This directory contains data and model code for the paper: 

Oliver Hellwig, Sven Sellmer, Sebastian Nehrdich, Paul Widmer, Rico Sennrich: Sanskrit Word Sense Disambiguation Based on Lexicographic Definitions. In: Proceedings of the ISCLS 2026.

`train-model01.sh` and `apply-model01.sh` are example slurm scripts for training and inference.

`/path/to/sentence/model` must point to a directory containing the qqq model available at https://huggingface.co/chronbmm/qqq.
