#Author:Ādiśeṣa
#Text:Paramārthasāra
#Data entry:Daniele Cuneo
#Edition:Henry Danielson: The Essence of Supreme Truth (Paramārthasāra). Sanskrit text with translation and notes. Leiden 1980 (Nisaba 10).

paraṃ parasyāḥ prakṛter anādim ekaṃ niviṣṭaṃ bahudhā guhāsu /
sarvālayaṃ sarvacarācarasthaṃ tvām eva viṣṇuṃ śaranaṃ prapadye // 1 //
ātmāmburāśau nikhilo 'pi loko magno 'pi nācāmati nekṣate ca /
āścaryam etan mṛgatṛṣṇikābhe bhavāmburāśau ramate mṛṣaiva // 2 //
garbhagṛhavāsasambhavajanmajarāmaraṇaviprayogābdhau /
jagad ālokya nimagnaṃ prāha guruṃ prāñjaliḥ śiṣyaḥ // 3 //
tvaṃ sāṅgavedavettā bhettā saṃśayagaṇasya ṛtavaktā /
saṃsārārṇavataraṇapraśnaṃ pṛcchāmy ahaṃ bhagavan // 4 //
dīrghe 'smin saṃsāre saṃsarataḥ kasya kena sambandhaḥ /
karma śubhāśubhaphaladam anubhavati gatāgatair iha kaḥ // 5 //
karmaguṇajālabaddho jīvaḥ saṃsarati kośakāra iva /
mohāndhakāragahanāt tasya kathaṃ bandhanān mokṣaḥ // 6 //
guṇapuruṣavibhāgajñe dharmādharmau na bandhakau bhavataḥ /
iti gaditapūrvavākyaiḥ prakṛtiṃ puruṣaṃ ca me brūhi // 7 //
ity ādhāro bhagavān pṛṣṭaḥ śiṣyeṇa taṃ sa hovāca /
viduṣām apy atigahanaṃ vaktavyam idaṃ śṛṇu tathāpi tvam // 8 //
satyam iva jagad asatyaṃ mūlaprakṛter idaṃ kṛtaṃ yena /
taṃ praṇipatyopendraṃ vakṣye paramārthasāram idam // 9 //
avyaktād aṇḍam abhūd aṇḍād brahmā tataḥ prajāsargaḥ /
māyāmayī pravṛttiḥ saṃhriyata iyaṃ punaḥ kramaśaḥ // 10 //
māyāmayo 'py acetā guṇakaraṇagaṇaḥ karoti karmāṇi /
tadadhiṣṭātā dehī sacetano 'pi na karoti kiṃcid api // 11 //
yadvad acetanam api san nikaṭasthe bhrāmake bhramati loham /
tadvat karaṇasamūhaś ceṣṭati cidadhiṣṭhite dehe // 12 //
yadvat savitary udite karoti karmāṇi jīvaloko 'yam /
na ca tāni karoti ravir na kārayati tadvad ātmāpi // 13 //
manaso 'haṃkāravimūrcchitasya caitanyabodhitasyeha /
puruṣābhimānasukhaduḥkhabhāvanā bhavati mūḍhasya // 14 //
kartā bhoktā draṣṭāsmi karmaṇām uttamādīnām /
iti tat svabhāvavimalo 'bhimanyate sarvago 'py ātmā // 15 //
nānāvidhavarṇānāṃ varṇaṃ dhatte yathāmalaḥ sphaṭikaḥ /
tadvad upādher guṇabhāvitasya bhāvaṃ vibhur dhatte // 16 //
gacchati gacchati salile dinakarabimbaṃ sthite sthitiṃ yāti /
antaḥkaraṇe gacchati gacchaty ātmāpi tadvad iha // 17 //
rāhur adṛśyo 'pi yathā śaśibimbasthaḥ prakāśate jagati /
sarvagato 'pi tathātmā buddhistho dṛśyatām eti // 18 //
sarvagataṃ nirupamam advaitaṃ tac cetasā gamyam /
yad buddhigataṃ brahmopalabhyate śiṣya bodhyaṃ tat // 19 //
buddhimano'haṃkārās tanmātrendriyagaṇāś ca bhūtagaṇaḥ /
saṃsārasargaparirakṣaṇakṣamāḥ prākṛtā heyāḥ // 20 //
dharmādharmau sukhaduḥkhakalpanā svarganarakavāsaś ca /
utpattinidhanavarṇāśramā na santīha paramārthe // 21 //
mṛgatṛṣṇāyām udakaṃ śuktau rajataṃ bhujaṃgamo rajjvām /
taimirikacandrayugavad bhrāntaṃ nikhilaṃ jagadrūpam // 22 //
yadvad dinakara eko vibhāti salilāśayeṣu sarveṣu /
tadvat sakalopadhiṣv avasthito bhāti paramātmā // 23 //
kham iva ghaṭādiṣv antar bahiḥ sthitaṃ brahma sarvapiṇḍeṣu /
dehe 'ham ity anātmani buddhiḥ saṃsārabandhāya // 24 //
sarvavikalpanahīnaḥ śuddho buddho 'jarāmaraḥ śāntaḥ /
amalaḥ sakṛd vibhātaś cetana ātmā khavad vyāpī // 25 //
rasaphāṇitaśarkarikāguḍakhaṇḍā vikṛtayo yathaivekṣoḥ /
tadvad avasthābhedāḥ paramātmany eva bahurūpāḥ // 26 //
vijñānāntaryāmiprāṇavirāḍdehajātipiṇḍāntāḥ /
vyavahārās tasyātmany ete 'vasthāviśeṣāḥ syuḥ // 27 //
rajjvāṃ nāsti bhujaṃgaḥ sarpabhayaṃ bhavati hetunā kena /
tadvad dvaitavikalpabhrāntir avidyā na satyam idam // 28 //
etat tad andhakāraṃ yad anātmany ātmatā bhrāntyā /
na vidanti vāsudevaṃ sarvātmānaṃ narā mūḍhāḥ // 29 //
prāṇādyanantabhedair ātmānaṃ saṃvitatya jālam iva /
saṃharati vāsudevaḥ svavibhūtyā krīḍamāna iva // 30 //
tribhir eva viśvataijasaprājñais tair ādimadhyanidhanākhyaiḥ /
jāgratsvapnasuṣuptair bhramabhūtaiś chāditaṃ turyam // 31 //
mohayatīvātmānaṃ svamāyayā dvaitarūpayā devaḥ /
upalabhate svayam evaṃ guhāgataṃ puruṣam ātmānam // 32 //
jvalanād dhūmodgatibhir vividhākṛtir ambare yathā bhāti /
tadvad viṣṇau sṛṣṭiḥ svamāyayā dvaitavistarā bhāti // 33 //
śānta iva manasi sānte hṛṣṭe hṛṣṭa iva mūḍha iva mūḍhe /
vyavahārastho na punaḥ paramārthata īśvaro bhavati // 34 //
jaladaharadhūmodgatibhir malinīkriyate yathā na gaganatalam /
tadvat prakṛtivikārair aparāmṛṣṭaḥ paraḥ puruṣaḥ // 35 //
eksaminn api ca ghaṭe dhūmādimalavṛte śeṣāḥ /
na bhavanti malopetā yadvad jīvo 'pi tadvad iha // 36 //
dehendriyeṣu niyatāḥ karma guṇāḥ kurvate svabhogārtham /
nāhaṃ kartā na mameti jānataḥ karma naiva badhnāti // 37 //
anyaśarīreṇa kṛtaṃ karma bhaved yena deha utpannaḥ /
tad avaśyaṃ bhoktavyaṃ bhogād eva kṣayo 'sya nirdiṣṭaḥ // 38 //
prāgjñānotpatticitaṃ yat karma jñānaśikhiśikhālīḍham /
bījam iva dahanadagdhaṃ janmasamarthaṃ na tad bhavati // 39 //
jñānotpatter ūrdhvaṃ kriyamānaṃ karma yat tad api nāma /
na śliṣyati kartāraṃ puṣkarapattraṃ yathā vāri // 40 //
vāgdehamānasair iha karmacayaḥ kriyata iti buddhāḥ prāhuḥ /
eko 'pi nāham eṣāṃ kartā tatkarmaṇāṃ nāsmi // 41 //
karmaphalabījanāśāj janmavināśo na cātra saṃdehaḥ /
buddhvaivam apagatatamāḥ savitevābhāti bhārūpaḥ // 42 //
yadvad iṣīkātūlaṃ pavanoddhūtaṃ hi daśa diśo yāti /
brahmaṇi tattvajñānāt tathaiva karmāṇi tattvavidaḥ // 43 //
kṣīrād uddhṛtam ājyaṃ kṣiptaṃ yadvan na pūrvavat tasmin /
prakṛtiguṇebhyas tadvat pṛthakkṛtaś cetano nātmā // 44 //
guṇamayamāyāgahanaṃ nirdhūya yathā tamaḥ sahasrāṃśuḥ /
bāhyābhyantaracārī saindhavaghanavad bhavet puruṣaḥ // 45 //
yadvad deho 'vayavā mṛd eva tasya vikārajātāni /
tadvat sthāvarajaṅgamam advaitaṃ dvaitavad bhāti // 46 //
ekasmāt kṣetrajñād bahvyaḥ kṣetrajñajātayo jātāḥ /
lohagatād iva dahanāt samantato viṣphuliṅgagaṇāḥ // 47 //
te guṇasaṃgamadoṣād baddhā iva dhānyajātayaḥ svatuṣaiḥ /
janma labhante tāvad yāvan na jñānavahninā dagdhāḥ // 48 //
triguṇā caitanyātmani sarvagate 'vasthite 'khilādhāre /
kurute sṛṣṭim avidyā sarvatra spṛśyate tayā nātmā // 49 //
rajjvāṃ bhujaṃgahetuḥ prabhavavināśau yathā na staḥ /
jagadutpattivināśau na ca kāraṇam asti tadvad iha // 50 //
janmavināśanagamanāgamamalasambandhavarjito nityam /
ākāśa iva ghaṭādiṣu sarvātmā sarvadopetaḥ // 51 //
karmaśubhāśubhaphalasukhaduḥkhair yogo bhavaty upādhīnām /
tatsaṃsargād bandhas taskarasaṃgād ataskaravad // 52 //
dehaguṇakaraṇagocarasaṃgāt puruṣasya yāvad iha bhāvaḥ /
tāvan māyāpāśaiḥ saṃsāre baddha iva bhāti // 53 //
mātṛpitṛputrabāndhavadhanabhogavibhāgasaṃmūḍhāḥ /
janmajarāmaraṇamaye cakra iva bhrāmyate jantuḥ // 54 //
lokavyvahārakṛtāṃ ya ihāvidyām upāsate mūḍhāḥ /
te jananamaraṇadharmāṇo 'ndhaṃ tama etya khidyante // 55 //
himaphenabudbudā iva jalasya dhūmo yathā vahneḥ /
tadvat svabhāvabhūtā māyaiṣā kīrtitā viṣṇoḥ // 56 //
evaṃ dvaitavikalpāṃ bhramasvarūpāṃ vimohanīṃ māyām /
utsṛjya sakalaniṣkalam advaitaṃ bhāvayed brahma // 57 //
yadvat salile salilaṃ kṣīre kṣiraṃ samīraṇe vāyuḥ /
tadvad brahmaṇi vimale bhāvanayā tanmayatvam upayāti // 58 //
itthaṃ dvaitasamūhe bhāvanayā brahmabhūyam upayāte /
ko mohaḥ kaḥ śokaḥ sarvaṃ brahmāvalokayataḥ // 59 //
vigatopādhiḥ sphaṭikaḥ svaprabhayā bhāti nirmalo yadvat /
cidīpaḥ svaprabhayā tathā vibhātīha nirupādhiḥ // 60 //
guṇagaṇakaraṇaśarīraprāṇais tanmātrajātisukhaduḥkhaiḥ /
aparāmṛṣṭo vyāpī cidrūpo 'yaṃ sadā vimalaḥ // 61 //
draṣṭā śrotā ghrātā sparśayitā rasayitā grahītā ca /
dehī dehendriyadhīvivarjitaḥ syān na kartāsau // 62 //
eko naikatrāvasthito 'ham aiśvaryayogato vyāptaḥ /
ākāśavad akhilam idaṃ na kaścid apy atra saṃdehaḥ // 63 //
ātmaivedaṃ sarvaṃ niṣkalasakalaṃ yadaiva bhāvayati /
mohagahanād viyuktas tadaiva parameśvarībhūtaḥ // 64 //
yad yat siddhāntāgamatarkeṣu prabruvanti rāgāndhāḥ /
anumodāmas tat tat teṣāṃ sarvātmavādadhiyā // 65 //
sarvākāro bhagavān upāsyate yena yena bhāvena /
taṃ taṃ bhāvaṃ bhūtvā cintāmaṇivat samabhyeti // 66 //
nārāyaṇam ātmānaṃ jñātvā sargasthitipralayahetum /
sarvajñaḥ sarvagataḥ sarvaḥ sarveśvaro bhavati // 67 //
ātmajñas tarati śucaṃ yasmād vidvān bibheti na kutaścit /
mṛtyor api maraṇabhayaṃ na bhavaty anyat kutas tasya // 68 //
kṣayavṛddhivadhyaghātakabandhanamokṣair vivarjitaṃ nityam /
paramārthatattvam etad yad ato 'nyat tad anṛtaṃ sarvam // 69 //
evaṃ prakṛtiṃ puruṣaṃ vijñāya nirastakalpanājālaḥ /
ātmārāmaḥ praśamaṃ samāsthitaḥ kevalībhavati // 70 //
nalakadaliveṇuvāṇā naśyanti yathā svapuṣpam āsādya /
tadvat svabhāvabhūtāḥ svabhāvatāṃ prāpya naśyanti // 71 //
bhinne 'jñānagranthau chinne saṃśayagaṇe śubhe kṣīṇe /
dagdhe ca janmabīje paramātmānaṃ hariṃ yāti // 72 //
mokṣasya naiva kiṃcid dhāmāsti na cāpi gamanam anyatra /
ajñānamayagranther bhedo yas taṃ vidur mokṣam // 73 //
buddhvaivam asatyam idaṃ viṣṇor māyātmakaṃ jagadrūpam /
vigatadvandvopādhikabhogāsaṃgo bhavec chāntaḥ // 74 //
buddhvā vibhaktāṃ prakṛtiṃ puruṣaḥ saṃsāramadhyago bhavati /
nirmuktaḥ sarvakarmabhir ambujapattraṃ yathā salilaiḥ // 75 //
aśnan yadvā tadvā saṃvīto yena kenacic chāntaḥ /
yatra kvacana ca śāyī vimucyate sarvabhūtātmā // 76 //
hayamedhasaharāṇy apy atha kurute brahmaghātalakṣāṇi /
paramārthavin na puṇyair na ca pāpaiḥ spṛśyate vimalaḥ // 77 //
madakopaharṣamatsaraviṣādabhayaparuṣavarjy avāgbuddhiḥ /
niḥstotravaṣaṭkāro jaḍavad vicared agādhamatiḥ // 78 //
utpattināśavarjitam evaṃ paramārtham upalabhya /
kṛtakṛtyasaphalajanmā sarvagatas tiṣṭhati yatheṣṭam // 79 //
vyāpinam abhinnam itthaṃ sarvātmānaṃ vidhūtanānātvam /
nirupamaparamānandaṃ yo veda sa tanmayo bhavati // 80 //
tīrthe śvapacagṛhe vā naṣṭasmṛtir api parityajan deham /
jñānasamakālamuktaḥ kaivalyaṃ yāti hataśokaḥ // 81 //
puṇyāya tīrthasevā nirayāya śvapacasadananidhanagatiḥ /
puṇyāpuṇyakalaṃkasparśābhāve tu kiṃ tena // 82 //
vṛkṣāgrāc cyutapādo yadvad anicchan naraḥ kṣitau patati /
tadvad guṇapuruṣajño 'nicchann api kevalībhavati // 83 //
paramārthamārgasādhanam ārabhyāprāpya yogam api nāma /
suralokabhogabhogī muditamanā modate suciram // 84 //
viṣayeṣu sārvabhaumaḥ sarvajanaiḥ pūjyate yathā rājā /
bhuvaneṣu sarvadevair yogabhraṣṭas tathā pūjyaḥ // 85 //
mahatā kālena mahān mānuṣyaṃ prāpya yogam abhyasya /
prāpnoti divyam amṛtaṃ yat tat paramaṃ padaṃ viṣṇoḥ // 8 //
vedāntaśāstram akhilaṃ vilokya śeṣas tu jagadādhāraḥ /
āryapañcāśītyā babandha paramārthasāram idam // 87 //
//iti paramārthasāraṃ samāptam
