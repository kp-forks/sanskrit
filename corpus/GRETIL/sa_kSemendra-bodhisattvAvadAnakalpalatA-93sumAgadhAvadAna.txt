#Alternative:Bodhisattvāvadānakalpalatā 93
#Data entry:Klaus Wille
#Edition:P.L. Vaidya, Ksemendra's Avadanakalpalata, vol. II. Darbhanga: Mithila Institute 1959 (Buddhist Sanskrit Texts, 23), pp. 526-534 (Avadāna 93).


ślāghyā jayanti jinabhaktiviśeṣabhājāṃ śraddhāsudhāprasaranirjharaśīkarās te /
niścetano 'py ucitacetanatām ivaiti yaḥ pūjyapūjanavidhau kusumādivargaḥ // 93.1 //
śrāvastyāṃ vijanāsīnaṃ jinaṃ jetavane purā /
anāthapiṇḍado 'bhyetya bhagavantam abhāṣata // 93.2 //
tvadbhaktir iva sarvatra mahārhaguṇaviśrutā /
bhagavan prauḍhim āyātā kanyā mama sumāgadhā // 93.3 //
śrīmataḥ sārthanāthasya nagare puṇḍravardhane /
sūnur vṛṣabhadattākhyas tatpāṇigraham iṣyate // 93.4 //
kanyāṃ dadāmi tāṃ tasmai bhagavan yadi manyase /
tvadadhīnadhanaprāṇas tvadājñāśaraṇo hy aham // 93.5 //
ity ukte tena bhagavān vātsalyavimalāśayaḥ /
ko doṣas tanayā tasmai dīyatām ity abhāṣata // 93.6 //
śāstuḥ śāsanam ādāya sādaraṃ pranipatya tam /
anāthapiṇḍadaḥ śrīmān prayayau nijamandiram // 93.7 //
tataḥ sumāgadhāṃ kanyāṃ vibhavena mahīyasā /
arhāyaṃ sa dadau tasmai bhūriratnāmbarapradaḥ // 93.8 //
dattā dūrataraṃ deśam atha yāntī sumāgadhā /
bhagavaccaraṇasmṛtyā sabāṣpanayanābhavat // 93.9 //
cireṇa sā samāsādya nagaraṃ puṇḍravardhanam /
śuśrūṣābhiratā patyuḥ sadā bhartṛgṛhe 'vasat // 93.10 //
kadācid atha tāṃ śvaśrūr bhojyasaṃbhārakāriṇī /
uvāca dhanavatyākhyā saṃkhyātītavyayodyatā // 93.ll //
pūjyopakaraṇaṃ sajjaṃ kuru sarvaṃ sumāgadhe /
gṛhān naḥ prātar āgantā jagatpūjyaguṇo jinaḥ // 93.12 //
guravo gauravapadaṃ pūjyāḥ sarvajanasya te /
sameṣyanti kṣapaṇakāḥ saṃmohakṣapaṇodyatāḥ // 93.l3 //
ity uktā sā tayā tatra babhūvārambhatatparā /
jñātvā bhikṣugaṇāyeva tāṃ pūjāṃ parikalpitām // 93.14 //
athāpare 'hni viviśur nagnāḥ kṣapaṇakā gṛham /
ulluñcitakacaśmaśrusaṃkleśaniśitavratāḥ // 93.15 //
tān alajjān avasanān avalokya sumāgadhā /
māṣaśaṣpāśanābhyāsapīvarān mahiṣān iva // 93.16 //
lajjitā vāsasācchādya vadanaṃ gurusaṃnidhau /
khedanirvedavinatā śvaśrūjanam abhāṣata // 93.17 //
aho vatāyam ācāraḥ sucirād avalokitaḥ /
digambarāṇām apy agre yad ihāste vadhūjanaḥ // 93.18 //
ete bhavadgṛhe 'dāntā bhuñjante śṛṅgavarjitāḥ /
amānuṣatvān naiteṣāṃ lajjante nūnam aṅganāḥ // 93.19 //
asthāne bhavatāṃ bhaktiḥ ko 'yam ucchṛṅkhalaḥ kramaḥ /
na tyaktam aśanaṃ yena sa kathaṃ vastram ujjhati // 93.20 //
keśonmūlanakarmaṇaiva niśitaṃ nairghṛṇyam āveditaṃ
kaupīnāṃśukavarjanena sujane śīlasya vārtaiva kā /
dambhārambhabhayaṃkare ca vadane krodhaḥ svayaṃ lakṣyate
nagnānām aśanaiṣiṇāṃ niyaminām eṣāṃ paśūnām iva // 93.21 //
yatraite paśavaḥ pūjyās tatrotsāryā bhavanti ke /
athavā deśadoṣo 'yaṃ gatānugatikā sthitiḥ // 93.22 //
iti bruvāṇāṃ tāṃ śvaśrūr viṣaṇṇā pratyabhāṣata /
pitus te bhavane bhadre pūjyate vada kīdṛśaḥ // 93.23 //
sāvadan matpitur gehe pūjyate bhagavān jinaḥ /
kāruṇyāt sarvajagatāṃ kuśalātiśayodyataḥ // 93.24 //
dhyānādhīnaḥ stimitanayanaḥ pūrṇalāvaṇasindhur nāsāvaṃśaṃ vipulasaralaṃ setubhūtaṃ dadhānaḥ /
bhūṣāśūnyaprasṛtarucimatkarṇapāśābhirāmaḥ kāntyaivāsau kim api viduṣāṃ śāntim antas tanoti // 93.25 //
śirasi sahajaprājyālokaprasekamayaṃ maṇiṃ karipatikarākārau bāhūkavat kanakadyutiḥ /
karatalagatām lekhāṃ śaṅkhadhvajāmbujamālikāṃ śamamayamahāsāṃrājyārhaṃ bibharti sa lakṣaṇam // 93.26 //
tasyābhilāṣajanakasya mahāmunīnāṃ sarvābhilāṣakalanārahitaḥ svabhāvaḥ /
niḥśeṣitasmṛtibhuvaḥ pramadāśrayasya rāgojjhitasya sutarām adharaḥ sarāgaḥ // 93.27 //
mūrtir nirbharasaṃgamapraṇayinī maitrī manaḥśāyinī kṣāntis tanmayakāriṇī hṛdi dayā gāḍhaṃ samāśleṣiṇī /
sarvāśābharaṇasya bhūridayitāsaktasya saṃlakṣyate tasyāpūrvamaṇer ananyamahimā vairāgyagarbhaḥ śamaḥ // 93.28 //
pūjyaḥ sa bhavane 'smākaṃ yasya pravrajyayā satām /
dhatte śīladukūlānām nirāvaraṇatāṃ manaḥ // 93.29 //
viśvarakṣāmaṇer yasya smṛtyāpi bhavabhoginā /
rāgadveṣogradaṃṣṭreṇa bādhyate na punar janaḥ // 93.30 //
iti tasyā vacaḥ śvaśrūḥ śrutvā śrotrarasāyanam /
sadyaḥ prasādaviśadā tāṃ jagāda pramodinī // 93.31 //
api taddarśanopāyaḥ kaścid asti varānane /
api tvatpuṇyasaṃbandhād vayam apy amṛtāspadam // 93.32 //
iti sānunayaṃ śvaśrvā sā sādaradhiyārthitā /
taṃ vaḥ saṃdarśayāmīti babhāṣe bhaktimāninī // 93.33 //
mahāpratijñāsaṃbhārabharanirvahaṇaiṣiṇī /
sā saṃśayatulāruḍhā kṣaṇaṃ dhyānaparābhavat // 93.34 //
tataḥ prāsādam āruhya bhagavatsevitāṃ diśam /
praṇipatyāsṛjat pūjyapūjārhakusumāñjalim // 93.35 //
puṣpadhūpodakair arcāṃ sā kṛtvā tatpadonmukhī /
ānandabāṣpasaṃruddhavistīrṇanayanāvadat // 93.36 //
anukampyaiva bhagavan ratnatrayavivarjitā /
tavāśramamṛgīvāhaṃ dūraṃ deśam imam gatā // 93.37 //
tvatpādapadmayugalaṃ śaraṇam prapannāṃ dūrasthitām api dṛśā spṛśa māṃ dayālo /
vātsalyapeśaladhiyāṃ mahatāṃ pravāsadurīkṛteṣu karuṇā na tanutvam eti // 93.38 //
bhagavan dāsasutayā tvaṃ mayādya nimantritaḥ /
prātar āgamanenaiva mānaṃ vitara me vibho // 93.39 //
ity udīrya tayotsrṣṭā vicitrakusumāvalī /
sajīviteva prayayau nabhasā bhaktidūtikā // 93.40 //
sā śvetaraktaharitāsitapuṣpapālī khe dhūpadhūmaśabalā śanakaiḥ prayāntī /
bālāmbudapraṇayinī suciraṃ cakāśe saṃcālicāpalatikeva śacīdhavasya // 93.41 //
atha jetavanaṃ prāpya kṣaṇena kusumāvalī /
sā bhaktiśālinī śāstuḥ papāta caraṇābjayoḥ // 93.42 //
bhagavān api sarvajñaḥ sarvaṃ jñātvā samīhitam /
sumāgadhāyāḥ kāruṇyād ānandam avadat puraḥ // 93.43 //
prātar gantavyam asmābhir nagaraṃ puṇḍravardhanam /
sumāgadhā prārthayate sasaṃghasya mamārcanam // 93.44 //
śataṣaṣṭyadhikaṃ tatra yojanānām itaḥ param /
ekenāhnā ca gantavyaṃ na vilambo 'tra yujyate // 93.45 //
vyomnā maharddhyā śaknoti gantuṃ yo yaḥ prabhāvavān /
nimantraṇaśalākāṃ tvaṃ tasmai tasmai samarpaya // 93.46 //
preritaḥ sugateneti sa bhikṣubhyo nyavedayat /
ekāhagamanaprāpyaṃ śalākābhir nimantraṇam // 93.47 //
śalākāsu gṛhītāsu tatra sarvair maharddhibhiḥ /
pūrṇaḥ kumbhopadhānīyaḥ sthaviro 'py agrahīt kramāt // 93.48 //
tena prāptaprabhāvena śalākāyāṃ prasārite /
pāṇau prāha tam ānandaḥ kiṃcit smitasitānanaḥ // 93.49 //
anāthapiṇḍadagṛhaṃ na gantavyaṃ padadvayam /
saṣaṣṭiyojanaśataṃ dinārdhenaiva laṅghyate // 93.50 //
ity uktaḥ sthaviras tena vailakṣyavinatānanaḥ /
acintayat svavargāgre nyūnabhāvo hi duḥsahaḥ // 93.5l //
anādikālopacitāḥ kleśajanmajarādayaḥ /
hantuṃ sapatnaiḥ śakyante prāptum ṛddhipadaṃ kiyat // 93.52 //
iti cintayatas tasya tīvrasaṃvegayā dhiyā /
prādur āsīt kṣaṇenaiva maharddhiḥ śuddhacetasaḥ // 93.53 //
atha rātryāṃ vyatītāyāṃ prabhāte sarvabhikṣavaḥ /
nānātridaśaveśeṇa vimānair nabhasā yayuḥ // 93.54 //
atrāntare mahārambhasambhāraparipūrite /
sumāgadhā bhartṛgṛhe bhagavaddarśanotsukā // 93.55 //
saha prāsādam āruhya śvaśrūśvaśurabhartṛbhiḥ /
tasthau kusumadhūpārdhyāracanāsaṃgrahonmukhī // 93.56 //
bhikṣur ājñātakauṇḍinyas tato 'śvaratham āsthitaḥ /
divyarddhivividhāścaryaḥ prathamaṃ pratyadṛśyata // 93.57 //
taṃ dṛṣṭvā sūryasaṃkāśaṃ vismitāḥ śvaśurādayaḥ /
ūcuḥ sumāgadhāṃ prītyā kim eṣa bhagavān iti // 93.58 //
sāvadad bhagavān nāyaṃ dṛśyate taraṇiprabhaḥ /
ayam ājñātakauṇḍinyo bhikṣur akṣuṇṇadīdhitiḥ // 93.59 //
āpatatsu krameṇātha ratheṣu śvaśurādayaḥ /
kim ayaṃ kim ayaṃ bhadre bhagavān iti tāṃ jaguḥ // 93.60 //
sābravīn naiva śāstāyam ete tacchāsanocitāḥ /
bhikṣavaḥ praśamaślāghyās tapodīptataratviṣaḥ // 93.6l //
yaḥ kāntahemadrumaramyaśailaśrṅgādhirūḍhaḥ parato 'bhyupaiti /
āścaryakṛn mūrta iva prabhāvaḥ śrīmān mahākāśyapa eṣa bhikṣuḥ // 93.62 //
pañcānanasyandanam āsthito yaḥ satoyajīmūtagabhīraghoṣam /
vyomnā samabhyeti gunaiḥ pragītaḥ sa eṣa bhikṣur bhuvi śāriputraḥ // 93.63 //
āruhya kailāsam ivātiśubhraṃ dvipam caturdantam anantakāntiḥ /
āyāti yaḥ puṇyavatāṃ jagatsu maudgalyanāmā ca sa eṣa bhikṣuḥ // 93.64 //
vaidūryanālaṃ kanakāravindam āruhya ratnāṅkurakeśarāḍhyam /
upaiti yaḥ saurabhapūritāśaḥ sa eṣa bhikṣuḥ prathito 'niruddhaḥ // 93.65 //
yaś cāmbarāgraṃ garuḍādhirūḍhaḥ pakṣānilotsāritavārivāhaḥ /
vigāhate sphitavanena bhikṣur maitrāyaṇīsūnur ayaṃ sa pūrṇaḥ // 93.66 //
anantam āsthāya nitāntaśāntam īrṣyāpathaṃ sattvamahodadhir yaḥ /
prabhāmṛtaiḥ sarpati tarpitāśaḥ prabhāvavān eṣyajid eṣa bhikṣuḥ // 93.67 //
vilolavallīvalayābhirāmaṃ viśālam āruhya suvarṇatālam /
yaḥ puṇyapūrṇadyutir abhyupaiti sa eṣa bhikṣur matimān upālī // 93.68 //
āruhya vaiḍūryavimānaśṛṅgaṃ suvarṇaratnojjvalapatralekham /
limpann ivāgacchati yaḥ prabhābhiḥ kātyāyano nāma sa eṣa bhikṣuḥ // 93.69 //
śarīriṇaṃ dharmam ivādhiruhya vigāhate khaṃ vṛṣavāhano yaḥ /
pṛṣṭhaḥ pratiṣṭhāpariniṣṭhitānāṃ gariṣṭhadhīḥ kauṣṭhila eṣa bhikṣuḥ // 93.70 //
vimānahaṃsadyutibhir muhūrtaṃ smitormiramyaṃ muhur antarīkṣam /
kurvan samabhyeti taponidhir yaḥ pilindavatsāhvaya eṣa bhikṣuḥ // 93.7l //
yo 'yaṃ samutphullalatāvitānavanāntarāle viharann upaiti /
sa śroṇakoṭiḥ śruta eṣa bhikṣur akṣuṇṇalakṣmīr gṛhanirvyapekṣaḥ // 93.72 //
yaś cakravartī divi bhāti so 'yaṃ śāstuḥ suto rāhulakābhidhānaḥ /
hemaprabhābhūṣitadigvibhāgaḥ saṃlakṣyate merur ivānyarūpaḥ // 93.73 //
ete giribhyo 'tha digantarebhyaḥ kṣmāmaṇḍalād vyomataṭāntarāc ca /
āyānty asaṃkhyādbhutabhikṣusaṃghā vicitraratnāsanavāhanasthāḥ // 93.74 //
tayā krameṇeti nivedyamānaṃ te bhikṣusaṃghaṃ vimukhaṃ vilokya /
yayuḥ praharṣādbhutasaṃbhramāṇāṃ vidheyatāṃ tulyam ananyalakṣmyāḥ // 93.75 //
atha jvalatkāñcanacūrṇavarṇaṃ jagad babhūvārkaśataprakāśam /
aśeṣasaṃtāpaviśeṣaśāntyā śītāṃśumālāśataśītalaṃ ca // 93.76 //
atha dhanapatiśakrabrahmamukhyair amartyair vipulagaganayātrādattasevānuyātraḥ /
amarapurapurastrīkīrṇapuṣpaprabhāvān nayanapatham ayāsīt puṇyabhājāṃ jinendraḥ // 93.77 //
aṣṭādaśadvārapathā puraṃ tat sa tulyam aṣṭādaśamūrtir eva /
praviśya cakre śaśikāntaratnaśikhāmayaṃ veśma sumāgadhāyāḥ // 93.78 //
abhyarcitas tatra bahuprakāraiḥ pūrṇopacāraiḥ praṇipatya sarvaiḥ /
apūjayat paurajanaḥ samantād bahiś ca bhittipratibimbitaṃ tam // 93.79 //
sumāgadhāyā dayayā dayāluḥ pūjāṃ gṛhītvā bhagavān sasaṃghaḥ /
anugrahālokanasaṃvibhāgaiḥ sarvānvavāyā vidadhe prasādam // 93.80 //
sumāgadhā saśvaśurādivargā sahāparaiḥ paurajanaiś ca sarvāḥ /
śāstus tayā deśanayā babhūvuḥ śuddhāśayās tatkṣaṇadṛṣṭasatyāḥ // 93.81 //
sumāgadhāyāḥ kuśalānubandhaṃ puṇyaṃ prabhāvaṃ vipulaṃ vilokya /
te bhikṣavas tatra kutūhalena papracchū ramyaṃ jinam ādivṛttam // 93.82 //
pṛṣṭaḥ sa taiḥ saṃsadi sarvadarśī sumāgadhāyāḥ kuśalasya hetum /
dantaprabhābhiḥ kakubhāṃ mukheṣu diśan prakāśam bhagavān babhāṣe // 93.83 //
vārāṇasyām abhūt pūrvaṃ kṛkeḥ kantasya bhūpateḥ /
sutā kāñcanamālākhyā kucakāñcanamālikā // 93.84 //
kāśyapākhyasya śāstuḥ sā satataṃ bhaktiśālinī /
paricaryāṃ vyadhāt sārdhaṃ sakhīnāṃ pañcabhiḥ śataiḥ // 93.85 //
sa kadācin narapatir vikṛtasvapnadarśanāt /
bhayasaṃśayasaṃbhrāntaḥ papraccha phalakovidān // 93.86 //
te taṃ rājasutādveṣān nimittajñā babhāṣire /
atipriyasya hṛdayaṃ hutvāgnau labhyate śivam // 93.87 //
iti teṣām anādṛtya vacaḥ krūrataraṃ nṛpaḥ /
bhagavantaṃ yayau draṣṭuṃ kāśyapaṃ duhitur girā // 93.88 //
sa tam etyāvadat svapnaḥ savikāraḥ paraṃ mayā /
dṛṣṭo 'dya sarvaṃ sarvajña tatphalaṃ vaktum arhasi // 93.89 //
vātāyanena nirgacchan ruddhapuccho mayā gajaḥ /
tṛṣitasya tathā paścāt kupo dhāvan vilokitaḥ // 93.90 //
tṛptaś ca śaktuprasthena mauktiprasthavikrayaḥ /
samīkṛtāni dṛṣṭāni kudārūṇi ca candanaiḥ // 93.9l //
kalabhena mahāhastī samāhūtas tathāhave /
parān aśuciliptāṅgaḥ pralimpan viplutaḥ kapiḥ // 93.92 //
sphītarājyām abhiṣiktaś ca kucāpalanidhiḥ kapiḥ /
paṭo 'ṣṭādaśabhiḥ kṛṣṭaḥ puruṣair apy asaṃkṣayaḥ // 93.93 //
ramyapuṣpaphalārāmaś caurair api viluṇṭhitaḥ /
dveṣopahāsakalahāsaktaś ca vipulo janaḥ // 93.94 //
svapnādbhutānām eteṣāṃ manye ghorataraṃ phalam /
iti pṛṣṭaḥ kṣitibhujā bhagavān kāśyapo 'bravīt // 93.95 //
śatāyuṣi jane śāstā śāntaḥ śākyamunir jinaḥ /
bhaviṣyaty amṛtāmbhodhiḥ sa drṣṭaḥ kuñjaras tvayā // 93.96 //
tasyāpi paścime kāle śrāvakāḥ kalisaṃśrayāt /
tyaktaśīlaguṇācārā bhaviṣyanti saviplavāḥ // 93.97 //
apakvālpavivekānāṃ balāt te gṛhavāsinām /
svayaṃ sevāṃ samālambya kariṣyanty eva deśanām // 93.98 //
arthanīyo 'rthibhāvena yasmāt sevāsu dhāvati /
tṛṣitasya vrajan paścāt kūpas tasmād vilokitaḥ // 93.99 //
te kariṣyanti lobhāndhaḥ saṃmohopahatāḥ param /
bodhyaṅgamuktāprasthasya śaktuprasthena vikrayam // 93.loo //
tīrthavākyakudārūṇi buddhabhāṣitacandanaiḥ /
sāmyam āpādayiṣyanti te maugdhyād aviśeṣiṇaḥ // 93.101 //
kvacid bhadraṃ samāsādya vinītaṃ bhikṣukuñjaram /
duḥśīlakalabho bhikṣuḥ spardhayā dhik kariṣyati // 93.102 //
cāpalāśuciliptāṅgaḥ suśīlān bhikṣumarkaṭaḥ /
anulimpan svadoṣeṇa kariṣyaty ātmasannibhān // 93.103 //
ṣaṇḍakasyābhiṣekaś ca bhaviṣyati kaper iva /
saṃbuddhaśāsanapaṭaṃ kṛṣyamāṇaṃ na naṃkṣyati // 93.104 //
saṃghadravyaphalārāmeṣv api yāsyanti cauratām /
mithaḥ kalahaśīlāś ca bhaviṣyanty apavādinaḥ // 93.105 //
tava svapnavipākānte phalāny etāni bhūtale /
iti śāstur vacaḥ śrutvā vismito 'bhūn mahīpatiḥ // 93.106 //
sānugasyātha nṛpateḥ sa śāstā dharmadeśanām /
kṛtvā kāñcanamālāyā dideśa kuśalārhatām // 93.l07 //
janmāntale cakārārcāṃ stūpe nāraṅgamālayā /
sā tena sukṛtenaiva jātā hemasrajāṅkitā // 93.108 //
seyaṃ sumāgadhā puṇyaprabhāveṇa mahīyasā /
prāptādya janakāyasya samyak kuśalasetutām // 93.109 //
adhidhāyeti bhagavān nabhasā bhikṣubhiḥ saha /
rucā saṃpūrayann āśāṃ yayau jetavanaṃ jinaḥ // 93.ll0 //
puṃsāṃ satkulabhūtaye balavatī mithyaiva putraspṛhā sūnuś ced aguṇas tad eva viphalaṃ tat kiṃ na sarvaṃ kulam /
sā kanyā kila jāyate guṇavatī puṇyaprabhāvād yayā saṃsārorusaritpatau kulayugam nāv eva saṃtāryate // 93.111 //
||iti kṣemendraviracitāyāṃ bodhisattvāvadānakalpalatāyāṃ sumāgadhāvadānaṃ trinavatitamaḥ pallavah
