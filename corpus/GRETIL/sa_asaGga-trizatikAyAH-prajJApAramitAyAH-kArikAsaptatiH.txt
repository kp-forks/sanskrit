#Author:Asaṅga
#Text:Triśatikāyāḥ prajñāpāramitāyāḥ kārikāsaptatiḥ
#Data entry:Klaus Wille
#Edition:G. Tucci: Minor Buddhist Texts, part I. Roma 1956 (Serie Orientale Roma, 9), pp. 54-92.



paramo 'nugraho jñeyaḥ śārīraḥ saparigrahaḥ /
prāptāprāptāvihānau ca paramā syāt parīndanā // 1 //
vipulaḥ paramo 'tyanto 'viparyastaś ca cetasi /
upakārāśayaḥ sthānaṃ yāne 'smin guṇapūritaḥ // 2 //
dānaṃ pāramitāṣatkam āmiṣābhayadharmataḥ /
ekadvayatrayeṇeha pratipat sā 'pratiṣṭhitā // 3 //
ātmabhāve pratikṛtau vipāke cāpy asaktatā /
apravṛttitadanyārthapravṛttiparivarjane // 4 //
pragraho maṇḍale tredhā nimittāc cittavāraṇam /
uttarottarasaṃdehajanmataś ca nivāraṇā // 5 //
saṃskṛtatvena saṃkalpya saṃpat prāptau nivāryate /
trailakṣaṇyānyathābhāvāt tadabhāvāt tathāgataḥ // 6 //
sahetuphalagāmbhīryadeśanāsmin yugādhame /
na niṣphalā yataḥ santi bodhisattvās trayānvitāḥ // 7 //
śikṣayopāsānāt pūrvaṃ kuśalasyāvaropaṇāt /
śīlavanto 'nyabuddheṣu guṇavantaś ca kīrtitāḥ // 8 //
sapudgaleṣu dharmeṣu saṃjñāyā viprahāṇataḥ /
prajñāvantaś ca saṃjñāyā aṣṭadhāṣṭārthabhedataḥ // 9 //
pṛthagbhāvena saṃtatyā vṛtter ājīvitasthiteḥ /
punaś ca gatilīnatvād ātmasaṃjñā caturvidhā // 10 //
sarvābhāvād abhāvasya sadbhāvān nābhilāpyataḥ /
abhilāpaprayogāc ca dharmasaṃjñā caturvidhā // 11 //
adhimuktivaśāt teṣāṃ bhūtasaṃjñā prasādataḥ /
yathārutāgrahāt saṃyagdeśitatvasya codgrahāt // 12 //
phalato na mitā buddhaiḥ praṇidhijñānalakṣitāḥ /
iābhasatkārakāmānāṃ tadvādavinivṛttaye // 13 //
asthānād ānukūlyāc ca dharmeṣv adhigamasya hi /
kolasyeva parityāgo dharme saṃdhis tato mataḥ // 14 //
nairmāṇikena no buddho dharmo nāpi ca deśitaḥ /
deśitas tu dvayāgrāhyo 'vācyo 'vākpathalakṣaṇāt // 15 //
grahaṇadeśaṇā cāsya nāpārthā puṇyasaṃgrahāt /
puṇyaṃ bodhyanupastaṃbhād upastaṃbhād dvayasya ca // 16 //
svābhāvikāptihetutvāt tadanyasya ca janmanaḥ /
kaivalyād buddhadharmāṇām agryatvaṃ puṇyasādhanam // 17 //
agrāhyānabhilapyatvaṃ svaphalānām anudgrahāt /
dvayāvaraṇanirmokṣāt subhūtāv araṇādvayaṃ // 18 //
buddhadīpaṃkarāgrāhād vakyenādhigamasya hi /
tataś cādhigame siddhā agrāhyānabhilāpyatā // 19 //
jñānaniṣyandavijñaptimātratvāt kṣetranodgrahaḥ /
avigrahatvād agratvād avyūhaṃ vyūhatā matā // 20 //
sumeror iva rājatve saṃbhoge nāsti codgrahaḥ /
sāsravatvena cābhāvāt saṃskṛtatvena cāsya hi // 21 //
bahutvabhedakhyātyarthaṃ viśeṣasya ca siddhaye /
paurvāparyeṇa puṇyasya punar dṛṣṭāntadeśanā // 22 //
dvayasya pātrīkaraṇān niṣyandatvamahatvataḥ /
asaṃkleśasya hetutvād dhīnābhibhavanād api // 23 //
tatphalaśreṣṭhaduḥkhatvād durlabhārthottamārthataḥ /
jñeyāpāramitatvāc ca parāsādhāraṇatvataḥ // 24 //
gāḍhagaṃbhīrabhāvāc ca parasūtraviśiṣṭataḥ /
mahāśuddhānvayatvāc ca puṇyāt puṇyaṃ viśiṣyate // 25 //
sahiṣṇutā ca caryāyāṃ duṣkarāyāṃ śubhā yataḥ /
tadguṇāparimāṇatvād agrārthena nirucyate // 26 //
ātmavyāpādasaṃjñāyā abhāvād duḥkhatā na ca /
sasukhā karuṇābhāvāc caryāduḥkhaphalā tataḥ // 27 //
cittātyāgābhinirhāre yatnaḥ kāryo dṛḍho yataḥ /
kṣāntipāramitāprāptau tatprāyogika eva ca // 28 //
pratipattiś ca sattvārthā vijñeyā hetubhāvataḥ /
sattvavastunimittāt tu vijñeyā parivarjitā // 29 //
nāmaskandhāś ca tadvastu tatsaṃjñāpagamāj jine /
tadabhāvo hi buddhānāṃ tattvadarśanayogataḥ // 30 //
phalāpratiṣṭhito mārgas tatphalasyāpi kāraṇam /
buddhānāṃ bhūtavāditvāt tac ca jñeyaṃ caturvidham // 31 //
pratijñā hīnayānasya mahāyānasya deśanā /
sarvavyākaraṇānāṃ ca na visaṃvādinī yataḥ // 32 //
aprāpter ānukulyāc ca na satyā na mṛṣā matā /
yathārutaniveśasya pratipakṣeṇa deśanā // 33 //
............. alābhatā /
ajñānāt sapratiṣṭhena jñānād anyena lābhatā // 34 //
tamaḥprakāśam ajñānaṃ jñānam ālokavan matam /
pratipakṣavipakṣasya lābhahānyāmukhatvataḥ // 35 //
yādṛśyā pratipattyā......... /
yat karmikā ca sā dharme pratipattis tad ucyate // 36 //
vyañjane trividhā dharmadharatve śrutavistare /
arthasya parato 'dhyātmam āptau śravaṇacintanāt // 37 //
............ paripācane /
vastukālamahatvena puṇyāt puṇyaṃ viśiṣyate // 38 //
agocaratvaṃ kaivalyaṃ mahātmāśritatā tathā /
durlabhaśravatā caiva dhātupuṣṭir anuttarā // 39 //
........... pātratāśraye /
śodhanāvaraṇānāṃ ca kṣiprābhijñātvam eva ca // 40 //
vicitraiokasaṃpattivipākaḥ sumahānn api /
karmāṇi etāni dharme pratipatter matāni. // 41 //
......... bodhisattvakalpanā /
cittāvaraṇam ākhyātaṃ yac cittam apratiṣṭhitaṃ // 42 //
paścādvyākaraṇān no ca caryā dīpaṃkare parā /
bodhis tac caryayā tulyā na sa...... // 43 //
........ na mṛṣā paridīpitā /
dharmās tato buddhadharmāḥ sarve 'bhāvasvalakṣaṇāḥ // 44 //
dharmakāyena buddhas tu mataḥ saḥ puruṣopamaḥ /
nirāvaraṇato........... // 45 //
guṇamahātmyataś cāpi mahākāyaḥ sa eva hi /
abhāvakāyabhāvāc ca akāyo 'sau nirucyate // 46 //
dharmadhātāv akuśalaḥ sattvanirvāpaṇe matiḥ /
kṣetrāṇāṃ śodhane caiva........ // 47 //
sattvānāṃ bodhisattvānāṃ dharmān yaś ca nairātmakān /
buddhyādhimucyate 'nārya āryo dhīmān sa kaṭhyate // 48 //
nopalambhe 'pi dharmāṇāṃ cakṣur na hi na vidyate /
buddhānāṃ pañcadhā tac ca vitathārthasya darśanāt // 49 //
nānāvithatavijñapteḥ sṃṛtyupasthānavarjanāt /
nirādhāraprabandho 'syā vitathāto nirucyate // 50 //
jñānasyādhārato jñeyā puṇye vitathatā na ca /
tataḥ puṇyanimittaṃ hi punar dṛṣṭāntakīrtanam // 51 //
na dharmakāyaniṣpattir anuvyañjanam ucyate /
na ca lakṣaṇasaṃpattis tad akāyatvato matā // 52 //
dharmakāyāvinirbhāgān na dvayaṃ na tathāgataḥ /
saṃpattir ucyate bhūyo dvayaṃ nāsty astitā tataḥ // 53 //
............. kalpitā /
dharmakāyāvinirbhāgād deśanāpy asvalakṣaṇā // 54 //
deśyadaiśikagāṃbhīryaśraddhā na ca na santi hi /
na sattvā nāpi cāsattvās te 'nāryārya.... // 55 //
........ jñeyā bodhir anuttarā /
na vṛddhyā dharmadhātau hi śuddhisāmyāt svalakṣaṇāt // 56 //
upāyānuttaratvāc ca sāsravatvād adharmataḥ /
śubhā na dharmā........... // 57 //
naiva? cāvyākṛtatve 'pi deśanā 'prāptaye matā /
dharmaratnaṃ tataś caikaṃ ratnād anyād viśiṣyate // 58 //
saṃkhyāprabhavajātīnāṃ saṃbandhasya viśeṣaṇe /............. labhyate // 59 //
samatvād dharmadhātoś ca na sattvā mocitā jinaiḥ /
sahanāmnā yataḥ skandhā dharmadhātvabahirgatāḥ // 60 //
ātmagrāhasamo dosas ta....... /
..... grāhe hi agrāhagrāhatā matā // 61 //
na caiva rūpakāyena so 'numeyas tathāgataḥ /
dharmakāyo yataś cakravartī mābhūt tathāgataḥ // 62 //
na ca lakṣaṇavaipākyapuṇy....... /
dharmakāyasya lābho hi upāyo yad vilakṣaṇaḥ // 63 //
rūpānuśravamātreṇa na buddhajñaḥ pṛthagjanaḥ /
tathatādharmakāyo hi yato 'vijñānagocaraḥ // 64 //
na ca puṇyas............ /
kṣāntilābhe 'pi nocchedo nirmalasyāsya lābhataḥ // 65 //
punaḥ puṇyanimittaṃ hi tasmād dṛṣṭāntadeśanā /
tat puṇyasyāvipākatvān nodgrahaḥ saparigrahaḥ // 66 //
tan nirmāṇaphalaṃ teṣāṃ puṇya...... /
anābhogena yat karma buddhāḥ kurvanti dikṣu ca // 67 //
gatyādayas tu nirmāṇair buddhās tv avicalāḥ sadā /
dharmadhātau ca tatsthānaṃ naikatvānyatvato matam // 68 //
rajomaṣīkriyā dhātor dṛṣṭāntas tasya dyotakaḥ /
maṣīkaraṇatā kleśakṣayasyeha nidarśanam // 69 //
asaṃcayatvāpiṇḍatvam anekatvanidarśanam /
saṃhatasthānatā tasmin nānyatve ca nidarśanam // 70 //
vyavahāramātratāyā bālānām udgraho 'nyathā /
dvayābhāvān na bodhyāptiḥ prahāṇād ātmadharmayoḥ // 71 //
tasmād dṛṣṭir adṛṣṭiś ca nairarthyābhūtakalpataḥ /
sūkṣmam āvaraṇaṃ hy etat tathā jñānāt prahīyate // 72 //
jñānadvayasamādhānapraheyaṃ tac ca deśitam /
nirmāṇaiḥ kaśaṇāt puṇyaṃ tad buddhānāṃ na nākṣayam // 73 //
nirmito 'smīti cātmānaṃ kāśayantas tathāgatāḥ /
prakāśayanti nātmānaṃ tasmāt sā kāśanā satī // 74 //
saṃskāro na tathā nānyaṃ nirvāṇaṃ hi tathāgate /
navadhā saṃbhūtasyeha saṃyagjñānaparīkṣaṇāt // 75 //
dṛṣṭir nimittaṃ vijñānaṃ pratiṣṭhādehabhogatā /
atītaṃ vartamānaṃ ca parīkṣyaṃ cāpy anāgatam // 76 //
lakṣaṇasyopabhogasya pravṛtteś ca parīkṣaṇāt /
nirmalāṃ teṣu vaśitāṃ saṃskāreṣu samāpnute // 77 //
triśatikāyāḥ prajñāpāramitāyāḥ kārikāsaptatiḥ samāptā //
kṛtir iyam āryāsaṅgapādānām iti //
