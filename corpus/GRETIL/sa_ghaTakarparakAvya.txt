#Text:Ghaṭakarparakāvya
#Data entry:Daniele Cuneo
#Edition:B. Parlier: La Ghaṭakarparavivṛti d'Abhinavagupta, Paris: Institut de civilisation indienne 1975. (Publications de l'Institut de civilisation indienne, 39).



chādite dinakarasya bhāvane khāj jale patati śokabhāvane /
manmathe ca hṛdi hantum udyate proṣitapramadayedam udyate // 1  //
sarvakālam ativāhya toyadā āgatāḥ stha dayito gato yadā /
nirdayena paradeśasevinā māriṣyatha hi tena māṃ vinā // 2  //
brūta taṃ pathikapāṃsulaṃ ghanā yūyam eva pathiśīghralaṅgan /
anyadeśaratir adya mucyatāṃ sāthavā tava vadhūḥ kim ucyatām // 3  //
haṃsapaṅktir api nātha samprati prasthitā viyati mānasaṃ prati /
cātakaś ca tṛṣito 'mbu yācate duḥkhitā pathika sā priyā ca te // 4  //
nīlaśaṣpam abhibhāti komalaṃ vāri vindati ca cātako 'malam /
ambudaiḥ śikhigaṇo vinādyate kā ratiḥ priya mayā vinādya te // 5  //
meghaśabdamuditāḥ kalāpinaḥ proṣitāhṛdayaśokalāpinaḥ /
toyadāgamakṛśā ca sādya te durdhareṇa madanena sādyate // 6  //
kiṃ kṣamāpi tava nāsti kāntayā pāṇḍugaṇḍapatitālakāntayā /
śokasāgarajale ca pātitāṃ tvadguṇasmaraṇam eva pāti tām // 7  //
kusumitakuṭajeṣu kānaneṣu priyarahiteṣu samutsukānaneṣu /
dravati ca kaluṣaṃ jalaṃ nadīnāṃ kim iti ca māṃ samavekṣase na dīnām // 8  //
mārgeṣu meghasalilena vināśiteṣu kāmo dhanuḥ spṛśati tena vinā śiteṣu /
gambhīramegharasitavyathitā kadāhaṃ jahyāṃ sakhi priyaviyogajaśokadāhaṃ // 9  //
susugandhitayā vane 'jitānāṃ svanadambhodharavātavījitānām /
madanasya kṛte niketanānāṃ pratibhānty adya vanāni ketanānām // 10  //
tat sādhu yat tvāṃ sutaruṃ sasarja prajāpatiḥ kāmanivāsa sarja /
tvaṃ mañjarībhiḥ pravaro vanānāṃ netrotsavaś cāsi sayauvanānām // 11  //
navakadamba śiro 'vanatāsmi te vasati vo madanaḥ kusumasmite /
kuṭaja kiṃ kusumair avahasyate praṇipatāmi ca duṣprahasya te // 12  //
taruvara vinatāsmi te sadāhaṃ hṛdayaṃ me prakaroṣi kiṃ sadāham /
tava puṣpanirīkṣitāpadehaṃ visṛjeyaṃ sahasaiva nīpa deham // 13  //
kusumair upaśobhitāṃ sitair ghanamuktāmbulavaprahāsitaiḥ /
madhunaḥ samavekṣya kālatāṃ bhramaraś cumbati yūthikālatām // 14  //
nicitaṃ kham upetya nīradaiḥ priyahīnahṛdayāvanīradaiḥ /
salilair vihataṃ rajaḥ kṣitau ravicandrāv api nopalakṣitau // 15  //
haṃsā nadanmeghabhayād dravanti niśāmukhāny adya na candravanti /
navāmbumattāḥ śikhino nadanti meghāgame kundasamānadanti // 16  //
meghāvṛttaṃ niśi na bhāti nabho vitāraṃ nidrābhyupaiti ca hariṃ śubhasevitāram /
sendrayudhaś ca jalado 'dya rasann ibhānāṃ saṃrambham āvahati bhūdharasannibhānām // 17  //
sataḍijjaladārpitaṃ nageṣu nadadambhodharabhītapannageṣu /
paridhīraravaṃ jalaṃ darīṣu prapaty adbhutarūpasundarīṣu // 18  //
kṣipraṃ prasādayati samprati ko 'pi tāni kāntāmukhāni rativigrahakopitāni /
utkaṇṭhayanti pathikāṇ jaladāḥ svanantaḥ śoko vivardhati ca tadvinatāsv anantaḥ // 19  //
tāsām ṛtuḥ saphala eva hi yā dineṣu sendrāyudhāmbudharagarjitadurdineṣu /
ratyutsavaṃ priyatamaiḥ saha mānayanti meghāgamaṃ priyasakhi śvasamānayanti // 20  //
bhāvānuraktalalanāsurataiḥ śapeyam ālabhya cāmbu tṛṣitaḥ karakośapeyam /
jīyeya yena kavinā yamakaiḥ pareṇa tasmai vaheyam udakaṃ ghaṭakarpareṇa // 21  //
