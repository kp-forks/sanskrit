pāpaṃ na kuryān manasā na vācā kāyena vā kiṃcana sarvaloke /
riktaḥ kāmaiḥ smṛtimān saṃprajānaṃ duḥkhaṃ na seveta anarthasaṃhitam // 1.1 //
ākhyeyasaṃjñinaḥ satvā ākhyeye 'smin pratiṣṭhitāḥ /
ākhyeyam aparijñāya yogam āyānti mṛtyunaḥ // 2.1 //
ākhyeyaṃ tu parijñāya ākhyātāraṃ na manyate /
tad vai na vidyate tasya vadeyur yena taṃ pare // 2.2 //
samo viśeṣa uta vāpi hīno yo manyate sa vivadeta tena /
vidhātraye 'smin na vikampate yaḥ samo viśiṣṭaś ca na tasya bhavati // 2.3 //
ācchidya tṛṣṇām iha nāmarūpe prahāya mānaṃ ca na saṃgam eti /
taṃ śāntadhūmam anighaṃ nirāśaṃ nādrākṣus te devamanuṣyaloke /
iha bāhirataś ca // 2.4 //
kāmarāgābhibhūtatvāc cittaṃ me paridahyate /
aṃga me gautama brūhi śāntiṃ tvam anukaṃpayā // 3.1 //
viparyāsena saṃjñānāṃ cittaṃ te paridahyate /
nimittaṃ varjyatāṃ tasmāc chubhaṃ rāgopasaṃhitaṃ // 3.2 //
aśubhāṃ bhāvaya sadā tvam ekāgraḥ susamāhitaḥ /
nirvāpayāśu rāgāgniṃ dahyase mā punaḥ punaḥ // 3.3 //
saṃskārāvaratāṃ paśya duḥkhato 'nātmatas tathā /
smṛtiṃ kāyagatāṃ kṛtvā nirvedabahulo bhava // 3.4 //
bhāvyatām animittaṃ ca mānānuśayanāśanaṃ /
tato mānābhisamayād duḥkhasyāntaṃ kariṣyasi // 3.5 //
||kumārikāpraśnagāthā:
kathaṃvihārabahulo bhikṣuḥ pañcaughatīrṇas taratīha ṣaṣṭhaṃ /
kathaṃdhyāyī vipulāṃ kāmatṛṣṇāṃ tīrṇo bhavaty apratilabdhayoktraḥ // 4.1 //
praśrabdhakāyaḥ suvimuktacitto hy asaṃskurvan smṛtimān akopyaḥ ājñāya dharmam avitarkadhyāyī kopaspṛhāstyānadoṣaiḥ viyuktaḥ // 4.2 //
evaṃvihārabahulo bhikṣuḥ pañcaughatīrṇas taratīha ṣaṣṭhaṃ /
evaṃdhyāyī vipulāṃ kāmatṛṣṇāṃ tīrṇo bhavaty apratilabdhayoktraḥ // 4.3 //
nityotrasto hy ayaṃ loko nityodvignā iyaṃ prajā /
anutpanneṣu duḥkheṣu samutpanneṣu vā punaḥ /
yadi kiṃcid anutrastaṃ pṛṣṭa ācakṣva tan mama // 5.1 //
nānyatra jñānatapaso nānyatrendriyanigrahāt /
nānyatra sarvasaṃtyāgān mokṣaṃ paśyāmi devate // 5.2 //
cirasya bata paśyāmi brāhmaṇaṃ parinirvṛtaṃ /
sarvavairabhayātītaṃ tīrṇaṃ loke viṣaktikām // 5.3 //
kenābhivarṇā janatā praṇītā mārgaś ca nairyāṇikataḥ prayuktaḥ /
kutra sthitaḥ kutra ca śikṣamāṇo nāyaṃ martyaḥ paralokād bibheti // 6.1 //
yaḥ śīlavāñ jñānavān bhāvitātmā samāhitaḥ smṛtimān ṛjugataś ca /
sarve 'sya śokajvarathāḥ prahīṇāḥ samyaksmṛto yasya cittaṃ vimuktaṃ // 6.2 //
tenābhivarṇā janatā praṇītā mārgaś ca nairyāṇikataḥ prayuktaḥ /
atra sthitaḥ atra ca śikṣamāṇo nāyaṃ martyaḥ paralokād bibheti // 6.3 //
kathaṃ yaśasvī bhavati kathaṃ bhavati bhogavān /
kathaṃ kīrtim avāpnoti kathaṃ mitrāṇi vindati // 7.1 //
śīlād yaśasvī bhavati dānād bhavati bhogavān /
satyena kīrtim āpnoti dadan mitrāṇi vindati // 7.2 //
kutaḥ sarā nivartante kutra vartma na vartate /
kutra duḥkhasukhaṃ loke niḥśeṣam uparudhyate // 8.1 //
cakṣuḥ śrotraṃ tathā ghrāṇaṃ jihvā kāyo manas tathā /
yatra nāma ca rūpaṃ ca niḥśeṣam uparuhyate // 8.2 //
tataḥ sarā nivartante tatra vartma na vartate /
tatra duḥkhasukhaṃ loke niḥśeṣam uparudhyate // 8.3 //
kena svid oghaṃ tarati kenottarati cārṇavaṃ /
duḥkhaṃ tyajati kena svit kena svit pariśudhyati // 9.1 //
śraddhayā tarati hy ogham apramādena cārṇavaṃ /
vīryeṇa duḥkhaṃ tyajati prajñayā pariśudhyati // 9.2 //
ka etam oghaṃ tarati rātriṃdivam atandritaḥ /
anālambe 'pratiṣṭhe ca ko gaṃbhīre na sīdati // 10.1 //
sarvataḥ śīlasaṃpannaḥ prajñāvān susamāhitaḥ /
adhyātmacintī smṛtimāṃs taratīmaṃ sudustaraṃ // 10.2 //
viraktaḥ kāmasaṃjñābhyo rūpasaṃyojanātigaḥ /
anālambe 'pratiṣṭhe ca sa gaṃbhīre na sīdati // 10.3 //
rāgadveṣau bhagavan kinnidānāv aratiratī romaharṣaḥ kuto 'yam /
kutaḥsamutthāś ca mano vitarkāḥ kumārakā dhātrīm ivāśrayante // 11.1 //
snehajā ātmasaṃbhūtā nyagrodhaskandhakā yathā /
pṛthagviṣaktāḥ kāmeṣu mālutā vā latā vane // 11.2 //
rāgaś ca dveṣaś ca itonidānāv aratiratī romaharṣaḥ ito 'yaṃ /
itaḥsamutthāś ca mano vitarkāḥ kumārakā dhātrīm ivāśrayante // 11.3 //
ye tān prajānanti yatonidānāṃs te tāṃ janā yakṣa vinodayanti /
ta arṇavaṃ saṃpratarantīhaugham atīrṇapūrvam apunarbhavāya // 11.4 //
kāryam etad brāhmaṇena prahāṇam akilāsinā /
kāmānāṃ viprahāṇārthaṃ na hi kāṃkṣaty asau bhavaṃ // 12.1 //
sa kāryaṃ brāhmaṇasyāsti kṛtārtho brāhmaṇaḥ smṛtaḥ // 12.2 //
yāvan na gādhaṃ labhate na tīram āyūhate sarvagātraiḥ sa tāvat /
tīraṃ tu labdhveha saṃtiṣṭhati sthale nāyūhate pāragato nirucyate // 12.3 //
eṣopamā dāmale brāhmaṇasya kṣīṇāsravo yo nipako dhyānalābhī /
sarve 'sya śokajvarathāḥ prahīṇāḥ samyaksmṛto yasya cittaṃ vimuktam // 12.4 //
||geya:
bhikṣo bhikṣo ogham atārṣīḥ /
āma devate /
anālambe 'pratiṣṭhe ogham atārṣīḥ /
āma devate /
yathā kathaṃ tvaṃ bhikṣo anālambe apratiṣṭhe ogham atārṣīḥ /
yathā yathāhaṃ devate āyūhāmi tathā tathā saṃsīdāmi /
yathā yathā saṃsīdāmi tathā tathā saṃtiṣṭhe /
yathā yathā saṃtiṣṭhe tathā tathā uhye /
yathā yathāhaṃ devate nāyūhāmi tathā tathā na saṃsīdāmīti vistareṇa śuklapakṣo veditavyaḥ // 13 //
||kumārikāpraśnagāthā:
eko 'raṇye prasṛto dhyāyase tvaṃ vittād vihīna uta vā prārthayānaḥ /
grāmasya vā kiṃcanāgo nv akārṣīḥ kasmāj janena na karoṣi sakhyaṃ sakhyaṃ na saṃvidyate kena cit tava // 14.1 //
arthāprāptyā hṛdayasyeha śāntir jitveha senāṃ priyaśātarūpaṃ /
eko dhyāyī sukham asmy anvabhotsaṃ tasmād janena na karomi sakhyaṃ sakhyaṃ na saṃvidyate kena cid mama // 14.2 //
anityā bata saṃskārā utpādavyayadharmiṇaḥ /
utpadya hi nirudhyante teṣāṃ vyupaśamaḥ sukham // 15 //
apramādo 'mṛtapadaṃ pramādo mṛtyunaḥ padaṃ /
apramattā na mriyante pramattās tu sadā mṛtāḥ // 16 //
vitarkapramathitasya dehinas tīvrarāgasya śubhānudarśinaḥ /
bhūyas tṛṣṇā vivardhate sā gāḍhīkurute 'sya bandhanaṃ // 17 //
dharmasthaṃ śīlasaṃpannaṃ hrīmantaṃ satyavādinaṃ /
ātmanaḥ priyakartāraṃ taṃ janaḥ kurute priyaṃ // 18 //
yat pareṣūpanidhyāyet karma dṛṣṭveha pāpakam /
ātmanā tan na kurvīta karmabaddho hi pāpakaḥ // 19 //
subhāṣitaṃ hy uttamam āhur āryāḥ priyaṃ vaden nāpriyaṃ tad dvitīyaṃ /
satyaṃ vaden nānṛtaṃ tat tṛtīyaṃ dharmaṃ vaden nādharmaṃ tat caturtham // 20 //
śraddhātha hrīḥ śīlam athāpi dānaṃ dharmā ime satpuruṣapraśastāḥ /
etaṃ hi mārgaṃ divigaṃ vadanti etena vai gacchati devalokam // 21 //
śrutvā dharmaṃ vijānāti śrutvā pāpān nivartate /
śrutvā anarthaṃ tyajati śrutvā prāpnoti nirvṛtim // 22 //
ākāśasamo na lipyate indrakīlapratimo na kaṃpate /
hrada iva samupetakardame saṃsāre ramate na paṇḍitaḥ // 23 //
ye rūpeṇa pramiṇvanti māṃ ghoṣeṇānuyānti ca /
cchandarāgavaśopetā na mā jānanti te janāḥ // 24.1 //
adhyātmaṃ ca vijānāti bahirdhā ca na paśyati /
adhyātmaphaladarśī yaḥ sa vai ghoṣeṇa nīyate // 24.2 //
adhyātmaṃ ca na jānāti bahirdhā ca vipaśyati /
bahirdhāphaladarśī yaḥ so 'pi ghoṣeṇa nīyate // 24.3 //
adhyātmaṃ ca na jānāti bahirdhā ca na paśyati /
samantāvaraṇo bālaḥ so 'pi ghoṣeṇa nīyate // 24.4 //
adhyātmaṃ ca vijānāti bahirdhā ca vipaśyati /
dhīro niḥsaraṇaprajño na sa ghoṣeṇa nīyate // 24.5 //
ṣaṣṭhe adhipatau rājñi rajyamāne rajasvalaḥ /
arajasy arajā bhavati rakto bālo nirucyate // 25 //
nagaram asthiprākāraṃ snāyumāṃsānulepanaṃ /
yatra rāgaś ca dveṣaś ca māno mrakṣaś ca gāhate // 26 //
kūrmaḥ svake 'ṅgāni yathā kapāle bhikṣur nidadhyān manaso vitarkāni /
aniśrito 'nyān aviheṭhamānaḥ parinirvṛto nāpavadeta kaṃ cit // 27 //
tulyam atulyaṃ ca saṃbhavaṃ bhavasaṃskāram avāsṛjan muniḥ /
adhyātmarataḥ samāhita abhinat kośam ivāṇḍasaṃbhavaḥ // 28 //
nāsti kāmasamaḥ paṃko nāsti dveṣasamo grahaḥ /
nāsti mohasamaṃ jālaṃ nāsti tṛṣṇāsamā nadī // 29 //
ākāśe vai padaṃ nāsti śramaṇo nāsti bāhyakaḥ /
prapañcābhiratā bālā niṣprapañcās tathāgatāḥ // 30 //
sthitiḥ prapaṃcāś ca na santi yasya yaḥ saṃdānaṃ parighaṃ cātivṛttaḥ /
taṃ nistṛṣṇaṃ muniṃ carantaṃ na vijānāti sadevako 'pi lokaḥ // 31 //
yasya vitarkā vidhūpitā adhyātmam avikalpitā aśeṣaṃ /
saṃgaṃ so 'tītya rūpasaṃjñāṃ caturyogāpagato na jātim eti // 32 //
dadataḥ puṇyaṃ pravardhate vairaṃ saṃyamato na cīyate /
kuśalī prajahāti pāpakaṃ kleśānāṃ kṣayatas tu nirvṛtaḥ // 33 //
sarvapāpasyākaraṇaṃ kuśalasyopasaṃpadā /
svacittaparyavadamanam etaṃ buddhānuśāsanaṃ // 34 //
durnigrahasya laghuno yatrakāmanipātinaḥ /
cittasya dāmanaṃ sādhu cittaṃ dāntaṃ sukhāvaham // 35 //
cittanimittasya kovidaḥ pravivekasya ca vindate rasaṃ /
dhyāyī nipakaḥ pratismṛto bhuṃkte prītisukhaṃ nirāmiṣaṃ // 36 //
aśilpajīvī laghur ātmakāmo jitendriyaḥ sarvato vipramuktaḥ /
anokasārī hy amamo nirāśaḥ kāmān prahāyaikacaro yas sa bhikṣuḥ // 37 //
dūraṃgamam ekacaram aśarīraṃ guhāśayaṃ /
damayati durdamaṃ cittaṃ brāhmaṇaṃ taṃ bravīmy ahaṃ // 38 //
pārāyaṇeṣv ajitapraśnaḥ:
kenāyaṃ nivṛto lokaḥ kenāyaṃ na prakāśate /
kiṃ cābhilepanaṃ brūṣe kiṃ ca tasya mahad bhayam // 39.1 //
avidyānivṛto lokaḥ pramādān na prakāśate /
jalpābhilepanaṃ brūmi duḥkhaṃ tasya mahad bhayaṃ // 39.2 //
sravanti sarvataḥ srotāḥ srotasāṃ kiṃ nivāraṇaṃ /
srotasāṃ saṃvaraṃ brūhi kena srotaḥ pidhīyate // 39.3 //
yāni srotāṃsi lokasya smṛtiḥ teṣāṃ nivāraṇaṃ /
srotasāṃ saṃvaraṃ brūmi prajñayā hi pidhīyate // 39.4 //
prajñāyāś ca smṛteś caiva nāmarūpasya sarvaśaḥ /
ācakṣva pṛṣṭa etan me kutraitad uparudhyate // 39.5 //
prajñā caiva smṛtiś caiva nāmarūpaṃ ca sarvaśaḥ /
vijñānasya nirodhād dhi atraitad uparudhyate // 39.6 //
kathaṃ smṛtasya carato vijñānam uparudhyate /
ācakṣva pṛṣṭa etan me yathātatham asaṃśayaḥ // 39.7 //
adhyātaṃ ca bahirdhā ca vedanāṃ nābhinandataḥ /
evaṃ smṛtasya carato vijñānam uparudhyate // 39.8 //
ye ca saṃkhyātadharmāṇo ye ca śaikṣāḥ pṛthagvidhāḥ /
teṣāṃ me nipakasyeryāṃ pṛṣṭaḥ prabrūhi mārṣa // 39.9 //
kāmeṣu nābhigṛdhyeta manasānāvilo bhavet /
kuśalaḥ sarvadharmeṣu smṛto bhikṣuḥ parivrajet // 39.10 //
||arthavargīyeṣu kāmān ārabhya gāthā:

tasya cet kāmāyamānasya chandajātasya jaṃtunaḥ /
te kāmāḥ parihīyaṃte śalyaviddha iva rūpyate // 40.2 //
yaḥ kāmāṃ parivarjayati sarpasyeva śirāt padaṃ /
sa imāṃ viṣaktikāṃ loke smṛtaḥ samativartate // 40.3 //
kṣetravastuhiraṇ@yaṃ ca gavāśvamaṇikuṇḍalaṃ /
