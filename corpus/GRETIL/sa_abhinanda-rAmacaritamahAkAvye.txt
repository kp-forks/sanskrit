#Text:Rāmacaritamahākāvya
#Data entry:Harunaga Isaacson
#Edition:K.S. Ramaswami Sastri Siromani: Ramacarita of Abhinanda. Baroda: Oriental Institute, 1930. Gaekwad's Oriental Series No. XLVI.

||prathamaḥ sargaḥ
atha mālyavataḥ prasthe kākutsthasya viyoginaḥ /
durnivārāśrusaṃvego jagāma jaladāgamaḥ // 1.1 //
śaśāma vṛṣṭir meghānām utsaṅge tasya bhūbhṛtaḥ /
virarāma na rāmasya dhārāsantatir aśruṇaḥ // 1.2 //
itas tataḥ pariṇatiṃ bheje barhiṇakūjitam /
hā priye rājaputrīti na rāmaparidevitam // 1.3 //
kāpy abhikhyā virajasoḥ sūryācandramasor abhūt /
dāśarathyos tathaivāsīd ayogopahatā ruciḥ // 1.4 //
nirmalendu nabho reje vikacābjaṃ babhau saraḥ /
paraṃ paryaśruvadanau mamlaturbhrātarāvubhau // 1.5 //
smartavyakalikājālaṃ jajñe kuṭajakānanam /
āsīt tathaiva tu mano rāmasyotkalikākulam // 1.6 //
prapede punar udbhedaḥ śucinā kacchaketakaiḥ /
upakriyāyāḥ sadṛśaṃ nārebhe ravisūnunā // 1.7 //
ākṛṣṭamālatīgandhaḥ siṣeve rāghavaṃ marut /
ānītamaithilīvārtas tam ārtaṃ na tu mārutiḥ // 1.8 //
rāmāya pūrāpagamavyaktoddeśāś cakāśire /
āste 'smāsu na sīteti śaṃsantya iva nimnagāḥ // 1.9 //
yadārdraviraho rāmaḥ prāpitaḥ prāṇasaṃśayam /
nūnaṃ tenānutāpena jagmur jaladarājayaḥ // 1.10 //
dyaur muktameghāvaraṇā savitarkonmukhaṃ muhuḥ /
yayāce sārasarutair vicāram iva rāghavam // 1.11 //
diśo dadṛśire tena stenaṃ taṃ parimārgatā /
mārgadānāparādhinyo gatā dūraṃ bhayād iva // 1.12 //
tasyākṛtiviśeṣasya viśeṣāt kaṣṭam īdṛśam /
kacchāḥ kāśacchalād ūhur itīva palitāgamam // 1.13 //
paśyantyā iva kaṣṭāṃ tām avasthāṃ maithilīpateḥ /
śuṣkapaṅkacayavyājād vidadre hṛdayaṃ bhuvaḥ // 1.14 //
vṛṣṭivicchedajātena tānavenāsamāptinā /
prapedire jaladharās tasya sabrahmacāritām // 1.15 //
yayau virahasaṃvartaḥ padminīrājahaṃsayoḥ /
apāraduḥkhārṇavayor na sītārāmabhadrayoḥ // 1.16 //
śāntanirjharajhātkāradhārāḥ śikhariṇo babhuḥ /
rāmasya viśadālāpaśuśrūṣānibhṛtā iva // 1.17 //
tasya cakruś camatkāraṃ vyatītasamayā api /
smitābhamukulodbhedāḥ kadambavanarājayaḥ // 1.18 //
adṛśyanta puras tena khelāḥ khañjanapaṅktayaḥ /
asmaryanta ca niśvasya priyānayanavibhramāḥ // 1.19 //
abravīd atha kālajñas tatkālapratipattaye /
praṇamya lakṣmaṇo rāmaṃ rāmāvirahaniḥsaham // 1.20 //
jātaṃ vibhātam amalaṃ paśya prāvṛṇṇiśā gatā /
utsāhakamalasyāyaṃ vikāsāvasaras tava // 1.21 //
ucchrayaṃ tīrataravo raver ūrjam abhīṣavaḥ /
madaṃ gopatayo gṛhṇanty abhiyogaṃ jigīṣavaḥ // 1.22 //
prasīda dīyatāṃ karṇaḥ kūjadbhiḥ kurarair amī /
prārthayantīva saṃcāraṃ taṭāḥ kardamadurgatāḥ // 1.23 //
bhuvi bhāsvadabhīṣūṇāṃ tvadiṣūṇāṃ ca śatruṣu /
ayaṃ sampatituṃ kālaḥ sa duṣkālo 'tivāhitaḥ // 1.24 //
sudūronnataparyantās tīryante saṃkramair iva /
analpagādhāḥ saritas tarubhiḥ pūrapātitaiḥ // 1.25 //
āviṣkṛtaviśuddhibhyāṃ nadyaḥ sumahilā iva /
kulābhyām iva kūlābhyām āpatsv api cakāsati // 1.26 //
jalajānāṃ sumanasāṃ vayasāṃ jalasevinām /
asya pratijalādhāraṃ sāmagryam avalokyate // 1.27 //
yuyutsārabhasotkhātasaritkhātakarodhasaḥ /
kukudmanto madakalāḥ kalayanti jaganty adhaḥ // 1.28 //
viśveśa kuru viśvāsam āvartante na te ghanāḥ /
amī śyāmalayanty āśāḥ śukāḥ śālivanonmukhāḥ // 1.29 //
mṛgāḥ samprati śāleyaṃ sampannaṃ sampatanty amī /
sāketam adhitiṣṭhantam arthinas tvām ivonmukhāḥ // 1.30 //
kāryadig gṛhyatāṃ kāpi kim ārya sthīyate vṛthā /
evam evāyam asmākam iṣṭo 'pi kim upaiṣyati // 1.31 //
vijayāya bhavāsīno mayy arpitavapurbharaḥ /
mūrcchānuvalanavyastaḥ prastaras tyajyatām ayam // 1.32 //
pratīṣṭopatyakāsāladalāgrajalabindavaḥ /
sāryantāṃ kṣaṇam uddhūya sthagitāṃsataṭā jaṭāḥ // 1.33 //
muhūrtaṃ kriyatām ārya kapolavirahī karaḥ /
vyuṣito valkalagranthir aṃsād unmocyatām ayam // 1.34 //
ito vitatya dīyantām iṣavaḥ klinnayantraṇāḥ /
nirmuktārdrajaraccailam ādattām āyataṃ dhanuḥ // 1.35 //
ito 'vatīryatāṃ prāsthād asvāsthyam idam ujjhyatām /
jahīṣubhir daśagrīvaṃ daśadigvedhaśodhitaiḥ // 1.36 //
ṛjur ādīyatāṃ panthāḥ kāryakanthā garīyasī /
utsāryatām idaṃ dūraṃ tamaḥ kiṃkāryatāmayam // 1.37 //
āryāyāḥ śivam evāsti śatrur nihata eva te /
alaṃ dattvāvasādāya trailokyaśaraṇaṃ vapuḥ // 1.38 //
rākṣasāpaśadaḥ ko 'sau tvayi sajyaśarāsane /
anena te 'vasādena dūram āropitaḥ punaḥ // 1.39 //
aśruvegaiḥ pramīlābhir vilāpair akhilaḥ sa te /
āmṛṣṭadehayātrasya māsaḥ prauṣṭhapado gataḥ // 1.40 //
kriyatām adhunotsāhaś cintyatāṃ nidhanaṃ dviṣaḥ /
kaḥ svabāhusahāyānāṃ mahatām ātmanigrahaḥ // 1.41 //
bhadraṃ bhādrapadacchedacchinnābhrājinarajjave /
tubhyaṃ puruṣanāgāya nirvighnatarase 'dhunā // 1.42 //
prasīda kuru bhūyo 'pi dehayātrāṃ yathā tathā /
anullaṅghitavijñaptir lakṣmaṇo 'varajeṣu te // 1.43 //
agādhaṃ dhairyam āryasya triṣu lokeṣu gīyate /
janamātrocitaḥ ko 'yam ākalpakaparigrahaḥ // 1.44 //
aropaśamitodagrajāmadagnyāgnitejasaḥ /
pratīcchati śarāsāraṃ tavārya ruṣitasya kaḥ // 1.45 //
svakṛtyabhārasannyāsī yāvad itthaṃ viṣīdasi /
tāvad antakadaṃṣṭrāṇām arātir atithir na te // 1.46 //
ātatāyī sa vijñāto jīvaty āryeti ca śrutam /
prāpto yātrāsahaḥ kālaḥ kālakṣepasya ko guṇaḥ // 1.47 //
na jñāyate paraṃ vyaktam arer māyāvinaḥ padam /
tanmārgaṇaṃ ca suhṛdā sugrīveṇa pratiśrutam // 1.48 //
diṅmukhāhūtavikhyātakapiyūthaḥ kapīśvaraḥ /
kṣaṇān nūnam asav āryapādamūle patiṣyati // 1.49 //
śālipiṅgamahīpṛṣṭhaspardhayeva bhaviṣyati /
raṃhasviharisaṃghātasampātakapilaṃ nabhaḥ // 1.50 //
teṣām āroṣarucirair acirān mukhamaṇḍalaiḥ /
drakṣyate taruṇādityaśatacchannam ivāmbaram // 1.51 //
aho dīptir aho kāntir aho śīlam aho balam /
aho śaktir aho bhaktir aho prajñā hanūmataḥ // 1.52 //
pragalbhenāpramattena tenābhyantaramantriṇā /
nūnam asmatkṛte svaptuṃ sugrīvāya na dīyate // 1.53 //
sa ca pratyupakārāya tvarayaty enam ādṛtaḥ /
dhruvaṃ buddhivayovṛddhaḥ sacivo jāmbavān api // 1.54 //
tad evaṃ hetunā kena vānarendro vilambate /
nāsau kṛtaghnasya bhaved guṇavadbhṛtyasaṃgrahaḥ // 1.55 //
āgantavyaṃ dhruvaṃ tena mārgitāryā pravṛttinā /
prapannaparakṛtyānām atyuṣṇo hi samudyamaḥ // 1.56 //
tvatprasādāptasambhogasukhasaṃmīlitena vā /
na tenāmūr dadṛśire diśaḥ proṣitakālikāḥ // 1.57 //
kiyacciraṃ punar asau tasya na śrutim eṣyati /
krīḍātaṭākahaṃsānāṃ ninādo madamūrchitaḥ // 1.58 //
uktaiḥ kim atha bhūyobhir adya śvo vāgamiṣyati /
paryutsukitakiṣkindhaḥ satyasandho harīśvaraḥ // 1.59 //
vyatireke 'pi yat kāryaṃ tadapyārya pradhāryatām /
gṛhyatāṃ kāryasiddhir nas tadvilambatiraskṛtā // 1.60 //
yas tathopakṛtaḥ pūrvam āryeṇāryākṛte 'dhunā /
sa śītalaḥ syād athavā paracittāni vetti kaḥ // 1.61 //
sa pratiśrutam arthaṃ naḥ sphīto vismṛtavān iva /
gatvāśu smāryatām ārya kāryataptasya kā kṣamā // 1.62 //
tam ahaṃ bodhayiṣyāmi daṇḍagarbhābhir uktibhiḥ /
sāma saṃmīlayaty eva sukhasaktān pramādinaḥ // 1.63 //
prasīda jīyatāṃ mohaḥ ko 'ham ity avadhīyatām /
drutam uttīrṇa evāste vyasanābdhiḥ kiyān ayam // 1.64 //
sūnṛtair iti saumitreḥ prītaḥ pīyūṣavarṣibhiḥ /
tam antarvedanāvegaṃ jigāyeva raghūdvahaḥ // 1.65 //
unmīlayāmāsa dṛśau diśaḥ kiṃcid alokata /
unnanāma ca talpāntān mocayan pakṣmalā jaṭāḥ // 1.66 //
satvaraprahvasaumitricīrotsāritareṇuni /
śucini kvacid āsanne niṣasāda śilātale // 1.67 //
upaviṣṭas tadā rāmo na reje sītayā vinā /
dhvajo nirvaijayantīkaḥ paurandara ivocchritaḥ // 1.68 //
anūpaviṣṭam anujaṃ danujārim ivādrijit /
pratyabhāṣata vātsalyagadgadāni padāni saḥ // 1.69 //
ehy ehi priyasaṃlāpa hlādayāliṅgitena mām /
ruddhaṃ ruddham upaity aśru tvām apīkṣe na lakṣmaṇa // 1.70 //
hā vatsa bhavato 'pīyam avasthāṅgeṣu vartate /
dhriyatāṃ rāmahatakaḥ kleśayann ekabāndhavam // 1.71 //
nāsti vatsa cikitsāsya vyādher vidhurajanmanaḥ /
yasya yāpanayā kṣīṇam idaṃ te sāntvanāmṛtam // 1.72 //
naikadurvāraghorādhighaṭṭanācalitā api /
tavāśvāsanayā vatsa na gacchanti mamāsavaḥ // 1.73 //
pitur naḥ priyamitrasya lūnapakṣasya maddviṣā /
saṃsmaran notsahe vatsa marmacchedaṃ jaṭāyuṣaḥ // 1.74 //
tātasya tātamitrasya vatsavatsalayos tayoḥ /
aham eva gataḥ pāpo nidhanāya nimittatām // 1.75 //
priyāpahārajaḥ śāmyet samyag unmūlite ripau /
manyur dehāvadhir ayaṃ tātamitravipattibhūḥ // 1.76 //
nabhastalāt khagapatau patati cchinnapakṣatau /
ko 'paras tāṃ mama suhṛn mā bhaiṣīr ity abhāṣata // 1.77 //
syād vatsa rāmahatakāt klībaḥ kaḥ puruṣo 'paraḥ /
ekapatnīm anugatām arāter yo na rakṣati // 1.78 //
tayā teṣu tapasvinyā sīdator āvayor api /
daṇḍakāraṇyakaṣṭeṣu mukhaṃ na vidhurīkṛtam // 1.79 //
videharājasya sutā snuṣā daśarathasya sā /
hatarāmasya jāyeti bhramati sma vane vane // 1.80 //
svam apy apuṣṇatā kāyam api jāyām arakṣatā /
mayā samaḥ śrutaḥ ko 'pi jātaḥ klībo raghoḥ kule // 1.81 //
tvaṃ kevalaṃ na subhrātā tathā vilapatā muhuḥ /
tiryañco 'pi mayāmuṣmin vane lakṣmaṇa roditāḥ // 1.82 //
dikṣu sannaddhameghāsu mama ghorādhisākṣiṇām /
avahan nirjharanibhān nagānām aśrunimnagāḥ // 1.83 //
ārād ākarṇitasnigdhanavābdaninadair api /
matkāruṇyād araṇye 'pi nānṛtyata kalāpibhiḥ // 1.84 //
satyam eṣyati sugrīvaḥ satyaṃ jīvati jānakī /
tvadvaco na viparyeti bhrātar brūhi punaḥ punaḥ // 1.85 //
tvadvākyaśīkarair ebhir niruddhabahirudgamaḥ /
mamāntar leḍhi marmāṇi sītāvirahamurmuraḥ // 1.86 //
tasmād api dahaty uccair ayaṃ mām aparaḥ śikhī /
yad vanaukasi nirvaire mukto vālini mārgaṇaḥ // 1.87 //
gurvī punaś ca lajjeyam uttamarṇair ivādhunā /
mārgitavyo yad asmābhiḥ plavagāḥ pratyupakriyām // 1.88 //
nāsti pratyupakārāśā tatra naś capale kapau /
yenātilaghunottuṅgo laṅghitaḥ satyapādapaḥ // 1.89 //
uttiṣṭha vatsa gacchāvaḥ sādhayāmo 'nyato 'dhunā /
navaiśvaryasukhavyagraḥ sugrīvo nāgam iṣyati // 1.90 //
tapasvī ramatām evaṃ cirād dāraiḥ samāgataḥ /
sa pīḍyamānaḥ praṇayād virasaḥ kiṃ kariṣyati // 1.91 //
kiṃ ca vatsa daśāsāmyāj jātāḥ smaḥ suhṛdaḥ purā /
tasyādhunā tu vistīrṇavibhavāndhasya ke vayam // 1.92 //
gṛhītasādhuvartmāpi sadyaḥ samprāpya sampadam /
patati skhalito 'tīva pramattaḥ prahate pathi // 1.93 //
sa vatsa viralo jantur vipatkālapratiśrutam /
sadyaḥ sampanmado yasya na vilumpati cetasi // 1.94 //
astu sādhur asādhur vā na vicāryaḥ sa me 'dhunā /
vatsa notsahate cetas tatra dogdhum upakriyām // 1.95 //
asāv anāgataḥ śreyān āgacchan vatsa vāryatām /
praveśayanti suhṛdaṃ na dhīrāḥ svārthasaṃkaṭe // 1.96 //
kartavyopakṛtiḥ sadbhir yatra tatra vipadgate /
avimṛśya kṛto 'smābhiḥ sa punaḥ kṛpaṇaḥ paṇaḥ // 1.97 //
kiṃ kṣiptam acalaprāntād antaḥsuptaśivāśatam /
nirdeśapuruṣeṇeva kaṅkālaṃ dundubher mayā // 1.98 //
kasyārthasya kṛte viddhāḥ sapta sālās tapasvinaḥ /
sa mā pratyetu vaidheyaḥ kiṃ na sādhitam anyataḥ // 1.99 //
akṣātraṃ kṣatriyayuvā ko nu rāma iva kṣipet /
viśikhaṃ vālini vyagre sugrīvasya paśoḥ kṛte // 1.100 //
na parityajyate mārgas tāvan nirvācyakardamaḥ /
yāvat pihitakaṣṭāya kāryāyātmā na dīyate // 1.101 //
iṣvaikayā mayā viddho bāliśena valīmukhaḥ /
anuddhārāḥ śarās tena ropitā mayi vāṅmayāḥ // 1.102 //
anavāptavipatpāraḥ smariṣyāmi kiyanti vā /
vatsa kālocitaṃ śādhi na rāmaḥ kiṃcid īkṣate // 1.103 //
avalambasva māṃ bhrātaḥ sītāsmaraṇanissaham /
dūraṃ punaḥ prabhavatā mano mohena lupyate // 1.104 //
sanāthobhayapārśvasya tvayā vatsa tayā ca me /
vaneṣu ca sataḥ kaṣṭam ajaniṣṭa na kiṃcana // 1.105 //
hā bhīru hā priyatame hā madvirahakātare /
kvāsīty ardhoktinissaṃjño jagāma sa mahītalam // 1.106 //
mātaḥ kaikeyi tuṣṭāsi bhuṅkṣva rājyam akaṇṭakam /
vadann ity apatat tasya lakṣmaṇaś caraṇābjayoḥ // 1.107 //
tatas tau sāśrunayanau mṛgair vidhuravīkṣitau /
nipetatur diva iva bhraṣṭau vidhuravī kṣitau // 1.108 //
iti nipatitayos tayor dvayor daśarathanandanayor mahītale /
daśabhir api digaṅganāmukhair adhṛtivaśād iva tatyaje ruciḥ // 1.109 //
ete nikāmarasikasya jayanti pādāḥ śrīhāravarṣayuvarājamahītalendoḥ /
yair dvādaśārkakiraṇotkaradurnivāraḥ sṛṣṭo 'bhinandakumudasya mahāvikāsaḥ // 1.110 //
||ity abhinandakṛtau rāmacarite mahākāvye prathamaḥ sargaḥ samāptaḥ |
||dvitīyaḥ sargaḥ
tām anīkṣitacarīm adhīratām ātmano 'tha viniyamya lakṣmaṇaḥ /
sāntvanārtham iti vācam abravīt tīvramanyurayamūḍham agrajam // 2.1 //
kāryam ārya pariśiṣṭam ucyatāṃ mucyatām idam ayogajaṃ tamaḥ /
rājatūdyamadivākarodayasmeram āśu mukhapaṅkajaṃ tava // 2.2 //
āsituṃ samaya eṣa nāvayor moham ittham avalambituṃ na ca /
utsahasva nanu hṛṣyatāṃ cirādiṣv adhīnam adhunābhikāṅkṣitam // 2.3 //
kevalaṃ phalabhujo na tasya nas taskarasya nilaye 'sti niścayaḥ /
abhyupaitu ravinandanaḥ kṣaṇād andhakāram idam apy apaiṣyati // 2.4 //
aurasān api sutān arīn iva ghnanti kāraṇavaśān mahībhujaḥ /
nigrahād anujavidviṣaḥ kapeḥ kiṃ pṛthagjana ivānutapyase // 2.5 //
krīḍatāṃ mṛgavaneṣu dhanvināṃ jīvakoṭikadane 'pi kā kṣatiḥ /
khidyase kim aniśaṃ prayojanād ekavānaravadhaḥ kutaḥ kṛtaḥ // 2.6 //
tasya ca pravayaso jaṭāyuṣaḥ svargiṇaḥ kim adhunānuśocyate /
yena jarjarakalevaravyayāt krītam indukiraṇojjvalaṃ yaśaḥ // 2.7 //
etad ekam ativāhyatām ahaḥ śvo na sa plavagarāṭpratīkṣyate /
tat prasīda parito vicīyatātmanaiva sanagārṇavāvaniḥ // 2.8 //
tat kabandhaśabarīhanūmatām āvayor ajani saṅgataṃ yathā /
iṣṭaśaṃsi vanaśailapānthayoḥ kiṃ tathānyad api nāpatet punaḥ // 2.9 //
yatra yatra tava siddhir agrataḥ kaḥ kariṣyati na te samīhitam /
ujjvale kṛtavidāṃ na kevalaṃ tena vartmani durātmanā sthitam // 2.10 //
sambhavaty abhimaraḥ pure 'thavā tasya tīvram ayaśaḥ kim ucyate /
duḥkham utkhanati kaṇṭakāvalīr aṅgasandhiṣu navaḥ kileśvaraḥ // 2.11 //
samprati prathamam ārya gṛhyatāṃ tatpravṛttim upalabdhum udyamaḥ /
marṣaṇīyaparilambadūṣaṇaḥ so 'dhunāpy akapaṭo ghaṭate cet // 2.12 //
muñca moham avatīryatām itaḥ sevyatām iyam anāvilā nadī /
eṣa gacchati javāj japāruṇo vāruṇīm aruṇasārathir diśam // 2.13 //
unmimīla raghunandanaś cirāt tan niśamya vacanaṃ camatkṛtaḥ /
apy ayogavidhurā na śerate saṃmukhīnavidhayas tathāvidhāḥ // 2.14 //
muktamoham avalokya lakṣmaṇas taṃ mṛgāṅkam iva meghanirgatam /
ādade 'tha śaśikāntabhūdharasyandabinduśiśirāṃ sarasvatīm // 2.15 //
eṣa paśya patito nabhastalād astaśailagahaneṣu naśyati /
sandhyayā caramadikpragalbhayā bhagnasāndrakarapañjaraḥ khagaḥ // 2.16 //
śakyam arcitum anuccapāṇibhiḥ pāśapāṇinagarīnivāsibhiḥ /
sānurodha iva dūrabandhuro bandhujīvakusumāruṇo raviḥ // 2.17 //

asmāsu nāvedayati sma garvāt kurvīta śeṣārivadhaṃ bhujābhyām /
vidhāya saurājyam asaūpeyād ity arthitā paśyati na pramāṇam // 3.16 //
tad eṣa me nirmalasargasargaḥ ṣaḍvargaruddho ripuvat sa heyaḥ /
nāsau niṣiddhas tava yānapakṣo mayā bata prāg asamāhitena // 3.17 //
asaṃśayaṃ sa pratihārapālīsasambhramāveditasannikarṣam /
tvāṃ praty asūyāṃ hṛdayena dhāsyaty upasthitāntaḥpuraviprayogaḥ // 3.18 //
tadvaktram ārkes taruṇārkatāmraṃ tvatprītivākyair akṛtapramodam /
bhaviṣyati pratyupakārabhāranirvāhacintāgamasāndhakāram // 3.19 //
smariṣyati tvāṃ na sa bodhyamāno 'py ālokayiṣyaty athavā na pṛṣṭaḥ /
pratyeṣi kāntājanamadhyavartī sa cintayāsmadgatayādhunāste // 3.20 //
āvāṃ yadaivādritaṭe niviṣṭau sa kālikāṃ vīkṣya purīṃ praviṣṭaḥ /
āmṛṣṭam āsīc capalena tena sauhārdam asmadviṣayaṃ tadaiva // 3.21 //
ahaḥsv amīṣu kṣaṇa eva nāsīd āsīn na saṃdeśaharo 'nurūpaḥ /
ityādi sa vyaktam alīkam eva tvatprītyupālambhapadeṣu vaktā // 3.22 //
kṛtāvahelaḥ praṇayaprasannaṃ niruttaraḥ satvarakāryavādam /
prekṣiṣyate roṣalavānuraktaṃ valīmukhaḥ śuṣkamukho mukhaṃ te // 3.23 //
tasyāgamād yaḥ sahasā bhaviṣyaty arthasya tasyābhavanaṃ mamāstu /
bādhiṣyate vālivadhātiriktas tadveśmayātrānuśayaḥ sadā nau // 3.24 //
sa prāktano nūnam anarthabhājā nītas talaṃ tena hitaiṣivargaḥ /
utthāpitair dūram asāv idānīṃ divāniśaṃ dīvyati durvidagdhaiḥ // 3.25 //
akṣā mṛgākṣyo mṛgayā madhūni prekṣāś ca tasyādya haranti kālam /
na prāpnuvanti kṣaṇam āptavācaḥ sadādṛtāś cāraṇacāṭuvādāḥ // 3.26 //
vadanti bhītāḥ sacivā na kiṃcit kiṃ cintayāsmadgatayā viṭānām /
svayaṃ na sampraty anusaṃdadhāti pūrvāparaṃ kāmaparaḥ sa rājā // 3.27 //
na sūrayo na sthavirā na dhīrāḥ puraḥsthitāḥ prāśnikatāṃ prayānti /
saṃtiṣṭhate saṃsadi saṃśayānaḥ pṛṣṭhopaviṣṭāsu vilāsinīṣu // 3.28 //
itthaṃgate tatra gatasya kena manyus tavodīrṇarayo nivāryaḥ /
vicāyamāṇo na guṇāya naḥ syād asau kapīnāṃ kadanānubandhaḥ // 3.29 //
prapannaśāṭhyasya haṭhopanāmād āneṣyamāṇasya guṇaṃ na vīkṣe /
tathāvidhā hi dviṣatāṃ stavena kaṣanti karṇau param antikasthāḥ // 3.30 //
spṛṣ.o na ced vikriyayā kayāpi nopasthitaḥ kāryavaśād idānīm /
etu svayaṃ na kriyate vṛthaiva sa tvadbalātkāraniruddhaśobhaḥ // 3.31 //
yathāvayos tātajaṭāyuṣoktam uktaṃ kabandhena yathā ca tena /
tathopayāsyaty acirād avaśyam āhūtacakro haricakravartī // 3.32 //
yat satyam ādityasamudbhavatvād udbhūtim eṣyanti guṇās tadīyāḥ /
tulāgrasampātavilambadoṣe tasmin visaṃvādam ahaṃ na śaṅke // 3.33 //
viropataptasya videśavāsād ājagmuṣas tasya nijādhivāsam /
iyanty ahāny utsukabandhuvargasaṃvargaṇavyagratayaiva yānti // 3.34 //
sa nūnam āyāsyati vānarendraḥ saṃdeśakārpaṇyam asaṃmataṃ me /
sarvasvabhūtaṃ visṛjanti vatsa kāryātipāte 'pi na dhairyam āryāḥ // 3.35 //
vyatītavighnaḥ suhṛdarthanighno na śītalaḥ sthāsyati kiṃ gatena /
sa tvaryamāṇas tvaramāṇa eva dhairyasya naḥ pāram avāpsyatīva // 3.36 //
dhig astu māṃ taṃ prati vipratīpam āśaṅkitaṃ tad bahu yena yad vā /
svakāryadolāsu jano 'lpabhāgyaḥ prāṅ niścinoti vyatirekam eva // 3.37 //
itthaṃ na māṃ pratyupapattim eti tatrādyapakṣadvitaye 'pi yātrā /
varaṃ visoḍhāni kiyanty apīha saṅkhyātakālābhyadhikāny ahāni // 3.38 //
svālocitaṃ vaktu bhavān idānīm ity anuṣṭheyam asaṃśayaṃ yat /
bālyāt prabhṛty eva tavottarā dhīr na pūrvapakṣe 'bhiniveśam eti // 3.39 //
iti prasannāṃ giram agrajasya śrutvā cirād vītaviṣādapaṅkaḥ /
mukhaśriyaivojjvalayābhinandann idaṃ sumitrātanayo jagāda // 3.40 //
samarthitaṃ sādhu samastam eva kṛtī kapiḥ kiṃ bahunoditena /
pravyaktam apy etad adoṣadṛṣṭyā gṛhītam āryeṇa na yat tadāgaḥ // 3.41 //
tasyorjitenātiraver avadyaṃ naivekṣate 'ntaḥkaraṇaṃ mamāpi /
ayaṃ punas tena kuto na jānāmy atikramaḥ kālakṛto na dṛṣṭaḥ // 3.42 //
visarjitā vā punarāgamāya samāpatanty adya na te nalādyāḥ /
samāharatiudyatavittavarṣo varṣāntarebhyaḥ pravarān harīn vā // 3.43 //
aṣṭādaśadvīpakapīndramaulir maulān asau vālivadhapralīnān /
naikābhyupāyopahitān idānīm uta pratīcchaty atisatkriyābhiḥ // 3.44 //
aprāptasaṃtoṣam asaṃvibhaktasambhuktasarvasvam anuplaveṣu /
athādiśaty arpitasārakośaḥ prakāmapātheyaparigrahāya // 3.45 //
atho sa bhītāturabālavṛddhān vṛddhopasevī sanibhaṃ niyuṅkte /
yathāyathaṃ sadmani padmasaṃkhyāsametaśākhāmṛgayūthanāthaḥ // 3.46 //
tad ittham utthānasamākulatvān na tvaryamāṇo 'pi sa dūṣaṇāya /
āryeṇa tatrāpy asamāgatasya chāyācyutiḥ kaiva nirūpitā me // 3.47 //
jñeyo 'sti naḥ pūrvam arer udantas tatas tadantāya vidheyam asti /
ity ākulatvān mama tāvad ārya kāryopataptiḥ kṣamate na dhairyam // 3.48 //
anarthibhāvād upacāravādād ākāragupteś ca parībhavanti /
yathātathārthapratipādanena prayānti mitrāṇi nirantaratvam // 3.49 //
mithaḥ prasādaskhalanād upādher upaiti sakhyaṃ khalayoḥ khilatvam /
gṛhṇāti satpūruṣayor na doṣam audāryasaṃvādakṛtā tu maitrī // 3.50 //
niryantraṇaṃ yatra na vartitavyaṃ na moditavyaṃ praṇayātivāde /
viśaṅkitānyonyabhayaṃ vidūrān namaskriyām arhati sauhṛdaṃ tat // 3.51 //
visrabdham ājñāpaya kiṃ karomi kathaṃ ghaṭena plavagādhirājaḥ /
yat kiṃcid ādiśya nibhena kiṃ māṃ niṣedhasi kleśam ivekṣamāṇaḥ // 3.52 //
sthito 'smi tāvad dinam etad eṣyaty anyedyur ājñāṃ tava na pratīkṣe /
āryo 'nurodhān na yunakti yatra tatrāham ambāniyamāt svatantraḥ // 3.53 //
tad eṣa niṣkramya tathā karomi tatpūrvasaṅgaḥ plavago yathāśu /
taṃ vairisaṃhāramahopakāram āryāya niryātayituṃ yatena // 3.54 //
dhik taṃ dhanuṣpāṇim anuplavaṃ māṃ mudhā manaḥparyuṣitāvalepam /
anvīkṣate yasya mamāvasādād āryaḥ sahāyāntaram antarajñaḥ // 3.55 //
dhikśātatāṃ sannataparvatāṃ dhig dhik pātavegaṃ ca mamāśugānām /
alakṣitoddeśam api dviṣantaṃ nihatya nāyānti hi yat kṣaṇena // 3.56 //
kharādirakṣaḥkadaneṣu dīrgham amī parijñātarasāḥ kim anyat /
asṛgbhujāṃ bhoktum asṛk tvarantāṃ bhūyo 'pi bhūcandra tavārdhacandrāḥ // 3.57 //
ātāṭakāsaṃjñapanāt prapannas tvayaiṣa varṇāśramarakṣaṇāya /
samāpyatāṃ śeṣanibarhaṇena kṣapācarāṇāṃ nidhanādhikāraḥ // 3.58 //
uddhṛtya ghorāyudhavṛndavṛṣṭim anuṣṭhitām oghaśaravyayena /
sa rākṣasebhyo 'mbaragocarebhyas trātas tvayaikena makho maharṣeḥ // 3.59 //
taṃ kālakalpaṃ kiyatā śrameṇa tathāvadhīr mārgarudhaṃ virādham /
āsan muhūrtāḥ kati ca ghnatas te sainyaṃ janasthānapateḥ svarasya // 3.60 //
āsīt kṣuraprasya kiyān vilambaḥ krūreṣu teṣu triśiraḥśiraḥsu /
katīṣavo dūṣaṇadehayātrānivāraṇārthaṃ ca raṇe visṛṣṭāḥ // 3.61 //
kasyāvalambād akarot parāsuṃ mārīcam uccāvacamāyam āryaḥ /
jñātā ca diksaṃjñapitas tvayaiva kṣapācarāṇām adhipaś ca so 'pi // 3.62 //
ity aprapañcoditakāraṇāni mithas tayos tathyaguṇagrahāṇi /
neduḥ stuvānā iva sūnṛtāni khagāḥ kulāyeṣu kṛtaikamatyāḥ // 3.63 //
āropya sandhyāṃ diśi vajrapāṇeḥ paryasya candraṃ kakubhi pratīcyām /
utsaṅgitoḍuprakarā khagānām ādāya nidrāṃ rajanī jagāma // 3.64 //
kaviś ca jīvaś ca sutaś ca bhūmeḥ sa dakṣiṇāśātilako muniś ca /
āsannasūryāṃśubhayadrutasya kṣaṇaṃ rarakṣur bhagaṇasya pṛṣṭham // 3.65 //
dig darśayāmāsa mukhaṃ sarāgaṃ na yāvad aindrī gaṇikeva mattā /
tāvad dvijeśo 'ṅkam iyāya sindhoḥ sandhyāsurāśīkarasekaśaṅkī // 3.66 //
visarjitoḍuḥ kṣaṇam indur āsīd āsīd upānteṣv api nāndhakāram /
bhṛśaṃ diśīndrasya rarāja sandhyā sindūriṇīva graharājamudrā // 3.67 //
uccetum uccair uḍukuḍmalāni vyākurvatī rāgam atipragāḍham /
śākhām ivālambya diśaṃ maghonaḥ sandhyā viyad bhūruham āruroha // 3.68 //
śeṣoḍupuṣpastabakoccayāya saṃjātarāgātiśayeva sandhyā /
apy uccakaiḥ pūrvamahīdhramūrdhni sthitvā viyadbhūruham āruroha // 3.69 //
visṛjya pāṇḍucchadanaughatulyaṃ tārāgaṇaṃ vyomatarus tadānīm /
babhāra sandhyārucimañjarīṇāṃ jālāni bālādharakomalāni // 3.70 //
utsāritevābhrakareṇukumbhāj jambhāriyugyādhikṛtair marudbhiḥ /
tatāna sandhyeti samullasantī nabhoṅgaṇaṃ raṅgavikāradhūliḥ // 3.71 //
saṃvittayas tattvavigāhaśubhrāḥ sāṃsārike rāga ivādhirūḍhe /
śanaiḥ śanair mlānaruco mamajjus tārāḥ prasarpaty aruṇe cireṇa // 3.72 //
kaiḥ śikṣitā vartayituṃ tadāsīd unmīlitaṃ tūlikayeva citram /
taraṅgavat tuṅgaśaradghanālīvisaṃsthule vyomani bālasandhyā // 3.73 //
pratyuptamātraṃ diśi devabhartur abhyaktakhaṇḍābhrataraṅgamālam /
babhau visarpaty ayasīva tailaṃ nabhastale bālapataṅgatejaḥ // 3.74 //
prasattinirvyūḍhasudhāmayūkhaṃ kham uṣṇaraśmāv api raktim ūhe /
bharaṃ pratīcchanti mahodayānāṃ pātrāṇi paryāyavaśaṃvadāni // 3.75 //
tāpiñchakalpaḥ kakubhi pratīcyām ācchādayāmāsa mṛgo mṛgāṅkam /
dūrādhirūḍhaṃ janam āpadīva prakāśamāno 'tiśayād avarṇaḥ // 3.76 //
pradīptasandhyāgni nadaddvijaughaṃ niraṃśukadhvastaparāṅmukhendu /
viyadviluptoḍukulaṃ tadābhūc cakrāntarākrāntapuropameyam // 3.77 //
mithaḥ kṛtālokamahotsavānāṃ kṛtāravaḥ sārasadampatīnām /
na bālabhānoḥ kiraṇād abhaiṣīn nadīṣu nīhāramayo 'ndhakāraḥ // 3.78 //
tatāra saṃtīrya himāndhakāraṃ tatā rasantī saritaḥ khagālī /

smaravyayaklāntavalatpulindīsarvāṅgasaṃvāhakalāvidagdhaḥ /

atītya śaityaṃ taṭavālukānāṃ koṣṇāsu sasnur munayo nadīṣu /
apām apūryanta ca nairjharīṇāṃ kuṭā girigrāmakuṭumbinībhiḥ // 3.81 //
jātaprakāśāsu bahiḥsthalīṣu prakāmacārotsavajāgarūkaḥ /

papau raveḥ saṃjvalitograbhāsaṃ bhujaṅgam aughaḥ śritanākurandhraiḥ // 3.84 //

āgantave stokam api dvirephaḥ sthito dadau nāntaram abjamadhye // 3.94 //

dṛśaṃ pramīlāpagamāt kaṣāyām unmīlayāmāsa cireṇa cakrī // 3.96 //

śrutapriyāhnvānaravaḥ pratasthe sīmantayann abjavanaṃ vinidram // 3.97 //
