#Author:Abhinanda
#Text:Kādambarīkathāsāra
#Data entry:Harunaga Isaacson
#Edition:Pdt. Durgaprasad and Kasinath Pandurang Parab, Kavyamala [second series], vol. 11, 1888.


śriyaṃ diśantu vaḥ śaurer dvaye tulyaśramāḥ kramāḥ /
ye cādau goṣpadaṃ paścāt trailokyaṃ krāmyataś ca ye // 1.1 //
sarasāḥ sadalaṅkārāḥ prasādamadhurā giraḥ /
kāntās tātajayantasya jayanti jagatīguroḥ // 1.2 //
guṇoddyotanadīpānāṃ satāṃ na param ujjvalam /
yāvan malinam apy eṣāṃ karma dṛṣṭeḥ prasādhanam // 1.3 //
guṇaḥ kṛśo 'pi prathate pṛthur apy apacīyate /
prāpya sādhukhalau candraḥ pakṣāv iva sitāsitau // 1.4 //
śaktināmābhavad gauḍo bhāradvājakule dvijaḥ /
dārvābhisāram āsādya kṛtadāraparigrahaḥ // 1.5 //
tasya mitrābhidhāno 'bhūd ātmajas tejasāṃ nidhiḥ /
janena doṣoparamaprabuddhenārcitodayaḥ // 1.6 //
sa śaktisvāminaṃ putram avāpa śrutaśālinam /
rājñaḥ karkoṭavaṃśasya muktāpīḍasya mantriṇam // 1.7 //
kalyāṇasvāmināmāsya yājñavalkya ivābhavat /
tanayaḥ śuddhayogarddhinirdhūtabhavakalmaṣaḥ // 1.8 //
agādhahṛdayāt tasmāt parameśvaramaṇḍanam /
ajāyata sutaḥ kāntaś candro dugdhodadher iva // 1.9 //
putraṃ kṛtajanānandaṃ sa jayantam ajījanat /
āsīt kavitvavaktṛtvaphalā yasya sarasvatī // 1.10 //
vṛttikāra iti vyaktaṃ dvitīyaṃ nāma bibhrataḥ /
vedavedāṅgaviduṣaḥ sarvaśāstrārthavādinaḥ // 1.11 //
jayantanāmnaḥ sudhiyaḥ sādhusāhityatattvavit /
sūnuḥ samudabhūt tasmād abhinanda iti śrutaḥ // 1.12 //
||yugmam
kāvyavistarasaṃdhānakhedālasadhiyaḥ prati /
tena kādambarīsindhoḥ kathāmātraṃ samuddhṛtam // 1.13 //
puruhūtapuraspardhivibhavā bhūṣaṇaṃ bhuvaḥ /
babhūva vidiśā nāma purī vetravatītaṭe // 1.14 //
yā puṇyakarmiṇāṃ tejaḥśālināṃ dyaur ivāspadam /
reje ratnagṛhajyotir jālair indrāyudhaprabhaiḥ // 1.15 //
||*yugmam*
tasyāṃ nijabhujodyogavijitārātimaṇḍalaḥ /
ākhaṇḍala iva śrīmān rājā śūdraka ity abhūt // 1.16 //
aṣṭānāṃ lokapālānāṃ rūpaṃ bibhrad api prajāḥ /
śaśvad ānandayaṃs tābhiś candramā ity avedi yaḥ // 1.17 //
||*yugmam*
sa medinīṃ vinirjitya caturambhodhimekhalām /
sacivārpitatadbhāras tasyām āsta yathāsukham // 1.18 //
tenānudvejitajanair arthadharmāvirodhibhiḥ /
vinodair atyanīyanta nave vayasi vāsarāḥ // 1.19 //
tasya cānanyalabhyānāṃ sukhānāṃ pāradṛśvanaḥ /
yūno 'pi suratakrīḍāparāṅ mukham abhūn manaḥ // 1.20 //
prayatnenāpi nāvāpi sā yoṣid viṣameṣuṇā /
hartuṃ śaśāka yā tasya hṛdayaṃ cakṣur eva vā // 1.21 //
atha bāhyāṅgaṇasthānavartinaṃ taṃ kadācana /
prasṛtya praṇayaprahvā pratīhārī vyajijñapat // 1.22 //
dūrād upāyanīkṛtya śukam āścaryaceṣṭitam /
caṇḍāladārikā kācid āgatā darśanārthinī // 1.23 //
śrutvā kṣaṇaṃ parāmṛśya mukham ālokya mantriṇām /
samādideśa tāṃ rājā ko doṣaḥ praviśatv iti // 1.24 //
nijena lakṣmaṇā lokasparśaṃ pariharanty atha /
mātaṅgakanyakāvikṣad anujñātā nṛpāṅgaṇam // 1.25 //
tayā ca saha hastasthasaśukasvarṇapañjaraḥ /
vahann ahiṃsram ākāraṃ caṇḍālasthaviro 'viśat // 1.26 //
praviśantīṃ tu tāṃ dṛṣṭvā kanyām adbhutadarśanām /
paraṃ vyasmayata kṣmāpaḥ kṣaṇaṃ caivam acintayat // 1.27 //
idaṃ rūpam iyaṃ kāntir ime lāvaṇyaśīkarāḥ /
martyatām api na prāhurmātaṅgatve tu kā kathā // 1.28 //
tat satyam eva paśyāmi nūnaṃ mātaṅgamāyayā /
channeyaṃ devatā kācin martyalokam upāgatā // 1.29 //
ityādi cintayantaṃ sā pragalbhavaniteva tam /
praṇanāma mahīpālam anākulavilokinī // 1.30 //
tataḥ kṣititalanyastajānuḥ praṇatakandharaḥ /
gāḍhobhayakarāsaktapañjaraḥ sthaviro 'bravīt // 31 //
vaiśampāyananāmāyaṃ ratnam atyadbhutaṃ śukaḥ /
ratnānāṃ cāspadaṃ devo dugdhodadhir ivāparaḥ // 1.32 //
ity ādareṇa naḥ svāmiduhitrāyam upāhṛtaḥ /
prasādaḥ kriyatām asyā vihagaḥ parigṛhyatām // 1.33 //
ity uktvā so 'grato rājñaḥ pañjaraṃ maṇikuṭṭime /
vinidhāya praṇamyainaṃ sapraśrayam apāsarat // 1.34 //
iṅgitajñā pratīhārī vivṛtadvārasampuṭam /
akarot pañjaraṃ tasmān niryayau ca vihaṅgamaḥ // 1.35 //
utkṣipya dakṣiṇaṃ pādaṃ jayaśabdam udīrya saḥ /
prayuktavedamantrāśīr imām āryām athāpaṭhat // 1.36 //
stanayugamaśrusnātaṃ samīpataravarti hṛdayaśokāgneḥ /
carati vimuktāhāraṃ vratam iva bhavato ripustrīṇām // 1.37 //
tām ākarṇya mahīpālo vismayotphullalocanaḥ /
kumārapālanāmānaṃ pradhānāmātyam abravīt // 1.38 //
dṛṣṭaṃ bhavadbhir āścaryam īdṛśaṃ śrutam eva vā /
tiryaktve 'tiśayaḥ ko 'yaṃ yo manuṣyeṣu durlabhaḥ // 1.39 //
śrutādhyayanasaṃskāro mātaṅgajanasaṃgatiḥ /
mātaṅgeṣu ca kanyeyam ity āścaryaparamparā // 1.40 //
amātyas tv abhyadhād deva pūrvasaṃskārakāritāḥ /
bhavanty atiśayāḥ prāyaḥ prāṇināṃ ko 'tra vismayaḥ // 1.41 //
etāni tāni liṅgāni paralokānumāṃ prati /
tārkikair upapādyante lokāyatanirākṛtau // 1.42 //
vicitrā vedhasaḥ sṛṣṭirvicitrā karmavāsanā /
vicitras tadvipākaś ca vicitraḥ kālaparyayaḥ // 1.43 //
ityādiśukasambandhikathāvicchedakāraṇam /
udabhūt tāragambhīro madhyāhṇapaṭahadhvaniḥ // 1.44 //
bahiḥ prakalpyatām asyāḥ sukhāyāvasathādikam /
ayaṃ cābhyantaraṃ patrarathabhūpaḥ praveśyatām // 1.45 //
ity ādiśya pratīhārīṃ sabhām aujjhan mahīpatiḥ /
visasarja ca rājanyacakraṃ sevārtham āgatam // 1.46 //
tataḥ kṛtālpavyāyāmaḥ snātaḥ pūjitadaivataḥ /
hutāgniḥ śrutaviprāśīr bhuktvā śayyāgṛhaṃ yayau // 1.47 //
tatra paryaṅkam āruhya dhyātvaiva kim api kṣaṇam /
tam antaḥpuraviśrāntaṃ śukam ānāyayat punaḥ // 1.48 //
taṃ ca kañcukinānītamādau pṛṣṭvā sukhāsikām /
papraccha kautukākṣiptas tadvṛttāntam anantaram // 1.49 //
vaiśampāyana ko nu tvaṃ kva jātaḥ kvoṣitaś ciram /
kutastyam idam īdṛk te kalāśāstrādikauśalam // 1.50 //
manye jātismaraḥ ko 'pi bhavān bhūmim imāṃ gataḥ /
na hy asty atiśayaprāptir iyatī pakṣijātiṣu // 1.51 //
kena vā bhavato nāma kṛtaṃ viprajanocitam /
īdṛśānyabhidhānāni kuto 'ntyajanaveśmasu // 1.52 //
kathaṃ caṇḍālasamparka iha vā katham āgamaḥ /
iti kārtsnyena vṛttāntam ātmano vaktum arhasi // 1.53 //
iti pṛṣṭaḥ kṣitīśena bahumānapuraḥsaram /
kṣaṇaṃ buddhyānusaṃdhāya śuko vaktuṃ pracakrame // 1.54 //
deva vindhyāṭavī tāvattava śrutipathaṃ gatā /
pūrvāparapayorāśivelāsaṃsparśaśālinī // 1.55 //
yadekadeśe kaṇḍūlagaṇḍebhadalitadrume /
uṣitau daṇḍakāraṇye sasītau rāmalakṣmaṇau // 1.56 //
||*yugmam*
yatrāgastyamuner āsīd āśramaḥ śamināṃ mataḥ /
lopāmudrākarākṣiptajalasaṃvardhitadrumaḥ // 1.57 //
tasyopakaṇṭhe pampākhyam asti puṇyajalaṃ saraḥ /
sītākaratalollūnaśravaṇābharaṇotpalam // 1.58 //
tasyāsti paścime tīre jīrṇaḥ śālmalipādapaḥ /
ārohapariṇāhābhyāṃ vyāptavyomadigantaraḥ // 1.59 //
tasmin kṛtakulāyāni vītabhītīni durgame /
vihaṅgamasahasrāṇi nivasanti nirantaram // 1.60 //
divā tāny abhinirvartya prāṇayātrām itas tataḥ /
tatrāgatya svanīḍeṣu sukhaṃ rātriṣu śerate // 1.61 //
tatraiva baddhavasatī abhūtāṃ pitarau mama /
jāto 'ham ekas tanayas tayor vayasi paścime // 1.62 //
udvoḍhumasamarthā sā tīvrāṃ prasavavedanām /
tayaiva pañcatāṃ nītā jananī me durātmanaḥ // 1.63 //
jāyāviyogadagdho 'pi tātaḥ putraiṣaṇāturaḥ /
śokamutsṛjya matprāṇapoṣaṇe yatnam agrahīt // 1.64 //
na śaśāka kvacid gantum ātmanaḥ prāṇavṛttaye /
māṃ vakṣasi nidhāyāsta nīḍa eva divāniśam // 1.65 //
anyapatrarathānītabhuktojjhitaphalādinā /
madāpyāyanaśiṣṭena so 'karod vṛttim ātmanaḥ // 1.66 //
atikrāmati kāle 'tha kiñcidudbhinnapakṣatau /
tātavakṣaḥsthalasthe 'pi mayy uḍḍayanalālase // 1.67 //
akasmād eva sambhrāntamṛgapakṣikulaśrutaḥ /
abhūt kalakalas tasminn araṇye yamadundubhiḥ // 1.68 //
||yugmam
tatas tātorasaḥ kiñcid unnamayya śirodharām /
kutūhalavaśāc cakṣur dikṣu nikṣiptavānaham // 1.69 //
athāpaśyam asaṃkhyātakirātaparivāritam /
vane śabarasenānyam āyāntaṃ mṛgayārasāt // 1.70 //
tato mṛtam ivātmānam utkhātam iva śālmalim /
parivṛttam ivāraṇyaṃ bhayenāham acintayam // 1.71 //
hataprāṇisahasrāṅgasravadrudhiracarcitaiḥ /
murhūrtād eva tadvyādhair athāraṇyam apūryata // 1.72 //
senāpatiḥ sa nistriṃśo mṛgayājanitaśramaḥ /
niṣasāda tataḥ pampāsarasas tīraśādvale // 1.73 //
kṣaṇaṃ viśramya pītvāpaḥ prāśya bālamṛṇālikāḥ /
pulindaiḥ saha senānīḥ sa jagāma yathāgatam // 1.74 //
eko jaratpulindas tu pāpas tatra vyalambata /
prāṃśur bhayaṅkarākāraḥ kṛtāntasyeva kiṅkaraḥ // 1.75 //
ajātapiśitaprāptir anubhūtavṛthāśramaḥ /
yamadūtīm iva dṛśaṃ sa dadau tatra śālmalau // 1.76 //
sa durārohatāṃ tasya jānann api mahātaroḥ /
jātānadhyavasāyo 'pi yatnam ārohaṇe 'karot // 1.77 //
athāham asamatrāsaviparyāsitamānasaḥ /
aviśaṃ pitur utsaṅgaṃ tatpakṣāvṛtavigrahaḥ // 1.78 //
sa tu pāpas tam āruhya taruṃ nīḍanivāsinaḥ /
hatvā hatvā yathāśakti śukān bhūmāv apātayat // 1.79 //
krameṇa sa durācāras tātasyāpi śirodharam /
babhañja taṃ ca cikṣepa kṣitau vigatajīvitam // 1.80 //
tadā mama na tacchoko nānyat kiñcic ca cetasi /
kevalaṃ sahajātena bhayenaivāsmi viplutaḥ // 1.81 //
tato 'haṃ pitur utsaṅgādvinirgatya sasādhvasam /
antaraṃ taruśīrṇānāṃ parṇānāṃ tūrṇamāviśam // 1.82 //
pāpaḥ so 'py avaruhyātha śālmaler mūlavartinaḥ /
śukān gatāsūn ādāya vyādhān anusarann agāt // 1.83 //
ahaṃ vidhiniyogāt tu tena duṣkṛtakāriṇā /
jīrṇaparṇasavarṇatvāttadānīṃ nāvadhāritaḥ // 1.84 //
tato dūram atikrānte tasmin dāruṇakarmaṇi /
saṃjātajīvitāśaṃ māṃ pipāsā paryabādhata // 1.85 //
anuḍḍayanasāmarthyāt padbhyām eva yathātathā /
pampāpulinaparyantaṃ gantum aicchaṃ pipāsitaḥ // 1.86 //
tathāpy atinirutsāhaḥ patan muhur itas tataḥ /
tam eva pāpakarmāṇaṃ śaṅkamānaḥ pade pade // 1.87 //
ciraṃ hā tāta tāteti śocan kaṇṭhāntavartibhiḥ /
prāṇair āyāsyamāno 'ham atiṣṭham atiniṣṭhuraḥ // 1.88 //
||*yugmam*
tasmād anatidūre 'sti sarasas tapasāṃ nidheḥ /
nirdhūtakleśajālasya jābāler āśramo muneḥ // 1.89 //
tatas tasmin sutas tasya sarasi snātum āgataḥ /
hārītanāmā māṃ tatra tadavasthaṃ vyalokayat // 1.90 //
kaṣṭaṃ kenāpi pāpena pātitaḥ śukaśāvakaḥ /
kulāyātklāmyatītyuktvā kṛpayā māṃ kare 'grahīt // 1.91 //
apāyayat payobindūn marutāśvāsayan muhuḥ /
kṛtāhnikaś ca kāruṇyād anayat pitur āśramam // 1.92 //
samāśvastasya hārītatātaṃ dṛṣṭyaiva paśyataḥ /
padaṃ sthiram iva nyastaṃ hṛdi me jīvitāśayā // 1.93 //
nīḍacyuto 'yamānīto mayā vihagapotakaḥ /
iti pitre praṇamyāsau pāṇisthaṃ māmadarśayat // 1.94 //
atha jābālim adrākṣaṃ sākṣād iva pitāmaham /
mahātmabhiḥ parivṛtaṃ sānukampais tapodhanaiḥ // 1.95 //
jarādhavalamūrdhānaṃ dhāmnā prasaratā bahiḥ /
anatisphuṭasaṃlakṣyaśarīrāvayavākṛtim // 1.96 //
sa tu māṃ kṣaṇam udvīkṣya smayamāna ivāvadat /
idaṃ hi bhujyate 'nena phalaṃ svasyaiva karmaṇaḥ // 1.97 //
trikāladarśī bhagavānsa hi divyena cakṣuṣā /
viśvaṃ paśyati pāṇisthabadarāmalakādivat // 1.98 //
vacas tad ākarṇya kutūhalākulāḥ sametya sarve munayas ta mabruvan /
babhūva ko 'yaṃ bhagavan bhavāntare kim asya vā duścaritaṃ tad ucyatām // 1.99 //
abhyarthito munibhir ity uparudhyamānas tān abravīt kuruta tāvadahaḥkriyāḥ svāḥ /
sāṃdhyaṃ vidhāya vidhim askhalitāvadhānam ākarṇayiṣyatha kathāṃ pṛthum etadīyām // 1.100 //
uktās tathā bhagavatā munayo nijena karmakrameṇa divasaṃ gamayāṃbabhūvuḥ /
hārītahastavihitaiś ca manovinodais tais tair ānīyata mayāpi dināvaśeṣaḥ // 1.101 //
||iti śrīvipaścidvarāgragaṇayācāryabhaṭṭaśrījayantasūnorabhinandasya kṛtau kādambarīkathāsāre prathamaḥ sargaḥ
||dvitīyaḥ sargaḥ
atha saṃdhyāsamādhyantāḥ samāpya munayaḥ kriyāḥ /
śuśrūṣavaḥ samāgamya yathāsthānam upāviśan // 2.1 //
aham apy ātmavṛttāntavistaraśravaṇotsukaḥ /
saṃhṛtya cāpalaṃ sarvam āsaṃ sthiramanās tadā // 2.2 //
atha dugdhābdhikalloladhavalair amṛtāñcitaiḥ /
siñcann iva karair indur udagādāśramasthalīm // 2.3 //
bhagavānatha jābāliḥ sukhāsīnaḥ svaviṣṭare /
valkalāhitaparyaṅkabandho vaktuṃ pracakrame // 2.4 //
sarve dattāvadhānāḥ stha śṛṇutemāṃ tapodhanāḥ /
amuṣya pūrvavṛttāntaprastāvopanatāṃ kathām // 2.5 //
asty avantiṣu saṃsāralakṣmyāḥ prasavabhūr iva /
nidhir ujjayinī nāma nagarī guṇasampadām // 2.6 //
laṅkālakāmarāvatyo yeṣāṃ vasatayaḥ priyāḥ /
te 'pi yām abhipaśyantaḥ sukhaṃ tāsu na śerate // 2.7 //
kim anyad yatra kailāsanivāsaprītim ujjhatā /
mahākālābhidhānena rudreṇa svayam āsyate // 2.8 //
||tilakam
tasyāṃ bharatamāndhātṛbhagīrathapṛthūpamaḥ /
tārāpīḍa iti śrīmān babhūva pṛthivīpatiḥ // 2.9 //
parasparavirodhena viprakīrṇair itas tataḥ /
visrabdham āsyate yatra sambhūya sakalair guṇaiḥ // 2.10 //
||*yugmam*
vihasan buddhivibhavair bhṛgūdvahabṛhaspatī /
amātyaḥ śukanāsākhyo brāhmaṇas tasya cābhavat // 2.11 //
trivikramapadākramya sa rājā pṛthivīm imām /
tasmin mantriṇi tadbhāraṃ nicikṣepa bharakṣame // 2.12 //
tathā ca pālayāmāsa sa mantrī sakalāḥ prajāḥ /
yathā trividham apy etā na duḥkhaṃ kiñcid aspṛśan // 2.13 //
yathā yathā ca loko 'smin pitarīvānvarajyata /
rājā tathā tathātmānaṃ kṛtakṛtyam amanyata // 2.14 //
akleśamarpitaṃ tena svabhuje jagadarjanam /
sambhogavighnakārīti pālanaṃ mantrisātkṛtam // 2.15 //
kṛtakṛtyatayā cāsau kāryaśeṣam acintayan /
viṣayeṣvāhitaprītir āsāṃcakre yathāsukham // 2.16 //
śobhā hi kṛtakṛtyasya rājño bhogavibhūtayaḥ /
asamāptajigīṣasya tā eva tu viḍambanāḥ // 2.17 //
sa yuvā madhurākāras taruṇīnayanotsavaḥ /
kalāsu kuśalaḥ kāmī bubhuje bhogasampadaḥ // 2.18 //
na te bhogā na tāḥ krīḍā na tā goṣṭhyo na te rasāḥ /
na te vilāsā yeṣv asya pāragaṃ nābhavan manaḥ // 2.19 //
samagrasukhasambhogabhūmijño 'pi sa bhūpatiḥ /
sukham ekaṃ tu nājñāsīd apatyālokanodbhavam // 2.20 //
yathā yathāsya kālena vyatīyāya navaṃ vayaḥ /
nirapatyatayā śoko vardhate sma tathā tathā // 2.21 //
mukhyabhūtā ca sarvasminn antaḥpurakadambake /
rājñī vilāsavatyākhyā tasya prāṇasamābhavat // 2.22 //
sā rājamahiṣī gaurīm abhyarcayitum ekadā /
devatāyatanaṃ bāhyaṃ yayau bhartur anujñayā // 2.23 //
purāṇe vācyamāne 'tha prasaṅgapatitāṃ śrutim /
aputrasya kuto lokā iti śuśrāva tatra sā // 2.24 //
prataptalohasūcīva sā śrutiḥ śravaṇāntaram /
viveśa dadatī tasyāḥ parāṃ kām api vedanām // 2.25 //
tataḥ pratinivṛttātha praviśya bhavanaṃ nijam /
vastrāvṛtamukhī bhūmāvupaviśya ruroda sā // 2.26 //
sakhībhiḥ prārthyamānāpi na svām abhajata sthitim /
nodasthāt kartum āhāraṃ virarāma na rodanāt // 2.27 //
athopalabhya tatpīḍāv ārtāṃ parijanān nṛpaḥ /
praviśyāntaḥpuraṃ prāha svairam enāṃ prasādayan // 2.28 //

kṛtā pratikriyā tatra tadālasyaṃ hi māmakam // 8.15 //

na yāvad bhavatā tāvan na gantavyam itaḥ kvacit // 8.16 //

tātena tatra karmaikadeśaleśāṅgatāṃ gataḥ // 8.17 //

krameṇāmantrya sa munīn pratasthe divam utpatan // 8.18 //

nīto vāritavāmena kāmena vaśam ātmanaḥ // 8.19 //

akārṣaṃ dhiyam utkrāntagurvājño gamanaṃ prati // 8.20 //

ekākī sahasoḍḍīya prasthito 'smy uttarāṃ diśam // 8.21 //
adūroḍḍīna evāham anabhyastādhvalaṅghanaḥ /
krāntas tṛṣā ca klāntyā ca jalaṃ kvacid atarkayam // 8.22 //
tataḥ kvāpi payaḥ pītvā kiñcid ādaśya cañcunā /
viśrāntaye 'viśaṃ śrāntas taroḥ kasyāpi koṭare // 8.23 //
kṣaṇaṃ tatra niṣaṇṇasya tadā me nidrayā manaḥ /
gāḍhayā pāśapaṅktyā ca śarīrakam agṛhyata // 8.24 //

kṛtārtham iva mātaṅgayuvānaṃ mama bandhanāt // 8.25 //
ākṛtyā karmaṇā cāsya samarthyāpiśitārthitām /
prārthanāpeśalaṃ kiñcid ātmamokṣārtham abhyadhām // 8.26 //

na hi māṃsārthino yuktaṃ nidrāchedapratīkṣaṇam // 8.27 //

mayā baddhena ko 'rthas te kā vimuktena vā kṣatiḥ // 8.28 //
so 'bravīn na svatantreṇa gṛhīto 'si mayā śuka /
prathitas tvaṃ hi jālāber āśrame suciraṃ sthitaḥ // 8.29 //
sa tvaṃ gato mama svāmiduhituḥ karṇagocaram /
tanniyogād gṛhīto 'si katham unmucyase mayā // 8.30 //
punar ākhyām amuṃ saumya yuvāsi smaravedanām /
jānāsi na priyotkaṇṭhākṛṣṭaṃ māṃ roddhum arhasi // 8.31 //
so 'bhāṣata na bhṛtyena taruṇenāpi śāsanam /
svāmino 'laṅghanīyaṃ tat kathaṃ tvāṃ tyaktum utsahe // 8.32 //
avocaṃ punar apy enaṃ śāpāj jātismaro muniḥ /
śukībhūto 'smi māṃ muñca paraṃ puṇyam avāpsyasi // 8.33 //
sa tvāha nāhaṃ jānāmi muniṃ gandharvameva vā /
rākṣasaṃ vā piśācaṃ vā nīta evāsi pakvaṇam // 8.34 //

prāveśayac ca caṇḍālakumārīveśma kaśmalam // 8.35 //
dṛṣṭvā tadārdragomāṃsavasāsṛkpaṅkapicchalam /
procchalatkrimi matprāṇair utkrāntam iva durbhagaiḥ // 8.36 //

svāminy ayam ihānītaḥ sa vaiśampāyanaḥ śukaḥ // 8.37 //

kṣaṇaṃ vakṣasi dhṛtvaiṣā nyakṣipad baddhapañjare // 8.38 //

munir bhūtvādya caṇḍālyā ahaṃ krīḍāśukaḥ sthitaḥ // 8.39 //
vyaktāṃ vidadhatā vācaṃ jātiṃ smarayatā nijām /
aho jābālinā divyadṛṣṭināpakṛtaṃ mama // 8.40 //
aho bata mahat kaṣṭam aviṣahyam upasthitam /
brāhmaṇena mayā neyaḥ kālaś caṇḍālaveśmani // 8.41 //
bhavatvanaśanenaiva tyajāmi hatajīvitam /
tāḍyamāno 'pi nāśnāmi na vācaṃ visṛjāmi ca // 8.42 //
evamudvejitaiṣā māṃ hanyādvā yadi vā tyajet /
bhavaty ubhayathāpy eṣa panthāḥ śreyaskaro mama // 8.43 //
ity upārūḍhasaṃkalpam upāhṛtaphalodakā /
eṣā caṇḍālakanyā mām ākhyan madhurayā girā // 8.44 //
phalāny aśāna pānīyam upayuṅkṣva kim īkṣase /
bhagnamaunavrato 'py adya kiṃ maunam avalambase // 8.45 //
munir jātismaro 'si tvaṃ yadi ko 'pi tathāpi te /
tiryagjātyucitā prāṇavṛttir evādya saṃmatā // 8.46 //
na tiryagadhikāraṃ hi śāstraṃ śāstravido viduḥ /
tiraścāṃ kva ca saṃskārāś cauḍopanayanādayaḥ // 8.47 //
bhakṣyābhakṣyavicārādi yadi cādya cikīrṣasi /
tat kim ucchiṣṭaśaucādi nāśrame 'py āśritaṃ tvayā // 8.48 //

hastād asmākam ādatta viśvāmitraḥ śvajāghanīm // 8.49 //
kṛtadvitropavāso 'pi jihāsur iva jīvitam /
lakṣyase vatsa mā maivaṃ kṛthā bhuṅkṣva phalodakam // 8.50 //
ity acaṇḍālasadṛśair vākyair astrījanocitaiḥ /
tasyā vismayamāno 'haṃ ciram evam acintayam // 8.51 //
nūnam eṣā na caṇḍālī madvat kenāpi karmaṇā /
manye gatim imāṃ prāptā madarthenaiva vā punaḥ // 8.52 //
bibharty urasi māteva paśyati snigdhayā dṛśā /
vakti caivam ato 'muṣyāḥ karomi vacanaṃ varam // 8.53 //
iti matvāsṛjaṃ vācam abhavaṃ bhojanonmukhaḥ /
abhojayac ca mām eṣā jananīva dine dine // 8.54 //

na tv eṣā prārthyamānāpi nijaṃ rūpam adīdṛśat // 8.55 //

sa rājā śūdrakaḥ śayyāgṛhād aṅganam āgamat // 8.60 //

muktvā manobhavarujaṃ suciraṃ sa rājā prāṇān mumoca sahasaiva vihaṅgamo 'pi // 8.75 //
