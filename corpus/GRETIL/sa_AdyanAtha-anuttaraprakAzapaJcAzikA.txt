#Author:Ādyanātha
#Text:Anuttaraprakāśapañcāśikā
#Data entry:Somadeva Vasudeva
#Edition:Mukundarama Shastri, Bombay: Tatva Vivechaka Press 1918 (Kashmir Series of Sanskrit Texts and Studies, 13).


oṃ namo 'nuttaraprakāśaśaṅkarāya
akṛtrimāhamāmarśaprakāśaikaghanaḥ śivaḥ /
śaktyā vimarśavapuṣā svātmano 'nanyarūpayā // 1 //
śivādikṣitiparyantaṃ viśvaṃ vapur udañcayan /
pañcakṛtyamahānāṭyarasikaḥ krīḍati prabhuḥ // 2 //
pṛthagartham arthavattā viśvaviśvaśarīrayoḥ /
na viśvaviśvavapuṣor bhinnatā kāpi tāttvikī // 3 //
bhede sattāsphurattābhyāṃ bhinnaṃ kiṃ tu jagad bhavet /
sattāsphurattāsaṃbandhāt sattābhānaṃ ca tan na cet // 4 //
asataḥ kiṃ satas tābhyāṃ saṃbandhaḥ so 'yam iṣyate /
svarūpalābhasulabhaḥ saṃbandho na hy avastunaḥ // 5 //
na labdharūpasaṃbandhe tābhyāṃ kiṃ cid upekṣyate /
upekṣāyām avasthānaṃ na syād etasya kiṃ cana // 6 //
svabhāvataḥ sphurattā ca sattā ca na vināśinī /
vināśānabhupagame jaḍatāpi nivartate // 7 //
tato gatyantarābhāvāc cid eva pariśiṣyate /
sataś cidaikyaṃ prakṛtaṃ kadā cin na nivartate // 8 //
prakāśe 'nanyato bhāvaḥ svātantryollāsakevalaḥ /
parichinnātmikā śaktiḥ śambhor viśvātiśāyinaḥ // 9 //
yat prakāśātmakaṃ sarvaṃ tamaḥkevalatāṃ gatam /
yac ca kiṃ cij jagad etat prakāśād atiricyate // 10 //
vimṛśyasaraṇīṃ prāptam ity eṣā tāttvikī matiḥ /
avabhāsaikatānānāṃ matir ekātra sākṣiṇī // 11 //
kiṃ pramāṇāir varākais taiś cidānuprāṇitātmabhiḥ /
na hi vaikartanaṃ jyotir dīpālokam apekṣate // 12 //
śaktipātapavitre 'smin dhītattve ca parīkṣyatām /
ādarśavimalābhoge na tu sarvaṃ prakāśate // 13 //
itthaṃ cidātmakaṃ sarvaṃ ṣaṭtriṃśadbhedatattvataḥ /
ādau śuddhātmakaṃ tattvaṃ pañcadhā tamasaḥ param // 14 //
śivaḥ śaktiś ca sādākhyam īśo vidyeti bhidyate /
akṣādiśāntavarṇātmāniramāyi śivena tu // 15 //
kalāvidyārāgakālaniyatir bandha ucyate /
māyāpūrvo vakārādikṣakārāntākṣarātmakaḥ // 16 //
pumāñ śaktir mano buddhir ahaṃkārādipañcakam /
śrotrādipañcakaṃ tādi ṭādi vāgādipañcakam // 17 //
tanmātrapañcakaṃ cādi kādi vyomādipañcakam /
sisṛkṣoḥ prathamaspandaḥ śivatattvaṃ vibhoḥ smṛtam // 18 //
icchaiva sāparimlānā śaktitattvam udañcayan /
svecchayā sūcitaṃ viśvam ācchādyāhantayā sthitam // 19 //
sa eva tattvaṃ sādākhyaṃ sarvānugrahaṇodyatam /
sa eveśvaratattvaṃ syāt paśyan viśvam idantayā // 20 //
idantāhantayor aikyam iti vidyā nigadyate /
svāṅgakalpeṣu bhāveṣu māyātattvaṃ vibhedadhīḥ // 21 //
māyāgṛhītasaṃkocaḥ śivaḥ puṃtattvam ucyate /
ayam eva hi saṃsārī jīvo bhoktaiva dṛśyate // 22 //
jñatvakartṛtvapūrṇatvanityatvānyasya śaktayaḥ /
tatsaṃkocāt saṃkucitaḥ kalādyātmatayā mataḥ // 23 //
māyātmakaṃ kalā nāma kiṃcitkartṛtvakāraṇam /
kālaḥ paricchedakaro niyatiś cedam eva me // 24 //
kartavyaṃ nānyad ity eṣā vyavasthā yantraṇākṛitiḥ /
prakṛtir guṇasāmyaṃ syād ahaṃkārādijanmabhūḥ // 25 //
ahaṃ mamedam ity etad buddhihetur ahaṃkṛtiḥ /
buddhir adhyavasāyasya kāraṇaṃ niścayātmanaḥ // 26 //
saṃkalpasya vikalpasya bījaṃ mana udīryate /
vacanādeś ca śabdāder vāgādiśravaṇādikam // 27 //
kāraṇaṃ śravaṇādīnāṃ grāhyaṃ tanmātrapañcakam /
ākāśādyavakāśādikāraṇaṃ bhūtapañcakam // 28 //
parāparāśaktimaye śuddhe vidyādipañcakam /
tadanyad aparāśaktir ity etat tattvam īritam // 29 //
iyaṃ devī parā śaktiḥ śuddhāśuddhādhvagarbhiṇī /
pṛthivyādīni tattvāni yadā līnāni kāraṇe // 30 //
tadā kāraṇamātrāṇi bahir udvamate vibhuḥ /
anuttarecche unmeṣe ānandeśanam ūnatā(?) // 31 //
kriyecchājñānaśaktīnāṃ sattā ṣaṭ . . . . .* /
iccheśanāntarārūḍhā sphuṭāsphuṭajaganmayī // 32 //
catvāraḥ parato varṇāḥ ṣaṇḍātmānaḥ pracoditāḥ /
anuttarānandaśāktis trikoṇād vṛttiyogataḥ // 33 //
tathaivoneṣayogena kriyāśakteḥ sphuṭaṃ vapuḥ /
uktaṃ triśaktisaṃghaṭṭāt triśūlaṃ dvaitaghasmaram // 34 //
parasparavirodhe tu kāryeṣu pravirohati /
na kathaṃcid upādeyam āsāṃ rūpam idaṃ bhavet // 35 //
bindur vedyasya saṃskāro vimarśaḥ sarga ity asau /
kalāṣoḍaśakākārā śaktir vijayate parā // 36 //
tithayaḥ pratipatpūrvāḥ pañcadaśeti māyayā /
sūryācandramasau svāntaścarantau sthitihetave // 37 //
yathā vimarśavapuṣaḥ sargasyādyāḥ kalāḥ smṛtāḥ /
dvidheyaṃ mātṛkā devī bījayonyātmanā sthitā // 38 //
nityapravṛttaśṛṅgāṭavapurviśvasya janmabhūḥ /
hṛdayaṃ bījam etasyāṃ sāraṃ yat tat paraṃ mahaḥ // 39 //
vaṭabīje yathā vṛkṣas tathātra nihitaṃ jagat /
vicāryamāṇe naivedaṃ kāraṇād atiricyate // 40 //
mṛdādeḥ kalaśādīnāṃ tattvaṃ nānyan nirūpaṇe /
ity āhus tattvavādinyaḥ śrutīnām antimā giraḥ // 41 //
idaṃ sarvaṃ sad evāsīd agre iti viniścayāt /
sattāvācini bīje 'smin bhāti māyā tv idaṃ jagat // 42 //
viluptapratyayākāram etat sa pariśiṣyate /
tato jñānakriyāsāravidyeśvarasadāśivāḥ // 43 //
śaktitriśūle līyante caturdaśakalātmani /
ūrdhvādhaḥ sṛṣṭivapuṣi sarvaṃ līnam ataḥ [param] // 44 //
itthaṃ parasyāṃ saṃvittau sarvaṃ saṃkucitaṃ kramāt /
athavā manasātīte yatra kvāpi nirañjane // 45 //
ṣaṭtriṃśattattvalaharīkalahātītagocare /
viśvātmani mahāmantre svabhāve sā vilīyate // 46 //
kṛtaś cainmathite dhāmni dīpte kenāpi hetunā /
sarvaṃ havir idaṃ juhvan na dāridryeṇa pīḍyate // 47 //
pañcapañcātmakaṃ viśvaṃ pañcaspandavijṛmbhitam /
saṃkocayatparāmarśāt sāmānyaspandakevalam // 48 //
ahami pralayaṃ kurvann idamaḥ pratiyoginaḥ /
parākramaparo bhuṅkte svabhāvam aśivāpaham // 49 //
iti ṣaḍadhikatriṃśadbhedaprasaktajagattanuḥ prasarati mahāśaktyullekhavicitramahāpaṭī /
jayati bahuśaḥ spandākārā parā cid anuttarā vimṛśata janāḥ kiṃ no svabhāvavijṛmbhitam // 50 //
iti śrīmadādyanāthaviracitā anuttaraprakāśapañcāśikā samāptā
seyaṃ kāśmīrabhūpālasaṃśritena prakāśitā /
satā mukundarāmeṇa sanmude 'stu śive 'rpitā // *1 //
