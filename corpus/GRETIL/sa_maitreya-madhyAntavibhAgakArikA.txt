||lakṣaṇaparicchedaḥ prathamaḥ
lakṣaṇaṃ hyāvṛtistattvaṃ pratipakṣasya bhāvanā /
tatrāvasthā phalaprāptiryānānuttaryameva ca // 1.1 //
abhūtaparikalpo 'sti dvayaṃ tatra na vidyate /
śūnyatā vidyate tvatra tasyāmapi sa vidyate // 1.2 //
na śūnyaṃ nāpi cāśūnyaṃ tasmāt sarvaṃ vidhīyate /
sattvādasattvāt sattvācca madhyamā pratipacca sā // 1.3 //
arthasattvātmavijñaptipratibhāsaṃ prajāyate /
vijñānaṃ nāsti cāsyārthastadabhāvāttadapyasat // 1.4 //
abhūtaparikalpatvaṃ siddhamasya bhavatyataḥ /
na tathā sarvathābhāvāt tatkṣayānmuktiriṣyate // 1.5 //
kalpitaḥ paratantraśca pariniṣpanna eva ca /
arthādabhūtakalpācca dvayābhāvācca deśitaḥ // 1.6 //
upalabdhiṃ samāśritya nopalabdhiḥ prajāyate /
nopalabdhiṃ samāśritya nopalabdhiḥ prajāyate // 1.7 //
upalabdhestataḥ siddhā nopalabdhisvabhāvatā /
tasmācca samatā jñeyā nopalambhopalambhayoḥ // 1.8 //
abhūtaparikalpaśca cittacaittāstridhātukāḥ /
tatrārthadṛṣṭirvijñānaṃ tadviśeṣe tu caitasāḥ // 1.9 //
evaṃ pratyayavijñānaṃ dvitīyaṃ caupabhogikam /
upabhogaparicchedaprerakāstatra caitasāḥ // 1.10 //
chādanādropaṇāccaiva nayanātsaṃparigrahāt /
pūraṇāt triparicchedādupabhogācca karṣaṇāt // 1.11 //
nibandhanādābhimakhyād duḥkhanāt kliśyate jagat /
tredhā dvedhā ca saṃkleśaḥ saptadhābhūtakalpanāt // 1.12 //
lakṣaṇaṃ cātha paryāyastadartho bheda eva ca /
sādhanañceti vijñeyaṃ śūnyatāyāḥ samāsataḥ // 1.13 //
dvayābhāvo hyabhāvasya bhāvaḥ śūnyasya lakṣaṇam /
na bhāvo nāpi cābhāvao na pṛthaktvaikalakṣaṇam // 1.14 //
tathatā bhūtakoṭiścānimittaṃ paramārthatā /
dharmadhātuśca paryāyāḥ śūnyatāyāḥ samāsataḥ // 1.15 //
ananyathāviparyāsatannirodhāryagocaraiḥ /
hetutvāccāryadharmaṇāṃ paryāyārtho yathākramam // 1.16 //
saṃkliṣṭā ca viśuddhā ca samalā nirmalā ca sā /
abdhātukanakākāśaśuddhivacchuddhiriṣyate // 1.17 //
bhoktṛbhojanataddhehapratiṣṭhāvastuśūnyatā /
tacca yena yathā dṛṣṭaṃ yadarthaṃ tasya śūnyatā // 1.18 //
śubhadvayasya prāptyartha sadā sattvahitāya ca /
saṃsārātyajanārthaṃ ca kuśalasyākṣayāya ca // 1.19 //
gotrasya ca viśuddhayarthaṃ lakṣaṇavyañjanāptaye /
śuddhaye bu[ddha]dharmāṇāṃ bodhisattvaḥ prapadyate // 1.20 //
pudgalasyātha dharmāṇāmabhāvaḥ śūnyatātra hi /
tadabhāvasya sadbhāvastasmin sā śūnyatāparā // 1.21 //
saṃkliṣṭā cedbhavennāsau muktāssyuḥ sarvadehinaḥ /
viśuddhā cedbhavennāsau vyāyāmo niṣphalo bhavet // 1.22 //
na kliṣṭā nāpi va 'ākliṣṭā śuddhāśuddhā na caiva sā /
prabhāsvaratvāccittasya kleśasyāgantukatvataḥ // 1.23 //
||iti lakṣaṇaparicchedaḥ prathamaḥ //
||āvaraṇaparicchedo dvitīyaḥ
vyāpi prādeśikodriktasamādānavivarjanam /
dvayāvaraṇamākhyātaṃ navadhā kleśalakṣaṇam // 2.1 //
saṃyojanānyāvaraṇamudvegasamupekṣayoḥ /
tattvadṛṣṭeśca satkāyadṛṣṭestadvastuno 'pi ca // 2.2 //
nirodhamārgaratneṣu lābhasatkāra eva ca /
saṅkleśasya parijñāne śubhādau daśadhāparam // 2.3 //
aprayogo 'nāyatane 'yogavihitaśca yaḥ /
notpattiramanaskāraḥ sambhārasyāprapūrṇatā // 2.4 //
gotramitrasya vaidhuryaṃ cittasya parikheditā /
pratipatteśca vaidhuryaṃ kuduṣṭajanavāsatā // 2.5 //
dauṣṭhulyamavaśiṣṭatvaṃ trayāt prajñāvipakvatā /
prakṛtyā caiva dauṣṭhulyaṃ kausīdyaṃ ca pramāditā // 2.6 //
saktirbhave ca bhoge ca līnacittatvameva ca /
aśraddhānadhimuktiśca yathārutavicāraṇā // 2.7 //
saddharme 'gauravaṃ lābhe gurutākṛpatā tathā /
śrutavyasanamalpatvaṃ samādhyaparikarmitā // 2.8 //
śubhaṃ bodhiḥ samādānaṃ dhīmattvābhrāntyanāvṛtī /
nṛtyatrāso 'matsaritvaṃ vaśitvañca śubhādayaḥ // 2.9 //
trīṇi trīṇi ca eteṣāṃ jñeyānyāvaraṇāni hi /
pakṣyapāramitābhūmiṣvanyadāvaraṇaṃ punaḥ // 2.10 //
vastvakauśalakausīdyaṃ samādherdvayahīnatā /
aropaṇātha daurbalyaṃ dṛṣṭidauṣṭhulyaduṣṭatā // 2.11 //
aiśvaryasyātha sugateḥ sattvātyāgasya cāvṛtiḥ /
hānivṛddhyośca doṣāṇāṃ guṇānāmavatāraṇe // 2.12 //
vimocane 'kṣayatve ca nairantarye śubhasya ca /
niyatīkaraṇe dharmasambhogaparipācane // 2.13 //
sarvatragārthe agrārthe niṣyandāgrārtha eva ca /
niṣparigrahatārtha ca santānābheda eva ca // 2.14 //
nissaṅkleśaviśuddhyarthe 'nānātvārtha eva ca /
ahīnānadhikārthe ca caturdhāvaśitāśraye // 2.15 //
dharmadhātāvavidyeyamakliṣṭā daśadhāvṛtiḥ /
daśabhūmivipakṣeṇa pratipakṣāstu bhūmayaḥ // 2.16 //
kleśāvaraṇamākhyātaṃ jñeyāvaraṇameva ca /
sarvāṇyāvaraṇānīha yatkṣayānmuktiriṣyate // 2.17 //
||ityāvaraṇaparicchedo dvitīyaḥ //
||tattvaparicchedastṛtīyaḥ
mūlalakṣaṇatattvamaviparyāsalakṣaṇam /
phalahetumayaṃ tattvaṃ sūkṣmaudārikameva ca // 3.1 //
prasiddhaṃ śuddhiviṣayaṃ saṅgrāhyaṃ bhedalakṣaṇam /
kauśalyatattvaṃ daśadhā ātmadṛṣṭivipakṣataḥ // 3.2 //
svabhāvastrividho 'sacca nityaṃ saccāpyatattvataḥ /
sadasattattvataśceti svabhāvatrayamiṣyate // 3.3 //
samāropāpavādasya dharmapudgalayoriha /
grāhyagrāhakayoścāpi bhāvābhāve ca darśanam // 3.4 //
yajjñānānna pravarteta taddhi tattvasya lakṣaṇam /
asadartho hyanityārtha utpādadavyayalakṣaṇaḥ // 3.5 //
samalāmalabhāvena mūlatattve yathākramam /
duḥkhamādānalakṣmākhyaṃ sambandhenāparaṃ matam // 3.6 //
abhāvaścāpyatadbhāvaḥ prakṛtiḥ śūnyatā matā /
alakṣaṇañca nairātmyaṃ tadvilakṣaṇameva ca // 3.7 //
svalakṣaṇañca nirdiṣṭaṃ duḥkhasatyamato matam /
vāsanātha samutthānamavisaṃyoga eva ca // 3.8 //
svabhāvadvayanotpattirmalaśāntidvayaṃ matam /
parijñāyāṃ prahāṇe ca prāptisākṣātkṛtāvayam // 3.9 //
mārgasatyaṃ samākhyātaṃ prajñaptipratipattitaḥ /
tathodbhāvanayaudāraṃ paramārthantu ekataḥ // 3.10 //
arthaprāptiprapattyā hi paramārthastridhā mataḥ /
nirvikārāviparyāsapariniṣpattito dvayam // 3.11 //
lokaprasiddhamekasmāt trayād yuktiprasiddhakam /
viśuddhagocaraṃ dvedhā ekasmādeva kīrtitam // 3.12 //
nimittasya vikalpasya nāmnaśca dvayasaṅgrahaḥ /
samyagjñānasatattvasya ekenaiva ca saṅgrahaḥ // 3.13 //
pravṛttitattvaṃ dvividhaṃ sanniveśakupannatā /
ekaṃ lakṣaṇavijñaptiśuddhisamyakprapannatā // 3.14 //
ekahetutvabhoktṛtvakartṛtvavaśavartane /
ādhipatyārthanityatve kleśaśuddhyāśraye 'pi ca // 3.15 //
yogitvāmuktamuktatve ātmadarśanameṣu hi /
parikalpavikalpārthadharmatārthena teṣu te // 3.16 //
anekatvābhisaṅkṣepaparicchedārtha āditaḥ /
grāhakagrāhyatadgrāhabījārthaścāparo mataḥ // 3.17 //
veditārthaparicchedabhogāyadvārato 'param /
punarhetuphalāyāsānāropānapavādataḥ // 3.18 //
aniṣṭeṣṭaviśuddhīnāṃ samotpattyādhipatyayoḥ /
samprāptisamudācārapāratantryārthato 'param // 3.19 //
grahaṇasthānasandhānabhogaśuddhidvayārthataḥ /
phalahetūpayogārthanopayogāttathāparam // 3.20 //
vedanāsanimittārthatannimittaprapattitaḥ /
tacchamapratipakṣārthayogādaparamiṣyate // 3.21 //
guṇadoṣāvikalpena jñānena parataḥ svayam /
niryāṇādaparaṃ jñeyaṃ saprajñaptisahetukāt // 3.22 //
nimittāt praśamāt sārthāt paścimaṃ samudāhṛtam //
||iti tattvaparicchedastṛtīyaḥ //
||pratipakṣabhāvanāvasthāphalaparicchedaś caturthaḥ
dauṣṭhulyāt tarṣahetutvād vastutvādavimohataḥ /
catussatyāvatārāya smṛtyupasthānabhāvanā // 4.1 //
parijñāte vipakṣe ca pratipakṣe ca sarvathā /
tadapāyāya vīryaṃ hi caturdhā sampravartate // 4.2 //
karmaṇyatā sthitestatra sarvārthānāṃ samṛddhaye /
pañcadoṣaprahāṇāṣṭasaṃskārāsevanānvayā // 4.3 //
kausīdyamavavādasya sammoṣo laya uddhavaḥ /
asaṃskāro 'tha saṃskāraḥ pañca doṣā ime matāḥ // 4.4 //
āśrayo 'thāśritastasya nimittaṃ phalameva ca /
ālambane 'sammoṣo layauddhatyānubuddhyanā // 4.5 //
tadapāyābhisaṃskāraḥ śāntau praśaṭhavāhitā /
ropite mokṣabhāgīye cchandayogādhipatyataḥ // 4.6 //
ālambane 'sammoṣāvisāravicayasya ca /
vipakṣasya hi saṃlekhāt pūrvasya phalamuttaram // 4.7 //
dvau dvau nirvedhabhāgīyāvindriyāṇi balāni ca /
āśrayāṅgaṃ svabhāvāṅgaṃ niryāṇāṅgaṃ tṛtīyakam // 4.8 //
caturthamanuśaṃsāṅgaṃ niḥkleśāṅgaṃ tridhā matam /
nidānenāśrayeṇeha svabhāvena ca deśitam // 4.9 //
paricchedo 'tha samprāptiḥ parasambhāvanā tridhā /
vipakṣapratipakṣaśca mārgasyāṅgaṃ tadaṣṭadhā // 4.10 //
dṛṣṭau śīle 'tha saṃlekhe paravijñaptiriṣyate /
kleśopakleśavaibhutvavipakṣapratipakṣatā // 4.11 //
anukūlā viparyastā sānubandhā viparyayā /
aviparyastaviparyāsānanubandha ca bhāvanā // 4.12 //
ālambanamanaskāraprāptitastad viśiṣṭatā /
hetvavasthāvatārākhyā prayogaphalasaṃjñitā // 4.13 //
kāryākāryaviśiṣṭā ca uttarānuttarā ca sā /
adhimuktau praveśe ca niryāṇe vyākṛtāvapi // 4.14 //
kathikatve 'bhiṣeke ca samprāptāvanuśaṃsane /
kṛtyānuṣṭhāna uddiṣṭā dharmādhātau tridhā punaḥ // 4.15 //
aśuddhāśuddhaśuddhā ca viśuddhā ca yathārthataḥ /
pudgalānāṃ vyavasthānaṃ yathāyogamato matam // 4.16 //
bhājanatvaṃ vipākākhyaṃ balantasyādhipatyataḥ /
rucirvṛddhirviśuddhiśca phalametad yathākramam // 4.17 //
uttarottaramādyañca tadabhyāsat samāptitaḥ /
ānukūlyād vipakṣācca visaṃyogād viśeṣataḥ // 4.18 //
uttarānuttaratvācca phalamanyat samāsataḥ /
||iti pratipakṣabhāvanādiparicchedaścaturthaḥ //
||yānānuttaryaparicchedaḥ pañcamaḥ
ānuttaryaṃprapattau hi punarālambane matam /
samudāgama uddiṣṭaṃ pratipattistu ṣaḍvidhā // 5.1 //
paramātha manaskāre anudharme 'ntavarjane /
viśiṣṭā cāviśiṣṭā ca paramā dvādaśātmikā // 5.2 //
audāryamāyatatvañca adhikaro 'kṣayātmatā /
nairantaryamakṛcchratvaṃ vittatvañca parigrahaḥ // 5.3 //
ārambhaprāptiniṣyandaniṣpattiḥ paramā matā /
tataśca paramārthena daśa pāramitā matāḥ // 5.4 //
dānaṃ śīlaṃ kṣamā vīryaṃ dhyānaṃ prajñā upāyatā /
praṇidhānaṃ balaṃ jñānametāḥ pāramitā daśa // 5.5 //
anugraho 'vighātaśca karma tasya ca marṣaṇam /
guṇavṛddhiśca sāmarthyamavatāravimocane // 5.6 //
akṣayatvaṃ sadā vṛttirniyataṃ bhogapācane /
yathāprajñaptito dharmamahāyānamanaskriyā // 5.7 //
bodhisattvasya satataṃ prajñayā triprakārayā /
dhātupuṣṭayaipraveśāya cārthasiddhyai bhavatyasau // 5.8 //
saṃyuktā dharmacaritaiḥ sā jñeyā daśabhiḥ punaḥ /
lekhanā pūjanā dānaṃ śravaṇaṃ vācanod grahaḥ // 5.9 //
prakāśanātha svādhyāyaścintanā bhāvanā ca tat /
ameyapuṇyaskandhaṃ hi caritaṃ tad daśātmakam // 5.10 //
viśeṣādakṣayatvācca parānugrahato 'śamāt /
avikṣiptāviparyāsapraṇatā cānudhārmikī // 5.11 //
vyutthānaṃ viṣaye sārastathāsvādalayoddhavaḥ /
sambhāvanābhisandhiśca manaskāre 'pyahaṃkṛtiḥ // 5.12 //
hīnacittañca vikṣepaḥ parijñeyo hi dhīmatā /
vyañjanārthamanaskāre 'visāre lakṣaṇadvaye // 5.13 //
aśuddhaśuddhāvāgantukatve 'trāsitānunnatau /
saṃyogāt saṃstavāccaiva viyogādapyasaṃstavāt // 5.14 //
arthasattvamasattvañca vyañjane so 'viparyayaḥ /
dvayena pratibhāsatvaṃ tathā cāvidyamānatā // 5.15 //
arthe sa cāviparyāsaḥ sadasattvena varjitaḥ /
tajjalpabhāvito jalpamanaskārastadāśrayaḥ // 5.16 //
manaskāre 'viparyāso dvayaprakhyānakāraṇe /
māyādivadasattvañca sattvañcārthasya tanmatam // 5.17 //
so 'visāre 'viparyāso bhāvābhāvāvisārataḥ /
sarvasya nāmamātratvaṃ sarvakalpāpravṛttaye // 5.18 //
svalakṣaṇe 'viparyāsaḥ paramārthe svalakṣaṇe /
dharmadhātuvinirmukto yasmād dharmo na vidyate // 5.19 //
sāmānyalakṣaṇantasmāt sa ca tatrāviparyayaḥ /
viparyastamanaskārāvihāniparihāṇitaḥ // 5.20 //
tadaśuddhirviśuddhiśca sa ca tatrāviparyayaḥ /
dharmadhātorviśuddhatvāt prakṛtyā vyomavatpunaḥ // 5.21 //
dvayasyāgantukatvaṃ hi sa ca tatrāviparyayaḥ /
saṃkleśaśca viśuddhiśca dharmapudgalayorna hi // 5.22 //
asattvāt trāsatāmānau nātaḥ so 'trāviparyayaḥ /
pṛthaktvaikatvamantaśca tīrthyaśrāvakayorapi // 5.23 //
samāropāpavādānto dvidhā pudgaladharmayoḥ /
vipakṣapratipakṣāntaḥ śāśvatocchedasaṃjñitaḥ // 5.24 //
grāhyagrāhakasaṃkleśavyavadāne dvidhā tridhā /
vikalpadvayatāntaśca sa ca saptavidho mataḥ // 5.25 //
bhāvābhāve praśāmye 'tha śamane trāsyatadbhaye /
grāhyagrāhe 'tha samyaktvamithyātve vyāpṛtau na ca // 5.26 //
ajanmasamakālatve sa vikalpadvayāntatā /
viśiṣṭā cāviśiṣṭā ca jñeyā daśasu bhūmiṣu // 5.27 //
vyavasthānantato dhātuḥ sādhyasādhanadhāraṇā /
avadhārapradhārā ca prativedhaḥ pratānatā // 5.28 //
pragamaḥ praśaṭhatvañca prakarṣālambanammatam /
avaikalyāpratikṣepo 'vikṣepaśca prapūraṇā // 5.29 //
samutpādo nirūḍhiśca karmaṇyatvāpratiṣṭhitā /
nirāvaraṇatā tasyāpraśrabdhisamudāgamaḥ // 5.30 //
śāstraṃ madhyavibhāgaṃ hi gūḍhasārārthameva ca /
mahārthañcaiva sarvārthaṃ sarvānarthapraṇodanam // 5.31 //
||iti yānānuttarya paricchedaḥ pañcamaḥ //
||samāptā madhyāntavibhāgakārikāḥ
