ācāryadiṅnāgakṛtā
namaḥ sarvabuddhabodhisattvebhyaḥ
yadyapīndriyavijñaptergrāhyāṃśaḥ kāraṇaṃ bhavet /
atadābhatayā tasyā nākṣavadviṣayaḥ sa tu // 1 //
yadābhāsā na tasmāt sā dravyābhāvāt dvicandravat /
evaṃ bāhyadvayañcaiva na yuktaṃ matigocaraḥ // 2 //
sādhanaṃ sañcitākāramicchanti kila kecana /
aṇvākāro na vijñapterarthaḥ kaṭhinatādivat // 3 //
bhaveddhaṭaśarāvādestathā sati samā matiḥ /
ākārabhedādbhedaścet, nāsti tu dravyasatyaṇau // 4 //
pramāṇabhedābhāvāt saḥ, adravye 'sti tataḥ sa hi /
aṇūnāṃ parihāre hi tadābhajñānaviplavāt // 5 //
yadantarjñeyarūpaṃ tu vahirvadavabhāsate /
so 'rtho vijñānarūpatvāttatpratyayatayāpi ca // 6 //
ekāṃśaḥ pratyayo 'vītāt śaktyarpaṇātkrameṇa[vā] /
sahakārivaśādyaddhi śaktirūpaṃ [tat] indriyam // 7 //
sā cāviruddhā vijñapterevaṃ viṣayarūpakam /
pravartate 'nādikālaṃ śaktiścānyonyahetuke // 8 //
ityācāryadiṅnāgakṛtā ālambanaparīkṣāprakaraṇakārikā samāptā
