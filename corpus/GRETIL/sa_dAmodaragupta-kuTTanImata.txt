#Text:Kuṭṭanīmata
#Data entry:Csaba Dezso and Dominic Goodall
#Edition:Dāmodaraguptaviracitaṃ Kuṭṭanīmatam : The Bawd's Counsel, Being an eighth-century verse novel in Sanskrit by Dāmodaragupta Newly edited and translated into English by Csaba Dezso & Dominic Goodall. Groningen: Egbert Forsten, 2012. ISBN 978 90 6980 158 2.

sa jayati saṃkalpabhavo ratimukhaśatapatracumbanabhramaraḥ /
yasyānuraktalalanānayanāntavilokitaṃ vasatiḥ // 1 //
avadhīrya doṣanicayaṃ guṇaleśe saṃniveśya matimāryāḥ /
kuṭṭanyā matametaddāmodaraguptaviracitaṃ śṛṇuta // 2 //
asti khalu nikhilabhūtalabhūṣaṇabhūtā vibhūtiguṇayuktā /
yuktābhiyuktajanatā nagarī vārāṇasī nāma // 3 //
anubhavatāmapi yasyāmupabhogān kāmataḥ śarīravatām /
śaśadharakhaṇḍavibhūṣitadehalayaḥ kila na duṣprāpaḥ // 4 //
candravibhūṣitadehā bhūtiratāḥ sadbhujaṅgaparivārāḥ /
vārastriyo 'pi yasyāṃ paśupatitanutulyatāṃ yātāḥ // 5 //
atituṅgasuraniketanaśikharasamutkṣiptapavanacalitābhiḥ /
mañjaritamiva virājati yatra nabho vaijayantībhiḥ // 6 //
aviratasaṃcaradabalācaraṇatalālaktakadravāruṇitam /
sthalakamalavatīṃ lakṣmīṃ bibharti vasudhātalaṃ yatra // 7 //
yatra ca ramaṇībhūṣaṇaravabadhiritasakaladiṅnabhobhāge /
śiṣyāṇāṃ nācāryairavadyamavadhāryate paṭhatām // 8 //
vindhyadharādharabhūriva yā rājati mattavāraṇopetā /
bahulaniśīthavatīva projjvaladhiṣṇyopaśobhitā yā ca // 9 //
yatigaṇaguṇasamupetā yā nityaṃ chandasāmiva pracitiḥ /
vanapaṅktiriva sasālā turuṣkaseneva bahalagandharvā // 10 //
tārāgaṇo 'kulīnaḥ priyadoṣā yatra kauśikāḥ satatam /
gadye vṛttacyavanaṃ paragṛharodhastathākṣeṣu // 11 //
śūlabhṛto vyālāsthā padavediṣu yatra dhātuvāditvam /
surateṣvabalākramaṇaṃ dānacchedo madacyutau kariṇām // 12 //
tīvrakaratvaṃ bhānoraviveko yatra mitrahṛdayānām /
yogiṣu daṇḍagrahaṇaṃ saṃdhicchedaḥ pragṛhyeṣu // 13 //
chandaḥprastāravidhau guravo yasyāmanārjavasthitayaḥ /
vīṇāyāṃ parivādo dvijanilayeṣvaprasannatvam // 14 //
anurūpavṛttaghaṭanā satkavikṛtarūpakeṣu loke ca /
ramaṇīvacane yasyāṃ mādhuryaṃ kāvyabandhe ca // 15 //
yasyāmupavanavīthyāṃ tamālapattrāṇi yuvativadane ca /
nakharaprahārakaṇitaṃ tantrīvādye ca suratakalahe ca // 16 //
nandanavanābhirāmā vibudhavatī nākavāhinījuṣṭā /
amarāvatīva yānyā viśvasṛjā nirmitā jagati // 17 //
tasyāṃ khagapatitanuriva vilāsināṃ hṛdayaśokasaṃjananī /
ākṛṣṭeśvarahṛdayā prāleyanagādhirājatanayeva // 18 //
saṃsaktabhoginetrā mandaradharaṇībhṛto yathā mūrtiḥ /
uparigatā śūlānāmandhāsuragātralekheva // 19 //
samuvāsa vārarāmā mānasavasateḥ śarīriṇī śaktiḥ /
niḥśeṣaveśayoṣidvibhūṣaṇaṃ mālatī nāma // 20 //
peśalavacasāṃ vasatirlīlānāmālayaḥ sthitiḥ premnaḥ /
bhūmiḥ parihāsānāmāvasatho vakrakathitānām // 21 //
sā śuśrāva kadā ciddhavalālayapṛṣṭhadeśamadhirūḍhā /
kenāpi gīyamānāṃ prasaṅgapatitāmimāmāryām // 22 //
yauvanasaundaryamadaṃ dūreṇāpāsya vāravanitābhiḥ /
yatnena veditavyāḥ kāmukahṛdayārjanopāyāḥ // 23 //
śrutvātha vipulajaghanā manasīdaṃ mālatī cakāra ciram /
atisāmpratamupadiṣṭaṃ suhṛdevānena sādhunā paṭhatā // 24 //
tadgatvā pṛcchāmo vikarālāṃ kalitasakalasaṃsārām /
yasyāḥ kāmijanaugho divāniśaṃ dvāradeśamadhyāste // 25 //
iti manasi sā niveśya drutataramavatīrya veśmanaḥ śikharāt /
vikarālābhavanavaraṃ parijanaparivāritā prayayau // 26 //
atha viralonnatadaśanāṃ nimnahanuṃ sthūlacipiṭanāsāgrām /
ulbaṇacūcukalakṣitaśuṣkakucasthānaśithilakṛttitanum // 27 //
gambhīrāraktadṛśaṃ nirbhūṣaṇalambakarṇapālīṃ ca /
katipayapāṇḍuracikurāṃ prakaṭaśirāsaṃtatāyatagrīvām // 28 //
sitadhautavasanayugalāṃ vividhauṣadhimaṇisanāthagalasūtrām /
tanvīmaṅgulimūle tapanīyamayīṃ ca vālikāṃ dadhatīm // 29 //
gaṇikāgaṇaparikaritāṃ kāmijanopāyanaprasaktadṛśam /
āsandyāmāsīnāṃ vilokayāmāsa vikarālām // 30 //
avalokya sā vidhāya kṣitimaṇḍalalīnamaulinā praṇatim /
paripṛṣṭakuśalavārtā samanujñātāsanaṃ bheje // 31 //
atha viracitahastapuṭā sapraśrayamāsanaṃ samutsṛjya /
idamūce vikarālāmavasaramāsādya mālatī vacanam // 32 //
vidadhāsi harimakaustubhamahariṃ ravimagajanāthamamarendram /
adraviṇaṃ draviṇapatiṃ niyataṃ matigocare patitam // 33 //
ayameva buddhivibhavaṃ hṛtavibhavaste paṭaccarāvaraṇaḥ /
kāmukalokaḥ kathayati sattrāgāreṣu bhuñjānaḥ // 34 //
upasaṃhṛtānyakarmā dhanavarmā narmadāṅghriyugalasya /
sakalasamarpitasampadyadupetaḥ pādapīṭhatvam // 35 //
yadupanato nayadattaḥ sāgaradattasya madhyamaḥ putraḥ /
prīṇayati madanasenāṃ vidhāya pitṛmandiraṃ riktam // 36 //
yallīlārpitacaraṇau mañjaryā bhaṭṭaputranarasiṃhaḥ /
paritoṣaṃ vrajati paraṃ mṛdu mṛdnan pāṇiyugalena // 37 //
yanniḥśeṣitavibhavo dīkṣitabhavadevaputraśubhadevaḥ /
nirbhartsito 'pi nojjhati kesarasenāgṛhadvāram // 38 //
anyā api kāmijanaṃ sādhāraṇayoṣito yadākramya /
vidadhati karpaṭaśeṣaṃ vilasitametattavopadeśānām // 39 //
hīnānvayajanmāno guṇahīnā rogiṇo nirākṛtayaḥ /
upasevitā mayāpi prakaṭīkṛtarāgasauṣṭhavaṃ puruṣāḥ // 40 //
mātaḥ kiṃ vidadhāmo hatadhāturvāmatābhiyogena /
nāsādayāma iṣṭaṃ nijatanupaṇyaprasārakeṇāpi // 41 //
tatkuru mātaranugrahamabhidhatsva mamāpi dehino bhogyān /
teṣāṃ ca veśaceṣṭitamanasijaśarajālapātanopāyān // 42 //
iti giramudīrayantīṃ sapremāmṛśya pāṇinā pṛṣṭhe /
ruciravaco vikarālā rucirākṛtimālatīmūce // 43 //
ayameva dahyamānasmaranirgatadhūmavartikākāraḥ /
cikurabharastava sundari kāmijanaṃ kiṅkarīkurute // 44 //
ayameva te kṛśodari mandollasitabhruvibhramādhāraḥ /
adharīkaroti dhīrān madhurasmitasubhagavīkṣitaviśeṣaḥ // 45 //
iyameva vadanakāntī ratikāntākūtamatitarāṃ kurute /
śrutipathamapyupayātā niyataṃ tava kāmināṃ manasi // 46 //
iyameva daśanapaṅktī rucirācirakāntikāntisamakāntiḥ /
utpādayati nitāntaṃ tava manmathadāhavedanāṃ puṃsām // 47 //
idameva samullapitaṃ līlāvati vijitaparabhṛtadhvānam /
tava niḥśeṣabhujaṅgavyākarṣaṇasiddhamantravaccitram // 48 //
idameva makaraketananiketanaṃ stanayugaṃ tavābhogi /
bhogavati bhogasādhanamaparopāyagraho vyarthaḥ // 49 //
idameva bāhuyugalaṃ mṛṇālatanusundaraṃ tavābhogi /
kasya na janayati madanaṃ kanakāṅgadabhūṣaṇaṃ sutanu // 50 //
ayameva madhyadeśaḥ kandarpādeśakaraṇacaturaste /
prakṛśo 'pi śarīrabhṛto daśamīṃ prāpayati manmathāvasthām // 51 //
iyameva romarājiḥ saṅkalpajacāpayaṣṭiguṇaśobhām /
dadhatī vidadhāti tava smarasāyakaśalyaviklavān yūnaḥ // 52 //
idameva pṛthulajaghanaṃ kaladhautaśilātalābhiramaṇīyam /
tava taruṇavaśīkaraṇaṃ yatisaṃyatināśakāri karabhoru // 53 //
idameva tavoruyugaṃ rambhāgarbhopamaṃ manohāri /
vada sundari nābhimataṃ madanajvaratāpaśāntaye kasya // 54 //
yauvanakalpataroste kanakalatāvibhramaṃ suvṛttam idam /
jaṅghāyugalaṃ necchati kāmaphalaprāptaye ka iva // 55 //
nirjitadāḍimarāgaṃ vijitasthalakamalinīvilāsamidam /
tava caraṇasarojayugaṃ kasya na mānasamalaṅkurute // 56 //
hrepayati vāraṇendraṃ haṃsaṃ hasati prayātamidameva /
tava līlāvati lalitaṃ yūnāṃ hṛdayāni mathnāti // 57 //
tadapi yadi te kutūhalamavadhānaṃ saṃvidhāya tanumadhye /
ākarṇaya kathayāmi svabuddhivibhavānusāreṇa // 58 //
svīkuru tāvatprathamaṃ nṛpasevakabhaṭṭasūnumatiyatnāt /
svādhīnāmativipulāṃ yadi sampadamīhase sutanu // 59 //
pratyāsannagrāme svayaṃ prabhuḥ pitari nityakaṭakasthe /
bhaṭṭasutaścintāmaṇirākṛṣṭo bhavati putri niyamena // 60 //
śṛṇu tasya cāruhāsini veṣagrahaṇaṃ ca ceṣṭitaṃ caiva /
nipatati ca yathā tūrṇaṃ priyasurabhikusumaśarāsanaprasare // 61 //
sthūlasthāpitacūlakapañcāṅgulamātrakeśavinyāsaḥ /
lambaśravaṇaniveśitakarapatrikaghaṭitadaśanapaṅktiśca // 62 //
karaśākhāśritamudrikacāmīkarakaṇṭhasūtrikābharaṇaḥ /
parimṛṣṭagātrakuṅkumakiñcitpiñjaritavasanasaṃvītaḥ // 63 //
pravilambikusumadāmakagalamaṇḍanajātarūpakṛtaśobhaḥ /
antarniviṣṭasikthitatauruṣkikakhuśśikādicaraṇatraḥ // 64 //
nānāvarṇaviveṣṭitabahaladaśāpāśabaddhatatakeśam /
ekasmindalavīṭakamaparasminsīsapatrakaṃ karṇe // 65 //
uccaṇḍakanakagarbhitakuṅkumapiñjaritavastiparidhānam /
sthūlatarakācavartakamālāṃ ca gale dadhānena // 66 //
vṛścikarañjitakararuhakaramūlanibaddhaśaṅkhacakreṇa /
prathamavayastvaṃ bhajatā tāmbūlakaraṅkavāhinānugataḥ // 67 //
śreṣṭhivaṇigviṭakitavapradhānaraṅgasya sumahato madhye /
śūlāpālasthāpitakatipayavadhrorupīṭhikāsīnaḥ // 68 //
utsaṅgārpitakhaḍgairayathātathabhāṣibhirmahauddhatyam /
bibhrāṇairanujīvibhiradhiṣṭhitaḥ pañcaṣaiḥ puruṣaiḥ // 69 //
caturatarasevakārpitapṛṣṭhaparikṣiptapūrvadehāṃśaḥ /
antarvidhṛtāvelaprocchūnakapolakalitakaraparṇaḥ // 70 //
anapekṣitaprasaṅgaḥ punaḥ punaḥ paṭhati sonnatabhrūkaḥ /
gāthāślokaprāyaṃ bhāvitacetā yathātathādhītam // 71 //
vismayalolitamauliḥ pārśvagatāntāḍayanrasāvegāt /
hākaṣṭasādhuvādairantarayati parasubhāṣitaśravaṇam // 72 //
idamukto rahasi ruṣā tātena nṛpo nṛpeṇa tāto 'pi /
iti piturāviṣkurute mahībhṛtaḥ praṇayaviśvāsau // 73 //
patracchedamajānan jānanvā kauśalaṃ kalāviṣaye /
prakaṭayati janasamāje bibhrāṇaḥ patrakartarīṃ satatam // 74 //
brahmoktanāṭyaśāstre gīte murajādivādane caiva /
abhibhavati nāradādīn prāvīṇyaṃ bhaṭṭaputrasya // 75 //
vasunandacitradaṇḍakamuktāyudhakhaḍgadhenubandheṣu /
vrajati purato 'sya niyataṃ bhārgavatāṃ paraśurāmo 'pi // 76 //
vātsyāyanamayamabudhaṃ bāhyaṃ dūreṇa dattakācāryam /
gaṇayati manmathatantre paśutulyaṃ rājaputraṃ ca // 77 //
yaḥ prārthito 'pi yatnātkavacaṃ rādhāsuto dadāti sma /
avicintitavasuvarṣastyāgaguṇaṃ hasati tasyāyam // 78 //
prapalāyanaikahṛdaye yo vikramamātanoti hariṇe 'pi /
siṃhasya tasya śauryaṃ trapākaraṃ bhaṭṭaputrasya // 79 //
ākheṭake 'pi kautukamastyeva jayaśca cañcale lakṣye /
bhaṭṭabhayena na khelati bhaṭṭasutaḥ kiṃ tvatiprakaṭam // 80 //
iti nijasevakanigaditarāmaṇikāvacanajanitaparituṣṭyā /
antarmudito brūte māmeṣa khalīkarotīti // 81 //
katamatkatamallagnaṃ prasthānaṃ kā ca nartakī bhadrā /
ṣiṅgaṭake kā nṛtyati kohalabharatoditakriyayā // 82 //
kīdṛk tvaṃ layamārge dhavalukaracite ca bhāṇake kīdṛk /
prekṣaṇakādāvevaṃ pṛcchati nṛttopadeśakaṃ yatnāt // 83 //
sumanomālāṃ kaṇṭhātsādaracetā dadāti nartakyai /
apanīya satāmbūlakamanavasare sādhuvādaṃ ca // 84 //
bhujapatanagātrasaṃsthitilālityodvahanapārśvavalitāni /
anayaiva nirmitāni sthānakaśuddhiśca cāturasryaṃ ca // 85 //
pravibhaktairbhāvarasairabhinayabhaṅgyā parikramaiścitraiḥ /
rambhāmapyatiśete kimutetaramartyanartakīlokam // 86 //
ityapasārakaviratāvaviratamutsnāyukaṇṭhamatyuccaiḥ /
varṇayati bhāvitātmā lakṣitapadamātrayā pātram // 87 //
prāyeṇa bhaṭṭatanayo bhavatīdṛśaveṣaceṣṭito vatse /
taṃ madanavāgurāntaḥ pātayasi yathā tathā brūmaḥ // 88 //
caturā prāgalbhyavatī paracittajñānakauśalopetā /
yojyā tasmindūtī vakroktivibhūṣitā prayatnena // 89 //
sa upetya tayāvasare tāmbūlaṃ sumanasaśca dattvettham /
abhidhātavyaḥ sundari makaradhvajadīpakairvacanaiḥ // 90 //
janmasahasropacitaiḥ puṇyacayairadya phalitamasmākam /
yattvaṃ nayanānandana nayanāvasaraṃ sameto 'si // 91 //
cāṭukramamanurāgaṃ praṇayaruṣau virahajanitaśokārtim /
prakaṭayati vāraramaṇī naṭīva śikṣābhiyogena // 92 //
pravayasi yauvanaśālini hīnakule satkulaprasūte ca /
rogavati dṛḍhaśarīre samacittā yoginaśca gaṇikāśca // 93 //
upacaritāpyatimātraṃ prakaṭavadhūḥ kṣīṇasampadaḥ puṃsaḥ /
pātayati dṛśaṃ vrajataḥ spṛhayā paridhānamātre 'pi // 94 //
itthaṃ dṛḍhataravāsitamanasāṃ puṃsāṃ tvasāmprataṃ purataḥ /
veśavilāsavatīnāmaśarīraśaravyathākathanam // 95 //
kevalamagaṇitalāghavadūraparityaktadhīratābharaṇā /
mukharayati māṃ durāśādagdhasakhī tena kathayāmi // 96 //
hṛdayamadhiṣṭhitamādau mālatyāḥ kusumacāpabāṇena /
caramaṃ ramaṇīvallabha locanaviṣayaṃ tvayā bhajatā // 97 //
kṣaṇamutkaṇṭakitāṅgī kṣaṇamulbaṇadāhavedanāvasthā /
kṣaṇamupajātākampā svedārdravapuḥ kṣaṇaṃ bhavati // 98 //
muhuravibhāvitahāsyā muhurujjhitadhīrabhāvamatyuccaiḥ /
roditi gāyati ca punaḥ punaśca maunāvalambinī bhavati // 99 //
patati muhuḥ paryaṅke muhuraṅke parijanasya muhuravanau /
kisalayakalpitatalpe muhurambhasi muhuranaṅgasaṃtaptā // 100 //
mahiṣīva paṅkadigdhā haṃsīva mṛṇālavalayaparivārā /
subhaga mayūrīvāsau bhujaṅgavidveṣiṇī jātā // 101 //
kadalī candanapaṅkaḥ paṅkeruhanīrahāraghanasāram /
śaśadharaśaśadharakāntaṃ śāntyai no madanahutabhujastasyāḥ // 102 //
apasāraya ghanasāraṃ kuru hāraṃ dūra eva kiṃ kamalaiḥ /
alamalamāli mṛṇālairiti vadati divāniśaṃ bālā // 103 //
saṃkalpairupanītaṃ tvāmantikamullasanmanovṛttiḥ /
dṛḍhamāliṅgati paścātsvabhujāpīḍena yāti vailakṣyam // 104 //
kusumāmodī pavanaḥ pikakūjitabhṛṅgasārtharasitāni /
iyamiyatī sāmagrī ghaṭitā vidhinaiva tadvināśāya // 105 //
abalāṃ balinā nītāṃ daśāmimāṃ makaraketunā rakṣa /
āpatpatitoddhṛtaye bhavati hi śubhajanmanāṃ janma // 106 //
no gṛhṇanti yathārthā arthijanairnigaditā giraḥ prāyaḥ /
mālatyā guṇaleśaṃ śṛṇu dhṛṣṭatayā tathāpi kathayāmi // 107 //
āsphālayato nūnaṃ dhanuratanoḥ kausumaṃ rajaḥ patitam /
saṃhṛtya sā sumadhyā viśvasṛjā nirmitā tena // 108 //
upahasati girisutāyā lāvaṇyaṃ yena satatalagnena /
na dravatāmupanītaṃ bhogīndravibhūṣaṇasya dehārdham // 109 //
śaśadharabimbārdhagatāṃ chāyāmiva saiṃhikeyavadanasya /
alipaṭalanīlakuṭilāmalakāvalimalikasaṃnidhau vahati // 110 //
sarasijamasthiraśobhaṃ vibhramarahitaṃ ca maṇḍalaṃ śaśinaḥ /
kena sametu samatvaṃ hṛdayapriya mālatīvadanam // 111 //
alirupari tadīkṣaṇayorbhrāntvāsaugandhyasūcitaviśeṣaḥ /
nipatati karṇāmburuhe nirguṇatāpyavasare sādhvī // 112 //
bibhrāṇe 'ruṇimānaṃ sahajaṃ jitabandhujīvarucimadhare /
yadalaktakavinyasanaṃ tattasyā maṇḍanakrīḍā // 113 //
citramidaṃ yatkṛśatā tasyā valiparigṛhītamadhyasya /
athavā no vidhivihitā mahatāpyapanīyate tanutā // 114 //
āstāmaparastāvattasyāḥ smaravasatipṛthutaranitambaḥ /
ślathayati kapilamunerapi dṛkpathapatitaḥ samādhānam // 115 //
tasyā rambhāvapuṣo rambhopamamūruyugalamavalokya /
makaradhvajo 'pi sahasā nijasāyakalakṣyatāṃ yāti // 116 //
jaghanabharālasayātā no yātā sā vilocanaprasaram /
tiṣṭhati tena manohara śarajanmā brahmacaryeṇa // 117 //
yadi kathamapi madhumathanaḥ paśyati tāmasamabāṇasarvasvam /
tadasārabhārabhūtaṃ lakṣmīvapururasi vinihitaṃ manute // 118 //
yadi patati sā kathaṃ cidvīkṣaṇaviṣayaṃ harasya tadavaśyam /
tribhuvanamaśivaṃ kurute vāmetaradehabhāgamāsādya // 119 //
saundaryaṃ tattādṛśamaśeṣayoṣidvilakṣaṇaṃ sṛjataḥ /
yanniṣpannaṃ dhātustanmanye kākatālīyam // 120 //
sahajavilāsanivāsaṃ tasyā vapuranabhivīkṣamāṇasya /
manye nākādhipateḥ sahasramapi cakṣuṣāṃ viphalam // 121 //
śithilayatu kusumacāpaṃ kṣipatu śarān bāṇadhau manojanmā /
saṃsārasārabhūtā vicarati bhuvi mālatī yāvat // 122 //
vātsyāyanamadanodayadattakaviṭavṛttarājaputrādyaiḥ /
ucchvasitaṃ yatkiṃ cittattasyā hṛdayadeśamadhyāste // 123 //
bharataviśākhiladattilavṛkṣāyurvedacitrasūtreṣu /
patracchedavidhāne bhramakarmaṇi pustasūdaśāstreṣu // 124 //
ātodyavādanavidhau nṛtte gīte ca kauśalaṃ tasyāḥ /
abhidhātuṃ yadi śakto vadanasahasreṇa bhogināmīśaḥ // 125 //
parigaladālolāṃśukamapayantraṇamupari mālatī rabhasāt /
nipatati nāpuṇyavatāṃ ratilālasamānasā rahasi // 126 //
ratirasarabhasāsphālanacalavalayaninādamiśritaṃ tasyāḥ /
tatkālocitamaṇitaṃ śrutipathamupayāti nālpapuṇyasya // 127 //
itthamabhidhīyamānaḥ śubhamadhye yadi bhavedudāsīnaḥ /
evaṃ tato 'bhidheyaḥ saṃdarśitakopayā dūtyā // 128 //
kiṃ saubhāgyamado 'yaṃ yauvanalīlābhirūpatādarpaḥ /
sahajapremopanatāṃ mālatikāṃ na bahu manyase yena // 129 //
na gaṇayati yā kulīnān draviṇavataḥ śāstravedinaḥ praṇatān /
sā bhavadarthe śuṣyati kusthānaniveśitaṃ dhiganurāgam // 130 //
kamalavatī tīvrarucau bahubhasmani śambhuśirasi śaśilekhā /
sā ca tvayi paśukalpe yadabhiratā tena me kṛśatā // 131 //
asaralamarasaṃ kaṭhinaṃ durgrahamasnigdhamāśritā khadiram /
yadupaiti vācyapadavīṃ mālatikā tatkimāścaryam? // 132 //
athavā kaḥ khalu doṣo yadatulyatayopajanitavailakṣyaḥ /
svādhīnāmapi sarasāṃ pariharati mṛṇālikāṃ dhvāṅkṣaḥ // 133 //
mātra kariṣyasi khedaṃ niṣṭhuramukto 'si yanmayā subhaga /
yūnāṃ hi raktataruṇīsuhṛdabhihitaparuṣamābharaṇam // 134 //
candramaseva jyotsnā kaṃsāsuravairiṇeva vanamālā /
kusumaśarāsanalatikā kusumākaravallabheneva // 135 //
madalīlā halineva stanayugaleneva hāralatā /
ramyāpi sā sugātrī ramyatarā bhavatu saṃgatā bhavatā // 136 //
kiṃ bahunā yadi yūnāmupari vidhātuṃ samīhase caraṇam /
tatkuru ramaṇīratnaṃ premojjvalamaṅkatastūrṇam // 137 //
atha tadvacanaśravaṇapravijṛmbhitamadanabhaṭṭadāyādaḥ /
upacaraṇīyaḥ sundari nijavasatimupāgatastvayāpyevam // 138 //
dūrādabhyutthānaṃ praṇamanamātmāsanapradānaṃ ca /
pravidheyamañcalena prasphoṭanamaṅghriyugalasya // 139 //
īṣadayatnaprakaṭaṃ kakṣodarabāhumūlakucabhāgam /
saṃdarśya jhaṭiti yāsyasi nāyakadṛggocarāttūrṇam // 140 //
atha paryaṅkasanāthaṃ dīpojjvalakusumadhūpagandhāḍhyam /
vitatavitānakaramyaṃ praveśito vāsakāvāsam // 141 //
mātrā te gurujaghane sādaramavatāraṇādikaṃ kṛtvā /
abhinandanīya ebhirvacanaviśeṣaiḥ prayatnena // 142 //
adyāśiṣaḥ samṛddhāḥ parituṣṭā iṣṭadevatā adya /
kalyāṇālaṅkāro yadalaṅkṛtavānidaṃ veśma // 143 //
anurūpapātraghaṭanaṃ kurvāṇasyādya kusumabāṇasya /
sucirādbata saṃjātaḥ śarāsanākarṣaṇaśramaḥ saphalaḥ // 144 //
vinyasya śirasi caraṇaṃ subhagā gaṇikājanasya sakalasya /
saubhāgyavaijayantīṃ samprati vatsā samutkṣipatu // 145 //
duhitara eva ślāghyā dhiglokaṃ putrajanmasaṃtuṣṭam /
jāmātara āpyante bhavādṛśā yadabhisambandhāt // 146 //
dṛḍhaparicayā guṇajñā bhavadvidhā nārthanārhakā yadapi /
tadapi hṛdayābhinandana duhitṛsnehādahaṃ vacmi // 147 //
sahajapremopahṛtā nyastā tvayi mālatī tathā kāryam /
na yathā bhavati varākī tvadvipriyajanmanāṃ śucāṃ vasatiḥ // 148 //
mṛdudhautadhūpitāmbaramagrāmyaṃ maṇḍanaṃ ca bibhrāṇā /
paripītadhūpavartiḥ sthāsyasi ramaṇāntike sutanu // 149 //
sasnehaṃ savrīḍaṃ sasādhvasaṃ saspṛhaṃ ca paśyantī /
kiṃ ciddṛśyaśarīrā praviralaparihāsapeśalālāpā // 150 //
mātari niryātāyāṃ parijanamukte ca vāsakasthāne /
abhiyuñjāne ramaṇe vāmācaraṇaṃ kṣaṇaṃ kāryam // 151 //
ratisaṃgaranihitamatāvākarṣati rabhasataḥ purastasmin /
kuṭṭamitamācarantī janayiṣyasi kiṃ cidaṅgasaṃkocam // 152 //
prārabdhe suratavidhau kramadarśitacittayonisaṃvegā /
apaśaṅkamarpayiṣyasi nirvyājaṃ putri gātrāṇi // 153 //
yadyadvāñchati hantuṃ yaddaṃṣṭuṃ yacca vilikhituṃ gātram /
tattadapasāraṇīyaṃ sāvegaṃ ḍhaukanīyaṃ ca // 154 //
daṃśe savyathahuṅkṛtimāmarde vividhakaṇṭharasitāni /
nakhavilikhane ca sītkṛtamāghāteṣūlbaṇaṃ kvaṇitam // 155 //
hrasvāyāsaśvāsān muñcantī pulakadanturaśarīrā /
svidyatsarvāvayavā prakariṣyasi rāgavṛddhaye puṃsām // 156 //
parabhṛtalāvakahaṃsakapārāvataturagahṛdayaniḥsvanitam /
anukāryamucitakāle kalakaṇṭharutaistvayā rasataḥ // 157 //
mā mā māmatipīḍaya muñca kṣaṇamadaya no samarthāsmi /
iti gadgadāsphuṭākṣaramabhidhātavyastvayā kāmī // 158 //
anubandhamānukūlyaṃ vāmatvaṃ prauḍhatāmasāmarthyam /
surateṣu darśayiṣyasi kāmukabhāvaṃ sphuṭaṃ buddhvā // 159 //
asamañjasamaślīlaṃ dūrojjhitadhairyamavinayaprasaram /
vyavahāramācariṣyasi vṛddhimupete ratāvege // 160 //
avivecitanakharakṣatirāmīlitalocanā nirutsāhā /
nāyakakāryasamāptau sthāsyasi śithilīkṛtāvayavā // 161 //
jhagiti nitambāvaraṇaṃ niḥsahatanutāṃ smitaṃ savailakṣyam /
khedālasāṃ ca dṛṣṭiṃ janayiṣyasi mohanacchede // 162 //
vṛtte ratābhiyoge spṛṣṭvā salilaṃ vikṛṣṭabhūbhāge /
prakṣālya pāṇipādaṃ sthitvā kṣaṇamāsane samūhya kacān // 163 //
upayuktavadanavāsā śayyāmāruhya darśitapraṇayā /
iti vakṣyasi taṃ ramaṇaṃ dṛḍhataramāliṅgya rabhasataḥ kaṇṭhe // 164 //
bhaṭṭasuta nūnamiṣṭā tava jāyā yadanuraktahṛdayasya /
janayati parituṣṭimalaṃ nāpararāmāpariṣvaṅgaḥ // 165 //
saphalaṃ tasyā janma spṛhaṇīyā saiva sakalalalanānām /
gaurī tayaiva mahitā subhagaṅkaraṇaṃ tapastayācaritam // 166 //
saivaikā guṇavasatistasyā evānvayaḥ sadā ślāghyaḥ /
yasyāḥ śubhaśatabhājaḥ pāṇigrahaṇaṃ tvayā vihitam // 167 //
tiṣṭhatu sā puṇyavatī vaṃśadvayabhūṣaṇaṃ varārohā /
yā nāpayāti bhavato lakṣmīriva narakavairiṇo hṛdayāt // 168 //
pātayasi kuvalayanibhe kautukamātreṇa locane yāsu /
tā api satyaṃ sundara harṣollasitā na mānti gātreṣu // 169 //
tanurapi nāthapraṇayaḥ prāyo mukharīkaroti laghumanasaḥ /
svārthaniveśitacittā karomi te 'bhyarthanāṃ tena // 170 //
tīvrasmaratāruṇyāccāpalataḥ kautukena ghṛṇayā vā /
madbhāgyasampadā vā dūtyā vā kauśalātsvabhāvādvā // 171 //
yo 'yaṃ premalavāṃśaḥ pradarśito 'smāsu jīvanopāyaḥ /
bādhā nātra vidheyā gaṇikājanavṛttamanyathā buddhvā // 172 //
yena snehaḥ krodhaḥ śāṭhyaṃ dākṣiṇyamārjavaṃ vrīḍā /
etāni santi tāsvapi jīvaddharmopanītāni // 173 //
nirvyājasamutpannaprabalapremābhibhūtahṛdayānām /
dayitavirahākṣamāṇāṃ gaṇikānāṃ tṛṇasamāḥ prāṇāḥ // 174 //
atrākarṇaya sādbhutamākhyānaṃ varṇayāmi yadvṛttam /
adyāpi bibharti vaṭo viśeṣaṇaṃ yadabhisambandhāt // 175 //
asti mahītalatilakaṃ sarasvatīkulagṛhaṃ mahānagaram /
nāmnā pāṭaliputraṃ paribhūtapurandarasthānam // 176 //
tribhuvanapuraniṣpādanakauśalamiva pṛcchato viriñcasya /
darśayituṃ nijaśilpaṃ varṇakamiva viśvakarmaṇā vihitam // 177 //
aśreyobhiranāśritamabhibhūtaṃ nābhibhūtidoṣeṇa /
na svīkṛtamupasargaiḥ kalikālamalairanālīḍham // 178 //
pātālatalaṃ bhogibhirambhodhirvāriratnanicayaiśca /
surasadanaṃ vibudhagaṇairdraviṇopacayaiḥ puraṃ kuverasya // 179 //
ramaṇībhirasuravivaraṃ kaṭakaṃ hemācalasya gāndharvaiḥ /
harinagaraṃ kratuyūpaiḥ śamavibhavairmunijanasthānam // 180 //
tiṣṭhantu sakalaśāstravyālocanavimalabuddhayo viprāḥ /
sadasadguṇanirṇītau lalanā api nikaṣabhūmayo yasmin // 181 //
kalikāloditabhītyā kratuhutavahadhūmakambalāvaraṇaḥ /
tiṣṭhannibhṛto 'pi kṛtaścaritairanumīyate yasmin // 182 //
apaharati pidhātumiva svakalaṅkaṃ śaśadharaḥ prasārya karān /
rātrau yatra vadhūnāṃ lāvaṇyaṃ vadanakoṣebhyaḥ // 183 //
timirapaṭalāsitāmbaramapaharadabhisārikājanaughasya /
nijatanukāntivitānaṃ vallabhasambhogavihataye yasmin // 184 //
yatra nitambavatīnāṃ vicalannayanāntaśitaśarairvraṇitaḥ /
śithilayati pathikalokaḥ svakalatrasamāgamotkaṇṭhām // 185 //
yatra ca kulamahilānāmalpatvaṃ vacasi pāṇipāde ca /
svacchatvamāśayeṣu vyālolaviśālanetre ca // 186 //
pīnapayodharabhāre ghanatā jīveśasahajarāge ca /
kuladevatārcanavidhau valiśobhā madhyabhāge ca // 187 //
gambhīratā svabhāve cetobhavabāṇatūṇanābhau ca /
vistīrṇatā nitambe gurujanapūjānuraktacitte ca // 188 //
hariṇāyatekṣaṇānāṃ vicchittiḥ koṣaharaṇamabjeṣu /
kuṭilatvamalakapaṅktau bālānāṃ kāmaceṣṭitaṃ yatra // 189 //
saṃyamanamindriyāṇāminopaghātagrahastamisrasya /
stabdhatvaṃ sālatarau hāralatā taralasaṃgatā yasmin // 190 //
bhujagāḥ pararandhradṛśaḥ khaṇḍyante priyatamādharā yatra /
sūcīvyathānubhūtirnṛtyābhyāsapravṛttānām // 191 //
natavapurapyatisaralā mantharagamanāpi narmadā yasmin /
gurujanaśāstraratāpi svabhāvamugdhāṅganājanatā // 192 //
tasminmakhaśatapūtaḥ puruhūta iva dvijanmanāṃ pravaraḥ /
gururiva vidyāvasatirvasati sma purandaro nāmnā // 193 //
dharmātmajasya satyaṃ tripurariporvijitakusumacāpatvam /
harinābhipaṅkajabhuvo vijitendriyatāṃ jahāsa yaḥ satatam // 194 //
nyakkṛtavṛṣa iti śarve yācaka iti kaustubhābharaṇe /
pīḍitavasudhāsuta iti kapile na babhūva yasya bahumānaḥ // 195 //
mārgānusṛtau lubdho yaḥ prāṇivapurvināśavimukho 'pi /
parihṛtaparadāro 'pi svākāṅkṣitagurujanapramadaḥ // 196 //
yasyānvaye mahīyasi sarasīva samastasattvanijavasatau /
saccaritajanmabhūmau vinivāritakalimalaprasare // 197 //
pitṛtarpaṇaprasaṅge khaḍgagrahaṇaṃ na śauryadarpeṇa /
truṭanaṃ mekhalikānāṃ vaṭukajane no ratājisaṃmarde // 198 //
śrutibhedeṣu vivādo no rikthavibhāgamanyunā janitaḥ /
tejasvitā havirbhuji na śamaikarateṣu bhūmideveṣu // 199 //
jaratāmeva skhalanaṃ japatāmevādharasphuraṇam /
yajatāmeva samidhruci reṇājina eva kṛṣṇasamparkaḥ // 200 //
tasyābhūtsakalakalodbhāsitapakṣadvayasya suta ekaḥ /
nāmnā sundarasenaḥ kaca iva vacasāmadhīśasya // 201 //
paśupatinayanahutāśanabhasmitamavadhārya yaṃ vapuṣmantam /
aparamiva kusumacāpaṃ ratirataye nirmame dhātā // 202 //
tiṣṭhantu tāvadanyāḥ kulalalanā yasya rūpamavalokya /
sāpi mahāmunidayitā kṛcchreṇa rarakṣa cāritram // 203 //
kaladhautaphalakaśobhāṃ bibhrāṇaṃ yasya pṛthutaraṃ vakṣaḥ /
dṛṣṭvā cirāya lakṣmīrharihṛdaye duḥsthitaṃ mene // 204 //
kathamīdṛgyadi na kṛtaḥ śaśiśakalairatha kṛtaḥ kathaṃ vyathakaḥ /
itthaṃ yamīkṣamāṇo nirṇayamagamanna kāminīsārthaḥ // 205 //
yo jagrāha himāṃśoḥ prasannamūrtitvamacalataḥ sthairyam /
jaladharata unnatatvaṃ gāmbhīryaṃ yādasāṃ patyuḥ // 206 //
yo vinayasya nivāso vaidagdhyasyāśrayaḥ sthiteḥ sthānam /
priyavācāmāyatanaṃ niketanaṃ sādhucaritasya // 207 //
yo madanaḥ pramadānāṃ tuhinakaraḥ sādhukumudaṣaṇḍasya /
nikaṣopalo guṇānāṃ mārgataruḥ pathikalokasya // 208 //
sajjanagoṣṭhīnirataḥ kāvyakathāsāranikaṣapāṣāṇaḥ /
praṇayijanakalpavṛkṣo lakṣmīlīlāvihārabhūmiśca // 209 //
jaladhiriva tuhinabhāsaḥ sahavṛddhiparikṣayaḥ suhṛttasya /
sakalopadhāviśuddho babhūva guṇapālito nāmnā // 210 //
tena samaṃ sa kadā cittiṣṭhanrahasi prasaṅgataḥ patitām /
kenāpi gīyamānāmaśṛṇodāryāmimāṃ sahasā // 211 //
deśāntareṣu veṣasvabhāvabhaṇitāni ye na budhyante /
samupāsate na ca gurūnviṣāṇavikalāsta ukṣāṇaḥ // 212 //
ākarṇyātha tamūce vacanamidaṃ sundaraḥ suhṛnmukhyam /
śobhanametadgītaṃ guṇapālita sādhunānena // 213 //
sādhūnāmācaritaṃ khalaceṣṭāṃ vividhalokahevākān /
narma vidagdhairvihitaṃ kulaṭājanavakrakathitāni // 214 //
gurugūḍhaśāstracarcāṃ viṭavṛttaṃ dhūrtavañcanopāyān /
vāridhiparikhāṃ pṛthvīṃ jānāti paribhramanpuruṣaḥ // 215 //
ata utsṛjya gṛhasthitisukhaleśaṃ vividhalābhapariṇāme /
sthāpaya gamanārambhe vayasya hṛdayaṃ mayā sahitaḥ // 216 //
itthaṃ nigaditavantaṃ suhṛduttaralābhalālasātmānam /
ūce sundarasenaṃ lajjita iva sahacaro vacanam // 217 //
abhyarthanānubandho lajjākara eva mādṛśāṃ kiṃ tu /
ākarṇaya kathayāmaḥ pathikānāṃ yāni duḥkhāni // 218 //
karpaṭakāvṛtamūrtirdūrādhvapariśramāvasitaśaktiḥ /
pāṃsūtkaradhūsarito dināvasāne pratiśrayākāṅkṣī // 219 //
mātarbhagini dayāṃ kuru mā maivaṃ niṣṭhurā bhava tavāpi /
kāryavaśena gṛhebhyo niryānti bhrātaraśca putrāśca // 220 //
kiṃ vayamutpāṭya gṛhaṃ prātargantāra īdṛgeva satām /

adya rajanīṃ nayāmo yathā kathaṃ cittvadāśraye mātaḥ /
astaṅgato vivasvānvada samprati kutra gacchāmaḥ // 222 //
iti bahuvidhadīnavacāḥ pratigehaṃ dvāradeśamadhitiṣṭhan /
nirbhartsyate varāko gṛhiṇībhiridaṃ vadantībhiḥ // 223 //
na sthita iha gehapatiḥ kiṃ raṭasi vṛthā prayāhi devakulam /
kathite 'pi nāpagacchati paśya manuṣyasya nirbandham // 224 //
atha yadi kathaṃ cidaparaḥ punaḥ punaryācito gṛhasvāmī /
nirdiśati sāvadhīraṇamatra svapihīti śīrṇagṛhakoṇe // 225 //
tatra kalahāyamānā tiṣṭhati gṛhiṇī vibhāvarīpraharam /
ajñātāya kimarthaṃ vāso dattastvayeti saha bhartrā // 226 //
īdṛgayaṃ saralātmā kiṃ kuruṣe bhagini tāvako bhartā /
sthāsyasi gehe 'vahitā bhramanti khalu vañcakā evam // 227 //
iti bhājanādiyācñāṃ buddhau vinidhāya nikaṭavartigṛhāt /
nārī samabhyupetā brūte tāmāptabhāvena // 228 //
gṛhaśatamadhikamaṭitvā kalamakulatthāṇucaṇamasūrādi /
ekībhūtaṃ bhuṅkte kṣudhopatapto 'dhvago bhaikṣam // 229 //
paravaśamaśanaṃ vasudhāṃ śayanīyaṃ suraniketanaṃ sadma /
pathikasya vidhiḥ kṛtavānupadhānakamiṣṭakākhaṇḍam // 230 //
iti nigaditavati tasmin sundarasenasya cottarāvasare /
iyamupagītā gītiḥ kenāpi kathāprasaṅgena // 231 //
nijavarabhavanaṃ suragṛhamurvītalamatimanoharaṃ śayanam /
kadaśanamamṛtamabhīpsitakāryaikaniviṣṭacetasāṃ puṃsām // 232 //
samupaśrutya ca suhṛdaṃ paurandariridamuvāca parituṣṭaḥ /
mama hṛdayagataṃ prakaṭitametena sataiva bhavatu gacchāvaḥ // 233 //
atha sahacaradvitīyaḥ kleśasamudrāvataraṇakṛtacittaḥ /
niragātsundarasenaḥ kusumapurādaviditaḥ pitrā // 234 //
paśyan vidagdhagoṣṭhīrabhyasyannāyudhāni sakalāni /
śāstrārthānavagacchanvilokayankautukāni vividhāni // 235 //
jānan patrachedanamālekhyaṃ sikthapustakarmāṇi /
nṛtyaṃ gītopacitaṃ tantrīmurajādivādyabhedāṃśca // 236 //
budhyanvañcakabhaṅgīrviṭakulaṭānarmavakrakathitāni /
babhrāma suhṛtsahitaḥ sundaraseno mahīmakhilām // 237 //
atha viditasakalaśāstro vijñātāśeṣajanasamācaraṇaḥ /
nijagṛhagamanākāṅkṣī sa śiloccayamarbudaṃ prāpa // 238 //
tatpṛṣṭhadeśadarśanalolamalaṃ sundaraṃ parijñāya /
guṇapālito babhāṣe vilokyatāmadrirāja iti // 239 //
eṣa sutaḥ sānutaṭasyandacchītācchasalilasampannaḥ /
lokānukampayeva prāleyamahībhṛtā marau nyastaḥ // 240 //
śiśirakarakāntamauliḥ kaṭakasthitapavanabhojanaḥ saguhaḥ /
vidyādharopasevyo bibharti lakṣmīmayaṃ śambhoḥ // 241 //
atra taruśikharasaṃgatasumanasa iti jātaniścayo manye /
abhilaṣati samuccetuṃ tārā niśi mugdhakāminīsārthaḥ // 242 //
āścaryaṃ yadupānte tiṣṭhantyetasya sapta munayo 'pi /

avagatya niravalambanamambaramārgaṃ pataṅgaturagāṇām /
ayamavanidharo manye viśrāntyai vedhasā vihitaḥ // 244 //
imamāśritya himāṃśoroṣadhayaḥ saṃnikarṣamupayātāḥ /
pratyāsattiḥ prabhuṇā prāyo 'nugrāhakavaśena // 245 //
sektumivāśākariṇo visṛjatyayamavanidharaṇaparikhinnān /
nirjharasalilakaṇaughān bhavati hi sauhārdamekakāryāṇām // 246 //
hārītāhṛtaśobho muditaśuko vyāsayogaramaṇīyaḥ /
viśrāntabharadvājaḥ samatāmayameti muninivāsasya // 247 //
asminniḥsaṅgā api paralokaprāptyupāyakṛtayatnāḥ /
gandhavahabhojanā api na hiṃsakāḥ phalabhujo 'pi na plavagāḥ // 248 //

anabhimataraudracaritāḥ śivapriyā api vasanti śamaniratāḥ // 249 //
mūrtiriva śiśiraraśmerhariṇavatī saptapatrakṛtaśobhā /

sotkaṇṭheva samadanā vāsakasajjeva tilakaparibhūṣā /
dhavaharipīlusanāthā naranāthadvārabhūmiriva // 251 //
arjunabāṇavrātaiḥ kurunāthavarūthinīva saṃchannā /
ṛkṣasahasropacitā lakṣmīriva gaganadeśasya // 252 //
dhvajinīva dānavānāṃ riṣṭakasamadhiṣṭhitā triyāmeva /
uddyotarohiṇīkā ramyeyamutpatyakā bhāti // 253 //
iti darśayati vayasye sundarasene ca paśyati prītyā /
svaprastāvopagatā gītiriyaṃ kena cidgītā // 254 //
atiśayitanākapṛṣṭhaṃ pṛṣṭhaṃ ye nārbudasya paśyanti /
bahuviṣayaparibhramaṇaṃ manye kleśāya kevalaṃ teṣām // 255 //
ākarṇya ca sa babhāṣe mahātmanānena yuktamupagītam /
śikhariśiraḥ paśyāmo vayasya ramyaṃ samāruhya // 256 //
atha girivaramārūḍho vilokayanvividhavibudhabhavanāni /
vāpīrudyānabhuvaḥ sarāṃsi saritaścacāra vismeraḥ // 257 //
acirābhāmiva vighanāṃ jyotsnāmiva kumudabandhunā vikalām /
ratimiva manmatharahitāṃ śriyamiva harivakṣasaḥ patitām // 258 //
hastoccayaṃ vidhātuḥ sāraṃ sakalasya jantujātasya /
dṛṣṭāntaṃ ramyāṇāmastraṃ saṃkalpajanmano jaitram // 259 //
vikasitakusumasamṛddhiṃ śṛṅgārarasāpagaikakalahaṃsīm /
līlāpallavavallīṃ vratināmavadhānavarmaṇāṃ bhallīm // 260 //
vicarannupavanamaṇḍapapuṣpaprakarābhirāmabhūpṛṣṭhe /
ramamāṇāṃ saha sakhyā lalanāmālokayāmāsa // 261 //
avalokayatastasya smaramārgaṇavedhyatāmupetasya /
idamāsīnmanasi ciraṃ vismayabhārābhibhūyamānasya // 262 //
kvedaṃ khalu viśvasṛjaḥ kauśalamatyadbhutaṃ samupajātam /
yena viruddhānāmapi ghaṭitaikatra sthitistathā hīyam // 263 //
lalitavapurnirdoṣā sphuradujjvalatārakābhirāmā ca /
nirvācyavadanakamalā jitavīṇākvaṇitavāṇī ca // 264 //
saṃvihitavigrahasthitiratiśobhanaghaṭitasandhibandhā ca /
unnatapayodharāḍhyā śaradindukarāvadātā ca // 265 //
abhimatasugatāvasthitirabhinanditacaraṇaracanā ca /
ativipulajaghanadeśā vidhvastaśarīravihitaśobhā ca // 266 //
āvirbhavadanurāge tasminnatha valitalocanā sahasā /
sāpi babhūva mṛgākṣī hastagatā kusumacāpasya // 267 //
tarumūlamāśritāyā vismṛtasakalānyakarmaṇaḥ sapadi /
tasyā gātralatāyāmaṅkuritaṃ sāttvikairbhāvaiḥ // 268 //
saivopavanasamṛddhistasminneva kṣaṇe smaraṃ smṛtvā /
tāṃ vyathayitumārebhe prabhorhi kṛtyaṃ karoti khalu sarvaḥ // 269 //
gātrasarasendhanebhyaḥ prasvedajalaṃ viniryayau tasyāḥ /
antarjvalitamanobhavahavyabhujā dahyamānebhyaḥ // 270 //
kusumaśarajālapatitā muhurmuhurvidadhatī vivṛttāni /
animeṣaṃ paśyantī matsyavadhūmanucakāra sā tanvī // 271 //
stabdhatanuṃ sotkampāṃ pulakavatīṃ svedinīṃ saniḥśvāsām /
vidadhe tāmasamaśaraḥ krīḍati hi śaṭho viśiṣṭamāsādya // 272 //
ucchvāsairullasanaṃ kucayugale sauṣṭhavaṃ vilāsānām /
abhilaṣitena premṇā snigdhatvaṃ cakṣuṣormanohāri // 273 //
anuraktyā vadanaruciṃ vacasi ca gamane ca sādhvasaskhalanam /
tasyā madanaḥ kurvannupaninye cārutāmadhikām // 274 //
pārśvagate 'pi preyasi kāmaśarāsāratāḍyamānāpi /
na śaśāka sābhidhātuṃ cittagataṃ praṇayabhaṅgato bhītā // 275 //
atha viditacittavṛttiḥ saktadṛśaṃ priyatame samākṛṣya /
madanena dahyamānāṃ vihasitaviśadaṃ jagāda tāmālī // 276 //
ayi hāralate saṃhara harahuṅkṛtidagdhadehasaṃkṣobham /
sadbhāvajānuraktirna hi paṇyaṃ paṇyanārīṇām // 277 //
avadhīraya dhanavikalaṃ kuru gauravamakṛśasampadaḥ puṃsaḥ /
asmādṛśāṃ hi mugdhe dhanasiddhyai rūpanirmāṇam // 278 //
abhirāme 'bhiniveśaṃ vidadhānā vibhavalābhanirapekṣā /
upahasyase sumadhye vidagdhavārāṅganāvāraiḥ // 279 //
yeṣāṃ ślāghyaṃ yauvanamabhimukhatāmupagato vidhiryeṣām /
phalitaṃ yeṣāṃ sukṛtaṃ jīvitasukhitārthitā yeṣām // 280 //
te 'vaśyaṃ svayameva tvāmanubadhnanti madanaśarabhinnāḥ /
na hi madhulehāḥ sundari mṛgyante cūtamañjaryā // 281 //
iti gaditavatīmālīṃ kāmaśarāsārabhinnasarvāṅgī /
avyaktaskhalitākṣaramūce kṛcchreṇa hāralatā // 282 //
sakhi kuru tāvadyatnaṃ bahumatamativedanāpratīkāre /
kroḍīkṛtā vipattyā na bhavantyupadeśayogyā hi // 283 //
asvāyattaḥ preyānmṛdupavanaḥ surabhikusumamudyānam /
iyatī khalu sāmagrī bhavati kṣīṇāyuṣāmeva // 284 //
matvā madanāśīviṣaviṣavegākulitavigrahāmālīm /
samupetya śaśiprabhayā paurandarirabhidadhe kṛtapraṇatiḥ // 285 //
yadi nāma ruṇaddhi giraṃ gaṇikābhāvopajanitavailakṣyam /
tadapi kathitavyameva snigdhāpadi na hi nirūpyate yuktam // 286 //
etāvati saṃsāre parigaṇitā eva te sujanmānaḥ /
āpatsu paritrāṇavyākulamanasāṃ sphuranti ye buddhau // 287 //
yasminneva muhūrte cakṣurviṣayaṃ gato 'si mama sakhyāḥ /
tata evārabhya gatā vidheyatāṃ dagdhamadanasya // 288 //
romodgamasaṃnahanaṃ bhittvāntarvigrahaṃ parāpatitāḥ /
tasyā mānasasambhavakodaṇḍaviniḥsṛtā iṣavaḥ // 289 //
kiṃ vidadhātu varākī kutra samāśvasitu yātu kaṃ śaraṇam /
pīḍayati bhṛśaṃ yasyā nityaśucirdakṣiṇo mṛduḥ pavanaḥ // 290 //
vacasi gate gadgadatāmujjhitamaunavratāścirāya pikāḥ /
hṛṣṭā vyathayanti sakhīṃ jātāvasarā nirargalaṃ virutaiḥ // 291 //
skhalitākulite gamane tanvaṅgyā agaṇitaśramā haṃsāḥ /
sucirāllabdhāvasarāḥ kurvanti gatāgatāni parituṣṭāḥ // 292 //
uṣṇocchvasitasamīrairvidahyamāno 'pi madhukarastasyāḥ /
alakakusumaṃ na muñcati kṛcchreṣvapi dustyajā viṣayāḥ // 293 //
no vārayasi tathā māṃ sāmpratamiti kathayatīva madhulehaḥ /
niḥsahavapuṣaḥ karṇe śrutipūrakapuṣpasaṃgato guñjan // 294 //
praśithilabhujalatikātastasyāḥ patitasya hemakaṭakasya /
yatprāpaṇaṃ pṛthivyāstasminkhalu muktahastatā hetuḥ // 295 //
raśanāguṇena vigalitamekapade tannitambataścitram /
patanāya niyatamathavā niṣevaṇaṃ gurukalatrasya // 296 //
aṅgīkṛtya manobhavamurasi tathā lālito 'pi hatahāraḥ /
tāpayati sakhīṃ tatkṣaṇamantarbhinnātkutaḥ kuśalam // 297 //
vāsasitaṃ svedajalaṃ kajjalamalināśruvāriṇā miśram /
kucataṭapatitaṃ tasyāḥ prayāgasambhedasalilamanukurute // 298 //
pikarutamalayasamīraṇasumanaḥsmarabhṛṅgadahanaparikaritā /
pañcatapaścarati bhavatparirambhaṇasaukhyalampaṭā bālā // 299 //
na parāṃ patati varākī daśamīṃ yāvanmanobhavāvasthām /
trāyasva subhaga tāvaccharaṇāgatarakṣaṇaṃ vrataṃ mahatām // 300 //
atha tadvacasi kṛtādaramudbhūtamanobhavaṃ samavadhārya /
avagītibhītacetā ūce guṇapālitaḥ suhṛdam // 301 //
yadyapi māraprasaro durvāraḥ prāṇināṃ nave vayasi /
cintyaṃ tadapi vivekibhiravasānaṃ veśayoṣitāṃ premṇaḥ // 302 //
vārastrīṇāṃ vibhramarāgapremābhilāṣamadanarujaḥ /
sahavṛddhikṣayabhājaḥ prakhyātāḥ sampadaḥ suhṛdaḥ // 303 //
tābhiravadātajanmā kurvīta samāgamaṃ kathaṃ yāsām /
kṣaṇadṛṣṭo 'pi praṇayī rūḍhapraṇayo 'pi janmano 'pūrvaḥ // 304 //
pradyumnaḥ pradyumno virūpakaḥ khalu virūpakaḥ satatam /
susnigdhaḥ susnigdho rūkṣo rūkṣastu gaṇikānām // 305 //
yāsāṃ jaghanāvaraṇaṃ parakautukavṛddhaye na tu trapayā /
ujjvalaveṣā racanā kāmijanākṛṣṭaye na tu sthitaye // 306 //
māṃsarasābhyavahāraḥ puruṣāhatipīḍayā na tu spṛhayā /
ālekhyādau vyasanaṃ vaidagdhyakhyātaye na tu vinodaḥ // 307 //
rāgo 'dhare na cetasi saralatvaṃ bhujalatāsu na prakṛtau /
kucabhāreṣu samunnatirācaraṇe nābhinandite 'sadbhiḥ // 308 //
jaghanasthaleṣu gauravamākṛṣṭadhaneṣu no kulīneṣu /
alasatvaṃ gamanavidhau no mānavavañcanābhiyogeṣu // 309 //
varṇaviśeṣāpekṣā prasādhane no ratiprasaṅgeṣu /
oṣṭhe madanāsaṅgo no puruṣaviśeṣasambhoge // 310 //

klībeṣvapi kāntadṛśaḥ sākāṅkṣā dīrgharoge 'pi // 311 //
svedāmbukaṇopacitā anārdratānijanivāsamanasaśca /
āviṣkṛtavepathavo vajropalasārakaṭhināśca // 312 //
jaghanacapalā anāryāḥ parabhṛtayaḥ kṛtakanetrarāgāśca /
sarvāṅgārpaṇadakṣā asamarpitahṛdayadeśāśca // 313 //
na-kulasamutpannā api bhujaṅgadaśanakṛtavedanābhijñāḥ /
kandarpadīpikā api rahitāḥ snehaprasaṅgena // 314 //
ujjhitavṛṣayogā api ratisamaye naraviśeṣanirapekṣāḥ /
kṛṣṇaikābhiratā api hiraṇyakaśipupriyāḥ satatam // 315 //

nītaya iva bhūmibhṛtāṃ suparihṛtānarthasaṃyogāḥ // 316 //
bahumitrakarajadāraṇalabdhābhyudayāḥ saroruhiṇya iva /
ḍākinya iva ca raktavyākarṣaṇakauśalopetāḥ // 317 //
pratipuruṣaṃ saṃnihitāḥ kṛtyaparā vividhavikaraṇopacitāḥ /
bahulārthagrāhiṇyaḥ prakṛtaya iva durgrahā gaṇikāḥ // 318 //
sādaramāśliṣya ciraṃ kusumastabakaṃ ca naraviśeṣaṃ ca /
riktīkartuṃ nipuṇaṃ kṣudrāḥ kṣudrāśca cumbanti // 319 //
paramārthakaṭhorā api viṣayagataṃ lohakaṃ manuṣyaṃ ca /
cumbakapāṣāṇaśilāḥ rūpājīvāśca karṣanti // 320 //

āhanyamānajaghanāḥ kareṇavo vārayoṣāśca // 321 //

jhagiti patanti mukhena prakaṭapramadāḥ kalādhikāśca tulāḥ // 322 //
bahirupapāditaśobhā antastucchāḥ svabhāvataḥ kaṭhināḥ /
dāsyaḥ samudgikā iva maṇanti yantraprayogeṇa // 323 //

ityupadiśati vayasye sundarasene ca manmathavyathite /
āvādupayātaṃ gītitrayamabhyadhāyi kenāpi // 325 //

laḍahanitambavatījanasambhogasukhena yāti tāruṇyam // 327 //

prabalapremapravaṇāḥ pramadāḥ spṛhayanti nālpapuṇyebhyaḥ // 328 //
evamupaśrutya vacaḥ samuvāca purandarātmajaḥ suhṛdam /
mama hṛdayādiva kṛṣṭvā gītamidaṃ sādhunānena // 329 //
tadatanusāyakavikalāṃ hāralatāṃ hariṇaśāvataralākṣīm /
āśvāsayituṃ yāmo guṇapālita kiṃ vikalpitairbahubhiḥ // 330 //
atha yatra kāpi gaṇikā gaṇayantī paricitaṃ hṛtadraviṇam /
praviśantameva mandiramīrṣyāvyājena nirurodha // 331 //
kā cidvañcakadattaṃ luṇḍīkṛtajīrṇavasanamavalokya /
veśyā viṣīdati sma kṣapākṣaye vṛttakartavyā // 332 //
daivasmṛtyā patitaṃ dṛṣṭipathaṃ bhagnamūlyaviṭamekā /
jvalitā ruṣā bhujiṣyā jagrāha javena dhāvitvā // 333 //
antaḥsthitakāmigṛhadvāragataṃ luptavittanaramanyā /
samuvāca kuṭṭanī vraja kallolākalpadeheti // 334 //
prakaṭitadaśananakhakṣatirabhidadhatī rājaputraratiyuddham /
aparā puraḥ sakhīnāṃ vāravadhūrātatāna saubhāgyam // 335 //
anyā kāmispardhāvardhitabhāṭī samucchritā kheḍā /
saubhāgyagarvadarpaṃ samuvāha vilāsinīmadhye // 336 //
ekagaṇikānubandhakrodhodyataśastrakāminoḥ kāpi /
sambhramato dhāvitvā nivārayāmāsa kuṭṭanī kalaham // 337 //
dhanamāhṛtya bahubhyo bhujyata ekena kena citsārdham /
iti dhanavantaṃ kāminamāvarjayati sma kāpi vāravadhūḥ // 338 //
gāyangāthāmātraṃ dvipadakamatha sauṣṭhavena viṭa ekaḥ /
babhrāma puro dāsyā vidadhadvikṛtīranekavidhāḥ // 339 //
kaścitpaṇyastrīṇāṃ vibhavopacitānyapuruṣayojanayā /
vidadhāti smārādhanamadhanatvamupāgataḥ kāmī // 340 //

tadapi taraṅgavatī sā kandarpaka kiṃ vṛthā garvaḥ // 347 //
jīvadveṣin lāsaka parihara dūreṇa mūḍha harisenām /
baddhāveśastasyāṃ vyāpṛtaputro mahāviṣamaḥ // 348 //
kesarayā kṣaṇadattaṃ kṛtvāṃśukamupari kāmijālasya /
stabdhagrīvaṃ bhramataścandrodaya paśya māhātmyam // 349 //
kaumārakaṃ vidhātuṃ vāñchasi kila ramaṇa madanasenāyāḥ /
icchāmi kiṃ tu tasyā mātrātīva prasāritaṃ vadanam // 350 //
vibhrama kiyatastapasaḥ phalametadyadupabhujyate madirā /
svakareṇa pītaśeṣā madaghūrṇitadevadattayā dattā // 351 //
kuvalayamālānilayo līlodaya kimiti samprati tyaktaḥ /
kiṃ vidadhāmastasmin bhrātardāsyā vinā mūlyam // 352 //
muṣitāśeṣavibhūterindīvarakasya yāminī yāti /
saṃvāhayataḥ samprati mañjīraka tilakamañjarīcaraṇau // 353 //
adyāpi bālabhāvaṃ nikhilaṃ na jahāti madanikā tadapi /
prauḍhimnā makarandaka sakalā lalanā adhaḥ kurute // 354 //
kubje gatvā vakṣyasi taṃ nirdayacittanartanācāryam /
hārā sukumāratanuḥ kimiti śramamadya kāritā bhavatā // 355 //
niḥsara ko 'bhiniveśaḥ śukaśāvakapāṭhane suratadevi /
tiṣṭhati bahirupaviṣṭaḥ pratīkṣamāṇastava preyān // 356 //

utthāpaya tāṃ tvaritaṃ smaralīlāṃ bhaṭṭaputra āyātaḥ // 357 //
kimidaṃ yathāsthitatvaṃ tava mādhavi yanmuhurvadantyā me /

īdṛk śūnyamanastvaṃ kiṃ kurmo mātarindulekhāyāḥ /
pānakrīḍāsaktyā patitāpi na cetitā kanakatāḍī // 359 //
nakulaḥ payo na pāyita iti roṣavaśādiyaṃ hi duḥśīlā /
nāśnāti kāmasenā punaḥ punaḥ prārthyamānāpi // 360 //
śrībalasutaparipālita ūrṇāyuḥ kimanayā vijetavyaḥ /
mukulā muktasukhasthitiraharniśaṃ meṣapoṣaṇe lagnā // 361 //
ātāmratāṃ samupagatamucchūnaṃ karatalaṃ tava sulalite /
mā punaraticiramevaṃ pravidhāsyasi kandukakrīḍām // 362 //
abhirāma ḍomba bhāṭī prathamamiyaṃ gṛhyate samutpanne /
snehe tu kusumadevyāstvaṃ prabhavasi jīvitasyāpi // 363 //
grahaṇakamarpaya tāvadyadi kautukamupari candrasenāyāḥ /
nirvartitakartavyo dāsyasi kiṃ cidyathābhimatam // 364 //
na paramadātā mātaḥ sūnurasau nagararoṭabhaṭṭasya /
nirlajjaḥ śaṭhavṛttiḥ punaḥ punarvāryamāṇo 'pi // 365 //

itthaṃprāyā vācaḥ śṛṇvan viṭakuṭṭanīsamudgīrṇāḥ /

ākṛṣṭamivotkatayā snapitamiva snigdhacakṣuṣaḥ prasaraiḥ /
tamupāgatamabhyarṇaṃ hāralatā pūjayāmāsa // 369 //
saṃvihitasamucitasthitiravanataśirasā praṇamya tatsakhyā /
idamabhidadhe 'tinamraṃ sundarasenaḥ śubhe 'vasare // 370 //
priyadarśana kiṃ bahubhiḥ smarapīḍitadīnavacanasaṃdarbhaiḥ /
iyamāste hāralatā jīvitamasyāstvadāyattam // 371 //
niryantrakeliviśadaṃ sahajapremānubandharamaṇīyam /
kāryāntarāntarāyaiḥ suparihṛtaṃ yātu yauvanaṃ bhavatoḥ // 372 //
nirdayamaviratavāñchaṃ dhvastatrapamavyavasthitācaraṇam /
upacīyamānarāgaṃ satataṃ bhūyādbhavatsuratam // 373 //
iti dattvāśiṣamantarniryāte parijane tadaṅgeṣu /
visrambhaviviktaraso vavṛdhe kusumāyudhaḥ sutarām // 374 //
yadamandamanmathocitamanurūpaṃ yannavānurāgasya /
yadyauvane 'bhirāmaṃ yacca phalaṃ jīvitavyasya // 375 //
avinaya eva vibhūṣaṇamaślīlācaraṇameva bahumānaḥ /
niḥśaṅkataiva sauṣṭhavamanavasthitireva gauravādhānam // 376 //
keśagrahaṇamanugraha upakārastāḍanaṃ mude daṃśaḥ /
nakhavilikhanamabhyudayo dṛḍhadehanipīḍanaṃ samutkarṣaḥ // 377 //
nigaraṇalolaṃ cumbanamavayavaniṣpeṣaṇaspṛho mardaḥ /
antaḥpraveśanecchaṃ nirbharaparirambhaṇaṃ yasmin // 378 //
yadanaṅgairiva vihitaṃ rāgairiva dīptimattvamupanītam /
premabhiriva niścalitaṃ śṛṅgārairiva vikāśamānītam // 379 //
aprāgalbhyaṃ vyasanaṃ dhairyamakāryaṃ viveka upaghātaḥ /
hrepaṇamaguṇo yasmin tatsurataṃ prastutaṃ tābhyām // 380 //
prārambha eva tāvatprajvalito dhagiti manasijo yasmin /
tasya viśeṣāvasthā vaktumaśakyāḥ pravṛddhasya // 381 //
sahajarasena jaḍīkṛtamiti yūnoḥ kāmaśāstranirṇīte /
nānākaraṇagrāme mālinyamavāpa pāṇḍityam // 382 //
avidheyamanākhyeyaṃ pravicāryaṃ chādanīyamaviṣahyam /
na babhūva tayostasminnāviddhārabdhasuratasaṃmarde // 383 //
abhyastā yā tanvyā suratavidhau vividhacāṭuparipāṭī /
tāmālūnaviśīrṇāṃ cakāra sahajaḥ smarāveśaḥ // 384 //
sadbhāvarāgadīpitamadanācāryopadiṣṭaceṣṭānām /
kaḥ parigaṇanaṃ kartuṃ raticakrāviṣṭaramaṇayoḥ śaktaḥ // 385 //
bālā mṛdugātralatā dṛḍhapuruṣākrāntavigrahā na param /
na vyathitā mudamāpa prabhavati khalu cittajanmanaḥ śaktiḥ // 386 //
kiṃ ramaṇīṃ ramaṇo 'viśaduta ramaṇī ramaṇamiti na jānīmaḥ /
svāvayavāvagamastu praṇāśamagamattayostadā nipuṇam // 387 //
tasyā nimīlitadṛśo niḥspandatanorbabhūva suratānte /
liṅgamanaṅgacchāyā jīvitasattānumānasya // 388 //
śramajalabindūpacitā vṛttasmaraṇena jātavailakṣyā /
sā śuśubhe rativiratau paryākulakeśabhūṣaṇā nitarām // 389 //
nirvyājārpitavapuṣornirvṛtimayameva gaṇayatorviśvam /
kṣaṇadā virarāma tayorakṣīṇākāṅkṣayoreva // 390 //
mohanavimardakhinnā vijṛmbhamāṇā skhaladgatirmandam /
nidrākaṣāyitākṣī hāralatā vāsaveśmano niragāt // 391 //
paricitapārśvagatāhaṃ tena samaṃ pānabhojanaṃ kṛtvā /
nītā niśā kathābhirmohanakāryaṃ tu yatkiṃ cit // 392 //
avidagdhaḥ śramakaṭhino durlabhayoṣidyuvā jaḍo vipraḥ /
apamṛtyurapakrāntaḥ kāmivyājena me rātrau // 393 //
necchāviratiḥ kṣaṇamapi na ca śaktirvastuśūnyaratiyatnaiḥ /
kevalamalamadyāhaṃ kadarthitā vṛddhapuruṣeṇa // 394 //
madyavaśādabhiyoktari mṛtakalpe talpabhāgamagnāyāḥ /
anirodhitanidrāyāḥ sukhena me yāminī yātā // 395 //
sukumārasamprayogaḥ peśalavacanaḥ savakraparihāsaḥ /
śakunavaśenopanato mama sakhi ramaṇo manoharākāraḥ // 396 //
paryaṅkāntanilīnaḥ parāṅmukho muktamandaniḥśvāsaḥ /
maccodanayā sutarāṃ niṣpandaḥ svedasalilasaṃsiktaḥ // 397 //
paryastamitānaṅgo vyapagatanidraḥ kṣapākṣayākāṅkṣī /
grāmoṣitaḥ prahīṇo niṣpratipattiḥ sthito 'dya sakhi manujaḥ // 398 //
śṛṇu sakhi kautukamekaṃ grāmīṇakakāminā yadadya kṛtam /
suratasukhamīlitākṣī mṛteti bhītena muktāsmi // 399 //
aviditadeśaprakṛteḥ śaṭhātmakāddurvidagdhato 'smābhiḥ /
anubhūto rājasutādapi bhāṇḍaviḍambanākleśaḥ // 400 //
priyasakhi lokasamakṣaṃ nagaraprabhuṇā haṭhena nītāsmi /
evaṃ bandhakadāturdviguṇārthaprārthane kuto 'nyāyaḥ // 401 //
ākarṣantī jaghanaṃ vrajasi yathā vikṣatā nakhaistilaśaḥ /
manye tathopabhuktā tvaṃ kerali dākṣiṇātyena // 402 //
adhare binduḥ kaṇṭhe maṇimālā kucayuge śaśaplutakam /
tava sūcayanti ketaki kusumāyudhaśāstrapaṇḍitaṃ ramaṇam // 403 //
iti śṛṇvannuṣasi giro nirvṛttaniśābhiyogagaṇikānām /
so 'pi yathākriyamāṇaṃ pravidhātuṃ nirjagāma kartavyam // 404 //
suracitarāgopacitisvīkṛtamanasastayā samaṃ tasya /
yauvanasukhamanubhavato jagāma saṃvatsaraḥ sārdhaḥ // 405 //
viśrambhakathāḥ kurvanvicarannudyānavedikāpṛṣṭhe /
sahacarakarasaktakaraḥ sundarasenaḥ kadā cittu // 406 //
sthūlaghanatantusantatitānitanānāmbarāvaraṇam /
yaṣṭiprāntaniyantritadalavīṭakakutapatumbakaprāyam // 407 //
truṭitacaraṇatrasaṃgatavisphuṭitābhyaktapādamalinatanum /
tvaritagatilekhavāhakamārādāyāntamadrākṣīt // 408 //
pratyāsannībhūtaṃ krameṇa paurandariḥ parijñāya /
sākūtamanā ūce vayasya hanumānayaṃ prāptaḥ // 409 //
avanitalalīnaśirasā kṛtanatinā tena vinihitaṃ bhūmau /
utkṣipya jhaṭiti lekhaṃ sundara iti vācayāmāsa // 410 //
svasti śrīkusumapurātpurandaraḥ sundaraṃ samabhidhatte /
antarjṛmbhitaśokagrasto 'vispaṣṭavarṇapadam // 411 //
kulamakalaṅkaṃ na gaṇitamavadhīritamagrajanmanāmucitam /
nāpekṣitamavagītaṃ śaṭhasevitavartmani tvayā patatā // 412 //
vaṃśe 'kuṭilagatīnāṃ dvijihvatādoṣarahitacaritānām /
aparavināśaratānāmutpannaḥ kathamasi bhujaṅgaḥ // 413 //
kva puroḍāśapavitritavedapadodgāragarbhavadanaṃ te /
kva ca madirāsavavāsitavāravadhūmukharasāsvādaḥ // 414 //
kva kuśavipāṭanajanmā sahasoditavedanācamatkāraḥ /
kva ca dāsīratasaṅgaranirdayanakharakṣatiḥ prītyai // 415 //
kva tretānaladhūmakṣobhitanayanāmbudhautavadanatvam /
kva ca gaṇikānirbhartsanaśokabharāyātabāṣpasalilaughaḥ // 416 //
kva vaṣaṭkāradhvānaḥ ṣaṭkarmavibhūṣaṇaṃ śravaṇapūraḥ /
kva ca sādhāraṇavanitāratimaṇitākarṇanautsukyam // 417 //
kvācāryapratanulatātāḍanasaṃkṣobhasambhavaḥ kampaḥ /
kva ca kupitavāralalanāniṣṭhurapādaprahāraviṣahitvam // 418 //
kva hariṇacarmāvaraṇaṃ smṛtiśāstraniveditaṃ vrataṃ carataḥ /
kva ca paṇyastrīgātraspṛṣṭāmbaradhāraṇeṣu bahumānaḥ // 419 //
samidhāmeva cchedanamabhyastaṃ śaiśavātsamārabhya /
śaṭhavanitādharakhaṇḍana utpannaṃ kauśalaṃ kuto bhavataḥ // 420 //
śuśrūṣaṇameva guroḥ pariśīlitamacalacetasā satatam /
kuṭilamatayo bhujiṣyāḥ kathaṃ tvayārādhitā nipuṇam // 421 //
āmnāyapāṭha eva sphuṭatarapadasauṣṭhavaṃ tava khyātam /
prakupitaveśyānunaye kva śikṣitaṃ vacanacāturyam // 422 //
athavā kiṃ kriyate 'sminnavadātakule 'pi labdhajanmānaḥ /
sadasaṃstutā bhavanti prāgupacitakarmadoṣeṇa // 423 //
tvayi viniveśya kuṭumbaṃ paralokahitārjanaikanihitātmā /
sthāsyāmīti samīhitamanudivasaṃ tadvisaṃvaditam // 424 //
ityavadhṛtalekhārthe sundarasene vidheyasammūḍhe /
āryāmagāyadanyaḥ svāvasare gītiparikaritām // 425 //
viṣayatimirāvṛtākṣṇāmavaṭe patatāmadṛṣṭamārgāṇām /
puṃsāṃ gurujanavacanadravyaśalākāñjanaṃ śaraṇam // 426 //
udvejayati tadātve sukhasampattiṃ karoti pariṇāme /
kaṭukauṣadhaprayogo gurunigaditakāryaniṣṭhuraṃ ca vacaḥ // 427 //
labdhvātha vacovasaraṃ mitramavādītpurandarāpatyam /
punarapi na hi khidyante priyajanahitabhāṣaṇe santaḥ // 428 //
agaṇitasahacaravacaso durvyasanamahābdhimagnavapuṣaste /
manyuvyathitasya pituryadi paramavalambanaṃ vacanam // 429 //
nijavaṃśadīpabhūtaḥ kṛtacaritālaṅkṛto mahāsattvaḥ /
sundara samprati tātaḥ spṛṣṭo duṣputradoṣeṇa // 430 //
putrābhāvaḥ śreyān duḥsutatā putriṇaḥ kulīnasya /
antastāpayati bhṛśaṃ saccaritakathāprasaṅgeṣu // 431 //
sāṃvyavahārika eva prāyo loke guṇonnatā niyatāḥ /
yena tu sutena jananī vandhyātvaṃ ślāghate sa pāpīyān // 432 //
viphalaṃ śāstrajñānaṃ gurugṛhasevāpi nopakārāya /
viṣayavaśīkṛtamanaso nyāyyaṃ panthānamutsṛjataḥ // 433 //
jīvanneva mṛto 'sau yasya jano vīkṣya vadanamanyonyam /
kṛtamukhabhaṅgo dūrātkaroti nirdeśamaṅgulyā // 434 //
no parihartuṃ viṣayāḥ śakyāḥ satyaṃ tathāpi nipuṇadhiyaḥ /
abhidheyatāṃ na gacchantyapavādaviśeṣitābhidhānasya // 435 //
guruparicaryā jāyā kulodgatā snigdhabandhusamparkaḥ /
brāhme karmaṇi saktirlokadvayasādhanaṃ sudhiyām // 436 //
sulabhā tasya vibhūtistasya guṇā yānti jagati vistāram /
bahu manute taṃ sujanastasmai spṛhayanti bāndhavāḥ satatam // 437 //
nāsādayati sa ekaḥ satsevitamārgataḥ pariskhalanam /
maṇḍayati so 'nvavāyaṃ sa nivāsaḥ śarmaṇāmaśeṣāṇām // 438 //
sa bhavati vinayādhāro yuktāyukte vivekitā tasya /
vṛddhopadeśavācaḥ śravaṇodaratarpaṇaṃ sadā yasya // 439 //
prāktanakarmavipākaḥ kṣudrāsu śarīriṇāṃ yadāsaktiḥ /
āyatanaṃ tu sukhānāṃ saṃsārabhuvāṃ kulodgatā rāmā // 440 //
nirviṇṇe nirviṇṇā mudite muditā samākulākulite /
pratibimbasamā kāntā saṃkruddhe kevalaṃ bhītā // 441 //
yāvadvāñchitasuratavyāyāmasahāviruddhasaṃbhāṣā /
cittānuvṛttikuśalā puṇyavatāmeva jāyate jāyā // 442 //
sadbhāvapremarasaṃ valayāvaliśabdaśaṅkitā nibhṛtam /
vidadhānāṅgasamarpaṇamunmīlitakusumasāyakākūtā // 443 //
hā hā kimuddhatatvaṃ śroṣyati kaścidgatatrapa svairam /
nikaṭe parivārajano vismṛta eva smarāturasya tava // 444 //
iti huṅkṛtisaṃvalitairāyāsaniveditārthapadavākyaiḥ /
dviguṇīkaroti kulajā nāyakakarmāṇi mohanaprasare // 445 //
itthamudīritavācaṃ suhṛdamavocatpurandarasya sutaḥ /
samupasthitajīvasamāviyogabhayakampito vacanam // 446 //
tātādeśe 'laṅghye hāralatāvirahapāvake tīvre /
vidhivaśavartini maraṇe no vidmaḥ kāryapariṇāmam // 447 //
anapekṣitadhanalābhāṃ snehaikanibaddhamānasāṃ dayitām /
daivākṛṣṭo muñcati ghaṭito vā lohavajrakaṇikābhiḥ // 448 //
atha kṛtagamanaviniścitirabhimatarāmāṃ cakāra viditārthām /
sāpi tamanuvavrāja prastutayātraṃ śucākulitā // 449 //
āsādya vaṭasya talaṃ bāṣpapayaḥkaṇacitākṣipakṣmāgrām /
vighnitacaraṇavihāro hāralatāmabhidadhāti sma // 450 //
ā kṣīravato vṛkṣādā salilādvā priye priyaṃ yāntam /
anuyāyāditi vacanaṃ tena tvamito nivartasva // 451 //
kiṃ kurmo daivahatāḥ prabhavati yasmin kṛśodari prasabham /
premagranthicchettā guruśāsanasāyako nirāvaraṇaḥ // 452 //
na draviṇacayaprāptirnaikāśrayaparicayo na cāṭuguṇaḥ /
na svāmisamādeśo nākāravilobhanaṃ na ca khyātiḥ // 453 //
hetustava pravṛtterasmāsu tathāpi daivavaśāt /
īdṛkko 'pyanubandho yasya vipāko 'pratīkāraḥ // 454 //
paruṣaṃ yadabhihitāsi praṇayaruṣā śaṅkitaṃ ca narmaṇi vā /
sudati na tatsmaraṇīyaṃ durbhāṣaṇakīrtanodghāte // 455 //
tava hṛdaye hṛdayamidaṃ vinyastaṃ nyāsapālanaṃ kaṣṭam /
yatnāttathā vidheyaṃ sthānabhraṃśo yathā na syāt // 456 //
atha viratavacodayitaṃ bāṣpabharakliṣṭavarṇapadayogāt /
iti kathamapi hāralatā saṃmūrchitavastubhāratīmūce // 457 //
aviśuddhakulotpannā dehārpaṇajīvikā śaṭhācaraṇā /
kvāhaṃ rūpājīvā kva bhavantaḥ ślāghanīyajanmaguṇāḥ // 458 //
yattu viṣayāvalokanakutūhalābhyāgatena viśrāntam /
iyato divasānasmin tanme parajanmakṛtaśubhasya phalam // 459 //
gurusevāṃ bandhujanaṃ svadeśavasatiṃ kalatramanukūlam /
anuṣaṅgadṛṣṭaparicita āsthāṃ pravidhāya kaḥ parityajati // 460 //
yauvanacāpalametadyanmādṛśi bhavati kautukaṃ bhavatām /
yattu sukhamanavagītaṃ tasya sthānaṃ nijā dārāḥ // 461 //
te madhurāḥ parihāsāstā vakragiraḥ sa vāmatāsamayaḥ /
no hṛdaye kartavyā rahasi kṣemārthinā bhavatā // 462 //
lāghavato yanmanasaḥ praṇayādvā yattavācaritam /
pratikūlaṃ tatra mayā nāthāñjalireṣa viracito mūrdhni // 463 //
duḥsaṃcārā mārgā dūre vasatirvisaṃṣṭhulaṃ hṛdayam /
guṇapālita tava suhṛdā bhavitavyamato 'pramattena // 464 //
hṛdayadvaya ekatvaṃ yāte yūnorviyogajaṃ kleśam /
anubhavatorapareṇa prasaṅgataḥ paṭhyate pathyā // 465 //
anyonyasudṛḍhaceṣṭitasadbhāvasnehapāśabaddhānām /
vicchedakaro mṛtyurdhīrāṇāṃ vā paricchedaḥ // 466 //
atha tacchravaṇānantaramāssva sukhaṃ dayitike vrajāmīti /
abhidhāya yāti mandaṃ sundarasene vivartitagrīvam // 467 //
vaṭaśākhālambibhujāṃ śvasitoṣṇasamīraśuṣyadadharamukhīm /
paryastāṃ bibhrāṇāṃ tanmārgavilokanānimeśadṛśam // 468 //
dolāyamānaveṇīṃ tiryaggatakaṇṭhabhūṣaṇaviśeṣām /
galadaśruvāripūrṇāṃ patitāṃśukabhāganiḥsahāṅgalatām // 469 //
rundhānāmiva hṛdayaṃ sphuṭaditarakareṇa kucayugāśrayiṇā /
pariśeṣitāṃ vilāsairutsṛṣṭāṃ jīvalokakartavyaiḥ // 470 //
aṅgīkṛtāṃ vipattyā vaśīkṛtāṃ marmaghaṭṭanairviṣamaiḥ /
hāralatāmaparisphuṭamantaḥparikṛṣyamāṇabhāratyā // 471 //
mā mā tāvadyāta kṣaṇamekaṃ yāvadeṣa niṣkaruṇaḥ /
vanagulmairna tirohita ityabhidadhatīṃ jahuḥ prāṇāḥ // 472 //
atha paścātsamupetaṃ papraccha purandarātmajaḥ pathikam /
dṛṣṭā śokavyathitā nivartamānāṅganā bhavatā // 473 //
sa uvāca vaṭataroradha urvyāṃ patitā viniścalāvayavā /
tiṣṭhati vanitā nānyā nayanāvasaraṃ gatāsmākam // 474 //
iti tadvacanāśmahato vihvalamūrtiḥ papāta bhūpṛṣṭhe /
utthāpitaśca suhṛdā so 'bhidadhe tena śokadīnena // 475 //
bhavatu kṛtārthastātastvamapi sumitrāssva sāmprataṃ prītaḥ /
samakālameva muktā pāpena mayāsubhiśca hāralatā // 476 //
hā hā hāva hato 'si dhvastā līlā vilāsa kiṃ kuruṣe /
uccchinnā vicchittirbhrama vibhrama daśa diśo nirādhāraḥ // 477 //
kilakiñcita gaccha vanaṃ moṭṭāyitamaśaraṇatvamupayātam /
kuṭṭamita pravrajyāṃ gṛhāṇa bibboka viśa bhuvo vivaram // 478 //
lalitamanāthībhūtaṃ vihṛtasya na vidyate gatiḥ kvāpi /
śaśadharabimbadyutimuṣi yātāyāmantakāntikaṃ tasyām // 479 //
vinivṛttya yāmi dagdhuṃ madvirahe muktavallabhaprāṇām /
bhavatu varākyāstasyāḥ saptārcirdānamātramupakāraḥ // 480 //
gatvātha tamuddeśaṃ yasmin sā pañcabhāvamāpannā /
vilalāpa muktanādaṃ viluṭhan bhuvi sahacareṇa dhṛtamūrtiḥ // 481 //
ete vayaṃ nivṛttā muñca ruṣaṃ dehi kopane vācam /
uttiṣṭha kimiti tiṣṭhasi bhūmitale reṇurūṣitaśarīrā // 482 //
vinimīlya dṛśau kasmādapratipattyā sthitāsi śubhavadane /
tvadavāritagamanavidheraparādhitayā na me yogaḥ // 483 //
nākādhipatipurandhrīrabhibhavituṃ tvayi divaṃ prayātāyām /
satsvapi śareṣu pañcasu nirāyudhaḥ sāmprataṃ madanaḥ // 484 //
vañcakavṛttā veśyā ityapavādo janeṣu yo rūḍhaḥ /
apanīto 'sau nipuṇaṃ tvayā priye jīvamokṣeṇa // 485 //
vandyaḥ sadvrata ekastripurāntakanandano mahāsenaḥ /
hṛdayaṃ yasya spṛṣṭaṃ na manāgapi vāmalocanāpremṇā // 486 //
manye 'bhīṣṭaviyogaṃ nimeṣamapi duḥsahaṃ samavadhārya /
hariṇā vakṣasi lakṣmīrvidhṛtā gaurī hareṇa dehārdhe // 487 //
ayi lokapāla sā bhuvi lalāmabhūtā tayā vinā śūnyam /
viśvamiti kiṃ na cintitamātmasthānaṃ priyāṃ nayatā // 488 //
bhagavan hutavaha mā mā lāvaṇyasamudrasāramuddhṛtya /
kathamapi vihitāṃ dhātrā dhakṣyasyenāṃ jagadbhūṣām // 489 //
iti vilapantaṃ bahuvidhamavadhīrya suhṛt purandarasya sutam /
kāṣṭhairviracayya citāṃ tāmakarodagnisādgaṇikām // 490 //
tasminniddhahutāśanavinipatane kṛtamatau śucākulite /
manasi sphuritāmāryāṃ papāṭha kaścitprasaṅgena // 491 //
anumaraṇe vyavasāyaṃ strīdharme kaḥ karoti savivekaḥ /
saṃsāramuktyupāyaṃ daṇḍagrahaṇaṃ vrataṃ muktvā // 492 //
śrutvā sundarasenaḥ sacivamavocadvyapetavaiklavyaḥ /
pratibodhitaṃ mano me dhīreṇānena yuktamupadiśatā // 493 //
kṣaṇadṛṣṭanaṣṭavallabhajanmajarāvyādhimaraṇaparibhūte /
āvartini saṃsāre kaḥ kuryādāgrahaṃ sumatiḥ // 494 //
yātu bhavān kusumapuraṃ vayamapyantyāśrame samāśrayaṇam /
aṅgīkurmo 'vidyāprahāṇasaṃsiddhaye vihitam // 495 //
so 'vadadabhijātajano bālyātprabhṛti tvayā ca na viyuktaḥ /
saṃnyasanabuddhimadhunā kathamujjhati viṣayaniḥspṛhaṃ suhṛdam // 496 //
evamiti so 'bhidhāya sthirayatiniyamaistapodhanairjuṣṭam /
guṇapālitena sahitaḥ sundaraseno jagāma vanam // 497 //
evaṃ bhavantu veśyāḥ svārthaikadṛśo vyapetasadbhāvāḥ /
abhilaṣitaviṣayasiddheḥ kā hānistadapi yuṣmākam // 498 //
ramaṇahṛdayānuvartanacaturacatuḥṣaṣṭikarmakuśalānām /
na spṛśati tattvacarcā paṇyavadhūnāṃ vidagdhacetāṃsi // 499 //
valitaplutacitragatisthitivegaiścodanānuvṛttyā ca /
rāgasparśena vinā viśati manaḥ sādināṃ turagaḥ // 500 //
gandho 'pi kutaḥ premṇaḥ parabhṛtahārītagṛhakapotānām /
ujjvalayantyasameṣuṃ virutaviśeṣaistathāpi te yūnām // 501 //
āhitayuktāhāryaḥ samyaksakalaprayoganiṣpattyā /
bhāvavihīno 'pi naṭaḥ sāmājikacittarañjanaṃ kurute // 502 //
ye 'pi dhanakṣayadoṣaṃ paśyanti jaḍā vilāsinīśleṣe /
praṣṭavyāste bhavatā kimakṛtakaśipuvyayā dārāḥ // 503 //
na ca lābha eka eva pravartane kāraṇaṃ manuṣyeṣu /
rāgādayo 'pi santi vaiśikaśāstrapraṇetṛbhiḥ kathitāḥ // 504 //
kā vā vibhūtirāptā sundarasenāttayā tapasvinyā /
tadvirahakuliśabhinnā mumoca yā jīvitaṃ kṣaṇārdhena // 505 //
uttamataruṇaprakṛtiḥ pulakādikasūcitānyatanusaktiḥ /
sphuṭasaṃnihitavibhāvo nivāryate kena śṛṅgāraḥ // 506 //
antaḥkaraṇavikāraṃ guruparijanasaṃkaṭe 'pi kulaṭānām /
budhyanti tadabhiyuktā bhrūbhaṅgāpāṅgamadhuradṛṣṭena // 507 //
anyā vihāya patigṛha mavicintitakulakalaṅkajanagarhāḥ /
rāgoparaktahṛdayā yānti digantaṃ manuṣya āsajya // 508 //
apamānaḥ pativihito guruparikaratīvratā gṛhe 'pi daurgatyam /
śīlakṣataye yāsāṃ tāsāmapi rāgato 'nyanarasaktiḥ // 509 //
yā apyacalitavṛttā bhartuḥ paricaraṇatatparāḥ pramadāḥ /
tā api rāgavimuktāstiṣṭhantyaucityamātreṇa // 510 //
tasmādastvabhigamanaṃ vividhanimittaṃ nivāryate kena /
nijaparapaṇyastrīṇāṃ rāgādhīnaṃ tu hṛdayanirvahaṇam // 511 //
evaṃvidhadṛṣṭāntairupapattiyutaistathedṛśairvākyaiḥ /
anyairapi cāṭupadairāvarjitamānasaṃ gamyam // 512 //
vihitasvāpavibodhaṃ kiñcitprakaṭīkṛtaśramaglānyā /
utpāditajṛmbhikayā parirabhya ghanaṃ niśāpagame // 513 //
vighaṭitapuṭamudradṛśā vilokya kakubhaḥ sadīrghaniḥśvāsam /
vaktavyamiti bhavatyā rajani khale kiṃ prayātāsi // 514 //
abalā viṣaheta kathaṃ dṛḍhaśaktimanuṣyaratirasaprasaram /
madanajanito 'nurāgo na vidadhyādyadi balādhānam // 515 //
dhanyā cakrāhvavadhūḥ priyatamasaṃghaṭanasamayasamprāptyā /
śaśinā viyujyamānā kumudvati kṣīṇapuṇyāsi // 516 //
vikasitasurabhimanoharasaṃsthānaṃ sarasakusumamaprāptam /
na karoti tathā pīḍāmāsvāditavicyutaṃ yathā bhṛṅgyāḥ // 517 //
vijñāpayāmyatastvāṃ racitāñjalimaulinā vidhāya natim /
paricārakajanamadhye gaṇanīyāhaṃ prasādena // 518 //
atha dīpitarāgāṅgairapahastitalābhadikkramopacitaiḥ /
mṛdubhiścittānugatairupacāraiḥ pātitasya viśvāse // 519 //
avalokito 'si lampaṭa kimapi vadan karṇasaṃnidhau nibhṛtam /
śaṅkarasenādhātryā adya mayā jālamārgeṇa // 520 //
mālatyā saha keliṃ vidadhāsi sakhī mameti na virodhaḥ /
yattu ciraṃ snigdhadṛśā paśyasi tāṃ tatra me śaṅkā // 521 //
tvāmāgatā na vīkṣitumanubadhya na yācitaḥ prayatnena /
āhūya vada kimarthaṃ tāmbūlaṃ grāhitā kamaladevī // 522 //
kañcukamapakarṣantyāḥ prakaṭībhavadaṃsakakṣakucapārśvam /
sābhiniveśaṃ dṛṣṭaṃ bhavatā kiṃ kundamālāyāḥ // 523 //
parihāsena gṛhītā yadyaṃśukapallave tvayā hīrā /
ācchoṭyāpakrāntā kiṃ māmavalokya pṛṣṭhataḥ sahasā // 524 //
vijñānena khyātāṃ kusumalatāṃ tvaṃ tu varṇayasyaniśam /
nṛtyantīṃ mṛgadevīṃ visphāritalocanaḥ paśyan // 525 //
kāraṇamatra na vedmyahamṛjupanthānaṃ prasiddhamutsṛjya /
vakreṇa yadeṣi sadā mādhavasenāgṛhāgreṇa // 526 //
iti serṣyopanyāsairanyaiścāmarmavedhilaghukopaiḥ /
praṇayaprabhavairvidite cchātodari rūḍharāgatve // 527 //
śrutiviśaye 'ntaritatanurjanitasthitirāyatākṣi saha mātrā /
paruṣagirā tvaṃ kuryā itthaṃ mithyāvacaḥkalaham // 528 //
akleśopanatadhanaḥ premaprahvo nirargalatyāgaḥ /
bhaṭṭānandasya suto nidhibhūto 'bhavyayā tvayā tyaktaḥ // 529 //
vyasanopahataviveko daivaikagatiḥ svadāravidveṣī /
māmavigaṇayya mūḍhe nirbhartsita eva keśavasvāmī // 530 //
agaṇitarājāpāyo 'vicchinnāyaḥ svabhāvatastyāgī /
kimupekṣito 'nurakto vāmadhiyā śaulkikādhyakṣaḥ // 531 //
pitureka eva putraścaturthavayaso gadābhibhūtasya /
draviṇavataḥ prabhurāto nirākṛto 'bhūtikāmayā so 'pi // 532 //
svakareṇa parityaktā tvayā vibhūtiḥ karomi kiṃ pāpā /
sarvabhareṇopanataṃ vasudevamanādareṇa paśyantyā // 533 //
puruṣāntarasaṃgharṣaprotsāhitacittavṛttiranapekṣam /
vasu visṛjati yo rabhasāttasya na vārtā tvayā pṛṣṭā // 534 //
citrādikalākuśalaḥ smaraśāstravicakṣaṇo vṛṣaprakṛtiḥ /
upakurvannapi sarvo vidveṣigaṇe tvayā kṣiptaḥ // 535 //
candravatīmābharaṇaṃ dattaṃ madhusūdanasya putreṇa /
paśyantī bibhrāṇāmayi rāgiṇi kiṃ na jihreṣi // 536 //
grāmotpattiraśeṣā praviśantī siṃharājaviniyogāt /
manmathasenāvāsaṃ laghayati te rūpasaubhāgyam // 537 //
āstāmaparo lābho heḍāvukanandisenatanayena /
śivadevyā upacāraḥ kriyate yastena paryāptam // 538 //
paśyedaṃ dhavalagṛhaṃ pāśupatācāryabhāvaśuddhena /
kāritamanaṅgadevyā ābharaṇaṃ pattanasya sakalasya // 539 //
āpaṇikārthasya kuto rājā labhate caturthamapi bhāgam /
haṭṭapatirāmasenaprasādato narmadā tamupabhuṅkte // 540 //
puṃstvakhyāpanakāmo na strī na pumān kila prabhusvāmī /
anubadhnannupahasitastvayā jaḍe svārthayogamanapekṣya // 541 //
vājīkaraṇaikamatirnaranāthānugraheṇa vikhyātaḥ /
pratyākhyātaḥ sa tathā ravidevaḥ kiṅkaratvamākāṅkṣan // 542 //
kiṃ kandarpakuṭumbe jāto 'sāvuta vaśīkaraṇayogam /
jānāti kamapi siddhaṃ yenākṛṣṭāsi sarvabhāvena // 543 //
bālye tāvadayogyā paścādapi vṛddhabhāvaparibhūtā /
tāruṇye rāgahṛtā yadi gaṇikā bhramatu tadbhikṣām // 544 //
upanaya bhāṇḍakametadyadarjitaṃ māmakena dehena /
vidadhāmi tīrthayātrāmāssva sukhaṃ preyasā sārdham // 545 //
āryajananinditānāṃ pāpaikarasaprakāśanārīṇām /
etāvāneva guṇo yadabhīṣṭasamāgamo nirāvaraṇaḥ // 546 //
no dhanalābho lābho lābhaḥ khalu vallabhena saṃsargaḥ /
akṣigatādarthāptirna bhavati manasaḥ pramodāya // 547 //
gāḍhānurāgabhinnaṃ tāruṇyarasāmṛtena saṃsiktam /
na bhajati sahṛdayahṛdayaṃ vibhavārjanasambhavā cintā // 548 //
lābhaḥ sa eva paramaḥ paryāptaṃ tena tena tṛptāsmi /
viniveśya yadutsaṅge nikṣipati mukhe mukhena tāmbūlam // 549 //
surataśramavārikaṇān parimārṣṭi nijāṃśukena gātreṣu /
yadurasi nidhāya vihasaṃstasya na mūlyaṃ vasundharā sakalā // 550 //
śithilitanijadāraratirmayi saktamanā ananyakartavyaḥ /
yadasau jitanalarūpastiraskṛtaṃ tena gāṇikyam // 551 //
bahukusumarasāsvādaṃ kurvāṇā madhukarī vidhiniyogāt /
īdṛkprasavaviśeṣaṃ labhate khalu yena bhavati kṛtakṛtyā // 552 //
ayi sarale tāvadimā upadeśagiro viśanti karṇāntaḥ /
yāvannāntarbhūtaṃ taccetasi māmakaṃ cetaḥ // 553 //
śrīrastu durgatirvā veśmani vāso mahatyaraṇye vā /
svarloke narake vā kiṃ bahunā tena me sārdham // 554 //
idamāste 'laṅkaraṇaṃ durjanani gṛhāṇa kiṃ mamaitena /
tenaiva bhūṣitāhaṃ guṇanidhinā bhaṭṭaputreṇa // 555 //
ucitasthānaniyuktānyapanīya vibhūṣaṇāni sāvegam /
evamabhidhāya yāsyasi mātuḥ purataḥ samutsṛjya // 556 //
iti rāgāndhaḥ śrutvā cetasi kurute kadā cidevamidam /
snehādhiṣṭhitamanasāmavidheyaṃ nāsti nārīṇām // 557 //
jananīṃ janmasthānaṃ bāndhavalokaṃ vasūni jīvaṃ ca /
puruṣaviśeṣāsaktāḥ sīmantinyastṛṇāya manyante // 558 //
raṇaśirasi hate vajre vajropamayantranirgatagrāvṇā /
prāṇānmumoca dayitā na mantravidhinā hṛtā nāma // 559 //
kālavaśenāyāsītpañcatvaṃ dākṣiṇātyamaṇikaṇṭhaḥ /
premopagatā veśyā tenaiva samaṃ jagāma bhasmatvam // 560 //
bhāskaravarmaṇi yāte suravasatiṃ vāritāpi bhūpatinā /
tadduḥkhamasahamānā praviveśa vilāsinī dahanam // 561 //
jvālākarālahutabhuji nagnācāryaḥ papāta narasiṃhaḥ /
tasminneva śarīraṃ nijamajuhocchokapīḍitā dāsī // 562 //
prītibharākrāntamatistridaśālayajīvikāṃ kramopanatām /
aṅgīcakāra muktvā jīhallā bhaṭṭaviṣṇumā mṛtyoḥ // 563 //
deśāntarādupete prasādamātreṇa vīkṣite vanite /
pādayugaṃ tatyajaturna vāmadevasya samiti nihatasya // 564 //
bhaṭṭakadambakatanaye yāte vasatiṃ paretanāthasya /
cakre dehatyāgaṃ raṇadevī vārayoṣitāṃ mukhyā // 565 //
asyāmeva nagaryāṃ draviṇamadāt kālasañcitamaśeṣam /
premṇākṛṣṭā gaṇikā bhaṭṭātmajanīlakaṇṭhāya // 566 //
iyamapi mayi vihitāsthā mātṛvacaḥśravaṇakaluṣitā kva gatā /
tyaktvābharaṇaṃ sarvaṃ pravijṛmbhitamanyusaṃvegā // 567 //
utsṛṣṭālaṅkaraṇāṃ pariśeṣitamātṛmuktaparivārām /
saṃtarpayāmi samprati sarvasvenāpi hariṇākṣīm // 568 //
gehena kiṃ prayojanamanyairapi bandhudāraparivāraiḥ /
saṃsāragrahakāraṇamekā khalu mālatī mama hi // 569 //
amṛtakarāvayavairiva ghaṭitā sā dṛḍhataraṃ pariṣvaktā /
ceto nayati samatvaṃ brahmaṇa ānandarūpasya // 570 //
āvirbhavadātmabhavakṣobhakṣatadhīratā ghanaṃ rabhasāt /
vigalitakucayugalāvṛtirāliṅgati mālatī dhanyam // 571 //
nirdayataroṣṭhakhaṇḍanasavyathahuṅkāramūrchitaṃ surate /
ahaheti vacastasyā apuṇyabhājo na śṛṇvanti // 572 //
smṛtijanmajanitavikṛtivrataticchannaṃ karoti saṃsāram /
āviddhasuratasaṅgaravimardasaṃkṣobhitā dayitā // 573 //
gāḍhatarāśliṣṭavapurbhajate kāntā pramodasammoham /
śithilīkṛtā tu kiṃ cidvividhavikāraṃ samucchvasiti // 574 //
santyanyā api satyaṃ puruṣocitakarmapaṇḍitāḥ pramadāḥ /
sṛṣṭā tu tayā niyataṃ viparītaratikriyāgoṣṭhī // 575 //
tantrīvādyaviśeṣānuddāmānanyajanmanastasyāḥ /
kuharitarecitakampitasampādananaipuṇaṃ karoti jaḍān // 576 //
lalitāṅgahārajṛmbhitavalitasmitavepanāni mālatyāḥ /
paśyañjahāti kāmaḥ ratimohanaceṣṭiteṣu bahumānam // 577 //
na grāmyaṃ parihasitaṃ nāvibhramataralito 'kṣivikṣepaḥ /
suratodyoganirodho dohadadānaṃ na puṣpabāṇasya // 578 //
nārthaparo lapanaraso na parāśayavedane 'vicakṣaṇatā /
nāsauṣṭhavaṃ prasaṅgenolvaṇaguṇakīrtaneṣu bhāratyāḥ // 579 //
nāparapuruṣaślāghā na tyāgaḥ kāladeśaveśasya /
vaidagdhyajanmabhūmergurujaghanabhareṇa mandayātāyāḥ // 580 //
cakrāhvapariṣvajanaṃ haṃsasamāśleṣanakulaparirambham /
pārāvatāvagūhanamācarati sumadhyamā yathāvasaram // 581 //
tadvakravacanahāsavyavahṛtihṛtamānasasya jāyante /
anukūlasundarā api bharaṇīyabharāya kevalaṃ dārāḥ // 582 //
sūcayati pṛthakkaraṇaṃ bhrātaṛṇāṃ vakti viṣamaśīlatvam /
vivṛṇoti gṛhavisaṃsthāmabhinandati pitṛkulasya guṇavattām // 583 //
anyasutapakṣapātaṃ kathayati mātustiraskaroti patim /
pārśvanimagnā jāyā mā yātu vimucya kārmukaṃ madanaḥ // 584 //
evaṃ kṛte 'pi sundari yadi tiṣṭhati nāyakaḥ prakṛtyaiva /
itthaṃ pathi parimoṣastvatsakhyā naipuṇena vaktavyaḥ // 585 //
gṛhakāryavyagratayā cittagrahaṇāya vā kulastrīṇām /
nāyāte bhavati sakhī prāvṛḍghanakaluṣite diśāṃ cakre // 586 //
pragrīvakaśayanagatā sphārībhavadātmasambhavavikārā /
tvanmārganihitanetrā gītāmanyena gītikāmaśṛṇot // 587 //
yadi jīvitena kṛtyaṃ saṃbhāvaya virahiṇi priyaṃ tūrṇam /
ghanarasitasya hi purataḥ kadalīdalakomalaḥ kuliśapātaḥ // 588 //
ākarṇya māmavādīddhanyāstā yuvatayaḥ sakhi kaṭhorāḥ /
yā viṣahante dīrghaṃ priyatamavirahānalāsaṅgam // 589 //
mama tu dināntarite 'pi preyasi labdhvā sahāyasāmagrīm /
vidadhāti makaraketana utkalikāvidhuritaṃ hṛdayam // 590 //
utkaṇṭhayati bhṛśaṃ māṃ samīraṇo bakulakusumagandhāḍhyaḥ /
pracyāvayanti dhairyānmadhuradhvanitaiḥ kalāpabhṛtaḥ // 591 //
sataḍinmiladbalākāmasitāmbudharāvalīṃ samudyantīm /
utsahate sā vīkṣitumaviralamāliṅgito yayā kāntaḥ // 592 //
svecchāgamanalaghutvaṃ bahulāpāyaṃ niśāsu panthānam /

rañjayati mano nitarāṃ kaladhautaniveśitaṃ ratnam // 594 //

vidadhānayā kubuddhiṃ rāgāndhe kimidamārabdham // 595 //
vacanaprapañcasāraṃ jāyāśritamanyadeśasambaddham /
puruṣamabhigantukāmā naveyamabhisārikā dṛṣṭā // 596 //
daradhautatilakaracanāṃ galadambhobindululitakeśāgrām /
timyattanulīnāvṛticaṇḍānilasalilapātakaṇṭakitām // 597 //
avibhāvitasamaviṣamapraskhaladaṅghriṃ sahāyakaralagnām /
purato 'dhvanaḥ pramāṇaṃ muhurmuhuḥ sādhvasena pṛcchantīm // 598 //

kiṃ premṇo 'yaṃ mahimā kimutānantyaṃ dhanapralobhasya /
kiṃ vānyataḥ pravṛttā praveśitā vātavarṣeṇa // 600 //
saṃnihitakalatrāṇāmanucitamiti bāhyalokasaṃvadanāt /
anyasminnudavasite visarjitāmiṣṭamālatīkena // 601 //
lokena hāsyamānāṃ bibhrāṇāṃ vāsasī jalaklinne /
rūpamadamutsṛjantīṃ vailakṣyavrīḍitena natavadanām // 602 //
paścāttāpagṛhītāṃ kaṇṭakadarbhāgrabhinnapādatalām /
asmadvacaḥ smarantīṃ drakṣyantyabhisārikāṃ sukarmāṇaḥ // 603 //

caurahatakā vrajantīṃ vidrāvitarakṣiṇaḥ sakhīṃ mumuṣuḥ // 604 //

vaṇigidamupetya vakṣyati sahāyaparicodito bhavatīm // 605 //
pūrvādattasyopari muktāhārasya kedarāstriṃśat /
paricārikayānītā anyānapi mṛgayate vyayasya kṛte // 606 //
yattu ghanasārakuṅkumacandanadhūpādi muktakaṃ dattam /
tatsampuṭake likhitaṃ śṛṇu piṇḍanikāṃ karomi te purataḥ // 607 //

riktaṃ bhāṇḍasthānaṃ sāmpratamiti yācanā kriyate // 608 //
evaṃvādini tasminkiṃ cillajjānatā kṣaṇaṃ sthitvā /
 praśritayā vācā vācyaḥ savailakṣyam // 609 //
hārastavaiva tiṣṭhatu madhyasthasthāpitena mūlyena /
śeṣaṃ tato yadanyattaddivasaiḥ saṃpādayiṣyāmi // 610 //
iyamapi kapaṭagrathanā ṣaṇḍasamā cettad īdṛgabhidheyam /
āśaṅkante 'niṣṭaṃ kātarahṛdayā hi yoṣitaḥ prāyaḥ // 611 //
apaṭuśarīre svāmini vijñaptā bhagavatī mayā gatvā /
bhavatu nirāmayadeho jīvitanāthastava prasādena // 612 //
sampannavāñchitāhaṃ balyupahāreṇa pūjayiṣyāmi /
sāmagrīviraheṇa tu na vitīrṇā tatra me manasi śūkā // 613 //
asmin vyarthībhūte riktīkṛtaśīrṇaveśmano dāham /
utpādya mandagāmini sarvavināśaḥ prakāśamupaneyaḥ // 614 //
snigdhatvamalaṃ buddhvā sahabhojanaśayanavasanaliṅgena /
ebhirupāyadravyairvāntaviriktastvayā kāryaḥ // 615 //
vārdhuṣikakadarthanayā bhogadhvaṃsātsahāyavacanairvā /
avadhārite 'pi nipuṇaṃ varagātri viluptasāratve // 616 //
paruṣavaconirdhāraṇamāyatyāmīhitopaghātīti /
yatnādamī vidheyā gamyasya vimokṣaṇopāyāḥ // 617 //
pṛthagāsananirdeśaḥ pratyutthānādikeṣu śaithilyam /
sāsūyasopahāsā ālāpā marmavedhi parihasitam // 618 //
tatpratipakṣaślāghā tadadhikaguṇarāgakīrtanāvṛttiḥ /
vadati priyamābhīkṣṇyaṃ bahupralāpitvadūṣaṇākhyānam // 619 //
vacanāntaropaghātaistatprastutasaṃkathāsamākṣepaḥ /
tadvyavahārajugupsā savyapadeśastadantikatyāgaḥ // 620 //
vyājena kālaharaṇaṃ svāpāvasare vivartanaṃ śayane /
nidrābhibhavakhyāpanamudvegaḥ sammukhīkaraṇe // 621 //
guhyasparśanirodhaḥ svabhāvasaṃsthā ratābhiyogeṣu /
cumbati vadanavikūṇanamāliṅgati kaṭhinagātrasaṃkocaḥ // 622 //
asahiṣṇutvaṃ prahaṇanakararuhadaśanakṣatiprasaṅgeṣu /
dīrgharate nirvedaḥ svapihīti vaco 'bhiyojake bhūyaḥ // 623 //
tadaśaktāvanubandho vaidagdhyavikāśane tathā hāsaḥ /
rātryavasānaspṛhayā punaḥ punaryāmikapraśnaḥ // 624 //
niḥsaraṇaṃ vāsagṛhāduṣasi samutthāya talpatastvarayā /
sarabhasamudīrayantyā niśā prabhātā prabhāteti // 625 //
ubhayecchayā pravṛttaṃ nirupādhi prema bhavati ramaṇīyam /
anyonyasamāsaktau saṃsthānamivābhijātamaṇihemnoḥ // 626 //
yastvekāśrayarāgaḥ paribhavadaurbalyadainyanāśānām /
sa nidānamasaṃdigdhaṃ sītāṃ prati daśamukhasyeva // 627 //
yāni haranti manāṃsi smitavīkṣitajalpitāni raktānām /
tānyeva viraktānāṃ pratibhānti vivartitānīva // 628 //
vidadhātu kimapi kathamapi nigṛhyamāṇā muhūrtamāsiṣye /
iti yatra manaḥ strīṇāṃ tatrāpi rate ramanti paśutulyāḥ // 629 //
yatra na madanavikārāḥ sadbhāvasamarpaṇaṃ na gātrāṇām /
tasminnujjhitabhāve paśukarmaṇi paśava eva rajyante // 630 //
avadhīraṇayopahataḥ pratidivasaṃ hīyamānasadbhāvaḥ /
abhimānavānmanuṣyo yoṣitamūḍhāmapi tyajati // 631 //
sākṣinikocaṃ sakhyāḥ pāṇitalaṃ pāṇinā samāhatya /
yaṃ naramupahasati strī dadātu tasmai mahī randhram // 632 //
puruṣāntaraguṇakīrtanamanyoddeśena cātmano nindām /
śṛṇvannapi yaḥ svasthaḥ svastho 'sau kālapāśabaddho 'pi // 633 //
avagatyābhiprāyaṃ svāminyāḥ parijano 'pi yaṃ puruṣam /
vyavaharati tiraskurvaṃstasya na mūlyaṃ varāṭikāḥ pañca // 634 //
tattvātattvasamutthavyavahṛtayoryo 'ntaraṃ na jānāti /
sthānaṃ bhavati sa paśupatirapasaṃśayamardhacandralābhasya // 635 //
kramagalitagauravāṃśo riktatayā lāghavaṃ parāpatitaḥ /
aprāptaparicchedataṭaḥ plavate 'sau yuvatisarasi kumanuṣyaḥ // 636 //
yatnena kapaṭaghaṭitāñśṛṅgārajñāpanārthamanubhāvān /
ratiśilpajīvikābhirmūḍhāstattvena gṛhṇanti // 637 //
yā dhanahāryā nāryo nirmaryādāḥ svakāryatātparyāḥ /
tābhirapi samīhante bata mandāḥ saṃgatamajaryam // 638 //
aparokṣadhano gamyaḥ śrīmānapi nānyatheti nirdiṣṭam /
kandarpaśāstrakāraiḥ kutaḥ kathā luptavibhaveṣu // 639 //
vyāsamunināpi gītaṃ dvāveva narādhamau mano dahataḥ /
yo 'nāḍhyaḥ kāmayate kupyati yaścāprabhutvayukto 'pi // 640 //
kṣīṇadravye dehini dārā api nādareṇa vartante /
kimutādānaikarasāḥ śarīrapaṇavṛttayo dāsyaḥ // 641 //
aviditaheyādeyāstiryañco 'pi tyajanti pītarasam /
kusumaṃ kimu kāryavido veśyā naramāttasarvasvam // 642 //
utpādayati sadāno rāgaṃ rāgātmako yathābhyadhikam /
nirdāno 'pi sadā no niḥsaṃdehaṃ tathaiva manujanmā // 643 //
yadatītaṃ tadatītaṃ bhāvini lābhe 'pi nātibahumānaḥ /
tatkālahastanipatitamaniyatapuṃsāṃ mude vittam // 644 //
pīḍitamadhu madhujālaṃ tucchībhūtaṃ ca manmathāgrastam /
muñcanti madanaśeṣaṃ kṣudrāśca prakaṭarāmāśca // 645 //
ekaḥ krīṇātyadya prāto bhavitā tathāparaḥ kretā /
anyavaśe kṣaṇamekaṃ na vikrayaḥ śāśvato 'sti veśyānām // 646 //
saṃdarśitaparamārthaṃ bhrūkṣepakaṭākṣadṛṣṭahasitādi /
śṛṇvantu ye sakarṇāstatkṛtamanyatra saṃkrāntam // 647 //
yadi nāma nirākaraṇe na samarthā chinnakāryabandhe 'pi /
kā cinmahānubhāvā boddhavyaṃ tadapi cetanāvadbhiḥ // 648 //
tenārthenopakṛtaṃ tayāpi tasya svadehadānena /
taccātītaṃ samprati nirarthakaḥ śuṣkaśṛṅgāraḥ // 649 //
avadhīraṇā rasāyanamapamāno bhavati yasya parituṣṭyai /
yogyo 'sau puruṣakharaḥ kharataranirbhartsanoktilakuṭānām // 650 //
dīpajvālālalane vrajataḥ khalu nirvṛtiṃ tayostviyān bhedaḥ /
prathamā snehena vinā tathāparā snehayogena // 651 //
dharmaḥ kāmādabhinavaguṇavanniḥsvasya madanarogavataḥ /
artho 'rthavato 'bhigamātkāmaḥ samaratanaropabhogena // 652 //
yastu na dharmaprāptyai nārthāya na kāmasādhanopāyaḥ /
sa pumānsaccaritadhanaiḥ paryanuyuktaḥ kimācaṣṭe // 653 //
kāmodvegagṛhītaṃ dhūrtairupahasyamānaśṛṅgāram /
dāridryahataṃ yauvanamabudhānāṃ kevalaṃ vipade // 654 //
vyapagatakoṣe rāgiṇi yāti layaṃ pānamātralābhahṛtā /
kṣudrā madhukarikābje na tu gaṇikā cintitasvārthā // 655 //
yāsāṃ kāryāpekṣā sakaṭākṣanirīkṣaṇe 'pi veśyānām /
darśanamātrakṣubhitairvañcyante tāḥ kathaṃ puruṣaiḥ // 656 //
kleśāya durgatānāṃ mānastutigātrabhaṅgavinyāsam /
gaṇikābhinayacatuṣṭayamākṛṣṭyai svāpateyapuṣṭānām // 657 //
kiṃ dhakṣyati bhūyo 'pi jvalanastaṃ tādṛśaṃ kulāṅgāram /
yo dahyate 'virāmaṃ viraktadāsītiraskāraiḥ // 658 //
gṛhametadīśvarāṇāṃ kāntāraṃ duṣpraveśamanyeṣām /
pūtkṛtamidamudbhujayā na mālatī kāmasattradānaparā // 659 //
iti coditagṛhaceṭīnigaditakaṭukākṣarānyakṛtalakṣyāḥ /
ākarṇayato vāco daivopahatasya tasya marmabhidaḥ // 660 //
evamabhidhīyamāno no budhyati yadi paśurnarākāraḥ /
tadidaṃ sundari vācyaḥ praśritavacasā tvayā kāmī // 661 //
prīyata eva tavopari hṛdayaṃ me kiṃ tu gurujanādhīnā /
mātṛvacotikramaṇaṃ na samarthā saṃvidhātumaham // 662 //
arhasi tāvadatastvaṃ gantumitaḥ katipayānyapi dināni /
punarapi bhavataiva samaṃ bhoktavyaṃ jīvalokasukham // 663 //
nirvāsite tu tasmin yaḥ kāmī pūrvamujjhito bhuktvā /
tasya prāptavibhūteryuktiriyaṃ bhinnasaṃdhāne // 664 //
upavanalīlāviharaṇahāvojjvalamañjulasya saha tena /
varṇanamitivṛttasya smarajavikārāśca vīkṣite tasmin // 665 //
idamupavanamatidhanyaṃ nirbharamāliṅgitaṃ surabhilakṣmyā /
matskandhārpitapāṇirbabhrāma sa yatra jīvitādhīśaḥ // 666 //
sakhya ito bhramarakulatrāsitayā priyatamo mayā sahasā /
vakrībhavatpayodharamupagūḍho dhīrasītkāram // 667 //
raṇadindindiravṛnde kūjatkalakaṇṭharāvaramaṇīye /
atrātimuktakagṛhe marudīraṇavidhutakusumasaṃchanne // 668 //
mayi jātādhikarāgo balavati madane sahāyasāmagryā /
kāntaḥ pallavaśayane no tṛptimagādviviktakāryeṣu // 669 //
preṅkholanavitaraṇayuktyā vidhyan pārśvayornakhairdhūrtaḥ /
cakre māṃ madanamayīṃ vratatipreṅkhāmimāṃ samārūḍhām // 670 //
spṛhaṇīyo 'yamaśokaḥ spṛṣṭo yadvallabhena hastena /
asmadavataṃsakārthaṃ nūtanadalapallavān vicārayatā // 671 //
asminsahakāratale tasyotsaṅge salīlamāsīnā /
aśṛṇavamahamiti vācaḥ paśyantī vilasitāni taruṇānām // 672 //
utthāpaya mānarase dayitaṃ caraṇāgranipatitaṃ tūrṇam /
atyākṛṣṭaṃ truṭyati sudṛḍhamapi premabandhanaṃ mūḍhe // 673 //
tiṣṭhannapi yātasamaḥ kiṃ tena nivāritena sakhi paśunā /
yāmīti niṣprakampaṃ viniḥsṛtā yasya mādhave vāṇī // 674 //
āyuḥsāraṃ yauvanamṛtusāraḥ kusumasāyakavayasyaḥ /
sundari jīvitasāro ratibhogasukhāmṛtāsvādaḥ // 675 //
ramyaṃ kusumastabakaṃ kuru me priya kaiṅkirātamavataṃsam /
tiṣṭhatu vā kimanena pratyagramaśokakisalayaṃ cāru // 676 //
āstāmāstāmetatprāpaya māṃ sinduvāramabhirāmam /
nahi nahi rājati sutarāṃ cūtadrumamañjarī karṇe // 677 //
dhiktāruṇyamakāntaṃ dhikkāntaṃ yauvanena rahitaṃ ca /
dhiktaddvayamapi manmathasāmarthyavikāsitaṃ vinā suratam // 678 //
janito 'pyaparādhaśatairvāme tasmiṃściraprarūḍho 'pi /
avagatamadhunā sakhyo na vasantamatītya vartate mānaḥ // 679 //
varṣaśatasya sa sāraḥ kālalavaḥ prathamamelakasthānam /
sacakitamāgacchantī sotkalikā yatra dṛśyate ramaṇī // 680 //
kiṃ nirmito 'si dhātrā navo 'paraḥ kimu vasantaguṇa eṣaḥ /
kusumaśarapūrṇatūṇaḥ kimutābhavadanya eva kandarpaḥ // 681 //
no paśyasi yadi kakubhaḥ pracurojjvalasurabhikusumaramaṇīyāḥ /
parabhṛtakūjitamiśrānna śṛṇoṣi yadi dvirephajhaṅkārān // 682 //
gandhaṃ yadi no labhase vāsitadigvyoma sumanasāṃ hṛdyam /
anubhavasi yadi sparśaṃ no śītaladākṣiṇātyapavanasya // 683 //
rasanendriyaikaśeṣaḥ parasaṃcāryo janena paribhūtaḥ /
nārhasi tato 'pi muktvā nijāśrayaṃ gantumanyato nitarām // 684 //
asminsarasi salīlaṃ karayantraviniryadambudhārābhiḥ /
dayitena tāḍitāhaṃ mayāpyasāvāhato mṛṇālikayā // 685 //
punarantarjalamagno māmupagamyāvibhāvitaḥ sahasā /
uccikṣepa sahāsaṃ hāsitasaṃnihitaparivāraḥ // 686 //
saṃsaktārdrāvaraṇaṃ jaghanaṃ naḥ paśyatastadā tasya /
prathamākāṅkṣākūtaṃ bheje sambhogaśṛṅgāraḥ // 687 //
kālapradeśaveśavyāpārasthitiviśeṣaghaṭanābhiḥ /
cirarūḍho 'pi hi yūnāṃ navatvamupanīyate rāgaḥ // 688 //
sādaramarpayato 'bjaṃ gotraskhalanāparādhinastasya /
sakhyaḥ smarāmi sahasā vilakṣatāṃ kliṣṭahasitasya // 689 //
pratyagranakhavraṇite stanāntare kṣipati locane spṛhayā /
preyasi hrītācchādanamakaravamahamabjinīpatram // 690 //
kṣiptvātarkitamambho garbhitanalinīpalāśapuṭamārāt /
āhatayā yadvikṛtaṃ svasthadhiyā tanna śakyate kartum // 691 //
suśliṣṭo hāvavidhirmadanālasagātrajṛmbhitaṃ lalitam /
gūḍhasthānaprakaṭanamaṅgulivisphoṭanaṃ smitaṃ subhagam // 692 //
nīvivimocanabandhau muhurmuhuḥ keśapāśaviśleṣaḥ /
svādharadaśanagrahaṇaṃ bālakaparicumbanaṃ ratotsukatā // 693 //
sākāṅkṣitaṃ kṣipantyāstaralāyatalocane muhuḥ kāmye /
uddiśya tadvayasyakamiti śokagrastavarṇagiraḥ // 694 //
ekībhāvaṃ gatayorjalapayasormitracetasośca tathā /
vyatirekakṛtau śaktirhaṃsānāṃ durjanānāṃ ca // 695 //
atipelavamatiparimitavarṇaṃ laghutaramudāharati śaṇṭhaḥ /
paramārthataḥ sa hṛdayaṃ vahati punaḥ kālakūṭaghaṭitamiva // 696 //
yena tadā māmūce parijanamutsārya vivṛtanaṭamanyuḥ /
darśitahitasvarūpaḥ parapīḍākaraṇapaṇḍitaḥ prakhalaḥ // 697 //
aviditaguṇāntarāṇāṃ ko doṣaḥ prāntadeśavāsānām /
svādhīnakuṅkumā api yadvidadhati bahumatiṃ nīle // 698 //
kva mahītalarambhā tvaṃ nyakkṛtacandraprabhā svadeharucā /
citralatā kva varākī nīcairupasevitā voṭā // 699 //
yasyārthe na vigaṇitāḥ prahvātmāno mahādhanāḥ kulajāḥ /
so 'dya hṛdayena tasyāṃ tvayi tiṣṭhati bāhyavṛttena // 700 //
tāmeva samācaraṇāṃ sadasadbhāvapravartitāṃ nipuṇaiḥ /
vindanti tatra kuśalāḥ snehavirūkṣaprabhedena // 701 //
tava tu virūḍhapremṇastatkarmavivecanaṃ manovṛttiḥ /
nāruhatīti mayaivaṃ niveditaṃ pāricityena // 702 //
iti durjanāhiniḥsṛtavāgviṣadūṣitasamastavapuṣo me /
īrṣyārujaḥ pravṛddhāścirarūḍhapraṇayakhaṇḍanaprabhavāḥ // 703 //
laghuhṛdayatayā yasmāddurbhāṣitavajrapātavihatānām /
vaktṛviśeṣavitarko na spṛśati prāyaśo manaḥ strīṇām // 704 //
priyamapi vadan durātmā kṣipati vipatsāgare duruttare /
āsādya prāṇabhṛto mṛtaye parileḍhi jihvayā khaḍgaḥ // 705 //
hitamadhurākṣaravāṇīvyavahāramanupraviśya tallīnam /
saralā durāśayānāmupaghātaṃ phalata eva vindanti // 706 //
parasaṃtāpavinodo yatrāhani na prayāti niṣpattim /
antarmanā asādhurna gaṇayati tadāyuṣo madhye // 707 //
divasāntānabhinandati bahu manute teṣu janmano lābham /
ye yānti duṣṭabuddheḥ paropatāpābhiyogena // 708 //
vikasitavadanaḥ prakhalaḥ protphullavilocano yathā bhramati /
manye tathā na jātaḥ sadahitakaraṇaśramo vandhyaḥ // 709 //
śaṭhamṛgayuḥ kusṛtiśarairajñātapratividhānasādhumṛgān /
abhyastalakṣavedho nighnanna pariśramaṃ vrajati // 710 //
anukūlavarapurandhriṣu puruṣāṇāṃ baddhamūlarāgāṇām /
nayati mano duḥśīlaḥ kusumāstro hīnapātreṣu // 711 //
sāvaraṇaṃ vrajato 'nyāṃ kautukadṛṣṭyā prasaṅgato dayitān /
buddhvāpi vidagdhadhiyo vartante nāṭyadharmeṇa // 712 //
satyaṃ premaṇi vṛddhe vyathayati hṛdayaṃ manāgapi skhalitam /
avadhṛtanijamāhātmyāstadapi na dhīrā vimuhyanti // 713 //
svacchandaṃ pibatu rasaṃ bhrāntvā bhrāntvā vanāni kusumeṣu /
anubhūtaguṇaviśeṣaḥ punareṣyati mālatīṃ madhupaḥ // 714 //
mālatyā guṇavattāṃ samyagno vetti madhukarastāvat /
anubhavameti na yāvatsumanontarasaṃgamāsvāde // 715 //
komalamānakadarthāṃ bhajamāno bhajati dīptatāmadhikām /
saṃcālyamānadāruḥ pāvaka iva suprabhaḥ snehaḥ // 716 //
yaḥ punaratikopānalasaṃtāpavaśena dūramākṛṣṭaḥ /
kācamaṇiḥ sa khalu yathā pariṇāme khaṇḍakhaṇḍamupayāti // 717 //
vetanalobhādbahavaḥ sevyante sauṣṭhavena pañcajanāḥ /
viśrāmyati yatra manaḥ sa tu duṣprāpaḥ sahasreṣu // 718 //
manvādimunivarairapi kālatrayavedibhiḥ sudurjñeyam /
tatsukṛtaṃ yasya phalaṃ rabhasāgatavallabhāśleṣaḥ // 719 //
yāte 'pi nayanamārgaṃ preyasi yasyāḥ smṛtirvyalīkeṣu /
manye tāṃ prati niyataṃ kuṇṭhitaśarapañcakaḥ madanaḥ // 720 //
jīvyata eva kathaṃ ciddhigvṛttimimāṃ mahadbhiravagītām /
vijahāti yanna gaṇikā tadvāñchitaramaṇalābhalobhena // 721 //
kaṇṭakinaḥ kaṭukarasān karīrakhadirādiviṭapatarugulmān /
upabhuñjānā karabhī daivādāpnoti madhuramadhujālam // 722 //
kā śrīrapraṇayivaśā kā vilasitayo manobhavavihīnāḥ /
ko dharmo nirupaśamaḥ kiṃ saukhyaṃ vallabhena rahitānām // 723 //
svācchandyaphalaṃ bālyaṃ tāruṇyaṃ rucitasuratabhogaphalam /
sthaviratvamupaśamaphalaṃ parahitasampādanaṃ ca janmaphalam // 724 //
abhidadhatīmidamālīmavakarṇya gṛhītayeva bhūtena /
yauvanasukhena sārdhaṃ mayaiva yūyaṃ paricchinnāḥ // 725 //
adhunānutāpapāvakamadhyagatā pacyamānasarvāṅgī /
niṣphalajanmaprāptirjīvāmyucchvāsamātreṇa // 726 //
sthāneṣu yeṣu yuṣmatsaṃgatayā krīḍitaṃ ciraṃ dhṛtyā /
tāni khalu vīkṣamāṇā bhavāmi kaṇṭhasthitaprāṇā // 727 //
anyavaśena visaṃjñā kṛtabhūṣā yantrasūtrasaṃcārā /
dārumayīva pratimā vidadhāmi viḍambanā bahvīḥ // 728 //
yadi nāmodarabharaṇaprāptyai kurute 'nyapuṣpasaṃśleṣam /
tadapi na puṣṭirbhṛṅgyā apibantyā āravindamakarandam // 729 //
āstāmaparo lokaḥ krīḍāpekṣī parāpadi prītaḥ /
vyasanārṇave patantī na vāritā parijanenāsmi // 730 //
kiṃ vā bahubhiḥ kathitaiḥ samprati niyame tu niyamitā buddhiḥ /
sthāsyāmi saṃniyuktā bhavadgṛhe preṣyakāryeṣu // 731 //
iti netrādivikārairvaśamupanītaṃ pralīnadhairyāṅgam /
māragrahābhibhūtaṃ parimṛṣṭaprāṅnirākṛtismaraṇam // 732 //
prādurbhūtariraṃsaṃ kṣaṇe kṣaṇe jatrudeśagatadṛṣṭim /
pakvāmramiva vimokṣyasi pūrvavadācūṣya subhru niḥśeṣam // 733 //
svaśarīrāmiṣadigdhaṃ vakrasmitadṛṣṭipātavāgbaḍiśam /

jaghanacapalā tathāryā sa piṅgalaste kathaṃ tulyaḥ // 780 //

sadrūpakajātiguṇairnātha tvaṃ gāmalaṅkuruṣe // 783 //
anyaiva varṇanaiṣā bhavatsu lokottarā sthitā kāpi /

paragṛhavināśapiśunāḥ subhagaṃmanyābhirūpyakṛtadarpāḥ /
kṛkalāsatulyarāgā bhavanti yuṣmadvidhā eva // 854 //
anabhīṣṭavyavahāraprabhavaśucā pīḍitākṣarā ittham /
sopālambhā vijane dhanyāḥ śṛṇvanti bandhakīvācaḥ // 855 //
parataruṇīsadbhāvasnehārpitanayanabhāgadṛṣṭasya /
veśyāracitavilāsāḥ kathitāḥ purataḥ purāṇatṛṇatulyāḥ // 856 //
upanamati ratimahotsava ārādhitadevatāviśeṣāṇām /
vacanamapi premārdraṃ svairiṇyāḥ śravaṇameti puṇyavatām // 857 //
kā gaṇanā viṣayarate puṃsi varāke parāṅganā spṛhayā /
vyājena vīkṣamāṇā dhyānadhiyāṃ spṛśati majjānam // 858 //
śirasā racitāñjalayo dadhati nideśaṃ triviṣṭape gaṇikāḥ /
paradārarasākṛṣṭastathāpi bheje śacīpatirahalyām // 859 //
apsarasaḥ kiṃ na vaśā vaidagdhyavatāṃ ca kiṃ na dhaureyaḥ /
yena cakārāsaktiṃ govindo gopadāreṣu // 860 //
trailokyagatā veśyāḥ svādhīnā yātudhānanāthasya /
tadapi jahāra kalatraṃ daśarathatanayasya rāmasya // 861 //
atha mañjaryā jananī nijapakṣasamarthane kṛtotsāhā /
ākṣeptumācacakṣe nṛpasutasacivāśritaṃ vākyam // 862 //
ghaṭayuvatiṣu pragalbho nāgarikādarśanena hṛtapuṃstvaḥ /
grāmoṣito 'vidagdho nindati gaṇikāṃ bhavadvidho 'vaśyam // 863 //
nārdrayati manaḥ puṃsāmavagāhitamīnaketuśāstrāṇām /
nakhadaśanakṣatihīnaṃ jīvatpatibandhakīsuratam // 864 //
sthāpaya ghaṭakaṃ tāvatkuru bhūmitale tṛṇaiḥ samāstaraṇam /
suratopakrama īdṛkprāyo grāmīṇataruṇamithunānām // 865 //
bahalośīraviliptasthitajūṭakakāṇamallikāmālyaḥ /
pāmaranāryā dṛṣṭaḥ smaro 'hamiti manyate viṭo grāmyaḥ // 866 //
gṛhakarmakṛtāyāsaprasvinnāṃ salilakāryaniryātām /
upapatirupaiti harṣaṃ niśāgame pāmarīṃ prāpya // 867 //
kūpāvatāritaghaṭāyā nāryāḥ kāṣṭhanihitacaraṇāyāḥ /
valitagrīvaṃ vīkṣitamunnamayati mānasaṃ yūnaḥ // 868 //
lagno 'si yatra gātre kathamapi daivena devayātrāyām /
adyāpi tanna muñcati pulakodgamakaṇṭakaṃ tasyāḥ // 869 //
uccetuṃ karpāsaṃ praviṣṭayā gahanavāṭikāṃ śūnyām /
khaṅkāritena saṃjñā kṛtā tayā tvaṃ tu vetsi no mūrkhaḥ // 870 //
āliṅgitamusalāyāstvayyeva niviṣṭacakṣuṣastasyāḥ /
āvṛttyā bhramati puro jātaḥ khalu śālikhaṇḍane vighnaḥ // 871 //
tvāṃ vilvakaṃ kṣipantaṃ pārśvasthaiḥ stūyamānasāmarthyam /
gṛhakartavyaṃ tyaktvā sāpaśyad dvārarandhreṇa // 872 //
tvayi mārganikaṭavartinyavivecitakhedayā tayā subhaga /
pratyāsannagṛheṣvapi kṛtaḥ prasahya smarāturo lokaḥ // 873 //
iti caturadūtikoditavardhitasaubhāgyagarvapūrṇasya /
ūrmisahasrollalitaṃ bhavati mano grāmaṣiṅgasya // 874 //
vinivārya taṃ pravartitavākyavikāsaṃ natottamāṅgena /
śrīsiṃhabhaṭasya sutaṃ samuvāca vaco 'tha nartakācāryaḥ // 875 //
nāyakabhūmau bharataḥ kuśīlavāḥ kohalādayo munayaḥ /
apsarasaḥ strīnāṭye gāndharvaḥ kamalajanmanastanayaḥ // 876 //
suṣirasvaraprayogapratipādanapaṇḍitaḥ mataṅgamuniḥ /
yadi rañjayanti hṛdayaṃ bhavato bhūmispṛśāṃ kutaḥ śaktiḥ // 877 //
abhyadhikaṃ dhṛṣṭatvaṃ prāyeṇa hi śilpajīvināṃ bhavati /
āśritanartakavṛtterviśeṣato vijitaraṅgasya // 878 //
vijñāpayāmyatastvāṃ nirmitanāṭye prajāsṛjā sadṛśam /
avalokayāṅkamekaṃ mā bhavatu mama śramo vandhyaḥ // 879 //
iti kathayannarabhartuḥ putreṇa sa codito bhruvonnatayā /
racite sakalātodye niyojayāmāsa sūtradhṛtam // 880 //
vāṃśikadattasthānaka udgrāhitabhinnapañcame samyak /
prāveśikyā dhruvayā dvipadīgrahaṇāntare 'viśatsūtrī // 881 //
utsāhabhāvayuktaḥ sāmājikahṛdayarañjanaṃ kurvan /
kavinaipuṇavatseśvaracaritasvavidheyadākṣyasāmagryā // 882 //
aṣṭakalāparimāṇāṃ dhruvāṃ parikramya tālalayayuktām /
āhūya naṭīṃ kṛtvā tayā samaṃ svagṛhakāryasaṃlāpam // 883 //
sūcitapātrāgamanaḥ kiṃ ciddattvā padāni lalitāni /
niścakrāma gṛhiṇyā sārdhaṃ niḥsaraṇagītena // 884 //
āśritya kathodghātaṃ praviveśa tataḥ savismayo 'mātyaḥ /
durghaṭasaṃghaṭanena kṣitināthasyodayena muditaśca // 885 //
prāsādamāruhantaṃ kusumāyudhaparvacarcarīṃ draṣṭum /
nirdiśya vatsanāthaṃ samanantarakāryasiddhaye niragāt // 886 //
atha viśati sma narendraḥ prāsādagataḥ samaṃ vayasyena /
avalokayanpramodaṃ pramuditacetāḥ svasaukhyasampattyā // 887 //
vismayabhāvākṛṣṭaḥ protphullavilocane tato visṛjan /
nṛtyati paurajanaughe provāca vayasya paśya paśyeti // 888 //
tulyaśiśutaruṇavṛddhaṃ samaguptāguptayuvatipariceṣṭam /
agaṇitavācyāvācyaṃ krīḍati janatā pravṛddhaharṣarasā // 889 //
piṣṭātakapiñjaritaṃ racitolbaṇavividhakusumaniryūham /
gātrāyāsasamutthitabahuniḥśvāsaprakīrṇapadagītam // 890 //
tūryaravavyāmiśritakaratālairudbhujaṃ pranṛtyantam /
muhurapi jātaskhalanaṃ saṃdarśitadārḍhyasauṣṭhavaṃ sthaviram // 891 //
astu vasantaḥ satataṃ svādhīnābhīṣṭajanasamāśleṣaḥ /
iti gāyantī rabhasādāliṅgati madavaśāttaruṇī // 892 //
krīḍantyāśayarahitaṃ śṛṅgakasalilena tāḍitastaruṇaḥ /
sīmantinyā gaṇayati duṣṭātmā subhagamātmānam // 893 //
bhagne lajjāsetau parvāvasareṇa kulavadhūvadanāt /
aślīloktijalaughe niryāte kena vāryate prasaraḥ // 894 //
tulyavyāpāragirāṃ lalanānāṃ devanaprasaktānām /
āryānāryāvagamaṃ vadanāvṛtijālikā kurute // 895 //
atha sahacaranirdiṣṭe madaskhalaccaraṇavighaṭitābhinayam /
vāsavadattāprahite nṛtyantyau viviśatuśceṭyau // 896 //
darśitasarojavartanamātrābhinaye śare 'bhinetavye /
vidadhāne vīradṛśāvāyudhamātraṃ samāśritya // 897 //
tadvalitanayanavṛttiḥ kautukahṛtamānaso narādhipatiḥ /
nijagāda nirbharamaho krīḍitamanayorvilāsinyoḥ // 898 //
karapīḍanopamardavyatikarasamaye kadarthyamāno 'pi /
stanamaṇḍale sthito 'haṃ tvaṃ punarākṛṣya kutra citkṣiptaḥ // 899 //
adhunāntarayasi māmiti kopādiva vāravāṇamabhirāmam /
bahucitrapadanyāsairvalgantyā hanti hāra ucchalitaḥ // 900 //
cūtalatā dhammillasthānacyutaśekharaṃ dadhau ślāghyam /
adhṛta patanniryūhāṃ na tveṣā madanikā veṇīm // 901 //
stanabhārāvanatasya pratanormadhyasya nāsti te 'pekṣā /
itthamiva pādalagnau krīḍantyā nūpurau rasataḥ // 902 //
vahati sma yaṃ nitambaṃ kathamapi kṛcchreṇa mandasaṃcārā /
kalayati taṃ tūlalaghuṃ jayati manojanmano mahimā // 903 //
udayanasamanujñāto narnarti vasantako 'pi muditātmā /
hāsyatrapābhirāmaṃ carcaritālena tanmadhye // 904 //
dhīroddhatalalitapadaiḥ krīḍitvā te cirāya naranātham /
pradyotasya sutāyāḥ saṃdeśakamūcatuḥ samupagamya // 905 //
ādiśati deva devītyardhokte te salajjamanyonyam /
avalokya mukhaṃ na hi na hi vijñāpayati praṇamya vinayena // 906 //
makaradhvajasya pūjāṃ tvatpādasarojasaṃnidhau kartum /
pṛthivīmaṇḍalamaṇḍana samīhate me manovṛttiḥ // 907 //
priyaratibhogo madano dayitavasanto janasya manasi vasan /
bhāvena bhavān pūjyo lokasthityā tu kusumaśarapāṇiḥ // 908 //
iti dattvā saṃdeśaṃ prakṛtivayaḥkālasamucitaṃ bhrāntvā /
te madamadanāviṣṭe babhūvaturjavanikāntarite // 909 //
apanītatiraskariṇī tato 'bhavannṛpasutā samaṃ ceṭyā /
aviditaratnāvalyā pūjocitavastuhastayānugatā // 910 //
atha dṛṣṭvā sāgarikāṃ pramāditāṃ parijanasya ninditvā /
kāñcanamālāmavadannṛpamahiṣī jātasaṃkṣobhā // 911 //
preṣaya kanyāmenāmavarodhaṃ tvaṃ gṛhāṇa kusumādi /
yāvanna bhavati viṣaye vīkṣaṇayorbhūmināthasya // 912 //
upagamya tataśceṭī tāmavadattvaṃ kimarthamāyātā /
medhāvinīṃ vimucya vraja tasminmā vilambasva // 913 //
vihite devyādeśe manasīdaṃ sannidhāya sā tasthau /
vihagī susaṅgatāyā haste nihitā manobhavasaparyām // 914 //
avalokayāmi tāvattirohitā sinduvāraviṭapena /
tātāntaḥpurikābhiryathārcyate kiṃ tatheha no veti // 915 //
piṇḍīkṛtamiva rāgaṃ hṛcchayamiva labdhavigrahotkarṣam /
samupetya vatsarājaṃ jagāda sā jayatu deva iti // 916 //
paribhuktamapi navatvaṃ śṛṅgāraṃ madanaparvaṇā nītam /
bhajamāno bhajamānāṃ svāgatavacasābhinandya tāmūce // 917 //
śarvavilocanapāvakadāhābhyadhikāṃ manobhavo manye /
prāpsyati tava karasaṅgatisukhavirahasamutthitāṃ pīḍām // 918 //
atha manmathamabhyarcya kṣitināthaṃ tadanu samadhikaṃ tasyām /
paramāṃ mudaṃ vahantyāṃ vigrahavanmadanamanasi kanyāyām // 919 //
śṛṅgārarasasamudraṃ sotkalikaṃ nipatite tathā nṛpatau /
tāramadhurasphuṭārthaṃ nagnācāryaḥ papāṭha nepathye // 920 //
nayanānandamakhaṇḍitamaṇḍalamabhirāmamamṛtaraśmimiva /
sāyantana āsthāne kṣitipatayastasthurudayanaṃ draṣṭum // 921 //
uccārite 'tha nāmni tridaśamatau tatkṣaṇaṃ vyapetāyām /
utpannavismayaratirnidadhe narabharturātmajā hṛdaye // 922 //
ayamudayanaḥ sa rājā tātaḥ satkṛtya māṃ dadau yasmai /
hanta parapreṣaṇamapi na niṣphalaṃ sāmprataṃ jātam // 923 //
yāvanna vetti kaścittāvaditastvaritameva niryāmi /
iti kathamapi nāyakataḥ kṛṣṭvā dṛśamutsasarja raṅgabhuvam // 924 //
kandarpamahamahotsavahṛtahṛdayairnāvadhārito 'smābhiḥ /
saṃdhyātikramakālaḥ paśya tvaṃ priyavayasyaka tathā hi // 925 //
udayataṭāntaritamiyaṃ prācī sūcayati diṅniśānātham /
paripāṇḍunā mukhena priyamiva hṛdayasthitaṃ ramaṇī // 926 //
devi tvanmukhapadmaḥ padmān vidadhāti paśya vicchāyān /
alayo 'pi lajjitā iva śanaiḥ śanaistadudareṣu līyante // 927 //
evamabhidhāya citraiścaraṇanyāsaiḥ parikramaṃ kṛtvā /
naiṣkrāmikyā dhruvayā viniryayau nāyako 'pi saha sarvaiḥ // 928 //
aṅke jātasamāptau gītātodyadhvanau ca viśrānte /
prekṣaṇakaguṇagrahaṇaṃ nṛpasūnuḥ pravavṛte kartum // 929 //
nāṭyaprayogatattvaṃ matayo na viśanti mādṛśāṃ prāyaḥ /
vāhanayajanapadātigrāmādikakāryadattahṛdayānām // 930 //
āste likhito grāmo gṛhāṇa taṃ satpradeśabahubhūmim /
vāsaya tatrāvāsaṃ bhavasi tataḥ ṭhakkuro divasaiḥ // 931 //
kṛtajīvanasaṃstho 'pi tvamapi kimarthaṃ karoṣi vijñaptim /
arpaya vā yadi necchasi kuru sthitiṃ hastadānena // 932 //
na ca pattayo na saptirna ca poṣyajanastathāpyasaṃtuṣṭaḥ /
labhamāno 'pi sa dāyaṃ cirantanatvābhimānena // 933 //
vijñaptikonmukhatvaṃ dūrata evāvadhāritaṃ bhavataḥ /
tūṣṇīṃ kriyatāmasmācchroṣyasi kāryaṃ pratīhārāt // 934 //
yūyaṃ kuṭumbamadhye kva gamyate gotraputrasāmānyam /
ādāya saṃvibhāgaṃ svagṛha iva sthīyatāṃ yathāsaukhyam // 935 //
abhyantaravyayārthaṃ na vilabdho yo mayā mahodraṅgaḥ /
tatrāpi te 'nubandho no jāne kiṃ karomīti // 936 //
prathamatarameva kalpitamanalpaphalajīvanaṃ pradeśastham /
adyāpi te na jātaṃ niyogināṃ paśya mantharatām // 937 //
evaṃprāyairanudinalābhodayamohakāribhirvacanaiḥ /
phalaśūnyairanujīvī pratāritaḥ kaḥ kiyatkālam // 938 //
etadviṣaye naipuṇamatra tu bhūmijñatāṃ samāśritya /
mukharatayā kathayāmo jaḍamatisāmājikocitaṃ kiṃ cit // 939 //
saptāśrayaḥ ṣaḍātmā śārīrastripramāṇaparimāṇaḥ /
sattvādhikyājjyeṣṭho vyastasamastaistribhirviniṣpādyaḥ // 940 //
sukumārāviddhakriya uparañjakarañjito vividhavṛttiḥ /
ādeyaheyamadhyairbhāvaiḥ saṃpāditaḥ prayogo 'yam // 941 //
gambhīramadhuraśabdaṃ parirakṣitagītavividhabhaṅgayutam /
darśayato vaicitryaṃ na bhraṣṭo vādakasya layakālaḥ // 942 //
aparityaktasthānakarasakākuvyañjitasphuṭārthapadam /
abhirāmāviśrāntaṃ paṭhitaṃ niravadyamakhilabhāṣāsu // 943 //
niyamitadīpanagamanaṃ drutamadhyavilambitālasaṃyuktam /

prakṛtiviśeṣāvasthāpratipādakaveṣaracanasāmagryā /
anukaraṇamabhyatītaṃ siddhidvayasampadādhāram // 945 //
bharatasutairupadiṣṭaṃ kṣitipatinaghuṣāvarodhanārīṇām /

suśliṣṭasandhibandhaṃ satpātrasuvarṇayojitaṃ sutarām /

saṃgrāmādanapasṛtiḥ prekṣābhijñā subhāṣitābhiratiḥ /

ārabdhakathācchedakamākheṭakavarṇanaṃ cakre // 950 //
calalakṣyavedhakauśalamaśvaprajave sthirāsanābhyasanam /
bhūmivibhāgajñānaṃ bhavanti mṛgayābhiyogena // 951 //
vahati javena turaṅge nibiḍasthitapādakaṭakapādāgraḥ /
tiryakpraṇihitakāyo nimnonnatamagrato bhuvaḥ paśyan // 952 //

dāvānalasaṃtāpānniryāntaṃ gahanavīrudho 'bhimukham /
yo niruṇaddhi sa vandyaḥ sūkaramekaprahāreṇa // 956 //
ghanakakṣodarasuptaṃ samupetya svairamakṛtapadaśabdam /
vyādhavara eva kurute nirjīvaṃ helayā śaśakam // 957 //
iti vidadhati saiṃhabhaṭāvākheṭakaśaktilāghavaślāghām /
hṛdayāgatāmagāyatprasaṅgato gītikāmaparaḥ // 958 //
āstāṃ vyāpārarasaḥ pravartitā saṃkathāpi mṛgayāyāṃ /
antarayati tanmanasāmāhārādikriyocitaṃ kālam // 959 //
avadhārya gītikārthaṃ dānaṃ prati dhananiyuktamabhidhāya /
uttasthau samarabhaṭaḥ mañjarikāṃ samavalokayanpremṇā // 960 //

mañjarikākṛṣṭamanā abhidadhyau sacivasaṃnidhāvevam // 961 //
bhrūbhaṅgasmitavīkṣitamṛduvakravacoṅgahāragamaneṣu /
kusumapraharaṇa eko yugapadvihitāśrayaḥ kathaṃ tasyāḥ // 962 //
sundopasundanāśaḥ phalamātmabhuvastilottamāsṛṣṭeḥ /
janamṛtaye tāṃ sṛjatā kiṃ dṛṣṭaṃ surahitaṃ tena // 963 //
sumanobhiḥ parikaritā mṛgaśāvakataralacakṣuṣastasyāḥ /
kāmocitaphalaheturdehabhṛtāṃ dīrghikā veṇī // 964 //

rūpaṃ yauvanacitritamanaṅgavikṛtāni nāṭyadīptāni /
śamināmapi śamagarvaṃ muṣṇantyavikalpitaṃ tasyāḥ // 970 //

tannirmāṇe jātā lāvaṇyamayāḥ kaṇā vidheraṇavaḥ // 973 //
āsādya samucchrāyaṃ tasyāḥ stanayugalamavihataprasaram /
kṣapayati yajjanamevaṃ kaḥ sprakṣyati tadvivekavān patitam // 974 //
sa kathaṃ na spṛhaṇīyo viṣayarataistannitambavinyāsaḥ /
śāntātmanāpi vihitaṃ viśvasṛjā gauravaṃ yasya // 975 //
smaraṇādyasyotpattiḥ sumanasa iṣavo 'balāśrayā śaktiḥ /
so 'pi vyaṅgaḥ praharati dhāturaho citramācaritam // 976 //
tiṣṭhantvanye dṛṣṭvā sāraṃ jagatāṃ tadaṅganāratnam /
naṣṭapaṭhanāvadhāno bhavati brahmā sanirvedaḥ // 977 //
yadi paśyati tāṃ śarvastadapararāmāsamāgamādvimukhaḥ /
nindati mūrdhani somaṃ smarāgnisaṃdhukṣaṇaṃ śarīraṃ ca // 978 //
keśava iha saṃnihitaḥ sāpi manohārirūpasampannā /
tadvakṣaścyavanabhuvaṃ kathamujjhati saindhavīśaṅkām // 979 //
udayati na paṇḍitānāṃ kathamātmani kautukaṃ gajendragatiḥ /
yannavavayasāṃ puṃsāṃ vinā kriyāyogamupasargaḥ // 980 //
śrutikuvalayamīkṣaṇatāṃ kuvalayatāṃ vā vilocanaṃ yāyāt /
hariṇadṛśo yadi na syātkanakojjvalakesaraṃ madhye // 981 //
lalanāstadatulyatayā puruṣā api tadupabhogaviraheṇa /
gacchanti śoṣamaniśaṃ prakṛtidvayavarjitāḥ svasthāḥ // 982 //
durvṛttayorna vṛttaṃ ślāghāspadameti tatpayodharayoḥ /
yau dattvāmalamūrtiṃ madhye hāraṃ janakṣayaṃ kurutaḥ // 983 //
bhūmaṇḍale 'tra sakale nātaḥ paramaparamadbhutaṃ kiṃ cit /
no jātā yadapārthā kṛśodarī dhārtarāṣṭrayātāpi // 984 //
kṛśa eṣa madhyadeśastanvyā nāhāryamaṇḍanaṃ voḍhum /
śakta iti kṛtaṃ vidhinā romāvalibhūṣaṇaṃ sahajam // 985 //
sākampo 'dhara īkṣaṇayugalasyādhīratā bhruvorbhaṅgaḥ /
tanvaṅgyā balamīdṛgjayati jagattadapi niḥśeṣam // 986 //
vahatu nitambaḥ sthūlo raśanāṃ hāraṃ ca kucayugaṃ pīnam /
tadbāhumṛṇālikayoḥ sāpāyaṃ kaṭakayojanamayuktam // 987 //
bahalopāyābhijñā guṇaviṣaye satatamāhitaprītiḥ /
balinaḥ sthāpayati vaśe karabhorurvigraheṇa mṛdunaiva // 988 //
iti tatstutimukharamukhe rājasute makaraketanākulite /
samupāgatā pragalbhā mañjarikācoditā dūtī // 989 //
sā sapraṇatiḥ purataḥ sumanastāmbūlapaṭalakaṃ nidadhe /
vyajñāpayacca tadanu svāvasare sahacarīkāryam // 990 //
muraripunābhisaroruhamavataṃsīkartumīhate mūḍhā /
nakṣatrarājamaṇḍalamicchati viyataḥ samādātum // 991 //
niścetanābhikāṅkṣati pīyūṣaṃ tridivasadmanāmaśanam /
abhilaṣati śayanamuṣṇe navacandanapallavāstaraṇe // 992 //
vidadhāti pārijātakasumanoniryūhadhāraṇaśraddhām /
durvyavasitā jighṛkṣati nārāyaṇavakṣaso ratnam // 993 //
aniyatapuruṣaspṛśyāḥ pāpā vayamanyathā kva hīnakulāḥ /
kva ca yūyamindrakalpā analpamanaso guṇābharaṇāḥ // 994 //
duṣprakṛteḥ prakṛtiriyaṃ tasya tu dagdhātmajanmanaḥ kāpi /
agaṇitayuktāyukto lagayati ceto yadasthāne // 995 //
yā hasati sarojavatīṃ rasānvitā sahajarāgarakteti /
dhyānadhiya ātmavṛttiṃ nindatyekatra puruṣa āsaktām // 996 //
snigdheti nābhinandati janmaśatenāpi sarpiṣo dhārām /
pañcākṣadyūtagatiṃ nānarthakaramaṇasaṃgatāṃ stauti // 997 //
na stauti candanalatāṃ bhujaṅgapariveṣṭitāṃ rasārdreti /
na śṛṇoti kīrtyamānāṃ svapneṣvapi madanamūrchitāṃ matsīm // 998 //
vidveṣṭi karaṇamadhye rasanāṃ tāmbūlarāgayukteti /
śaṃsati matiṃ mumukṣoraviśiṣṭāṃ śaśavṛṣāśvapuruṣeṣu // 999 //
no bahu manute rambhāṃ nalakūbaramabhisṛteti kāmārtā /
garhati ca devagaṇikāmanuraktāmurvaśīṃ purūravasi // 1000 //
harati mano no hriyate rañjayati na rajyate kadā cidapi /
gṛhṇāti citracaritairupakṛtibhirgṛhyate na bahvībhiḥ // 1001 //
premamayīvābhāti prema tu nāmnaiva kevalaṃ vetti /
kaṇṭakitā bhavati rate ratabhogasukhaṃ śṛṇoti lokāttu // 1002 //
kurute viviktacāṭūñśilpaviśeṣeṇa na tu rasāveśāt /
anabhijñā madanarujāmākalpakavedanāṃ samāvahati // 1003 //
bālaivārjavarahitā sphuratīśvarametya candralekheva /
hṛtadhanapatimāhātmyā pravṛttiriva rakṣasāṃ patyuḥ // 1004 //
naranātha kiṃ bravīmi tripurāntakanayanadāhadagdho 'pi /
duḥsādhyasādhanagrahamutsṛjati na pāpakusumāstraḥ // 1005 //
tvaddarśanāvakāśaṃ samprāpya yato durātmanā tena /
cirasambhṛtakopena prārabdhā sāpi hantumiṣudhāraiḥ // 1006 //
avahelayaiva bhavatā saṃspṛṣṭā yena vetradaṇḍena /
jātaḥ sa eva tasyā ananyabhavamārgaṇaḥ prathamaḥ // 1007 //
vijñānārjitadarpo nibhṛtaṃ hasitaḥ samānaśilpābhiḥ /
tvayi saktadṛśaḥ sakhyā visaṃṣṭhule nāṭyanirmāṇe // 1008 //
avadhīryācāryaruṣaṃ bharatoditadoṣakaraṇasaṃbhūtām /
vistāritaḥ prayogastvadavasthitivāñchayā tanvyā // 1009 //
bhagne 'pi prekṣaṇake tadanantarabhūmikāśrayāvasthāḥ /
gṛha eva niravasānaṃ vitanoti na nāṭyadharmeṇa // 1010 //
dhyāyata ekaṃ puruṣaṃ paramātmavidaḥ śaśaṃsa yā na purā /
tānanukurute saiva dhyāyantī tvāṃ mahāpuruṣam // 1011 //
gatamevamevamāsitamālokitamevamevamālapitam /
iti vismṛtānyakāryā smarati kṛśāṅgī tvadīyalīlānām // 1012 //
nalakūbaro varāko ratiramaṇo ramaṇa eva kiṃ tena /
aniruddho 'pi na buddho vidagdhavihitāsu suratagoṣṭhīṣu // 1013 //
na jayanto 'nantaguṇo na kumāraḥ mārakarmaṇo 'bāhyaḥ /
kena samatāṃ nayāmastamiti sakhī vahati mānasaṃ kleśam // 1014 //
āgatamāgacchantaṃ purataḥ pārśve prasannamatha kupitam /
paśyati bhavantamekaṃ saṃkalpaniveśitaṃ bālā // 1015 //
rucyaḥ kānto hṛdyaḥ subhagaḥ sukhado manoharo ramaṇaḥ /
iṣṭaḥ svāmī dayitaḥ prāṇeśaḥ kelikaraṇanipuṇa iti // 1016 //
muktānyasamārambhā varatanuranupaplutena cittena /
japati samīhitasiddhyai tvaddvādaśanāmakaṃ mahāstotram // 1017 //
tāmeva gaccha yasyāmāsajya vilambito 'si gatalajja /
velāmiyatīmalamalametairadhunā śaṭhānunayaiḥ // 1018 //
vakṣyāmi sāparādhaṃ krodhasphuradadharamañcitabhrūkam /
iti vidadhāti sumadhyā hṛdayena manorathāvṛttim // 1019 //
utsahate na draṣṭuṃ pratibimbitamānanaṃ kutaḥ śaśinam /
kā saṃkathā mṛṇāle kṣipati bhujau sarvato vyathitā // 1020 //
dūre kadalīdaṇḍā ūrvorapi na sahate samāśleṣam /
karasamparkādvimukhī viśrāmyati pallaveṣviti viruddham // 1021 //
ayi mañjari saiva tvaṃ vidagdhajanamaṇḍitā purī saiva /
kusumāyudhaḥ sa eva vyasanaṃ kuta etadāyātam // 1022 //
yasyāḥ kāmaḥ kṛpaṇo rāgākṛṣṭistṛṇolapaprakhyā /
sāpi gatā bhūmimimāṃ jīvantyā nekṣyate kimiha // 1023 //
abhiyogaśikṣitānāmaśikṣitānāṃ ca madanaceṣṭānām /
sutanu viśeṣagrahaṇe sāmarthyaṃ tadvidāmeva // 1024 //
vyathayannapi sacchāyaḥ parijanacintākaro 'pi ramaṇīyaḥ /
ādhatte tvayi lakṣmīmabhinavarāgāśrayo rāgaḥ // 1025 //
ekaḥ sa eva jāto bhuvane 'sminnasamasāyakaspardhī /
tena śaśibimbaphalake sujanmanā lekhitaṃ nijaṃ nāma // 1026 //
pādastena salīlaṃ vinyastaḥ subhagamānināṃ mūrdhni /
saubhāgyayaśaḥkusumaṃ dhanapatisūnoḥ kadarthitaṃ tena // 1027 //
naravañcanapaṭubuddhiḥ saṃpāditakapaṭacāṭusaṃghaṭanā /
tvamapi vilāsini nītā gatimiyatīṃ yena subhagena // 1028 //
tadvada tasya sthānaṃ yatāmahe kāryasādhanāyālam /
kurvata eva hi yatnaṃ bhiṣagjanāḥ kṛcchrasādhyaroge 'pi // 1029 //
iti gadite sakhyā sā tadabhimukhaṃ cakṣuṣī samunmīlya /
vitarati kṛcchreṇa cirādvibhāvitaṃ kliṣṭahuṅkāram // 1030 //
kā puruṣārthasamīhā dyotayataḥ śarvarīṃ śaśāṅkasya /
tarpayatāṃ bhuvamakhilāṃ salilamucāṃ ko 'bhikāṅkṣito lābhaḥ // 1031 //
maṇḍayituṃ viyadudayati puruhūtadhanurvinaiva phalavāñchām /
anapekṣitātmakāryaḥ parahitakaraṇagrahaḥ satāṃ sahajaḥ // 1032 //
prāyeṇa yannidānaṃ tadasevanamupaśamāya rogāṇām /
smaramāndyaṃ tu yadutthaṃ tadeva khalu bheṣajaṃ yatastasya // 1033 //
tena spṛhayati sutanustvatpādayugābjareṇusaṃgataye /
āśīrviṣayopete sambhogasukhodaye tu nākāṅkṣā // 1034 //
pramadamupaiti mayūrī paramaṃ śabdena vārivāhasya /
animiṣavilokitena prāpnoti jhaṣī kṛtārthatāmeva // 1035 //
na vṛthāstutimukharatayā na ca yuṣmallobhanābhiyogena /
vidadhāmi tadguṇākhyāṃ svarūpamātraprasaṅgena // 1036 //
sadbhāvabaddhamūle smitadṛṣṭibhrūvikārapallavite /
sevante hṛdyarasāṃ rāgatarau mañjarīṃ dhanyāḥ // 1037 //
tiṣṭhatu tadaṅgasaṅgo vilokitā yena jhagiti varagātrī /
tasyānyo yuvatijanaḥ pratibhāti manuṣyarūpeṇa // 1038 //
sakṛdapi yairanubhūtastattanuparirambhasukharasāsvādaḥ /
viddhi narādhipa teṣāṃ dūrībhūtaṃ prajākāryam // 1039 //
āsthā kā khalu tasyā viṣayagrahadurbaleṣu puruṣeṣu /
yasyā vilāsajālakapatitaḥ śakunāyate kapilaḥ // 1040 //
dagdhvāpi punardagdho nūnamanaṅgaḥ hareṇa tāṃ tanvīm /
dṛṣṭvāpi yena tiṣṭhasi nirākulaḥ svasthavṛttena // 1041 //
atha viratoktau tasyāmullāsitamānase ca nṛpasūnau /
kaścidagāyadgītiṃ smṛtisaṃgatimāgatāṃ prasaṅgena // 1042 //
anyonyagāḍharāgaprabalīkṛtacittajanmanoryūnoḥ /
kālātyayo manāgapi samāgamānandavighnakaraḥ // 1043 //
śrutvā siṃhabhaṭasutaḥ priyāpriyāṃ prītimān smitaprathamam /
nijagāda cārubhāṣiṇi gītikayā samayasammataṃ kathitam // 1044 //
abhinandya sā tatheti prayayau pramadāvatī nijaṃ bhavanam /
akarocca viditakāryāṃ yukte 'vasare manoramāṃ gaṇikām // 1045 //
atha sā kṛtasaṃkalpā satvaramādāya ruciravicchittim /
āsādya nṛpaniśāntaṃ viveśa saṃcārikāsahitā // 1046 //
vihitanamaskṛtirāsanamadhitaṣṭhau nāyakena nirdiṣṭam /
pṛṣṭe ca dehakuśale vinayānvitamabhyadhāddūtī // 1047 //
śrīmannadya śreyaḥsampannā gurujanāśiṣo 'śeṣāḥ /
adya madanaḥ prasanno bhāgyacayairadya pariṇataṃ phalataḥ // 1048 //
adya jananī prasūtā saubhāgyaguṇodayo 'dya niṣṇātaḥ /
tvayi vitarati sasnehaṃ nirāmayapraśnabhāratīṃ tasyāḥ // 1049 //
utkalikākulamanasāmudriktariraṃsayābhibhūtānām /
audāsīnyaṃ bhajatāṃ samāgatā bhavati nālikā yūnām // 1050 //
dhṛtasumanaḥśaradhanuṣā sahāyavāṃstiṣṭha dayitayā sārdham /
yāmo vayaṃ na rājati vijanasthitamithunasaṃnidhāvaparaḥ // 1051 //
eṣā nṛtyaśrāntā madanenāyāsitātisukumārā /
tvamapi ratisamaraśūraḥ svargabhuvaḥ santu kuśalāya // 1052 //
yāvadyāvadaśaktiṃ prathayati lalanā hi mohanākrāntā /
tāvattāvatpuṃsāmutsāhaḥ pallavān samutsṛjati // 1053 //
iti śūnyīkṛtaveśmani harati śanaiḥ sahajamaṃśukaṃ tasmin /
darśitasādhvasalajjā jagāda sā kiṃ karoṣīti // 1054 //
ayi mugdhe tatkriyate puruṣārthacatuṣṭayasya yatsāram /
iti nigaditasasmeraḥ smaravidhurita ātatāna ratikalaham // 1055 //

tadyanmayopadiṣṭaṃ kāmijanārthāptikāraṇaṃ tena /
mahatīṃ samṛddhimeṣyasi kāmukalokāhṛtena vittena // 1057 //
ityupadeśaśravaṇaprabodhatuṣṭā jagāma dhāma svam /
mālatyapagatamohā vikarālāpādavandanāṃ kṛtvā // 1058 //
