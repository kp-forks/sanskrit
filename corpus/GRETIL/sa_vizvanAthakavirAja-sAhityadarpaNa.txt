#Text:Sāhityadarpaṇa
#Data entry:Jan Brzezinski
#Edition:Satyavrata Simha, 9th edition. Varanasi: Chowkhamba Vidya Bhavan, 1992. (Vidya Bhavan Sanskrit Granthamala, 29).


||śrīḥ sāhityadarpaṇaḥ
||kārikāmātraḥ
||(1) prathamaḥ prakāśaḥ
śaradindusundararuciś cetasi sā me girāṃ devī /
apahṛtya tamaḥ santatam arthān akhilān prakāśayatu // 1.1 //
caturvargaphalaprāptiḥ sukhād alpadhiyām api /
kāvyād eva yatas tena tatsvarūpaṃ nirūpyate // 1.2 //
kāvyaṃ rasātmakaṃ kāvyaṃ doṣās tasyāpakarṣakāḥ /
utkarṣahetavaḥ proktā guṇālaṅkārarītayaḥ // 1.3 //
||iti sāhityadarpaṇe kāvyasvarūpanirūpaṇo nāma prathamaḥ paricchedaḥ
||(2) dvitīyaḥ prakāśaḥ
vākyaṃ syād yogyatākāṅkṣāsattiyuktaḥ padoccayaḥ /
vākyoccayo mahāvākyam itthaṃ vākyaṃ dvidhā matam // 2.1 //
varṇāḥ padaṃ prayogārhān anvitaikārthabodhakāḥ /
artho vācyaś ca lakṣyaś ca vyaṅgaś ceti tridhā matā // 2.2 //
vācyo 'rtho 'bhidhayā bodhyo lakṣyo lakṣaṇayā mataḥ /
vyaṅgyo vyañjanayā tāḥ syus tisraḥ śabdasya śaktayaḥ // 2.3 //
tatra saṅketitārthasya bodhanād agrimā matā /
saṅketo gṛhyate jātau guṇadravyakriyāsu ca // 2.4 //
mukhyārthabādhe tadyukto yayānyo 'rthaḥ pratīyate /
rūḍheḥ prayojanād vāpi lakṣaṇā śaktir arpitā // 2.5 //
mukhyārthasyetarākṣepo vākyārthe 'nvayasiddhaye /
syād ātmano 'py upādānād eṣopādānalakṣaṇā // 2.6 //
arpaṇaṃ svasya vākyārthe parasyānvayasiddhaye /
upalakṣaṇahetutvād eṣā lakṣaṇalakṣaṇā // 2.7 //
āropādhyavasānābhyāṃ pratyekaṃ tā api dvidhā /
viṣayāsyānigīrṇasyānyatādātmyapratītikṛt // 2.8 //
sāropā syān nigīrṇasya matā sādhyavasānikā /
sādṛśyetarasaṃbandhāḥ śuddhās tāḥ sakalā api // 2.9 //
sādṛśyāt tu matā gauṇyas tena ṣoḍaśabhedikā /
vyaṅgyasya gūḍhāgūḍhatvād dvidhā syuḥ phalalakṣaṇā // 2.10 //
dharmidharmagatatvena phalasyaitā api dvidhā /
tad evaṃ lakṣaṇābhedāś catvāriṃśan matā budhaiḥ // 2.11 //
padavākyagatatvena pratyekaṃ tā api dvidhā /
viratāsvabhidhādyāsu yathārtho bodhyate 'paraḥ // 2.12 //
sā vṛttir vyañjanā nāma śabdasyārthādikasya ca /
abhidhālakṣaṇāmūlā śabdasya vyañjanā dvidhā // 2.13 //
anekārthasya śabdasya saṃyogādyair niyantrite /
ekatrārthe 'nyadhīhetur vyañjanā sābhidhāśrayā // 2.14 //
lakṣaṇopāsyate yasya kṛte tat tu prayojanam /
yayā pratyākhyāyate sā syād vyañjanā lakṣaṇāśrayā // 2.15 //
vaktṛbodhavyavākyānām anyasannidhivācyayoḥ /
prastāvadeśakālānāṃ kākoś ceṣṭādikasya ca // 2.16 //
vaiśiṣṭyād anyam arthaṃ yā bodhayet sārthasaṃbhavā /
traividhyād iyam arthānāṃ pratyekaṃ trividhā matā // 2.17 //
śabdabodhyo vyanakty arthaḥ śabdo 'py arthāntarāśrayaḥ /
ekasya vyañjakatve tad anyasya sahakāritā // 2.18 //
abhidhāditrayopādhivaiśiṣṭyāt trividho mataḥ /
śabdo 'pi vācakas tadval lakṣako vyañjakas tathā // 2.19 //
tātparyākhyāṃ vṛttim āhuḥ padārthānvayabodhane /
tātparyārthaṃ tadarthaṃ ca vākyaṃ tadbodhakaṃ pare // 2.20 //
||iti sāhityadarpaṇe vākyasvarūpanirūpaṇo nāma dvitīyaḥ paricchedaḥ
||(3) tṛtīyaḥ paricchedaḥ
vibhāvenānubhāvena vyaktaḥ saṃcāriṇā tathā /
rasatām eti ratyādiḥ sthāyī bhāvaḥ sacetasām // 3.1 //
sattvodrekād akhaṇḍasvarprakāśānandacinmayaḥ /
vedyāntarasparśaśūnyo brahmāsvādasahodaraḥ // 3.2 //
lokottaracamatkāraprāṇaḥ kaiścit pramātṛbhiḥ /
svākāravad abhinnatvenāyam āsvādyate rasaḥ // 3.3 //
karuṇādāv api rase jāyate yatparaṃ sukham /
sacetasām anubhavaḥ pramāṇaṃ tatra kevalam // 3.4 //
kiṃ ca teṣu yadā duḥkhaṃ na ko 'pi syāt tadunmukhaḥ /
tathā rāmāyaṇādīnāṃ bhavitā duḥkhahetutā // 3.5 //
hetutvaṃ śokaharṣāder gatebhyo lokasaṃśrayāt /
śokaharṣādayo loke jāyantāṃ nāma laukikāḥ // 3.6 //
alaukikavibhāvatvaṃ prāptebhyaḥ kāvyasaṃśrayāt /
sukhaṃ sañjāyate tebhyaḥ sarvebhyo 'pīti kā kṣatiḥ // 3.7 //
aśrupātādayas tadvad drutatvāc cetaso matāḥ /
na jāyate tadāsvādo vinā ratyādivāsanām // 3.8 //
vyāpāro 'sti vibhāvāder nāmnā sādhāraṇīkṛtiḥ /
tatprabhāveṇa yasyāsan pāthodhiplavanādayaḥ // 3.9 //
pramātā tadabhedena svātmānaṃ pratipadyate /
utsādādisamudbodhaḥ sādhāraṇyābhimānataḥ // 3.10 //
nṛṇām api samudrādilaṅghanādau na duṣyati /
sādhāraṇyena ratyādir api tadvat pratīyate // 3.11 //
parasya na parasyeti mameti na mameti ca /
tadāsvāde vibhāvādeḥ paricchedo na vidyate // 3.12 //
vibhāvanādivyāpāram alaukikam upeyuṣām /
alaukikatvam eteṣāṃ bhūṣaṇaṃ na tu dūṣaṇam // 3.13 //
kāraṇakāryasañcārirūpā api hi lokataḥ /
rasodbodhe vibhāvādyāḥ kāraṇāny eva te matāḥ // 3.14 //
pratīyamānaḥ prathamaṃ pratyekaṃ hetur ucyate /
tataḥ saṃvalitaḥ sarvo vibhāvādiḥ sacetasām // 3.15 //
prapāṇakarasanyāyāc carvyamāṇo raso bhavet /
sadbhāvaś ced vibhāvāder dvayor ekasya vā bhavet // 3.16 //
jhaṭity anyasam ākṣepe tadā doṣo na vidyate /
pārimityyāl laukikatvāt sāntarāyatayā tathā // 3.17 //
anukāryasya ratyāder udbodho na raso bhavet /
śikṣābhyāsādimātreṇa rāghavādeḥ svarūpatām // 3.18 //
darśayan nartako naiva rasasyāsvādako bhavet /
kāvyārthabhāvenāyam api sabhyapadāspadam // 3.19 //
nāyaṃ jñāpyaḥ svasattāyāṃ pratīty avyabhicārataḥ /
yasmād eṣa vibhāvādisamūhālambanātmakaḥ // 3.20 //
tasmān na kāryaḥ no nityaḥ pūrvasaṃvedanojjhitaḥ /
asaṃvedanakāle hi na bhāvo 'py asya vidyate // 3.21 //
nāpi bhaviṣyan sākṣād ānandamayasvaprakāśarūpatvāt /
kāryajñāpyavilakṣaṇabhāvān no vartamāno 'pi // 3.22 //
vibhāvādiparāmarśaviṣayatvāt sacetasām /
parānandamayatvena saṃvedyatvād api sphuṭam // 3.23 //
na nirvikalpakaṃ jñānaṃ tasya grāhakam iṣyate /
tathābhilāpasaṃsargayogyatvavirahān na ca // 3.24 //
savikalpakasaṃvedyaḥ sākṣātkāratayā na ca /
parokṣas tatprakāśo nāparokṣaḥ śabdasaṃbhavāt // 3.25 //
tasmād alaukikaḥ satyaṃ vedyaḥ sahṛdayair ayam /
pramāṇaṃ carvaṇaivātra svābhinne viduṣāṃ matam // 3.26 //
niṣpattyā carvaṇasyāsya niṣpattir upacārataḥ /
avācyatvādikaṃ tasya vakṣye vyañjanarūpaṇe // 3.27 //
ratyādijñānatādātmyād eva yasmād raso bhavet /
ato 'sya svaprakāśatvam akhaṇḍatvaṃ ca sidhyati // 3.28 //
ratyādyudbodhakā loke vibhāvāḥ kāvyanāṭyayoḥ /
ālambanoddīpanākhyau tasya bhedāv ubhau smṛtau /
ālambanaṃ nāyakādis tam ālambya rasodgamāt // 3.29 //
tyāgī kṛtī kulīnaḥ suśrīko rūpayauvanotsāhī /
dakṣo 'nuraktalokas tejovaidagdhyaśīlavān netā // 3.30 //
dhīrodātto dhīroddhatas tathā dhīralalitaś ca /
dhīrapraśānta ity ayam uktaḥ prathamaś caturbhedaḥ // 3.31 //
avikatthanaḥ kṣamāvān atigambhīro mahāsattvaḥ /
stheyān nigūḍhamāno dhīrodātto dṛḍhavrataḥ kathitaḥ // 3.32 //
māyāparaḥ pracaṇḍaś capalo 'haṅkāradarpabhūyiṣṭhaḥ /
ātmaślāghānirato dhīrair dhīroddhataḥ kathitaḥ // 3.33 //
niścinto mṛdur aniśaṃ kalāparo dhīralalitaḥ syāt /
sāmānyaguṇair bhūyān divjādiko dhīrapraśāntaḥ syāt // 3.34 //
ebhir dakṣiṇadhṛṣṭānukūlaśaṭharūpibhis tu ṣoḍaśadhā /
eṣu tv anekamahilāsu samarāgo dakṣiṇaḥ kathitaḥ // 3.35 //
kṛtāgā api niḥśaṅkas tarjito 'pi na lajjitaḥ /
dṛṣṭadoṣo 'pi mithyāvākvathito dhṛṣṭanāyakaḥ // 3.36 //
anukūla ekanirataḥ śaṭho 'yam ekatra baddhabhāvo yaḥ /
darśitabahiranurāgo vipriyam anyatra gūḍham ācarati // 3.37 //
eṣāṃ ca traividhyād uttamamadhyādhamatvena /
uktā nāyakabhedāś catvāriṃśat tathāṣṭau ca // 3.38 //
dūrānvartini syāt tasya prāsāṅgiketivṛtte tu /
kiñcit tadguṇahīnaḥ sahāya evāsya pīṭhamardākhyaḥ // 3.39 //
śṛṅgāro 'sya sahāyā viṭaceṭavidūṣakādyāḥ syuḥ /
bhaktā narmasu nipuṇāḥ kupitavadhūmānabhañjanāḥ śuddhāḥ // 3.40 //
saṃbhogahīnasampad viṭas tu dhūrtaḥ kalaikadeśajñaḥ /
veśopacārakuśalo vāgmī madhuro 'tha bahumato goṣṭhyām // 3.41 //
kusumavasantādyabhidhaḥ karmavapur veṣabhāṣādyaiḥ /
hāsyakaraḥ kalaharatir vidūṣakaḥ syāt svakarmajñaḥ // 3.42 //
mantrī syād arthānāṃ cintāyāṃ tadvad avarodhe /
vāmanaṣaṇḍakirātamlecchābhīrāḥ śakārakubjādyāḥ // 3.43 //
madamūrkhatābhimānī duṣkulataiśvaryasaṃyuktaḥ /
so 'yam anūḍhabhrātā rājñaḥ śyālaḥ śakāra ity uktaḥ // 3.44 //
daṇḍe suhṛtkumārāṭavikāḥ sāmantasainikādyāś ca /
ṛtvikpurodhasaḥ syur brahmavidas tāpasās tathā dharme // 3.45 //
uttamāḥ pīṭhamardādyāḥ madhyau viṭavidūṣakau /
tathā śakāraceṭādyā adhamā parikīrtitāḥ // 3.46 //
nisṛṣṭārtho mitārthaś ca tathā sandeśahārakaḥ /
kāryapreṣyas tridhā dūto dūtyaś cāpi tathāvidhāḥ // 3.47 //
ubhayor bhāvam unnīya svayaṃ vadati cottaram /
suśliṣṭaṃ kurute kāryaṃ nisṛṣṭārthas tu sa smṛtaḥ // 3.48 //
mitārthabhāṣī kāryasya siddhakārā mitārthakaḥ /
yāvad bhāṣitasandeśahāraḥ sandeśahārakaḥ // 3.49 //
śobhā vilāso mādhuryaṃ gāmbhīryaṃ dhairyatejasī /
lalitaudāryam ity aṣṭau sattvajāḥ pauruṣā guṇāḥ // 3.50 //
śūratā dakṣatā satyaṃ mahotsāho 'nurāgitā /
nīce ghūṇādhike spardhā yataḥ śobheti tāṃ viduḥ // 3.51 //
dhīrā dṛṣṭir gatiś citrā vilāse sasmitaṃ vacaḥ /
sāṅkṣobheṣv apy anudvego mādhuryaṃ parikīrtitam // 3.52 //
bhīśokakrodhaharṣādyair gāmbhīryaṃ nirvikāratā /
vyavasāyād acalanaṃ dhairyaṃ vighne mahaty api // 3.53 //
adhikṣepāpamānādeḥ prayuktasya pareṇa yat /
prāṇātyaye 'py asahanaṃ tattejaḥ samudāhṛtam // 3.54 //
vāgveśayor madhuratā, tadvac chṛṅgāraceṣṭitaṃ lalitam /
dānaṃ sapriyabhāṣaṇam audāryaṃ śatrumitrayoḥ samatā // 3.55 //
atha nāyikā tribhedā svānyā sādhāraṇā strīti /
nāyakasāmānyaguṇair bhavati yathāsaṃbhavair yuktā // 3.56 //
vinayārjavādiyuktā gṛhakarmaparā pativratā svīyā /
sāpi kathitā tribhedā mugdhā madhyā pragalbheti // 3.57 //
prathamāvatīrṇayauvanamadanavikārā ratau vāmā /
kathitā mṛduś ca māne samadhikalajjāvatī mugdhā // 3.58 //
madhyā vicitrasuratā prarūḍhasmarayauvanā /
īṣatpragalbhavacanā madhyamavrīḍitā matā // 3.59 //
smarāndhā gāḍhatāruṇyā samastaratakovidā /
bhāvonnatā daravrīḍā pragalbhākrāntanāyakā // 3.60 //
te dhīrā cāpy adhīrā ca dhīrādhīreti ṣaḍvidhe /
priyaṃ sotprāsavakroktyā madhyā dhīrā dahed ruṣā // 3.61 //
dhīrādhīrā tu ruditair adhīrā puruṣoktibhiḥ /
pragalbhā yadi dhīrā syāc channakopākṛtis tadā // 3.62 //
udāste surate tatra darśayanty ādarān bahiḥ /
dhīrādhīrā tu solluṇṭhabhāṣitaiḥ khedayaty amum // 3.63 //
tarjayet tāḍayed anyā pratyekaṃ tā api dvidhā /
kaniṣṭhajyeṣṭharūpatvān nāyakapraṇayaṃ prati // 3.64 //
madhyāpragalbhayor bhedās tasmād dvādaśa kīrtitāḥ /
mugdhā tv ekaiva tena syuḥ svīyābhedās trayodaśā // 3.65 //
parakīyā dvidhā proktā paroḍhā kanyakā tathā /
yātrādinirastānyoḍhā kulaṭā galitatrapā // 3.66 //
kanyā tv ajātopayamā salajjā navayauvanā /
dhīrā kalāpragalbhā syād veśyāsāmānyanāyikā // 3.67 //
nirguṇān api na dveṣṭi na rajyati guṇiṣv api /
vittamātraṃ samālokya sā rāgaṃ darśayed bahiḥ // 3.68 //
kāmaṃ aṅgīkṛtam api parikṣīṇadhanaṃ naram /
mātrā niḥsārayed eṣā punaḥsandhānakāṅkṣayā // 3.69 //
taskarāḥ ṣaṇḍakā mūrkhāḥ sukhaprāptadhanās tathā /
liṅginaś channakāmādyā asyāḥ prāyeṇa vallabhāḥ // 3.70 //
eṣāpi madanāyattā kvāpi satyānurāgiṇī /
raktāyāṃ vā viraktāyāṃ ratam asyāṃ satyānurāgiṇī // 3.71 //
avasthābhir bhavanty aṣṭāv etāḥ ṣoḍaśabheditāḥ /
svādhīnabhartṛkā tadvat khaṇḍitāthābhisārikā // 3.72 //
kalahāntaritā vipralabdhā proṣitabhartṛkā /
anyā vāsakasajjā syād virahotkaṇṭhitā tathā // 3.73 //
kānto ratiguṇākṛṣṭo na jahāti yadantikam /
vicitravibhramāsaktā sā syāt svādhīnabhartṛkā // 3.74 //
pārśvam eti priyo yasyā anyasaṃyogacihnitaḥ /
sā khaṇḍiteti kathitā dhīrair īrṣyākaṣāyitā // 3.75 //
abhisārayate kāntaṃ yā manmathavaśaṃvadā /
svayaṃ vābhisaraty eṣā dhīrair uktābhisārikā // 3.76 //
saṃlīnā sveṣu gātreṣu mūkīkṛtavibhūṣaṇā /
avaguṇṭhanasaṃvītā kulajābhisared yadi // 3.77 //
vicitrojjvalaveṣā tu raṇannūpurakaṅkaṇā /
pramodasmeravadanā syād veśyābhisared yadi // 3.78 //
madaskhalitasaṃlāpā vibhramotphullalocanā /
āviddhagatisañcārā syāt preṣyābhisared yadi // 3.79 //
kṣetraṃ vāṭī bhagnadevālayo dūtīgṛhaṃ vanam /
mālāpañcaḥ śmaśānaṃ ca nadyādīnāṃ taṭī tathā // 3.80 //
evaṃ kṛtābhisārāṇāṃ puṃścalīnāṃ vinodane /
sthānāny aṣṭau tathā dhvāntacchanne kutracid āśraye // 3.81 //
cāṭukāram api prāṇanāthaṃ roṣād apāsya yā /
paścāt tāpam avāpnoti kalahāntaritā tu sā // 3.82 //
priyaḥ kṛtvāpi saṅketaṃ yasyā nāyāti saṃnidhim /
vipralabdhā tu sā jñeyā nitāntam avamānitā // 3.83 //
nānākāryavaśād yasyā dūradeśaṃ gataḥ patiḥ /
sā manobhavaduḥkhārtā bhavet proṣitabhartṛkā // 3.84 //
kurute maṇḍanaṃ yasyāḥ sajjite vāsaveśmani /
sā tu vāsakasajjā syād viditapriyasaṅgamā // 3.85 //
āgantuṃ kṛtacitto 'pi daivān nāyāti yat priyaḥ /
tadanāgataduḥkhārtā virahotkaṇṭhitā tu sā // 3.86 //
iti sāṣṭāviṃśatiśatam uttamamadhyādhamasvarūpeṇa /
caturadhikāśītiyutaṃ śatatrayaṃ nāyikābhedāḥ // 3.87 //
kvacidanyonyasāṅkaryam āsāṃ lakṣyeṣu dṛśyate /
itarā apy asaṅkhyās tā noktā vistaraśaṅkayā // 3.88 //
yauvane sattvajās tāsām aṣṭāviṃśatisaṅkhyakāḥ /
alaṅkārās tatra bhāvahāvahelās trayo 'ṅgajāḥ // 3.89 //
śobhā kāntiś ca dīptiś ca m ādhuryaṃ ca pragalbhatā /
audāryaṃ dhairyam ity ete saptaiva syur ayatnajāḥ // 3.90 //
līlā vilāso vicchittir bibbokaḥ kilakiñcitam /
moṭṭāyitaṃ kuṭṭamitaṃ vibhramo lalitaṃ madaḥ // 3.91 //
vihṛtaṃ tapanaṃ maugdhyaṃ vikṣepaś ca kutūhalam /
hasitaṃ cakitaṃ kelir ity aṣṭādaśasaṅkhyakāḥ // 3.92 //
svabhāvājāś ca bhāvādyā daśa puṃsāṃ bhavanty api /
nirvikārātmake citte bhāvaḥ prathamavikriyā // 3.93 //
bhrūnetrādivikārais tu saṃbhogecchāprakāśakaḥ /
bhāva evālpasaṃlakṣyavikāro hāva ucyate // 3.94 //
helātyantasamālakṣyavikāraḥ syāt sa eva tu /
rūpayauvanalālityabhogādyair aṅgabhūṣaṇam // 3.95 //
saiva kāntir manmathāpyāyitadyutiḥ /
kāntir evātivistīrṇā dīptir ity abhidhīyate // 3.96 //
sarvāvasthāviśeṣeṣu mādhuryaṃ ramaṇīyatā /
niḥsādhvasatvaṃ prāgalbhyam audāryaṃ vinayaḥ sadā // 3.97 //
muktātmaślāghanā dhairyaṃ manovṛttir acañcalā /
aṅgair veṣair alaṅkāraiḥ premibhir vacanair api // 3.98 //
prītiprayojaitair līlāṃ priyasyānukṛtiṃ viduḥ /
yānasthānāsanādīnāṃ mukhanetrādikarmaṇām // 3.99 //
viśeṣas tu vilāsaḥ syād iṣṭasandarśanādinā /
stokāpy ākalparacanā vicchittiḥ kāntipoṣakṛt // 3.100 //

kāryo viṣkambhako nāṭya āmukhākṣiptapātrakaḥ /
yadā tu sarasaṃ vastu mūlād eva pravartate // 6.62 //
ādāv eva tadāṅke syād āmukhākṣepasaṃśrayaḥ /
viṣkambhakādyair api no vadho vācyo 'dhikāriṇaḥ // 6.63 //
anyo 'nyena tirodhānaṃ na kuryād rasavastunoḥ /
bījaṃ binduḥ patākā ca prakarī kāryam eva ca // 6.64 //
arthaprakṛtayaḥ pañca jñātvā yojyā yathāvidhi /
alpamātraṃ samuddiṣṭaṃ bahudhā yad visarpati // 6.65 //
phalasya prathamo hetur bījaṃ tad abhidhīyate /
avāntarārthavicchede bindur acchedakāraṇam // 6.66 //
vyāpi prāsāṅgikaṃ vṛttaṃ patākety abhidhīyate /
patākānāyakasya syān na svakīyaṃ phalāntaram // 6.67 //
garbhe sandhau vimarśe vā virvāhas tasya jāyate /
prāsaṅgikaṃ pradeśasthaṃ caritaṃ prakarī matā // 6.68 //
prakarī nāyakasya syān na svakīyaṃ phalāntaram /
apekṣitaṃ tu yat sādhyam ārambho yan nibandhanaḥ // 6.69 //
samāpanaṃ tu yatsiddhyai tat kāryam iti saṃmatam /
avasthāḥ pañca kāryasya prārabdhasya phalārthibhiḥ // 6.70 //
ārambhayatnaprāptyāśāniyatāptiphalāgamāḥ /
bhaved ārabhya autsukyaṃ yan mukhyaphalasiddhaye // 6.71 //
prayatnas tu phalāvāptau vyāpāro 'titvarānvitaḥ /
upāyāpāyaśaṅkābhyāṃ prāptyāśā prāptisaṃbhavaḥ // 6.72 //
apāyābhāvataḥ prāptir niyatāptis tu niścitā /
sāvasthā phalayogaḥ syād yaḥ samagraphalodayaḥ // 6.73 //
yathāsaṅkhyam avasthābhir ābhir yogāt tu pañcabhiḥ /
pañcadhaivetivṛttasya bhāgāḥ syuḥ pañca sandhayaḥ // 6.74 //
antaraikārthasaṃbandhaḥ sandhir ekāntaye sati /
mukhaṃ pratimukhaṃ garbho vimarśa upasaṃhṛtiḥ // 6.75 //
iti pañcāsya bhedāḥ syuḥ kramāl lākṣaṇam ucyate /
yatra bījasamutpattir nānārtharasasaṃbhavā // 6.76 //
prārambheṇa samāyuktā tan mukhaṃ parikīrtitam /
phalapradhānopāyasya mukhasandhiniveśinaḥ // 6.77 //
lakṣyālakṣya ivodbhedo yatra pratimukhaṃ ca tat /
phalapradhānopāyasya prāgudbhinnasya kiñcana // 6.78 //
garbho yatra samudbhedo hāsānveṣaṇavān muhuḥ /
yatra mukhyaphalopāya udbhinno garbhato 'dhikaḥ // 6.79 //
śāpādyaiḥ sāntarāyaś ca sa vimarśa iti smṛtaḥ /
bījavanto mukhādyarthā viprakīrṇā yathāyatham // 6.80 //
ekārtham upanīyante yatra nirvahaṇaṃ hi tat /
upakṣepaḥ parikaraḥ parinyāso vilobhanam // 6.81 //
yuktiḥ prāptiḥ samādhānaṃ vidhānaṃ paribhāvanā /
udbhedaḥ karaṇaṃ bheda etāny aṅgāni vai mukhe // 6.82 //
kāvyārthasya samutpattir upakṣepa iti smṛtaḥ /
samutpannārthabāhulyaṃ jñeyaḥ parikaraḥ punaḥ // 6.83 //
tanniṣpattiḥ parinyāsaḥ guṇākhyānaṃ vilobhanam /
saṃpradhāraṇam arthānāṃ yuktiḥ prāptiḥ sukhāgamaḥ // 6.84 //
bījasyāgamanaṃ yat tu tat samādhānam ucyate /
sukhaduḥkhakṛto yo 'rthas tad vidhānam iti smṛtam // 6.85 //
kutūhalottarā vācaḥ proktā tu paribhāvanā /
bījārthasya prarohaḥ syād udbhedaḥ karaṇaṃ punaḥ // 6.86 //
prakṛtārthasamārambhaḥ bhedaḥ saṃhatabhedanam /
vilāsaḥ parisarpaś ca vidhutaṃ tāpanaṃ tathā // 6.87 //
narma narmadyutiś caiva tathā pragamanaṃ punaḥ /
virodhaś ca pratimukhe tathā syāt paryupāsanam // 6.88 //
puṣpaṃ vajram upanyāso varṇasaṃhāra ity api /
samīhā ratibhogārthā vilāsa iti kathyate // 6.89 //
iṣṭanaṣṭānusaraṇaṃ parisarpaś ca kathyate /
kṛtasyānunayasyādau vidhutaṃ tv aparigrahaḥ // 6.90 //
upāyādarśanaṃ yat tu tāpanaṃ nāma tad bhavet /
parihāsavaco narma dhṛtis tu parihāsajā // 6.91 //
narmadyutiḥ pragamanaṃ vākyaṃ syād uttarottaram /
virodho vyasanaprāptiḥ kruddhasyānunayaḥ // 6.92 //
syāt paryupāsanaṃ puṣpaṃ viśeṣavacanaṃ matam /
pratyakṣaniṣṭhuraṃ vajram upanyāsaḥ prasādanam // 6.93 //
cāturvarṇyopagamanaṃ varṇasaṃhāra iṣyate /
abhūtāharaṇaṃ mārgo rūpodāharaṇe kramaḥ // 6.94 //
saṅgrahaś cānumānaṃ ca prārthanā kṣiptir eva ca /
troṭakādhibalodvegā garbhe syur vidravas tathā // 6.95 //
tatra vyājāśrayaṃ vākyam abhūtāharaṇaṃ matam /
tattvārthakathanaṃ mārgo rūpaṃ vākyaṃ vitarkavat // 6.96 //
udāharaṇam utkarṣayuktaṃ vacanam ucyate /
bhāvatattvopalabdhis tu kramaḥ syāt saṅgrahaḥ punaḥ // 6.97 //
sāmadānārthasaṃpanno liṅgād ūho 'numānatā /
ratiharṣotsavānāṃ tu prārthanaṃ prārthanā bhavet // 6.98 //
rahasyārthasya tadbhedaḥ kṣiptiḥ syāt toṭakaṃ punaḥ /
saṃrabdhavāk adhibalam abhisandhicchalena yaḥ // 6.99 //
nṛpādijanitā bhītir udvegaḥ parikīrtitaḥ /
śaṅkābhayatrāsakṛtaḥ saṃbhramo vidravo mataḥ // 6.100 //
apavādo 'tha saṃpheṭo vyavasāyo dravo dyutiḥ /
śaktiḥ prasaṅgaḥ khedaś ca pratiṣedho virodhanam // 6.101 //
prarocanā vimarśe syād ādānaṃ chādanaṃ tathā /
doṣaprakhyāpavādaḥ syāt saṃpheṭo roṣabhāṣaṇam // 6.102 //
vyavasāyas tu vijñeyaḥ pratijñāhetusaṃbhavaḥ /
dravā guruvyatikrāntiḥ śokāvegādisaṃbhavā // 6.103 //
tarjanodvejane proktā dyutih śaktiḥ punar bhavet /
virodhasya praśamanaṃ prasaṅgo gurukīrtanam // 6.104 //
manaś ceṣṭāsamutpannaḥ śramaḥ kheda iti smṛtaḥ /
īpsitārthapratīghātaḥ pratiṣedha itīṣyate // 6.105 //
kāryātyayopagamanaṃ virodhanam iti smṛtam /
prarocanā tu vijñeyā saṃhārārthapradarśinī // 6.106 //
kāryasaṅgraha ādānaṃ tad āhuś chādanaṃ punaḥ /
kāryārtham apamānādeḥ sahanaṃ khalu yad bhavet // 6.107 //
sandhir vibodho grathanaṃ nirṇayaḥ paribhāṣaṇam /
kṛtiḥ prasāda ānandaḥ samayo 'py upagūhanam // 6.108 //
bhāṣaṇaṃ pūrvavākyaṃ ca kāvyasaṃhāra eva ca /
praśastir iti saṃhāre jñeyāny aṅgāni nāmataḥ // 6.109 //
bījopagamanaṃ sandhiḥ vibodhaḥ kāryamārgaṇam /
upanāyasas tu kāryāṇāṃ grathanaṃ nirṇayaḥ punaḥ // 6.110 //
anubhūtārthakathanaṃ vadanti paribhāṣaṇam /
parivādakṛtaṃ vākyaṃ labdhārthaśamanaṃ kṛtiḥ // 6.111 //
śuśrūṣādiḥ prasādaḥ syād ānando vāñchitāgamaḥ /
samayo duḥkhaniryāṇaṃ tad bhaved upagūhanam // 6.112 //
yat syād adbhutasaṃprāptiḥ sāmadānādi bhāṣaṇam /
pūrvavākyaṃ tu vijñeyaṃ yathoktārthopadarśanam // 6.113 //
varapradānasaṃprāptiḥ kāvyasaṃhāra iṣyate /
nṛpadeśādiśāntis tu praśastir abhidhīyate // 6.114 //
catuḥṣaṣṭividhaṃ hy etad aṅgaṃ proktaṃ manīṣibhiḥ /
kuryād aniyate tasya sandhāv api niveśanam // 6.115 //
rasānuguṇatāṃ vīkṣya rasasyaiva hi mukhyatā /
iṣṭārtharacanāścaryalābho vṛttāntavistaraḥ // 6.116 //
rāgaprāptiḥ prayogasya gopyānāṃ gopanaṃ tathā /
prakāśanaṃ prakāśyānām aṅgānāṃ ṣaḍvidhaṃ phalam // 6.117 //
aṅgahīno naro yadvan naivārambhakṣamo bhavet /
aṅgahīnaṃ tathā kāvyaṃ na prayogāya yujyate // 6.118 //
saṃpādayetāṃ sandhyaṅgaṃ nāyakapratināyakau /
tadabhāve patākādyās tadabhāve tathetarat // 6.119 //
rasavyaktim apekṣyaiṣām aṅgānāṃ saṃniveśanam /
na tu kevalayā śāstrasthitisaṃpādanecchayā // 6.120 //
aviruddhaṃ tu yad vṛttaṃ rasādivyaktaye 'dhikam /
tad apy anyathayed dhīmān vaded vā kadācana // 6.121 //
śṛṅgāre kauśikī vīre sāttvaty ārabhaṭī punaḥ /
rase raudre ca bībhatse vṛttiḥ sarvatra bhāratī // 6.122 //
catasro vṛttayo hy etāḥ sarvanāṭyasya mātṛkāḥ /
syur nāyikādivyāpāraviśeṣā nāṭakādiṣu // 6.123 //
yā ślakṣṇanepathyaviśeṣacitrā strīsaṅkulā puṣkalanṛtyagītā /
kāmopabhogaprabhavopacārā sā kaiśikī cāruvilāsayuktā // 6.124 //
narma ca narmasphūrjo narmasphoṭo 'tha narmagarbhaś ca /
catvāry aṅgāny asyā vaidagdhyakrīḍitaṃ narma // 6.125 //
iṣṭajanāvarjanakṛt tac cāpi trividhaṃ matam /
vihitaṃ śuddhahāsyena saśṛṅgāramayena ca // 6.126 //
narmasphūrjaḥ sukhārambho bhayānto navasaṅgamaḥ /
narmasphoṭo bhāvaleśaiḥ sūcitālparaso mataḥ // 6.127 //
narmagarbho vyavahṛtir netuḥ pracchannavartinaḥ /
sāttvatī bahulā sattvaśauryatyāgadayārjavaiḥ // 6.128 //
saharṣā kṣudraśṛṅgārā viśokā sādbhutā tathā /
utthāpako 'tha sāṅghātyaḥ saṃlāpaḥ parivartakaḥ // 6.129 //
viśeṣā iti catvāraḥ sāttvatyāḥ parikīrtitāḥ /
uttejanakaro śatror vāgutthāpaka ucyate // 6.130 //
mantrārthadaivaśaktyādeḥ sāṅghātyaḥ saṅghabhedanam /
saṃlāpaḥ syād gabhīroktir nānābhāvasamāśrayaḥ // 6.131 //
prārabdhād anyakāryāṇāṃ karaṇaṃ parivartakaḥ /
māyendrajālasaṅgrāmakrodhodbhrāntādiceṣṭitaiḥ // 6.132 //
saṃyuktā vadhabandhādyair uddhatārabhaṭī matā /
vastūtthāpanasaṃpheṭo saṃkṣipir avapātanam // 6.133 //
iti bhedās tu catvāra ārabhaṭyāḥ prakīrtitāḥ /
māyādyutthāpitaṃ vastu vastūtthāpanam ucyate // 6.134 //
saṃpheṭas tu samāghātaḥ kruddhasatvarayor dvayoḥ /
saṅkṣiptā vasturacanā śilpair itarathāpi vā // 6.135 //
saṅkṣiptiḥ sthān nivṛttau ca netur netrantaragrahaḥ /
praveśatrāsaniṣkrāntiharṣavidravasaṃbhavam // 6.136 //
avapātanam ity uktaṃ pūrvam uktaiva bhāratī /
aśrāvyaṃ khalu yad vastu tad iha svagataṃ matam // 6.137 //
sarvaśrāvyaṃ prakāśaṃ syāt tad bhaved apavāritam /
rahasyaṃ tu yad anyasya parāvṛtya prakāśyate // 6.138 //
tripatākakareṇānyān apavāryāntarā kathām /
anyonyāmantraṇaṃ yat syāt tajjanānte janāntikam // 6.139 //
kiṃ bravīṣīti yannāṭye vinā pātraṃ prayujyate /
śrutvevānuktam apy arthaṃ tat syād ākāśabhāṣitam // 6.140 //
dattāṃ siddhāṃ ca senāṃ ca veśyānāṃ nāma darśayet /
dattaprāyāṇi vaṇijāṃ ceṭaceṭyos tathā punaḥ // 6.141 //
vasantādiṣu varṇyasya vastuno nāma yad bhavet /
nāma kāryaṃ nāṭakasya garbhitārthaprakāśakam // 6.142 //
nāyikānāyakākhyānāt saṃjñā prakaraṇādiṣu /
nāṭikāsaṭṭakādīnāṃ nāyikābhir viśeṣaṇam // 6.143 //
prāyeṇa ṇyantakaḥ sādhir gameḥ sthāne prayujyate /
rājā svāmīti deveti bhṛtyair bhaṭṭeti cādhamaiḥ // 6.144 //
rājarṣibhir vayasyeti tathā vidūṣakeṇa ca /
rājann ity ṛṣibhir vācyaḥ so 'patyapratyayena ca // 6.145 //
svecchayā nāmabhir viprair vipra āryeti cetaraiḥ /
vayasyety athavā nāmnā vācyo rājñā vidūṣakaḥ // 6.146 //
vācyau naṭīsūtradhārāvāryanāmnā parasparam /
sūtradhāraṃ vaded bhāva iti vai pāripārśvikaḥ // 6.147 //
sūtradhāro māriṣeti haṇḍe ity adhamaiḥ samāḥ /
vayasyety uttamair haṃho madhyair āryeti cāgrajaḥ // 6.148 //
bhagavann iti vaktavyāḥ sarvair devarṣiliṅginaḥ /
vaded rājñīṃ ca ceṭīṃ ca bhavatīti vidūṣakaḥ // 6.149 //
āyuṣman rathinaṃ sūto vṛddhaṃ tāteti cetaraḥ /
vatsaputrakatāteti nāmnā gotreṇa vā sutaḥ // 6.150 //
śiṣyo 'nujaś ca vaktavyo 'mātya āryeti cādhamaiḥ /
viprair ayam amātyeti saciveti ca bhaṇyate // 6.151 //
sādho iti tapasvī ca praśāntaś cocyate budhaiḥ /
svagṛhītābhidhaḥ pūjyaḥ śiṣyādyair vinigadyate // 6.152 //
upādhyāyeti cācāryo mahārājeti bhūpatiḥ /
svāmīti yuvarājas tu kumāro bhartṛdārakaḥ // 6.153 //
bhadrasaumyamukhety evam adhamais tu kumārakaḥ /
vācyā prakṛtibhī rājñaḥ kumārī bhartṛdārikā // 6.154 //
patir yathā tathā vācyā jyeṣṭhamadhyādhamaiḥ striyaḥ /
haleti sadṛśī preṣyā hajje veśyājjukā tathā // 6.155 //
kuṭṭiny ambety anugatiḥ pūjyā ca jaratī janaiḥ /
āmantraṇaiś ca pāṣāṇḍā vācyāḥ svasamayāgataiḥ // 6.156 //
śakyādayaś ca saṃbhāṣyā bhadradattādināmabhiḥ /
yasya yat karma śilpaṃ vā vidyā vā jātir eva vā // 6.157 //
tenaiva nāmnā vācyo 'sau jñeyāś cānye yathocitam /
puruṣāṇām anīcānāṃ saṃskṛtaṃ syāt kṛtātmanām // 6.158 //
saurasenī prayoktavyā tādṛśīnāṃ ca yoṣitām /
āsām eva tu gāthāsu mahārāṣṭrīṃ prayojayet // 6.159 //
atroktā māgadhī bhāṣā rājāntaḥpuracāriṇām /
ceṭānāṃ rājaputrāṇāṃ śreṣṭhānāṃ cārdhamāgadhī // 6.160 //
prācyāṃ vidūṣakādīnāṃ dhūrtānāṃ syād avantijā /
yodhanāgarikādīnāṃ dākṣiṇātyā hi dīvyatām // 6.161 //
śabarāṇāṃ śakādīnāṃ śābarīṃ saṃprayojayet /
vāhlīkamāṣodīcyānāṃ drāviḍī drāviḍādiṣu // 6.162 //
ābhīreṣu tathābhīrī cāṇḍālī pukkasādiṣu /
ābhīrī śābarī cāpi kāṣṭhapātropajīviṣu // 6.163 //
tathaivāṅgārakārādau paiśācī syāt piśācavāk /
ceṭānām apy anīcānām api syāt saurasenikā // 6.164 //
bālānāṃ ṣaṇḍakānāṃ ca nīcagrahavicāriṇām /
unmattānām āturāṇāṃ saiva syāt saṃskṛtaṃ kvacit // 6.165 //
aiśvaryeṇa pramattasya dāridryopadrutasya ca /
bhikṣuvalkadharādīnāṃ prākṛtaṃ saṃprayojayet // 6.166 //
saṃskṛtaṃ saṃprayoktavyaṃ liṅginīṣūttamāsu ca /
devīmantrisutāveśyāsv api kaiścit tathoditam // 6.167 //
kāryataś cottamādīnāṃ kāryo bhāṣāviparyayaḥ // 6.168 //
yoṣitsakhībālaveśyākitavāpsarasāṃ tathā /
vaidagdhyārthaṃ pradātavyaṃ saṃskṛtaṃ cāntarāntarā // 6.169 //
ṣaṭtriṃśallakṣaṇāny atra nāṭyālaṅkṛtayas tathā /
trayastriṃśatprayojyāni vīthyaṅgāni trayodaśa // 6.170 //
lāsyāṅgāni daśa yathālābhaṃ rasavyapekṣayā /
bhūṣaṇākṣarasaṅghātau śobhodāharaṇaṃ tathā // 6.171 //
hetusaṃśayadṛṣṭāntās tulyatarkaḥ padoccayaḥ /
nidarśanābhiprāyo ca prāptir vicāra eva ca // 6.172 //
diṣṭopadiṣṭe ca guṇātipātātiśayau tathā /
viśeṣaṇaniruktī ca siddhibhraṃśaviparyayau // 6.173 //
dākṣiṇyānunayau mālārthāpattir garhaṇaṃ tathā /
pṛcchā prasiddhiḥ sārūpyaṃ saṃkṣepo guṇakīrtanam // 6.174 //
leśo manoratho 'nuktasiddhiḥ priyavacas tathā /
lakṣaṇāni guṇaiḥ sālaṅkārair yogas tu bhūṣaṇam // 6.175 //
varṇanākṣarasaṅghātaś citrārthair akṣarair mitaiḥ /
siddhair arthaiḥ samaṃ yatrāprasiddho 'rthaḥ prakāśate // 6.176 //
śliṣṭalakṣaṇacitrārthā sā śobhety abhidhīyate /
yatra tulyārthayuktena vākyenābhipradarśanāt // 6.177 //
sādhyate 'bhimataś cārthas tadudāharaṇaṃ matam /
hetur vākyaṃ samāsoktam iṣṭakṛd dhetudarśanāt // 6.178 //
saṃśayo 'jñātatattvasya vākye syād yad aniścayaḥ /
dṛṣṭānto yas tu pakṣe 'rthasādhanāya nidarśanam // 6.179 //
tulyatarko yadarthena tarkaḥ prakṛtigāminā /
sañcayo 'rthānurūpo yaḥ padānāṃ sa padoccayaḥ // 6.180 //
yatrārthānāṃ prasiddhānāṃ kriyate parikīrtanam /
parapakṣavyudāsārthaṃ tan nidarśanam ucyate // 6.181 //
abhiprāyas tu sādṛśyād abhūtārthasya kalpanā /
prāptiḥ kenacid aṃśena kiñcid yatrānumīyate // 6.182 //
vicāro yuktivākyair yad apratyakṣārthadarśanam /
deśakālasvarūpeṇa varṇanā diṣṭam ucyate // 6.183 //
upadiṣṭaṃ manohāri vākyaṃ śāstrānusārataḥ /
guṇātipātaḥ kāryaṃ vad viparītaṃ guṇān prati // 6.184 //
yaḥ sāmān guṇodrekaḥ sa guṇātiśayo mataḥ /
siddhān arthān bahūn uktvā viśeṣoktir viśeṣaṇam // 6.185 //
pūrvasiddhārthakathanaṃ niruktir iti kīrtyate /
bahūnāṃ kīrtanaṃ siddhir abhipretārthasiddhaye // 6.186 //
dṛptādīnāṃ bhavad bhraṃśo vācyād anyatarad vacaḥ /
vicārasyānyathābhāvaḥ sandehāt tu viparyayaḥ // 6.187 //

yatra bimbānubimbatvaṃ bodhayets sā nidarśanā // 10.51 //
ādhikyam upameyasyopamānān nyūnatāthavā /
vyatireka eka ukte 'nukte hetau punas tridhā // 10.52 //
caturvidho 'pi sāmyasya bodhanāc chabdato 'rthataḥ /
ākṣepāc ca dvādaśadhā śleṣe 'pīti triraṣṭadhā // 10.53 //
pratyekaṃ syān militāṣṭacatvāriṃśadvidhaḥ punaḥ /
sahārthasya balād ekaṃ yatra syād vācakaṃ dvayoḥ // 10.54 //
sā sahoktir mūlabhūtātiśayoktir yadā bhavet /
vinoktir yad vinānyena nāsādhv anyad asādhu vā // 10.55 //
samāsoktiḥ samair yatra kāryaliṅgaviśeṣaṇaiḥ /
vyavahārasamāropaḥ prakṛtenāsya vastunaḥ // 10.56 //
uktair viśeṣaṇaiḥ sābhiprāyaiḥ parikaro mataḥ /
śabdaiḥ svabhāvād ekārthe śleṣo 'nekārthavācanam // 10.57 //
kvacid viśeṣaḥ sāmānyāt sāmānyaṃ vā viśeṣataḥ /
kāryān nimittaṃ kāryaṃ ca hetor atha samāt samam // 10.58 //
aprastutāt prastutaṃ ced gamyate pañcadhā tataḥ /
aprastutapraśaṃsā syād uktā vyājastutiḥ punaḥ // 10.59 //
nindāstutibhyāṃ vācyābhyāṃ gamyatve stutinindayoḥ /
paryāyoktaṃ yadā bhaṅgyā gamyam evābhidhīyate // 10.60 //
sāmānyaṃ vā viśeṣeṇa viśeṣas tena vā yadi /
kāryaṃ ca kāraṇenedaṃ kāryeṇa ca samarthyate // 10.61 //
sādharmyenetareṇārthāntaranyāso 'ṣṭadhā tataḥ /
hetor vākyapadārthatve kāvyaliṅgo nigadyate // 10.62 //
anumānaṃ tu vicchityā jñānaṃ sādhyasya sādhanāt /
abhedenābhidhāhetur hetor hetumatā saha // 10.63 //
anukūlaṃ prātikūlyam anukūlavidhāyi cet /
vastuno vaktum iṣṭasya viśeṣapratipattaye // 10.64 //
niṣedhābhāsa ākṣepo vakṣyamāṇoktago dvidhā /
aniṣṭasya tathārthasya vidhyābhāsaḥ paro mataḥ // 10.65 //
vibhāvanā vinā hetuṃ kāryotpattir yad ucyate /
uktānuktanimittatvād dvidhā sā parikīrtitā // 10.66 //
sati hetau phalābhāve viśeṣoktis tathā dvidhā /
jātiś caturbhir jātādyair guṇo guṇādibhis tribhiḥ // 10.67 //
kriyā ca kriyādravyābhyāṃ dravyaṃ dravyeṇa vā mithaḥ /
viruddham iva bhāseta virodho 'sau daśākṛtiḥ // 10.68 //
kāryakāraṇayor bhinnadeśatāyām asaṅgatiḥ /
guṇau kriye vā yat syātāṃ viruddhe hetukāryayoḥ // 10.69 //
yad vārabdhasya vaiphalyam anarthasya ca saṃbhavaḥ /
virūpayoḥ saṃghaṭanā yā ca tad viṣamaṃ matam // 10.70 //
samaṃ syād ānurūpyeṇa ślāghā yogyasya vastunaḥ /
vicitraṃ yad viruddhasya kṛtir iṣṭaphalāptaye // 10.71 //
āśrayāśrayinor ekasyādhikye 'dhikam ucyate /
anyonyam ubhayor ekakriyāyāḥ kāraṇaṃ mithaḥ // 10.72 //
yad ādheyam anādhāram ekaṃ cānekagocaram /
kiñcit prakurvataḥ kāryam aśakyasyetarasya vā // 10.73 //
kāryasya karaṇaṃ daivād viśeṣas trividhas tataḥ /
vyāghātaḥ sa tu kenāpi vastu yena yathākṛtam // 10.74 //
tenaiva ced upāyena kurute 'nyas tad anyathā /
saukāryeṇa ca kāryasya viruddhaṃ kriyate yadi // 10.75 //
paraṃ paraṃ prati yadi pūrvapūrvasya hetutā /
tadā kāraṇamāla syāt tan mālādīpakaṃ punaḥ // 10.76 //
dharmiṇām ekadharmeṇa saṃbaddho yady athottaram /
pūrvaṃ pūrvaṃ prati viśeṣaṇatvena paraṃ param // 10.77 //
sthāpyate 'pohate vā cet syāt tadaikāvalī dvidhā /
uttarottaram utkarṣo vastunaḥ sāra ucyate // 10.78 //
yathāsāṅkhyam anuddeśa uddiṣṭānāṃ krameṇa yat /
kvacid ekam anekasmin ekaṃ caikaśaḥ kramāt // 10.79 //
bhavati kriyate vā cet tadā paryāya iṣyate /
parivṛttir vinimayaḥ samanyūnādhikair bhavet // 10.80 //
praśnād apraśnato vāpi kathitād vastuno bhavet /
tādṛg anyavyāpohaś cec chābda ārtho 'thavā tadā // 10.81 //
parisaṅkhyottaraṃ praśnasyottarād unnayo yadi /
yac cāsakṛd asaṃbhāvyaṃ saty api praśna uttaram // 10.82 //
daṇḍāpūpikayānyārthāgamo 'rthāpattir iṣyate /
vikalpas tulyabalayor virodhaś cāntarāyataḥ // 10.83 //
samuccayo 'yam ekasmin sati kāryasya sādhake /
khalekapotikānyāyāt tatkaras tatparo 'pi cet // 10.84 //
guṇau kriye vā yugapat syātāṃ yad vā guṇakriye /
samādhiḥ sukare kārye daivād vastv antarāgamāt // 10.85 //
pratyanīkam aśaktena pratīkāre ripor yadi /
tadīyasya tiraskāras tasyotkarṣasya sādhakaḥ // 10.86 //
prasiddhasyopamānasyopameyatvaprakalpanam /
niṣphalatvābhidhānaṃ vā pratīpam iti kathyate // 10.87 //
uktvā cātyantam utkarṣam atyutkṛṣṭasya vastunaḥ /
kalpite 'py upamānatve pratīpaṃ kecid ūcire // 10.88 //
mīlitaṃ vastuno guptiḥ kenacit tulyalakṣmaṇā /
sāmānyaṃ prakṛtasyānyatādātmyaṃ sadṛśair guṇaiḥ // 10.89 //
tadguṇaḥ svaguṇatyāgād atyutkṛṣṭaguṇagrahaḥ /
tadrūpānahāras tu hetau saty apy atadguṇaḥ // 10.90 //
saṃlakṣitas tu sūkṣmo 'rtha ākāreṇeṅgitena vā /
kayāpi sūcyate bhaṅgyā yatra sūkṣmaṃ tad ucyate // 10.91 //
vyājoktir gopanaṃ vyājād udbhinnasya tu vastunaḥ /
svabhāvoktir durūhārthaḥ svakriyārūpavarṇanam // 10.92 //
adbhutasya padārthasya bhūtasyārthe bhaviṣyataḥ /
yat pratyakṣāyamānatvaṃ tad bhāvikam udāhṛtam // 10.93 //
lokātiśayasaṃpattir vastunodāttam ucyate /
yad vāpi prastutasyāṅgaṃ mahatāṃ caritaṃ bhavet // 10.94 //
rasabhāvau tadābhāsau bhāvasya praśamas tathā /
guṇībhūtatvam āyānti yadālaṅkṛtayas tadā // 10.95 //
rasavat preya ūrjasvi samāhitam iti kramāt /
bhāvasya codaye sandhau miśratve ca tadākhyakāḥ // 10.96 //
yady eta evālaṅkārāḥ parasparavimiśritāḥ /
tadā pṛthagalaṅkārau saṃsṛṣṭiḥ saṅkaras tathā // 10.97 //
aṅgāṅgitve 'laṃkṛtīnāṃ tadvad ekāśrayasthitau /
sandigdhatve ca bhavati saṅkaras trividhaḥ punaḥ // 10.98 //
śrīcandraśekharamahākavicandrasūnuśrīviśvanāthakavirājakṛtaṃ prabandham /
sāhityadarpaṇam amuṃ sudhiyo vilokya sāhityatattvam akhilaṃ sukham eva vitta // 10.99 //
yāvat prasannendunibhānanā śrīnārāyaṇasyāṅkam alaṅkaroti /
tāvan manaḥ saṃmadayan kavīnām eṣa prabandhaḥ prathito 'stu loke // 10.100 //
