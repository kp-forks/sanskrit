#Author:Kṣemarāja
#Text:Bhairavānukaraṇastotra
#Data entry:Daniele Cuneo
#Edition:Raniero Gnoli: »Miscellanea Indica. 3. The Bhairavānukaraṇastotra by Kṣemarāja.« In: East and West 9.3 (1958), pp. 451-455.


oṃ namaś cidbhairavāya /
citibhūmisamāśrayaṇāś citibhūtiṃ śrayata mā mudhā bhramata /
prathayann iti bhaktimatāṃ jayati śivo 'nugrahapravaṇaḥ // 1 //
cidbhairavam eva paraṃ paramāmṛtarūpam ekam atidīptam /
ullasitakaraṇacakragrastasamastaṃ śivaṃ vande // 2 //
stutyaḥ stotā stutir iti yad api vibhinnaṃ na kiṃcid astīha /
mṛṣati yathā yad rūpaṃ cidrūpatayā tathā bhavaty etat // 3 //
vigalitasarvavibhedaṃ sarvavibhedātma cidghanānandam /
yat tava tattvaṃ bhagavaṃs tasyānukṛtiṃ tvadākṛtiṃ vande // 4 //
rekhāpuruṣaḥ puruṣaṃ varṇalipir varṇasaṃcayaṃ yadvat /
tadvat vicitrarūpaṃ tvām ākāro vyanakti paramātman // 5 //
tatrāvikalpam eva tvāṃ cidrūpaṃ samāviśya /
vimṛśāmaḥ kim api manāṅ na hi tattvaṃ kalpanāviṣayaḥ // 6 //
svātmaprabhāṃ vidhāya vāgdevīṃ viṣayakusumakḷptārcaḥ /
prathamonmeṣaṃ gurum atha vanditvā hṛdayam āviśya // 7 //
staumi vimṛśan maheśvaram ātmānaṃ svaṃ cidekaghanam /
srāvitapāśakadambakam etasmin dehadevagṛhe // 8 //
cityānandecchāvitkaraṇākhyāḥ śaktayo jayanti vibho /
sūkṣmasthūlabhidāptā vaktratvaṃ brahmamuṇḍabhidhā // 9 //
sarvajñatādiviṣayaṃ hṛdayādikam aṅgaṣaṭkaṃ te /
sakalaṃ sadāśivāntaṃ pretātmatayā tavāsanaṃ svāmin // 10 //
paramaprakāśavapuṣo vimarśaśaktiḥ prabho parā devī /
asyā eva prasaraḥ sarvā brāhmyādikā devyaḥ // 11 //
dahanajalāmṛtalekhāviṣādi bibhrat tvam ādiśasy etat /
viśvaṃ viruddham api saccidrūpe mayi virudhyate naitat // 12 //
saṃvitsūtranilīno niḥsāro mātṛnivaho 'yam /
karaṇamuṇḍamālabhārin darśayasītīva naḥ svāmin // 13 //
paśyata paśyata pāśā viṣamās ta ime bahiṣkṛtāḥ śaśvat /
ity antradhāraṇamiṣat kathayaṃs tvaṃ naḥ samābhāsi // 14 //
māyāpaṭaḥ sarāgo vyomni pare 'smin sa eṣa nikṣiptaḥ /
iti gajacarma saraktaṃ mūrdhni vahan kathayasīśa // 15 //
antaḥśaktikṛpāṇīm vyanakṣi saṃsṛtivibhedinīm asinā /
nijaśaktimahimasvīkṛtasamastaviśvā hi vīravarāḥ // 16 //
bhavabhayahantā so 'ham sānāthye 'vasthito 'smi mā bhaiṣṭa /
ity āsphoṭitakheṭakadarśanato diśasi naḥ svāmin // 17 //
nijaśaktipāśavalitaś cidvapuṣā kalyate mayā kālaḥ /
iti pāśadhāraṇavaśāt prathayasi naḥ kālakālas tvam // 18 //
bhedamayam akhilam etan nijaśaktyaivākṣipāmi saṃhartum /
ity aṅkuśadhāraṇataḥ sphuṭayati parabhairavo 'smākam // 19 //
kodaṇḍārūdhaśarapradarśanād brahmaviṣṇurudreśān /
sasadākhyaśivāñ chaktyā kāraṇahariṇaṃ bhinatsi mṛgavat tvam // 20 //
varadena pāṇinā tvaṃ viśvavibhūtipradatvam abhinayasi /
na khalu paratattvaniṣṭho vibhavamalaiḥ spṛśyate jātu // 21 //
abhayena ca bhavabhayāni unmūlayatā prakāśyate satatam /
viśvānugrahakāraṇasvabhāvatā tava karatalena // 22 //
akhyātirūpam asamaṃ māyāyāḥ śakalitaṃ śiro 'traitat /
parabodhamayena mayā muṇḍaṃ bibhrat prakāśayasy evam // 23 //
niḥśeṣahṛtasārā mayy eva jagatsthitiś cidekaghane /
iti khaṭvāṅgakaraṅkodvahanacchalato dadāsy ājñām // 24 //
vīṇāghaṇtāḍamarūn uḍḍāmara vādayann idaṃ diśasi /
vṛttitribhedabhinnaṃ nādaṃ śṛṇvan sthito 'smīti // 25 //
kiṃ ca parādikaśaktitritayārāsundarām mahāśaktim /
devānukaroṣi bahis triśūladhāraṇamiṣeṇa nityaṃ tvam // 26 //
icchādikanijaśaktiprakāśitādhaḥsthagocaratritayām /
svām eva parāṃ śaktiṃ vajramayīṃ vahasi ṣaḍarāṃ tvam // 27 //
jagad akhilaṃ macchaktyā damitaṃ sarvavyāvasthitīr dhatte /
iti daṇḍadhāraṇavaśād vyanakṣi cidbhairavāsmākam // 28 //
mudgaraparaśū bibhrad baindavanādānukārarūpau tvam /
bhedavibhedanaśakalanaparatvam īśāna nirdiśasi // 29 //
darśanabhedāt svāminn anyānyām āhṛtiṃ dadhānas tvam /
bhairavaparabodhamayaṃ prathayasi nijadhāma sarvatra // 30 //
śāktāṇḍakhaṇḍamadhye viśvarasāḥ sarvam ahaṃ pibāmi sadā /
vyañjayasi narakapālagarudhiramiṣād etad iva me 'ntaḥ // 31 //
abhimānahṛdayabandhānutkhātau tāv imau saśikhau /
mūrtāv iva muṇdābjau prathayan dhatse mahākālaḥ // 32 //
akhilajagadbījabhuvaṃ prathamasthām anukaroṣi bodhanidhe /
karatalagabījapūrakadarśanalīlāyitād deva // 33 //
bhairavarūpo 'py asi yat tatrāpi kare 'kṣamālikāṃ dhatse /
tat sarvākṣavikāsād viśvaṃ parivartayasy antaḥ // 34 //
asthikaramuṇḍamālābhasmābharaṇo nṛraktaruciḥ /
viśvaṃ brahmamayatvāc chuddham itīśaḥ samādiśasi // 35 //
svātmārāmamayānāṃ viṣamā api viṣayabudbudhāḥ kim amī /
smaradahanam alikanayanam prathayan paramāgnirūpam ādiśasi // 36 //
antarnibaddhalakṣyaḥ pravikāsikaraṇo 'pi yat sadā bhavasi /
tat prathayasi sarvadaśaṃ gatamohakalaṅkam ātmānam // 37 //
deva digambaratā te vadati nirāvaraṇam ātmano rūpam /
paramabrahmamayārkān māyāvaraṇam kathaṃ vṛṇuyāt // 38 //
saṃhartā sarveṣām arkādīnām ahaṃ tamobhūmiḥ /
nirmalaśāradagaganaprabheṇa vapuṣeti darśayasi // 39 //
paramabrahmamayas tvaṃ deva yadāśrayasi bhairavākāram /
tat prathayasi tattvajñaḥ saty api bhede vimukta iti // 40 //
vividhaśmaśānamālāvetālāvṛtaḥ prakāśayase /
saṃvittiviṣayakalanāmadhyaś citibhairavo 'smīti // 41 //
tvayy evāmṛtalekhā paraśaktimayī parapramātātaḥ /
nātha tvam eva jagatām jīvanam iti bhāsi no hṛdaye // 42 //
tava śabdarāśivapuṣaḥ pratyaṣṭakam eva bāhyaparivāraḥ /
parasūkṣmādivibhedād bhairava lokeśvarapramukhāḥ // 43 //
mudrayati nikhilam etan niḥśeṣasvasvarūpasaṃkramaṇāt /
modayati bhaktavargaṃ gatimudaṃ pūrṇatāprathanāt // 44 //
mocayati pāśajālaṃ drāvayati vibhedam īśa te mūrtiḥ /
mudrayati vividhasaṃviddraviṇaṃ mudrā tataḥ kathitā // 45 //
jayati paratattvabhūmipravare nipāyabhūtaiṣa /
sphuṭam adhivasatu satattvaprathayā hṛdayasya meyamalam // 46 //
pratikalam udeti saṃvid yā kila tatra sphuraty adho viśvam /
śivaśaktivibhavarūpaṃ jānanti maheśa yāgasthāḥ // 47 //
itthaṃ sphurasi sadā me cidbhairava sarvabhūtātma /
tad yāgādijapāntaṃ sampannam ayatnataḥ sarvam // 48 //
iti śrīmahāmāheśvarācāryaśrīkṣemarājaviracitaṃ bhairavānukaraṇastotraṃ samāptam //
