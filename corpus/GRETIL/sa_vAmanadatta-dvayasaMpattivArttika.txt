#Author:Vāmanadatta
#Text:Dvayasaṃpattivārttika
#Data entry:Daniele Cuneo
#Edition:Raniero Gnoli: »Il Dvayasampattivārttikam di Vāmanadatta.« In: Gururajamañjarikā. Napoli 1974, pp. 451-455.

oṃ namaḥ śivāya /
namaḥ paramanāthāya gaṇanāthāya śūline /
cidpadmāsanasaṃsthāya caturdhābhinnamūrtaye // 1 //
avarṇam asvaraṃ dhyātvā vyāpakaṃ parameśvaram /
vyutthāne yoginām itthaṃ śivatvaṃ jāyate sphuṭam // 2 //
tataḥ śaktimayo yogī nādabindvātmavigrahaḥ /
dvayasampattim āpnoti nirādhārapade sthitaḥ // 3 //
ślokadvayam idaṃ yasmāc chivavaktrāgamoditam /
śuddhaiṣā yogikoṣasya vivṛtyeha pravartate // 4 //
ahaṃkāramayī bhūmir buddhibhūmeḥ parā hi sā /
tayā vyāptam idaṃ sarvaṃ saiva jñātā vimuktidā // 5 //
saṃvidrūpā parā śaktir abhinnā pārameśvarī /
jñānaśaktiḥ punaḥ saiva mantrarūpā prakāśate // 6 //
ahaṃkāraḥ smṛto mantraḥ śivaśaktyātmarūpakaḥ /
yad atrādyam avarṇākyaṃ sa śivaḥ parameśvaraḥ // 7 //
nādabindvātmakaṃ yat tu dvitīyā śaktir aiśvarī /
dvayam etat samākhyātaṃ bījabhūtaṃ carācare // 8 //
anena mudritāḥ sarve varṇā aṅgārarūpiṇā /
śabdabrahma tataḥ prāhur yoginas tattvacintakāḥ // 9 //
aprāṇam asvaraṃ dhyātvā paśyantyā nirvibhāgayā /
cinmātrāmarśayogena vilīnaḥ syāt pare pade // 10 //
tathā hi siddhair ādiṣṭaṃ kvacit siddhāntasaṃgrahe /
nābher ūrdhvaṃ vitastis tu kaṇṭhādhastāt ṣaḍaṅgulam // 11 //
hṛdayaṃ madhyadeśe tu padmaṃ tu hṛdi madhyataḥ /
tatra madhye sthitā sūkṣmā karṇikā tripathopagā // 12 //
tatra madhye hy akāraṃ tu svātmatattvaṃ vyavasthitam /
akārāntaḥ paraṃ sūkṣmaṃ cinmātratattvalakṣaṇam // 13 //
tasyāpy upāsā kathitā vijñāne bhairavena tu /
abindum avisargaṃ ca akāraṃ japato mahān // 14 //
udeti devi sahasāj jñānaughaḥ parameśvaraḥ /
karṇasthāne nirādhārā dvitiyā śaktir aiśvarī // 15 //
bījabhūtā kuṇḍalinī nādabindusvarūpiṇī /
prāṇaśaktir iyaṃ dehe hakārākhyā nadaty alam // 16 //
karṇāntaravivare yogī sthagitvā lakṣayet sphuṭam /
bindurūpā bhruvor madhye viśvasya jananī dhruvā // 17 //
lakṣyate yogibhir nityaṃ bhrājamāna svatejasā /
tasya bimbau tu candrārkau vāmadakṣinanetrayoḥ // 18 //
nirāśrayau tu tau kṛtvā viṣed brahma sanātanam /
vijñānabhairave 'py uktaṃ śastre siddhaniṣevite // 19 //
sampradāyam imaṃ samyak śṛṇu devi vadāmy aham /
kaivalyaṃ jāyate sadyo netrayoḥ stabdhamātrayoḥ // 20 //
saṃkocaṃ karṇayoḥ kṛtvā hy adhodvāre tathaiva ca /
anackam ahalaṃ dhyāyan viṣed brahmabilaṃ kṣaṇāt // 21 //
iti siddhamukhodgītaṃ svayam eva udāhṛtam /
mārgenānena saṃsiddhā labhante yoginīpadam // 22 //
jākadeśasamudbhūto mīmāṃsāvanakesarī /
harṣadattetināmā yas tasya sūnor iyaṃ kṛtiḥ // 23 //
