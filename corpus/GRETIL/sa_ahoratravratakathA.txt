#Text:Ahoratravratakathā
#Data entry:Klaus Wille
#Edition:R. Handurukande, Three Sanskrit Texts on Caitya Worship. In Relation to the Ahorātravrata, Tokyo 2000 (Studia Philologica Buddhica, Monograph Series, 16).

||ahorātravratakathā
oṃ namaḥ śrīdharmadhātave /
ākāśanirmalo bhūto niṣprapañcaguṇāśrayaḥ /
pañcaskandhātmakaḥ śāntas tasmai stūpātmane namaḥ // 1 //
evaṃ mayā śrutam ekasmin samaye buddho bhagavāñchrāvastyāṃ viharati sma jetavane 'nāthapiṇḍadasyārāme mahatā bhikṣusaṃghena sārdham anekair devanāgayakṣagandharvāsuragaruḍakinnaramahoragaiḥ sthavirabhikṣuvītarāgasaṃghair bodhisattvaśatasahasraiḥ saṃbahulaiś ca śrāvakair daśabhir lokapālaiś caturmahārājakāyikair anekaiś caturvarṇakair manuṣyair anekaśatasahasraiḥ parivāraiḥ parivṛtaiḥ /
tasmin sabhāmaṇḍale sarve dharmaśravaṇikā lokā bhagavantam abhivīkṣya caityavratakathām anudhyāyantaḥ samāsthitāḥ /
atha khalu subhūtiḥ sthaviraḥ svāsanāt samutthāyaikāṃsam uttarāsaṅgaṃ kṛtvā triḥ pradakṣiṇīkṛtya savyajānubimbaṃ bhuvi sthāpya bhagavantaṃ praṇamyaitad avocat /
ahorātravrataṃ nātha kasmin kāle pravartate /
tadvidhiṃ ca kathaṃ brūhi tat sarvaṃ jagatāṃ prabho // 2 //
bhagavān āha /
sādhu sādhu subhūte tvaṃ śṛṇu sattvānukampaka /
ahorātravrataṃ nityaṃ tadvidhiṃ kathitaṃ mayā // 3 //
āsīt pūrvavideheṣu nāmnā gandhavatī purī /
tasyāṃ rājā mahātejā indrapṛṣṭo mahāprabhuḥ // 4 //
so 'ntaḥpure sabhāṃ kṛtvā vasubandhuṃ jagadgurum /
praṇamya sahasotthāya papraccha vidhivan mudā // 5 //
śrotum icchāmi tat sarvaṃ yad vratānāṃ samuttamam /
yasya puṇyaprabhāvena sattvā yānti sunirvṛtim // 6 //
tat sarvaṃ gadatāṃ dhīra sarvasattvānukampaka /
bhavatā cānupālyo 'yaṃ lokaś ca duṣkṛtākulaḥ // 7 //
vasubandhur uvāca /
śṛṇu rājan mayā khyātaṃ vratānāṃ vratam uttamam /
ahorātravrataṃ pūrvaṃ vipaśyinā prabhāṣitam // 8 //
aśvinīpūrṇamāsyāṃ tu vrataṃ kuryād mahodayam /
caityaṃ saṃpūjayec chuddhaḥ ṣoḍaśair vidhibhis tathā // 9 //
bhojanānte trayodaśyāṃ kuryād dantādiśodhanam /
caturdaśyāṃ viśeṣeṇa kṣurakarmāṇi kārayet // 10 //
snānaṃ kṛtvā ca tīrtheṣu śucivastrāvṛtaḥ sudhīḥ /
madhyāhne pūjayec caityaṃ pradīpaṃ ca pradāpayet // 11 //
sūrye ṣaṣṭhagate bhāge prāśayec ca kṣīrodanam /
punar śaucādikaṃ kṛtvā dharmaṃ ca śruṇuyān mudā // 12 //
pratyūṣe pūrṇamāsyāṃ tu snātvā vrataṃ samārabhet /
ādau bhūmitalaṃ śodhya dharmaśālāṃ prakārayet // 13 //
gandhodakaiḥ suśītaiś ca caityapuṃgavaṃ snāpayet /
gokṣīraiḥ pañcaratnaiś ca śilābhasmābhilepayet // 14 //
catvāraś ca dhvajāḥ sthāpyā vitānaṃ saha cāmaraiḥ /
digmālā vividhaiḥ puṣpair lambayec ca jinālaye // 15 //
tadanu vratam ārabhya guruṃ vajradharaṃ namet /
triratnaśaraṇaṃ kṛtvā dhyāpayed dharmadhātukam // 16 //
dharmadhātumayaṃ caityaṃ jinānāṃ gṛham uttamam /
anādibuddha-m-ākhyātaṃ sarvatathāgatālayam /
sarvadevālayaṃ caityam ārādhya vratam ācaret // 17 //
sahasradalapadmasthaṃ candrabimbaprabhāsvaram /
raktaṃ bhoḥkārasaṃbhūtaṃ caturmukhaṃ virājitam // 18 //
raktagauramukhaṃ mūlaṃ dakṣiṇe kuṅkumāruṇam /
paścime padmaraktaṃ ca [!] uttare pītaraktakam // 19 //
aṣṭabhujaṃ virājantaṃ vāgīśvaraṃ jagadgurum /
dhanurbāṇadharaṃ dvābhyāṃ pāśāṅkuśadharaṃ tathā /
prajñāpustaṃ tathā khaḍgaṃ vajraghaṇṭāṃ tathāparaiḥ // 20 //
sarvarasasamāyuktaṃ lalitāsanasaṃsthitam /
divyābharaṇabhūṣāṅgaṃ pañcabuddhamukūṭinam // 21 //
vāgīśvaraṃ mahāśuklaṃ sarvadharmapratiṣṭhitam /
evaṃ svahṛdi saṃbhāvya puraḥ sthāpya prapūjayet // 22 //
pādyācamanam arghaṃ ca dattvā bhaktisamanvitaḥ /
jalajaiḥ sthalajaiḥ puṣpaiḥ sugandhaiḥ pañcagandhakaiḥ // 23 //
sāṃyojikaiḥ sugandhaiś ca sudhūpair dhūpanais tathā /
ghṛtaprayuktadīpaiś ca naivedyair divyabhojanaiḥ // 24 //
chattradhvajapatākābhir nṛtyagītasuvādanaiḥ /
phalamūlādibhiḥ pattraiḥ pūgatāmbūlakais tathā // 25 //
nānauṣadhisamāyuktair divyair mānuṣyakais tathā /
kundapuṣpākṣatair dūrvair lājayā pañcaratnakaiḥ // 26 //
kṣīraiḥ śaṃkhasupuṣpārghaṃ pradadyāc caityapuṃgave /
dhāraṇīṃ caiva stotraṃ ca paṭhed gītaṃ ca gāyayet // 27 //
pradakṣiṇāsahasraṃ ca yāvat kuryāt pramoditaḥ /
rātriṃ divā susaṃpūrṇaṃ pracarec ca pradakṣiṇām // 28 //
yatra devālaye tīrthe sarvān devān prapūjayet /
hīnajātiprasaṅgaṃ ca sparśanaṃ naiva kārayet // 29 //
sparśite saṃcaret snātvā pañcagavyena śodhayet /
asparśe 'pi caret snānaṃ trisaṃdhyaṃ saṃcared vratī // 30 //
ahorātraṃ susaṃpūrṇaṃ na nidrāṃ kārayed vratī /
caityadhyānasamāyuktaḥ japan stotraṃ paṭhaṃś caret // 31 //
nānāgītamahotsāhair nṛtyair vādyaiḥ sughoṣakaiḥ /
hāhākāraiḥ samullāsaiḥ kārayet saṃpradakṣiṇām // 32 //
viśrāmaṃ vā na kuryāc ca khedaṃ vā vratamānasaḥ /
ahorātre susaṃpūrṇe prātaḥ snātvā vrataṃ caret // 33 //
caityapūjāṃ yathā kṛtvā gurupūjāṃ samācaret /
annaṃ vastraṃ suvarṇaṃ ca vrīhiratnāni bhūṣitān // 34 //
ācāryāya pradātavyā yathāśakti subhaktitaḥ /
yajñaṃ kuryāt tato vajrī vrataṃ pūrṇaṃ prapūrayet // 35 //
abhiṣekaṃ tato dattvā pañcasūtraṃ ghaṭaṃ tathā /
pratyekaṃ vratināṃ dattvā āśīrvādyaṃ pradāpayet // 36 //
kaumārīpūjanaṃ kṛtvā śeṣānnaṃ ca prabhuñjayet /
dadhiśarkarasaṃyuktaṃ pañcāmṛtaṃ subhojanam // 37 //
gaṇacakraṃ tataḥ kuryād vidhijñas tadvidhiṃ yathā /
ity evaṃ vratarājaṃ tad yaḥ kuryāc chuddhamānasaḥ // 38 //
vipulaṃ bhogam ābhujya sarvasukhasamanvitaḥ /
paratra sukham āpnoti dharmaloke mahīpate // 39 //
indrapṛṣṭa uvāca /
bhagavan sarvavin nātha punar vaktuṃ tvam arhasi /
saṃsāre bahudhā jātis teṣāṃ pracāraṇaṃ katham // 40 //
vratapūjāphalaṃ sarvaṃ kālaṃ ca samayaṃ dinam /
kutra sthāne vrataṃ kāryaṃ tat sarvaṃ vada sāṃpratam // 41 //
vasubandhur uvāca /
śṛṇu rājan mahābhāga saṃkṣepād gaditaṃ mayā /
svayaṃbhuvo viśeṣeṇa sthāne puṇyaṃ mahottamam // 42 //
tadabhāve svadeśeṣu sthāvare caityapuṃgave /
tadabhāve svayaṃ kṛtvā kārayitvāparais tathā // 43 //
suvarṇena kṛte vāpi ratnaiś cānyaiś ca dhātubhiḥ /
pāṣāṇena tathā kāṣṭhair iṣṭikāmṛnmayais tathā // 44 //
vālukaudanapiṇḍena kṛte paṭṭe ca citrite /
devālaye nadītīre tīrthe puṇye mahītale // 45 //
dharmaśālāṃ viśodhyādau yathāvidhi vrataṃ caret /
puṣpākṣatasugandhaiś ca vāriṇā caityapuṃgave // 46 //
arghaṃ dadyād raveḥ pūrvaṃ puṣpaṃ kṣipet salājakam /
maṇḍalaṃ pañcadevānāṃ likhed gomayavāriṇā // 47 //
avandhyākṣarayuktena adbhir vratīn susiñcayet /
havargāntākṣarair mantrair vratimūrdhni pratiṣṭhitam // 48 //
oṃ āḥ hūm iti mantreṇa diśobandhaṃ satarjitam /
aṅganyāsaṃ karanyāsaṃ kṛtvā guruṃ ca vandayet // 49 //
puṣpākṣataṃ kare dhṛtvā kṣipet puṣpāñjalitrayam /
pañcabhūtasamutpannaṃ padmakarṇikasaṃsthitam // 50 //
āvāhayed dharmadhātuṃ vāgīśvaraṃ samujjvalam /
jñānarūpaṃ samākṛṣya saṃsthāpya samaye 'rcayet // 51 //
āvāhayāmi tvaṃ nātha tiṣṭha tatra manohare /
sthiro bhava sadā mā gāḥ sāṃnidhyaṃ kuru sthāvare // 52 //
dugdhākṣatāmbusaṃyuktaṃ navaratnāni hāṭakaiḥ /
śaṅkhasthaṃ pratigṛhṇātu arghaṃ dattaṃ mayā mudā // 53 //
paṭṭavastrāṇi divyāni cīvarāṇi mahāprabho /
nānāraṅgavicitrāṇi pratigṛhṇātu modatām // 54 //
candanaṃ malayodbhūtaṃ kastūrikuṅkumādikam /
śaileyaṃ ghanasāraṃ ca gṛhyatām anumodatām // 55 //
jalajān sthalajān divyān pañcavarṇena granthitān /
sumodavāsanāyuktān puṣpān gṛhya pramodatām // 56 //
dhūpo 'yaṃ ghrāṇasaṃtarpo divyagandhasuyojitaḥ /
pradattas tava toṣārthaṃ gṛhyatāṃ bhaktavatsala // 57 //
dīpo 'yaṃ harati dhvāntaṃ sarvatimiranāśakaḥ /
ghṛtatailasamujjvālo mayā dattaḥ pragṛhyatām // 58 //
annaṃ caturvidhaṃ nātha rasānāṃ ṣaḍvidhaṃ tathā /
ghṛtādyān śarkarāhārān bhuñjatāṃ ca sukhaṃ mama // 59 //
nānāphalasumodyaṃ ca koṣakāraṇḍavādikam /
skandhamūlādikaṃ pattraṃ gṛhāṇa pratimodatām // 60 //
tāmbūlapūgasaṃyuktaṃ karpūranirmalāni ca /
divyauṣadhigaṇādīni gṛhyatāṃ hitakāṅkṣayā // 61 //
lājākṣataphalārcaiva dūrvākundāni bhaktitaḥ /
prakṣiptāni mayā tubhyaṃ gṛhāṇa hitamānasaḥ // 62 //
sauvarṇādikṛtaṃ chattraṃ dhvajāni vividhāni ca /
patākādīni sarvāṇi pragṛhya jina modatām // 63 //
dhīmān pujāṃ tathā kṛtva stotram evaṃ ca kārayet /
jinendratoṣaṇārthāya kārayec ca pradakṣiṇām // 64 //
dharmadhātumayaṃ caityaṃ sarvalokasvarūpiṇam /
ālayaṃ sarvabuddhānāṃ jinadhātuṃ namāmy aham // 65 //
namas te jinadhātūnāṃ pratyakṣaṃ samalīlayā /
triyānasamayānandaṃ satataṃ tvāṃ samāśraye // 66 //
jinajaṃ sakalasthaṃ ca puṇyakṣetraṃ namāmy aham /
pavitrajanasukhārthaṃ sevitārthapradāyakam // 67 //
praṇamāmi muniśreṣṭhaṃ maṅgalārthapradāyakam /
janānāṃ bhayahantāraṃ lokanāthaṃ maheśvaram // 68 //
devatāḥ sarvalokasthāḥ saṃbhūtā jinamaṇḍale /
dadyāt tebhyo baliṃ tuṣṭyā pūjābhaktyarcanair ataḥ // 69 //
bahumaṅgalavādyais ca chattrasindūrapātrayā /
ācāryaḥ svastivākyaiś ca kārayec ca mahotsavam // 70 //
aśvinīpūrṇamāsyāṃ tu (...) ravivāsare /
āyuṣmatīyute yoge praśastaś ca viśeṣataḥ // 71 //
yaḥ prakuryād vrataṃ tasmin vipulaṃ bhogam āpnuyāt /
paratra sukhasaṃprāpto jinaloke mahīpate // 72 //
śṛṇu rājan mahābhāga vakṣye pūjāphalaṃ punaḥ /
yena saṃsthāpitaṃ caityaṃ vītaśoko bhaved asau /
māyāmohavinirmuktas trimalenāviliptitaḥ // 73 //
pañcāmṛtaiś candanagandhagarbhair ye snāpayantī jinacaityabimbam /
mandākinītoyasugandhagandhais te cāpsarobhī ratim āvahanti // 74 //
ye dhūpayanti kṣaṇam atra lokāḥ śokā na teṣāṃ prabhavanti jātu /
surāsuraiḥ pūjitapādapadmāḥ sugandhamālyāmbarabhūṣitāṅgāḥ // 75 //
sugandham asmā upaḍhaukayanti prasannacittā manujā ajasram /
sugandhidehā ravibhānudīptā bhavanti lokāḥ paramārthalokāḥ // 76 //
ye kuṅkumādyai racitaṃ suvastraṃ dadanti ye caityatathāgatebhyaḥ /
śītārtiduḥkhaṃ na hi jātu teṣāṃ taddhetutaḥ koṣaviśālasāram // 77 //
aśeṣapuṣpaṃ jalajaṃ bhavantaṃ śvetāruṇaśyāmasupītakṛṣṇam /
samantabhadrāya viśālavaṃśam ārogyasaukhyaṃ prabhavanti dānāt // 78 //
śraddhāratā ye manujā udārair mālāsugandhair bahupuṣpayuktaiḥ /
caranti pūjāṃ bahubhaktitas te samastasaukhyāni labhanti loke // 79 //
netrābhirāmā bahuratnakoṣā narādhipair arcitapādapadmāḥ /
jñānapradīpāhatamohajālā ye dīpamālā racayanti buddhe // 80 //
naivedyam asmai pradadāti loke yo bhaktiyuktaḥ surasaṃ sugandhim /
balī viśiṣṭo naracakracūḍāmaṇyarcitāṅghrī nṛpasevyamānaḥ // 81 //
phalamūlādikaṃ dattvā caityeṣu bhaktimānasaḥ /
yatheṣṭaṃ phalam āpnoti nirogī sukhavān sadā // 82 //
rogādibhiḥ prabaladuḥkhakarair vimuktāḥ snigdhānanāḥ kanakatulyamanojñavarṇāḥ /
rājyaṃ hi yad vigatakaṇṭakam āpnuvanti bhaiṣajyadānavidhinā tad uśanti caitye // 83 //
tāmbūlādigaṇaṃ dattvā rūpavān sukhavāñ chuciḥ /
divyāṅganā manohārī rājalakṣmīm avāpnuyāt // 84 //
vitānam uccair vitanoti caitye dhanyas triloke paripūjanīyaḥ /
viśālavaṃśo guṇarakṣaṇīyo jāto mahādīptaśarīrabhāsaḥ // 85 //
dhvajān patākān abhiropayanti ye caityabimbe khalu bhaktiyuktāḥ /
mahīśvarās te jitaduṣṭasaṃghā bhavanti nāthā divi bhūtale ca // 86 //
chattrāṇi nānāmaṇibhir mahārhaiḥ saṃpādayanty āttamanā dadanti /
caityeṣu bhūmīśvaravṛndavandyāḥ śrīsaṃyutā dharmaratā bhavanti // 87 //
vīṇādibhir ghaṇṭamṛdaṅgavādyaiḥ śṛṅgaiś ca śaṅkhādisudivyaghoṣaiḥ /
saṃghoṣaṇāṃ ye vitaranti caitye te 'bhijñayā pañcabhir āśrayante // 88 //
mahotsāhakarair gītaiḥ sumanojñasvarair jine /
nṛtyaṃ ca kārayen nityaṃ kuryāc ca divyaghoṣaṇām // 89 //
te narāḥ susvarā bhonti sundarā divi bhūtale /
divyaśrotrā mahāvijñāḥ suśabdaṃ śṛṇuyuḥ sadā // 90 //
nṛtyagītādikaṃ kṛtvā caityabimbe jinālaye /
vandyamānāḥ sugotrāś ca niṣpāpāś ca bahuśrutāḥ // 91 //
dīrghāyuṣkāḥ sahasrākṣā nirāpadāḥ guṇāśrayāḥ /
grahavyādhivimuktāṅgā bhavanti te śubhākarāḥ // 92 //
trāsāya duṣṭasattvānāṃ rakṣārthaṃ ca jinālaye /
śastrahastāḥ sthitā ye ca jinabhaktisamanvitāḥ // 93 //
āyuṣmanto nirogāś ca putrapautrīsamanvitāḥ /
bhāgyavanto dhanāḍhyāś ca gacchanty ante jinālayam // 94 //
dūrvākundākṣatān puṣpān lājayā saha bhaktimān /
prakṣipej jinacaityeṣu svastivākyam udāharan // 95 //
durgatiṃ nābhijānāti sukulī lokanandanaḥ /
sarvavidyāgurur vandyaś cānte mokṣam avāpnuyāt // 96 //
ye ramamuktāmaṇivajrahemaṃ rūpyaṃ suvarṇādikadhātudānāt /
vinaṣṭaśokāḥ subhagāḥ suvaṃśāḥ pūrṇendriyāḥ pūrṇamanorathāḥ syuḥ // 97 //
ye śuddhacittāḥ stutim ācaranti gadyātmikāṃ padyamayīṃ vicitrām /
te śabdasaṃghān bahuvādyajātān śṛṇvanti gītān surabhāminīnām // 98 //
yāvatī bhūmir ākrāntā hy adhaḥ kāñcanacakrataḥ /
yojanānāṃ sahasrāṇi rāśibhyaḥ parikīrtitāḥ // 99 //
yāvatyo vālukās tatra tāvatyaḥ parisaṃkhyayā /
rājā bhavati vīreśaś cakravartī mahītale // 100 //
ye jātamātrāḥ prabhutāṃ prayānti śreṣṭhīkule janma sadaiva yeṣām /
hastyaśvayānaiś ca paribhramanti kṛtvātra te caityavarapraṇāmam // 101 //
akāra-m-ādyakṣaramantrarājair balyūpahāraṃ khalu yo hi dadyāt /
tasmai sudevāḥ sagaṇāḥ sutṛptā dāsyanti satkarmaphalābhilāṣam // 102 //
yasmin sudhālepam udācaranti te rogaśokādivimuktadehāḥ /
dhaneśvarā dīrghatarāyuṣas te narāmarāṇāṃ subhagā bhavanti // 103 //
ye buddham uddiśya mahānti nityaṃ kurvanti mṛdgomayalepanāni /
bhaktyā suśuddhā jalamaṇḍalaṃ vā labhanti te rājabalaṃ subhāgyāḥ // 104 //
stūpāṅgaṇaṃ dhātuvaraṃ vihāraṃ ye śodhayantīha narāḥ prayatnaiḥ /
nirmālyam ebhyaś ca samuddharanti te hemavarṇāḥ sudṛśo bhavanti // 105 //
pradakṣiṇāṃ ye vitaranti tasmin bhavanti jātismaralābhinas te /
suvarṇavarṇā bahuputrapautrāḥ pūjyā manuṣyāmaranāthasaṃgḥaiḥ // 106 //
ālasyenāpi yaḥ kuryāj jinālayapradakṣiṇām /
pade pade suvarṇaikakarṣadānaphalaṃ labhet // 107 //
brahmakṣatriyavaiśyānāṃ sacchūdrāṇāṃ prapūjanam /
tebhyo vihīnajātīnām adhikāro na pūjane // 108 //
gandhapuṣpadvijātīnāṃ ... ca pūjayet /
gandhapuṣpadvijātīnāṃ viśeṣeṇa prakathyate // 109 //
chattrāvarohaṇaṃ rājñāṃ vaiśyānāṃ tv annaḍhaukane /
dhvajapātākavastreṣu śudrāṇāṃ ca pracakṣyate // 110 //
svarṇakārā lohakārāḥ kāṃsyakṛtkṛṣikādayaḥ /
eteṣāṃ ceha hastena caityapuṃgavapūjanam // 111 //
taṇḍulī vyañjanī cāpi krayavikrayakāriṇaḥ /
etaiḥ svavṛttivastūni ḍhaukanīyāni bhaktitaḥ // 112 //
śaulinī dhvananavartī māṃsavartī tathā surī /
etair dīpaḥ pradātavyaḥ kuladharmapramāṇataḥ // 113 //
carmakāraś ca mātaṅgo niyogī rajako dhvajī /
etaiś ca dūrataḥ sthitvā vandanīyo jinālayaḥ // 114 //
ṣaṭtriṃśajjātibhir evaṃ svakulavṛtticāraṇaiḥ /
anyaiś ca vividhair nityaṃ pūjyo mānyo jinālayaḥ // 115 //
yeṣāṃ vratavikāro na pūjane jinapuṃgave /
etaiḥ pradakṣiṇāṃ kṛtvā vandanīyaḥ svabhaktitaḥ // 116 //
evaṃ bhūyaś cared dharmī vratarājam aharniśam /
tasya puṇyaṃ yathā proktaṃ kuṣṭhavyādhikṣayaṃ vrajet // 117 //
pañcānantaryapāpāni yāsyanty api na saṃśayaḥ /
kiṃ punaś cānyapāpāni janmāntarakṛtāny api // 118 //
na hī sa vidyate rājan sarvatra bhuvaneṣv api /
ahorātravrataṃ puṇyaṃ yo 'nyathātvaṃ kariṣyati // 119 //
śravād api haret pāpaṃ darśanāt puṇyam āpnuyāt /
anumodāt tathā caivaṃ vratināṃ tu kim ucyate // 120 //
vratināṃ saptabhāgāni kṛtvā puṇyaṃ viśāradaḥ /
ekabhāgaṃ labhed rājā caikaṃ tu sārathījanaḥ /
sarvasattvasya caikaṃ hi tac cheṣaṃ labhate vratī // 121 //
iti śrutvā guror vākyam indrapṛṣṭo mahāprabhuḥ //
tataḥ sattvahitārthāya vrataṃ kartuṃ samudyataḥ // 122 //
vārāṇasīpuraṃ gatvā vasubandhuvaco yathā /
vihāraṃ ca pratiṣṭhāpya caityarājaṃ jinālayam /
saṃghabhojyaṃ tataḥ kṛtvā yayau svargaṃ janaiḥ sudhīḥ // 123 //
tato bhikṣavo bhagavantam etad ūcuḥ /
bhagavan muniśārdūla punar vaktuṃ tvam arhasi /
kena kena purā nātha vratarājam idaṃ kṛtam // 124 //
bhagavān āha /
purā svargeṣu śakreṇa ahorātravrataṃ kṛtam /
sa ṣaṭpāramitāṃ dhārya caityarājam apūjayat /
pradakṣiṇāṃ sadā kṛtvā yathāvidhi mahotsavaiḥ // 125 //
pātāle ca purā vīro daityendro balavān abhūt /
nirghoṣadamano nāma devāsurapramardakaḥ // 126 //
indraṃ svarge nirākṛtya pṛthivīm avadhāritam /
tenāsurasahāyena trailokyaṃ svavaśe kṛtam // 127 //
pṛthivīmaṇḍalaṃ gatvā gato 'sau maghavatpurim /
indrapṛṣṭamahāsthāne devatāpratimāṃ sthitām /
pañcapradakṣiṇāṃ kṛtvā prajñāpārasthito 'bhavat // 128 //
pātāleṣu punar gatvā so 'carat tad vratottamam /
nāginyo nāgarājāś ca pātāleṣv acaraṃs tathā // 129 //
nāgaiḥ phaṇavibhūṣābhir devanāgendrakalpitam /
pūjitaṃ satataṃ bhaktyā caityarājaṃ vidhānataḥ // 130 //
ṛṣayaś ca purā sapta jalāśrayapratiṣṭhitam /
pupūjuḥ jinadhātusthaṃ vidhibhiś ca viśāradāḥ // 131 //
nāginyo viṣadarpāndhāḥ pūjyamānāñ jinālayam /
tāpitān kārayām āsur viṣāṅgāraiḥ pradāhanaiḥ // 132 //
tatas taiḥ śāpitā evaṃ nāginyaḥ śambukāḥ khalu /
bhavadhvaṃ saptajanmāni punaś caityasya sevakāḥ // 133 //
tataś cānte punaḥ prāpya mānuṣyaṃ puṇyabhāvataḥ /
eṣā sā nāyakā yā tu dharmadakṣaṃ patiṃ labhet // 134 //
tatas ta vañcitāḥ sarvāḥ śambukā dharmamānasāḥ /
dhānyāṃ pradakṣiṇāṃ cakrur bhāvayantyo jinālayam // 135 //
tataḥ puṇyānubhāvena devyaḥ saptakumārikāḥ /
bhūtā rājasutā dhanyāḥ prālabhan paramaṃ padam // 136 //
tāsām ekā vratī nāmnā dharmadakṣapativratā /
tasyāḥ puṇyaprabhāvena dharmadakṣo 'labhad vratam // 137 //
dharmadakṣo mahārājā yathāvidhi vrataṃ caran /
bhāryayā saha dharmātmā sukhāvatyāṃ mahīpatiḥ // 138 //
ity evaṃ bhikṣavo jñātvā vratānāṃ vratam uttamam /
caityasevā sadā kāryā vratapuṇyapracāraṇaiḥ // 139 //
caityavratasamaṃ puṇyaṃ nāsti satyaṃ tribhūvane /
tasmāc caityaḥ sadā vandyaḥ pūjanīyaḥ prayatnataḥ // 140 //
||ity ahoratravratakathā
