praṇamya paramātmānaṃ vakṣyāmi hitakāmyayā /
sattvānām alpabuddhīnāṃ śāstravārttāsamuccayam // 1,1.1 //
yaṃ śrutvā sarvaśāstreṣu prāyas tattvaviniścayaḥ /
jāyate dveṣaśamanaḥ svargasiddhisukhāvahaḥ // 1,1.2 //
duḥkhaṃ pāpāt sukhaṃ dharmāt sarvaśāstreṣu saṃsthitiḥ /
na kartavyam ataḥ pāpaṃ kartavyo dharmasaṃcayaḥ // 1,1.3 //
hiṃsānṛtādayaḥ pañca tattvāśraddhānam eva ca /
krodhādayaś ca catvāra iti pāpasya hetavaḥ // 1,1.4 //
viparītāstu dharmasya eta evoditā buddhaiḥ /
eteṣu satataṃ yatnaḥ samyak kāryaḥ sukhaiṣiṇā // 1,1.5 //
sādhusevā sadā bhaktyā maitrī sattveṣu bhāvataḥ /
ātmīyagrahamokṣaś ca dharmahetuprasādhanam // 1,1.6 //
upadeśaḥ śubho nityaṃ darśanaṃ dharmacāriṇām /
sthāne vinaya ityetat sādhusevāphalaṃ mahat // 1,1.7 //
maitrīṃ bhāvayato nityaṃ śubho bhāvaḥ prajāyate /
tato bhāvodayājjantor dveṣāgnir upaśāmyati // 1,1.8 //
aśeṣadoṣajananī niḥśeṣaguṇaghātinī /
ātmīyagrahamokṣeṇa tṛṣṇāpi vinivarttate // 1,1.9 //
evaṃ guṇagaṇopeto viśuddhātmā sthirāśayaḥ /
tattvaviddhiḥ samākhyātaḥ samyag dharmasya sādhakaḥ // 1,1.10 //
upādeyaś ca saṃsāre dharma eva buddhaiḥ sadā /
viśuddho muktaye sarvaṃ yato 'nyad duḥkhakāraṇam // 1,1.11 //
anityaḥ priyasaṃyoga iherṣyāśokavatsalaḥ /
anityaṃ yauvanaṃ cāpi kutsitācaraṇāspadam // 1,1.12 //
anityāḥ sampadas tīvrakleśavargasamudbhavāḥ /
anityaṃ jīvitaṃ ceha sarvabhāvanibandhanam // 1,1.13 //
punarjanma punarmṛtyur hīnādisthānasaṃśrayaḥ /
punaḥ punaś ca yad ataḥ sukham atra na vidyate // 1,1.14 //
prakṛtyasundaraṃ hy evaṃ saṃsāre sarvam eva yat /
ato 'tra vada kiṃ yuktā kvacidāsthā vivekinām // 1,1.15 //
muktvā dharmaṃ jagad vandyam akalaṅkaṃ sanātanam /
parārthasādhakaṃ dhīraiḥ sevitaṃ śīlaśālibhiḥ // 1,1.16 //
āha tatrāpi no yuktā yadi samyag nirūpyate /
dharmasyāpi śubho yasmād bandha eva phalaṃ matam // 1,1.17 //
na cāyasasya bandhasya tadā hemamayasya ca /
phale kaścid viśeṣo 'sti pāratantryāviśeṣataḥ // 1,1.18 //
tasmād adharmavat tvājyo dharmo 'py evaṃ mumukṣubhiḥ /
dharmādharmakṣayānm uktir munibhir varṇitā yataḥ // 1,1.19 //
ucyate evam evaitat kintu dharmo dvidhā mataḥ /
saṃjñānayoga evaikas tathānyaḥ puṇyalakṣaṇaḥ // 1,1.20 //
jñānayogas tapaḥ śuddham āśaṃsādoṣavarjitam /
abhyāsātiśayād uktaṃ tad vimukteḥ prasādhanam // 1,1.21 //
dharmas tadapi cet satyaṃ kiṃ na bandhaphalaḥ sa yat /
āśaṃsā varjito 'nyo 'pi kiṃ naivaṃ ced na yat tathā // 1,1.22 //
bhogamuktiphalo dharmaḥ sa pravṛttītarātmakaḥ /
samyagmithyādirūpaś ca gatis tantrāntareṣv api // 1,1.23 //
tam antareṇa tu tayoḥ kṣayaḥ kena prasādhyate /
sadā syān na kadācid vā yady ahetuka eva saḥ // 1,1.24 //
tasmād avaśyam eṣṭavyaḥ kaścid hetus tayoḥ kṣayeṃ /
sa eva dharmo vijñeyaḥ śuddho muktiphalapradaḥ // 1,1.25 //
dharmādharmakṣayān muktir yac coktaṃ puṇyalakṣaṇam /
heyaṃ dharmaṃ tadāśritya na tu saṃjñānayogakam // 1,1.26 //
atas tatraiva yuktāsthā yadi samyag nirūpyate /
saṃsāre sarvam evānyat darśitaṃ duḥkhakāraṇam // 1,1.27 //
tasmāc ca jāyate muktir yathā mṛtyādivarjitā /
tathopariṣṭād vakṣyāmaḥ samyakśāstrānusārataḥ // 1,1.28 //
idānīṃ tu samāsena śāstrasamyaktvam ucyate /
kuvādiyuktyapavyākhyānirāsenāvirodhataḥ // 1,1.29 //
||(2) bhūtacaitanyavāda-khaṇḍana
pṛthivyādimahābhūtakāryamātram idaṃ jagat /
na cātmadṛṣṭasadbhāvaṃ manyante bhūtavādinaḥ // 1,2.30 //
acetanāni bhūtāni na taddharmo na tatphalam /
cetanāsti ca yasyeyaṃ sa evātmeti cāpare // 1,2.31 //
yadīyaṃ bhūtadharmaḥ syāt pratyekaṃ teṣu sarvadā /
upalambhyeta sattvādikāṭhinatvādayo yathā // 1,2.32 //
śaktirūpeṇā sā teṣu sadāto nopalabhyate /
na ca tenāpi rūpeṇa satyasaty eva cen na tat // 1,2.33 //
śaktivetanayoraivyaṃ nānātvaṃ vātha sarvathā /
aikye sā cetaneveti nānātve 'nyasya sā yataḥ // 1,2.34 //
anabhivyaktir apy asyā nyāyato nopapadyate /
āvṛtir na yad anyena tattvasaṃkhyāvirodhataḥ // 1,2.35 //
na cāsau tatsvarūpeṇa teṣām anyatareṇa vā /
vyañjakatvapratijñānāt nāvṛtir vyañjakaṃ yataḥ // 1,2.36 //
viśiṣṭapariṇāmabhā-ve 'pi hy atrāvṛtir na vai /
bhāvatāptestathā nāmavyañjakatvaprasaṅgataḥ // 1,2.37 //
na cāsau bhūtabhinno yat tato vyaktiḥ sadā bhavet /
bhede tvadhikabhāvena tattvasaṃkhyā na yujyate // 1,2.38 //
svakāle 'bhinna ity evaṃ kālābhāve na saṅgatam /
lokasiddhāśraye tv ātmā hanta ! nāśrīyate katham // 1,2.39 //
nātmāpi loke no siddho jātismaraṇasaṃśrayāt /
sarveṣāṃ tadabhāvaś ca citrakarmavipākataḥ // 1,2.40 //
loke 'pi naikataḥ sthānād āgatānāṃ tathekṣyate /
aviśeṣeṇa sarveṣām anubhūtārthasaṃsmṛtiḥ // 1,2.41 //
divyadarśanataś caiva tacchiṣṭāvyabhicārataḥ /
pitṛkarmādisiddheś ca hanta ! nātmāpy alaukikaḥ // 1,2.42 //
kāṭhinyābodharūpāṇi bhūtāny adhyakṣasiddhitaḥ /
cetanā tu na tadrūpā sā kathaṃ tatphalaṃ bhavet ? // 1,2.43 //
pratyekam asatī teṣu na ca syād reṇutailavat /
satī ced upalabhyeta bhinnarūpeṣu sarvadā // 1,2.44 //
asat sthūlatvam aṇvādau ghaṭādau dṛśyate yathā /
tathāsatyeva bhūteṣu cetanāpīti cen matiḥ // 1,2.45 //
nāsat sthūlatvam aṇvādau tebhya eva tadudbhavāt /
asatastatsamutpādo na yukto 'tiprasaṅgataḥ // 1,2.46 //
pañcamasyāpi bhūtasya tebhyo 'sattvāviśeṣataḥ /
bhaved utpattir evaṃ ca tattvasaṃkhyā na yujyate // 1,2.47 //
na tajjananasvabhāvāś cet te 'tra mānaṃ na vidyate /
sthūlatvotpāda iṣṭaś cet tatsadbhāve 'py asau samaḥ // 1,2.48 //
na ca mūrttāṇusaṅghātabhinnaṃ sthūlatvam ity adaḥ /
teṣām eva tathābhāvo nyāyyaṃ mānāvirodhataḥ // 1,2.49 //
bhede tadadalaṃ yasmāt kathaṃ sadbhāvam aśnute /
tadabhāve 'pi tadbhāve sadā sarvatra vā bhavet // 1,2.50 //
na caivaṃ bhūtasaṅghātamātraṃ caitanyam iṣyate /
aviśeṣeṇa sarvatra tadvat tadbhāvasaṅgateḥ // 1,2.51 //
evaṃ sati ghaṭādīnāṃ vyaktacaitanyabhāvataḥ /
puruṣān na viśeṣaḥ syāt sa ca pratyakṣabādhitaḥ // 1,2.52 //
atha bhinnasvabhāvāni bhūtāny eva yatastataḥ /
tatsaṃghāteṣu caitanyaṃ na sarveṣv etad apy asat // 1,2.53 //
svabhāvo bhūtamātratve sati nyāyān na bhidyate /
viśeṣaṇaṃ vinā yasmān na tulyānāṃ viśiṣṭatā // 1,2.54 //
svarūpamātrabhede ca bhedo bhūtetarātmakaḥ /
anyabhedakabhāve tu sa evātmā prasajyate // 1,2.55 //
havir guḍakaṇikkādidravyasaṅghātajāny api /
yathā bhinnasvabhāvāni khādyakāni tatheti cet // 1,2.56 //
vyaktimātrata evaiṣāṃ nanu bhinnasvabhāvatā /
rasavīryavipākādikāryabhedo na vidyate // 1,2.57 //
tadātmakatvamātratve saṃsthānādivilakṣaṇā /
yatheyam asti bhūtānāṃ tathā sāpi kathaṃ na cet // 1,2.58 //
kartrabhāvāt tathā deśakālabhedādyayogataḥ /
na cāsiddhamado bhūtamātratve tadasaṃbhavāt // 1,2.59 //
tathā ca bhūtamātratve na tatsaṅghātabhedayoḥ /
bhedakābhāvato bhedo yuktaḥ samyag vicintyatām // 1,2.60 //
ekas tathāparo neti tanmātratve tathāvidhaḥ /
yatas tad api no bhinnaṃ tatas tulyaṃ ca tat tayoḥ // 1,2.61 //
syādetad bhūtajatve 'pi grāvādīnāṃ vicitratā /
lokasiddheti siddhaiva na sā tanmātrajā nanu // 1,2.62 //
adṛṣṭākāśakālādisāmagrītaḥ samudbhavāt /
tathaiva lokasaṃvitter anyathā tadabhāvataḥ // 1,2.63 //
na ceha laukiko mārgaḥ sthito 'smābhir vicāryate /
kiṃ tv ayaṃ yujyate kveti tvannītau coktavan na saḥ // 1,2.64 //
mṛtadehe ca caitanyam upalabhyeta sarvathā /
dehadharmādibhāvena tat taddharmādi nānyathā // 1,2.65 //
na ca lāvaṇyakārkaśyaśyāmatvair vyabhicāritā /
mṛtadehe 'pi sadbhāvād adhyakṣeṇaiva saṃgateḥ // 1,2.66 //
na cel lāvaṇyasadbhāvo na sa tanmātrahetukaḥ /
ata evānyasadbhāvād asty ātmeti vyavasthitam // 1,2.67 //
na prāṇādir asau mānaṃ kiṃ tadbhāve 'pi tulyatā /
tadabhāvād abhāvaś ced ātmābhāve na kā pramā // 1,2.68 //
tena tadbhāvabhāvitvaṃ na bhūyo nalikādinā /
saṃpādite 'py etat siddheḥ so 'nya eveti ced na tat // 1,2.69 //
vāyusāmānyasaṃsiddhes tatsvabhāvaḥ sa neti cet /
atrāpi na pramāṇaṃ vaś caitanyotpattir eva cet // 1,2.70 //
na tasyām eva saṃdehāt tavāyaṃ kena neti cet /
tattatsvarūpabhāvena tadabhāvaḥ kathaṃ nu cet // 1,2.71 //
tadvailakṣaṇyasaṃvitteḥ mātṛcaitanyaje hy ayam /
sute tasmin na doṣaḥ syān na na bhāve 'sya mātari // 1,2.72 //
na ca saṃsvedajādyeṣu mātrabhāvena tad bhavet /
pradīpajñātam apy atra nimittatvān na bādhakam // 1,2.73 //
itthaṃ na tadupādānaṃ yujyate tat kathaṃcana /
anyopādānabhāve ca tad evātmā prasajyate // 1,2.74 //
na tathābhāvinaṃ hetum antareṇopajāyate /
kiñcin naśyati naikāntād yathāha vyāsamaharṣiḥ // 1,2.75 //
nāsato vidyate bhāvo nābhāvo vidyate sataḥ /
abhayor api dṛṣṭo 'ntas tv anayos tattvadarśibhiḥ // 1,2.76 //
nābhāvo bhāvam āpnoti śaśaśṛṅge tathāgateḥ /
bhāvo nābhāvam etīha dīpaś cen na sa sarvathā // 1,2.77 //
evaṃ caitanyavān ātmā siddhaḥ satatabhāvataḥ /
paraloky api vijñeyo yuktimārgānusāribhiḥ // 1,2.78 //
