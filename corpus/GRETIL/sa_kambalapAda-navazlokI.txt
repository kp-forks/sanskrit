||āryāṣṭasāhaśrikāyāḥ prajñāpāramitāyāḥ piṇḍārthaḥ
prajñāpāramitāmbodhau śubharatnākare svayam /
sarvā pāramitās tatra tādātmyena vyavasthitāḥ // 1 //
niṣprapañcā nirābhāsā nirvikalpā nirālayā /
niḥsvabhāvā parā sūkṣmā bindunādavivarjitā // 2 //
prajñāpāramitā mātā sarvabuddhodayā parā /
trayānupalabdhirūpā sarvajñajñānagocarā // 3 //
prajñāpāramitāṃ saṃyag yo bhāvayitum icchati /
tenārthato navaślokāś cintanīyāḥ samāsataḥ // 4 //
karmaprabhāvasaṃbhūtaṃ ṣaḍāyatanalakṣaṇam /
punarbhavam iti khyātaṃ pratibhāsopamaṃ hi tat // 5 //
nirmitaṃ nagaraṃ yadvad vilokayati nirmitaḥ /
tadvat paśyati rūpāṇi karmabhir nirmitaṃ jagat // 6 //
dharmaṃ deśayataḥ śabdā ye kecit śrutigocarāḥ /
pratiśrutkopamāḥ sarve prodbhūtaśrutinaḥ śrutāḥ // 7 //
āghrataṃ svāditaṃ tathā spraṣṭaṃ viṣayalālāsaiḥ /
svapnatulyam idaṃ sarvam upalabdhaṃ na vidyate // 8 //
māyāyantro naro yadvad bhinnāṃ ceṣṭāṃ karoti vai /
tadvad ceṣṭāṃ karoty eva dehayantro nirātmakaḥ // 9 //
nānopalabdhayo yāś ca pratikṣaṇasamudbhavāḥ /
marīcisadṛśāś caite dṛṣṭanaṣṭāḥ vilakṣaṇāḥ // 10 //
pratibimbanibhaṃ grāhyam anādicittasaṃbhavam /
tadākāraṃ ca vijñānam anyonyapratibimbavat // 11 //
dhyāyināṃ svacchasaṃtāne yaj jnanendusamudbhavam /
udakacandropamaṃ tad dhi pratyakṣaṃ na vidyate // 12 //
yoginām api yaj jñānaṃ tad apy ākāśalakṣaṇam /
tasmāj jñānaṃ ca jñeyaṃ ca sarvam ākāśalakṣaṇam // 13 //
iti cintayataḥ tattvaṃ sarvabhāveṣv anāśritam /
bodhipraṇidhicittena jñānaṃ agraṃ bhaviṣyati // 14 //
āryāṣṭasāhasrikāyāḥ prajñāpāramitāyāḥ piṇḍārthaḥ samāptaḥ kṛtir iyaṃ śrīkambalāmbarapādanām
