#Author:Asaṅga
#Text:Mahāyānasūtrālaṃkāra
#Data entry:Jens Braarvig
#Edition:Sylvain Lévi: Mahāyānasūtrālaṃkāra, Exposé de la doctrine du Grand Véhicule, vol. I, Paris 1907.
#Notes:Some lines of the mula text are probably not complete. Mula text extracted from the commented version.

arthajño 'rthavibhāvanāṃ prakurute vācā padaiś cāmalair duḥkhasyottaraṇāya duḥkhitajane kāruṇyatas tanmayaḥ /
dharmasyottamayānadeśitavidheḥ satveṣu tadgāmiṣu śliṣṭām arthagatiṃ niruttaragataṃ pañcātmikāṃ darśayan // 1.1 //
ghaṭitam iva suvarṇaṃ vārijaṃ vā vibuddhaṃ sukṛtam iva subhojyaṃ bhujyamānaṃ kṣudhārtaiḥ /
vidita iva sulekho ratnapeṭeva muktā vivṛta iha sa dharmaḥ prītimagryāṃ dadhāti // 1.2 //
yathā bimbaṃ bhūṣāprakṛtiguṇavad darpaṇagataṃ viśiṣṭaṃ prāmodyaṃ janayati nṛṇāṃ darśanavaśāt /
tathā dharmaḥ soktaprakṛtiguṇayukto 'pi satataṃ vibhaktārthas tuṣṭiṃ janayati viśiṣṭām iha satām // 1.3 //
āghrāyamāṇakaṭukaṃ svādurasaṃ yathāuṣadhaṃ tadvat /
dharmarmo dvayavyavasthāstho vyañjanato 'rtho na ca jñeyaḥ // 1.4 //
rājeva durārādho dharmo 'yaṃ vipulagāḍhagambhīraḥ /
ārādhitaś ca tadvadvaraguṇadhanadāyako bhavati // 1.5 //
ratnaṃ jātyamanarthaṃrghaṃ yathā 'parīkṣakajanaṃ na toṣayati /
dharmas tathāyam abudhaṃ viparyayāt toṣayati tadvat // 1.6 //
ādāv avyākaraṇāt samapravṛtter agocarāt siddheḥ /
bhāvābhāve 'bhāvāt pratipakṣatvād rutānyatvāt // 1.7 //
pratyakṣacakṣuṣo buddhāḥ śāsanasya ca rakṣakāḥ /
adhvany anyanāvṛtajñānā upekṣāto na yujyate // 1.8 //
vaikalyato virodhād anupāyatvāt tathāpy anupadeśāt /
na śrāvakayānam idaṃ bhavati mahāyānadharmākhyam // 1.9 //
āśayasyopadeśasya prayogasya virodhataḥ /
upastambhasya kālasya yat hīnaṃ hīnam eva tat // 1.10 //
svake 'vatārāt svasyaiva vinaye darśanād api /
audāryād api gāmbhīryād aviruddhaiva dharmatā // 1.11 //
niśrito 'niyato 'vyāpī sāṃvṛtaḥ khedavān api /
bālāśrayo matas tarkas tasyāto viṣayo na tat // 1.12 //
audāryād api gāmbhīryāt paripāko 'vikalpanā /
deśanā'to dvayasyāsmin sa copāyo niruttare // 1.13 //
tadasthānatrāso bhavati jagatāṃ dāhakaraṇo mahāpuṇyaskandhaprasavakaraṇād dīrghasamayam /
agotro 'sanmitro 'kṛtamatir apūrvācitaśubhas trasaty asmin dharme patati mahato 'rthād gata iha // 1.14 //
tadanyānyānyābhāvāt paramagahanatvād anugamāt vicitrasyākhyānād dhruvakathanayogād bahumukhāt /
yathākhyānaṃ nārthād bhagavati ca bhāvātigahanāt na dharme 'smiṃs trāso bhavati viduṣāṃ yonivicayāt // 1.15 //
śrutaṃ niśrityādau prabhavati manaskāra iha yo manaskārāj jñānaṃ prabhavati ca tatvārthaviṣayam /
ahaṃ na boddhā na gabhīraboddhā buddho gabhīraṃ kim atarkagamyaṃ /
kasmād gabhīrārthavidāṃ na mokṣa ity etad uttrāsapadaṃ na yuktam // 1.17 //
hīnādhimuktaḥ sunihīnadhātor hīnaiḥ sahāyaiḥ parivāritasya /
audāryagāmbhīryasudeśite 'smin dharme 'dhimuktir yadi nāsti siddham // 1.18 //
śrutānusāreṇa hi buddhimattāṃ labdhvā śrute yaḥ prakaroty avajñāṃ /
śrute vicitre sati cāprameye śiṣṭe kuto niścayam eti mūḍhaḥ // 1.19 //
yathārute 'rthe parikalpyamāne svapratyayo hānim upaiti buddheḥ /
svākhyātatāṃ ca kṣipati kṣatiṃ ca prāpnoti dharme pratighāvatīva // 1.20 //
manaḥ pradoṣaḥ prakṛtipraduṣṭo 'yathārute cāpi hy ayuktarūpaḥ /
prāg eva saṃdehagatasya dharme tasmād upekṣaiva varaṃ hy adoṣā // 1.21 //
ratnāni yo hi śaraṇapragato 'tra yāne jñeyaḥ sa eva paramaḥ śaraṇaṇagatānām /
sarvatragābhyupagamādhigamābhibhūtibhedaiś caturvidhamayārthaviśeṣaṇena // 2.1 //
yasmād ādau duṣkara eṣa vyavasāyo duḥsādho 'sau naikasahasrair api kalpaiḥ /
siddho yasmāt satvahitādhānamahārthas tasmād agre yāna ihāgraśaraṇārthaḥ // 2.2 //
sarvān satvāṃs tārayituṃ yaḥ pratipanno yano jñāne sarvagate kauśalyayuktaḥ /
yo nirvāṇe saṃsaraṇe 'py ekaraso 'sau saṃsṛtiśāntyekaraso 'sau jñeyo dhīmān eṣa hi sarvatraga evam // 2.3 //
śaraṇagatim imāṃ gato mahārthāṃ guṇagaṇavṛddhim upaiti so 'prameyām /
sphurati jagad idaṃ kṛpāśayena prathayati cāpratimaṃ mahārthadharmam // 2.4 //
satvāgratvaṃ svabhāvaś ca liṅgaṃ gotraprabhedatā /
ādīnavo 'nuśaṃsaś ca dvidhāupamyaṃ caturvidhā // 3.1 //
dhātūnām adhimukteś ca pratipatteś ca bhedataḥ /
phalabhedopalabdheś ca gotrāstitvaṃ nirūpyate // 3.2 //
udagratve 'tha sarvatve mahārthatve 'kṣayāya ca /
śubhasya tannimittatvāt gotrāgratvaṃ vidhīyate // 3.3 //
prakṛtyā paripuṣṭaṃ ca āśrayaś cāśritaṃ ca tat /
sadasac caiva vijñeyaṃ guṇottāraṇatārthataḥ // 3.4 //
kāruṇyam adhimuktiś ca kṣāntiś cādiprayogataḥ /
samācāraḥ subhasyāpi gotre liṅgaṃ nirupyate // 3.5 //
niyatāniyataṃ gotram ahāryaṃ hāryam eva ca /
pratyayair gotrabhedo 'yaṃ samāsena caturvidhaḥ // 3.6 //
kleśābhyāsaḥ kumitratvaṃ vidhātaḥ paratantratā /
gotrasyādīnavo jñeyaḥ samāsena caturvidhaḥ // 3.7 //
cirād apāyagamanam āśumokṣaś ca tatra ca /
tanuduḥkhopasaṃvittiḥ sodvegā satvapācanā // 3.8 //
suvarṇagotravat jñeyam ameyaśubhatāśrayaḥ /
jñānanirmalatāyogaprabhāvāṇāṃ ca niśrayaḥ // 3.9 //
suratnagotravat jñeyaṃ mahābodhinimittataḥ /
mahājñānasamādhyāryamahāsatvārthaniśrayāt // 3.10 //
aikāntiko duścarite 'sti kaścit kaścit samudghātitaśukladharmā /
amokṣabhāgīyaśubho 'sti kaścin nihīnaśuklo 'sty api hetuhīnaḥ // 3.11 //
gāmbhīryāudāryavāde parahitakaraṇāyodite dīrghadharme ajñātvaivādhimuktir bhavati suvipulā saṃprapattikṣamā ca /
saṃpattiś cāvasāne dvayagataparamā yad bhavaty eva teṣāṃ tajjñeyaṃ bodhisatvaprakṛtiguṇavatas tatprapuṣṭāc ca gotrāt // 3.12 //
suvipulaguṇabodhivṛkṣavṛddhyai dhanasukhaduḥkhaśamopalabdhaye ca /
svaparahitasukhakriyāphalatvād bhavati samudagrasamūlam udagragotram etat // 3.13 //
mahotsāhā mahārambhā mahārthātha mahodayā /
cetanā bodhisatvānāṃ dvayārthā cittasaṃbhavaḥ // 4.1 //
cittotpādo 'dhimokṣo 'sau śuddhādhyāśayiko 'paraḥ /
vaipākyo bhūmiṣu matas tathāvaraṇavarjitaḥ // 4.2 //
karuṇāmūla iṣṭo 'sau sadāsatvahitāśayaḥ /
dharmādhimokṣas tajjñānaparyeṣṭyālambanas tathā // 4.3 //
uttaracchandayāno 'sau pratiṣṭhāśīlasaṃvṛtiḥ /
utthāpanā vipakṣasya paripantho 'dhivāsanā // 4.4 //
śubhavṛddhyanuśaṃso 'bhau puṇyajñānamayaḥ sa hi /
sadā pāramitāyoganiryāṇaś ca sa kathyate // 4.5 //
bhūmiparyavasāno 'sau pratisvaṃ tatprayogataḥ /
vijñeyo bodhisatvānāṃ cittotpādaviniścayaḥ // 4.6 //
mitrabalād hetubalān mūlabalāc chrutabalāc chubhābhyāsāt /
adṛḍhadṛḍhodaya uktaś cittotpādaḥ parākhyānāt // 4.7 //
sūpāsitasaṃbuddhe susaṃbhṛtajñānapuṇyasaṃbhāre /
dharmeṣu nirvikalpajñānaprasavāt paramatāsya // 4.8 //
dharmeṣu ca satveṣu ca tatkṛtyeṣūttame ca buddhatve /
samacittopāttopalambhāt prāmodyaviśiṣṭatā tasya // 4.9 //
janmāudāryaṃ tasminn utsāhaḥ śuddhir āśayasyāpi /
kauśalyaṃ pariśiṣṭe niryāṇaṃ caiva vijñeyam // 4.10 //
dharmādhimuktibījāt pāramitāśreṣṭhamātṛto jātaḥ /
dhyānamaye sukhagarbhe karuṇā saṃvardhikā dhātrī // 4.11 //
audāryaṃ vijñeyaṃ praṇidhānamahādaśābhinirhārāt /
utsāho boddhavyo duṣkaradīrghādhikākhedāt // 4.12 //
āsannabodhibodhāt tadupāyajñānalābhataś cāpi /
āśayaśuddhir jñeyā kauśalyaṃ tv anyabhūmigatam // 4.13 //
niryāṇaṃ vijñeyaṃ yathāvyavasthānamanasikāreṇa /
tatkalpanatājñānād avikalpanayā ca tasyaiva // 4.14 //
pṛthivīsama utpādaḥ kalyāṇasuvarṇasaṃnibhaś cānyaḥ /
śuklanavacandrasadṛśo vahniprakhyo 'parocchrāyaḥ jñeyaḥ // 4.15 //
bhūyo mahānidhānavadanyo ratnākaro yathaivānyaḥ /
sāgarasadṛśo jñeyo vajraprakhyo 'calendranibhaḥ // 4.16 //
bhaiṣajyarājasadṛśo mahāsuhṛtsaṃnibho 'paro jñeyaḥ /
cintāmaṇiprakāśo dinakarasadṛśo 'paro jñeyaḥ // 4.17 //
gandharvamadhuraghoṣavadanyo rājopamo 'paro jñeyaḥ /
koṣṭhāgāraprakhyo mahāpathasamas yathaivānyaḥ // 4.18 //
yānasamo vijñeyo gandharvasamaś ca vetasagacetasaḥprabhavaḥ /
ānandaśabdasadṛśo mahānadīśrotasadṛśaś ca // 4.19 //
meghasadṛśaś ca kathitaś cittotpādo jinātmajānāṃ hi /
tasmāt tathā guṇāḍhyaṃ cittaṃ muditaiḥ samutpādyam // 4.20 //
parārthacittāt tadupāyalābhato mahābhisaṃdhyarthasutatvadarśanāt /
mahārhacittodayavarjitā janāḥ śamaṃ gamiṣyanti vihāya tat sukham // 4.21 //
sahodayāc cittavarasya dhīmataḥ susaṃvṛtaṃ cittam anantaduṣkṛtāt /
sukhena duḥkhena ca modate sadā śubhī kṛpāluś ca vivardhanadvayam // 4.22 //
yad ānapekṣaḥ svaśarīrajīvite parārtham abhyeti paraṃ pariśramam /
paroghātena tathāvidhaḥ kathaṃ sa duṣkṛte karmaṇi saṃpravatsyati // 4.23 //
māyopamān vīkṣya sa sarvadharmān udyānayātrām iva copapattīḥ /
kleśāc ca duḥkhāc ca bibheti nāsau saṃpattikāle 'tha vipattikāle // 4.24 //
svakā guṇāḥ satvahitāc ca modaḥ saṃcintyajanma rddhivikurvitaṃ ca /
vibhūṣaṇaṃ bhojanam agrabhūmiḥ krīḍāratir nityakṛpātmakānām // 4.25 //
parārtham udyogavataḥ kṛpātmano hy avīcir apy eti yato 'sya ramyatām /
kutaḥ punas trasyati tādṛśo bhavan parāśrayair duḥkhasamudbhavair bhave // 4.26 //
mahākṛpācāryasadoṣitātmanaḥ parasya duḥkhair upataptacetasaḥ /
parasya kṛtye samupasthite punaḥ paraiḥ samādāpanato 'tilajjanā // 4.27 //
śirasi vinihitoccasatvabhāraḥ śithilagatir na hi śobhate 'grasatvaḥ /
svaparavividhabandhanātibaddhaḥ śataguṇam utsaham arhati prakartum // 4.28 //
mahāśrayārambhaphalodayātmikā jinātmajānāṃ pratipattir iṣyate /
sadā mahādānamahādhivāsanā mahārthasaṃpādanakṛtyakārikā // 5.1 //
paratralabdhvātmasamānacittatāṃ svato 'dhi vā śreṣṭhatareṣṭatāṃ pare /
tathātmano 'nyārthaviśiṣṭasaṃjñinaḥ svakārthatā kā katamā parārthatā // 5.2 //
paratra loko na tathātinirdayaḥ pravartate tāpanakarmaṇā ripau /
yathā parārthaṃ bhṛśaduḥkhatāpane kṛpātmakaḥ svātmani saṃpravartate // 5.3 //
nikṛṣṭamadhyotttamadharmatāsthite sudeśanāvarjanatā 'vatāraṇā /
vinītir arthe paripācanā śubhe tathāvavādasthitibuddhimuktayaḥ // 5.4 //
guṇair viśiṣṭaiḥ samudāgamas tathā kulodayo vyākaraṇābhiṣiktatā /
tathāgatajñānam anuttaraṃ padaṃ parārtha eṣa tryadhiko daśātmakaḥ // 5.5 //
janānurūpāviparītadeśanā nirunnatā cāpy amamā vicakṣaṇā /
kṣamā ca dāntā ca sudūragā 'kṣayā jinātmajānāṃ pratipattir uttamā // 5.6 //
mahābhaye kāmijanaḥ pravartate cale viparyāsasukhe bhavapriyaḥ /
pratisvam ādhipraśame śamapriyaḥ sadā tu sarvādhigame kṛpātmakaḥ // 5.7 //
jano vimūḍhaḥ svasukhārtham udyataḥ sadā tad aprāpya paraiti duḥkhatām /
sadā tu dhīro hi parārtham udyato dvayārtham ādhāya paraiti nirvṛtim // 5.8 //
yathā yathā hy akṣavicitragocare pravartate cāragato jinātmajaḥ /
tathā tathā yuktasamānatāpadair hitāya satveṣv abhisaṃskaroti tat // 5.9 //
sadā 'svatantrīkṛtadoṣacetane jane na saṃdoṣam upaiti buddhimān /
akāmakāreṇa hi viprapattayo jane bhavantīti kṛpāvivṛddhitaḥ // 5.10 //
bhavagati sakalābhibhūyagantrī paramaśamānugatā prapattir eva /
vividhaguṇagaṇair vivardhamānā jagad upaguhya sadā kṛpāśayena // 5.11 //
na san na cāsan na tathā na cānyathā na jāyate vyeti na cāvahīyate /
na vardhate nāpi viśudhyate punar viśudhyate tatparamārthalakṣaṇaṃ // 6.1 //
na cātmadṛṣṭiḥ svayam ātmalakṣaṇā na cāpi duḥsaṃsthitatā vilakṣaṇā /
dvayān na cānyad bhrama eṣa tatūditas tataś ca mokṣo bhramamātrasaṃkṣayaḥ // 6.2 //
kathaṃ jano vibhramamātram āśritaḥ paraiti duḥkhaprakṛtiṃ na saṃtatām /
avedako vedaka eva duḥkhito na duḥkhito dharmamayo na tanmayaḥ // 6.3 //
pratītyabhāvaprabhave kathaṃ janaḥ samakṣavṛttiḥ śrayate 'nyakāritam /
tamaḥ prakāraḥ katamo 'yam īdṛśo yato 'vipaśyan sadasan nirīkṣate // 6.4 //
ta cāntaraṃ kiṃcana vidyate 'nayoḥ sadarthavṛttyā śamajanmanor iha /
tathāpi janmakṣayato vidhīyate śamasya lābhaḥ śubhakarmakāriṇām // 6.5 //
saṃbhṛtya saṃbhāram anantapāraṃ jñānasya puṇyasya ca bodhisatvaḥ /
dharmeṣu cintāsuviniśriś citatvāj jalpānvayām arthagatiṃ paraiti // 6.6 //
arthān sa vijñāya ca jalpamātrān saṃtiṣṭhate tannibhacittamātre /
pratyakṣatām eti ca dharmadhātus tasmād viyukto dvayalakṣaṇena // 6.7 //
nāstīti cittāt param etya buddhyā cittasya nāstitvam upaiti tasmāt /
dvayasya nāstitvam upetya dhīmān saṃtiṣṭhate 'tadgatidharmaghātau // 6.8 //
akalpanājñānabalena dhīmataḥ samānuyātena samantataḥ sadā /
tadāśrayo gahvaradoṣasaṃcayo mahāgadeneva viṣaṃ nirasyate // 6.9 //
munivihitasudharmasuvyavastho matim upadhāya samūladharmadhātau /
smṛtim agatim avagamya kalpamātrāṃ vrajati guṇārṇavapāramāśudhīraḥ // 6.10 //
utpattivākcittaśubhāśubha-adhitatsthānaniḥsārapadā parokṣam /
jñānaṃ hi sarvatragasaprabhedeṣv avyāhataṃ dhīr agataḥ prabhāvaḥ // 7.1 //
dhyānaṃ caturthaṃ suviśuddham etya niṣkalpanājñānaparigraheṇa /
yathāvyavasthānamanaskriyātaḥ prabhāvasiddhiṃ paramāṃ paraiti // 7.2 //
yenāryadivyāpratimair vihārair brāhmaiś ca nityaṃ viharaty udāraiḥ /
buddhāṃś ca satvāṃś ca sa dikṣu gatvā saṃmānayaty ānayate viśuddhim // 7.3 //
māyopamān paśyati lokadhātūn sarvān sasatvān savivartanāśān /
saṃdarśayaty eva ca tān yatheṣṭaṃ vaśī vicitrair api sa prakāraiḥ // 7.4 //
raśmipramokṣair bhṛśaduḥkhitāṃś ca āpāyikān svargagatān karoti /
mārānvayān kṣubdhavimānaśobhān saṃkampayaṃ srāsayate samārān // 7.5 //
samādhivikrīḍitam aprameyaṃ saṃdarśayaty agragaṇasya madhye /
sakarmajanmottamanirmitaiś ca satvārtham ātiṣṭhati sarvakālam // 7.6 //
jñānavaśitvāt samupaiti śuddhiṃ kṣetraṃ yathākāmanidarśanāya /
abuddhanāmeṣu ca buddhanāma saṃśrāvaṇāt tān kṣipate 'nyadhātau // 7.7 //
śakto bhavaty eva ca satvapāke saṃjātapakṣaḥ śakunir yathaiva /
buddhāt praśaṃsāṃ labhate 'timātrām ādeyavākyo bhavati prajānām // 7.8 //
ṣaḍdhāpy abhijñā trividhā ca vidyā aṣṭau vimokṣā 'bhibhuvas tathā'ṣṭau /
daśāpi kṛtsnāyatanāny ameyāḥ samādhayo dhīragataḥ prabhāvaḥ // 7.9 //
sa hi paramavaśitvalabdhabuddhir jagadavaśaṃ svavaśe vidhāya nityam /
parahitakaraṇaikatābhirāmaś carati bhaveṣu hi siṃhavat sudhīraḥ // 7.10 //
rūciḥ prasādaḥ praśamo 'nukampanā kṣamātha medhā prabalatvam eva ca /
ahāryatāṅgaiḥ samupetatā bhṛśaṃ jinātmaje tatparipākalakṣaṇam // 8.1 //
sumitratāditrayam ugravīryatā parārdhaniṣṭhottamadharmasaṃgrahaḥ /
kṛpālusaddharmamahāparigrahe mataṃ hi samyakparipākalakṣaṇam // 8.2 //
guṇajñatāthāśusamādhilābhitā phalānubhūtir manaso 'dhyabhedatā /
jinātmaje śāstari saṃprapattaye mataṃ hi samyakparipākalakṣaṇam // 8.3 //
susaṃvṛttiḥ kliṣṭavitarkavarjanā nirantarāyo 'tha śubhābhirāmatā /
jīnātmaje kleśavinodanāya tan mataṃ hi samyakparipākalakṣaṇam // 8.4 //
kṛpā prakṛtyā paraduḥkhadarśanaṃ nihīnacittasya ca saṃpravarjanam /
viśeṣagatvaṃ jagadagrajanmatā parānukampāparipākalakṣaṇam // 8.5 //
dhṛtiḥ prakṛtyā pratisaṃkhyabhāvanā suduḥkhaśītādyadhivāsanā sadā /
viśeṣagāmitvaśubhābhirāmatā mataṃ kṣamāyāḥ paripākalakṣaṇam // 8.6 //
vipākaśuddhiḥ śravaṇādyamoṣatā praviṣṭatā sūktaduruktayos tathā /
smṛter mahābuddhyudaye ca yogyatā sumedhatāyāḥ paripākalakṣaṇam // 8.7 //
śubhadvayena dvayadhātupuṣṭatā phalodaye cāśrayayogyatā parā /

sarvasyānupalambhāc ca bhūtajñānaṃ caturvidhaṃ /
sarvārthasiddhyai dhīrāṇāṃ sarvabhūmiṣu jāyate // 19.48 //
pratiṣṭhābhogabījaṃ hi nimittaṃ bandhanasya hi /
sāśrayāś cittacaittās tu badhyante 'tra sabījakāḥ // 19.49 //
purataḥ sthāpitaṃ yac ca nimittaṃ yat sthitaṃ svayaṃ /
sarvaṃ vibhāvayan dhīmān labhate bodhim uttamām // 19.50 //
tathatālambanaṃ jñānaṃ dvayagrāhavivarjitaṃ /
dauṣṭhulyakāyapratyakṣaṃ tatkṣaye dhīmatāṃ matam // 19.51 //
tathatālambanaṃ jñānam anānākārabhāvitaṃ /
sadasattārthe pratyakṣaṃ vikalpavibhu cocyate // 19.52 //
tatvaṃ saṃcchādya bālānām atatvaṃ khyāti sarvataḥ /
tatvaṃ tu bodhisatvānāṃ sarvataḥ khyāty apāsya tat // 19.53 //
akhyānakhyānatā jñeyā asadarthasadarthayoḥ /
āśrayasya parāvṛttir mokṣo 'sau kāmacārataḥ // 19.54 //
anyonyaṃ tulyajātīyaḥ khyāty arthaḥ sarvato mahān /
antarāyakaras tasmāt parijñāyainam utsṛjet // 19.55 //
paripācyaṃ viśodhyaṃ ca prāpyaṃ yogyaṃ ca pācane /
samyaktvadeśanāvastu aprameyaṃ hi dhīmatām // 19.56 //
bodhisatvacittasya cotpādo notpādakṣāntir eva ca /
cakṣuś ca nirmalaṃ hīnam āśravakṣaya eva ca // 19.57 //
saddharmasya sthitir dīrghā vyutpatticchittibhogatā /
deśanāyāḥ phalaṃ jñeyaṃ tatprayuktasya dhīmataḥ // 19.58 //
ālambanamahatvaṃ ca pratipatter dvayos tathā /
jñānasya vīryārambhasya upāye kauśalasya ca // 19.59 //
udāgamamahatvaṃ ca mahatvaṃ buddhakarmaṇaḥ /
etan mahatvayogād dhi mahāyānaṃ nirucyate // 19.60 //
gotraṃ dharmādhimuktiś ca cittasyotpādanā tathā /
dānādipratipattiś ca nyāyām āvakrāntir eva ca // 19.61 //
satvānāṃ paripākaś ca kṣetrasya ca viśodhanā /
apratiṣṭhitanirvāṇaṃ bodhiḥ śreṣṭhā ca darśanāt darśanā // 19.62 //
nimitte cānimitte ca cāry apy anabhisaṃskṛte /
bodhisatvā hi vijñeyāḥ pañcaite sarvabhūmiṣu // 19.63 //
kāmeṣv asaktas triviśuddhakarmā krodhābhibhūmyaṃ guṇatatparaś ca /
dharme 'calas tatvagabhīradṛṣṭir bodhau spṛhāvān khalu bodhisatvaḥ // 19.64 //
anugraheccho 'nupaghātadṛṣṭiḥ paropaghāteṣv adhivāsakaś ca /
dhīro 'pramattaś ca bahuśrutaś ca parārthayuktaḥ khalu bodhisatvaḥ // 19.65 //
ādīnavajñaḥ svaparigraheṣu bhogeṣv asakto hy anigūḍhavairaḥ /
yogī nimitte kuśalo 'kudṛṣṭir adhyātmasaṃsthaḥ khalu bodhisatvaḥ // 19.66 //
dayānvito hrīguṇasaṃniviṣṭo duḥkhādhivāsāt svasukheṣv asaktaḥ /
smṛtipradhānaḥ susamāhitātmā yānāvikāryaḥ khalu bodhisatvaḥ // 19.67 //
duḥkhāpaho duḥkhakaro na caiva duḥkhādhivāso na ca duḥkhabhītaḥ /
duḥkhād vimukto na ca duḥkhakalpo duḥkhābhyupetaḥ khalu bodhisatvaḥ // 19.68 //
dharme rato 'dharmarataḥ dharme 'rato 'dharmarataḥ prakṛtyā dharme jugupsī dharamābhiyuktaḥ /
dharme vaśī dharmanirandhakāro dharmapradhānaḥ khalu bodhisatvaḥ // 19.69 //
bhogāpramatto niyamāpramatto rakṣāpramattaḥ kuśalāpramattaḥ /
sukhāpramatto dharamāpramatto yānāpramatto khalu bodhisatvaḥ // 19.70 //
vimānalajjas tanudoṣalajja amarṣalajjaḥ parihāṇilajjaḥ /
viśālavisāralajjas tanudṛṣṭilajjaḥ yānānyalajjaḥ khalu bodhisatvaḥ // 19.71 //
ihāpi cāmutra upekṣaṇena saṃskārayogena vibhutvalābhaiḥ /
śamopasamaupadeśena mahāphalena anugrahe vartati bodhisatvaḥ // 19.72 //
bodhisatvo mahāsatvo dhīmāṃś caivottamadyutiḥ /
jinaputro jinādhāro vijetātha jināṅkuraḥ // 19.73 //
vikrāntaḥ paramāścaryaḥ sārthavāho mahāyaśāḥ /
kṛpāluś ca mahāpuṇya īśvaro dhārmikas tathā // 19.74 //
sutatvabodhaiḥ sumahārthabodhaiḥ sarvāvarthabodhair api nityabodhaiḥ /
upāyabodhaiś ca viśeṣaṇena tenocyate hetuna bodhisatvaḥ // 19.75 //
ātmānubodhāt tanudṛṣṭibodhād vicitravijñaptivibodhataś ca /
sarvasya cābhūtavikalpabodhāt tenocyate hetuna bodhisatvaḥ // 19.76 //
abodhabodhād anubodhabodhād abhāvabodhāt prabhavānubodhāt /
abodhabodhapratibodhataś ca tenocyate hetuna bodhisatvaḥ // 19.77 //
anarthabodhāt paramārthabodhāt sarvāvabodhāt sakalārthabodhāt /
boddhavyabodhāśrayabodhabodhāt tenocyate hetuna bodhisatvaḥ // 19.78 //
niṣpannabodhāt padabodhataś ca garbhānubodhāt kramadarśanasya /
bodhād bhṛśaṃ saṃśayahānibodhāt tenocyate hetuna bodhisatvaḥ // 19.79 //
lābhī hy alābhī dhīsaṃsthitaś ca boddhānuboddhā pratideśakaś ca /
nirjalpabuddhir hatamānamānī hy apakkasaṃpakkamatiś ca dhīmān // 19.80 //
anukampā priyākhyānaṃ dhīratā muktahastatā /
gambhīrasaṃdhinirmokṣo liṅgāny etāni dhīmatāṃ // 20-21.1 //
parigrahe 'dhimuktyāptāvakhede dvayasaṃgrahe /
āśayāc ca prayogāc ca vijñeyaṃ liṅgapañcakaṃ // 20-21.2 //
bodhisatvā hi satataṃ bhavantaś cakravartinaḥ /
prakurvanti hi satvārthaṃ gṛhiṇaḥ sarvajanmasu // 20-21.3 //
ādānalabdhā pravrajyā dharmatopagatā parā /
nidarśikā ca pravrajayā dhīmatāṃ sarvabhūmiṣu // 20-21.4 //
aprameyair guṇair yuktaḥ pakṣaḥ pravrajitasya tu /
gṛhiṇo bodhisatvād dhi yatis tasmād viśiṣyate // 20-21.5 //
nirvāṇecchā ca dhīrāṇāṃ satveṣv āśaya iṣyate /
aśuddhaś ca viśuddhaś ca suviśuddhaḥ sarvabhūmiṣu // 20-21.6 //
praṇidhānāt samāc cittād ādhipatyāt parigrahaḥ /
gaṇasya karṣaṇatvāc ca dhīmatāṃ sarvabhūmiṣu // 20-21.7 //
karmaṇaś cādhipatyena praṇidhānasya cāparā /
samādheś ca vibhutvasya cotpattir dhīmatāṃ matā // 20-21.8 //
lakṣaṇāt pudgagalāc chikṣāskandhaniṣpattiliṅgataḥ /
nirukteḥ prāptitaś caiva vihāro bhūmir eva ca // 20-21.9 //
śūnyatā paramātmasya karmanāśe vyavasthitiḥ /
vihṛtya sasukhair dhyānair janma kāme tataḥ param // 20-21.10 //
tataś ca bodhipakṣāṇāṃ saṃsāre pariṇāmanā /
vinā ca cittasaṃkleśaṃ satvānāṃ paripācanā // 20-21.11 //
upapattau ca saṃcitya saṃkleśasyānurakṣaṇā /
ekāyanapathaśliṣṭā 'nimittaikāntikaḥ pathaḥ // 20-21.12 //
animitte 'py anābhogaḥ kṣetrasya ca viśodhanā /
satvapākasya niṣpattir jāyate ca tataḥ param // 20-21.13 //
samādhidhāraṇīnāṃ ca bodheś caiva viśuddhatā /
etasmāc ca vyavasthānād vijñeyaṃ bhūmilakṣaṇam // 20-21.14 //
viśuddhadṛṣṭiḥ suviśuddhaśīlaḥ samāhito dharmavibhūtamānaḥ /
saṃtānasaṃkleśaviśuddhibhede nirmāṇa ekakṣaṇalabdhabuddhiḥ // 20-21.15 //
upekṣakaḥ kṣetraviśodhakaś ca syāt satvapāke kuśalo maharddhiḥ /
saṃpūrṇakāyaś ca nidarśane ca śakto 'bhiṣiktaḥ khalu bodhisatvaḥ // 20-21.16 //
dharmatāṃ pratividhyeha adhiśīle 'nuśikṣaṇe /
adhicitte 'py adhiprajñe prajñā tu dvayagocarā // 20-21.17 //
dharmatatvaṃ tadajñānajñānād yā vṛttir eva ca /
prajñāyā gocaras tasmād dvibhūmau tadvyavasthitiḥ // 20-21.18 //
śikṣāṇāṃ bhāvanāyāś ca phalam anyac caturvidham /
animittasasaṃskāro vihāraḥ prathamaṃ phalam // 20-21.19 //
sa evānabhisaṃskāro dvitīyaṃ phalam iṣyate /
kṣetraśuddhiś ca satvānāṃ pākaniṣpattir eva ca // 20-21.20 //
samādhidhāraṇīnāṃ ca niṣpattiḥ paramaṃ phalaṃ /
caturvidhaṃ phalaṃ hy etat caturbhūmisamāśritam // 20-21.21 //
samādhiprajñāskandhasya tata ūrdhvaṃ viśodhanā // 20-21.22 //
vimuktimuktijñānasya tadanyāsu viśodhanā /
caturvidhād āvaraṇāt pratighātāvṛter api // 20-21.23 //
aniṣpannāś ca niṣpannā vijñeyāḥ sarvabhūmayaḥ /
niṣpannā apy aniṣpannā niṣpannāś ca punar matāḥ // 20-21.24 //
niṣpattir vijñeyā yathāvyavasthānamanasikāreṇa /
tatkalpanatājñānād avikalpanayā ca tasyaiva // 20-21.25 //
bhāvanā api niṣpattir acintyaṃ sarvabhūmiṣu /
pratyātmavedanīyatvāt buddhānāṃ viṣayād api // 20-21.26 //
adhimuktir hi sarvatra sālokā liṅgam iṣyate /
alīnatvam adīnatvam aparapratyayātmatā // 20-21.27 //
prativedhaś ca sarvatra sarvatra samacittatā /
aneyānunayopāyajñānaṃ maṇḍalajanma ca // 20-21.28 //
nācchando na ca lubdhahrasvahṛdayo na krodhano nālaso nāmaitrīkarūṇāśayo na kumatiḥ kalpair vikalpair hataḥ /
no vikṣiptamatiḥ sukhair na ca hato duḥkhair na vā vyāvartate satyaṃ mitram upāśritaḥ śrutaparaḥ pūjāparaḥ śāstari // 20-21.29 //
sarvaṃ puṇyasamuccayaṃ suvipulaṃ kṛtvānyasādhāraṇaṃ saṃbodhau pariṇāmayaty aharaharyo hy uttamopāyavit /
jātaḥ svāyatane sadā śubhakaraḥ krīḍaty abhijñāguṇaiḥ sarveṣām uparisthito guṇanidhir jñeyaḥ sa buddhātmanaḥ // 20-21.30 //
śamathe vipaśyanāyāṃ ca dvayapañcātmako mataḥ /
dhīmatām anuśaṃso hi sarvathā sarvabhūmiṣu // 20-21.31 //
paśyatāṃ bodhim āsannāṃ satvārthasya ca sādhanaṃ /
tīvra utpadyate modo muditā tena kathyate // 20-21.32 //
mahādharmāvabhāsasya karaṇāc ca prabhākarī // 20-21.33 //
arcirbhūtā yato dharmā bodhipakṣāḥ pradāhakāḥ /
arciṣmatīti tadyogāt sā bhūmir dvayadāhataḥ // 20-21.34 //
satvānāṃ paripākaś ca svacittasya ca rakṣaṇā /
dhīmadbhir jīyate duḥkhaṃ durjayā tena kathyate // 20-21.35 //
ābhimukhyād dvayasyeha saṃsārasyāpi nirvṛteḥ /
uktā hy abhimukhī bhūmiḥ prajñāpāramitāśrayāt // 20-21.36 //
ekāyanapathaḥ pūrvaṃ nirdiṣṭas tadupaśleṣatvāt dūraṃ gatā bhavati prayogapayantagamanāt /
dvayasaṃjñāvicalanād acalā ca nirucyate // 20-21.37 //
pratisaṃvinmateḥ sādhutvād iti pradhānatvāt /
dharmameghā dvayavyāpter dharmākāśasya meghavat // 20-21.38 //
vividhe śubhanirhāre ratyā viharaṇāt sadā /
sarvatra bodhisatvānāṃ vihārabhūmayo matāḥ // 20-21.39 //
bhūyo bhūyo 'mitāsv āsu ūrdhvaṃgamanayogataḥ /
bhūtāmitābhayārthāya ta eveṣṭā hi bhūmayaḥ // 20-21.40 //
bhūmilābhebho 'dhimukteś ca cariteṣu ca vartanāt /
prativedhāc ca bhūmīnāṃ niṣpatteś ca caturvidhaḥ // 20-21.41 //
dvayor āvarjanārthāya vinayāya ca deśitāḥ /
caryāś catasro dhīrāṇāṃ yathāsūtrānusārataḥ // 20-21.42 //
anukampakasatveṣu saṃyogavigamāśaya /
aviyogāśaya saukhyahitāśaya namo 'stu te // 20-21.43 //
sarvāvaraṇanirmukta sarvalokābhibhū mune /
jñānena jñeyaṃ vyāptaṃ te muktacitta namo 'stu te // 20-21.44 //
aśeṣaṃ sarvasatvānāṃ sarvakleśavināśaka /
kleśaprahāraka kliṣṭasānukrośa namo 'stu te // 20-21.45 //
anābhoga nirāsaṅga avyāghāta samāhita /
sadaiva sarvapraśnānāṃ visarjaka namo 'stu te // 20-21.46 //
āśraye 'thāśrite deśye vākye jñāne ca deśike /
avyāhatamate nityaṃ sudeśika namo 'stu te // 20-21.47 //
upetya vacanais teṣāṃ carijña āgatau gatau /
niḥsāre caiva satvānāṃ svavavāda namo 'stu te // 20-21.48 //
satpauruṣyaṃ prapadyante tvāṃ dṛṣṭvā sarvadehinaḥ /
dṛṣṭamātrāt prasādasya vidhāyaka namo 'stu te // 20-21.49 //
ādānasthānasaṃtyāganirmāṇapariṇāmane /
samādhijñānavaśitāmanuprāpta namo 'stu te // 20-21.50 //
upāye śaraṇe śuddhau satvānāṃ vipravādane /
mahāyāne ca niryāṇe mārabhañja namo 'stu te // 20-21.51 //
jñānaprahāṇaniryāṇavighnakārakadeśika /
svaparārthe 'nyatīrthyānāṃ nirādhṛṣya namo 'stu te // 20-21.52 //
vigṛhyavaktā parṣatsu dvayasaṃkleśavarjita /
nirārakṣa asaṃmoṣa gaṇakarṣa namo 'stu te // 20-21.53 //
cāre vihāre sarvatra nāsty asarvajñaceṣṭitaṃ /
sarvadā tava sarvajña bhūtārthika namo 'stu te // 20-21.54 //
sarvasatvārthakṛtyeṣu kālaṃ tvaṃ nātivartase /
abandhyakṛtya satatam asaṃmoṣaḥ namo 'stu te // 20-21.55 //
sarvalokamahorātraṃ ṣaṭkṛtvaḥ pratyavekṣase /
mahākaruṇayā yukta hitāśaya namo 'stu te // 20-21.56 //
cāreṇādhigamenāpi jñānenāpi ca karmaṇā /
sarvaśrāvakapratyekabuddhottama namo 'stu te // 20-21.57 //
tribhiḥ kāyair mahābodhiṃ sarvākārām upāgata /
sarvatra sarvasatvānāṃ kāṅkṣāchida namo 'stu te // 20-21.58 //
niravagraha nirdoṣa niṣkāluṣyānavasthita /
āniṅkṣya sarvadharmeṣu niṣprapañca namo 'stu te // 20-21.59 //
niṣpannaparamārtho 'si sarvabhūmiviniḥsṛtaḥ /
sarvasatvāgratāṃ prāptaḥ sarvasatvavimocakaḥ // 20-21.60 //
akṣayair asamair yukto guṇair lokeṣu dṛśyase /
maṇḍaleṣv apy adṛśyaś ca sarvathā devamānuṣaiḥ // 20-21.61 //
