#Author:Kṣemendra
#Text:Sevyasevakopadeśaḥ
#Data entry:Daniele Cuneo
#Edition:»kṣemendrakṛtaḥ sevyasevakopadeśaḥ.« In: Paṇḍit Durgāprasād (ed.) et al.: Kāvyamālā Part 2. Bombay 1932, pp. 79-85.


mahākaviśrīkṣemendrakṛtaḥ sevyasevakopadeśaḥ /
vibhūṣaṇāya mahate tṛṣṇātimirahāriṇe /
namaḥ saṃtoṣaratnāya sevāviṣavināśine // 1 //
utsṛjya nijakāryāṇi sadbhir bāṣpākulekṣaṇam /
sevyasevakasevānāṃ kriyatām anuśāsanam // 2 //
darpād ekaḥ paro lobhād dvāv andhau sevyasevakau /
dhanoṣmadainyavikṛtī mukhe kaḥ kasya paśyati // 3 //
durvāramohalobhāndho yadi na syād ayaṃ janaḥ /
kaḥ krūrakrodhavidhuraṃ saheta dhanināṃ mukham // 4 //
yaḥ pṛthvīm api darpāndho na paśyati puraḥsthitām /
sa dainyalaghutāṃ yātaṃ kathaṃ sevakam īkṣate // 5 //
agatiṃ vāhayaty eko badhiraṃ stauti cāparaḥ /
aho jagati hāsyāya nirlajjau sevyasevakau // 6 //
dūraṃ huṃkāramātreṇa visṛṣṭo mārgaṇaḥ sadā /
guṇabhraṣṭaḥ kriyāhīno nodvegaṃ yāti sevakaḥ // 7 //
manye sukṛtinā tena bhāgīrathyāṃ kṛtaṃ tapaḥ /
vairāgyabhāgī rathyāṃ yaḥ sevāsu na vigāhate // 8 //
kathitakleśavāpena śāpeneva vipākinā /
sevātāpena pacyante na hy aduṣkṛtino narāḥ // 9 //
adainyapuṇyamanasāṃ yaśas teṣāṃ virājate /
sevāpaṅkakalaṅkānāṃ yair na pātrīkṛtaṃ śiraḥ // 10 //
prabhupraṇāme jaṭharaṃ dainyamūkaṃ vilokayan /
praveṣṭuṃ sevakaḥ śaṅke vilakṣaḥ kṣitim īkṣate // 11 //
sevādhvajo 'ñjalir mūrdhni hṛdi dainyaṃ mukhe stutiḥ /
āśāgrahagṛhītānāṃ kiyatīyaṃ viḍambanā // 12 //
akālāgamanakrodhavidhureśvaracakṣuṣā /
kṣipraṃ smara ivāyāti sevako dugdhabhūtitām // 13 //
punaḥ sevāvamānānāṃ tena dattaṃ tilodakam /
praviśya vāhinīṃ yena khaḍgapātrīkṛtaṃ vapuḥ // 14 //
kaṇavṛttiparikliṣṭaḥ kaṣṭaṃ sevakacātakaḥ /
ghanāśānirato nityam udgrīvaḥ pariśuṣyati // 15 //
niḥsaṃtoṣaḥ parityajya bhramaraḥ puṣpitaṃ vanam /
sevate dānalobhena mātaṅgam api sevakaḥ // 16 //
jaḍasevāparikliṣṭas tīvradambhabakavrataḥ /
kṛcchreṇa kṣaṇikāṃ prītim āsādayati sevakaḥ // 17 //
nityam unnatikāmo 'pi mānabhaṅgād adhomukhaḥ /
yat satyam ubhayabhraṣṭas triśaṅkur iva sevakaḥ // 18 //
svamāṃsavikrayāsaktaḥ prabhuvetālaghaṭṭitaḥ /
niḥsattvaḥ pralayaṃ yāti rātrisevāsu sevakaḥ // 19 //
bhūmiśāyī nirāhāraḥ śītavātātapakṣataḥ /
munivrato 'pi narakakleśam aśnāti sevakaḥ // 20 //
karoti sevakaḥ sevāṃ tāvad āśām ivāyatām /
yāvad bhogarasajñena yauvanena viyujyate // 21 //
cakṣur dvāry añjalau pāṇiḥ stutau jihvā natau śiraḥ /
aho nu sevakaiḥ kāyaḥ paropakaraṇīkṛtaḥ // 22 //
hanta yācñāvamānena sevako laghutāṃ gataḥ /
sevāpaṅkapade magno yad anviṣṭo na labhyate // 23 //
aviśrāntyāviraktasya dīrghocchvāsena śuṣyataḥ /
jareva duḥsahā jantoḥ sevā saṃkocakāriṇī // 24 //
truṭyati klinnatantrīva māleva mlāyati kṣaṇāt /
sevā nāracayaty eva dinacchedena naśyati // 25 //
sevāsaṃkṣapitāṅgānāṃ rājarathyāsu śuṣyatām /
tīvratṛṣṇāviṣārtānāṃ śaraṇam śamavāridaḥ // 26 //
lobhasya mohādhvani vṛddhabhāve jātā durāśāśayitasya sevā /
adhomukhī sā satataṃ salajjā jaratkumārīva na kasya śocyā // 27 //
āśeva śūnyeṣu vivartamānā tṛṣṇeva saṃtoṣaparāṅmukhīyam /
divāniśaṃ karṣaṇadīrgharajjuḥ sevā surāṇām api dainyabhūmiḥ // 28 //
nisargavandhyāsu phalāśayā ye kurvanti sevāsu sadā prayatnam /
kṛṣipravṛttāḥ śaśaśṛṅgakoṭyā khananti mūrkhāḥ khalu svasthalīṃ te // 29 //
antaḥsthite 'py ātmani yaiḥ kriyante sevāpravṛttaiḥ puruṣaiḥ praṇāmāḥ /
dugdhābdikallolini kūlakacche tṛṣṇāturās te jalam arthayante // 30 //
bhāgīrathītīravanāntareṣu phalāvanamreṣu mahādrumeṣu /
kṣuttāpatṛṣṇāśamaneṣu satsu kiṃ dainyasevāvyasanāvamānaiḥ // 31 //
nityaprasaktyā na sṛjaty avajñāṃ na laulyagarhāgaṇanāṃ karoti /
atyantayācñāsu na yāti khedaṃ bhikṣābhujām pātram ato nu sevyam // 32 //
jayanti te svasti namo 'stu tebhyaḥ prabhūpraṇāmacyutamānaratnam /
sevāprayāsavyasaneṣu mithyā rathyārajobhāgi śiro na yeṣām // 33 //
kiṃ rājabhir durjanajanmajālavyāptair avāptair api kaṣṭadṛṣṭaiḥ /
santy eva sevyā bhuvi bhūbhṛtas te taṭeṣu yeṣāṃ munayo niviṣṭāḥ // 34 //
tṛṣṇāturāṇāṃ rajasā vṛtānāṃ sevākukūlānalatāpitānām /
śāntyai hitaṃ candrakirīṭajūṭamaitrīpavitraṃ tridaśāpagāmbhaḥ // 35 //
utsṛjya tīvraṃ viṣayābhilāṣaṃ saṃtoṣapoṣaṃ kuśalo bhajasva /
ity arthyamānaḥ sukhasevayāyam ātmāiva sarvaṃ sudhiyāṃ dadāti // 36 //
ekasya jāne sukhabhogabhājaḥ ślāghyasya sevāvyasanaṃ nivṛttam /
sadaiva śiśnodaravarjitatvād ayācyayācñārahitasya rahoḥ // 37 //
sevāvrate prāya ivāpaviṣṭaḥ sarvātmanā niścitajīvanāśaḥ /
dvāḥsthaiḥ prayatnena nivārito 'pi na durgrahaṃ muñcati naṣṭasaṃjñaḥ // 38 //
tamo'dhvatāpena ca cintayā ca śitena dainyena ca piṇḍitāṅgaḥ /
sukhāśayā duḥkhasahasrabhāgī sevāvrataṃ mugdhamatir bibharti // 39 //
gacchāmi gacchāmi dinaṃ gataṃ me dine gato 'haṃ na nṛpo 'dya dṛṣṭaḥ /
iti bruvāṇasya sadā janasya jīrṇātarāṃ tasya tanur na tṛṣṇā // 40 //
vicchedakārī sahasā kathānām anīpsito dīpa iva praviṣṭaḥ /
saṃsaktavastrāntaramakṣikeva na sevakaḥ kasya karoti duḥkham // 41 //
dhūmāyamāno 'kṣivipakṣabhūtaś cāṭukramotpāditakarṇaśūlaḥ /
kusevakaḥ pādatale 'valagnaḥ pade pade kaṇṭakatām upaiti // 42 //
ekaḥ kham eva kṣitim īkṣate 'nyaḥ sa nirjanārthī sa ca gāḍhalagnaḥ /
svasvārthitā tasya bhṛśaṃ sa cārthī kathaṃ sa sevyaḥ sa ca sevako 'stu // 43 //
chāyāgraho mūrta ivānuyāyī sthitaś ciraṃ vṛścikaniścalaś ca /
akālapātī sukhakelilīlāyantrotpalaḥ sevakadurvidagdhaḥ // 44 //
visṛjyamāno 'py aniśaṃ na yāti tīvroparodhapraṇayena mūrkhaḥ /
bhartuḥ prasāde vihitaprayatno jātaḥ paraṃ pratyuta kopahetuḥ // 45 //
śūtkāridīrghaśvasitānubandhair niveditodvegagatiprayāsaḥ /
atipraveśāt kharamārgaṇo 'sau nirasyamāno viphalatvam eti // 46 //
śrīratnacandravaravājigajorjite 'smin gambhīrarājakulanāmni mahāsamudre /
aśnāti mūḍhamatir īśvaratām avāptum āśāvaśāt kṣitipater bahu kālakūṭam // 47 //
virama virama neyaṃ pāntha namrāmramālā badhira khadirapālī niṣphalaiṣā prayāhi /
iti bahuvidham uktaḥ sevako 'nyāpadeśais tyajati na vipulāśāpāśabaddhaḥ kusevām // 48 //
pramlānā sarasatvam eti na punar māleva lagnātapā bhagnā kācamayīva saṃdhighaṭanā yogena na śliṣyati /
sevā dīpaśikheva durgatagṛhe saṃdhāryamāṇā paraṃ niḥsnehā kṣayam eti khedajanitair ucchvāsamālānilaiḥ // 49 //
kas tvaṃ gatvara sevakas tvaritatā kasmāt prabhur dṛśyate lābhaḥ kas tava tatra vetrivipulāghātaiḥ śirastāḍanam /
kiṃ mṛdgāsi mudā viśālacaraṇair mithyaiva rathyām imām bandho vandhyanarendrabhavyabhavanabhrāntyā viḍambyāmahe // 50 //
suptākarṣaṇadāma dhāma mahato mohasya dainyasya vā sevākleśam aharniśaṃ viṣahate yo vittaleśāśayā /
prasphūrjadvaḍavāgnigarbhagahanodbhūtormimālākulaṃ saṃkṣubdhaṃ makarākaraṃ praviśati śrīratnalabdhyai na kim // 51 //
vyarthātyarthagatāgatavyatikarair aśrāntatīvratvaro dvārālokanatatparaḥ parijanair utsāryamāṇaḥ param /
yatnenaiva puraḥ praveśakalanaiḥ kleśāvamānāśrayaḥ sarvāpatpiśunaḥ śunaḥ sadṛśatām ālapsyate sevakaḥ // 52 //
kūjatkrūrakapāṭapīḍanaruṣā dattaprahāraiḥ paraṃ dvārāvasthaganāya susthirabhujair dvāḥsthair bhṛśaṃ bhartsitaḥ /
kubjībhūya janasya jānuvivaraiḥ kṣiprapraveśotsukaḥ pṛcchaty antaranirgatānavasaraṃ maunavratān sevakaḥ // 53 //
dvāre ruddham upekṣate katham api prāptaṃ puro nekṣate vijñaptau gajamīlanāni kurute gṛhṇāti vākyacchalam /
niryātasya karoti doṣagaṇanāṃ svalpāparādhe yamaḥ sa svāmī yadi sevyate marutaṭe kiṃ naḥ piśācaiḥ kṛtam // 54 //
nityaṃ rājakule na pūjyayajane taddvārapālārcanaṃ bhaktiṃ bhūtisamudbhave na tu bhave dhyānaṃ dhane nātmani /
dṛṣṭādṛṣṭavicintanaṃ narapatau na svocite karmaṇi śrīhetor bata niṣphalaṃ prakurute sarvaṃ jaḍaḥ sevakaḥ // 55 //
sā tīvraṃ jaḍanimnapātanaratā no nityasaktā tvarā rathyāpaṅkakalaṅkitaṃ na vadanaṃ na dvārapālaiḥ kaliḥ /
yasmiṃs taccirasevakaḥ pravitataṃ puṇyaṃ vanaṃ gamyatāṃ krūrās te na bhavanti tatra vikṛtakrodhoddhatāḥ pārthivāḥ // 56 //
haṃho kaṣṭavinaṣṭa sevaka sakhe khinno 'si poṣāśayā rāgaś ced vibhave tad eṣa kupatiḥ klībaḥ kim āsevyate /
sevyaḥ ko 'pi maheśvaraḥ sa bhagavān yasya prasādekṣaṇair akṣuṇṇaiḥ karuṇāspadīkṛtamarunnātho maruttaḥ kṛtaḥ // 57 //
rājñām ajñatayā kṛtaṃ yad aniśaṃ dainyaṃ tad utsṛjyatāṃ saṃtoṣambhasi mṛjyatām api rajaḥ pādapraṇāmārjitam /
saṃtoṣaḥ paramaḥ purāṇapuruṣaḥ saṃvinmayaḥ sevyatāṃ yatsmṛtyā na bhavanti te sumanasāṃ bhūyo bhavagranthayaḥ // 58 //
utsṛjya prājyasevāṃ vijanasukhajuṣāṃ bhūbhujāṃ vyājabhājāṃ chittvāśāpāśabandhān vimalaśamajalais tīvratṛṣṇāṃ nivārya /
sthitva śuddhe samādhau kim aparam amṛtaṃ mṛgyatāṃ jyotir antar yasmin dṛṣṭe vinaṣṭotkaṭatimirabhare labhayte mokṣalakṣmīḥ // 59 //
vṛttyā jīvati lokaḥ sevāvṛttir nijaiva keṣāṃcit /
asthāne tīvratarā nindyā tu tadarthināṃ sevā // 60 //
vidvajjanārādhanatatpareṇa saṃtoṣasevārasanirbhareṇa /
kṣemendranāmnā sudhiyāṃ sadaiva sukhāya sevāvasaraḥ kṛto 'yam // 61 //
iti śrīvyāsadāsāparākhyamahākaviśrīkṣemendrakṛtaḥ sevyasevakopadeśaḥ samāptaḥ /
