oṃ
arcitaḥ saṃsmṛto dhyātaḥ kīrtitaḥ kathitaḥ śrutaḥ yo dadāty amṛtatvaṃ hi sa māṃ rakṣatu keśavaḥ // 1 //
tāpatrayeṇa saṃtaptaṃ yad etad akhilaṃ jagat vakṣyāmi śāntaye hy asya kṛṣṇāmṛtamahārṇavam // 2 //
te narāḥ paśavo loke kiṃ teṣāṃ jīvane phalam yair na labdhā harer dīkṣā nārcito vā janārdanaḥ // 3 //
saṃsāre 'smin mahāghore janmarogabhayākule ayam eko mahābhāgaḥ pūjyate yad adhokṣajaḥ // 4 //
sa nāma sukṛtī loke kulaṃ tena hy alaṃkṛtam ādhāraḥ sarvabhūtānāṃ yena viṣṇuḥ prasāditaḥ // 5 //
yajñānāṃ tapasāṃ caiva śubhānāṃ caiva karmaṇāṃ tadviśiṣṭaphalaṃ nṛṇāṃ sadaivārādhanaṃ hareḥ // 6 //
kalau kalimaladhvaṃsisarvapāpaharaṃ harim ye 'rcayanti narā nityaṃ te 'pi vandyā yathā hariḥ // 7 //
nāsti śreyaskaraṃ nṛṇāṃ viṣṇor ārādhanān mune yuge 'smiṃs tāmase loke satataṃ pūjyate nṛbhiḥ // 8 //
arcite sarvadeveśe śaṅkacakragadādhare arcitāḥ sarvadevāḥ syur yataḥ sarvagato hariḥ // 9 //
svarcite sarvalokeśe surāsuranamaskṛte keśave kaṃsakeśighne na yāti narakaṃ naraḥ // 10 //
sakṛd abhyarcya govindaṃ bilvapatreṇa mānavaḥ muktibhāgī nirātaṅkī viṣṇuloke ciraṃ vaset // 11 //
sakṛd abhyarcito yena heḷayāpi namaskṛtaḥ sa yāti paramaṃ sthānaṃ yat surair api durlabham // 12 //
nāradaḥ:
samastalokanāthasya devadevasya śārṅgiṇaḥ sākṣād bhagavato viṣṇoḥ pūjanaṃ janmanaḥ phalam // 13 //
pulastyaḥ:
bhaktyā dūrvāṅkuraiḥ pumbhiḥ pūjitaḥ puruṣottamaḥ harir dadāti hi phalaṃ sarvayajñaiś ca durlabham // 14 //
vidhinā devadeveśaḥ śaṅkhacakradharo hariḥ phalaṃ dadāti sulabhaṃ salilenāpi pūjitaḥ // 15 //
narake pacyamānas tu yamena paribhāṣitaḥ kiṃ tvayā nārcito devaḥ keśavaḥ kleśanāśanaḥ // 16 //
dharmaḥ:
narasiṃho hṛṣīkeśaḥ puṇḍarīkanibhekṣaṇaḥ smaraṇān muktido nṛṇāṃ sa tvayā kiṃ na pūjitaḥ // 17 //
dravyāṇām apy abhāve tu salilenāpi pūjitaḥ yo dadāti svakaṃ sthānaṃ sa tvayā kiṃ na pūjitaḥ // 18 //
garbhasthitāmṛtā vāpi muṣitāste sudūṣitāḥ na prāptā yair harer dīkṣā sarvaduḥkhavimocanī // 19 //
mārkaṇḍeyaḥ:
sakṛd abhyarcito yena devadevo janārdanaḥ yat kṛtaṃ tatkṛtaṃ tena samprāptaṃ paramaṃ padam // 20 //
dharmārthakāmamokṣāṇāṃ nānyopāyas tu vidyate satyaṃ bravīmi deveśa hṛṣīkeśārcanāhate // 21 //
tasya yajñavarāhasya viṣṇor amitatejasaḥ praṇāmaṃ ye 'pi kurvanti teṣām api namo namaḥ // 22 //
marīciḥ:
anārādhitagovindair naraiḥ sthānaṃ nṛpātmaja na hi samprāpyate śreṣṭhaṃ tasmād ārādhayācyutam // 23 //
atriḥ:
paraḥ parāṇāṃ puruṣas tuṣṭo yasya janārdanaḥ sa cāpnoty akṣayaṃ sthānam etat satyaṃ mayoditam // 24 //
aṅgirāḥ:
yasyāntaḥ sarvam evedam acyutasyākhyayātmanas tam ārādhaya govindaṃ sthānam agryaṃ yadīcchasi // 25 //
pulahaḥ:
paraṃ brahma paraṃ dhāma yo 'sau brahma sanātanam tam ārādhya hariṃ yāti muktim apy atidurlabhām // 26 //
aindram indraḥ paraṃ sthānaṃ yam ārādhya jagatpatim prāpa yajñapatiṃ viṣṇuṃ tam ārādhaya suvrata // 27 //
prāpnoty ārādhite viṣṇau manasā yad yad icchati trailokyāntargataṃ sthānaṃ kimu lokottarottarān // 28 //
ye smaranti sadā viṣṇuṃ śaṅkhacakragadādharam sarvapāpavinirmuktāḥ paraṃ brahma viśanti te // 29 //
tato 'niruddhaṃ deveśaṃ pradyumnaṃ ca tataḥ param tataḥ saṃkarṣaṇaṃ devaṃ vāsudevaṃ parāt param // 30 //
vāsudevāt paraṃ nāsti iti vedāntaniścayaḥ vāsudevaṃ praviṣṭānāṃ punar āvartanaṃ kutaḥ // 31 //
ātreyaḥ:
yo yān icchen naraḥ kāmān nārī vā varavarṇinī tān samāpnoti vipulān samārādhya janārdhanam // 32 //
brahmā:
bāhubhyāṃ sāgaraṃ tartuṃ ka iccheta pumān bhuvi vāsudevam anārādhya ko mokṣaṃ gantum icchati // 33 //
śaṃkaraḥ:
kṛte pāpe 'nutāpo vai yasya puṃsaḥ prajāyate prāyaścittaṃ tu tasyoktaṃ harisaṃsmaraṇaṃ param // 34 //
na hy apuṇyavatāṃ loke mūḍhānāṃ kuṭilātmanāṃ bhaktir bhavati govinde smaraṇaṃ kīrtanaṃ tathā // 35 //
tadaiva puruṣo mukto janmaduḥkhajarādibhiḥ jitendriyo viśuddhātmā yadaiva smarate harim // 36 //
prāpte kaliyuge ghore dharmajñānavivarjite na kaścit smarate devaṃ kṛṣṇaṃ kalimalāpaham // 37 //
na kalau devadevasya janmaduḥkhāpahāriṇaḥ karoti martyo mūḍhātmā smaraṇaṃ kīrtanaṃ hareḥ // 38 //
ye smaranti sadā viṣṇuṃ viśuddhenāntarātmanā te prayānti bhavaṃ tyaktvā viṣṇulokam anāmayam // 39 //
garbhajanmajarārogaduḥkha saṃsārabandhanaiḥ na bādhyate naro nityaṃ vāsudevam anusmaran // 40 //
yamamārgāṃ mahāghoraṃ narakāṇi yamaṃ tathā svapne 'pi ca na paśyeta yaḥ smared garuḍadhvajam // 41 //
hṛdi rūpaṃ mukhe nāma naivedyam udare hareḥ pādodakaṃ ca nirmālyaṃ mastake yasya so 'cyutaḥ // 42 //
govindasmaraṇaṃ puṃsāṃ pāparāśimahācalam asaṃśayaṃ dahaty āśu tūlarāśim ivānalaḥ // 43 //
agastyaḥ:
smaraṇād eva kṛṣṇasya pāpasaṅghātapañjaram śatadhā bhedam āyāti girir vajrahato tathā // 44 //
kṛṣṇe ratāḥ kṛṣṇam anusmarantaḥ tad bhāvitās tad gatamānasāś ca te bhinnadehāḥ praviśanti viṣṇuṃ havir yathā mantrahutaṃ hutāśe // 45 //
sā hānis tan mahat chidraṃ sā cāndhajaḍamūkatā yan muhūrtaṃ kṣaṇaṃ vāpi vāsudevo na cintyate // 46 //
nārāyaṇo nāma naro narāṇāṃ prasaṃhya coraḥ kathitaḥ pṛthivyām anekajanmārjitapāpasañcayaṃ dahaty aśeṣaṃ smṛtamātrayaiva // 47 //
yasya saṃsmaraṇād eva vāsudevasya śārṅgiṇaḥ koṭijanmārjitaṃ pāpaṃ takṣaṇād eva naśyati // 48 //
kiṃ tasya bahubhis tīrthaiḥ kiṃ tapobhiḥ kim adhvaraiḥ yo nityaṃ dhyāyate devaṃ nārāyaṇam ananyadhīḥ // 49 //
ye mānavā vigatarāgaparāvarajñā nārāyaṇaṃ suraguruṃ satataṃ smaranti dhyānena tena hatakilbiṣacetanāste mātuḥ payodhararasaṃ na punaḥ pibanti // 50 //
he citta cintaya sveha vāsudevam aharniśam nūnaṃ yaś cintitaḥ puṃsāṃ hanti saṃsārabandhanam // 51 //
āloḍya sarvaśāstrāṇi vicārya ca punaḥ punaḥ idam ekaṃ suniṣpannaṃ dhyeyo nārāyaṇaḥ sadā // 52 //
smṛte sakalakalyāṇabhājanaṃ yatra jāyate puruṣas tam ajaṃ nityaṃ vrajāmi śaraṇaṃ harim // 53 //
vedeṣu yajñeṣu tapasu caiva dāneṣu tīrtheṣu vrateṣu caiva iṣṭeṣu pūrteṣu ca yat pradiṣṭaṃ puṇyaṃ smṛte tat khalu vāsudeve // 54 //
aurvaḥ:
ārādhayaiva naro viṣṇuṃ manaso yad yad icchati phalaṃ prāpnoty avikalaṃ bhūri svalpam athāpi vā // 55 //
yan nāmakīrtanaṃ bhaktyā vilāyanam anuttamam maitreyāśeṣapāpānāṃ dhātūnām iva pāvakaḥ // 56 //
kalikalmaṣam atyugraṃ narakārtipradaṃ nṛṇām prayāti vilayaṃ sadyaḥ sakṛt saṃkīrtite 'cyute // 57 //
anāyāsena cāyānti muktiṃ keśavam āsritāḥ tadvighātāya jāyante śakrādyāḥ paripanthinaḥ // 58 //
catussāgaram āsādya jambūdvīpottame kvacit na pumān keśavād anyaḥ sarvapāpacikitsakaḥ // 59 //
yad abhyarcya hariṃ bhaktyā kṛte varṣaśatair api phalaṃ prāpnoty avikalaṃ kalau saṃkīrtya keśavam // 60 //
kṣīyate tu yadā dharmaḥ prāpte ghore kalau yuge tadā na kīrtayet kaścin muktidaṃ devam acyutam // 61 //
avaśenāpi yannāmni kīrtite sarvapātakaiḥ pumān vimucyate sadyaḥ siṃhatrastamṛgair iva // 62 //
nārāyaṇeti mantro 'sti vāg asti vaśavartinī tathāpi narake ghore majjantīty etad adbhutam // 63 //
ārtā viṣaṇṇāḥ śithilāś ca bhītā ghoreṣu ca vyādhiṣu vartamānāḥ saṃkīrtya nārāyaṅaśabdamātraṃ vimuktaduḥkhāḥ sukhino bhavanti // 64 //
kauśikaḥ:
sakṛd uccāritaṃ yais tu kṛṣṇeti na viśanti te garbhāgāraṃ gṛhaṃ mātur yamalokaṃ ca dussaham // 65 //
kva nākapṛṣṭhagamanaṃ punarāvṛttilakṣaṇam kva japo vāsudeveti muktibījam anuttamam // 66 //
buddhyā buddhvā vadasvainaṃ harir ity akṣaradvayam smaraṇād eva kṛṣṇasya na punarjāyate kvacit // 67 //
he jihve mama nisnehe hariṃ kiṃ nuna bhāṣase hariṃ vadasva kalyāṇi saṃsārodadhinaur hariḥ // 68 //
asāre khalu saṃsāre sārāt sārataro hariḥ puṇyahīnā na vindanti sāraṅgāś ca yathā jalam // 69 //
kurukṣetreṇa kiṃ tasya kiṃ kāśyā puṣkareṇa vā jihvāgre vartate tasya harir ity akṣaradvayam // 70 //
brahmovāca:
asāre khalu saṃsāre sāram ekaṃ nirūpitam samastalokanāthasya sāram ārādhanaṃ hareḥ // 71 //
sā jihvā yā hariṃ stauti tac cittaṃ yat tadarpaṇam tāv eva kevalau ślāghyau yau tatpūjākarau karau // 72 //
yas tu viṣṇuparo nityaṃ dṛḍhabhaktir jitendriyaḥ svagṛhe 'pi vasan yāti tad viṣṇoḥ paramaṃ padam // 73 //
śaṅkaraḥ:
sādhu sādhu mahābhāga sādhu dānavanāśan yan māṃ pṛcchasi dharmajña keśavārādhanaṃ prati // 74 //
nimiṣaṃ nimiṣārdhaṃ vā muhūrtam api bhārgava nādagdhāśeṣapāpānāṃ bhakti bhavati keśave // 75 //
kiṃ tena manasā kāryaṃ yan na tiṣṭhati keśave mano muktiphalāvāptyau kāraṇaṃ saprayojanam // 76 //
rogo nāma na sā jihvā yayā na stūyate hariḥ gartau nāma na tau karṇau yābhyāṃ tat karma na śrutam // 77 //
nūnaṃ tat kaṇṭhaśālūkam athavāpy upajihvikā rogo nāma na sā jihvā yā na vakti harer guṇān // 78 //
bhārabhūtaiḥ kāryam ebhiḥ kiṃ tasya nṛpaśor dvija caraṇau tau tu saphalau keśavālayagāminau te ca netre mahābhāga yābhyāṃ sandṛśyate hariḥ // 79 //
kiṃ tasya caraṇaiḥ kāryaṃ vṛthāsañcaraṇair dvija yair hi na vrajate jantuḥ keśavālayadarśane // 80 //
vedavedāṅgaviduṣāṃ munīnāṃ bhāvitātmanām ṛṣitvam api dharmajña vijñeyaṃ tatprasādajam // 81 //
vicitraratnaparyaṅke mahābhogena bhoginaḥ ramante nākanārībhiḥ keśavasmaraṇāt phalam // 82 //
aśvamedhasahasrāṇāṃ yaḥ sahasraṃ samācaret nāsau tat phalam āpnoti tadbhaktair yad avāpyate // 83 //
re re manuṣyāḥ puruṣottamasya karau na kasmān mukulīkurudhve kriyājuṣāṃ ko bhavatāṃ prayāsaḥ phalaṃ hi yat padam acyutasya // 84 //
viṣṇor vimānaṃ yaḥ kuryāt sakṛd bhaktyā pradakṣiṇam aśvamedhasahasrasya phalam āpnoti mānavaḥ // 85 //
pradakṣiṇaṃ tu yaḥ kuryād haribhaktyā samanvitaḥ haṃsayuktivimānena viṣṇulokaṃ sa gacchati // 86 //
tīrthakoṭisahasrāṇi vratakoṭiśatāni ca nārāyaṇapraṇāmasya kalāṃ nārhanti ṣoḍaśīm // 87 //
urasā śirasā dṛṣṭyā manasā vacasā tathā padbhyāṃ karābhyāṃ jānubhyāṃ praṇāmo 'ṣṭāṅga īritaḥ // 88 //
śāṭhyenāpi namaskāraṃ kurvataḥ śārṅgapāṇaye śātajanmārjitaṃ pāpaṃ naśyaty eva na saṃśayaḥ // 89 //
saṃsārārṇavam agrānāṃ narāṇāṃ pāpakarmaṇām nānyo dharttā jagannāthaṃ muktvā nārāyaṇaṃ param // 90 //
reṇukuṇṭhigātrasya kaṇā yāvanti bhārata tāvad varṣasahasrāṇi viṣṇuloke mahīyate // 91 //
pāvanaṃ viṣṇunaivedyaṃ subhojyam ṛṣibhiḥ smṛtam anyadevasya naivedyaṃ bhuktvā cāndrāyaṇaṃ caret // 92 //
koṭyaindavasahasrais tu māsopoṣaṇakoṭibhiḥ yat phalaṃ labhyate pumbhir viṣṇor naivedya bhakṣaṇāt // 93 //
trirātraphaladā nadyo yāḥ kāścid asamudragāḥ samudragās tu pakṣasya māsasya saritāṃ patiḥ // 94 //
ṣaṇmāsaphaladā godā vatsarasya tu jāhnavī viṣṇupādodakasyaite kalāṃ nārhanti ṣoḍasīm // 95 //
gaṇgāprayāgagayapuṣkaranaimiśāni saṃsevitāni bahuśaḥ kurujāṅgalāni kālena tīrthasalilāni punanti pāpaṃ pādodakaṃ bhagavataḥ prapunāti sadyaḥ // 96 //
yāni kāni ca tīrthāni brahmāṇḍāntargatāni vai viṣṇupādodakasyaite kalāṃ nārhanti ṣoḍaśīm // 97 //
snānaṃ pādodakaṃ viṣṇoḥ piban śirasi dhārayan sarva pāpavinirmukto vaiṣṇavīṃ siddhim āpnuyāt // 98 //
yathā pādodakaṃ puṇyaṃ nirmālyaṃ cānulepanam naivedyaṃ dhūpaśeṣaṃ ca ārārttiś ca tathā hareḥ // 99 //
tulasyās tu rajojuṣṭaṃ naivedyasya ca bhakṣaṇam nirmālyaṃ śirasā dhāryaṃ mahāpātakanāśanam // 100 //
bhaktyā vā yadi vābhaktyā cakrāṅkitaśilām prati darśanaṃ sparśanaṃ vāpi sarvapāpapraṇāśanam // 101 //
sāḷagrāmodbhavo devo devo dvāravatībhavaḥ ubhayoḥ snānatoyena brahmahatyāṃ vyapohati // 102 //
mlecchadeśe 'śucau vāpi cakrāṅko yatra tiṣṭhati yojanāni tathā trīṇi mama kṣetraṃ vasundhare // 103 //
śālagrāmodbhavo devo śailaṃ cakrāṅkamaṇḍitam yatrāpi nīyate tatra vāraṇasyā śatādhikam // 104 //
sāḷagrāmodbhavo devo devo dvāravatībhavaḥ ubhayoḥ saṃgamo yatra tatra muktir na saṃśayaḥ // 105 //
hariṇā muktidānīha muktisthānāni sarvaśaḥ sa yasya sarvabhāveṣu tasya taiḥ kiṃ prayojanam // 106 //
harir yāti harir yāti dasyuvyājena yo vadet so 'pi sadgatim āpnoti gatiṃ sukṛtino yathā // 107 //
vāsudevaṃ parityajya yo 'nyaṃ devam upāsate tyaktvāmṛtaṃ sa mūḍhātmā bhuṅkte hālāhalaṃ viṣam // 108 //
tyaktvāmṛtaṃ yathā kaścid anyapānaṃ piben naraḥ tathā hariṃ parityajya cānyaṃ devam upāsate // 109 //
svadharmaṃ tu parityajya paradharmaṃ yathā caret tathā hariṃ parityajya cānyaṃ devam upāsate // 110 //
gāṃ ca tyaktvā vimūḍhātmā gardabhīṃ vandate yathā tathā hariṃ parityajya cānyaṃ devam upāsate // 111 //
vāsudevaṃ parityajya yo 'nyaṃ devam upāsate tṛṣito jāhnavītīre kūpaṃ khanati durmatiḥ // 112 //
yathā gaṅgodakaṃ tyaktvā pibet kūpodakaṃ naraḥ tathā hariṃ parityajya yo 'nyaṃ devam upāsate // 113 //
svamātaraṃ parityajya śvapākīṃ vandate tathā tathā hariṃ parityajya yo 'nyaṃ devam upāsate // 114 //
yāvat svastham idaṃ piṇḍaṃ nirujaṃ karaṇānvitam tāvat kuruṣvātmahitaṃ paścāt tāpena tapyase // 115 //
yāvat svāsthyaṃ śarīreṣu karaṇeṣu ca pāṭavam tāvad arcaya govindam āyuśyaṃ sārthakaṃ kuru // 116 //
smaryatāṃ tu hṛṣīkeśo hṛṣīkeṣu dṛḍheṣu ca adṛḍheṣu hṛṣīkeṣu hṛṣīkeśaṃ smaranti ke // 117 //
yāvac cintayate jantur viṣayān viṣasannibhān tāvac cet smarate viṣṇuṃ ko na mucyeta bandhanāt // 118 //
yāvat pralapate jantur viṣayān viṣasannibhān tāvac cet smarate viṣṇuṃ ko na mucyeta bandhanāt // 119 //
sūta uvāca:
jñātvā viprās tithiṃ samyag daivajñaiḥ samudīritām kartavya upavāsaś ca anyathā narakaṃ vrajet // 120 //
kṣaye vāpy athavā vṛddhau samprāpte vā dinakṣaye upoṣyā dvādaśī puṇyā pūrvaviddhāṃ parityajet // 121 //
pūrvaviddhāṃ prakurvāṇo naro dharmān nikṛntati saṃtates tu vināśāya sampado haraṇāya ca // 122 //
kalāvedhe tu viprendrā daśamyaikādaśīṃ tyajet surāyā bindunā spṛṣṭaṃ gaṅgāmbha iva santyajet // 123 //
śvadṛtau pañcagavyaṃ ca daśamyā dūṣitāṃ tyajet ekādaśīṃ dvijaśreṣṭhāḥ pakṣayor ubhayor api // 124 //
tasmād viprā na viddhā hi kartavyaikādasī kvacit viddhā hanti purāpuṇyaṃ śrāddhaṃ ca vṛṣalīpatiḥ // 125 //
japtaṃ dattaṃ hutaṃ snātaṃ tathā pūjā kṛtā hareḥ tat sarvaṃ vilayaṃ yāti tamaḥ sūryodaye yathā // 126 //
ekādaśyāṃ yadā brahman dinakṣayatithir bhavet upoṣyā dvādaśī tatra trayodaśyāṃ tu pāraṇam // 127 //
pratipatprabhṛtayaḥ sarvā udayād udayād raveḥ sampūrṇā iti vijñeyā harivāsaravarjitāḥ // 128 //
aruṇodayakāle tu daśamī yadi dṛśyate pāpamūlaṃ tadā jñeyam ekādaśyupavāsanam // 129 //
aruṇodayakāle tu daśamī yadi dṛśyate na tatraikādaśī kāryā dharmakāmārthanāśinī // 130 //
catastro ghaṭikāḥ prātar aruṇodaya ucyate yatīnāṃ snānakālo 'yaṃ gaṅgāmbhaḥsadṛśaḥ smṛtaḥ // 131 //
udayāt prāg yadā viprā muhūrtadvayasaṃyutā sampūrṇaikādaśī nāma tatraivopavased vratī // 132 //
udayāt prāk trighaṭikā vyāpiny ekādaśī yadā sandigdhaikādaśī nāma varjyā dharmārthanāśinī // 133 //
putrapautravivṛddhyarthaṃ dvādaśyām upavāsayet tatra kratuśataṃ puṇyaṃ trayodaśyāṃ tu pāraṇaṃ // 134 //
udayāt prāg dvighaṭikāvyāpinyekādaśī yadā saṅkīrṇaikādaśī nāma varjyā dharmārthakāṅkṣibhiḥ // 135 //
putrarājyavivṛddhyarthaṃ dvādaśyām upavāsanam tatra kratuśataṃ puṇyaṃ trayodaśyāṃ tu pāraṇam // 136 //
daśamīśeṣasaṃyuktā gāndhāryā samupoṣitā tasyāḥ putraśataṃ naṣṭaṃ tasmāt tāṃ parivarjayet // 137 //
bahvāgamavirodheṣu brāhmaṇeṣu vivādiṣu upoṣyā dvādaśī puṇyā trayodaśyāṃ tu pāraṅam // 138 //
ekādaśyāṃ tu viddhāyāṃ samprāpte śravaṇe tathā upoṣyā dvādaśī puṇyā pakṣayor ubhayor api // 139 //
uparāgasahasrāṇi vyatīpātāyutāni ca amālakṣaṃ tu dvādaśyāḥ kalāṃ nārhanti ṣoḍaśīm // 140 //
śuddhāpi dvādaśī grāhyā parato dvādaśī yadi viṣaṃ tu daśamī jñeyāmṛtaṃ caikādaśī tithiḥ viṣapradhānā varjyā sāmṛtā grāhyā pradhānataḥ // 141 //
dvādaśyāṃ bhojanaṃ caiva viddhāyāṃ haryupoṣaṇam yaḥ kuryān mandabuddhitvān nirayaṃ so 'dhigacchati // 142 //
yāni kāni ca vākyāni viddhopāsyāparāṇi tu dhanadārcāparāṇi syur vaiṣṇavī na daśāyutā // 143 //
athavā mohanārthāya mohinyā bhagavān hariḥ arthitaḥ kārayām āsa vyāsarūpī janārdanaḥ // 144 //
dhanadārcāvivṛddhyarthaṃ mahāvittalayasya ca asurāṇāṃ mohanārthaṃ pāṣaṇḍānāṃ vivṛddhaye ātmasvarūpāvijñaptyai svarūpāprāptaye tathā // 145 //
evaṃ viddhāṃ parityajya dvādaśyām upavāsane koṭijanmārjitaṃ pāpam ekayaiva vinaśyati tataḥ koṭiguṇaṃ vāpi niṣiddhasyetarair janaiḥ // 146 //
yad anādikṛtaṃ pāpaṃ tad ūrdhvaṃ yat kariṣyati tat sarvaṃ vilayaṃ yāti pareṣām upavāsanān na ca tasmāt priyatamaḥ keśavasya mamāpi vā // 147 //
ekādaśyā hy avedhe tu dvādaśīṃ na parityajet pāraṇe maraṇe caiva tithis tātkālikī smṛtā // 148 //
brahmacārī gṛhastho vā vānaprastho yatis tathā brāhmaṇaḥ kṣatriyo vaiśyaḥ śūdro bhartṛmatī tathā // 149 //
abhartṛkā tathānye vā sūtavaidehikādayaḥ ekādaśyāṃ na bhuñjīta pakṣayor ubhayor api // 150 //
ekādaśyāṃ tu yo bhuṅkte mohenāvṛtacetanaḥ śuklāyām atha kṛṣṇāyāṃ nirayaṃ yāti sa dhruvam // 151 //
vivecayati yo mohāc chuklā kṛṣṇeti pāpakṛt ekādaśīṃ sa vai yāti nirayaṃ nātra saṃśayaḥ // 152 //
yathā gaur naiva hantavyā śuklā kṛṣṇeti bhāmini ekadāśyāṃ na bhuñjīta pakṣayor ubhayor api // 153 //
yāni kāni ca vākyāni kṛṣṇaikādaśivarjane bharaṇyādiniṣedhe 'pi tāni kāmyaphalārthinām // 154 //
kāmino 'pi hi nityārthaṃ kuryur evopavāsanam prīṇanārthaṃ harer nityaṃ na tu kāmavyapekṣayā // 155 //
tasmāc chuklām atho kṛṣṇāṃ bharaṇyādiyutām api pratyavāyaniṣedhārtham upavāsīta nityaśaḥ prīṇanārthaṃ hareś vāpi viṣṇulokasya cāptaye // 156 //
kalā vā ghaṭikā vāpi apare dvādaśī yadi dvādaśadvādaśīr hanti pūrvedyuḥ pāraṇe kṛte // 157 //
atiriktā dvādaśī cet sa tāṃ nopoṣayed yadi dvādaśadvādaśīr hanti dvādaśī cātilaṅghitā // 158 //
dvādaśyām atiriktāyāṃ yo bhuṅkte pūrvavāsare dvādaśadvādaśīr hanti dvādaśīṃ na parityajet // 159 //
dvādaśīṃ śravaṇopetāṃ yo nopoṣyāt sumandadhīḥ pañcasaṃvatsarakṛtaṃ puṇyaṃ tasya vinaśyati // 160 //
ekādaśīm upoṣyātha dvādaśīm apy upoṣayet na tatra vidhilopaḥ syād ubhayor devatā hariḥ // 161 //
pārayitvodakenāpi bhuñjāno naiva duṣyati aśitānaśitā yasmād āpo vidvadbhir īritāḥ ambhasā kevalenātha kariṣye vratapāraṇam // 162 //
na kāśī na gayā gaṅgā na reva na ca gautamī na cāpi kauravaṃ kṣetraṃ samā bhūpa harer dināt // 163 //
aśvamedhasahasrāṇi vājapeyāyutāni ca ekādaśyupavāsasya kalāṃ nārhanti ṣoḍaśīm // 164 //
ekādaśīsamutthena vahninā pātakendhanam bhasmībhavati rājendra api janmaśatodbhavam // 165 //
nedṛśaṃ pāvanaṃ kiñcin narāṇāṃ bhuvi vidyate yādṛśaṃ padmanābhasya dinaṃ pātakahānidam // 166 //
tāvat pāpāni dehe 'smiṃs tiṣṭhanti manujādhipa yāvan nopoṣayej jantuḥ padmanābhadinaṃ śivam // 167 //
ekādaśendriyaiḥ pāpaṃ yat kṛtaṃ bhavati prabho ekādaśyupavāsena tat sarvaṃ vilayaṃ nayet // 168 //
ekādaśīsamaṃ kiñcit pāpatrāṇaṃ na vidyate vyājenāpi kṛtā rājan na darśayati bhāskarim // 169 //
śrī vedavyāsa uvāca:
sa brahmahā sa goghnaś ca stenaḥ sa gurutalpagaḥ ekādaśyāṃ tu bhuñjānaḥ pakṣayor ubhayor api // 170 //
varaṃ svamātṛgamanaṃ varaṃ gomāṃsabhakṣaṇam varaṃ hatyā surāpānam ekādaśyāṃ tu bhojanāt // 171 //
ekādaśīdine puṇye bhuñjate ye narādhamāḥ avalokya mukhaṃ teṣām ādityam avalokayet // 172 //

rugmāṅgadaḥ:

dharmavibhūṣaṇaḥ:
prātar haridine lokās tiṣṭhadhvaṃ caikabhojanāḥ akṣāralavaṇāḥ sarve haviṣyān niṣeviṇaḥ // 176 //

brahmā:
upoṣya dvādaśīṃ puṇyāṃ sarvapāpakṣayapradām na paśyati yamaṃ vāpi narakān na ca yātanām // 179 //
raṭantīha purāṇāni bhūyo bhūyo varānane na bhoktavyaṃ na bhoktavyaṃ samprāpte harivāsare // 180 //
dvādaśī na pramoktavyā yāvad āyuḥ pravartate arcanīyo hṛṣīkeśo viśuddhenāntarātmanā // 181 //
bhaktyā grāhyo hṛṣīkeśo na dhanair dharaṇīsurāḥ bhaktyā sampūjito viṣṇuḥ phalaṃ datte samīhitam // 182 //
jalenāpi jagannāthaḥ pūjitaḥ kleśanāśanaḥ paritoṣaṃ prayāty āśu tṛṣārtās tu yathā jalaiḥ // 183 //
āsīnasya śayānasya tiṣṭhato vrajato 'pi vā ramasva puṇḍarīkākṣa hṛdaye mama sarvadā // 184 //
sarvagaś caiva sarvātmā sarvāvasthāsu cācyuta ramasva puṇḍarīkākṣa nṛsiṃha hṛdaye mama // 185 //
karāvalambanaṃ dehi śrī kṛṣṇa kamalekṣaṇa bhavapaṅkārṇave ghore majjato mama sarvadā // 186 //
trāhi trāhi jagannātha vāsudevācyutāvyaya māṃ samuddhara govinda duḥkhasaṃsārasāgarāt // 187 //
etat puṇyaṃ paraṃ guhyaṃ pavitraṃ pāpanāśanam āyuṣyaṃ ca yaśasyaṃ ca kaliduḥsvapnanāśanam // 188 //
kalau pāpaṃ kiyanmātraṃ hatyāsteyādisambhavam smṛte manasi govinde dahyate tūlarāśivat // 189 //
kalau keśavabhaktānāṃ na bhayaṃ vidyate kvacit smṛte saṃkīrtite dhyāte saṃkṣayaṃ yāti pātakam // 190 //
adhyetavyam idaṃ śāstraṃ śrotavyam anasūyayā bhaktebhyaś ca pradātavyaṃ dhārmikebhyaḥ punaḥ punaḥ // 191 //
adhīyāna idaṃ śāstram viṣṇor māhātmyam uttamam sarvapāpavinirmuktaḥ prāpnoti paramaṃ padam // 192 //
śrutvā dharmaṃ vijānāti śrutvā tyajati durmatim śrutvā jñānam avāpnoti śrutvā mokṣaṃ ca gacchati // 193 //
tasmād idaṃ samāśrāvyaṃ śrotavyaṃ ca sadaiva hi kutarkadāvadagdhebhyo na dātavyaṃ kathañcana // 194 //
saṃsāraviṣapānena ye mṛtāḥ prāṇino bhuvi amṛtāya smṛtas teṣāṃ kṛṣṇāmṛtamaharṇavaḥ // 195 //
klinnaṃ pādodakenaiva yasya nityaṃ kaḷebaram tīrthakoṭisahasrais tu snāto bhavati pratyaham // 196 //
tīrthakoṭisahasrais tu sevitaiḥ kiṃ prayojanam toyaṃ yadi piben nityaṃ sāḷagrāmaśilācyutam // 197 //
sāḷagrāmaśilāsparśaṃ ye kurvanti dine dine vāñchanti karasaṃsparśaṃ teṣāṃ devāḥ savāsavāḥ // 198 //
duḥsaho nārako vahnir duḥsahā yamakiṅkarāḥ viṣamaśvāntakapathaḥ pretatvaṃ cātidāruṇam // 199 //
vicintya manasāpy evaṃ pātakād vinivarttayet smaraṇaṃ kīrtanaṃ viṣṇoḥ sadaiva na parityajet // 200 //
śrī vedavyāsaḥ:

sakṛd uccāritaṃ yena harir ity akṣaradvayam baddhaḥ parikaras tena mokṣāya gamanaṃ prati // 202 //
evaṃ brahmādayo devā ṛṣayaś ca tapodhanāḥ kīrtayanti suraśreṣṭhaṃ devaṃ nārāyaṇaṃ prabhum // 203 //
kiṃ tasya dānaiḥ kiṃ tīrthaiḥ kiṃ tapobhiḥ kim adhvaraiḥ yo nityaṃ dhyāyate devaṃ nārāyaṇam ananyadhīḥ // 204 //
nityotsavo nityatā ca nityaśrīr nityaśo jayaḥ yeṣāṃ hṛdistho bhagavān maṅgalāyatano hariḥ // 205 //
jīvaṃś caturdaśād ūrdhvaṃ puruṣo niyamena tu strī vāpy anūnadaśakaṃ dehaṃ mānuṣam ārjate // 206 //
caturdaśordhvajīvīni saṃsāraś cādivarjitaḥ ato 'vittvā paraṃ devaṃ mokṣāśā kā mahāmune // 207 //
ācaturdaśamād varṣāt karmāṇi niyamena tu daśāvarāṇāṃ dehānāṃ kāraṇāni karoty ayam ataḥ karmakṣayān muktiḥ kuta eva bhaviṣyati // 208 //
samānāṃ viṣamā pūjā viṣamāṇāṃ samā tathā kriyate yena devo 'pi svapadād bhraśyate hi saḥ // 209 //
vittaṃ bandhur vayaḥ karma vidyā caiva tu pañcamī etāni mānyasthānāni garīyo yad yad uttaram // 210 //
guṇānusāriṇīṃ pūjāṃ samāṃ dṛṣṭiṃ ca yo naraḥ sarvabhūteṣu kurute tasya viṣṇuḥ prasīdati // 211 //
yathā suhṛtsu kartavyaṃ pitṛśatrusuteṣu ca tathā karoti pūjādi samabuddhiḥ sa ucyate // 212 //
tiryakpuṇḍraṃ na kurvīta samprāpte maraṇe 'pi ca na cānyan nāma vibrūyāt paraṃ nārāyaṇādṛte // 213 //
naivedyaśeṣaṃ devasya yo bhunakti dine dine sikthe sikthe bhavet puṇyaṃ cāndrāyaṇaśatādhikam // 214 //
ūrdhvapuṇḍram ṛjuṃ saumyaṃ lalāṭe dṛśyate sa caṇḍālo 'pi śuddhātmā pūjya eva na saṃśayaḥ // 215 //
aśucir vāpy anācāro manasā pāpam ācaran śucir eva bhaven nityam ūrdhvapuṇḍrānkito naraḥ // 216 //
ūrdhvapuṇḍravihīnasya śmaśānasadṛśaṃ mukham avalokya mukhaṃ teṣām ādityam avalokayet // 217 //
yajño dānaṃ tapaś caiva svādhyāyaḥ pitṛtarpaṇam vyarthaṃ bhavati tat sarvam ūrdhvapuṇḍraṃ vinā kṛtam // 218 //
gopīcandanaliptāṅgo yaṃ yaṃ paśyati cakṣuṣā taṃ taṃ śuddhaṃ vijānīyān nātra kāryā vicāraṇā // 219 //
āsphoṭayanti pitaraḥ pranṛtyanti pitāmahāḥ vaiṣṇavo 'smatkule jātaḥ sa naḥ santārayiṣyati // 220 //
jīvitaṃ viṣṇubhaktasya varaṃ pañcadināny api na tu kalpasahasrais tu bhaktihīnasya keśave // 221 //
kiṃ tena jātamātreṇa bhūbhāreṇānnaśatruṇā yo jāto nārcayed viṣṇuṃ na smaren nāpi kīrtayet // 222 //
yo dadāti dvijātibhyaś candanaṃ gopimarditam api sarṣapamātreṇa punāty āsaptamaṃ kulam // 223 //
jñānī ca karmāṇi sado 'ditāni kuryād akāmaḥ satataṃ bhavet // 224 //
atītānāgatajñānī trailokyodvaraṇakṣamaḥ etādṛśo 'pi nācāraṃ śrautaṃ smārtaṃ parityajet // 225 //
yad eva vidyayā karoti śraddhayopaniṣadā sad eva vīryavattaraṃ bhavati // 226 //
kurvan eveha karmāṇi jijīviṣec chataṃ samāḥ evaṃ tvayi nānyatheto 'sti na karma lipyate nare // 227 //
