|| oṃ namaḥ sarvajñāya ||
adhigataparamārthaḥ śrīghanavyūha eva prati tuṣitajanārthaṃ yo 'bhavac chvetaketuḥ /
tata iha jagadarthaṃ cābhavac chākyaketuḥ sa jayatu jitamṛtyur darśitānantamāyaḥ // 1.1 //
ajarāmaratāṃ prāptam ajarāmarakārakam /
ajarāmaratāprāptyai natvā saṃbuddham ādarāt // 1.2 //
durbodhaṃ mandabuddhīnāṃ duṣkaraṃ calacetasām /
sāriṣṭaṃ vañcanaṃ mṛtyor ucyate śāstrasaṃgatam // 1.3 //
sarvathā vañcanaṃ mṛtyor muktānām eva yujyate /
amuktānāṃ ca sarveṣāṃ kiyatkālavilambanam // 1.4 //
āyurindriyacittānāṃ nirodho mṛtyulakṣaṇam /
upāyair jīvitaṃ dīrghaṃ mṛtyuvañcanam ucyate // 1.5 //
yāvaj jīvati saṃsāre dharmācārī jitendriyaḥ /
tāvat puṇyādisaṃbhāraṃ prasūte bodhisādhakam // 1.6 //
na vinā jīvitāt sāraṃ saṃsāre kiṃ cid īkṣyate /
nānopāyair atas tūrṇaṃ mṛtyuvañcanam ārabhet // 1.7 //
na duṣkaraṃ yataḥ kiṃ cid upāyair yuktiyojitaiḥ /
maṇimantrauṣadhīnāṃ ca prabhāvaḥ sarvasaṃmataḥ // 1.8 //
lakṣaṇād dūrato mṛtyuṃ jñātvā yuktā pratikriyā /
maste patati vajrāgnau suprajño 'pi karoti kim // 1.9 //
bāhyādhyātmikabhedena dvidhā maraṇalakṣaṇam /
kathyate jñāyate yena maraṇaṃ bhāvi tatparaiḥ // 1.10 //
mṛtyor lakṣaṇam ājñātuṃ vāñchā cet kasya cid bhavet /
tena svasthaśarīreṇa jñātavyaṃ saṃśayo 'nyathā // 1.11 //
dhātuvaiṣamyato rogāt timirādiprabhāvataḥ /
bhayaśokādisāmarthyāt pratibhāti yato 'nyathā // 1.12 //
daivabhūtobhayakṣobhān mṛtyuṃ trividhalakṣaṇam /
prāpyāriṣṭasamudbhūtir ato bhedena lakṣayet // 1.13 //
bāhyābhyantarabhedena daivakṣobho dvidheṣyate /
bhūtakṣobho 'pi dvividhas tathaiveti nirūpayet // 1.14 //
daivasaṃkṣobhato mṛtyur ādhidaivika ucyate /
daivakṣobho 'naśanādivajrapātādilakṣaṇaḥ // 1.15 //
bhūtasaṃkṣobhato mṛtyur ādhibhautika ucyate /
bhūtakṣobhas tu pittādivahnitoyādyupadravaḥ // 1.16 //
ubhayakṣobhato mṛtyur ubhayājjo nigadyate /
ubhayakṣobho vajrādipittaśleṣmādyupadravaḥ // 1.17 //
ariṣṭānām anantatvān notpattir yugapan matā /
kramaśo 'pi na sarveṣām api tv ekādisaṃkhyayā // 1.18 //
adṛṣṭvā lakṣaṇaṃ yatra mṛtyur dṛṣṭaḥ kva cid bhavet /
niścetur dūṣaṇaṃ tatra na tu lakṣaṇanāstitā // 1.19 //
dṛṣṭvāpi lakṣaṇaṃ yatra na mṛtyur dṛśyate kva cit /
abhrāntau tatra dharmādibādhakāntarakalpanam // 1.20 //
niścite dūrato mṛtyau prasiddhir mahatī bhavet /
yathāśakti parityāge pratīkāre ca vartanam // 1.21 //
sadā samāhito bhūtvā tadgataikāgramānasaḥ /
bāhyāni prathamaṃ tāval lakṣaṇāny upalakṣayet // 1.22 //
lakṣaṇānāṃ paricchedakālo dvividha iṣyate /
niyato 'niyataś ceti vijñeyaḥ sa tu yatnataḥ // 1.23 //
saṃkrāntipūrṇamāsyādau prabhātādikṣaṇātmakaḥ /
kālo niyata ity ukto 'niyatas tu viparyayāt // 1.24 //
sphuratsaṃdhi kapolaṃ syād atyarthaṃ mṛtyudarśanāt /
vātaśītādisaṅgāc ca lakṣaṇīyam ataḥ pṛthak // 1.25 //
karkaṭo rāśir āṣāḍhaḥ puṣyas tu makaro mataḥ /
karkaṭe makare cāpi jāyate mṛtyulakṣaṇam // 1.26 //
āṣāḍhasaṃkrāntidine madhyāhne padamātrikā /
uttarābhimukhasthasya cchāyā svābhāvikī bhavet // 1.27 //
puṣyasaṃkrāntidivase saiva saptapadībhavet /
vyabhre nabhasi madhyāhne cchāyāyāś ca nirīkṣaṇam // 1.28 //
pratimāsaṃ padavṛddhiḥ pratimāsaṃ padakṣayaḥ /
amuṣmād anyathābhāve mṛtyuśaṅkā prajāyate // 1.29 //
ekaikapadavṛddhiḥ syād āṣāḍhān makarāvadheḥ /
ekaikapādyahrāsaḥ syān makarāt karkaṭāvadheḥ // 1.30 //
āṣāḍhāt kramaśo vṛddhiḥ puṣyād dhrāsaḥ kramāt tathā /
chāyāvikāram ālocya māsān maraṇam ādiśet // 1.31 //
kṣayakāle bhaved vṛddhir vṛddhikāle kṣayo yadi /
tam eva kālam ārabhya mṛtyur māsād viniścitaḥ // 1.32 //
vṛddhikāle 'py ativṛddhau nānārogād upadravaḥ /
kṣayakāle 'py atikṣaye mṛtyur eva na saṃśayaḥ // 1.33 //
āṣāḍhapuṣyayor ante pūrvaṃ karkaṭamatsyayoḥ /
yāni ṣaṭ ṣaḍ dināny atra kālākālanirūpaṇam // 1.34 //
āṣāḍhāntadinair ebhiḥ śrāvaṇādau śubhāśubham /
puṣyaśeṣadinaiś cāpi māghādiṣu śubhāśubham // 1.35 //
pādayos tālukāṃ viddhvā nābhau vedho yadā bhavet /
ahorātratrayād ūrdhvaṃ pañcatvaṃ jāyate tadā // 1.36 //
pādayos tālukāṃ viddhvā yadi cakṣuś ca vidhyati /
tadā māsatrayād ūrdhvaṃ gamanaṃ yamasadmani // 1.37 //
pādayos tālukāṃ viddhvā nāsikāṃ yadi vidhyati /
tribhir dinais tadā nūnam ekākī yāti pañcatām // 1.38 //
kuṭiprasrāvayoḥ kāle tulyaṃ syād yadi hañchikā /
tasyām eva hi velāyāṃ mṛtyur varṣeṇa tasya hi // 1.39 //
nāsikāgaddrikāvedhe maraṇaṃ pañcavatsaraiḥ /
jihvāgrādarśane mṛtyur vāsaraiḥ syāt tribhir nṛṇām // 1.40 //
māsaiś caturbhir mṛtyuḥ syāt karṇāgre tīkṣṇavedhanāt /
sadyo mṛtyur bhaven nūnam ūrṇāsthāne pravedhanāt // 1.41 //
tulyakālaṃ yadā vedhaś candrasūryagato bhavet /
tadā māsād bhaven mṛtyuḥ puṇyam evārjayet tataḥ // 1.42 //
bhagaliṅgasamāyoge madhye śeṣe ca hañchikā /
syāc cet tulyaṃ tadā māsaiḥ pañcatvaṃ yāti pañcabhiḥ // 1.43 //
hastayoḥ pādayoś caiva kaniṣṭhānāṃ ca saṃdhayaḥ /
catvāro yasya vidhyanti tulyaṃ māsāt sa mṛtyubhāk // 1.44 //
hṛtkaṇṭhamadhyayor vedhas tulyakālaṃ yadā bhavet /
pakṣatrayeṇa mṛtyuḥ syād yadi dharmaṃ na sevate // 1.45 //
tulyakālaṃ yadā vidhyet tālukānāṃ trayaṃ muhuḥ /
tribhir dinais tadā mṛtyur yadi śakrasamo bhavet // 1.46 //
vāmākṣiputtalīcchāyāṃ yo na paśyati darpaṇe /
saptāhān mriyate nūnaṃ yadi na syāt pratikriyā // 1.47 //
maithune sati ghaṇṭāyā nādaś cet karṇayor bhavet /
tribhir māsais tadā mṛtyur yadi brahmasamo bhavet // 1.48 //
kṛṣṇaṃ yadi bhavec chukraṃ śuklāyāṃ pratipattithau /
ṣaḍbhir māsais tadā mṛtyur lohitaṃ vyādhisūcakam // 1.49 //
karṇamūle bhruvor madhye mastakāgre pṛthakpṛthak /
yadi vedho 'pratīkāro mṛtyuḥ sadyas tadā bhavet // 1.50 //
pādāṅguṣṭhaṃ samārabhya nābhiparyantago bhavet /
vedhaḥ sadāho yasyāsau ṣaḍmāsān mriyate dhruvam // 1.51 //
pādāṅguṣṭhād dhṛdayāntaṃ yāvat kaṇṭhaṃ śiro 'thavā /
māsatrayāt pakṣatrayān mṛtyur ekadināt kramāt // 1.52 //
nāsāgramāṃsaśaithilyān mṛtyuḥ syāt saptarātrataḥ /
kapolamāṃsavicchedān mṛtyuḥ syāt pañcamāsataḥ // 1.53 //
cakṣurnāsikayor madhye spandanaṃ cen na dṛśyate /
jñātvā pañcadinān mṛtyum ādadyād dharmasaṃvalam // 1.54 //
netrayos tārake mliṣṭe sahasā bhavato yadi /
ṣaḍmāsābhyantare mṛtyuḥ pratikārakriyā na cet // 1.55 //
akasmān nāsikā vakrā karṇau bhraṣṭau svadeśataḥ /
netre ca vartulībhūte saptāhān mṛtyudarśanāt // 1.56 //
akasmād dhṛdayaṃ nimnaṃ syāc cet pakṣāt tadā mṛtiḥ /
grīvāpārśvasthayor nāḍyor vicchedād api pakṣataḥ // 1.57 //
akasmāt kṛṣṇarekhā syāj jihvāyā yadi madhyataḥ /
dṛḍho vā dantasaṃdaṃśas tadā mṛtyur dvirātrataḥ // 1.58 //
karṇaśaṣkulikāpṛṣṭhanāḍīvicchedadarśanāt /
tasminn eva dine mṛtyur athavā pañcavāsaraiḥ // 1.59 //
nakheṣu raktavarṇeṣu sahasā yady araktatā /
ṣaḍbhir māsais tadā mṛtyur yadi mantrādi nācaret // 1.60 //
akasmāj jāyate sthūlaḥ kṛśaḥ kruddho bhayākulaḥ /
yas tasya mṛtyur varṣeṇa yadi dharmaṃ na sevate // 1.61 //
indriyāṇi na gṛhṇanti yathāsvaṃ yasya gocarān /
muhuḥ skhalati vāṇī ca muhuś ca mativibhramaḥ // 1.62 //
cakṣuṣī sravato nityaṃ dṛṣṭarūpe 'pi vibhramaḥ /
darpaṇe salile vāpi svacchāyāṃ yo na paśyati // 1.63 //
rātrāv indradhanuḥ paśyed divā nakṣatramaṇḍalam /
ameghe vidyutaṃ paśyet sphurantīṃ dakṣiṇāśritām // 1.64 //
divā chāyāpathaṃ paśyed ulkāyāḥ patanaṃ tathā /
haṃsakākamayūrāṇāṃ paśyed ekatra melakam // 1.65 //
candradvayaṃ dvisūryaṃ vā svaśirojvalanaṃ tathā /
gandharvanagaraṃ paśyed vṛkṣāgre śikhare gireḥ // 1.66 //
paśyet pretapiśācān adṛśyān anyāṃś ca bhīṣaṇān /
prakampate bhṛśaṃ caiva mūrchito vā bhaven muhuḥ // 1.67 //
chardiṃ mūtraṃ purīṣaṃ ca suvarṇarajataprabham /
paśyed ekaikaśas tasya mṛtyur māsāvadher bhavet // 1.68 //
kaṇṭḥoṣṭhatālujihvānāṃ chedaṃ chidraṃ tanau tathā /
yaḥ paśyet tasya mṛtyuḥ syāt ṣaḍmāsābhyantare dhruvam // 1.69 //
kaṇṭhoṣṭhatālurasanādantā yasya pṛthakpṛthak /
śuṣyanty abhīkṣṇaṃ ṣaḍmāsān maraṇaṃ tasya nirdiśet // 1.70 //
nakṣatracakrasaptarṣidiśāṃ rātrāv adarśanam /
saptāhamṛtyor jāyeta jvālā vākasmikī tanau // 1.71 //
arundhatīṃ rohiṇīṃ ca dhruvaṃ chāyāpathaṃ tathā /
rātrāv apaśyato 'kasmāt ṣaḍmāsān mṛtyum ādiśet // 1.72 //
kalaṅkarahitaṃ candraṃ sūryaṃ raśmivivarjitam /
saraśmikaṃ citrabhānuṃ candraṃ cāpi saraśmikam // 1.73 //
rātrau sūryaṃ divā candraṃ svanetre jvalanaṃ tathā /
tārāṃ merupramāṇāṃ ca samudraṃ ca nadīm iva // 1.74 //
nābhau hikkāṃ gude hañchāṃ varṇaṃ padmopamaṃ mukhe /
galake piṭakān raktān gātre varṇavicitratām // 1.75 //
hṛdaye krandanaṃ sāśru prakampam atha tāluni /
candrāṃśuṃ vahnisaṃsparśaṃ sūryāṃśuṃ himaśītalam // 1.76 //
candracchidraṃ ravicchidraṃ chidraṃ bhūmau tathāmbare /
ātmanaiva hi yaḥ paśyet paśyen mṛtyuṃ sa pakṣataḥ // 1.77 //
mūtraśukrapurīṣāṇi tulyakālaṃ patanti cet /
varṣān mṛtyur bhavet tatra bhaiṣajyādi pratikriyā // 1.78 //
ardhaṃ śītaṃ tathā coṣṇam ardhaṃ yasya kalevaram /
saptarātrād bhaven mṛtyuḥ paralokaṃ vicintayet // 1.79 //
hakāraḥ śītasaṃsparśaḥ phukāro vahnisaṃnibhaḥ /
yasyānubhūyate tasya daśāhād yamadarśanam // 1.80 //
anāmikānāṃ mūleṣu dṛṣṭayā kṛṣṇarekhayā /
aṣṭādaśadinād ūrdhvaṃ mṛtyuḥ prākkarmadoṣataḥ // 1.81 //
yasya mastakam āruhya trivarṇo yāti vegataḥ /
kṛkalāso mṛtis tasya pañcamāsād viniścitā // 1.82 //
śabdo na śrūyate dehād dhastābhyām apamārjitāt /
sarvāṅgaśītalatvena daśāhān maraṇaṃ dhruvam // 1.83 //
yugapan nirgamo bāhujaṅghayor yasya gacchataḥ /
vināśo brahmasūtrasya mṛtyus tasyaikavarṣataḥ // 1.84 //
kardame pāṃśudeśe vā purataḥ pṛṣṭhato 'pi vā /
khaṇḍaṃ pādodaye nyūnaṃ mṛtyur māsacatuṣṭayāt // 1.85 //
akasmāt puruṣaṃ paśyet kṛṣṇapiṅgalavarṇakam /
varṣatrayātyayān mṛtyur yadi bhrāntyā na kalpitaḥ // 1.86 //
dvimāsamṛtyur hṛtpādau snātamātrasya śuṣyataḥ /
trirātramṛtyur durgandho jāyate vikṛtākṛtiḥ // 1.87 //
kharārkadivase toyair āpūrya mukhakoṭaram /
pratyahaṃ phūtkṛtaṃ kurvan kramāt tanmadhyasaṃsthitam // 1.88 //
paśyed indradhanuś citraṃ dīrghāyuḥ puruṣaḥ sadā /
ṣaḍmāsāyur na tat paśyed iti karmavicitratā // 1.89 //
kharārkadivase paśyet puro gaganamadhyataḥ /
muktāhārān ivālūnān muktāphalavibhūṣaṇān // 1.90 //
indranīlamaṇicchāyān keśoṇḍukasamākṛtīn /
dṛśyadeśaṃ parityajya deśāntaravisarpiṇaḥ // 1.91 //
nāgān anekaśaḥ paśyet saṃyatān iva saṃtatam /
teṣām adṛṣṭau ṣaḍmāsān maraṇaṃ parikīrtitam // 1.92 //
prabhāte vātha sāyāhne jyotsnāyāṃ vā ciraṃ puraḥ /
vitatya bāhū svacchāyāṃ dṛṣṭvā paśyec chanair nabhaḥ // 1.93 //
tatrāpi dṛśyate chāyā dhavalā nararūpiṇī /
śiraso 'darśanāt tasya mṛtyuḥ syād varṣamadhyataḥ // 1.94 //
putrabhāryāvināśaḥ syād vāmapāṇer adarśanāt /
dakṣiṇādarśanāt pitṛbhrātrādīnāṃ mahīyasām // 1.95 //
jānūpari sthitau bāhū kṛtvā mūrdhni tathāñjalim /
rambhāphalanibhāṃ chāyāṃ lakṣayen madhyatas tayoḥ // 1.96 //
yadā tv ekaṃ dalaṃ tatra vikāsi parilakṣyate /
tasyām eva tithau nūnaṃ ṣaḍmāsān maraṇaṃ bhavet // 1.97 //
sūryamārgagate vāyau lakṣaṇīyam idaṃ trayam /
śrotre śirasi netre ca śabdo dhūmo dyutis tathā // 1.98 //
karṇau pidhāya sudṛḍhaṃ gambhīro gulgulo dhvaniḥ /
na śrūyate pañcadinaṃ daśa pañcadaśāthavā // 1.99 //
viṃśatiṃ divasān yāvat pañcaviṃśatim eva vā /
bāṇābdhiguṇanetrenduvarṣair mṛtyur yathākramam // 1.100 //
eṣām apy antarāleṣu śodhyāḥ ṣoḍaśa vāsarāḥ /
varṣasyaiva caturthāṃśaiḥ kramaśaś cābhivardhitaiḥ // 1.101 //
mastakopari niryāntīṃ saptāhaṃ dhūmamālikām /
yo na paśyati tasya syān mṛtyur varṣatrayāvadheḥ // 1.102 //
svanetrayoś caturdikṣu dṛśyante 'ṅgulipīḍanāt /
mārjāranetratulyāni jyotīṃṣi jhaṭitikramāt // 1.103 //
apīḍane 'pi taddṛṣṭau mṛtyuḥ syāc chatavāsaraiḥ /
etat sāmānyato jñeyaṃ viśeṣas tv ayam ucyate // 1.104 //
candrasyādhas tathordhvaṃ ca karṇanāsopakaṇṭhayoḥ /
naṣṭe jyotiṣi mṛtyuḥ syāt ṣaṭtrinetrendumāsataḥ // 1.105 //
sūrye 'pi yadi candrasya kramaḥ syāt pūrvavat tadā /
digbāṇaguṇanetreṣu dineṣu maraṇaṃ bhavet // 1.106 //
svarūpasya viparyāsāt ṣaḍmāsān maraṇaṃ bhavet /
pañcamāsād bhaven mṛtyur nāsāgrasya vināśataḥ // 1.107 //
bādhirye śrotrayor mṛtyuś caturmāsātyayād bhavet /
svasthendriyasya patane mṛtyur māsatrayād bhavet // 1.108 //
bhrūmadhye jyotiṣo 'dṛṣṭau dvimāsād yamadarśanam /
ekamāsātyayān mṛtyur vṛṣaṇe golakakṣayāt // 1.109 //
parākṣṇoḥ pratibimbasyādṛṣṭau pakṣād vināśanam /
jihvāyā aparāvṛttau daśāhān mṛtyusaṃgamaḥ // 1.110 //
nābher viparyayān mṛtyuḥ pañcāhād upaśliṣyati /
pārśvadvayasyākṛṣṭau tu mṛtyuḥ syād vāsaratrayāt // 1.111 //
samastagātrastabdhatve mṛtyur ekāhiko bhavet /
lalāṭasthatrirekhāṇāṃ nāśān mṛtyur ahastrayāt // 1.112 //
svasthasya lālājihvā cen niḥsparśā dravavarjitā /
yāvat pañcadinaṃ mṛtyuḥ pañcāśaddivasais tathā // 1.113 //
amlāditvaṃ ca rasato nīlāditvaṃ ca varṇataḥ /
vikāraḥ śukramūtrāṇāṃ ṣaḍmāsān mṛtyudarśanāt // 1.114 //
yugapatpañcadhāraṃ vā vāmāvartaṃ vigandhi ca /
mūtraṃ yasya bhaven mṛtyus tasya ṣaḍmāsamadhyataḥ // 1.115 //
kṛṣṇe vā yadi vā śukle pakṣe reto 'tikṛṣṇakam /
mṛtyuḥ ṣaḍmāsato no cet tenaiva syāt pratikriyā // 1.116 //
kṛṣṇabhāgaṃ parityajya gṛhītvā śuklabhāgakam /
candre prabhañjane vāti praveśāt syāt pratikriyā // 1.117 //
sthiratve 'pi svadehasya tacchāyā cañcalā yadi /
caturmāsād bhaven mṛtyur ity āgamavicakṣaṇāḥ // 1.118 //
dakṣiṇāśāgatāṃ chāyām ātmano yadi paśyati /
adyaiva mṛtyur asmākam iti paśyed anityatām // 1.119 //
nirarthako visaṃvādī mithyetyādi prakāśanāt /
svapno 'py asatyaḥ sarveṣāṃ nātaḥ śraddhā na yujyate // 1.120 //
devatādarśitaḥ svapnaḥ satyasvapno 'tha yo naraḥ /
sa svapne pratyayaṃ kṛtvā mṛtyuliṅgāni paśyatu // 1.121 //
kiṃśukaṃ kovidāraṃ ca puṣpitaṃ karavīrakam /
yo 'bhirohati svapnānte sa syāt ṣaḍmāsamṛtyukaḥ // 1.122 //
vihārayaṣṭiṃ svapnānte yo 'bhirohati mānavaḥ /
vālukābhasmarāśiṃ vā maraṇaṃ tasya pūrvavat // 1.123 //
gardabhaṃ yaḥ samārūḍhaḥ prayāyād dakṣiṇāṃ diśam /
bhūyaś ca na nivarteta tadvat tasyāpi jīvitam // 1.124 //

tatrasthaś ca vibudhyeta mṛtyuḥ syāt tasya pūrvavat // 1.125 //
daṇḍaṃ vā caityayūpaṃ vā naro yas tv abhirohati /
vālmīkaṃ pāṃśurāśiṃ vā ṣaḍmāsān na sa jīvati // 1.126 //
kālī kumārī saṃkruddhā yā tu badhnāti bandhanaiḥ /
kālarātrī tu sā jñeyā saptāhān mṛtyukārikā // 1.127 //
kṛṣṇavastrā tu yā nārī kālī kāmayate naraṃ /
karavīramālā svapne yāty asau yamamaṇḍalam // 1.128 //
tamo vā praviśet svapne śvabhraṃ vā cārakaṃ tathā /
vṛkṣād vā prapatet svapne sa mṛtyor antike sthitaḥ // 1.129 //
vṛkṣaṃ tṛṇaṃ vā kāṣṭhaṃ vā viphalaṃ yas tu paśyati /
suptaḥ śīrṣe śarīre vā ṣaḍmāsān na sa jīvati // 1.130 //
yo vānararathārūḍho gacchet pūrvāṃ diśaṃ naraḥ /
pratibuddho vijānīyād rātrir eṣā mamāntimā // 1.131 //

ekābdavigame nūnaṃ maraṇaṃ tasya nirdiśet /
svapnadṛṣṭanimittānāṃ śāntir māyopamā matiḥ // 1.133 //
lohadaṇḍadharaṃ kṛṣṇaṃ naraṃ kṛṣṇaparicchadam /
prasuptenāgrato dṛṣṭvā trimāsān mṛtyur āpyate // 1.134 //
raktagandhapraliptāṅgaṃ raktamālyavibhūṣitam /
tailābhyaktam atibhītaṃ muṇḍitaṃ raktavāsasam // 1.135 //
kharam āruhya vegena vrajantaṃ dakṣiṇāṃ diśam /
ātmānam īdṛśaṃ svapne paśyet ṣaḍmāsamṛtyukaḥ // 1.136 //
raktamālyāni gandhāṃś ca raktāmbaram athāpi vā /
yaḥ svapne labhate tasya mṛtyuḥ syād aṣṭamāsataḥ // 1.137 //
saṃkhyātītāni bāhyāni mṛtyuliṅgāni tāni kaḥ /

|| iti śrīpaṇḍitavāgīśvarakīrtiviracite mṛtyuvañcanopadeśe prathamaḥ paricchedaḥ ||

tac ca vāyūdayāj jñeyaṃ vāyur ghrāṇāsyagocaraḥ // 2.1 //
bāhyalakṣaṇamātrān na mṛtyoḥ samyag viniścayaḥ /
yāvad ādhyātmikaṃ samyag lakṣaṇaṃ naiva niścitam // 2.2 //
ubhayor niścayān mṛtyur niścitaḥ syāt parisphuṭam /
bāhye buddhiviparyāso rogāder api saṃskṛteḥ // 2.3 //
puṣyādimāsasaṃkrāntau pūrṇamāsyādiparvasu /
sāmānyalakṣaṇaṃ mṛtyor lakṣayet prātarāditaḥ // 2.4 //
nirūpakanaro yatra dine kālaṃ nirūpayet /
tad ārabhya dinaṃ mṛtyur mitakālātyayād bhavet // 2.5 //
samaprakṛtiyuktānāṃ sarvaprāṇabhṛtāṃ sadā /
nāsikādakṣiṇetare puṭe pratyekam āśritaḥ // 2.6 //
vāyur vahati yāmārdhaṃ sūryaśītāṃśusaṃjñake /
ato viparyayāt kalpyaṃ maraṇādi manīṣibhiḥ // 2.7 //
pratipadaṃ samārabhya śuklāṃ vahati candragaḥ /
dināni trīṇi pavanaḥ tataḥ sūrye dinatrayam // 2.8 //
dinatrayaṃ punaś candre tataḥ sūrye dinatrayam /
krameṇānena niyamād yāvat pañcadaśī sitā // 2.9 //
evaṃ kṛṣṇaṃ samārabhya sūrye vahati mārutaḥ /
dināni trīṇi pūrvavad yāvat pañcadaśītarā // 2.10 //
kramasyāsya viparyāsāt tribhiḥ pakṣaiḥ suniścitāt /
ṣaḍbhir māsair bhaven mṛtyur dharmam eva smaret tataḥ // 2.11 //
pakṣadvayaviparyāsāt suhṛdbandhuvipad bhavet /
ekapakṣaviparyāsād dāruṇavyādhisaṃbhavaḥ // 2.12 //
ekadvitricatuḥpañcaṣaḍdināni viparyayāt /
vahed vāyur yadi tadā kalyudvegādi jāyate // 2.13 //
trimārgago bhaved vāyuḥ prakṛteḥ parivarjanāt /
madhyāhnāt parato mṛtyuḥ pratīkāravivarjitaḥ // 2.14 //
saṃkrāntirahito vāyur ubhayor api mārgayoḥ /
vahed yasya daśāhāni saṃkrāntau mriyate hi saḥ // 2.15 //
viṣuvatkṣaṇasaṃprāptau spandete yasya locane /
tasya nāśaṃ vijānīyād ahorātrān na saṃśayaḥ // 2.16 //
itaś cetaś ca bahudhā yasyārdhapraharaṃ vahet /
vāyus tasya vijānīyāt pūjanaṃ lābham eva ca // 2.17 //
saṃkrāntipañcakaṃ yasya samatītya mukhe vahet /
vāyus tasyārthanāśaḥ syād udvego vā bhayādi vā // 2.18 //
vāmanāsāpuṭe vāyuḥ saṃkrāntīś ca trayodaśa /
samatītya vahed yasya tasya rogādi jāyate // 2.19 //
puṭadvayaṃ parityajya yadā vaktreṇa gacchati /
tad ahar jīvitaṃ tasya yadi vajrasamo 'py asau // 2.20 //
mārgaśīrṣasya saṃkrāntikālam ārabhya mārutaḥ /
pañcāhaṃ ced vahet sūrye mṛtyur aṣṭādaśābdataḥ // 2.21 //
aśvayuṅmāsasaṃkrāntikālam ārabhya mārutaḥ /
pañcāhaṃ ced vahen mṛtyur bhavet pañcadaśābdataḥ // 2.22 //
śrāvaṇasyāpi ced evaṃ mṛtyur dvādaśavarṣataḥ /
jyeṣṭhe 'pi ced bhaved evaṃ mṛtyuḥ syān navavarṣataḥ // 2.23 //
caitramāsasya saṃkrāntikālam ārabhya ced vahet /
pañcāhaṃ māruto mṛtyuḥ ṣaḍbhir varṣais tadā bhavet // 2.24 //
māghamāsasya saṃkrānteḥ pañcāhaṃ ced vahen marut /
varṣatrayātyayān mṛtyur iti kālavicakṣaṇāḥ // 2.25 //
sarvatra dvitricaturo vāyuś ced divasān vahet /
abdabhāgais tu te śodhyā yathāvad anupūrvaśaḥ // 2.26 //
evam eva vahec candre vāyuś cen mṛtyuvarjitāḥ /
vyādhidurbhikṣakāntāraśokarājādyupadravāḥ // 2.27 //
sāmānyakālasaṃbodhaḥ sphuṭo 'yam upadarśitaḥ /
viśeṣakālabodhāya gatir anyā pradarśyate // 2.28 //
samasaptagate sūrye janmarkṣe candramā yadi /
kālo 'sau pauṣṇanāmeti mṛtyunirṇayakārakaḥ // 2.29 //
yatra rāśau naro jātas tasmād yaḥ saptamo 'paraḥ /
samasapta iti khyātas tatrārkaḥ samasaptagaḥ // 2.30 //
rāśau tatraiva candraś cej janmanakṣatrago bhavet /
pauṣṇaḥ kālaḥ sa vijñeyas tatra mṛtyor nirūpaṇam // 2.31 //
sarvatra sūryamārgāntargate satatagāmini /
kālaṃ nirūpayed vidvān itaratra vikālakam // 2.32 //
yatra velākṣaṇe vāyor gatir anyā pravartate /
tatra velākṣaṇe pūrṇe mṛtyur nāsty atra saṃśayaḥ // 2.33 //
pauṣṇe kāle vahed vāyur dinārdhaṃ yadi sūryagaḥ /
vatsarair manusaṃkhyātair lakṣayen maraṇaṃ tadā // 2.34 //
dinam ekam ahorātraṃ dinadvitricatuṣṭayam /
vahec ced arkadikśailaṣaḍvedābdaiḥ kramān mṛtiḥ // 2.35 //
pañcāhaṃ ca daśāhaṃ ca tathā pañcadaśāhakam /
vahec cet tridvyekavarṣair bhaven mṛtyur yathākramam // 2.36 //
tathaiva ced vahed vāyur viṃśatiṃ pañcaviṃśatim /
ṣaḍbhir māsais tribhiś caiva yathāsaṃkhyaṃ bhaven mṛtiḥ // 2.37 //
ṣaṭsaptāṣṭāviṃśatiṃ ced ūnatriṃśaddinaṃ vahet /
syād dvyekamāsaiḥ pakṣeṇa digdinaiś ca kramān mṛtiḥ // 2.38 //
triṃśat tu caikatriṃśac ca dvātriṃśad divasān yadi /
pañcatridvidinair mṛtyur yathāsaṃkhyaṃ bhavet tadā // 2.39 //
trayastriṃśaddinaṃ yāvat sūrye ced vāti mārutaḥ /
tasminn eva dine mṛtyur viṣṇor api na saṃśayaḥ // 2.40 //
pañcāhapañcaviṃśatyor antarāleṣu vāsarāḥ /
noktāḥ ṣoḍaśasaṃkhyā ye teṣāṃ śodhanam ucyate // 2.41 //
ṣaḍdinaṃ cet tribhir varṣaiś caturviṃśativarjitaiḥ /
saptāhaṃ cet tribhir varṣaiḥ pūrvoktadviguṇonitaiḥ // 2.42 //

navāhaṃ cet tribhir varṣaiḥ ṣaṇṇavatidinonitaiḥ // 2.43 //
vahed ekādaśāhaṃ cen mārutaḥ sūryagocaraḥ /
caturviṃśatirātronān mṛtyur varṣadvayād bhavet // 2.44 //
dvādaśāhaṃ varṣadvayāt pūrvoktadviguṇonitāt /
trayodaśāhaṃ dvivarṣād dvāsaptatidinonitāt // 2.45 //
caturdaśāhaṃ dvivarṣāt ṣaṇṇavatidinonitāt /
ṣoḍaśadinavāhena dvādaśonaikavarṣataḥ // 2.46 //
vahet saptadaśāhaṃ ced vāyuḥ sūryaikagocaraḥ /
caturviṃśatyahanyūnād varṣān mṛtyur udīritaḥ // 2.47 //
aṣṭādaśāhaṃ ṣaṭtriṃśaddinanyūnaikavarṣataḥ /
ūnaviṃśati ced aṣṭacatvāriṃśaddinonitāt // 2.48 //
ekaviṃśati ṣaḍmāsāt ṣaḍvāsaraviyojitāt /

vāyūdayavaśāj jñeyo nāta ūrdhvam asaṃbhavāt // 2.51 //
āha cātra |
ādau kṛtvā dinārdhaṃ sakaladinam athā-harniśaṃ yāvad eva tasmād ahnor dvayaṃ ca tridinam atha catur-vāsarān vyāpya yāvat /
prāṇo nāḍyāśrito yo vahati dinapater udgame savyahīne tasmin vijñeyam etad bhuvanaravidiśo maṅgalaṃ ṣaṭ catuṣkam // 2.52 //
pañcabhyaḥ pañcaviṃśaddivasagatir ihā-rohate pañcavṛddhyā tasmād ekottareṇa triguṇitadaśakaṃ tryuttaraṃ yāvad eva /

bāhyaś cādhyātmikaś cāpi jñeyo vañcanasaṃcayaḥ // 3.1 //
kāyavākkarmabhir bāhyam āntaraṃ dhyānakarmabhiḥ /
maṇimantrauṣadhair bāhyam āntaraṃ yogaśaktibhiḥ // 3.2 //
śraddhayā śakyate kartuṃ mṛtyuvañcanam udyataiḥ /
śraddhām ato dṛḍhīkuryād anyathā syāc chramaḥ paraḥ // 3.3 //
kāyakleśair bahuvidhair na cāpy arthasya rāśibhiḥ /
dharmaḥ saṃprāpyate sūkṣmaḥ śraddhāhīnaiḥ surair api // 3.4 //
śraddhā dharmaḥ paraṃ sūkṣmaḥ śraddhā jñānaṃ tapo hutam /
śraddhā svargaś ca mokṣaś ca śraddhā sarvam idaṃ jagat // 3.5 //
sarvasvaṃ jīvitaṃ cāpi dadyād aśraddhayā yadi /
naivāpnuyāt phalaṃ kiṃ cic chraddadhānas tato bhavet // 3.6 //
ādau dharmaparo bhūyān mṛtyuvañcanavāñchayā /
dharmo hi trāyate mṛtyor mṛtyur dharmeṇa vañcyate // 3.7 //
na dharmarahitas tiṣṭhet pramādāt kṣaṇamātrakam /
brahmāpi hi vinā dharmaṃ na prabhur mṛtyuvañcane // 3.8 //
viduṣāṃ saṃmato dharmaḥ satyaṃ dānaṃ tapaḥ kṣamā /
viśeṣeṇa kṛpā yeṣāṃ kiṃ teṣāṃ dharmavistaraiḥ // 3.9 //
kāruṇyasyandi hṛdayaṃ yeṣāṃ sattveṣu saṃtatam /
teṣāṃ jñānaṃ ca mokṣaś ca kim anyair dharmavistaraiḥ // 3.10 //
ahiṃsā paramo dharmaḥ sarvasiddhāntasaṃmataḥ /
tām evābhyasyato nityaṃ mṛtyur dūrāyate kramāt // 3.11 //
alpāyur api dīrghāyuḥ sadyaḥ syāj jīvayann imān /
pakṣimatsyamṛgavyāḍacaurādīn vadhyadeśagān // 3.12 //
dīrghāyur api mandāyuḥ sadyaḥ syān mārayann imān /
cauramatsyamṛgavyāḍapakṣiṇo jīvakāṅkṣiṇaḥ // 3.13 //
daśākuśalasaṃtyāgāt triratnaśaraṇāt tathā /

mātāpitṛgurujyeṣṭhajñānavṛddhādigauravāt /

mahāṣṭasthānapūjābhir āyur nityaṃ vivardhate // 3.20 //
chandakabhikṣayopāttair dravyaiḥ saṃghasya bhojanāt /

caityādivandanāc caiva dhruvam āyur vivardhate // 3.22 //

sakṣudrakaṃ mahāmeghaṃ gāthāḥ susvastisaṃjñitāḥ // 3.23 //

gaṇḍavyūhādisūtrāntān mahāyānaprakāśakān /
ekāgro vācayen nityaṃ mṛtyuvañcanam aśnute // 3.25 //

dhāraṇīsaṃcayādyuktadhāraṇījāpatatparāḥ /

evaṃ dharmaparo bhūtvā siddhamantrauṣadhādibhiḥ /

ratnalakṣaṇaśāstroktaparīkṣottīrṇavajradhṛk /
āyurmaṇidharo vāpi viśvarūpadharas tathā // 3.36 //
anarghasyendranīlāder anyasyāpi mahāmaṇeḥ /
dhārakaḥ sarvamṛtyūnāṃ vañcakaḥ syān na saṃśayaḥ // 3.37 //

pratyakṣasiddhaṃ sarveṣāṃ yathā nāsty atra saṃśayaḥ /
tathopadeśāt kalpādijīvane 'pi na saṃśayaḥ // 4.88 //
nāśonmukho 'pi dīpādis tailavartikṣayādibhiḥ /
punas tailādisaṃprāpteḥ kalpāntam api tiṣṭhati // 4.89 //
yathā tathāyaṃ deho 'pi mriyamāṇo jarādibhiḥ /
siddhopadeśasaṃprāpteḥ kalpāntaṃ kiṃ na tiṣṭhati // 4.90 //
yathā jīrṇasya gehasya saṃskāreṇa cirasthitiḥ /
tathā jīrṇasya dehasya saṃskāreṇa na kiṃ bhavet // 4.91 //
na duṣkaraṃ kiṃ cana vikrameṇa tasmād anikṣiptadhurā yatadhvam /
nityapravṛttā vibhinatti kāle śilātalaṃ mṛdv api vāridhārā // 4.92 //
kāṣṭhād agnir jāyate mathyamānād bhūmis toyaṃ khanyamānā dadāti /
sotsāhānāṃ nāsty asādhyaṃ narāṇāṃ nyāyārabdhāḥ sarvayatnāḥ phalanti // 4.93 //
subahūny api vijñāya vañcanāni mṛter iha /
ananuṣṭhānād viṣṭhābhir liptvā śayyāṃ mṛtā vayam // 4.94 //
prayogakuśalā ye ca paṇḍitāḥ śāstrapāragāḥ /
ananuṣṭhānadoṣeṇa dantān niṣkāśya te mṛtāḥ // 4.95 //
jñātvāpi yogaśāstrāṇi prayogāṃś cāpy anekaśaḥ /
adya śvo vā kariṣyāma iti cintāparā mṛtāḥ // 4.96 //
idaṃ kartāsmi śvaḥ punar idam idaṃ cāsya parato vṛthā prāptāśvāso vimṛśati na lolāṃ bhavagatim /
agatvā kāryāntaṃ viṣaya-salilāvartapatitā prajā mṛtyor vaktraṃ makara-mukhavan nauḥ praviśati // 4.97 //
iti nigaditam etad vañcanaṃ mṛtyuśatroḥ svaparasamayasiddhadhyānakarmaprasādhyam /
aghṛṇamaraṇabhīter yatnataḥ saṃvidheyaṃ yata iha kamanīyaṃ jīvitān nānyad asti // 4.98 //
vajraḍākaṃ catuṣpīṭhaṃ buddhapañjarakādi ca /
kālottaraṃ kālāvalīṃ kālapañcāśikām api // 4.99 //
jīvapañcāśikādīni śāstrāṇy ālokya yatnataḥ /
sattvānukampayā proktam idaṃ maraṇavañcanam // 4.100 //
śāradacandranabhaḥ pariśuddhaṃ śuddhaviyacchaśināpy asamāptam /
āptam ataḥ kuśalaṃ yad amṛtyus tena jano 'stu sa mṛtyuvimuktaḥ // 4.101 //
gaṇanāśramahānāya lekhakānāṃ tapasvinām /
granthapramāṇam ākhyātaṃ sāśītikaśatatrayam // 4.102 //
|| iti śrīpaṇḍitācāryavāgīśvarakīrtiviracite mṛtyuvañcanopadeśe caturthaḥ paricchedaḥ samāptaḥ ||
