#Data entry:Peter Gäng
#Edition:Yoginī Lakṣmīṅkarā: »Advayasiddhi.« In: Samdhong Rinpoche and Vrajvallabh Dwivedi (eds.): Guhyādi-aṣṭasiddhisaṅgraha. Sarnath, Varanasi 1987 (Rare Buddhist texts series 1), pp. 161-164.

advayasiddhi yoginī lakṣmīṅkarā viracitā

oṃ namaḥ śrīvajrasattvāya
prakṛtiprabhāsvaraṃ nāthaṃ sarvajñaṃ tribhavodbhavam /
praṇamya śirasā vajram īpsitārthaphalapradam // 1 //
deśakālatithivāranakṣatrair maṇḍalair vinā /
vakṣye haṃ vajrasattvasya saṃkṣepāt sādhanaṃ param // 2 //
niyamavratopavāsair akṣaroccāraṇabhāvanaiḥ /
atattvayogo na siddhyet kalpakoṭiśatair api // 3 //
viḍvajrodakabījādyair nāsikābhyantarodbhavaiḥ /
pūjayed satataṃ mantrī ātmānaṃ tattvabhāvanaiḥ // 4 //
jananīṃ bhaginīś caiva duhitṛbhāgineyikān /
prajñnopāyavidhānena pūjayed yogavit sadā // 5 //
ekāṅgavikalāṃ hīnāṃ śilpanīṃ śvapacīṃ tathā /
yoṣitāṃ pūjayen nityaṃ jñānavajraprabhāvanaiḥ // 6 //
tatremāni bījapadāni – oṃ āḥ hūṃ
yena yena hi baddhyante jantavo raudrakarmaṇā /
sopāyena tu tenaiva mucyante bhavabandhanāt // 7 //
samayān aharahaḥ kuryād divyān pañcakulodbhavān /
pūjayec ca pradīpādyaiḥ sakṣīrair viśvasambhavaiḥ // 8 //
protphullanayano mantrī nityaṃ prahasitānanaḥ /
cittam āropya saṃbodhau bhāvayej jñānasāgaram // 9 //
yāvat sthiracalā bhāvā santy atra tribhavālaye /
sarve te tattvayogena draṣṭavyā vajradhṛg yathā // 10 //
paravādinaś ca kecil lingabhedair vyavastitāḥ /
te 'py atra nāvamantavyā vajrasattvavikurvitaiḥ // 11 //
sarvān samarasīkrtya bhāvān nairātmyaniḥsṛtān /
bhāvayet satataṃ mantrī dehaṃ prakṛtinirmalam // 12 //
gandhamālyādibhir vastrair dhūpanaivedyakais tathā /
gītavādyais tathā nṛtyaiḥ sopāyair bhajate vibhuḥ // 13 //
na kaṣṭakalpanāṃ kuryān nopavāsaṃ na ca kriyām /
snānaṃ śaucaṃ na caivātra grāmadharmavivarjanaṃ // 14 //
na cāpi vandayed devān kāṣṭhapāṣāṇamṛṇmayān /
pūjām asyaiva kāyasya kurvān nityaṃ smāhitaḥ // 15 //
makṣikācchardisaṃmiśrair viṇmūtrādyaiś ca bhāvitaiḥ /
pañcapradīpasaṃyuktaiḥ pūjayed vajradhāriṇam // 16 //
abalā svayambhukusumaiḥ sakṣīrair viśvasambhavaiḥ /
pūjayed devatāṃ tena dehasthāṃ tattvabhāvanaiḥ // 17 //
parasvaharaṇaṃ kuryāt paradāraniṣevanam /
vaktavyaṃ ca mṛṣāvākyaṃ sarvabuddhāṃś ca ghātayet // 18 //
śailamṛṇmayacaityādīn na kuryāt pustake ratim /
na maṇḍalāni svapne 'pi kāyavākcittakarmaṇā // 19 //
jugupsāṃ naiva kurvīta sarvavastuṣu mantravit /
vajrasattvaḥ svayaṃ tatra sākṣād rūpeṇa saṃsthitaḥ // 20 //
gamyāgamyavikalpaṃ tu bhakṣyābhakṣyaṃ tathaiva ca /
peyāpeyaṃ tathā mantrī kuryān naiva samāhitaḥ // 21 //
vairocanasamudbhūtān sarvaprāṇyaṅgasambhavān /
prāṇakān guhyatattvajño bhakṣayet siddhihetunā // 22 //
sarvavarṇasamudbhūtā jugupsyā naiva yoṣitaḥ /
saiva bhagavatī prajñā saṃvṛtyā rūpam āśrityā // 23 //
na tithir na ca nakṣatraṃ nopavāso vidhīyate /
advayajñānayuktasya siddhir bhavati saugatī // 24 //
bahunātra kim uktena yad bhaved upalabdhikam /
tat sarvaṃ tattvayogena draṣṭavyaṃ tattvavedinā // 25 //
hastyaśvakharagāvoṣṭrapradīpaṃ śvānasambhavam /
mahāpradīpasaṃmiśraṃ bhakṣayed yogavit sadā // 26 //
na cādhyāsaktiṃ kurvīta ekasminn api yogavit /
samatācittayogena bhāvanīyo bhavārṇavaḥ // 27 //
utpattisthitinirodhaṃ cāsampṛktaṃ pṛthagjanaiḥ /
tasya bhāve tu saṃsāro nānyatra pralayodbhavaḥ // 28 //
dinaṃ tu bhagavān vajrī naktaṃ prajñā vidhīyate /
evaṃ tu bhāvayed yogī laghu siddhim avāpnuyāt // 29 //
yat tad avyaktarūpaṃ tu sarvasattveṣu saṃsthitam /
guruvaktrāt paraṃ tattvaṃ prāpyate nātra samśayaḥ // 30 //
apratiṣṭhitanirvāṇaṃ nirnimittaṃ nirālayam /
vyāpakaṃ sarvasattveṣu saṃbodhiḥ paramaṃ padam // 31 //
evaṃ matvā tu vai yogī yo 'bhyased buddhimān sadā /
sa siddhyati na sandeho mandapuṇyo 'pi mānavaḥ // 32 //
ācāryāt parataraṃ nāsti trailokye sacarācare /
yasya prasādāt prāpyante siddhayo 'nekadhā budhaiḥ // 33 //
vajrasattvaḥ sa vai jñeyaḥ sarvabuddhair namaskṛtaḥ /
ācāryaḥ paramo devaḥ pūjanīyaḥ prayatnataḥ // 34 //
sa eva tathatārūpī lokānugrahahetunā /
rūpam āśritya saṃvṛtyā saṃsthito yogapīṭhake // 35 //
mṛtyur eṣa vikalpo yaṃ na bhāvaḥ sarvavastuṣu /
hanyate svavikalpena pṛthagjanavijṛmbhitaiḥ // 36 //
śrīmadoḍīyānavinirgatā mahāyogapīṭhāgatā 'khilayogatantratattvagarbhā śrīlakṣmīmukhakamalād viniḥsṛtā svādhiṣṭhānakramodayā advayasiddhir nāma sādhanopāyikā samāpteti
