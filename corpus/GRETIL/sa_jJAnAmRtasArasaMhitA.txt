#Text:Jñānāmṛtasārasaṃhitā
#Alternative:Nāradapañcarātra


||śrīnāradapañcarātram

||prathamaikarātre prathamo 'dhyāyaḥ

||oṃ namo bhagavate vāsudevāya

||atha maṅgalācaraṇam

nārāyaṇaṃ namas kṛtya naraṃ caiva narottamam /
devīṃ sarasvatīṃ caiva tato jayam udīrayet //

gaṇeśaśeṣabrahmeśadineśapramukhāḥ surāḥ /
kumārādyāś ca munayaḥ siddhāś ca kapilādayaḥ // 1,1.1 //

lakṣmī sarasvatī durgā sāvitrī rādhikā parā /
bhaktyā namanti yaṃ śaśvat taṃ namāmi parāt paraṃ // 1,1.2 //

dhyāyante satataṃ santo yogino vaiṣṇavās tathā /
jyotirabhyantare rūpam atulaṃ śyāmasundaram // 1,1.3 //

dhyāyet taṃ paramaṃ brahma paramātmānam īśvaram /
nirīham atinirliptam nirguṇaṃ prakṛteḥ paraṃ // 1,1.4 //

sarveśaṃ sarvarūpaṃ ca sarvakāraṇakāraṇam /
satyaṃ nityaṃ ca puruṣaṃ purāṇaṃ param avyayam // 1,1.5 //

maṅgalyaṃ maṅgalārhaṃ ca maṅgalaṃ maṅgalālayam /
svecchāmayaṃ paraṃ dhāma bhagavantaṃ sanātanam // 1,1.6 //

stuvanti vedā yaṃ śaśvann ānantaṃ jānanti yasya te /
taṃ staumi paramānandaṃ sānandaṃ nandanandanam // 1,1.7 //

bhaktapriyaṃ ca bhakteśaṃ bhaktānugrahavigrahaṃ /
śrīdaṃ śrīśaṃ śrīnivāsaṃ śrīkṛṣṇaṃ rādhikeśvaram // 1,1.8 //

jñānāmṛtaṃ jñānasindhoḥ saṃprāpya śaṅkarād guroḥ /
parāvarāc ca paramād yogīndrāṇāṃ guror guroḥ // 1,1.9 //

vedebhyo dadhisindhubhyaś caturbhyaḥ sumanoharam /
taj jñānamanthadaṇḍena saṃnirmathya navaṃ navam // 1,1.10 //

navanītaṃ samuddhṛtya natvā śambhoḥ padāmbujam /
vidhiputro nārado 'haṃ pañcarātraṃ samārabhe // 1,1.11 //

oṃ nārāyaṇāśrame puṇye puṇyakṣetre ca bhārata /
siddhe nārāyaṇakṣetre vaṭamūle supuṇyade // 1,1.12 //

kṛṣṇāṃśaṃ kṛṣṇabhaktaṃ ca palaṃ kṛṣṇaparāyaṇam /
śrīkṛṣṇacaraṇāmbhojadhyānaikatānamānasam // 1,1.13 //

japantaṃ paramaṃ brahma kṛṣṇa ity akṣaradvayam /
sukhāsane sukhāsīnaṃ kṛṣṇadvaipāyanaṃ munim // 1,1.14 //

papraccha śukadevaś ca sarvajñaṃ pitaraṃ muniḥ /
kāraṇaṇ ca purāṇānāṃ purāṇaṃ param avyayam // 1,1.15 //

śrīśuka uvāca /
bhagavan sarvatattvajña vedavedāṅgapāraga /
yad yat prakāraṃ jñānaṃ ca nigūḍhaṃ śrutisammatam // 1,1.16 //

teṣu yat sārabhūtaṃ cāpy ajñānāndhapradīpakam /
tat tat sarvaṃ samālocya māṃ bodhayitum arhasi // 1,1.17 //

||atha śrīkṛṣṇabhaktipraśaṃśā

sa pitā jñānadātā yo jñānaṃ tat kṛṣṇabhaktidam /
sā bhaktiḥ paramā śuddhā kṛṣṇadāsyapradā ca yā // 1,1.18 //

tad eva dāsyaṃ śastaṃ yat sākṣāc caraṇasevanaṃ /
nityaṃ golokavāsaṃ ca purataḥ stavanaṃ hareḥ // 1,1.19 //

śaśvan nimeṣarahitaṃ tatpādapadmadarśanam /
śaśvat tatsārdham ālāpasevākarmaniyojanam // 1,1.20 //

tena sārdham avicchedasthānaṃ paraṃ śobhanam /
bhaktānāṃ vāñcchitaṃ vastu sārabhūtaṃ śrutau śrutam // 1,1.21 //

putrasya vacanaṃ śrutvā vyāsadevo jahāsa saḥ /
vijñāya jñāninaṃ putraṃ paramāhlādam āpa ha // 1,1.22 //

putraṃ śubhāśiṣaṃ kṛtvā sarvajñaḥ sarvabhāvanaḥ /
yathāprāptaṃ gurumukhāt pravaktum upacakrame // 1,1.23 //

śrīvyāsa uvāca /
śuka dhanyo 'si mānyo 'si puṇyarūpo 'si bhārate /
putreṇa bhavatāsmākaṃ kulaṃ muktaṃ ca pāvanam // 1,1.24 //

sa putraḥ kṛṣṇabhakto yo bhārate suyaśaskaraḥ /
punāti puṃsāṃ śatakaṃ janmamātreṇa līlayā // 1,1.25 //

mātāmahānāṃ śatakaṃ mātaraṃ mātṛmātaram /
sodarān bāndhavāṃś caiva bhṛtyān patnīṃ sahātmajām // 1,1.26 //

yatkanyāṃ pratigṛhṇāti tadādipuruṣatrayam /
kanyāpradātā śvaśuro jīvanmuktaḥ sabhāryakaḥ // 1,1.27 //

svayaṃ vidhātā bhagavān paraṃ kṛṣṇaparāyaṇaḥ /
kṛṣṇabhakto vasiṣṭas tu tat suto vaiṣṇavaḥ svayam // 1,1.28 //

vaiṣṇavas tat sutaḥ śaktiḥ kṛṣṇadhyānaikamānasaḥ /
parāśaraś ca tatputraḥ kṛṣṇapādābjasevayā // 1,1.29 //

jīvanmukto mahājñānī yogīndrāṇāṃ guror guruḥ /
ahaṃ vedavibhaktā ca śrīkṛṣṇapādasevayā // 1,1.30 //

gurur me bhagavān sākṣād yogīndro nārado muniḥ /
guror gurur me śambhuś ca yogīndrāṇāṃ guror guruḥ // 1,1.31 //

teṣāṃ puṇyena putras tvaṃ puṇyarāśiś ca mūrtimān /
padmānāṃ mama puṃsāṃ ca prakāśo bhāskaraḥ svayam // 1,1.32 //

śrīkṛṣṇacaraṇāmbhojaṃ pādābjaṃ nāradeśayoḥ /
sarasvatīṃ namas kṛtya jñānaṃ vakṣye sanātanam // 1,1.33 //

śrūyatām pañcarātraṃ ca vedasāram abhīpsitam /
pañcasaṃvādam iṣṭaṃ ca bhaktānām abhivāñcchitam // 1,1.34 //

prāṇādhikaṃ priyaṃ śuddhaṃ paraṃ jñānāmṛtaṃ śubhaṃ /
atha ṣaṭsaṃvādāḥ ?what? hi goloke śataśṛṅge ca parvate // 1,1.35 //

supuṇye virajātīre vaṭamūle manohare /
purato rādhikāyāś ca brahmāṇaṃ kamalodbhavam // 1,1.36 //

tam uvāca mahābhaktaṃ stuvantaṃ praṇataṃ sutam /
pañcarātram idaṃ puṇyaṃ śrutvā ca jagatāṃ vidhiḥ // 1,1.37 //

praṇamya rādhikāṃ kṛṣṇaṃ prayayau śivamandiram /
bhaktyā tam pūjayām āsa śaṅkaraḥ paramādaram // 1,1.38 //

sukhāsane sukhāsīnaṃ svasthaṃ bhaktaṃ ca pūjitam /
papraccha vārtāṃ vinayī vinayena sukhāvahām // 1,1.39 //

sarvaṃ taṃ kathayām āsa pañcarātrādikaṃ śubham /
vasantaṃ vaṭamūle ca svarge mandākinītaṭe // 1,1.40 //

yogīndrair api siddhendrair munīndraiś ca stutaṃ prabhuṃ /
jñānāmṛtaṃ tam uktvā sa brahmalokaṃ jagāma ha // 1,1.41 //

śambhuś ca kathayām āsa svaśiṣyaṃ nāradaṃ munim /
nāradaḥ kathayām āsa puṣkare sūryaparvaṇi // 1,1.42 //

māṃ bhaktam anuraktaṃ ca puṇyāhe munisaṃsadi /
pañcarātram idaṃ śuddhaṃ bhramāndhadhvaṃsadīpakam // 1,1.43 //

||atha pañcarātrapadavyākhyā

rātraṃ ca jñānavacanaṃ jñānaṃ pañcavidhaṃ smṛtam /
tenedaṃ pañcarātraṃ ca pravadanti manīṣiṇaḥ // 1,1.44 //

jñānaṃ paramatattvaṃ ca janmamṛtyujarāpaham /
tato mṛtyuñjayaḥ śambhuḥ saṃprāpa kṛṣṇavaktrataḥ // 1,1.45 //

jñānam dvitīyaṃ paramaṃ mumukṣūṇāṃ ca vāñcchitam /
paraṃ muktipradaṃ śuddhaṃ yato līnaṃ hareḥ pade // 1,1.46 //

jñānaṃ śuddhaṃ tṛtīyaṃ ca maṅgalaṃ kṛṣṇabhaktidam /
taddāsyam adhīṣṭaṃ ca yato dāsyaṃ labhed dhareḥ // 1,1.47 //

caturthaṃ yaugikaṃ jñānaṃ sarvasiddhipradaṃ param /
sarvasvaṃ yogināṃ putra siddhānāṃ ca sukhapradam // 1,1.48 //

aṇimā laghimā vyāptiḥ prākāmyaṃ mahimā tathā /
īśitvaṃ ca vaśitvaṃ ca tathākāmāvasāyitā // 1,1.49 //

sāvajñaṃ dūraśravaṇaṃ parakāyapraveśanam /
kāyavyūhaṃ jīvadānaṃ parajīvaharaṃ param // 1,1.50 //

sargakartṛtvaśilpaṃ ca sargasaṃhārakāraṇam /
siddhaṃ ca ṣoḍaśavidhaṃ jñānināṃ ca yato bhavet // 1,1.51 //

jñānaṃ ca paramaṃ proktaṃ tad vai vaiṣayikaṃ nṛṇāṃ /
yad iṣṭadevī māyā sā paraṃ saṃmohakāraṇaṃ // 1,1.52 //

viṣaye baddhacitaṃ ca sarvam indriyasevanam /
poṣanaṃ svakuṭumbānāṃ svātmanaś ca niranratam // 1,1.53 //

prathamaṃ sāttvikaṃ jñānaṃ dvitīyaṃ ca tad eva ca /
nairguṇyaṃ ca tṛtīyaṃ ca jñānaṃ ca sarvataḥ param // 1,1.54 //

caturtham ca rājasikaṃ bhaktas tan nābhivāñcchati /
pañcamaṃ tāmasaṃ jñānaṃ vidvāṃs tan nābhivāñcchati // 1,1.55 //

jñānaṃ pañcavidhaṃ proktaṃ pañcarātraṃ vidur budhāḥ /
pañcarātraṃ saptavidhaṃ jñānināṃ jñānadaṃ param // 1,1.56 //

brāhmaṃ śaivaṃ ca kaumāraṃ vāsiṣṭaṃ kāpilaṃ param /
gautamīyaṃ nāradīyam idaṃ saptavidhaṃ smṛtam // 1,1.57 //

||atha granthapraśaṃsā

ṣaṭ pañcarātraṃ vedāś ca purāṇāni ca sarvaśaḥ /
itihāsaṃ dharmaśāstraṃ śātraṃ ca siddhiyogajam // 1,1.58 //

dṛṣṭvā sarvaṃ samālokya jñānaṃ saṃprāpya śaṅkarāt /
jñānāmṛtaṃ pañcarātraṃ cakāra nārado muniḥ // 1,1.59 //

puṇyaṃ ca pāpavighnaṃ bhaktidāsyapradaṃ hareḥ /
sarvasvaṃ vaiṣṇavānāṃ ca priyaṃ prāṇādhikaṃ suta // 1,1.60 //

sārabhūtaṃ ca sarveṣāṃ vedānāṃ paramādbhutam /
nāradīyaṃ pañcarātraṃ purāṇeṣu sudurlabham // 1,1.61 //

sarvāntarātmā bhagavān brahmajyotiḥ sanātanam /
paripūrṇatamaḥ śrīmān yathā kṛṣṇaḥ sureṣu ca // 1,1.62 //

yathā devīṣu pūjyā sā mūlaprakṛtir īśvarī /
vaiṣṇavānāṃ ca siddhānāṃ jñānināṃ yogināṃ śivaḥ // 1,1.63 //

viśvastānāṃ indriyānām manaś ca śīghragāminām /
brahmā ca vedaviduṣāṃ pūjyānāṃ ca gaṇeśvaraḥ // 1,1.64 //

sanatkumāro bhagavān munīnāṃ pravaro yathā /
bṛhaspatir buddhimatāṃ siddhānāṃ kapilo yathā // 1,1.65 //

yogīndrānāṃ satāṃ śuddha ṛṣir nārāyaṇo yathā /
kavīnāṃ ca yathā śukraḥ paṇḍitānāṃ bṛhaspatiḥ // 1,1.66 //

saritām ca yathā gaṅgā samudrāṇāṃ jalārṇavaḥ /
vṛndāvanaṃ vanānāṃ ca varṣānāṃ bhārataṃ yathā // 1,1.67 //

puṣkaraṃ tatra tīrthānāṃ pūjyānāṃ vaiṣṇavo yathā /
ātmākāśo yathāptānāṃ yathā kāśī purīṣu ca // 1,1.68 //

vṛkṣāṇāṃ kalpavṛkṣaś ca surabhī kāmadhenuṣu /
puṣpāṇāṃ pārijātaś ca patrāṇāṃ tulasī yathā // 1,1.69 //

mantrāṇāṃ kṛṣṇamantraś ca yathā vidyā dhaneṣv api /
yathā tejasvināṃ sūryo miṣṭānām amṛtaṃ yathā // 1,1.70 //

ādhārāṇāṃ ca sthūlānāṃ mahāviṣṇur yathā suta /
sūkṣmāṇāṃ paramāṇuś ca guruṇāṃ mantratantradaḥ // 1,1.71 //

putraś ca snehapātrāṇāṃ nakṣatrāṇāṃ yathā śaśī /
yathā ghṛtaṃ ca gavyānāṃ śasyānāṃ dhānyam īpsitam // 1,1.72 //

śāstrāṇāṃ ca yathā vedāḥ sāśramāṇāṃ yathā dvijaḥ /
taijasānāṃ yathā ratnaṃ muktāmāṇikyahīrakam // 1,1.73 //

yathā chandasi gāyatrī durgā śaktimatīṣv api /
pativratāsu laksmīś ca kṣamāśīlāsu medinī // 1,1.74 //

saubhāgyāsu sundarīṣu rādhā kṛṣṇapriyāsu ca /
hanumān vānarāṇāṃ ca pakṣiṇāṃ garuḍo yathā // 1,1.75 //

vāhanānāṃ balavatāṃ śaṅkarasya yathā vṛṣaḥ /
śālagrāmaś ca yantrāṇāṃ pūjāsu kṛṣṇapūjanam // 1,1.76 //

ekādaśī vratānāṃ ca tapaḥsv anaśanaṃ yathā /
yajñānāṃ japayajñaś ca satyaṃ dharmeṣu putraka // 1,1.77 //

suśīlaṃ ca guṇānāṃ ca puṇyeṣu kṛṣṇakīrtanam /
śobhāśu sukhadṛśyeṣu prabhā tejaḥsu sarvataḥ // 1,1.78 //

poṣṭrīṇāṃ upakartṛṇāṃ mitrāṇāṃ jananī yathā /
lokānām api lokeśaḥ śeṣo nāgeṣu pūjitaḥ // 1,1.79 //

sudarśanaṃ ca śastrāṇāṃ viśvakarmā ca śilpinām /
dharmiṣṭheṣu dayāvatsu devarṣisu mahatsu ca // 1,1.80 //

viṣṇubhakteṣu vijñeṣu yathaiva nārado muniḥ /
evaṃ ca sarvaśāstreṣu pañcarātraṃ ca pūjitam // 1,1.81 //

yathā nipīya pīyūṣaṃ na spṛhā cānyavastuṣu /
pañcarātram abhijñāya nānyeṣu ca spṛhā satām // 1,1.82 //

sarvārthajñānabījaṃ cāpy ajñānāndhapradīpam /
vedasāroddhṛtaṃ tattvaṃ sarveṣāṃ samabhīpsitam // 1,1.83 //

||iti śrīnāradapañcarātre granthapraśaṃsanaṃ nāma prathamo 'dhyāyaḥ

||prathamaikarātre dvitīyo 'dhyāyaḥ

śuka uvāca /
kutra vā pañcarātraṃ ca nāradasya ca dhīmate /
pradattaṃ śambhunā tāta tan me vyākhyātum arhasi // 1,2.1 //

||atha nāradasya tapovarṇanam

vyāsa uvāca /
adhītya sarvān vedāṃś ca vedāṅgāṇ pitur antike /
jagāma tītrhaṃ kedāraṃ supraśastaṃ ca bhārate // 1,2.2 //

himālayasya pūrve ya gaṅgātīre manohare /
siddhe nārāyaṇakṣetre sarveṣāṃ abhivāñcchite // 1,2.3 //

tapaś cakāra sa munir divyaṃ varṣasahasrakam /
pitroktenaiva vidhinā satataṃ saṃyutaḥ śuciḥ // 1,2.4 //

śuśrāvākāśavāṇīṃ ca tapaso 'nte mahāmuniḥ /
svalpākṣarāṃ ca bahvarthāṃ pariṇāmasukhāvahām // 1,2.5 //

||atha nāradaṃ prati daivavāṇī

aśarīriṣy uvāca  /
ādhārito yadi haris tapasā tataḥ kim /
nādhārito yadi haris tapasā tataḥ kim /
antar bahir yadi haris tapasā tataḥ kim /
nāntar bahir yadi haris tapasā tataḥ kim // 1,2.6 //


virama virama brahman kiṃ tapasyāsu vatsa /
vraja vraja dvija śīghraṃ śaṅkaraṃ jñānasindhum /
labha labha haribhaktiṃ vaiṣṇavoktāṃ supakvāṃ /
bhavanigaḍanibandhachedinīṃ kartanīṃ ca // 1,2.7 //

iti śrutvā ca sa munir vimanāḥ svarṇadītaṭe /
cakārārthānusandhānaṃ na prasannam ca tan manaḥ // 1,2.8 //

ruroda svarṇadītīre smāraṃ smāraṃ hareḥ padam /
dadarśa puratas tātaṃ brahmāṇaṃ sakumārakam // 1,2.9 //

nanāma sahasā mūrdhnā pitaraṃ taṃ sahodaram /
pādyam arghyaṃ ca pradadau javena sādaraṃ muniḥ // 1,2.10 //

ślokadvayārthaṃ papraccha kumāraṃ jagatāṃ vidhim /
sukhāsīnaṃ susthiraṃ ca sasmitaṃ ca gataśramam // 1,2.11 //

svātmārāmaṃ pūrṇakāmaṃ jñānināṃ ca guror gurum /
sāśrunetraḥ pulakito bhaktyā praṇatakandharam // 1,2.12 //

nāradasya vacaḥ śrutvā dṛṣṭvā taṃ kātaraṃ vidhiḥ /
putreṇa sārddham ālocya vyākhyāṃ kartuṃ samārebhe // 1,2.13 //

||atha daivavāṇyarthaḥ

brahmovāca /
he vatsa pūrvaślokārthaṃ nigūḍhaṃ śrutisammatam /
vedārthaṃ dvividhaṃ śuddhaṃ vyākhyāṃ kurvanti vaidikāḥ // 1,2.14 //

ārādhito yadi harir yena puṃsā svabhaktitaḥ /
kiṃ tasya tapasā vyarthaṃ tīrthapūtasya nārada // 1,2.15 //

kṛṣṇamantropāsakasya jīvanmuktasya bhārate /
tapaś copahāsabījaṃ tathā carvitacarvaṇam // 1,2.16 //

mantragrahaṇamātreṇa puruṣāṇāṃ śataṃ suta /
punāti svasvabhaktaṃ ca cāndhavaṃś cāvalīlayā // 1,2.17 //

nahi dharmo nahi tapaḥ śrīkṛṣṇasevanāt param /
pariśramaṃ ca viphalaṃ tapasā vaiṣṇavasya ca // 1,2.18 //

kṛṣṇamantropāsakasya tīrthapūtasya putraka /
tīrthasnānam anaśanaṃ vedeṣu ca viḍambanam // 1,2.19 //

pūrvakarmānurodhena yat pāpaṃ vaiṣṇavasya ca /
mantragrahaṇamātreṇa naṣṭaṃ vahnau yathā tṛṇam // 1,2.20 //

pavitraḥ paramo vahniḥ pavitraṃ cāmalaṃ jalaṃ /
pavitraṃ bhārataṃ varṣaṃ tīrthaṃ yat tulasīdalam // 1,2.21 //

punāti līlayaitāni śuddhaḥ kṛṣṇaparāyaṇaḥ /
upasparśa ca bhaktasyāpy ete vāñcchanti sādaram // 1,2.22 //

bhaktasya pādarajasā sadyaḥ pūtā vasundharā /
nahi pūtastribhuvane śrīkṛṣṇasevakāt paraḥ // 1,2.23 //

śāligrāmaśilācakre karoti kṛṣṇapūjanam /
tatpādodakanaivedyaṃ nityaṃ bhuṅkte ca yaḥ pumān // 1,2.24 //

sa vaiṣṇavo mahāpūtas tanmantropāsakaḥ śuciḥ /
punāti puṃsāṃ śatakaṃ janmamātrāt sabāndhavam // 1,2.25 //

vatsa ślokasyaikapādaṃ vyākhyātaṃ ca yathāgamam /
vyākhyātaṃ karomy anyapādaṃ yathājñānaṃ miśāmaya // 1,2.26 //

nārādhito yadi harir yena puṃsādhamena ca /
kiṃ tasya tapasā vyarthaṃ niṣphalaṃ tatpariśramaḥ // 1,2.27 //

vratāny eva hi dānāni tapāṃsy anaśanāni ca /
vedopayuktā yajñāś ca karmāṇi ca śubhāni ca /
na niṣpunāty abhaktaṃ ca surākumbham ivāpagā // 1,2.28 //

abhaktasparśamātreṇa tīrthāni kampitāni ca /
abhaktabhāraduḥkhena kampitā sā vasundharā // 1,2.29 //

ślokārdhaṃ kathitaṃ vatsa kiṃcid eva yathāgamam /
tasyārdhasyāpi vyākhyānaṃ karomīti niśāmaya // 1,2.30 //

vedasāraṃ kṛṣṇamataṃ mamāpi nahi kalpanā /
antar bahir yadi hair yeṣāṃ puṃsāṃ mahātmanāṃ // 1,2.31 //

svapne jāgaraṇe śaśvat tapas teṣāṃ ca niṣphalam /
sa eva viṣṇutulyo hi tadaṃśo bhārate mune // 1,2.32 //

tasya rakṣānibandhena tadabhyāse sudarśanam /
dhyānamātreṇa niṣpāpaḥ punāti bhuvanatrayam // 1,2.33 //

datvā cakraṃ ca rakṣārthaṃ na niścinto janārdanaḥ /
svayaṃ tan nikaṭaṃ yāti draṣṭuṃ rakṣaṇāya ca // 1,2.34 //

tatparo hi priyo nāsti kṛṣṇasya paramātmanaḥ /
nahi bhaktāt paraś cātmā prāṇāś cāvayavādayaḥ /
na lakṣmī rādhikā vāṇī svayaṃbhuḥ śambhur eva ca // 1,2.35 //

bhaktaprāṇo hi kṛṣṇaś ca kṛṣṇaprāṇā hi vaiṣṇavāḥ /
dhyāyante vaiṣṇavāḥ kṛṣṇaṃ kṛṣṇaś ca vaiṣṇavāṃs tathā // 1,2.36 //

vyākhyātam ca tripādaṃ ca he munīndra yathāgamam /
śeṣapādasya vyākhyānaṃ karomīti niśāmaya // 1,2.37 //

nāntar bahir yadi harir yeṣāṃ puṃsāṃ ca nārada /
teṣām api tapo vyartham antarmalinacetasāṃ // 1,2.38 //

kiṃ tajjñānena tapasā vratena niyamena ca /
tīrthasnānena puṇyenāpi abhaktamūḍhacetasāṃ // 1,2.39 //

kṛṣṇabhaktivihīnebhyo dvijebhyaḥ śvapaco mahān /
śūkaro mlecchanivahaḥ svadharmācareṇa ca // 1,2.40 //

svadharmahīnā viprāś cāpy abhakṣyabhakṣaṇena ca /
nityaṃ nityaṃ vidharmeṇa patitāḥ śvapacādhamāḥ // 1,2.41 //

brahmaṇānāṃ svadharmaś ca santataṃ kṛṣṇasevanam /
nityaṃ te bhuñjate santas tannaivedyaṃ pādodakam // 1,2.42 //

na datvā haraye yas tu yadi bhuṅkte dvijādhamaḥ /
annaṃ viṣṭhāsamaṃ mūtrasamaṃ toyaṃ vidur budhāḥ // 1,2.43 //

bhuṅkte svabhakṣyaṃ kolaś ca mlecchaś ca śvapacādhamaḥ /
vipro nityam abhakṣyaṃ ca bhuṅkte ca patitas tataḥ // 1,2.44 //

ślokam ekaṃ ca vyākhyātaṃ yathājñānaṃ ca nārada /
sannibodha parasyārdhaṃ vyākhyānaṃ ca yathocitaṃ // 1,2.45 //

tapaso virama brahman vyarthaṃ bhaktatapo dhruvam /
śaṅkaraś ca guruṃ kṛtvā haribhaktiṃ labhāciram // 1,2.46 //

supakvā haribhaktiś ca taraṇī bhavatāraṇe /
gurur eva paraṃ brahma karṇadhārasvarūpakaḥ // 1,2.47 //

ity evam uktvā tvāṃ devī prajagāma sarasvatī /
vyākhyātas tadabhiprāyaḥ kiṃ bhūyaḥ kathayāmi te // 1,2.48 //

brahmāṇaś ca vacaḥ śrutvā jahāsa yogināṃ guruḥ /
sanatkumāro bhagavān uvāca pitaraṃ śuka // 1,2.49 //

sanatkumāra uvāca /
pūrvaślokasya vyākhyānaṃ na buddhaṃ śiśunā mayā /
putraṃ śiṣyam abodhaṃ ca yuktaṃ bodhayitum punaḥ // 1,2.50 //

ārādhito harir yena tasya vyarthaṃ tapo yadi /
nārādhito hair yena tasya vyartaṃ tapo yadi // 1,2.51 //

tasyārahitau tau dvau tapasaś ca sthalaṃ kutaḥ /
tapaḥ kurvanti ye tāta tvaṃ māṃ bodhaya bālakam // 1,2.52 //

putrasya vacanaṃ śrutvā sandigdho jagatāṃ guruḥ /
dadhyau kṛṣṇapadāmbhojaṃ paraṃ kalpataruṃ śuka // 1,2.53 //

kṣaṇaṃ saṃcitya pādābjaṃ prāpa rāddhāntam īpsitam /
vyākhyāṃ kartuṃ samārebhe vidhātā jagatām api // 1,2.54 //

||atha naivedyapraśaṃsā

brahmovāca /
dhanyo 'haṃ bhavataḥ putrāt jñānināṃ ca guror guroḥ /
viṣṇubhaktāc ca dharmiṣṭhāt satputrāc ca pitā sukhī // 1,2.55 //

dhanyo 'si paṇḍito 'si tvaṃ haribhakto 'si putraka /
mamāpi saphalaṃ janma jīvanaṃ ca tvayā budha // 1,2.56 //

nibodha pūrvaślokārthaṃ punar vyākhyāṃ karomi ca /
tathāpi cen na santoṣo bhavān vyākhyāṃ kariṣyati // 1,2.57 //

āśabdaḥ samyagarthe ca rādhitaḥ prāptavācakaḥ /
saṃprāptaś ca harir yena vyarthas tasya tapaḥśramaḥ // 1,2.58 //

yena samyakprakāreṇa saṃprāpto harir īśvaraḥ /
svapne jñāne ca jñātas teṣāṃ vyarthas tapaḥśramaḥ // 1,2.59 //

śrīkṛṣṇavimukhaṃ mūḍhaṃ dvijam eva narādhamam /
tīrthaṃ dānaṃ tāpaḥ puṇyaṃ vrataṃ naiva punāti tam // 1,2.60 //

yaś ca mūḍhatamo loke yaś ca bhaktiṃ parāṃ gataḥ /
tāv ubhau sukhasedhete tapaḥ kurvanti madhyamāḥ // 1,2.61 //

devān anyāṃś ca bhajate hariṃ jānāti tatparaḥ /
tapaḥ karoti taṃ prāptum ākāṅkṣan madhyamo janaḥ // 1,2.62 //

prāktanād anurāgī ca gṛhī saṃsārasaṃvṛtaḥ /
tapaḥ karoti śrīkṛṣṇapādapadmārtham īpsitam // 1,2.63 //

paraṃ śrīkṛṣṇabhajanaṃ dhyānaṃ tannāmakīrtanam /
tatpādodakanaivedyabhakṣaṇaṃ sarvavāñcchitam // 1,2.64 //

atīva mūḍho vipraś ca prāktanād gurudoṣataḥ /
tāmaso hi na jānāti śrīkṛṣṇaṃ triguṇāt param // 1,2.65 //

ajñānād atha vā jñānāt satsaṅgād eva prāktanāt /
bhuṅkte naivedyaṃ īśasya kṛṣṇasya paramātmanaḥ // 1,2.66 //

sa ca mukto bhavet putra mucyate sarvapātakāt /
sa yāti divyayānena golokaṃ lokam uttamam // 1,2.67 //

śṛṇu vatsa pravakṣyāmi pūrvākhyānaṃ purātanam /
atīva suśravaṃ cāru madhuraṃ muktidaṃ param // 1,2.68 //

kānyakubjaḥ sukṣubdhaś ca brāhmaṇo grāmayājakaḥ /
devalo vṛṣavāhaś ca mahāmūḍhaś ca pātakī // 1,2.69 //

svapne jñāne na jānāti puṇyaṃ vā kṛṣṇapūjanam /
kṛṣṇabhaktasahālāpadarśanasparśanaṃ śubhaṃ // 1,2.70 //

babhūva prāktanāt tasya kṣaṇamātraṃ sudurlabham /
tena puṇyena naivedyaṃ lebhe kṛṣṇasya brāhmaṇaḥ // 1,2.71 //

pituḥ puṇyena putraś ca mārge patitam alpakam /
svayaṃ bhuktāvaśeṣaṃ ca patitaṃ vaiṣṇavāj janāt // 1,2.72 //

susnigdhākṣatajīrṇaṃ ca rajasā miśritaṃ param /
gacchatas tatra viprasya patitaṃ bhakṣyavastu ca // 1,2.73 //

naivedyopari kṛṣṇasya tvarāyuktasya putraka /
tadvastu bhuktaṃ vipreṇa kṛṣṇanaivedyamiśritam // 1,2.74 //

saputreṇa kṣudhārtena tau yayatur gṛham /
viprocchiṣṭaṃ ca bubhuje tasya patnī pativratā // 1,2.75 //

paramparānusaṃbandhāt pavitrā sā babhūva ha /
jīvanmukto brāhmaṇaś ca babhūva sa saputrakaḥ // 1,2.76 //

kālena tena puṇyena vyāghrabhuktaś ca kānane /
sārdhaṃ ca vyāghraputrābhyāṃ golokaṃ prayayau dvija /
pativratā sahamṛtā bhartā sārdhaṃ jagāma sā // 1,2.77 //

||iti śrīnāradapañcarātre jñānāmṛtasāre prathamaikarātre brahmasanatkumārasaṃvāde naivedyapraśaṃsanaṃ nāma dvitīyo 'dhyāyaḥ

||prathamaikarātre tṛtīyo 'dhyāyaḥ

sanatkumāra uvāca /

aho tāta kim āścaryaṃ kṛṣṇasya paramātmanaḥ /
paraṃ naivedyamāhātmyaṃ vistarād vada sāmpratam // 1,3.1 //

||atha śrīkṛṣṇamahimavarṇanam

brahmovāca /
ekadā brāhmaṇo hṛṣṭaḥ praphullavadanekṣaṇaḥ /
putreṇa sārdhaṃ prayayau bāndhavasya gṛhaṃ mudā // 1,3.2 //

nimantrito vivāhena mahāsaṃbhārasaṃbhṛtaḥ /
bhuktvā pītvā ca tadgṛhe svagṛhaṃ prayayau mudā // 1,3.3 //

saputro brāhmaṇo mārge kṣutpipāsārditaḥ sutaḥ /
dadarśa candrabhāgāṃ tāṃ nadīm atimanoharām // 1,3.4 //

uvāca putraḥ pitraṃ snātvā bhokṣyāmi ceti bhoḥ /
kṣutpipāsā balavatī vardhate tāta vartmani // 1,3.5 //

putrasya vacanaṃ śrutvā tam uvāca dvijaḥ svayam /
bhayaṃkaraṃ vanam idaṃ samīpe saritaḥ sutaḥ // 1,3.6 //

suśīghraṃ gaccha grāmāntaṃ puro ramyasarovaram /
tatra snātvā ca bhokṣyāvo gaccha vatsa yathāsukham // 1,3.7 //

tātasya vacanaṃ śrutvā jahāsa ca cukopa ha /
pitaraṃ vaktum ārebhe raktapaṇgkajalocanaḥ // 1,3.8 //

śiśur uvāca /
bālo 'haṃ daśavarṣīyas tvaṃ ca vṛddhaś ca jñānadaḥ /
pitā dadāti putrāya jñānaṃ sarvatra bhūtale // 1,3.9 //

aho duratyayaḥ kālo vṛdhho vadati bālavat /
kathaṃ prāktanam ullaṅghya brūhi tāta duratyayam // 1,3.10 //

prāktanāt sukhaduḥkhaṃ ca rogaṃ śokaṃ bhayaṃ pitaḥ /
sumṛtyur apamṛtyur vā cirāyur alpajīvanaḥ // 1,3.11 //

yatra kāle ca yanmṛtyur bhavanaṃ śubhakarma ca /
nyūnādhikaṃ kṣaṇaṃ nāsti niṣekaḥ kena vāryate // 1,3.12 //

yasya haste ca yanmṛtyur vidhātrā likhitaḥ purā /
na ca taṃ khaṇḍituṃ śaktaḥ svayaṃ viṣṇuś ca śaṅkaraḥ // 1,3.13 //

tāta vyartham adhītaṃ te durbuddher janma niṣphalam /
subuddheḥ saphalaṃ janma tatkṣaṇam jīvanaṃ sukham // 1,3.14 //

yena śuklīkṛtā haṃsāḥ śukāś ca haritīkṛtāḥ /
mayūrāś citritā yena sa me rakṣāṃ kariṣyati // 1,3.15 //

yena kṛṣṇena viśvāni cāsaṃkhyāni kṛtāni ca /
carācaraṃ ca yo rakṣet sa me rakṣāṃ kariṣyati // 1,3.16 //

ghorāraṇye sukhaṃ śete yo hi kṛṣṇena rakṣitaḥ /
nirbandho 'pi yasya maraṇaṃ tasya mandire // 1,3.17 //

yaḥ śete nāgaśayyāsu prāktanān maṅgalāhitaḥ /
yo nāgabhakṣito bhogāt sa bhṛto garuḍāntike // 1,3.18 //

na samudre ca mriyate nāgnirāśau viṣānale /
na śastreṇa na cāstreṇāyurmarmāṇi rakṣati // 1,3.19 //

nāprāptakālo mriyate viddhaḥ śaraśatair api /
tṛṇāgreṇāpi saṃspṛṣṭaḥ prāptakālo na jīvati // 1,3.20 //

kaścid garbhe ca mriyate kaścid bhūmiṣṭamātrataḥ /
kaścid yauvanakāle ca kaścid eva hi vārddhake // 1,3.21 //

kaścid cirāyu rogī cāpy arogī cāpi kaścana /
kaścid dhanī daridraś ca kaścid eva hi karmaṇā // 1,3.22 //

kaścit kalpāntajīvī ca cirajīvī ca kaścana /
prāktanād amaraḥ kaścin niṣeko balavattaraḥ // 1,3.23 //

kaścid yāti ca rājendro divyayānena karmaṇā /
kaścit kīṭapataṅgeṣu kaścit paśvādiyoniṣu // 1,3.24 //

kaścid eva hi sannyāsī kaścic ca naraghātakaḥ /
kaścid gajendragāmī ca paśuyāyī ca kaścana // 1,3.25 //
kaścid sūkṣmāṃśukādhārī kaścij jīrṇapaṭī janaḥ // 1,3.26 //
kaścin nagno 'apy anāhārī sudhābhojī ca ca kaścana /
kaścic ca sundaraḥ śrīmān galatkuṣṭī ca kaścana // 1,3.27 //

kaścit kubjaś cāṅgahīno badhiraḥ kāṇa eva ca /
kaścid dīrgho madhyamaś ca kaścit khañjaś ca vāmanaḥ // 1,3.28 //

kaścit kṛṣṇaś ca gauraś ca śyāmalaś ca svakarmaṇā /
kaścid bhaktyā ca prāpnoti kṛṣṇadāsyaṃ sudurlabham // 1,3.29 //

brahmaṇaḥ paramaṃ sthānaṃ janmamṛtyujarāharam /
kaścit prāpnoti paramaṃ brahmalokaṃ nirāmayam // 1,3.30 //

kaścit svargam indrapadaṃ śivalokaṃ svakarmaṇā /
kaścit vargam indralokaṃ yamalokaṃ ca kaścana // 1,3.31 //

kaścic ca narake ghore prāpnoti kleśam ulvaṇam /
tādito yamadūtena kṣudhitas tṛṣitaḥ sadā // 1,3.32 //

bhuṅkte viṇmūtrakīṭaṃ tanmalaṃ śleṣmaṃ garaṃ vasām /
kṣuradhāre taptataile vahnau śite jale sthale // 1,3.33 //

prāpnoti dāruṇaṃ duḥkham ākalpaṃ pātakī pitaḥ /
tathā bhogāvaśeṣe ca labdhā janma svakarmaṇā // 1,3.34 //

vyādhiyuktaḥ pramucyeta tayā ced īśvarecchayā /
yadbhayād vāti vāto 'yaṃ sūryas tapati yadbhayāt // 1,3.35 //

varṣatīndro dahaty agnir mṛtyuś carati jantuṣu /
yasyājñayā sṛṣṭividhau mūrmo 'nantaṃ dadhāti ca // 1,3.36 //

sa ca sarvaṃ ca brahmāṇḍaṃ līlayā ceśvarecchayā /
yasyājñayā mahābhītā sarvādhārā vasundharā // 1,3.37 //

dharā sā sarvasyādyā ratnavāṃś ca himālayaḥ /
svayaṃ vidhātā bhagavān dhyāyate yam aharniśam // 1,3.38 //

yaṃ dhyāyate ca bhajate svayaṃ mṛtuñjayaḥ śivaḥ /
sahasravaktro 'yaṃ stauti dhyāyate bhajate sadā // 1,3.39 //

svayṃ sarasvatī stauti yam īśvaram abhīpsitam /
sevate pādapadmaṃ ca svayaṃ padmālayā pitaḥ // 1,3.40 //

māyā bhītā ca yaṃ stauti durgā durgatināśinī /
stuvanti vedāḥ satataṃ sāvitrī vedamātṛkā // 1,3.41 //

siddhendrāś ca munīndrāś ca yogīndrāḥ sanakādayaḥ /
rājendrāś cāsurendrāś ca surendrā manavas tathā // 1,3.42 //

dhyāyante ca bhajante ca bhaktāḥ santo hi santatam /
kecid vidanti yaṃ brahmāṃ bhagavantaṃ sanātanam // 1,3.43 //

kecit pradhānaṃ sarvādyaṃ kecic ca jyotir īśvaram /
kecic ca sarvarūpaṃ ca sarvakāraṇakāraṇam // 1,3.44 //

kecit svecchābhayaṃ rūpaṃ bhaktānugrahavigraham /
kecit suruciraṃ śyāmasundaraṃ manoharam // 1,3.45 //

sānandaṃ paramānandaṃ govindaṃ nandanandanam /
bhaja tāta paraṃ brahma smara śaśvat sureśvaram // 1,3.46 //

ity evam uktvā pitaraṃ candrabhāgānadījale /
snātvā papau jalaṃ svacchaṃ bubhuje miṣṭamodakam // 1,3.47 //

pitā tadvacanaṃ śrutvā sānandāśru mumoca saḥ /
cucumba gaṇḍaṃ putrasya samāśleṣaṇapūrvakam // 1,3.48 //

pitā snātvā samārebhe sandhyāṃ kartuṃ ca pūjanam /
susnātaṃ pitaraṃ dṛṣṭvā putraḥ sa prayayau vanam // 1,3.49 //

patraṃ bhojanapātrārtham arhatuṃ cañcalaḥ śiśuḥ /
cakāra cayanaṃ tūrṇaṃ praśastaṃ patrapañcakam // 1,3.50 //

sundaraṃ kusumaṃ vanyaṃ pūjanārthaṃ pitus tathā /
dadarśa purato bālaḥ supakvaṃ vadarīphalam // 1,3.51 //

cakāra cayanaṃ tāni phalāni śobhanāni ca /
dhātrīphalaṃ supakvaṃ ca pakvam āmrātakaṃ tathā // 1,3.52 //

supakvaṃ ca kadambaṃ ca cakāra cayanaṃ punaḥ /
supakvaṃ sundaraṃ ramyaṃ dāḍimaṃ śrīphalaṃ tathā // 1,3.53 //

ramyaṃ jambuphalaṃ caiva kharjūraṃ sumanoharam /
karañjakaṃ ca jāmbīraṃ sundaraṃ cikuraṃ tathā // 1,3.54 //

tat sarvaṃ cayanaṃ kṛtvā dadarśa purataḥ saraḥ /
sunirmalaṃ jalaṃ svacchaṃ śvetapadmaṃ manoharam // 1,3.55 //

ruciraṃ raktakahlāraṃ prasphuṭaṃ ca jalāntike /
vihāya tāni sarvāṇi saraḥśirasi susthale // 1,3.56 //

papau saraḥsvacchatoyaṃ jahāra padmam ulvaṇam /
kiṃcit suraktakahlāraṃ pakvaṃ padmaphalaṃ tathā // 1,3.57 //

sarvam āharaṇaṃ kṛtvā pitaraṃ gantum udyataḥ /
praphullavadanaḥ śrīmān sasmito dvijabālakaḥ // 1,3.58 //

praphullacampakataruṃ dadarśa purataḥ śiśuḥ /
mallikāmālatīkundayūthikāmādhavīlatāḥ // 1,3.59 //

cakāra cayanaṃ sphītaḥ puṣpāṇi sundarāṇi ca /
puṣpeṇa phalapatreṇa tasya bhāro babhūva ha // 1,3.60 //

bālo voḍhum aśakyantaś ca yayau gamanamantharaḥ /
na phalaṃ bubhuje so 'pi dharmādharmabhayena ca // 1,3.61 //

puro dadarśa sa śiśur ghoraṃ vyāghrālayaṃ bhiyā /
tāta tāteti śabdaṃ ca cakāra ha punaḥ punaḥ // 1,3.62 //

na dadarśa ca tātaṃ ca śārdulaṃ ca dadarśa saḥ /
bhiyā sasmāra govindapādāravaindam īpsitam // 1,3.63 //

hariṃ narahariṃ rāmaṃ kṛṣṇaṃ viṣṇuṃ ca mādhvam /
dāmodaraṃ hṛṣīkeśaṃ mukundaṃ madhusūdanam // 1,3.64 //

etāni daśa nāmāni japan vipraśiśur bhiyā /
prayayau purataḥ śīghraṃ punar eva sarovaram // 1,3.65 //

saraso nirmale tīre puṣpāṇi ca phalāni ca /
dadau bhaktyā bhagavate kṛṣṇāya paramātmane // 1,3.66 //

śrīkṛṣṇapūjāṃ kurvantaṃ dhyānamānāṃ padāmbujam /
nikaṭaṃ na yayau vyāghro dṛṣṭvā bālaṃ ca dūrataḥ // 1,3.67 //

vyāghraṃ dadarśa bālaś ca prakaṭāsyaṃ bhayānakam /
vikṛtākāradaśanaṃ vikaṭākṣaṃ mahodaram // 1,3.68 //

dṛṣṭvā ca durato vyāghram uvāsa sarasas taṭe /
dadhyau kṛṣṇapadāmbhojaṃ janmamṛtyujarāharam // 1,3.69 //

mūlādhāraṃ svādhiṣṭhānaṃ maṇipūram anāhatam /
viśuddhaṃ ca tathājñākhyaṃ ṣaṭcakraṃ ca vibhāvya ca // 1,3.70 //

kuṇḍalinyā svaśaktyā ca sahitaṃ parameśvaram /
sahasradalapadmasthaṃ hṛdaye svātmanaḥ prabhum // 1,3.71 //

dadarśa dvibhujaṃ kṛṣṇaṃ pītakauśeyavāsasam /
sasmitaṃ sundaraṃ śuddhaṃ navīnajaladaprabham // 1,3.72 //

koṭikandarpasaundaryalīlādhāmamanoharam /
koṭipārvaṇapūrṇenduprabhājuṣṭaṃ ca sundaram // 1,3.73 //

sukhadṛśyaṃ surūpaṃ ca bhaktānugrahakārakam /
candanokṣitasarvāṅgaṃ ratnabhūṣaṇabhūṣitam // 1,3.74 //

praphullapadmanayanaṃ rādhāvakṣaḥsthalasthitam /
mālatīmālyasambaddhacūḍācārusuśobhanam // 1,3.75 //

dhṛtaratnaṃ ratnapadmaṃ dakṣiṇena kareṇa ca /
vāmena maṇinirmāṇadīptadarpaṇam ujjvalam // 1,3.76 //

ratnakuṇḍālayugmena gaṇḍasthalavirājitam /
kausthubhena maṇīndreṇa cāruvakṣaḥsthalojjvalam // 1,3.77 //
muktārājivinindaikadantarājivirājitam /
ājānumālatīmālāvanamālāvibhūṣitam // 1,3.78 //


vedānanasarasvatyā stutaṃ brahmeśavanditam /
padmāpadmālayāmāyāsaṃsevitapadāmbujam // 1,3.79 //

paripūrṇatamam brahma paramātmānam īśvaram /
nirliptaṃ sākṣibhūtaṃ ca bhagavantaṃ sanātanam // 1,3.80 //

sarveśāṃ sarvarūpaṃ ca sarvakāraṇakāraṇam /
puruṣaṃ paramātmaikaṃ pareśaṃ prakṛteḥ paraṃ // 1,3.81 //

evaṃ bhūtaṃ vibhuṃ dṛṣṭvā manasā praṇanāma tam /
tuṣṭāva parayā bhaktyā tam īśaṃ sampuṭāñjaliḥ // 1,3.82 //

śrīsubhadra uvāca /
he nātha darśnaṃ dehi māṃ bhaktaṃ śaraṇāgatam /
śrīda śrīśa śrīnivāsa śrīnidhe śrīniketana // 1,3.83 //

śriyā sevitapādābja śrīsamutpattikāraṇa /
vedānirvacaniyeśa nirīha nirguṇādhipa // 1,3.84 //

sarvādya sarvanilaya sarvabīja sanātana /
śānta sarasvatīkānta nitānta sarvakarmasu // 1,3.85 //

sarvādhāra nirādhāra kāmapūra parāt para /
duṣpārāsārasaṃsārakarṇadhāra namo 'stu te // 1,3.86 //

ity evam uktvā sa śiśu ruroda ca punaḥ punaḥ /
dhyāyena tatpadāmbhojaṃ śaraṇaṃ ca cakāra saḥ // 1,3.87 //

iti viprakṛtaṃ stotraṃ trisandhyaṃ yaḥ paṭhen naraḥ /
mucyate sarvapāpebhyo viṣṇulokaṃ sa gacchati // 1,3.88 //

||iti śrīnāradapañcarātre jñānāmṛtasāre prathamaikarātre brahmasanatkumārasaṃvāde śrīkṛṣṇamahimopālambhanaṃ nāma tṛtīyo 'dhyāyaḥ

||prathamaikarātre caturtho 'dhyāyaḥ

brahmovāca /
brāhmaṇasya stavaṃ śrutvā parituṣṭo janārdanaḥ /
kṛpāṃ cakāra bhagavān bhakteśo bhaktavatsalaḥ // 1,4.1 //

etasminn antare tatra bhagavān nandanandanaḥ /
nārāyaṇarṣiḥ kṛpayā cājagāma sarovaram // 1,4.2 //

dadarśa brāhmaṇavaṭuṃ tam eva munipuṅgavam /
tejasā sukhadṛśyena sundaraṃ sumanoharam // 1,4.3 //

pītavastraparīdhānaṃ navīnajaladaprabham /
candanokṣitasarvāṅgaṃ vanamālāvibhūṣitam // 1,4.4 //

prasannavadanaṃ śuddhaṃ sasmitaṃ sarvapūjitam /
vibhāntaṃ ca japantaṃ śuddhasphaṭikamālayā // 1,4.5 //

dṛṣṭvā nanāma sahasā śirasā viprapuṅgavaḥ /
śubhāśiṣaṃ dadau tasmai datvā śirasi hastakam // 1,4.6 //

tam uvāca muniśreṣṭhaḥ kṛpayā dīnavatsalaḥ /
hitaṃ tathyaṃ nītisāraṃ pariṇāmasukhāvaham // 1,4.7 //

śrīnārāyaṇarṣir uvāca /
aye vipra mahābhāga saphalaṃ jīvanaṃ tava /
yasmin kule ca jāto 'si taddhanyaṃ supraśaṃsitam // 1,4.8 //

bhaja tvaṃ paramānandaṃ sānandaṃ nandanandanam /
dhruvaṃ yāsyasi golokaṃ paramānandam īpsitam // 1,4.9 //

tat kulaṃ pāvanaṃ dhanyaṃ yaśasyaṃ ca nirāpadam /
yasmin svayaṃ bhavān jātaḥ puṇyaḥ kṛṣṇaparāyaṇaḥ // 1,4.10 //

naivedyaṃ patitaṃ mārge jīrnaṃ śvāpadabhakṣitam /
bhuktvā tavaiṣā buddhiś ca kṛṣṇabhaktir babhūva ca // 1,4.11 //

kṛṣṇanaivedyamāhātmyaṃ ko vatsa kathituṃ kṣamaḥ /
yad vaktuṃ na hi śaktāś ca vedāś catvāra eva ca // 1,4.12 //

varaṃ vṛṇuṣva bhadraṃ te subhadra dvijapuṅgava /
sarvaṃ dātum ahaṃ śakto yat te manasi vāñcchitam // 1,4.13 //

nārāyaṇavacaḥ śrutvā tam uvāca śiṣuḥ svayam /
punaḥ kampitasarvāṅgaḥ sāśrunetraḥ puṭāñjaliḥ // 1,4.14 //

subhadra uvāca /
dehi me kṛṣṇapādābje dṛḍhaṃ bhaktiṃ sudurlabham /
taddāsyaṃ tatpade vāsaṃ jarāmṛtyuharaṃ param // 1,4.15 //

anyaṃ varaṃ na gṛhṇāmi na me kiṃcit prayojanam /
nāhaṃ varārthī kāmī ca rāgī vetanabhug yathā // 1,4.16 //

nārāyaṇarṣir uvāca /
śrīkṛṣṇo yasya bhaktiś ca tasyātra kiṃ sudurlabham /
aṇimādikadvatriṃśatsiddhiḥ karatale parā // 1,4.17 //

nirvikalpo dadāty asya naiva gṛhṇāti vaiṣṇavaḥ /
animittāṃ harer bhaktiṃ bhaktā vāñcchanti santatam // 1,4.18 //

gṛhāṇa mantraṃ kṛṣṇasya paraṃ kalpataruṃ varam /
bhaktidaṃ dāsyadaṃ śuddhaṃ karmamūlanikṛntanam // 1,4.19 //

lakṣmīr māyākāmabījaṃ ṅe 'ntaṃ kṛṣṇapadaṃ tathā /
vahnijāyāntamantraṃ ca mantrarājaṃ manoharam // 1,4.20 //

ity evam uktvā tatkarṇe kathayām āsa dakṣiṇe /
vāratrayaṃ muniśreṣṭhaḥ śuddhabhāvena putraka // 1,4.21 //

yena stotreṇa tuṣṭāva subhadraḥ parameśvaram /
ājñāṃ cakāra sa ṛṣis tad eva paṭhituṃ mudā // 1,4.22 //

kavacaṃ ca dadau tasmai jaganmaṅgalamaṅgalam /
dhyānaṃ ca sāmavedoktaṃ sarvapūjāvidhikramam // 1,4.23 //

harer dāsyaṃ ca tadbhaktir golokavāsam īpsitam /
janmadvayāntare caiva karmabhogakṣaye sati // 1,4.24 //

subhadra uvāca /
satyaṃ kuru mahābhāga varaṃ me yadi dāsyasi /
varaṃ vṛṇomi tat paścāt yan me manasi vāñcchitam // 1,4.25 //

nārāyaṇarṣir uvāca /
oṃ satyaṃ vatsa dāsyāmi varaṃ vṛṇu yathepsitam /
mamāśakyaṃ nāsti kiṃcit dātāhaṃ sarvasampadām // 1,4.26 //

subhadra uvāca /
kaṇṭhe te kiṃ ca kavacaṃ kasya vā sarvapūjitam /
amūlyaratnaguṭikāyuktaṃ ca sumanoharam // 1,4.27 //

kavacaṃ dehi me deva svasatyarakṣaṇaṃ kuru /
viprasya vacanaṃ śrutvā śuṣkakaṇṭhauṣṭhātālukaḥ // 1,4.28 //
vaktuṃ na śaktas tad vākyaṃ dadhyau kṛṣṇapadāmbujam /
pradadau guṭikāṃ tasmai novāca kavacaṃ muniḥ // 1,4.29 //

tam uvāca maharṣiś ca vituṣṭaś conmanāḥ suta /
vatsa krodho hi devasya varaṃ tulyaṃ ca vāñcchitam // 1,4.30 //

nārāyaṇarṣir uvāca /
triṃṣatsahasravarṣaṃ ca bhuṅkṣva rājyaṃ sudurlabham /
labhasva durlabhāṃ lakṣmīṃ māyayā mohito bhava // 1,4.31 //

madiṣṭadevakavacaṃ gṛhītaṃ yena hetunā /
saptakalpāntajīvasya paratra ca bhaviṣyati // 1,4.32 //

sucireṇaiva kālena golokaṃ ca prāyasyasi /
pare mṛkaṇḍuputras tvaṃ mārkaṇḍeyo bhaviṣyasi // 1,4.33 //

mayā dattaṃ ca kavacaṃ tvāṃ ca rakṣati putraka /
tava kaṇṭhe sthitiś cāśya prati janmani janmani // 1,4.34 //

punaś ca guṭikāyuktaṃ kṛtvā ca kavacaṃ muniḥ /
gale dadhāra bhaktyā ca tadbhakto dharmanandanaḥ // 1,4.35 //

varaṃ datvā ca sa munir yayau gehaṃ sa unmanāḥ /
viprāya kavacaṃ datvā naṣṭavatsā gaur yathā // 1,4.36 //

bhrātrā nareṇa pitrā ca dharmeṇa ca mahātmanā /
mātrā mūrtyā ca patnyā ca śāntyā ca bhartsito muniḥ // 1,4.37 //

vipraḥ samprāpya kavacaṃ mantraṃ kalpataruṃ param /
sarovarāt samutthāya prajvalan brahmatejasā // 1,4.38 //

kṣaṇaṃ tasthau sarastīre vaṭamūle manohare /
jajāpa paramaṃ mantraṃ sampūjya jagadīśvaram // 1,4.39 //

atha tattātavipro hi samanviṣya sutaṃ ciram /
gatvā ca svagṛhaṃ duḥkhī śokārtaḥ sa ruroda ca // 1,4.40 //

samudyatā tanuṃ tyaktuṃ tanmātā putravārtayā /
na tatyāja tanuṃ vipro dṛṣṭvā susvapnam uttamam // 1,4.41 //

vipro viprā gṛhaṃ tyaktvā putrānveṣaṇapūrvakam /
prayayau kānanaṃ ghoraṃ sarvaiś ca bāndhavaiḥ sahaḥ // 1,4.42 //

sarvaṃ vanaṃ samanviṣya prayayus te sarovaram /
dadṛśus te śiśuṃ gṛhyaṃ sūryābhaṃ vaṭamūlake // 1,4.43 //

cucumba gaṇḍaṃ putrasya vipro viprā ca sāradam /
āśiśleṣa krameṇaiva mātā tātaḥ punaḥ punaḥ // 1,4.44 //

putraś ca sarvavṛttāntaṃ kathayām āsa sādaram /
śrutvā putrasya vipraś ca viprā bāndhavas tathā // 1,4. 45 //

yayuḥ sarve svadeśaṃ ca paramāhlādamānasāḥ /
candrabhāgāṃ samuttīrya viveśa nagaraṃ param // 1,4.46 //

nagarastho nṛpendraś ca dṛṣṭvā tejasvinaṃ śiśum /
dadau tasmai svakanyāṃ ca ratnālaṅkārabhūṣitam // 1,4.47 //

yuvatīṃ sundarīṃ śyāmāṃ taptakañcanasaṃnibhām /
pativratāṃ mahābhāgāṃ sundarīṃ kamalākalāṃ // 1,4.48 //

gajendrāṇāṃ sahasraṃ ca pradadau yautukaṃ mudā /
aśvānāṃ daśalakṣaṃ ca ratnānāṃ ca sahasrakam // 1,4.49 //

dāsīnāṃ niṣkakaṇṭhīnāṃ saundarīṇāṃ sahasrakam /
vastraratnasahasraṃ ca bahumūlyaṃ sudurlabham // 1,4.50 //

dāsānāṃ ca sahasraṃ ca padātīnāṃ trilakṣakam /
daśalakṣaṃ suvarṇaṃ ca ratnamālāṃ sudurlabhām // 1,4.51 //

datvā tasmai ca kanyāṃ ca ruroda ca sabhāryakaḥ /
rājā ca kanyayā sārdhaṃ prayayau vipramandiram // 1,4.52 //

gatvā cāpi kiyad dūraṃ dadarśa nagaraṃ nṛpaḥ /
atīva sundaraṃ ramyaṃ vijitya cāmarāvatīm // 1,4.53 //

śuddhasphaṭikasaṃkāśaṃ ratnasāravinirmitam /
trikoṭicaṭṭālikāgehaṃ navakoṭisumandiram // 1,4.54 //

saptaprākārayuktaṃ ca parikhātrayasamyuktam /
durlaṅghyam atidurgamyaṃ ripūṇām api putraka // 1,4.55 //

śiśoś ca svāśramaṃ ramyaṃ sadratnasāranirmitam /
sphurat vajrakapāṭaṃ ca ratnendrakalaśānvitam // 1,4.56 //

sadratnadarpaṇāir dīpaṃ ratnakumbhair virājitam /
prāṅgaṇaṃ ratnasārāḍhyaṃ ratnasopānaśobhitam // 1,4.57 //

manoharaṃ rājamārgaṃ sindūrādipariṣkṛtam /
prākāraṃ maṇibhūṣāḍhyam uccair ākāśasparśi ca // 1,4.58 //

jagāma vismayaṃ rājā dṛṣṭvā nagaram uttamam /
pitrā mātrā saha śiśur vismayaṃ ca yayau mudā // 1,4.59 //

gajendrāṇāṃ trilakṣaṃ ca śvānāṃ śatalakṣakam /
caturguṇaṃ padātīnām āyayus te 'py anuvrajam // 1,4.60 //

vāraṇendraṃ puras kṛtya veśyāṃ ca nartakaṃ tathā /
dvijāṃś ca pūrṇakumbhāṃś ca patiputravatīṃ satīṃ // 1,4.61 //

mahāpātraḥ śiśuṃ dṛṣṭvā gajendroparisaṃsthitam /
mūrdhnā nanāma vegenāpy avaruhya gajād api // 1,4.62 //

śiśuṃ praveśayāṃ āsa ratnanirmāṇamandiram /
ratnasiṃhāsanaṃ tasmai pradadau sādaraṃ mudā // 1,4.63 //

kanyādātre ca pitre ca mātre ca sādaraṃ mudā /
ratnasiṃḥāsanaṃ ramyaṃ pradadau pātra eva ca // 1,4.64 //

śiśum siṣeva pātraś ca svayaṃ ca śvetacāmaraiḥ /
dadhāra ratnachatraṃ ca hīrāhārapariṣkṛtam // 1,4.65 //

uvāsa sa sabhāryāṃ ca sudharmyāṃ mahendravat /
śvasuraś ca yayau gehaṃ śiśunā ca puraskṛtaḥ // 1,4.66 //

triṃśatsahasravarṣaṃ ca rājā rājyaṃ cakāra saḥ /
kālāntare tatpitā ca vane vyāghreṇa bhakṣitaḥ // 1,4.67 //

pativratā mahābhāgā mātā sahamṛtā suta /
ratnayānena ramyeṇa sastrīkaḥ kṛṣṇamandiram // 1,4.68 //

prayayau sādaraṃ vipraḥ kṛṣṇanaivedyabhakṣaṇāt /
tadasthi bhuktvā vyāghraś ca pūtaḥ sadyaś ca sāṃpratam // 1,4.69 //

tābhyāṃ sārdhaṃ ca prayayau golokaṃ sumanoharam /
śiśur dehaṃ parityajya himādrau svarṇadītaṭe // 1,4.70 //

datvā putrāya rājyaṃ ca svargād api sudurlabham /
mṛkaṇḍupatnīgarbhe ca lebhe janma svakarmaṇā // 1,4.71 //

mārkaṇḍeyo muniśreṣṭho babhūva parajanmani /
saptakalpāntajīvī ca nārāyaṇavareṇa saḥ // 1,4.72 //

babhūva sāṃprataṃ vipraḥ kṛṣṇanaivedyabhakṣaṇāt /
śvabhakṣitaṃ ca naivedyaṃ bhuktvā ced īdṛśī gatiḥ /
akāmataś cāpy ajñāto jīrṇamārgasthitaṃ suta // 1,4.73 //
yo bhakṣet kāmato jñāto nityaṃ naivedyam īpsitam /
na jānanti gatis tasya vedāś catvāra eva ca // 1,4.74 //

iti te kathitaṃ brahmann itihāsaṃ purātanam /
āścaryaṃ madhuraṃ ramyaṃ kiṃ bhūyaḥ śrotum icchasi // 1,4.75 //

śrīnārada uvāca /
śrutaṃ naivedyamāhātmyam atīva sumanoharam /
īśvarasyāpi he tāta kṛṣṇasya paramātmanaḥ // 1,4.76 //

adhunā śrotum icchāmi svātmasandehabhañjanam /
nārāyaṇarṣer kaṇṭhe ca kavacaṃ tasya tad vada // 1,4.77 //

||atha kavacapraśnaḥ

sanatkumāra uvāca /
mamāpy astīti sandeho vacane prapitāmaha /
kasya tat kavacaṃ brahmann idaṃ vaktuṃ tvam arhasi // 1,4.78 //

sa pitā sa guruḥ svacchaḥ karoti bhramabhañjanam /
śīghraṃ brūhi mahābhāga nāradaṃ māṃ sutapriya // 1,4.79 //

putrayoś ca vacaḥ śrutvā śuṣkakaṇṭhauṣṭhatālukaḥ /
uvāca vacanaṃ brahmā smaran kṛṣṇapadāmbujam // 1,4.80 //

brahmovāca /
nārāyaṇena muninā jaganmaṅgalamaṅgalam /
viprāya kavacaṃ dattam dhyānaṃ ca parmātmanaḥ // 1,4.81 //

tad bravīmi mahābhāga tvām eva nāradaṃ prati /
kaṇṭhasthaṃ kavacaṃ vaktuṃ naiva śaknomi sāṃpratam // 1,4.82 //

matkaṇṭhe kavacaṃ yasya gopanīyaṃ sudurlabham /
nārāyaṇarṣikaṇṭhe ca tad eva paramādbhutam // 1,4.83 //

tad eva dharmakaṇṭhe ca narasya ca mahātmanaḥ /
agastyasya ca kaṇṭhe ca lomaśasya mahāmuneḥ // 1,4.84 //

tulasyāś cāpi saṃjñāyāḥ sāvitryāś cāpi putraka /
anyeṣāṃ ca bhāgyavatāṃ bhārate ca sudurlabhe // 1,4.85 //

nārada uvāca /
paścāt śroṣyāmi kavacaṃ jaganmaṅgalamaṅgalam /
dhyānaṃ pūjāṃ vidhānaṃ ca kṛṣṇasya paramātmanaḥ // 1,4.86 //

ādau kathaya bhadraṃ te paraṃ paramabhadrakam /
subhadraprāptaṃ kavacaṃ māhātmyaṃ yasya durlabham // 1,4.87 //

brahmovāca /
subhadraprāptaṃ kavacaṃ paścāt śroṣyasi putraka /
śaṅkarasya mukhād vipra svaguror jñāninas tathā // 1,4.88 //

||iti śrīnāradapañcarātre jñānāmṛtasāre brahmānāradasaṃvāde prathamaikarātre kavacapraśno nāma caturtho 'dhyāyaḥ

||prathamaikarātre pañcamo 'dhyāyaḥ

śrīsanatkumāra uvāca /
tavecchā yatra kavace dhyāne tad vada sāmpratam /
yac chṛṇomi śubhaṃ tac ca kena śreyasi tṛpyate // 1,5.1 //

brahmovāca /
dhyānaṃ sāmavedoktaṃ dattam nārāyaṇena vai /
kavacaṃ ca subhadrāya dharmiṣṭhāya mahātmane // 1,5.2 //

navīnajaladaśyāmaṃ pītakauśeyavāsasam /
candanokṣitasarvāṅgaṃ sasmitaṃ śyāmasundaram // 1,5.3 //

mālatīmālyabhūṣāḍhyaṃ ratnabhūṣaṇabhūṣitam /
munīndreśasusiddheśabrahmeśaśeṣavanditam // 1,5.4 //

sarvasvarūpaṃ sarveśaṃ sarvabījaṃ sanātanam /
sarvādyaṃ sarvajñaṃ puruṣaṃ prakṛteḥ param // 1,5.5 //

nirguṇaṃ ca nirīhaṃ ca nirliptam īsvaraṃ bhaje /
dhyātvā mūlena tasmai ca dadyāt pādyādikaṃ mudā // 1,5.6 //

tataḥ stotraṃ ca kavacaṃ bhaktyā ca prapaṭhen naraḥ /
japtvā ca mantraṃ bhaktyā daṇḍavat praṇamed bhuvi /
iti te kathitaṃ vatsa kiṃ bhūyaḥ śrotum icchasi // 1,5.7 //

śrīsanatkumāra uvāca /
brūhi me kavacaṃ brahman jaganmaṅgalamaṅgalam /
pūjyaṃ puṇyasvarūpaṃ ca kṛṣṇasya paramātmanaḥ // 1,5.8 //

||atha jaganmaṅgalamaṅgalakavacam

brahmovāca /
śṛṇu vakṣyāmi viprendra kavacaṃ paramādbhutaṃ /
śrīkṛṣṇenaiva kathitaṃ mahyaṃ ca kṛpayā parā // 1,5.9 //

mayā dattaṃ ca dharmāya tena nārāyaṇarṣaye /
ṛṣiṇā tena tad dattaṃ subhadrāya mahātmane // 1,5.10 //

atiguhyatamaṃ śuddhaṃ paraṃ snehād vadāmy aham /
yad dhṛtvā paṭhanāt siddhāḥ siddhāni prāpnuvanti ca // 1,5.11 //

evam indrādayaḥ sarve sarvaiśvaryam āpnuyuḥ /
ṛṣiś chandaś ca sāvitrī devo nārāyaṇaḥ svayam // 1,5.12 //

dharmārthakāmamokṣeṣu viniyogaḥ prakīrtitaḥ /
rādheśo me śiraḥ pātu kaṇṭhaṃ radheśvaraḥ // 1,5.13 //

gopīśaś cakṣuṣī pātu tālu ca bhagavān svayam /
gaṇḍayugmaṃ ca govindaḥ karṇayugmaṃ ca keśavaḥ // 1,5.14 //

galaṃ gadādharaḥ pātu skandhaṃ kṛṣṇaḥ svayaṃ prabhuḥ /
vakṣasthalaṃ vāsudevaś codaraṃ cāpi so 'cyutaḥ // 1,5.15 //

mattānāṃ pallavasthānāṃ mādhaveṣu manoharam // 1,7.29 //

śukānāṃ rājahaṃsānāṃ mayūrāṇāṃ ca putrakaiḥ /
kṣemakarīkhañjanānāṃ rājibhiś ca manoharam // 1,7.30 //

haritpītaraktakṛṣṇasupakvaphalapatrakaiḥ /
susnigdhākṣatapatraiś ca nūtanair abhibhūṣitam // 1,7.31 //

hiṃsābhayādirahitaṃ sarveṣāṃ paśupakṣiṇāṃ /
parasparaṃ ca suprītaṃ hiṃsrāṇāṃ kṣudrajantubhiḥ // 1,7.32 //

tatra krīḍāsthalaṃ ramyaṃ pārvatīparameśayoḥ /
munīndrair indranīlaiś ca padmarāgaiḥ pariṣkṛtam // 1,7.33 //

krośāyataṃ parimitaṃ vartulaṃ candravimbavat /
amlānarambhāstambhānāṃ lakṣalakṣaiś ca veṣṭitam // 1,7.34 //

citritaṃ sūkṣmasūtrāktair nūtanair abhibhūṣitam /
nūtanākṣatapatraiś ca lalitaiḥ pariśobhitam // 1,7.35 //

raktapītāsitaiḥ snigdhair amlānaiḥ sumanoharaiḥ /
paritaḥ paritaḥ śaśvan mālājālair vibhūṣitam // 1,7.36 //

śayyābhūtaṃ sutalpaiś ca snigdhacampakacandanaiḥ /
puṣpacandanayuktena vāyunā surabhīkṛtam // 1,7.37 //

kastūrīkuṅkamāsaktasugandhicandanaiḥ sitaiḥ /
mārjitaṃ citritaṃ citraiḥ parito raṅgavastubhiḥ // 1,7.38 //

dṛṣṭvā tad adbhutaṃ śīghraṃ prayayau svarṇadīṃ muniḥ /
śuddhasphaṭikasaṃkāsāṃ sarvapāpavināśinīm // 1,7.39 //

bhavābdhighorataraṇe taraṇīṃ nityanūtanām /
kṛṣṇapādaprasūtāṃ ca jagatpūjyāṃ pativratāṃ // 1,7.40 //

snāṭvā kṛṣṇaṃ ca saṃpūjya paramātmānam īśvaram /
prakṛteḥ parimiṣṭaṃ ca nirliptaṃ nirguṇaṃ param // 1,7.41 //

sākṣiṇaṃ karmaṇām eva brahma jyotiḥ sanātanam /
prayayau purato ramyaṃ rājamārgaṃ dadarśa saḥ // 1,7.42 //

maṇibhiḥ sphaṭikākārair amalair bahumūlyakaiḥ /
pariṣkṛtaṃ ca sarvatra nirmitaṃ viśvakarmaṇā // 1,7.43 //

satāṃ puṇyavatāṃ dṛṣṭam adṛṣṭaṃ kṛtapāpinām /
dhanuḥ śataṃ parimitaṃ citrarājivirājitam // 1,7.44 //

dadhyaṃ sarvāśramāntaṃ ca prakhyāt koṭiguṇottaram /
rathaṃ dadarśa purato manoyāyi manoharam // 1,7.45 //

amūlyaratnanirmāṇavimāṇasārasundaram /
dhanur lakṣaṃ parimitaṃ parito vartulākṛtam // 1,7.46 //

ūrdhvasthitam ūrdhvagaṃ ca sahasracakrasaṃyutam /
dhanur lakṣo 'pi sūtaṃ ca vahniśuddhāṃśukānvitam // 1,7.47 //

hīrāsāravinirmāṇaṃ sucārukalaśojjvalam /
ratnapradīpadīptāḍhyaṃ ratnadarpaṇabhūṣitam // 1,7.48 //

muktāśuktinibaddhaiś ca śobhitaṃ śvetacāmaraiḥ /
māṇikyasārahāreṇa maṇirājair virājitam // 1,7.49 //

pārijāṭaprasūtānāṃ māyājālaiḥ pariṣkṛtam /
grīṣmamadhyāhnamārtaṇḍaṃ sahasrasadṛśojjvalam // 1,7.50 //

īśvarecchāvinirmāṇaṃ kāmapuraṃ ca kāminām /
sarvabhogasamāviṣṭaṃ kalpavṛkṣaparaṃ varam // 1,7.51 //

sasaktacitritai ramyai ratimandirasundaraiḥ /
golokād āgataṃ pūrvaṃ krīḍārthaṃ śaṅkarasya ca // 1,7.52 //

vivāhe pariniṣpanne pārvatīparameśayoḥ /
rathaṃ dṛṣṭvā ca prayayau kiyad dūraṃ mahāmuniḥ // 1,7.53 //

atīva ramyaṃ ruciraṃ dadarśa śaṅkarāśramam /
ratnendrasāranirmāṇaṃ śiviraiḥ śatakoṭibhiḥ // 1,7.54 //

mitais tasmāt śataguṇais tatra sundaramandiraiḥ /
yuktaṃ ratnakapāṭaiś ca ratnadhātuvicitritaiḥ // 1,7.55 //

paramastambhasopānair vajramiśrair vibhūṣitam /
dadarśa śiviraṃ śambhoḥ parikhābhis tribhir yutam // 1,7.56 //

durlaghyābhirāmitrāṇāṃ sugamyābhiḥ satām aho /
prakāraiś ca tribhir yuktaṃ dhanur lakṣocchritaṃ suta // 1,7.57 //

sammitaṃ saptabhir dvārair nānārakṣakarakṣitaiḥ /
dhanuḥśatasahasraṃ ca caturasraṃ ca sammitam // 1,7.58 //

amūlyaratnanirmāṇaṃ catuḥśālāśatair yutam /
atīva ramyaṃ purato puradvāraṃ dadarśa saḥ // 1,7.59 //

purato ratnabhittau ca kṛtrimaṃ ca suśobhitam /
puṇyaṃ vṛndāvanaṃ ramyaṃ tanmadhye rāsamaṇḍalam // 1,7.60 //

sarvatra rādhākṛṣṇaṃ ca pratyekaṃ ratimandire /
ramyaṃ kuñjakuṭīrāṇāṃ sahasraṃ sumanoharam // 1,7.61 //

sugandhi puṣpaśayyānāṃ sahasraṃ candanokṣitam /
dvārapālaṃ ca tatraiva maṇibhadraṃ bhayaṃkaram // 1,7.62 //

triśūlapaṭṭiśadharaṃ vyāghracarmāmbaraṃ param /
taṃ saṃbhāṣya vilokyaivaṃ dvitīyadvāram īpsitam // 1,7.63 //

jagāma ca muniśreṣṭho dadarśa citram uttamam /
kadambānāṃ samūhaṃ ca tanmūlaṃ ca manoharam // 1,7.64 //

ratnabhittisamāyuktaṃ kālindīkūlam uttamam /
snātaṃ gopīsamūhaṃ ca nagnasarvāṅgam adbhutam // 1,7.65 //

kadambāgre ca śrīkṛṣṇaṃ vastrapuñjakaraṃ param /
tatraiva śūlahastaṃ ca mahākālaṃ dadarśa ca // 1,7.66 //

kṛpāluṃ dvārapālaṃ taṃ saṃbhāṣya nārado muniḥ /
prayayau śīghragāmī sa tṛtīyadvāram uttamam // 1,7.67 //

dadarśa tatra purataḥ kṛtrimaṃ vaṭamūlakam /
gopānāṃ ca samūhaṃ ca pītambaradharaṃ param // 1,7.68 //

bālakrīḍāṃ ca kurvantaṃ tanmadhye kṛṣṇam uttamam /
brāhmaṇībhiḥ pradattam ca bhuktavantaṃ supāyasam // 1,7.69 //

kurvantaṃ ca samādhānaṃ muneḥ vāmakareṇa ca /
gṛhītvā tadanujñāṃ ca caturthaṃ dvāram īpsitam // 1,7.70 //

prayayau brahmaputraś ca dadarśa citram uttamam /
govardhanaṃ parvataṃ ca tatra kṛṣṇakarasthitam // 1,7.71 //

gokulaṃ gokulasthānāṃ gopīnāṃ caiva rakṣaṇam /
vyākulaṃ gokulaṃ bhītaṃ śakravṛṣṭibhayena ca // 1,7.72 //

abhayaṃ dattavantaṃ ca kṛṣṇaṃ dakṣakareṇa ca /
nandinaṃ dvārapālaṃ ca śūlahastaṃ ca sasmitam // 1,7.73 //

vilokya prayayau vipraḥ pañcamaṃ dvāṛam uttamam /
nānākṛtimacitrāḍhyaṃ vīrabhadrānvitaṃ param // 1,7.74 //

tatraiva nīpamūlaṃ ca yamunākulam eva ca /
kālīyadamanaṃ tatra kṛtimaṃ ca dadarśa ḥa // 1,7.75 //

tad dṛṣṭvā sasmitas tuṣṭaḥ ṣaṣṭadvāṛaṃ jagāma saḥ /
dvāre niyuktaṃ bālaṃ ca śūlahastaṃ caturbhujam // 1,7.76 //

ratnasiṃhāsanasthaṃ ca sasmitaṃ svagaṇādhipam /
dadarśa citraṃ tatraiva mathurāgamanaṃ hareḥ // 1,7.77 //

gopikānāṃ vilāpaṃ ca yaśodānandayos tathā /
vyākulaṃ gokulaṃ cāpi rathasthaṃ śaraṇaṃ harim // 1,7.78 //

akrūraṃ ca tathā nandaṃ nirānandaṃ śucākulam /
tad dṛṣṭvā saptamadvāraṃ dvārapālaṃ dadarśa saḥ // 1,7.79 //

citraṃ kautukayuktaṃ ca mathurāyāḥ praveśanam /
sabalaṃ gopasahitaṃ śrīkṛṣṇaṃ prakṛteḥ param // 1,7.80 //

mathurānāgarībhiś ca bālakair vānirargalaiḥ /
vīkṣantaṃ sādaraṃ sarvair nagarasthair manoharam // 1,7.81 //

dhanur bhaṅgaṃ tathā śaṃbhoḥ kaṃsādinidhanādikam /
sabhāryaṃ vasudevaṃ ca nigaḍān muktam īpsitam // 1,7.82 //

dvāre niyuktaṃ deveśaṃ gaṇeśaṃ gaṇasaṃyutam /
dhyānasthaṃ ca vibhāntaṃ ca śuddhasphaṭikamālayā // 1,7.83 //

japantaṃ paramaṃ śuddhaṃ brahmajyotiḥ sanātanam /
nirliptaṃ nirguṇaṃ kṛṣṇaṃ paramaṃ prakṛteḥ param // 1,7.84 //

dṛṣṭvā taṃ ca suraśreṣṭhaṃ muniśreṣṭho 'pi nāradaḥ /
sāmavedoktastotreṇa puṣṭāva parameśvaram /
sāśrunetraḥ pulakito bhaktinamrātmakaṃdharaḥ // 1,7.85 //

||atha gaṇapatistotram

bho gaṇeśa suraśreṣṭha lambodara parāt para /
heramba maṅgalārambha gajavaktra trilocana // 1,7.86 //

muktida śubhada śrīda śrīdharasmaraṇe rata /
paramānanda parama pārvatīnandana svayam // 1,7.87 //

sarvatra pūjya sarveśa jagatpūjya mahāmate /
jagadguro jagannātha jagadīśa namo 'stu te // 1,7.88 //

yatpūjā sarvaparato yaḥ stutaḥ sarvayogibhiḥ /
yaḥ pūjitaḥ surendraiś ca munīndrais taṃ namāmy aham // 1,7.89 //

paramārādhanenaiva kṛṣṇasya paramātmanaḥ /
puṇyakena vratenaiva yaṃ prāpa pārvatī satī // 1,7.90 //

taṃ namāmi suraśreṣṭhaṃ sarvaśreṣṭhaṃ gariṣṭhakam /
jñāniśreṣṭhaṃ variṣṭhaṃ ca taṃ namāmi gaṇeśvaram // 1,7.91 //

ity evam uktvā devarṣis tatraivāntar dadhe vibhuḥ /
nāradaḥ prayayau śīghram īśvarābhyantaraṃ mudā // 1,7.92 //

idaṃ lambodarastotraṃ nāradena kṛtaṃ purā /
pūjākāle paṭhen nityaṃ jayas tasya pade pade // 1,7.93 //

saṃkalpitaṃ paṭhed yo hi varṣam ekaṃ susaṃyataḥ /
viśiṣṭaputraṃ labhate paraṃ kṛṣṇaparāyaṇam // 1,7.94 //

yaśasvinaṃ ca vidvāṃsaṃ dhaninaṃ cirajīvinam /
vighnanāśo bhavet tasya mahaiśvaryaṃ yaśo 'malam /
ihaiva ca sukhaṃ bhaktyānte yāti hareḥ padam // 1,7.95 //

||iti śrīnāradapañcarātre jñānāmṛtasāre prathamaikarātre gaṇapatistotraṃ nāma saptamo 'dhyāyaḥ

||prathamaikarātre aṣṭamo 'dhyāyaḥ

śrīvyāsa uvāca /
atha cābhyantaraṃ gatvā nārado hṛṣṭamānasaḥ /
dadarśa svāśramaṃ ramyam atīva sumanoharam // 1,8.1 //

payaḥ phenanibhaśayyāsahitaṃ ratnamandiram /
sākṣād gorocanābhaiś ca maṇistambhair vibhūṣitam // 1,8.2 //

maṇīndrasārasopānaiḥ kapāṭaiś ca pariṣkṛtam /
muktāmāṇikyahīrāṇāṃ mālārājivirājitam // 1,8.3 //

śuddhasphaṭikasaṃkāśaṃ prāṅgaṇaṃ maṇisaṃskṛtam /
sundaraṃ mandiracayaṃ sadratnakalaśojjvalam // 1,8.4 //

ratnapatrapaṭākīrṇaṃ vahniśuddhāṃśukānvitam /
sudhānāṃ ca madhūnām ca pūrṇakumbhakaṃ śataṃ śatam // 1,8.5 //

dāsadāsīsamūhaiś ca ratnālaṅkārabhūṣitaiḥ /
pārvatīpriyasaṅgaiś ca svakarmākulasaṅkalpam // 1,8.6 //

tad dṛṣṭvā ca muniśreṣṭhas tat parābhyantaraṃ yayau /
ratnasiṃhāsanasthaṃ ca śaṅkaraṃ ca dadarśa saḥ // 1,8.7 //

vyāghracarmāmbaradharaṃ sasmitaṃ candraśekharam /
prasannavadanaṃ svacchaṃ śāntaṃ śrīmantam īśvaram // 1,8.8 //

vibhūtibhūṣitāṅgaṃ ca paraṃ gaṅgājaṭādharam /
bhaktapriyaṃ ca bhakteśaṃ jvalantaṃ brahmatejasā // 1,8.9 //

trinetraṃ pañcavaktraṃ ca koṭicandrasamaprabham /
japantaṃ paramātmānaṃ brahma jyotiḥ sanātanam // 1,8.10 //

nirliptaṃ ca nirīhaṃ ca dātāraṃ sarvasampadām /
svecchāmayaṃ sarvabījaṃ śrīkṛṣṇaṃ prakṛteḥ paraṃ // 1,8.11 //

siddhendraiś ca munīndraiś ca devendraiḥ parisevitam /
pārśvadapravaraśreṣṭhasevitaṃ śvetacāmaraiḥ // 1,8.12 //

durgāsevitapādābjaṃ bhadrakālīpariṣṭutam /
purato hi vasantaṃ taṃ skandaṃ gaṇapatīm tathā // 1,8.13 //

gale baddhvā ca vasanaṃ bhaktinamrātmakaṃdharaḥ /
yogīndraṃ svaguruṃ śaṃbhuṃ śirasā praṇanāma saḥ // 1,8.14 //

tuṣṭāva parayā bhaktyā devarṣir jagatāṃ patim /
svaguruṃ ca paśupatiṃ vedoktena stavena ca // 1,8.15 //

śrīnārada uvāca /
namas tubhyaṃ jagannātha mama nātha mama prabho /
bhavarūpataror bīja phalarūpa phalaprada // 1,8.16 //

abījaja praja prāja sarvabīja namo 'stu te /
sadbhāva paramābhāva vibhāva bhāvanāśraya // 1,8.17 //

bhaveśa bhavabandheśa bhāvābdhināvināyaka /
sarvādhāra nirādhāra sādhāra dharaṇīdhara // 1,8.18 //

vedavidyādhārādhāra gaṅādhara namo 'stu te /
jayeśa vijayādhāra jayabīja jayātmaka // 1,8.19 //

jagadāde jayānanda sarvānanda namo 'stu te/

ity evam uktvā devarṣiḥ śambhoś ca purataḥ sthitaḥ /
prasannavadanaḥ śrīmān bhagavāṃs tam uvāca saḥ // 1,8.20 //

śrīmahādeva uvāca /
varaṃ vṛṇu mahābhāga yat te manasi vartate /
dāsyāmi tvāṃ dhruvaṃ putra dātāhaṃ sarvasaṃpadām // 1,8.21 //

sukhaṃ muktiṃ harer bhaktiṃ niścalām avināśinīm /
hareḥ pādaṃ taddāsyaṃ sālokyādicatuṣṭayam // 1,8.22 //

indratvam amaratvaṃ vā yamatvam anileśvaram /
prajāpatitvaṃ brahmatvaṃ siddhatvaṃ siddhasādhanam // 1,8.23 //

siddhaiśvaryaṃ siddhibījaṃ vedavidyādhipaṃ param /
aṇimādikasiddhiṃ ca manoyāyitvam īpsitam // 1,8.24 //

hareḥ padaṃ ca gamanaṃ saśarīreṇa līlayā /
eteṣu vāñchitārtheṣu kiṃ vā te vāñchitaṃ suta // 1,8.25 //

tan me brūhi muniśreṣṭha sarvaṃ dātum ahaṃ kṣamaḥ /
śaṅkarasya vacaḥ śrutvā tam uvāca mahāmuniḥ // 1,8.26 //
śrīnārada uvāca /
dehi me haribhaktiṃ ca tannāmasevane ruciḥ /
atitṛṣṇā guṇākhyāne nityam astu mameśvara // 1,8.27 //

nāradasya vacaḥ śrutvā jahāsa śaṅkaraḥ svayam /
pārvatī bhadrakālī ca kārttikeyo gaṇeśvaraḥ // 1,8.28 //

sarvaṃ dadau mahādevo nāradāya ca dhīmate /
sarvapradas tu sarveśaḥ sarvakāraṇakāraṇaḥ // 1,8.29 //

nāradena kṛtaṃ stotraṃ nityaṃ yaḥ prapaṭhet śuciḥ /
haribhaktir bhavet tasya tannāmni guṇato ruciḥ // 1,8.30 //

daśavārajapenaiva stotrasiddhir bhaven nṛṇām /
sarvasiddhir bhavet tasya siddhastotro bhaved yadi // 1,8.31 //

iha prāpnoti lakṣmīṃ ca niścalāṃ lakṣapauruṣīm /
paripūrṇam ahaiśvaryam ante yāti hareḥ padam // 1,8.32 //

putraṃ viśiṣṭaṃ labhate haribhaktaṃ jitendriyam /
susādhyāṃ suvinītāṃ suvratāṃ ca pativratām // 1,8.33 //

prajāṃ bhūmiṃ yaśaḥ kīrtiṃ vidyāṃ sakavitāṃ labhet /
prasūyate mahābandhyā varṣam ekaṃ śṛṇoti cet // 1,8.34 //

galatkuṣṭhī mahārogī sadyo rogāt pramucyate /
dhanī mahādaridraś ca kṛpaṇaḥ satyavān bhavet /
vipradgrasto rājabaddho mucyate nātra saṃśayaḥ // 1,8.35 //

||iti śrīnāradapañcarātre jñānāmṛtasāre prathamaikarātre aṣṭamo 'dhyāyaḥ

||prathamaikarātre navamo 'dhyāyaḥ

śrīvyāsa uvāca /
varaṃ datvā mahādevo bhaktyā taṃ brāhmaṇātithim /
pūjāṃ cakāra vedoktāṃ svayaṃ vedavidāṃ varaḥ // 1,9.1 //

bhuktvā pītvā muniśreṣṭho mahādevasya mandire /
tiṣṭhann upāsānāṃ cakre pārvatīparameśayoḥ // 1,9.2 //

ekadā cirakālānte tam uvāca mahāmunim /
mahādevaḥ sabhāmadhye kṛpayā ca kṛpānidhiḥ // 1,9.3 //

śrīmahādeva uvāca /
kiṃ vā te vāñchitaṃ vatsa brūhi māṃ yadi rocate /
varo dattaḥ kim aparaṃ yat te manasi vartate // 1,9.4 //

mahādevavacaḥ śrutvā tam uvāca mahāmuniḥ /
kailāse ca sabhāmadhye yat tanmanasi vāñchitam // 1,9.5 //

śrīnārada uvāca /
jñānam ādhyātmikaṃ nāma vedasāraṃ manoharam /
haribhaktipradaṃ jñānaṃ muktidaṃ jñānam īpsitam // 1,9.6 //

yogayuktaṃ ca yaj jñānaṃ jñānaṃ yat siddhidaṃ tathā /
saṃsāraviṣayajñānam eva pañcavidhaṃ smṛtam // 1,9.7 //

āśramāṇāṃ samācāraṃ teṣāṃ dharmapariṣkṛtam /
vidhavānāṃ ca bhikṣūṇāṃ yatīnāṃ brahmacāriṇāṃ // 1,9.8 //

pūjāvidhānaṃ kṛṣṇasya tatstotraṃ kavacaṃ manum /
puraścaryāvidhānaṃ ca sarvāhnikam abhīpsitam // 1,9.9 //

jīvakarmavipākaṃ ca karmamūlanikṛntanam /
saṃsāravāsanāṃ kāṃ vā lakṣaṇaṃ prakṛtīśayoḥ // 1,9.10 //

tayoḥ paraṃ vā kiṃ vastu tasyāvatāravarṇamam /
ko vā tadaṃśaḥ kaḥ pūrṇaḥ paripūrṇatamaś ca kaḥ // 1,9.11 //

nārāyaṇarṣikavacaṃ subhadrapravarāya ca /
yad dattaṃ kiṃ tad deveśa tad ārādhyaṃ prayatnataḥ // 1,9.12 //

mayā jñānaṃm anāpṛṣṭhaṃ yad yad asti surottama /
tan me kathaya tattvena mām evānugrahaṃ kuru // 1,9.13 //

guroś ca jñānodgiraṇāj jñānaṃ syān mantratantrayoḥ /
tat tantraṃ sa ca mantraḥ syād yato bhaktir adhokṣaje // 1,9.14 //

jñānaṃ syād viduṣāṃ kiṃcid vedavyākhyānacintayā /
svayaṃ bhavān vedakartā jñānādhiṣṭhātṛdevatā // 1,9.15 //

nāradasya vacaḥ śrutvā sasmitaḥ pārvatīpatiḥ /
nirīkṣya pārvatīvaktraṃ gajavaktram uvāca saḥ // 1,9.16 //

||atha nāradopadeśagrahaṇam

śrīmahādeva uvāca /
aho anantadāsānāṃ māhātmyaṃ paramādbhutam /

sa snātaḥ sarvatīrtheṣu sarvayajñeṣu dīkṣitaḥ /
phalaṃ sa lebhe pūjānāṃ vratī sarvavrateṣu ca // 1,9.26 //

viṣṇumantraṃ yo labhet vaiṣṇavāc ca dvijottamāt /

vaiṣṇavād darśanenaiva sparśanena ca pārvati /

darśanaṃ vaiṣṇavānāṃ ca devā vāñchanti nityaśaḥ /

ity uktvā saṅkaraḥ śīghraṃ nāradena sahātmajaḥ /
yayau mandākinītīraṃ nīraṃ kṣīropamaṃ param // 1,9.31 //

tatra snāto mahādevī nāradaś ca mahāmuniḥ /
samācāntaḥ śucis tatra dhṛtvā dhaute ca vāsavī // 1,9.32 //

kṛṣṇamantraṃ dadau tasmai nāradāya maheśvaraḥ /
paraṃ kaplataruvaraṃ sarvasiddhipradaṃ śuka // 1,9.33 //

lakṣmīr māyākāmabījaṃ ṅentaṃ kṛṣṇapadaṃ tataḥ /
jagatpūtapriyāntaṃ ca mantrarājaṃ prakīrtitam // 1,9.34 //

mantraṃ gṛhītvā sa muniḥ śivaṃ kṛtvā pradakṣiṇam /
sapta vārān namas kṛtya svātmānaṃ dakṣiṇāṃ dadau // 1,9.35 //

tatpādapadme vikrītamājanma mastakaṃ param /
muninā bhaktiyuktena svargamandākinītaṭe // 1,9.36 //

etasminn antare vatsa puṣpavṛṣṭir babhūva ha /
nāradopari tatraiva suśrāva dundubhir muniḥ // 1,9.37 //

nanarta brahmaṇaḥ putro brahmaloke nirāmaye /
brahmā jagāma tatraiva suprasannaś ca sasmitaḥ // 1,9.38 //

putraṃ śubhāśiṣaṃ kṛtvā tuṣṭāva candraśekharam /
śambhuś ca pūjayām āsa brāhmaṇam atithiṃ tathā /
śambhuṃ śubhāśiṣaṃ kṛtvā brahmalokaṃ yayau vidhiḥ // 1,9.39 //

||iti śrīnāradapañcarātre jñānāmṛtasāre prathamaikarātre nāradopadeśagrahaṇaṃ nāma navamo 'dhyāyaḥ

||prathamaikarātre daśamo 'dhyāyaḥ

śrīśuka uvāca /
nārado hi mahājñānī devarṣir brahmaṇaḥ sutaḥ /
sarvavedavidāṃ śreṣṭho gariṣṭhaś ca variṣṭhakaḥ // 1,10.1 //

kathaṃ sa nopadiṣṭaś ca jñānahīno mahāmuniḥ /
tan māṃ bodhaya vibho sandehabhañjanaṃ kuru // 1,10.2 //

śrīvyāsa uvāca /
nārado brahmaṇaḥ putraḥ purākalpe babhūva saḥ /
sarvajñānaṃ dadau tasmai vidhātā jagatām api // 1,10.3 //

vedāṃś ca pāṭhayām āsa vedāṅgāny api suvrata /
siddhavidyāṃ śilpavidyāṃ yogaśāstraṃ purāṇakam // 1,10.4 //

bhagavān ekadā putraṃ kathayām āsa saṃsadi /
sṛṣṭiṃ kuru mahābhāga kṛtvā dāraparigraham // 1,10.5 //

brahmaṇaś ca vacaḥ śrutvā koparaktāsyalocanaḥ /
uvāca pitaraṃ kopāt paraṃ kṛṣṇaparāyanaḥ // 1,10.6 //

śrīnārada uvāca /
sarveṣām api vandyānāṃ pitā caiva mahāguruḥ /
jñānadātuḥ paro vandyo na bhūto na bhaviṣyati // 1,10.7 //

stanadātrī garbhadhātrī snehakartrī sadāmbikā /
janmadātānnadātā syāt snehakartā pitā sadā // 1,10.8 //

na kṣamau tau ca pitarau putrasya karma khaṇḍitum /
karoti sadguruḥ śiṣyakarmamūlanikṛntanam // 1,10.9 //

guruś ca jñānodgiraṇāt jñānaṃ syān mantratantrayoḥ /
tat tantraṃ sa ca mantraś ca kṛṣṇabhaktir yato bhavet // 1,10.10 //

śrīkṛṣṇavimukho bhūtvā viṣaye yasya mānasam /
viṣam atyamṛtaṃ tyaktvā sa ca mūḍho narādhamaḥ // 1,10.11 //

sa guruḥ sa pitā vandyaḥ sā mātā sa patiḥ sutaḥ /
yo dadāti harau bhaktiṃ karmamūlanikṛntanī // 1,10.12 //

śrīkṛṣṇabhajanaṃ tāta sarvamaṅgalamaṅgalam /
karmopabhogarogaāṇām auṣadhaṃ tannikṛntanam // 1,10.13 //

aho jagadvidhātuś ca dharmaśāstur iyaṃ matiḥ /
svayaṃ māyāmohitaś ca paraṃ bhraṣṭaṃ karoti ca // 1,10.14 //

viṣṇus tvāṃ mohitām kṛtvā yuyoja sraṣṭum īśvaraḥ /
na dadau svātmabhaktiṃ tāṃ svadāsyaṃ cātidurlabham // 1,10.15 //

mātā dadāti putrāya modakaṃ kṣunnivārakam /
sa ca bālo na jānāti kathaṃbhūtaṃ ca modakam // 1,10.16 //

bālakaṃ vañcanaṃ kṛtvā miṣṭaṃ dravyaṃ pradāya saḥ /
pitā prayāti kāryārthaṃ viṣṇunā mohitas tathā // 1,10.17 //

saṃsārakūpapatito viṣṇunā prerito bhavān /
na yuktaṃ patanaṃ tatra tad uddhāram abhīpsitam // 1,10.18 //

jñānī guruś ca balavān bhavābdheḥ śiṣyam uddharet /
guruḥ svayam asiddhaś ca durbalaḥ katham uddharet // 1,10.19 //

guror atyavaliptasya kāryākāryam ajānataḥ /
utpathapratipannasya parityāgo vidhīyate // 1,10.20 //

sa guruḥ paramo vairī yo dadāti hy asanmatim /
taṃ namaskṛtya satśiṣyaḥ prayāti jñānadaṃ gurum // 1,10.21 //

saṃsāraviṣayonmatto gururārtaḥ svakarmaṇi /
durbalo durvahaṃ bhāraṃ dadāti janakāya ca // 1,10.22 //

nāradasya vacaḥ śrutvā kruddhaḥ putram uvāca saḥ /
kampitas tamasā dhātā koparaktāsyalocanaḥ // 1,10.23 //

brahmovāca /
jñānaṃ te bhavatu bhraṣṭaṃ strījito bhava pāmara /
sarvajātiṣu gandharvaḥ kāmī so 'pi bhavān bhava // 1,10.24 //

pañcāśatkāminīnāṃ ca svayaṃ bhartā bhavācirāt /
tāsāṃ vaśaś ca satataṃ strīṇāṃ krīḍā mṛgo yathā // 1,10.25 //

śṛṅgāraśūro bhava re śaśvatsusthirayauvanaḥ /
tāsāṃ nityaṃ yauvanānāṃ sundarīṇāṃ priyo bhava // 1,10.26 //

kāmabādhyo bhava ciraṃ divyavarṣasahasrakam /
nirjane nirjane ramye vane krīḍāṃ kariṣyasi // 1,10.27 //

tato varṣasahasrānte mayā śaptaḥ svakarmaṇā /
vipradāsyāṃ tu śūdrāyāṃ janiṣyasi na saṃśayaḥ // 1,10.28 //

tato vaiṣṇavasaṃsargāt viṣṇor ucchiṣṭabhojanāt /
viṣṇumantraprasādena viṣṇumāyāvimohitaḥ // 1,10.29 //

tātasya vacanaṃ śrutvā cukopa nārado muniḥ /
śaśāpa pitaraṃ śighraṃ dāruṇaṃ ca yathocitam // 1,10.30 //

apūjyo bhava duṣṭaṃ tvaṃ tvanmantropāsakaḥ kutaḥ /
agamyāgamanecchā te bhaviṣyati na saṃśayaḥ // 1,10.31 //

nāradasya tu śāpena so 'pūjyo jagatāṃ vidhiḥ /
dṛṣṭvā svakanyārūpaṃ ca paścād dhāvitavān purā // 1,10.32 //

punaḥ svadehaṃ tatyāja bhartsitaḥ sanakādibhiḥ /
lajjitaḥ kāmayuktaś ca punar brahmā babhūva saḥ // 1,10.33 //

nāradas tu namaskṛtya pitaraṃ kamalodbhavam /
vipradehaṃ parityajya gandharvaś ca babhūva saḥ // 1,10.34 //

navayauvanakālena balavān madanoddhataḥ /
jahāra kanyāḥ pañcāśat balāc citrarathasya tu // 1,10.35 //

gāndharvena vivāhena tā uvāha ca nirjane /
mūrcchāṃ prāpuś ca tāḥ kanyā dṛṣṭvā sundaram īśvaram // 1,10.36 //

visaspuruś ca pitaraṃ mātaraṃ bhrātaraṃ tathā /
remire tena sārdhaṃ ca kāmukyaḥ kāmukena ca // 1,10.37 //

kandarer kandare ramye ramye sundaramandire /
śaile śaile surahasi kānane kānane tathā // 1,10.38 //

puṣpodyane tarudyāne nadyāṃ nadyāṃ nade nade /
saraḥśreṣṭhe saraḥśreṣṭhe vare candrasarovare // 1,10.39 //

sureśasyāpi nikaṭe subhadrasya taṭe taṭe /
agamye ca mahāghore gandhamādanagahvare // 1,10.40 //

parijātataruṇāṃ ca puṣpitānāṃ manohare /
tadantare sundare cāmodite puṣpavāyunā // 1,10.41 //

malaye nilaye ramye sugandhe candanānvite /
candanokṣitasarvāṅgaś candanāktena kāminā // 1,10.42 //

ramyacampakaśayyāsu candanāktāsu sasmitāḥ /
divāniśaṃ na jānanti kāminā sasmitena ca // 1,10.43 //

visyandake śūrasene nandane puṣpabhadrake /
svāhāvane kāmyake ca ramyake pāribhadrake // 1,10.44 //

surandhake gandhake ca suraṅhre puṇḍrake 'pi ca /
kālañjare pañjare ca kāñcīkāñcanakānane // 1,10.45 //

madhumādhavamāse ca madhūre madhukānane /
vane kalpatarūṇāṃ ca viśvakārukṛtasthale // 1,10.46 //

ratnākarāṇāṃ nikare sundare sundarāntare /
suvele ca supārśve ca pravālāṃkurakānane // 1,10.47 //

mandāre mandire pūre gāndhāre ca yugandhare /
vane kelikadambānāṃ ketakīnāṃ manohare // 1,10.48 //

mādhavīmālatīnāṃ ca yūthikānāṃ vane vane /
campakānāṃ palāśānāṃ kundānāṃ vipine tathā // 1,10.49 //

nāgeśvaralavaṅgānāṃ antare lalitālaye /
kumudānāṃ paṅkajānāṃ paṅkile komalasthale // 1,10.50 //

sthalapadmaprakāro ca bhūmicampakakānane /
lāṅgalīnāṃ rasālānāṃ panasānāṃ sukhaprade // 1,10.51 //

kadalībadarīṇāṃ ca śrīphalānāṃ ca śrīyute /
jambīrāṇāṃ ca jambūnāṃ karañjānāṃ tathaiva ca // 1,10.52 //

kṛtvā bihāraṃ tābhiś ca gandharvaś copavarhaṇaḥ /
divyaṃ varṣasahasraṃ ca svāśramaṃ punar āyayau // 1,10.53 //

śrutvā vidhātur āhvānaṃ puṣkaraṃ ca yayau punaḥ /
dadarśa tatra brahmāṇam ratnasiṃhāsanasthitam // 1,10.54 //

devendraiś cāpi siddhendrair munīndraiḥ sankādibhiḥ /
samāvṛtaṃ sabhāyāṃ ca rakṣogandharvakinnaraiḥ // 1,10.55 //

suśobhitaṃ yathā candraṃ gagane bhagaṇaiḥ saha /
praṇanāma sabhāmadhye tābhiḥ sārdhaṃ jagadvidhim // 1,10.56 //

maheśaṃ ca gaṇeśaṃ ca dhaneśaṃ śeṣam īśvaram /
dharmaṃ dhanvantariṃ skandaṃ sūryasomahutāśanam // 1,10.57 //

upendrendraṃ viśvakāruṃ varuṇaṃ pavanaṃ smaram /
yamam aṣṭau vasūn rudrān jayantaṃ nalakūvaram // 1,10.58 //

sarvān devān namaskṛtya nanāma munipuṅgavam /
agastyaṃ ca pulastyaṃ ca pulahaṃ ca pracetasam // 1,10.59 //

sarvaśreṣṭhaṃ vasiṣṭhaṃ ca dakṣaṃ ca kardamaṃ tathā /
sanakaṃ sanandaṃ ca tṛtīyaṃ ca sanātanam // 1,10.60 //

sanatkumāraṃ yogīśaṃ jñānināṃ ca guror gurum /
voḍhuṃ pañcaśikhaṃ saṅkhaṃ bhṛgum aṅgirasaṃ tathā // 1,10.61 //

āsuriṃ kapilaṃ kautsaṃ kratuṃ nārāyaṇaṃ naram /
marīciṃ kaśyapaṃ kaṇvaṃ vyāsaṃ durvāsasaṃ kavim // 1,10.62 //

bṛhaspatiṃ ca cyavanaṃ mārkaṇḍeyaṃ ca lomaśam /
vālmīkiṃ paraśurāmaṃ saṃvartaṃ ca vibhāṇḍakam // 1,10.63 //

devalaṃ ca vāmadevam ṛśayaśṛṅgaṃ parāśāram /

brahmovāca /

tad dinaṃ saphalaṃ dhanyaṃ yaśasyaṃ sarvamaṅgalam /
śrīkṛṣṇakīrtanaṃ yatra tatraiva nāyuṣo vyayaḥ // 1,10.71 //

hantā yo viprabhikṣūṇāṃ yatīnāṃ brahmacāriṇāṃ /

bhrūṇāghnaś cāpi goghnaś ca śūdraghnaś ca kṛtaghnakaḥ /

kṛṣṇasaṃkīrtnadhyānāt tanmantragrahaṇād aho /

tapoyajñakṛtī pūtas tīrthasnātavratī tathā /
bhikṣur yatir brahmacārī vānaprasthaś ca tāpasaḥ // 1,10.80 //

pavitraḥ paramo vahniḥ supavitraṃ jalaṃ yathā /

viṣṇupādodakocchiṣṭaṃ bhuñjate ye ca nityaśaḥ /

jīvanmuktās ca te dhanyā haridāsāś ca bhārate /

nahi teṣāṃ parābhūtāḥ puṇyavanto jagattraye /

teṣāṃ ca darśanaṃ sparśaṃ vāñchanti munayaḥ surāḥ /
puruṣāṇāṃ sahasraṃ ca pūtaṃ tajjanmamātrataḥ // 1,10.85 //

ity uktvā jagatāṃ dhātā tatra tūṣṇīṃ babhūva saḥ /
āścaryaṃ menire śrutvā devāś ca munayas tathā // 1,10.86 //

etasminn antare tatra vidyādharyaḥ samāgatāḥ /
gandharvāś cāpi vividhā nanṛtuḥ kinnarā jaguḥ // 1,10.87 //

rambhor vaśī ghṛtācī ca menakā ca tilottamā /
sudhāmukhī pūrṇacittī mohinī kalikā tathā // 1,10.88 //

samāyayus tathā mandā daivajñāḥ stutipāṭhakāḥ /
lakṣmī sarasvatī durgā sāvitrī rohiṇī ratiḥ // 1,10.92 //

tulasī pṛthivī gaṅgā svāhā ca yamunā tathā /

aho mahotsavaṃ paramānandamānasāḥ /
vicitrāṃ ca brahmasabhāṃ puṣkaraṃ tīrtham āyayuḥ // 1,10.94 //

||atha gandharvamokṣaṇam

śrībhagavān uvāca /
gandharvarājapravara sthiro bhava bhayaṃ tyaja /
śubhāśrayasya bhaktasya bhayaṃ kiṃ te mayi sthite // 1,13.8 //

sarvebhyo 'pi bhayaṃ nāsti madbhaktānām akarmaṇām /
janmamṛtyujarāvyādhibhayaṃ teṣāṃ na vidyate // 1,13.9 //

manmantropāsakaś caiva svatantro nityavigrahaḥ /
punar na vidyate janma mantragrahaṇamātrataḥ // 1,13.10 //

nāsti kālād bhayaṃ tasya na niṣekād vidher api /
mantragrahaṇamātreṇa mucyate sarvakarmaṇaḥ // 1,13.11 //

manmantro hi dehāt pāpaṃ koṭijanmakṛtaṃ ca yat /
sudīpto jvaladagniś ca tṛṇapuñjaṃ dehād yathā // 1,13.12 //

manmantragrahaṇād yogān mannāmagrahaṇasya vā /
teṣāṃ pāpāni vepante koṭijanmakṛtāni ca // 1,13.13 //

yamas tannāmalikhanaṃ dūrībhūtaṃ karoti ca /
ante dāsyaṃ ca labhate gatvā golokam uttamam // 1,13.14 //

yāvad āyur bhramet tāvat svatantro mattakuñjaraḥ /
tataḥ pāpāḥ phalāyante vainateyād ivoragāḥ // 1,13.15 //

teṣāṃ ca pādarajasā sadyaḥ pūtā vasundharā /
punāti sarvatīrthāni dūrato darśanād api // 1,13.16 //

pūtaś ca pavano vahnir jalaṃ ca tulasīdalam /
pūtāny eva hi tīrthāni gaṅgādīni ca gāyana // 1,13.17 //

pūtā suśīlā dharmiṣṭhā suvratā strī pativratā /
manmantropāsakāś caiva tebhyaḥ pūtottamāḥ sadā // 1,13.18 //

mantropāsakānāṃ ca tīrthasthānaṃ vrataṃ suta /
śrāddhaṃ dānaṃ pūjanaṃ ca yathā carvitacarvaṇam // 1,13.19 //

bhaktyā tīrthāni pūtāni svataḥ pūto hi vaiṣṇavaḥ /
tat tantraṃ ca tathā dānamalaṃ śrāddhaṃ ca niṣphalam // 1,13.20 //

śrāddhasya sampradānaṃ ca kartuś ca puruṣatrayam /
puruṣāṇāṃ śataṃ muktaṃ ko bhuṅkte śrāddhavastu ca // 1,13.21 //

kecid evaṃ vadantīti pitṛlokārtham eva ca /
tadviruddhaṃ ca te tuṣṭā mantragrahaṇamātrataḥ // 1,13.22 //

teṣāṃ śubhāśiṣaṃ karma naiva bhogāya kalpate /
devān na prabhaved vatsa siddhadhānye yathāṅkuraḥ // 1,13.23 //

sākṣāt karoti teṣāṃ ca karmamūlanikṛntanam /
mantropāsakād anye karmabhogaṃ ca bhuñjate // 1,13.24 //

mayā svayaṃ pradattaś ca svamantraḥ puruṣāya ca /
paradvārād grāhayitvā bhaktaṃ muktaṃ karomy aham // 1,13.25 //

mayā pradattamantraś ca purā mṛtyuñjayas tathā /
mṛtyuñjayāya goloke śuddhasattvaguṇāya ca // 1,13.26 //

punaḥ sanatkumārāya dharmāya brahmaṇe tathā /
kapilāya ca śeṣāya gaṇeśāya ca mahāmate // 1,13.27 //

nārāyaṇarṣaye caiva dharmaputrāya dhīmate /
punar mahāviṣṇave ca viśvāni yasya lomasu // 1,13.28 //

kālādhiṣṭhātṛdevāya tasmai sarvāntakāya ca /
upendrāya ca kāmāya bhṛgave 'ṅgirase tathā // 1,13.29 //

sarasvatyai ca padmāyai rādhāyai virajātaṭe /
śavitryai viṣṇumāyāyai pārṣadebhyaś ca putraka // 1,13.30 //

tubhyaṃ na datto mantro 'tra śrūyatāṃ tan nimittakam /
janiṣyasi śūdrayonau brahmaṇo vākyapālanāt // 1,13.31 //

ity evaṃ kathitaṃ sarvaṃ gaccha vatsa yathā sukham /
dvādaśābdāntare śūdrayonau devāj janiṣyasi // 1,13.32 //

pañcavarṣābhyantare ca manmantraṃ prāpya viprataḥ /
daśābdānte vapus tyaktvā brahmaputro bhaviṣyasi // 1,13.33 //

manmantraṃ punar eveti śambhuvaktrāl labhiṣyasi /
ity evam uktvā sarvātmā tatraivāntaradhīyata // 1,13.34 //
gandharvaḥ prayayau tasmād yoṣidbhiḥ saha putraka /
ity evaṃ kathitaṃ sarvaṃ pūrvavṛttāntam eva ca // 1,13.35 //

||iti śrīnāradapañcarātre jñānāmṛtasāre prathamaikarātre gandharvamokṣaṇaṃ nāma trayodaśo 'dhyāyaḥ

||prathamaikarātre caturdaśo 'dhyāyaḥ

śrīśuka uvāca /
prayāte rādhikānāthe golokaṃ ca nirāmayam /
babhūva kiṃ rahasyaṃ ca gate gandharvapuṅgave // 1,14.1 //

śrīvyāsa uvāca /
sarve devāś ca munayaḥ prayāte paramātmani /
sarve babhūvus te tūṣṇīṃ vayāṃsīva dinātyaye // 1,14.2 //

uvāca śambhur brahmāṇaṃ nītisāraviśāradam /
jñānādhidevo bhagavān pariṇāmasukhaṃ vacaḥ // 1,14.3 //

śrīmahādeva uvāca /
rakṣitā yasya bhagavān kalyāṇaṃ tasya santatam /
sa yasya vighnakartā ca rakṣituṃ taṃ ca kaḥ kṣamaḥ // 1,14.4 //

smṛtimātreṇa nirvighnā ye ca kṛṣṇaparāyaṇaḥ /
vighnaṃ kartuṃ ke samarthās teṣāṃ ca munayaḥ surāḥ // 1,14.5 //

kopāgnināṃ sthalaṃ kutra stambhitānāṃ ca sāmpratam /
devānāṃ ca munīnāṃ ca kṣaṇenaiveśvarecchayā // 1,14.6 //

yadi tiṣṭhanti bhūmau ca dagdhaśasyā vasundharā /
jale yadi tatas taptaṃ naṣṭhāste jalajantavaḥ // 1,14.7 //

sthāle dahanti lokāṃś ca vṛkṣāṃś ca pralayāgnayaḥ /
vidhānaṃ kartum ucitam eṣāṃ ca jagatāṃ vidhe // 1,14.8 //

tvam eva dhātā jagatāṃ pitā ca viṣṇur īśvaraḥ /
kālāgnirudraḥ saṃhartā nedānī pralayakṣamaḥ // 1,14.9 //

ete viṣayiṇaḥ sarve kṛṣṇasya paramātmanaḥ /
ājñāvahāś ca satataṃ dikpālāś ca digīśvarāḥ // 1,14.10 //

tasyaivājñāvaho dharmaḥ sākṣī ca karmaṇāṃ nṛṇām /
bhramanti viṣaye śaśvan mohitā māyayā hareḥ // 1,14.11 //

ahaṃ na pātā na sraṣṭā na saṃhartā ca jīvanām /
nirlipto ' haṃ tapasvī ca harer ārādhanonmukhaḥ // 1,14.12 //

saṃhāraviṣayaṃ mahyaṃ śrīkṛṣṇaś ca purā dadau /
datvā rudrāya tad ahaṃ tapasyāsu rato hareḥ // 1,14.13 //

tadarcanena dhyānena tapasā pūjanena ca /
stavena kavacanenaiva nāmamantrajapena ca // 1,14.14 //

mṛtyuñjayo ' haṃ adhunā na ca kālād bhayaṃ mama /
kālaḥ saṃharate sarvaṃ māṃ vinā ca tatheśvaram // 1,14.15 //

purā sarvādisarge ca kasyacit sraṣṭur eva ca /
bhālodbhavāś ca te rudrās teṣv eko ' haṃ śaṅkaraḥ // 1,14.16 //

kalpaś ca brahmaṇaḥ pāte laye prākṛtike tathā /
sarve naṣṭā viṣayiṇo na bhaktāś ca yatheśvaraḥ // 1,14.17 //

asaṃkhyabrahmaṇaḥ pātaḥ kalpaś cāsaṃkhya eva ca /
samatītaḥ katividho bhavitā yā punaḥ punaḥ // 1,14.18 //

śrīkṛṣṇasya nimeṣeṇa brahmaṇaḥ patanaṃ bhavet /
tatra prākṛtikāḥ sarve tirobhūtāḥ punaḥ punaḥ // 1,14.19 //

na prākṛto na viṣayī nityadehī ca vaiṣṇavaḥ /
harer vareṇāmaro 'haṃ śivādhāras tatas tataḥ // 1,14.20 //

jalaplutaṃ ca viśvaughaṃ laye prākṛtike dhruvam /
ābrahmalokaparyantaṃ paraṃ kṛṣṇālayaṃ vinā // 1,14.21 //

sarvā devyo vilīnāś ca kṛṣṇaḥ satyaṃ suniścitam /
sarve pumāṃso līnāś ca satye nitye sanātane // 1,14.22 //

ahaṃ kṛṣṇaś ca prakṛtiḥ pārṣadapravaro hareḥ /
nityaṃ nityā vidyamānā goloke ca nirāmaye // 1,14.23 //

eka īśo na dvitīya iti sarvādisargataḥ /
nahi naśyanti tadbhaktāḥ prakṛtiprākṛte laye // 1,14.24 //

tasya bhaktottamānāṃ ca satataṃ smaraṇena ca /
āyurvyayo nahi bhavet kathaṃ mṛtyur bhaviṣyati // 1,14.25 //

na vāsudevabhaktānām aśubhaṃ vidyate kvacit /
teṣāṃ bhaktottamānāṃ ca satataṃ smaraṇena ca // 1,14.26 //

janmamṛtyujarāvyādhibhayaṃ nāpy upajāyate /
atra kalpe bhavān brahmā vyavasthātā ca karmasu // 1,14.27 //

sthalaṃ kopānalānāṃ ca vidhānaṃ yad vidhe kuru /
śambhoś ca vacanaṃ śrutvā kampitaḥ kamalāsanaḥ /
sthalaṃ cakāra vahnīnām ājñayā śaṅkarasya ca // 1,14.28 //

brahmovaca /
jvaras tripādas triśirāḥ ṣaḍbhujo navalocanaḥ /
bhasmapraharaṇo raudraḥ kālāntakayamopamaḥ // 1,14.29 //

bhave bhavatu sarvatra bhavakopānalo 'dhunā /
prākṛteṣu ca deheṣu vyāpāro 'sya mayā kṛtaḥ // 1,14.30 //

mama kopānalaḥ śambho saṃskṛtāgnir dvijasya ca /
bhave bhavatu sarvatra vyāpāro 'sya mayā kṛtaḥ // 1,14.31 //

śeṣasya kopavahniś ca śeṣāsye 'stv adhunā śiva /
yato viśvaṃ ca pralaye dahed gomayapiṇḍavat // 1,14.32 //

vahner mukhyānalo viśve vyavahārāgnir īśvaraḥ /
bhvatv eva hi sarvatra sarveṣām upakārakaḥ // 1,14.33 //

dharmāsyakopavahniś ca kṛṣṇāgniś ca bhavatv ayam /
adharmaṃ kurvatāṃ sarvaṃ dāhanaṃ ca kariṣyati // 1,14.34 //

sūryakopānalaś cāyaṃ dāvāgniś ca vaneṣu ca /
sthitir asya taroḥ skandhe tadbhakṣyāḥ paśupakṣiṇaḥ // 1,14.35 //

candrakopānalo viśve kāmināṃ virahānalaḥ /
dampatyor virahe śaśvad bhakṣyati sma dvayos tanum // 1,14.36 //

indrakopānalaḥ sadyo vajrāgniś ca babhūva ha /
upendrasyānalaś caiva vidyud eva bhvatv ayam // 1,14.37 //

rudrāṇām āsyavahniś ca maholkāgnir bhavatv ayam /
gaṇeśāgniḥ pṛthivyāṃ tu yathāsthāne tu tiṣṭhati // 1,14.38 //

yatra tiṣṭhet tad uṣaram evam evaṃ vidur budhāḥ /
skandakopānalaś caiva raṇāstrāgnir babhūva ha // 1,14.39 //

kāmetarāṇāṃ devānāṃ munīnāṃ ca mukhānalaḥ /
jagrāhaurvamunis tatra tejasi brahmaṇaḥ sutaḥ // 1,14.40 //

svadakṣiṇorau sa muniḥ saṃsthāpya vedamantrataḥ /
brahmaṇaṃ ca namaskṛtya śaṅkaraṃ tapase yayau // 1,14.41 //

kālena tasmān niḥsṛtya samudre vāḍavānalaḥ /
sa babhūva purā putra paramaurvānalaḥ svayam // 1,14.42 //

kāmāgnim ulvaṇaṃ dṛṣṭvā vicintya manasā vidhiḥ /
samālocya suraiḥ sārdhaṃ munīndraiḥ saha saṃsadi // 1,14.43 //

ajuhāva striyaḥ sarvāḥ suvratāś ca pativratāḥ /
āyuyur yoṣitaḥ sarvās tā ūcuḥ kamalodbhavam // 1,14.44 //

striya ūcuḥ /
kim asmān brūhi bhagavan śādhi naḥ karavāma kim /
ālocya manasā sarvaṃ dehi bhāraṃ vayaṃ striyaḥ // 1,14.45 //

brahmovāca /
gṛhītvā madanāgniṃ ca maithune sukhadāyakam /
viśve ca yoṣitaḥ sarvāḥ śaśvatkāmā bhavantu ca // 1,14.46 //

brahmaṇaś ca vacaḥ śrutvā koparaktāsyalocanaḥ /
tam ūcuḥ yoṣitaḥ sarvā bhayaṃ tyaktvā ca saṃsadi // 1,14.47 //

striya ūcuḥ /
dhik tvāṃ jagadvidhiṃ vyarthaṃ cakāra parameśvaraḥ /
apūjyo mohinīśāpāt putraśāpena sāmpratam // 1,14.48 //

gṛhītvā madanāgniṃ ca puruṣāś ca tathā striyaḥ /
nityaṃ dahanti satataṃ vāstavaṃ duḥsahaṃ param // 1,14.49 //

tadekabhāgaḥ puruṣe tribhāgaś cāpi yoṣiti /
tena dugdhāḥ striyaḥ sarvāś cāsmākam apareṇa kim // 1,14.50 //

samarpaṇaṃ cet puruṣe yad yasmāsu smarānalaḥ /
bhasmībhūtaṃ kariṣyāmo rakṣitā ko bhavet tava // 1,14.51 //

pativratāvacaḥ śrutvā tam uvāca śivaḥ svayam /
hitaṃ satyaṃ nitisāraṃ pariṇāmasukhāvaham // 1,14.52 //

||atha kulaṭotpattiḥ

śrīmahādeva uvāca /
tyaja dvandvaṃ mahābhāga suvratābhiḥ sahādhunā /
pativratānāṃ tejaś ca sarvebhyaś ca paraṃ bhavet // 1,14.53 //

nirmāṇaṃ kuru devendra kṛtyāṃ strījātim īśvara /
tasyai dehi duḥkhabījaṃ kāmakopānalaṃ param // 1,14.54 //

śaṅkarasya vacaḥ śrutvā satvaraṃ jagatāṃ vidhiḥ /
sasṛje tatkṣaṇaṃ mūrtiṃ strīrūpāṃ sumanoharāṃ // 1,14.55 //

aho rūpam aho veśam aho asyā navaṃ vayaḥ /
aho cakṣuḥ kaṭākṣaṃ ca munīnāṃ mohayan manaḥ // 1,14.56 //

aho kaṭhinaṃ cāru stanayugmaṃ suvartulam /
vicitraṃ kaṭhinaṃ sthūlaṃ śroṇiyugmaṃ ca sundaram // 1,14.57 //

nitambayugmaṃ valitaṃ cakrākāraṃ sukomalam /
śvetacampakavarṇābhaṃ sarvāvayam īpsitam // 1,14.58 //

śaratpārvaṇakoṭīnduvinindāsyaṃ suśobhanam /
īṣaddhāsyaprasannāsyaṃ vastreṇācchāditaṃ mudā // 1,14.59 //

vapuḥ sukomalam cālaṃ nātidīrghaṃ na vakharam /
vahniśuddhāṃśukaṃ ratnabhūṣaṇair bhūṣitaṃ sadā // 1,14.60 //

dāḍimbakumudākāraṃ sāndraṃ sindūrasundaram /
kastūrīvindunā sārdhaṃ snigdhacandanavindubhiḥ // 1,14.61 //

pakvabimbaphalākāram adharauṣṭhapuṭaṃ param /
dantapaṃktiyugaṃ caiva dāḍimbabījasannibham // 1,14.62 //

sucāru kavarībhāraṃ mālatīmālyamaṇḍitam /
tasyai dadau ca kāmāgniṃ dṛṣṭvā tāṃ kamalodbhavaḥ // 1,14.63 //

dṛṣṭvā sā candrarūpaṃ ca kāmonmattā vicetanā /
kṛtvā kaṭākṣaṃ smerāśyā māṃ bhajasvety uvāca sā // 1,14.64 //

saspitaḥ prayayau candro lajjayā ca sabhātalāt /
kāmaṃ dṛṣṭvā ca cakame kāmārtā sā gatatrapā // 1,14.65 //

dudrāva kāmas tasmāc ca tatpaścāt sā dadhāva ca /
jahasur devatāḥ sarvā munayaś cāpi saṃsadi // 1,14.66 //

lajjitā yoṣitah sarvās tāṃ vārayitum akṣamāḥ /
sarve cakruḥ parīhāsaṃ strīvargaṃ śaṅkarādayaḥ // 1,14.67 //

kāmaṃ na labdhvā sā ca strī nivṛtyāgatya saṃsadi /
tam aśvinīkumāraṃ cāpy uvāca surasannidhau // 1,14.68 //

kṛtyākāminy uvāca /
māṃ bhajasva raveḥ putra priyāṃ rasavatīṃ mudā /
śṛṅgāre sukhadāṃ śāntāṃ parāṃ kāmāturāṃ varāṃ // 1,14.69 //

tvayā śārdhaṃ bhramiṣyāmi sundare gahane vane /
rahasi rahasi krīḍāṃ kariṣyāmi divāniśam // 1,14.70 //

madhupānaṃ ca dāsyāmi vāsitaṃ cāmalaṃ jalam /
sakarpūraṃ ca tāmbūlaṃ bhogavastu manoharam // 1,14.71 //

śayyāṃ manoramāṃ kṛtvā sapuṣpacandanārcitām /
bhagavantaṃ kariṣyāmi puṣpacandanacarcitam // 1,14.72 //

kumāra uvāca /
vacanaṃ vada vāme mām ātmano hṛdayaṅgama /
vihāya kapaṭaṃ kānte kapaṭaṃ dharmanāśanam // 1,14.73 //

strīdharmaṃ strīmanaskāmaṃ strīsvabhāvaṃ ca kīdṛśam /
tadācāraṃ katividhaṃ tan māṃ vyākhyātum arhasi // 1,14.74 //

aśvinījavacaḥ śrutvā kāmārtā tam uvāca sā /
kāmārtānāṃ kva lajjā ca kva bhayaṃ mānam eva ca // 1,14.75 //

||atha kulaṭāsvabhāvakathanam

kāminy uvāca /
sthānaṃ nāsti kṣaṇaṃ nāsti nāsti dūtī tad uttamā /
tenaiva yuvatīnāṃ ca satītvam upajāyate // 1,14.76 //

suveśaṃ kāmukaṃ dṛṣṭvā kāminī madanāturā /
tad rātraṃ ca pulakitaṃ yonau kaṇḍūyanaṃ paraṃ // 1,14.77 //

vicetanā bhavet sā ca kāmajvaraprapīḍitā /
sarvaṃ tyajati taddhetoḥ putraṃ kāntaṃ gṛhaṃ dhanam // 1,14.78/ //

labdhvā yuvānaṃ puruṣaṃ deśatyāgaṃ karoti sā /
taduttamaṃ punar labdhvā taṃ tyajet sā kṣaṇena ca // 1,14.79 //

viṣaṃ dātuṃ samarthā sā svāminaṃ guṇināṃ varam /
mlecchaṃ yuvānaṃ samprāpya sarvasvaṃ dātum utsukā // 1,14.80 //

tyajet kulabhayaṃ lajjāṃ dharmaṃ bandhuṃ yaśaḥ śriyam /
samprāpya ratiśūraṃ ca yuvānaṃ suratonmukham // 1,14.81 //

sudṛśyaṃ sundaramukhaṃ śaśvanmadhuritaṃ vacaḥ /
hṛdayaṃ kṣuradhārābhaṃ ko vā jānāti tanmanaḥ // 1,14.82 //

vidyucchaṭā jale rekhā cāsthirā ca yathāmbare /
tathāsthirā ca kulaṭāprītiḥ svapnaṃ ca tadvacaḥ // 1,14.83 //

kulaṭānāṃ na satyaṃ ca na ca dharmo bhayaṃ dayā /
na laukikaṃ na lajjā syāj jārajintā nirantaram // 1,14.84 //

svapne jāgaraṇe caiva bhojane śayane sadā /
nirantaraṃ kāmacintā jāre sneho na cānyataḥ // 1,14.85 //

kulaṭā naraghātibhyo nirdayā duṣṭamānasāḥ /
jārārthe ca sutaṃ hanti bāndhavasya ca kā kathā // 1,14.86 //

na hi vedā vidanty evaṃ kulatāhṛdayaṅgamam /
kathaṃ devās ca munayaḥ santo jānanti niścayam // 1,14.87 //

ratiśūraṃ priyaṃ dṛṣṭvā kṣīraṃ ghṛtam ivācaret /
gate vayasi jīrṇaṃ taṃ viṣaṃ dṛṣṭvā tyajet kṣaṇāt // 1,14.88 //

na viśvaseyus tāṃ duṣṭo tasmāt santo hi santataṃ /
na ripuḥ puruṣāṇāṃ ca duṣṭastrībhyaḥ paro bhuvi // 1,14.89 //

viṣaṃ mantrād upaśamaṃ jalād vahniś ca niścitaṃ /
agneś ca kaṇṭhakocchannaṃ durjanaḥ stavanād vaśaḥ // 1,14.90 //

lubdho dhanena rājā ca sevayā satataṃ vaśaḥ /
miśraṃ svacchasvabhāvena bhayena ca ripur vaśaḥ // 1,14.91 //

ādareṇa vaśo vipro yauvatī premabhārataḥ /
bandhur vaśaḥ samatayā guruḥ praṇatibhiḥ sadā // 1,14.92 //

mūrkho vaśaḥ kathāyāṃ ca vidvān vidyāvicārataḥ /
na hi duṣṭā ca kulaṭā puṃsaś ca vaśagā bhavet // 1,14.93 //

svakārye tatparā śaśvat prītiḥ kāryānurodhataḥ /
na sarvasya vaśībhūtā vinā śṛṅgāram ulvaṇam // 1,14.94 //

na prītyā na dhanenaiva na stavān na sevayā /
na prāṇadānato veśyā vaśībhūtā bhavet kṣaṇam // 1,14.95 //

āhāro dviguṇas tāsāṃ buddhis tāsāṃ caturguṇā /
ṣaḍguṇā mantraṇā tāsāṃ kāmaś cāṣṭagunaḥ smṛtaḥ // 1,14.96 //

śaśvatkāmā ca kulaṭā na ca tṛptiś ca krīḍayā /
haviṣā kṛṣṇavartmeva bhūya evābhivardhate // 1,14.97 //

divāniśaṃ ca śṛṅgāraṃ kurute tatpumān yadi /
na tṛptiḥ kulaṭānāṃ ca pumāṃsaṃ grastum icchati // 1,14.98 //

nāgnis tṛpyati kāṣṭhānāṃ nāpagānāṃ mahodadhiḥ /
nāntakaḥ sarvabhūtānāṃ nāsāṃ tṛpyati sampadām // 1,14.99 //

na śreyasāṃ manas tṛptaṃ vāḍavāgnir na pāthasām /
vasundharā na rajasāṃ na puṃsāṃ kulaṭā tathā // 1,14.100 //

ity evaṃ kathitaṃ kiṃcit sarvaṃ vaktuṃ ca nocitam /
lajjā bījaṃ yoṣitāṃ ca nibodha bhāskarātmaja // 1,14.101 //

śrutvā ca kṛtyāstrīvākyaṃ jahasur munayaḥ surāḥ /
cukupur yoṣitaḥ sarvāḥ padmādyā lajjitāḥ suta // 1,14.102 //

lajjānatānanā lakṣmīr niryayau devamaṇḍalāt /
tat paścāt pārvatī sārdhaṃ sarasvatyā natānanā // 1,14.103 //

sāvitrī rohinī svāhā vāruṇī ca ratiḥ śacī /
sarvā babhūvur ekatra pracakrur mantraṇāṃ ca tāḥ // 1,14.104 //

kṛtyāstriyaḥ samāhūya tā ūcuś ca krameṇa ca /
rodhayām āsur iṣṭāṃ tāṃ sugopyāṃ api yoṣitaḥ // 1,14.105 //

tasyā mukhe dadau hastaṃ suśīlā kamalālayā /
salajjitā bhava sute śāntā ceti śubhāśiṣam // 1,14.106 //

sarasvatī dadau tasyai cābhimānaṃ ca dhairyatām /
maukharyaṃ vāvadūkatvaṃ mantraṇām ātmarakṣaṇam // 1,14.107 //

sāvitrī ca dadau tasyai sauśīlyaṃ cātidurlabham /
ātmasaṃgopanaṃ caiva gāmbhīryaṃ kulato bhayam // 1,14.108 //

pārvaty uvāca /
dhik tvāṃ svabhāvakulaṭāṃ lajjitā bhava sundari /
svamānaṃ gauravaṃ rakṣa hy asmākaṃ ca smarātura // 1,14.109 //

jani labha pṛthivyāṃ ca kāyavyūhaṃ vidhāya ca /
puṃsām aṣṭaguṇaṃ kāmaṃ labhasva ca pṛthak prṛthak // 1,14.110 //

lajjāṃ caturguṇāṃ cāpi dviguṇāṃ dhairyatāṃ tathā /
abhogecchadhame gaccha dūraṃ gaccha mamāntikāt // 1,14.111 //

puṃsāṃ ca dviguṇaḥ kāmo vāstavīṇāṃ ca yoṣitām /
lajjā cāṣṭaguṇā cāpi dhairyatā ca caturguṇā // 1,14.112 //

kuladharmaḥ kulabhayaṃ sauśīlyaṃ mānam ūrjitam /
śaśvat tiṣṭhatu puṃsy eva strīṣu ca mamājñayā // 1,14.113 //

yasmāt sadasi sarvebhyo lajjāhīnaḥ surādhamaḥ /
strīsvabhāvaṃ ca papraccha yajñabhāk na bhavet tataḥ // 1,14.114 //

adyaprabhṛti viśveṣu nāgrāhyaṃ pāpasaṃyutam /
cikitsakānāṃ viduṣāṃ na bhakṣya mamājñayā // 1,14.115 //

ity evam uktvā prayayur devyaś ca sarvayoṣitaḥ /
devāś ca munayaś cāpi ye cānye ca samāgataḥ // 1,14.116 //

devāś ca munayaś cāpi ye cānye sarvataḥ suta /
pativratāṇāṃ strīṇāṃ ca lajjā bījasvarūpiṇī // 1,14.117 //

||iti śrīnāradapañcarātre jñāṇāmṛtasāre prathamaikarātre kulaṭottpatir nāma caturdaśo 'dhyāyaḥ

||prathamaikarātre pañcadaśo 'dhyāyaḥ

gate niyamite kāle gandharvaś copavarhaṇaḥ /
svayogena jahau dehaṃ bhārate prāktanād aho // 1,15.1 //

sa jajñe śudrayonau ca pituḥ śāpena ca daivataḥ /
viṣṇuprasādaṃ bhuktvā ca babhūva brahmaṇaḥ sutaḥ // 1,15.2 //

vimuktas tātaśāpena samprāpya jñānam uttamam /
pratijanmasmṛtis tasya kṛṣṇamantraprasādataḥ // 1,15.3 //

pituḥ sakāśād āgatya samprāpa candraśekharāt /
śrīkṛṣṇamantram atulaṃ svargamandākinītaṭe // 1,15.4 //

svargamandākinītīrād guruṇā śaṅkareṇa ca /
sahitaḥ prayayau tūrṇaṃ pārvatīsannidhānataḥ // 1,15.5 //

uvāsa tatra śambhuś ca nāradaś ca mahāmuniḥ /
pārvatī bhadrakālī skando gaṇapatiḥ svayam // 1,15.6 //

mahākālaś ca nandī ca vīrabhadraḥ pratāpavān /
siddhā maharṣayaś ca munayaḥ sankādayaḥ // 1,15.7 //

yogīndrā jñāninaḥ sarve samūcuḥ śambhusaṃsadi /
yat stotraṃ kavacaṃ dhyānaṃ subhadrāya ca kānane // 1,15.8 //

nārāyaṇarṣibhagavān brāhmaṇāya dadau purā /
pūjāvidhānaṃ yad yac ca puraścaraṇapūrvakam // 1,15.9 //

tad eva bhagavān śambhuḥ pradadau nāradāya ca /
uvāca śambhuṃ devarṣir yogināṃ ca guror gurum /
pārvatīsannidhau tatra nāradaś ca mahāmuniḥ // 1,15.10 //

bhagavan sarvadharmajña sarvajña sarvakāraṇa /
sad yat pṛṣṭaṃ mayā pūrvaṃ tan māṃ vyākhyātum arhasi // 1,15.11 //

yad yat pṛṣṭaṃ tvayā brahman pratyekaṃ ca krameṇa ca /
punaḥ praśnaṃ kuru mune śṛṇvantu matsabhāsadaḥ // 1,15.12 //

ādhyātmikaṃ ca yaj jñānaṃ vedānāṃ sāram uttamam /
jñānaṃ jñāniṣu sāraṃ yat kṛṣṇabhaktipradaṃ śubhaṃ // 1,15.13 //

nirvāṇamuktidaṃ jñānaṃ karmamūlanikṛntanam /
tatsiddhiyogān muktiś ca yoginām api vāñcchitam // 1,15.14 //

saṃsāraviṣayaṃ jñānaṃ śaśvat sammohaveṣṭitam /
āśramāṇāṃ samācāraṃ teṣāṃ dharmapariṣkṛtam // 1,15.15 //

caturṇāṃ varṇanāṃ vidhānāṃ maheśvara /
bhikṣūṇāṃ vaiṣṇavānāṃ ca yatīnāṃ brahmacāriṇām // 1,15.16 //

vānaprasthāśramāṇāṃ ca paṇḍitānāṃ tathaiva ca /
pativratānāṃ yad yac ca śrīkṛṣṇapūjanaṃ ca yat // 1,15.17 //

yat stotraṃ kavacaṃ mantraṃ puraścaraṇam īpsitam /
sārvāhnikam abhīṣṭaṃ ca vipākaṃ karmajīvinām // 1,15.18 //

saṃsāravāsanābaddhaṃ lakṣaṇaṃ prakṛtīśayoḥ /
tayoḥ paraṃ vā yad brahma tasyāvatāravarṇanam // 1,15.19 //

kas tat kalāvatīrṇaś ca kas tadaṃśaś tathaiva ca /
paripūrṇatamaḥ kaś ca kaḥ pūrṇaḥ kaḥ kalāṃśakaḥ // 1,15.20 //

kasya vārādhane śambho kiṃ phalaṃ kiṃ yaśas tathā /
aṅgāṅginor bhedaphalaṃ vistīrṇaṃ nirapekṣakam // 1,15.21 //

nārāyaṇarṣikavacaṃ subhadrabrāhmaṇāya ca /
yad dattaṃ kiṃ tad deveśa tadārādhyaś ca kaḥ suraḥ // 1,15.22 //

atisaṃgopanīyaṃ ca kavacaṃ paramādbhutam /
sudurlabhaṃ ca viśveṣu noktaṃ māṃ brahmaṇā purā // 1,15.23 //

sanatkumāro jānāti noktaṃ tena purā ca mām /
mayā jñānam anāpṛṣṭaṃ yad yaj jānāsi maṅgalam // 1,15.24 //

vedasāram anupamaṃ karmamūlanikṛntanam /
tan me kathaya bhadreśa mām evānugrahaṃ kuru // 1,15.25 //

apūrvaṃ rādhikākhyānaṃ vedeṣu ca sudurlabham /
purāṇeṣv itihāse ca vedāṅgeṣu sudurlabham // 1,15.26 //

guroś ca jñānodigaraṇāt jñānaṃ syān mantratantrayoḥ /
tat tantaṃ sa ca mantraḥ syāt kṛṣṇabhaktir yato bhavet // 1,15.27 //

jñānaṃ syād viduṣāṃ kiṃcid vedavyākhyānataḥ prabho /
vedakāraṇapūjyaś tvaṃ jñānādhiṣṭhātṛdevatā // 1,15.28 //

tasmād bhavān paraṃ jñānaṃ vada vedavidāṃ vara /
māṃ bhaktam anuraktaṃ ca śaraṇāgatam īśvara // 1,15.29 //

nāradasya vacaḥ śrutvā yogināṃ ca guror guruḥ /
bhagavatyā sahālocya jñānaṃ vaktuṃ samudyataḥ // 1,15.30 //

ity evaṃ kathitaṃ sarvaṃ pūrvākhyānaṃ manoharam /
haribhaktipradaṃ sarvaṃ karmamūlanikṛntanam // 1,15.31 //

||iti śrīnāradapañcarātre jñānāmṛtasāre prathamaikarātre pañcadaśo 'dhyāyaḥ

||samāptaś cedaṃ nāradapañcarātraikarātram
