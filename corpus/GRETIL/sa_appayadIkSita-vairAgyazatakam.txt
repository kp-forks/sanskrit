#Author:Appayadīkṣita
#Text:Vairāgyaśatakam
#Data entry: Dhaval Patel
#Edition:Pdt. Durgaprasad and Kasinath Pandurang Parab, Kavyamala [old, cumulative series], vol. 1, 1886 (3rd ed., 1929), pp. 91-99.


śrīmadappayadīkṣitaviracitaṃ vairāgyaśatakam /
āste kaścana bhikṣuḥ saṃgṛhṇannavyayāni daśa /
na mametyavyayayugalaṃ yācāmastaṃ kimastyanyat // 1 //
dhīsacivaṃ dhairyabalaṃ saṃkalpavirodhi śāntidhanam /
viśvatrayaviṣayamidaṃ vairāgyaṃ nāma sāmrājyam // 2 //
rājño bibheti loko rājānaḥ punarito 'pi vairibhyaḥ /
ā brahmaṇaḥ kṛtāntādakutobhayamaspṛhārājyam // 3 //
bhikṣāpradā jananyaḥ pitaro guravaḥ kumārakāḥ śiṣyāḥ /
ekāntaramaṇahetuḥ śāntirdayitā viraktasya // 4 //
ye na kimapi ceṣṭante kāryapratikaryayorvirahāt /
santasta eva muktāḥ saṃdehe gautamaḥ sākṣī // 5 //
patatu nabhaḥ sphuṭatu mahī calantu girayo milantu vāridhayaḥ /
adharottaramastu jagatkā hānirvītarāgasya // 6 //
ke corāḥ ke piśunāha ke ripavaḥ ke 'pi dāyādāḥ /
jagadakhilaṃ tasya vaśe yasya vaśe syādidaṃ cetaḥ // 7 //
viṣayā upatiṣṭhantāṃ viṣayairvā samavayantu karaṇāni /
āntaramekaṃ karaṇaṃ śāntaṃ yadi kā tataścintā // 8 //
kiṃ viṣayānparihartuṃ vastavyaṃ merukaṃdareṣvabudhaiḥ /
nahyadbhiranāsektuṃ dhātuṣu rohanti padmāni // 9 //
ajñānamiha nidānaṃ prāgrūpaṃ jananameva bhavaroge /
paripākaḥ saṃsaraṇaṃ bhaiṣajyaṃ naiṣṭhikī śāntiḥ // 10 //
svenopabhuktamarthaṃ sūkarajātismaro manuṣya iva /
dūre jugupsamāno dhīro vairāgyamadriyate // 11 //
śaiśavamiva kaumāre tattaruṇimnīva sa iva vṛddhatve /
na svadate dhīrāṇāṃ kāmasya viceṣṭitaṃ śāntau // 12 //
śataśaḥ parīkṣya viṣayānsadyo jahati kvacitkvaciddhavyāḥ /
kākā iva vāntāśanamanye tāneva sevante // 13 //
caramau mātāpitarau caramā gṛhiṇī sutāścaramāḥ /
kartavye 'pi premaṇi kathamiha dhīrā virajyante // 14 //
tṛṇavadbhramanti capalāḥ strīnāmani caṇḍamārute calati /
dharaṇidharā iva santastatra na kiṃcitprakampante // 15 //
kāmijanaparamabhogye kāmasukhe dhārayanti bībhatsam /
santaḥ śamasukharasikāḥ sudhāśanāḥ sūkarānna iva // 16 //
vikṣepamātrabhājo vikāsakāṣṭāgatajñānāḥ /
svasyāpi ceṣṭitāni svayamīkṣante parasyeva // 17 //
asthāne 'bhiniviṣṭānmūrkhānasthāna eva saṃtuṣṭān /
anuvartante dhīrāḥ pitara iva krīḍato bālān // 18 //
puṣṇati puruṣe salilairmuṣṇati puṣpaṃ phalaṃ ca tarava iva /
vartante santaḥ samamupakartari cāpakartari ca // 19 //
nityānityavivekaḥ sarveṣāṃ ghaṭaghaṭatvayorāste /
sa viveko yaḥ śāntikṛdaviveko 'nyaḥ samasto 'pi // 20 //
anadhigate kāmasukhe kālena yathā pravartate taruṇaḥ /
evaṃ brahmasukhe 'pi pravartate ko 'pi bhāgyavaśāt // 21 //
putraguṇāḥ svātantrye dāraguṇāścādhivedanāvasare /
bhrātṛguṇā dāyavidhau draṣṭavyā mokṣyamāṇena // 22 //
kā me gatiriti pṛcchati caramaśvāse 'pi yaḥ svārtham /
tasya janasyāpi kṛte pāpāḥ pāpāni kurvanti // 23 //
pitṛbhiḥ kalahāyante putrānadhyāpayanti pitṛbhaktim /
paradārānupayantaḥ paṭhanti śāstrāṇi dāreṣu // 24 //
śāntiralabhyāduparatirapātrabhāvaḥ pratigrahanivṛttiḥ /
kṣāntirdurbalateti ca nivṛttidharmāḥ kalāvete // 25 //
nītijñā niyatijñā vedajñā api bhavanti śāstrajñāḥ /
brahmajñā api labhyāḥ svājñānajñānino viralāḥ // 26 //
kalikaluṣe manasi sve kathamiva jagadārjavaṃ labhate /
cakṣurdoṣe jāgrati candradvitvaṃ kuto yātu // 27 //
viṣayānanukūlayituṃ viṣayiṇi hṛdaye vidhīyatāṃ yatnaḥ /
dṛśi deyamauṣadhaṃ ko dṛśye dattvā sukhī bhavati // 28 //
dārāḥ putreṣu ratāḥ putrāḥ pitṛdhanaparigrahavyagrāḥ /
rodanaśaraṇā jananī paralokagatasya ko bandhuḥ // 29 //
paśyanti mriyamāṇānmariṣyato 'numimate smaranti mṛtān /
kathayante caivamasacceṣṭante nityavattu param // 30 //
kalahāyante mūḍhāḥ kaḥ pratibhūḥ śvaḥ prabhāta iti /
tasyāmeva rajanyāṃ kaḥ pratibhūḥ svasya sattāyām // 31 //
kulyāḥ kṛtā viśālāḥ kuḍyānyupalairnibaddhāni /
krītā balino mahipāḥ kṛtakṛtyāḥ sma iti manyante // 32 //
ā prapadamā śiraskaṃ cāntaḥ kalimalamalīmase vapuṣi /
viphalaṃ gaṅgājalamapi madyaghaṭe darbhamuṣṭiriva // 33 //
daṇḍyaṃ yatsukhahetoḥ puṣṇanti janāḥ kathaṃ tadeva vapuḥ /
nahi śarkarābhilāṣibhirikṣoḥ kāṇḍāni pūjyante // 34 //
prāyo muhyati cetaḥ prāṇabhṛtaḥ prāṇanirgamāvasare /
puṇyena yadi na muhyati putrāneva smaranmriyate // 35 //
śamayitumaudaramagniṃ saṃsārākhyāmbudhau nimajjanti /
tuhinavyathānivṛttyai nahi veśmani pāvako deyaḥ // 36 //
āhṛtya parityaktā janayantyarthāḥ sukhābhāsam /
atyantaparityaktāḥ paramānandāya kalpante // 37 //
arthānāmadhikānāṃ rājñā caureṇa vā nāśaḥ /
anne khalvatimukte vamanaṃ vā syādvireko vā // 38 //
praṇamati parisāntvayati[praṇi]patati sadā paribhramati /
āviṣṭa iva piśācyā puruṣastṛṣṇāvaśaṃ yātaḥ // 39 //
jananādṛṣṭātpitarau putrā jāmātaro 'pyayādṛṣṭāt /
kalahādṛṣṭājjñātaya iti nirṇīte kimeṣṭavyam // 40 //
svapituḥ paralokāya svayamanudivasaṃ yadācarati /
kriyatāmidamupamānaṃ kiṃ nālaṃ putravairāgye // 41 //
ananugate dāridrye kimanugataṃ lakṣaṇaṃ dṛṣṭam /
kāmasyāpūrtiryadi kṛpaṇaṃ jagadā caturvadanāt // 42 //
na khalu dhanatvaṃ jātiryasya yadiṣṭaṃ tadeva tasya dhanam /
tattadiva pāmarāṇāmākiṃcanyaṃ dhanaṃ viduṣām // 43 //
svīkriyate yadi tṛṣṇā svīkartavyaṃ jagatsamastamapi /
svīkriyate yadi śāntiḥ svātmāpi svasya bhavati na vā // 44 //
yaddātāraṃ vaśayati yatparijanamasya sāntvayati /
yadapatrapate nāntarbhāvālābhaḥ sa kasya samaḥ // 45 //
prādeśamātramudaraṃ paripūrayituṃ kiyānayaṃ yatnaḥ /
culakenāmbhaḥ pātuṃ svanitavyaḥ kiṃ taṭāko 'pi // 46 //
yāmārdhamasaṃskārādyāmadvayamanaśanācca suvyakte /
śārīre saundarye 'pyabhiniviśante kiyanmūḍhāḥ // 47 //
vandhyetyāhustaruṇīṃ jaratīti parityajanti bahuputrām /
abhinindantyalpasutāṃ kā gṛhiṇī kāmino hṛdyā // 48 //
yānti śucamakṛtadārā dve bhārye neti kṛtadārāḥ /
te paradārā neti strībhistṛptānna paśyāmaḥ // 49 //
madanasyājñākaraṇe manye jagadakhilamekarūpamidam /
tiryañca iti narā iti devā ityanyato bhedaḥ // 50 //
śukravimokasthānaṃ malamūtratyāgadeśavatkimapi /
striya iti vihitaṃ vidhinā kiyadatra janā nimajjanti // 51 //
vedānadhītya vidhivanmīmāṃsitvā tadarthaṃ ca /
dārāḥ kartavyā iti kenedaṃ prahasanaṃ kathitam // 52 //
duḥkhenopārjyante pālyante pratyahaṃ ca lālyante /
vāmāḥ striyo vimūḍhairupabhuñjānāḥ sukhaṃ viguṇam // 53 //
aśnīta pibata svādata jāgrata saṃviśata tiṣṭhata vā /
sakṛdapi cintayatāhnaḥ sāvadhiko dehabandha iti // 54 //
kiṃ vijitayā pṛthivyā kiṃ kāñcanabhūbhṛtā karasthena /
kiṃ divyābhiḥ strībhirmartavye brahmaṇā likhite // 55 //
jīvati katicinnimiṣānkaticitteṣu śrutīradhīta iva /
tāvatsaivākulayati tantrāṇi navāni cātanute // 56 //
svalpo jīvanakālaḥ svalpā dhīḥ paricayaḥ svalpaḥ /
tadapi tarema kathaṃcicchrutayo yadi nopajāyante // 57 //
kuta āgataṃ na jāne kva nu vā gantavyamidamapi na jāne /
saṃcarasi kvedānīṃ saṃsārapathe mahātamasi // 58 //
tamasāvṛttāścarantaḥ savidhe dūre ca nāvayantyarthān /
avayanti tu vispaṣṭaṃ taḍitābhihate śirasi mūḍhāḥ // 59 //
nimiṣantyatra taruṇyastatra taruṇyo na nimiṣanti /
īdṛkṣo hi viśeṣaḥ svargaḥ svarga iti kiṃ tatra // 60 //
kopo maitrāvaruṇeḥ śāpo vā tārkikasya muneḥ /
saṃsmaryate yadi sakṛcchatrorapi māstu śakrapadam // 61 //
gacchatyamarāvatyāṃ gacchatu caturānanasya vā nagare /
punarāgantavyaṃ yadi puṃsā kiṃ sādhitaṃ bhavati // 62 //
bhuktā bahavo dārā labdhāḥ putrāśca pautrāśca /
nītaṃ śatamapyāyuḥ satyaṃ vada martumasti manaḥ // 63 //
viśleṣaṇasvabhāvānpaśyanviṣayānkaroti ko mamatām /
naśyadavasthāpannaṃ kaḥ krīṇīte dhanairaśvam // 64 //
annābhāve mṛtyuḥ śālibhirannāni śālayo vṛṣṭyā /
vṛṣṭistapaseti vadannamṛtyave tattapaścaratu // 65 //
kiṃ na nigṛhṇanti manaḥ kiṃ na bhajante janāḥ śivaṃ śaraṇam /
abhisaṃdhibhedamātrānmokṣopāye na badhyante // 66 //
bhogāya pāmarāṇāṃ yogāya vivekināṃ śarīramidam /
bhogāya ca yogāya ca na kalpate durvidagdhānām // 67 //
brāhmaṇacaṇḍālā ityāha muniryanmahāpathikān /
bhavamārgamahāpathikānadhikṛtyaiva pravartitaṃ caitat // 68 //
ekadvāḥ kṣitipatiṣu dvitrā deveṣu pañcaṣā druhiṇe /
etāvanto jagati brahmānandārṇavasya kaṇāḥ // 69 //
ajñānenāpihite vijñāne karma kiṃ kurute /
vikale cakṣuṣi tamasā vyādāya mukhaṃ kimīkṣeta // 70 //
atikaluṣamāśunaśvaramāpātasphuraṇamanabhilāṣakaram /
api hṛṣyanti janāḥ kathamavalambya jñānakhadyotam // 71 //
ayutaṃ niyutaṃ vāpi pradiśantu prākṛtāya bhogāya /
krīṇanti na bilvadalaiḥ kaivalyaṃ pañcaṣairmūḍhāḥ // 72 //
yāvatkila ceṣṭante tāvatpāśe nibadhyate granthiḥ /
nibhṛtaṃ yadi vartante kālena sraṃsate pāśaḥ // 73 //
uparundhanti śvāsānmunayo nāśnanti na pibanti /
stūyante sujanaiḥ kiṃ kaṇṭhe kurvanti kanakapāśamime // 74 //
kāmaṃ janāḥ smayante kailāsavilāsavarṇanāvasare /
sādhanakathanāvasare sācīkurvanti vaktrāṇi // 75 //
vaṅgāḥ kathamaṅgāḥ kathamityanuyuṅkte vṛthā deśān /
kīdṛkkṛtāntapuramiti ko 'pi na jijñāsate lokaḥ // 76 //
tyaktavyo mamakārastyaktuṃ yadi śakyate nāsau /
kartavyo mamakāraḥ kiṃ tu sa sarvatra kartavyaḥ // 77 //
putrā iti dārā iti poṣyānmūrkho janānbrūte /
andhe tamasi nimajjannātmā poṣya iti nāvaiti // 78 //
yaccintitamadhigarbhaṃ yacca ciraṃ cintitaṃ narake /
viṣayānilasaṃsargānmamṛje tatsarvamekapade // 79 //
bāhyagatāgataśīlā prāṇasya śvāsalakṣaṇā vṛttiḥ /
karṣati manaso vṛttiṃ kulaṭeva kulastriyaṃ mugdhām // 80 //
atigambhīramanāvilamakṣobhyamadṛṣṭapāramavilaṅghyam /
aviralataraṅgasaṃkulamaikṣiṣi vijñānasāgaraṃ mahatām // 81 //
ghoraṃ bhavamapahātuṃ kecidaghoraṃ prapadyante /
saṃsaraṇakātarāṇāṃ saṃśaraṇaṃ śāṃbhavī śaktiḥ // 82 //
pāśo yadi moktavyaḥ paśupatirevopasartavyaḥ /
na khalu vyatimucyante paśavaḥ pāśena saṃbaddhāḥ // 83 //
alamalamanubhūtābhirmātṛbhiralamastu pitṛbhiśca /
bhavitavyaṃ yadi nityavadarthaṃ mātuḥ pituścāstu // 84 //
dhanyāste bahudevāḥ svāmini yeṣāṃ na durbhikṣam /
jātu na jānīmo vayamekaṃ taṃ svāminaṃ pūrṇam // 85 //
santu bṛhanto devāḥ kiṃ tu na tānnantumīhate cetaḥ /
āḍhyavadānyanyāyādantakajitameva cintayenmanasi // 86 //
nidhyāyasi viṣayasukhaṃ na dhyāyasi viṣamamasya paripākam /
bandhuṃ tameva cintaya baddhuṃ moktuṃ ca yaḥ kṣamate // 87 //
sadanaṃ gurūpasadanaṃ śaraṇaṃ pañcākṣarīpuraścaraṇam /
dhanamabhilāṣanirodhanamantyāśramavartināṃ puṃsām // 88 //
kau pitarau kaḥ putraḥ kaḥ svāmī yaḥ prapañcasya /
pratyastamite bhede kimidaṃ kimidamiti vipraśnaḥ // 89 //
sa vidhiryatte vidadhati sa pratiṣedho yato nivartante /
sopaniṣadyadbruvate śaivāśravamartino dhīrāḥ // 90 //
tyaja saṃsāramasāraṃ bhaja śaraṇaṃ pārvatīramaṇam /
viśvasihi śrutiśikharaṃ viśvamidaṃ tava nideśakaram // 91 //
bhavyamabhavyaṃ vā naḥ pralikhatu vedhāḥ sudurmedhāḥ /
savyamasavyaṃ vā naḥ śaraṇaṃ caraṇaṃ maheśasya // 92 //
vedhāḥ kathaṃ hariḥ kathamiti tu praśne vayaṃ mūkāḥ /
śivamekaṃ jānīmo na śivādanyaṃ vijānīmaḥ // 93 //
dāruṇamasipattravanaṃ dāruṇatamamandhatāmisram /
kā vā tataḥ kṣatirnaḥ śaivā vayamā caturvadanāt // 94 //
kṛtadīkṣo ghoramakhe kulakūṭastho bharadvājaḥ /
vidyeśvareṣu kaścana pitāmaho na iti visrambhaḥ // 95 //
kalahaḥ kadāpi māstviti kalitaśarīraikyayoḥ śivayoḥ /
ahamasmyahamasmīti prāptaḥ kalaho mama trāṇe // 96 //
narakāyāpi na bhogānnarādhamāyāpi nānyasurān /
manyante katicidamī māheśvaramāśritā yogam // 97 //
jñātuṃ hātuṃ viṣayaṃ śrotuṃ mantuṃ gṛhītumātmānam /
vatsā yadi na hi ghaṭate tatsādhayatāvimuktāya // 98 //
sāṃkhyaṃ yogaṃ nigamā bhaktiḥ karma pratītiriti /
ekatra sakalametatkevalamavimuktamekatra // 99 //
baddhaḥ kaste vakṣyati mukto muktiṃ vijānāti /
yāsyasi cedavimuktaṃ jñāsyasi viśveśvarasya mukhāt // 100 //
na gṛhītaṃ śrutihṛdayaṃ na ca na gṛhītaṃ pariplavaṃ hṛdayam /
icchāmi ca dhāma paraṃ gacchāmi tu viśvanāthapurīm // 101 //
iti śrīmadappayadīkṣitaviracitaṃ vairāgyaśatakaṃ saṃpūrṇam /
