#Text:Karatoyāmāhātmya
#Data entry:Arlo Griffiths
#Edition:included as Appendix I in: Prabhas Chandra Sen: Mahasthan and Its Environs, Rajshahi, Bengal : Varendra Research Society 1929 (Varendra Research Society's Monographs, 2).
#Notes:(A. Griffiths) This work is said to be included in a larger work the Uttara-Pauṇḍra-khaṇḍa, was first published in 1298 B.S. (1891 A.D.) with a Bengali translation, by Pandit Rajachandra Nyayapanchanan. It consists of two parts: verses 1 to 60, Paundra-khsetra-māhātmyam and verses 61 to 85, Karatoyā-māhātmyam. The text is printed here from that edition. Verse 41 of this work is quoted in Sarvānanda's Tikā sarvasva (1159 A.D.) in the tikā of sloka 32 in vārivarga; verses 41 and 64, in the Smritichandrikā of Devanabhaṭṭa who is quoted by Hemādri (12th century); verse 41, in the Vyākhyāsudhā of Bhanuji Dikshit son of Bhattoji Dikshit; verse 63, in Smritiratnakar of Vedāchārya who quotes Bhavadeva and Jimutavahana and is quoted by Raghunandanda.

||atha karatoyāmāhātmyam ||
aparaṃ kathyatāṃ deva nadīnāṃ ca viśeṣataḥ /
pauṇḍrakṣetrasya māhātmyaṃ na śrutaṃ vistarāt prabho // 01 //
kadotpattiḥ kathaṃ tasyāḥ kasmāc caiva viśeṣaṭaḥ /
śrotum icchāmi bhūteśa yadi syān mayy anugrahaḥ // 02 //
kena prākāśitā sā ca nadī karajalā bhuvi /
kathaṃ vā plāvitaṃ kṣetraṃ śubhaṃ pauṇḍram anuttamam // 03 //
pāṇigrahaṇakāle te devi himavatā jalam /
saṃpradattaṃ matkarāc ca nirgataṃ karajā bhuvi // 04 //
puraiva kathitaṃ sarvaṃ pauṇḍrasya ca sureśvari /
tatraiva kathitaṃ tubhyaṃ karatoyāphalaṃ yathā // 05 //
adhunāpi yathā tāsāṃ nadīnāṃ ca viśeṣataḥ /
kalipāpaharā puṇyā bhārgavena prakāśitā // 06 //
matputro 'pi guhas tatra tiṣṭhaty eva hi sarvadā /
yatrāste bhagavān viṣṇur garuḍāsana īśvaraḥ // 07 //
sarvadā sarvabhāvena pauṇḍre nārāyaṇo hariḥ /
puṣkaraṃ na tyajet brahmā nāhaṃ vārāṇasīṃ tyaje // 08 //
śrīpauṇḍravardhanaṃ kṣetraṃ naiva muñcati keśavaḥ /
dharitryā nābhikamalaṃ pūtaṃ karajalair mama // 09 //
śṛṇudhvaṃ munayaḥ sarve yad uktaṃ bhārgavena vai /
śrutvā śaṅkarato vākyaṃ saṃvādam ubhayor api // 10 //
putravātsalyabhāvena bhārgavāya prakāśitam /
sa eva bhārgavaḥ śrīmān ṛṣibhyo 'py aprakāśayat // 11 //
namas tasmai munīndrāya dānavendraniṣūdane /
śrīcakrapāṇaye tubhyaṃ brahmaviṣṇuśivātmane // 12 //
ekaḥ paraśumātreṇa nihatya kṣatriyān yudhi /
cakre niḥkṣatriyāṃ pṛthvīm ekaviṃśativārataḥ // 13 //
tataḥ paraśurāmeti pṛthivyāṃ khyātavikramaḥ /
jāmadagnyo mahāvīryas tretādye yasya saṃsthitiḥ // 14 //
sarvajñaḥ suvrataḥ śuddhaḥ sarvācāravidhāyakaḥ /
kautukākṛṣtahṛdayaḥ pṛthivyāṃ paramo hariḥ // 15 //
pauṇḍre koṭiśilādvīpe mahāpuṇye suviśrute /
karatoyāsarinnīraṃ śarīrādyantapāvanam /
bhaktimuktiphalārthāya yenākāri dvijārpaṇam // 16 //
adbhutā kāritā sṛṣṭiḥ kanakasya dinatrayam /
skandagovindayor madhye bhūmiḥ saṃskṛtavedikā // 17 //
vedīmadhyottare pārśve devī kālañjarī sthitā /
taddakṣiṇe 'rpitā devī koṭīśvarīti viśrutā // 18 //
nairṛte liṅgakoṭyaś ca vasanti bhṛguṇārpitāḥ /
vāraṇe vijayā caṇḍī uttare bhūtikeśvaraḥ // 19 //
tatkuṇḍe sutithau snātvā naraḥ pāpāt pramucyate /
bhūtikeśvaradevasya dakṣiṇe sūryamaṇḍapam // 20 //
vedīmadhye 'rpito yūpaḥ saṃśleṣād vardhate nṛṇām /
govindamaṇḍapāt pūrvaṃ kuṇḍaṃ kuṇḍaṃ viṣṇuvinirmitam // 21 //
skandamaṇḍapavāyavye sabhā rāmasya cādbhutā /
sapādalakṣaṃ viprāṇāṃ yatrāste 'dbhutakarmaṇām // 22 //
prabhāvāt tapaso devi munīndrasya mahātmanaḥ /
tatsabhā vāyukoṇe ca gartam īśvaranirmitam // 23 //
ādyaṃ bhuvo bhavanaṃ lakṣasapādavipraiḥ skandādiviṣṇubalabhadraśivādidevaiḥ /
adhyāsitaṃ karajalāmbuvidhūtapāpaṃ śrīpauṇḍravardhanapuraṃ śirasā namāmi // 24 //
karajāpaścime bhāge sadā vahati jāhnavī /
pūrvabhāge tu karajā pādonā jāhnavī jalā // 25 //
karatoyāpaścime tīre lohinī yatra mṛttikā /
muktikṣetraṃ samākhyātaṃ mahāpātakanāśanam // 26 //
karatoyānadīṃ prāpya trirātropoṣito naraḥ /
aśvamedham avāpnoti śakralokaṃ ca gacchati /
atraiva jñānam āsādya harisāyujyam āpnuyāt // 27 //
karatoyāṃ samāsādya ye tyajanti kalevaram /
teṣāṃ muktir na sandeho yāvad indrāś caturdaśa // 28 //
na mānuṣās te te devā nadīs tisraḥ pibanti ye /
devikāṃ karatoyāṃ ca vipāśāṃ pāpanāśinīm // 29 //
skandagovindayor madhye somavāre kuhūtithau /
prātar maunena yaḥ snāyāt kulakoṭīḥ samuddharet // 30 //
kiyanto reṇavaḥ pṛthvyāṃ kiyān ākāśasaṃbhavaḥ /
māhātmyaṃ karatoyāyā vaktuṃ naiva hi śakyate // 31 //
puruṣottame mahājyaiṣṭhīsamaye darśanāt phalam /
karatoyāmbhasi snātvā yat tat phalam avāpnuyāt // 32 //
karatoyājalaṃ prāpya yadi somayutā kuhūḥ /
aruṇodayavelāyāṃ sūryagrahaśataiḥ samā // 33 //
śilādvīpaṃ samāsādya tac ca koṭiguṇaṃ bhavet /
tad eva koṭiguṇitaṃ pauṣārke ca yadaiva sā // 34 //
vārāṇasyāṃ kurukṣetre yat puṇyaṃ rāhudarśane /
śilādvīpaṃ samāsādya tac ca koṭiguṇaṃ bhavet // 35 //
pauṣe vā māghamāse vā yadi somayutā kuhūḥ /
vyatipātena yogena koṭikoṭiguṇaṃ bhavet // 36 //
cāpārke mūlasaṃyukte yadi somayutā kuhūḥ /
nārāyaṇīti vikhyātā trikoṭikulam uddharet // 37 //
vārāṇasyāṃ kṛtā pūjā saṃpūrṇaphaladāyinī /
tato 'pi dviguṇā proktā karatoyānadījale // 38 //
dvārāvatyāṃ ca gaṇḍakyāṃ prayāge puṣkare tathā /
badaryākhye kurukṣetre yā pūjā phaladāyinī /
tataś caturguṇā proktā karatoyānadījale // 39 //
karatoyājale devi viṣṇupūjā viśeṣataḥ /
tato 'pi phalabāhulyaṃ śivaśaktyoḥ prapūjanāt // 40 //
ādau karkaṭake devi tryahaṃ gaṅgā rajasvalā /
sarvā raktavahā nadyaḥ karatoyāmbuvāhinī // 41 //
iyaṃ śrīsundarī devi sadānīravahā smṛtā /
ye kurvanti sadā snānaṃ tarpayanti ca ye sadā /
kiṃ bahūktena deveśi muktis teṣāṃ kare sthitā // 42 //
aho jalasya māhātmyaṃ mamaiva karasaṃbhavaṃ /
nṛṇāṃ pāpaharaṃ puṇyaṃ snānapānāc ca muktidam // 43 //
karatoyānadītīre vāso vā kriyate yadi /
vārāṇasīsamo vāsaḥ pātakān mucyate naraḥ // 44 //
karajāyās tīre devapūjā sarvārthasādhikā /
anyatra pūjanād devi saṃdeho nāsti sundari // 45 //
kāratoyena toyena udarasthena ye mṛtāḥ /
teṣāṃ muktir na saṃdeho yāvad indrāś caturdaśa /
tatraiva jñānam āsādya muktiḥ syāt kevalāmalā // 46 //
gaṅgāyāḥ karajāyāś ca jalam atreti sundari /
sarvapāpaharaṃ puṇyaṃ bhuvi pāvanam uttamam // 47 //
asthikeśādayo yasya karajāyāṃ tapodhane /
patanti tasya svargaḥ syād yāvad indrāś caturdaśa // 48 //
karatoyāpaścime tīre sadā vahati jāhnavī /
viśeṣo lohinī yatra mṛttikā muktidāyinī // 49 //
karatoyāpaścime tīre lohinī yatra mṛttikā /
muktikṣetraṃ samākhyātaṃ mahāpātakanāśanam // 50 //
karatoyājalaṃ puṇyaṃ pāvanaṃ bhuvi durlabham /
saṃpūrṇamāghamāsaṃ tu snātvā viṣṇupuraṃ vrajet // 51 //
karatoyānadītīre vedapūjāpārāyaṇaḥ /
viprabhojanamātreṇa hy aśvamedhaphalaṃ labhet // 52 //
viśeṣaḥ pauṇḍranagare koṭikoṭiguṇaṃ bhavet /
vipraikabhojanād eva sarvayajñaphalaṃ labhet // 53 //
japahomais tathā dānapūjāśrāddhakriyādibhiḥ /
koṭikoṭiguṇaṃ tatra pauṇḍrakṣetre ca sundari // 54 //
karatoyāmṛdā ye ca tilakaṃ dhārayanti vai /
viṣṇurūpadharāḥ pāpān muñcanti nātra saṃśayaḥ // 55 //
skandagivindayor madhye guptā vārāṇasī purī /
tatrārohaṇamātreṇa naro nārāyaṇo bhavet // 56 //
pañcakrośam idaṃ kṣetraṃ samantāt parikīrtitam /
tadantargatam etat tu krośamātraṃ maheśvari /
atiguhyatamaṃ kṣetraṃ yatrāste bhārgavo muniḥ // 57 //
paśor jñānaṃ kathayati guhas tadgṛhe tāmracūḍo dairghī haimī ghaṭitasurabhir yaṣṭivṛddhiḥ śilāsthiḥ /
kheṣu chhattraṃ na phaṇati phaṇī dvisvaro jīvalokaḥ kūpo dvīpaḥ kanakapatanaṃ pauṇḍrakṣetre 'dbhutāni // 58 //
proccā bhūmir bhavati taruṇaḥ snānataḥ kāmyakuṇḍe bhogo yajño bhramaṇanaṭanaṃ tatra vākyaṃ hi vedaḥ /
itthaṃ rāmo racayati padaṃ lakṣaṇānyūnaviṃśāny asmāt khyātaṃ sakalajagatāṃ śrīmahāsthānam etat // 59 //
snānād yatra nihanti pāpanicayaṃ śrīpāṇitoyā nadī yasyāṃ saṃsthitamānuṣāsthi sakalaṃ prāpnoti pāṣāṇatām /
devas tārakamārako 'pi nitarāṃ jñānaṃ dadāty adbhutaṃ kaupaṃ yat paya eva tailavipulaṃ pauṇḍraṃ puraḥ pātu vaḥ // 60 //
||ity uttarapauṇḍrakhaṇḍe pauṇḍrakṣetramāhātmyam ||
śṛṇudhvaṃ munayaḥ sarve māhātmyaṃ tajjalasya ca /
bāhudāyāś ca tīre 'smin jalaṃ sarvamalāpaham // 61 //
gaṅgā vā karatoyā vā viśeṣo nātra vidyate /
haramūrdhni sthitā gaṅgā sāparā karanirgatā // 62 //
karatoye sadānīre saricchreṣṭhe suviśrute /
pauṇḍrān plāvayase nityaṃ pāpaṃ hara karodbhave /
mantreṇānena vai snāyāt karatoyājale śubhe // 63 //
tattulyarūpāsti nadī na kā cid rajovihīnā taruṇo yato 'si /
dhanyāsi puṇyāsi saridvarāsi śrīkaṇṭhapāṇiprabhave namas te // 64 //
snānād iyaṃ karajalā duritāni hanti jñānaṃ dadāti bhagavān iha tārakāriḥ // 65 //
bāhudā bāhudānāc ca likhitasya muneḥ purā /
sadānīrā mahāpuṇyā śītavāhinikā śubhā // 66 //
ṛṣayo munayaś caiva mārkaṇḍeyo mahāmuniḥ /
aśvatthāmā kapivaro vāsudevaḥ svayaṃ vibhuḥ // 67 //
caturmukhaḥ pañcamukhaḥ karimukhaś ca ṣaṇmukhaḥ /
sarve te paścime bhāge karatoyāsarittaṭe // 68 //
tiṣṭhanti tapaso hetoḥ sadānīrājalārthinaḥ /
rajohīnā mahāpuṇyā viśeṣaḥ siṃhabhāskare // 69 //
karatoyāsarinnīraprāptimātreṇa sundari /
snānatarpaṇam āvaśyaṃ tadāpāraṃ vidhīyate // 70 //
asnātvā gacchataḥ pāraṃ pūrvadharmakṣayo bhavet /
snātvā pītvā tathā nīraṃ pārāvāre na dūṣayet // 71 //
pauṇḍrakṣetraṃ mahāpuṇyaṃ plāvitaṃ karasaṃbhavaiḥ /
tajjalasnānamātreṇa viṣṇuprītivivardhanam // 72 //
pauṇḍrakṣetrād uttare ca yojanadvayam antarā /
tatrāste caṇḍikā devī lohinī yatra mṛttikā // 73 //
taddhāre prārthanāmātrāt turagān bhavanāni ca /
dadāti caṇḍikā devī karatoyāsarittaṭe /
aho kṣetrasya māhātmyād brahmahatyāṃ vyapohati // 74 //
cāṇḍālāntyajasaṃspṛṣṭaṃ tīrthatoyaṃ na pāvanam /
vihāya karajāgaṅgānarmadāyamunājalam // 75 //
bhāṇḍānītajalenāpi snānaṃ kurvanti ye narāḥ /
pāpaughān aviśeṣeṇa muñcanti nātra saṃśayaḥ // 76 //
karatoyājale jñānājñānataḥ snāti yo naraḥ /
brahmalokam avāpnoti dinapāpān vinaśya vai // 77 //
ajñānenāpi karajājalaṃ ye manujāḥ sakṛt /
durvṛttāḥ pāparahitā prapibanti bhavanti te // 78 //
atraiva pauṇḍranagare puruṣārthasiddhir vārāṇasī vyasanam eva paraṃ narāṇām /
tatraiva viṣṇunilayaṃ sakalaṃ ca yajño jñānaṃ samādhivividhaṃ japasādhyasiddhiḥ // 79 //
kṛṣṇaveṇī tāmraparṇī sarayūr gaṇḍakī tathā /
viṣṇupādodbhavā puṇyā yamunā ca sarasvatī // 80 //
kāverī kauśikī candrabhāgā ca ciravallabhī /
svarṇā campā vetravatī tathātreyī punarbhavā // 81 //
vipāpā ca vipāśā ca citrā citrotpalā tathā /
gotamī gomukhī revā tathā cirasarasvatī // 82 //
pṛthvyāṃ vasantyaḥ saritaḥ sarvās toyacayāḥ priye /
āsāṃ nadīnāṃ ca jalaṃ sadaiva ca karāmbhasi // 83 //
pauṇḍradvīpe paṭhitvemaṃ śrīvīro bhārgavo muniḥ /
pauṇḍrān pradakṣiṇīkṛtya mucyate brahmahatyayā // 84 //
karatoyāmahātīrthamāhātmyaṃ yaḥ paṭhec chuciḥ /
trisandhyam ekasandhyaṃ vā śṛṇuyād vāpi yo naraḥ // 85 //
tasyeha bhogān sakalān bhuktvā tadgatamānasaḥ /
pretya yāti paraṃ sthānaṃ sarvāmalanivāraṇam // 85* //
||ity uttarapauṇḍrakhaṇḍe sūtaśaunakasaṃvāde paraśurāmaviracitaṃ karatoyāmāhātmyaṃ samāptam |
