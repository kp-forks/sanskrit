namaḥ śivāya śaktyai ca bindave śāśvatāya ca /
gurave ca gaṇeśāya kārtikeyāya dhīmate // 1 //
binduśaktiśivākhyāni trīṇi siddhāntasāgarāt /
samuddhṛtya satāṃ dhartuṃ hṛtkaṇṭhaśravaṇe sadā // 2 //
ratnāni vāṅmayaiḥ sūtrair nibadhyante mayā tataḥ /
kṣantumarhanti tatsanto nāsūyā paricārake // 3 //
rañjanātsarvatattveṣu rāgādebhyo 'dhikārayoḥ /
īśvarāṇāṃ śivānāṃ ca dhāraṇīyatayā dhiyā // 4 //
puṃsāmapratighodāramarīcinicayena ca /
ratnānīṣṭavidhānācca bindvādyāḥ śivaśāsane // 5 //
ratnatrayaṃ samāśritya vartate tattvasaṃhatiḥ /
adhvaitatprāpyate ṣoḍhā varṇyate parameṣṭhinā // 6 //
etadeva mataṃ bījaṃ trayaṃ dīkṣāpratiṣthayoḥ /
yogāśca vividhāḥ karma nityanaimittikaṃ tathā // 7 //
jñānāni siddhayo mokṣā api tasyāvabodhanāt /
etadeva parā kāṣṭhā mokṣākhyāṇorvicārataḥ // 8 //
śrotavyametadāptoktyā mantavyaṃ copapattibhiḥ /
dhyeyaṃ ca yogamārgeṇa śaktipātoditātmabhiḥ // 9 //
āptoktiratra siddhāntaḥ śiva evāptimān yataḥ /
na tābhyāṃ sadṛśaḥ kaścic chreya āptividhāyakaḥ // 10 //
siddhānta eva siddhāntaḥ pūrvapakṣāstataḥ pare /
āptastu śiva evaikaḥ śivānye tvaśivā matāḥ // 11 //
siddhāntaḥ sevyate sadbhiḥ śaktipātapavitritaiḥ /
kāmakāritayānyaistu nindyate paśuśāstravat // 12 //
na hi kastūrikāmodaḥ puruṣaiḥ pratihanyate /
hetubhiḥ sādhyate kintu dhanyairāghrāyate sadā // 13 //
vedāntaiśca kulāmnāyais tathānyaiḥ pratipādyate /
ānandavipralabdhānām ānandopahitā citiḥ // 14 //
cinmahodadhigāmbhīryam avasīyeta sūribhiḥ /
avagāhya parānandalaharī yadi notkṣipet // 15 //
tadatra kathitaṃ sarvasrotasāṃ jyāyasi prabhoḥ /
udakṣeṇordhvavaktreṇa tatastadavadhārayet // 16 //
hetūnapi parīkṣāyai lakṣayet tasya śāstrataḥ /
na hyatra śemuṣī śuddhā vipulāpi pragalbhate // 17 //
alaukikāni sūkṣmāṇi gopitāni śivena ca /
trīṇi ratnāni ko vetti siddhāntena vinā svayam // 18 //
hetūnapi kutarkāndhatamasārīn manīṣayā /
tadvadyogaṃ ca ko vetti sabījābījalakṣaṇam // 19 //
siddhāntaḥ sevitaḥ sadbhir api kāmān prayacchati /
sarvān sādhakacittasthān āptaścintāmaṇiryathā // 20 //
tamāsevya mayāpyeṣā binduśaktiśivāśrayā /
parīkṣā kriyate tatra bindurādau nirūpyate // 21 //
jāyate 'dhvā yataḥ śuddho vartate yatra līyate /
sa binduḥ paranādākhyaḥ nādabindvarṇakāraṇam // 22 //
uttīrṇamāyāmbudhayo bhagnakarmamahārgalāḥ /
aprāptaśivadhāmānas tridhā vijñānakevalāḥ // 23 //
vidyāvidyādhipatayaḥ paśupūrvāḥ sadāśivāḥ /
tatra vidyābhujaḥ pūrve mantrā vidyāśca nāmataḥ // 24 //
vidyeśvaraniyojyāste saṃkhyayā saptakoṭayaḥ /
teṣāṃ purāṇi viyāyāṃ vāmādīni yathottaram // 25 //
sphītāni nava jātāni tanubhogendriyādibhiḥ /
madhyāḥ praśāntakaluṣā vidyeśāḥ śivatejasaḥ // 26 //
adhikāramalopetās tattvamaiśaṃ samāśritāḥ /
śivārkakarasamparkavikāsātmīyaśaktayaḥ // 27 //
aṣṭāvanantapūrvāste yathāpūrvaṃ guṇādhikāḥ /
tathā purāṇi tatraiṣām anantaḥ parameśvaraḥ // 28 //
tatrāṇavaḥ pureṣvāsan nāpyanye tadvibhūtayaḥ /
pare sadāśivasamāḥ patikṛtyādhikāriṇaḥ // 29 //
mūrdhānamadhvanaḥ prāptāḥ prasanne parameśvare /
tattvameṣāmasūtāṅgapurabhogādiśobhitam // 30 //
sadāśivamadhiṣṭhātṛnāmnā te 'pi sadāśivāḥ /
aṣṭādaśabhiradhvāyaṃ bhuvanaiḥ saha bhoktṛbhiḥ // 31 //
tribhiśca tattvairuddiṣṭo viśuddhaḥ śivakartṛkaḥ /
sadāśivāditattvaugho nityopādānakāraṇaḥ // 32 //
vikāritvādyathā kumbhas tathā caiṣa tatastathā /
pārthivo 'pi bhavetkumbho māyopādānakāraṇaḥ // 33 //
vināśotpattimattvābhyāṃ pariṇāmitayā tathā /
yadyadutpadyate vastu tanmāyeyaṃ yathā kalā // 34 //
utpattināśau māyeyadharmāvāha maheśvaraḥ /
pariṇāmo hi vastūnāṃ pūrvāvasthāparicyuteḥ // 35 //
avasthāntarasamprāptiḥ kṣīrasya dadhibhāvavat /
dadhnaśca takravat tatra takrāvasthā nirūpyate // 36 //
na dadhno nāpi dugdhasya pūrvāvasthe hi te mate /
sadavasthaṃ hi vastvekaṃ pūrvaṃ kṣīraṃ tato dadhi // 37 //
paścāttakraṃ tathā māyā vicitrapariṇāmataḥ /
tattvatāttvikabhāvānām upādānamanaśvaram // 38 //
tato nidarśanaṃ sādhu prasiddhoktaviśeṣaṇam /
na māyā neśvaro nāṇur na śaktiḥ śuddhavartmanaḥ // 39 //
upādānamato binduḥ pariśeṣeṇa labhyate /
tathā hi māyā yā teṣāṃ kṣobhitānantatejasā // 40 //
jalādikṣitiparyantaṃ tattvajātamasūta sā /
aviśuddhajaḍatvena māyāmārgatayā tathā // 41 //
duḥkhānuṣaṅgānmāyeyakārmāṇavamalānvayaiḥ /
sakalāṇūpabhogyatvāt pariṇāmodayairapi // 42 //
māyā jaḍāntaravyāptā pariṇāmavatī ca yat /
niṣpādane kalādīnāṃ śarīrādisamanvitam // 43 //
puruṣaṃ gamayedeva parādhīnamasaṃśayam /
suvarṇamiva karmāraṃ makuṭotpādakarmaṇi // 44 //
saiṣā vikalpavijñānagocaraiva satī ca yat /
kṣobhyate 'nantanāthena kumbhakāreṇa mṛd yathā // 45 //
savikalpakavijñānaṃ citeḥ śabdānuvedhataḥ /
sa tu śabdaścaturdhā vāgvaikharyādivibhedataḥ // 46 //
jāyate bindusaṃkṣobhād anantasyārthadarśane /
vidyāśarīro bhagavān anantaḥ kṣobhako mataḥ // 47 //
māyāyāḥ sā ca vidvadbhir baindavaṃ tattvamucyate /
ato na māyopādānaṃ tathaivāyaṃ maheśvaraḥ // 48 //
cetanatvādavṛttitvāt pariṇāmāt tatastathā /
ātmā śaktiśca vijñeyau vistaro 'traiva vakṣyate // 49 //
ito 'pi lakṣyate bindur aṇuvaiṣamyadarśanāt /
dṛśyante pudgalāḥ kecid alpajñānakriyānvitāḥ // 50 //
tebhyo 'dhikāḥ pare 'nye tu sarvajñā balaśālinaḥ /
pudgalaścetano nityo vikārarahito mataḥ // 51 //
vikāritve jaḍānityabhāvaḥ syād ghaṭakuḍyavat /
tathaiva ca citiḥ śaktis tayorapyavikāriṇoḥ // 52 //
bahudhā yadavasthānaṃ tadupādhivaśād bhavet /
sampṛktā cidaṇoryena māyādyarthāvalokane // 53 //
yadupādhervicitrā ca sa bindurbahuvṛttikaḥ /
na karmaṇāṇorvaicitryam anapekṣeṇa jāyate // 54 //
vaicitryamapi bhogasya sāpekṣeṇaiva tena yat /
karmopabhogaṃ kurute vaicitryaṃ candanādayaḥ // 55 //
tadeva yadi tatkuryāt kiṃ tairiti vilupyatām /
pravṛttiḥ sarvabhūtānāṃ tyāgopādānakāraṇam // 56 //
kiñcātiśāyikaṃ prāhus tamambaramanaśvaram /
śivānāmasamaiśvaryabhājāṃ bhogādhikārayoḥ // 57 //
jyotirgaṇānāmākāśam iva bhūtādikāraṇam /
bindureva vikalpākhyāṃ savikalpakabuddhiṣu // 58 //
svavṛttibhedasambhedair ullikhan labhate citim /
na cāyaṃ bhāvanāsaṃjñaḥ saṃskāro 'dhyakṣabhāvataḥ // 59 //
saṃskārāḥ smṛtiliṅgā hi nāsmatpratyakṣagocarāḥ /
na buddheḥ pariṇāmo vā māyordhvamapi sambhavāt // 60 //
tathā vidyeśvaro 'nanto māyāmākramya tejasā /
tataḥ sṛṣṭiṃ prakurute savikalpakabodhavān // 61 //
aṇutvea sati kartṛtvād asmatpreṣyo yathā janaḥ /
anye vṛttiparīṇāmabhedavādaviśāradāḥ // 62 //
guravaḥ kathayantyenam anyathoktaviśeṣaṇam /
pariṇāmasya kartāyaṃ na tu vṛttestatastathā // 63 //
idamevaṃ mayā kṣubdham iṣṭaṃ sampādayed dhruvam /
iti jānāti yaḥ śaktaḥ sa kartā pariṇāminām // 64 //
pariṇāmiṣvayaṃ dharmo vṛttimatsvanyathā bhavet /
tathā hi sarvo nirdhūtavikalpamavalokayan // 65 //
vastu loko vijānāti savikalpakamanyathā /
anantenāpi śabdānuviddhavijñānapūrvakam // 66 //
sarvaṃ cediha vijñātam iṣyetaiṣa karoti ca /
savikalpaṃ vijānāmīty avabodhābhimānataḥ // 67 //
vṛttireva matā bindoḥ paṭasyeva kuṭī tataḥ /
nirvikalpakabodhe 'pi bindumīśo 'dhitiṣṭhati // 68 //
naivaṃ vidyeśvaro māyām eṣā hi pariṇāminī /
na vṛttipariṇāmābhyāṃ kartṛbhedo 'vadhāryate // 69 //
kurvato 'pi kuṭīṃ buddhiḥ savikalpā hi dṛśyate /
śabdatattvamaghoṣā vāg brahma kuṇḍalinī dhruvam // 70 //
vidyāśaktiḥ parā nādo mahāmāyeti deśikaiḥ /
bindurevaṃ samākhyāto vyomānāhatamityapi // 71 //
catasro vṛttayastasya yābhirvyāptāstridhāṇavaḥ /
vaikharī madhyamābhikhyā paśyantī sūkṣmasaṃjñitā // 72 //
tatra sā vaikharī śrotragrāhyā yārthasya vācikā /
sthāneṣu vidhṛte vāyau kṛtavarṇaparigrahā // 73 //
prayoktṝṇāmiyaṃ prāyaḥ prāṇavṛttinibandhanā /
kevalaṃ buddhyupādānā kramād varṇānupāyinī // 74 //
antaḥsaṃjalparūpā tu na śrotramupasarpati /
prāṇavṛttimatikramya vartate madhyamāhvayā // 75 //
avibhāgena varṇānāṃ sarvataḥ saṃhṛtikramāt /
svayamprakāśā paśyantī māyūrāṇḍarasopamā // 76 //
svarūpajyotirevāntaḥ sūkṣmā vāganapāyinī /
yasyāṃ dṛṣṭasvarūpāyām adhikāro nivartate // 77 //
puruṣe ṣoḍaśakale tāmāhuramṛtāhvayām /
kevalaḥ paramānando ghoro nityoditaḥ prabhuḥ // 78 //
nāstameti na codeti na śrānto na vikāravān /
sarvabhūtāntaracaraḥ śabdabrahmātmako raviḥ // 79 //
bhittvā yaṃ bodhakhaḍgena nirgacchantyaviśaṅkitāḥ /
tāmeva vāṇīṃ sūkṣmākhyām āhurātmavido janāḥ // 80 //
pratyātmaniyatā etā vṛttayo bandhanātmikāḥ /
ābhyo viviktamātmānaṃ na hi paśyanti pudgalāḥ // 81 //
paramātmaiva vāgātmā vāgevātmeti ca śruteḥ /
vaikharī śrotraje bodhe madhyamā savikalpake // 82 //
paśyantī madhyamotpādasamudyogeṣu lakṣyate /
yadāvṛttiraśeṣeṇa vilīnā cittasaṃśrayā // 83 //
tadā sūkṣmā viśuddheva cidābhātyavivekataḥ /
na so 'sti pratyayo 'ṇūnāṃ yaḥ śabdānugamādṛte // 84 //
anuviddhamiha jñānaṃ sarvaṃ śabdena jāyate /
saiṣā caturvidhā vṛttir nivṛttyādikalāśrayāt // 85 //
pañcadhā bhidyate bhūyaḥ kalāstā binduvṛttayaḥ /
nivṛttiśca pratiṣṭhā ca vidyā śāntiśca pañcamī // 86 //
śāntyatītāḥ kalā etā yābhivyāpto 'dhvapañcakam /
tattvādhvā bhuvanādhvā ca varṇādhvā ca padātmakaḥ // 87 //
mantrādhvā vyāpakasteṣāṃ kalādhvā bindumāśritaḥ /
tyaktvaikamekaṃ samprāpya kalādiṣvajarāmaram // 88 //
padamāsādyate pumbhir ato 'dhvānaḥ kalādayaḥ /
kalādhvā varṇitaḥ pūrvaṃ nivṛttyādivibhedataḥ // 89 //
vyāptiṃ tasyābhidhāsyāmi sādhārāṅkādhidevatām /
sākārakāraṇāmātmabinduśaktiśivāśrayām // 90 //
||nivṛttikalā:---
nivṛttau pārthivaṃ tattvaṃ purāṇyaṣṭottaraṃ śatam /
teṣu kālānalādīnām anantāṇḍasya ṣaḍ bahiḥ // 91 //
prācyādiṣu daśasvāsann āśāsvekaikaśo daśa /
rudrāṇāṃ śatasaṃkhyānāṃ vīrabhadrasya copari // 92 //
bhadrakālyāśca bhuvane kṣakāro 'rṇeṣu kīrtitaḥ /
ekāśītipadā devī pratisargapadakramāt // 93 //
aṣṭāviṃśatisaṃkhyasitu padairatra pratiṣṭhitā /
mantrāvajātahṛdayau tatra sā tu parā kalā // 94 //
bhuvanādīnabhivyāpya pañca pañcasu vartate /
pītaiṣā caturaśrā ca kaṭhinā vajralāñchitā // 95 //
dhyātavyodyadanekārcir māṃsalā hṛdayāmbuje /
tatpralīnāṇusaṃghasya saṃkalpo vinivartate // 96 //
anāsādya phalaṃ tena nivṛttirabhidhīyate /
brahmātra kāraṇaṃ mantraḥ sadyojāto 'dhidevatā // 97 //
||pratiṣṭhākalā:---
pratiṣṭhāyāṃ tu catvāri kalāyāmavanīṃ vinā /
bhūtāni pañca tanmātrāḥ pañca karmendriyāṇi ca // 98 //
pañca buddhīndriyāṇyāsan mano'haṅkārabuddhayaḥ /
avyaktamapi varṇāśca hādiṭāntā vilomataḥ // 99 //
trayoviṃśatirāmnātāḥ ṣaṭpañcāśat purāṇi ca /
jalatejo 'nilākāśabuddhyavyakteṣvahaṅkṛtau // 100 //
aṣṭāvaṣṭau padānyekaviṃśatistatra saṃkhyayā /
śiro vāmaśca mantrau dvau viṣṇuḥ kāraṇamucyate // 101 //
sā tu śuklārdhacandrābhā nīlotpaladalāṅkitā /
dhyātavyā galapadmāntarbahulālokaśālinī // 102 //
tatpralīnāṇusaṃghasya saṃkalpo 'rthaprasiddhaye /
pratiṣṭhito yatastena pratiṣṭhā nāma sā kalā // 103 //

||vidyākalā:---
vidyāyāṃ sapta puruṣo rāgo niyatisaṃyutaḥ // 104cd //
vidyā kalā ca kālaśca māyātattvāni tatra tu /
māyādibhuvanānāṃ ca vijñeyāḥ saptaviṃśatiḥ // 105 //
ñādayo 'tra ghakārāntā varṇāḥ sapta vilomataḥ /
padāni viṃśatirmantrau śikhāghorau vyavasthitau // 106 //
rudro 'tra kāraṇaṃ mantro bahurūpo 'dhidevatā /
sā tu sphuradanekārcis trikoṇā svastikāṅkitā // 107 //
dhyātavyā tālupadmāntarnīlāñjanasamadyutiḥ /
tatpralīnāṇusaṃghasya saṃkalpo 'śeṣagocaraḥ // 108 //
||śāntikalā:---
savārthadyotako yena tena vidyeti gīyate /
śāntau tu trīṇi tattvāni daśāṣṭau bhuvanāni ca // 109 //
kathitānyatra varṇāstu gādikāntāstrayo matāḥ /
padānyekādaśātrāsan mantrau vaktratanucchadau // 110 //
bhuvanādīnyabhivyāpya pañca ṣaḍbindulāñchitā /
ṣaṭkoṇoditamalpāntasahasrakiraṇadyutiḥ // 111 //
dhyātavyā sā parā śaktir bhrūmadhyakamalodare /
tatpralīnāṇusaṃghasya dveṣarāgādyabhāvataḥ // 112 //
saṃkalpasya praśāntatvāc chāntireṣā nigadyate /
īśvaraḥ kāraṇaṃ tatra mantrastatpuruṣāhvayaḥ // 113 //
tadadhiṣṭhāyako jñeyaḥ puruṣastvadhidevatā /
atra ca trīṇi tattvāni pañcamī tu parā kalā // 114 //
śāntyatītakalā:--- /
śivatattvātmakaṃ tatra purāṇi daśa pañca ca /
varṇā visargapūrvā ye ṣoḍaśa svarasaṃjñitāḥ // 115 //
omityādyaṃ padaṃ mantrāḥ śivāstreśānaśabditāḥ /
śāntyatītā ca pañcādhvagarbhiṇī paramā kalā // 116 //
cintanīyā mahākāśasvarūpā mūrdhapaṅkaje /
īśāno mantrarāḍenām adhitiṣṭhati kāraṇam // 117 //
devaḥ sadāśivastatra kalādhveti prakīrtitaḥ /
vyāpako bhuvanādīnām abhivyāptaḥ sa bindunā // 118 //
binduḥ śaktyā śivenaiṣā nānyena vyāpyate śivaḥ /
sa hi devaḥ samāvṛtya svaśaktyānanyabhūtayā // 119 //
sarvamākramya ca tayā vijānāti karoti ca /
anyasyānyatra sadbhāvaḥ sarvadāvyabhicāriṇī // 120 //
vyāptirāptapadārthātmalābhasthitinibandhanam /
dvidhā sā sannidhāvekā paramātmavidhānataḥ // 121 //
tatra sannidhimātreṇa vidadhātyakhilaṃ citiḥ /
yathārko dinaceṣṭānāṃ sannidherupakārakaḥ // 122 //
bindurātmani nādādīn adhvanaḥ ṣaḍapi kramāt /
vicitraiḥ pariṇāmaistair vidadhāti śiveritaḥ // 123 //
yathā mṛtkalaśādīni kulālādhiṣṭhitā satī /
yena yad vyāpyate vastu pariṇāmitayā svayam // 124 //
sā tasya vikṛtiḥ proktā hemnastu makuṭo yathā /
bindunā vyāpyate yo 'sau ṣaḍvidho 'dhvā kalādikaḥ // 125 //
sā tasya vikṛtistena vyāptatvānnādabinduvat /
kecidācakṣate binduḥ samavaiti śive tataḥ // 126 //
dṛkśaktivat kriyāśaktir iyaṃ kuṇḍalinī parā /
dve śaktī samavāyinyau śive jñānakriyātmike // 127 //
ādyā tu saṃvid vijñānaṃ kriyā kuṇḍalinī parā /
jñānaśaktyā vijānāti kriyayā kurute jagat // 128 //
kriyā hi phaladā puṃsāṃ na jñānaṃ syāt phalapradam /
tābhyāṃ na virahastasya te ca na staḥ śivaṃ vinā // 129 //
tayoḥ prasaratoḥ sāmye tattvātmā syāt sadāśivaḥ /
īśādhikakriyāśaktyor vidyājñānātiriktayoḥ // 130 //
tattveṣu teṣu vijñānakevalānāṃ mahātmanām /
bhuvanāni vicitrāṇi svabhāvalalitāḥ priyāḥ // 131 //
bhogānapyaparimlānān aṅgāni karaṇāni ca /
vidadhāti śivaḥ śakter ananyāyāḥ kriyātmanaḥ // 132 //
nādādīnapi tenaiṣa kartā śuddhādhvano mataḥ /
ato nāyaṃ pṛthak śakteḥ śaktireva kriyātmikā // 133 //
samavetā śive bindur iti tatrābhidhīyate /
so 'yamātmani tattvaugham anekabhuvanāvalim // 134 //
vidadhad vividhānalpatanubhogendriyādikam /
bindurākhyāyate yuktyā pariṇāmī pradhānavat // 135 //
