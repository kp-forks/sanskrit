#Author:Kṣemendra
#Text:Darpadalana
#Data entry:Members of the Rasala team
#Edition:Durgaparasad and Kashinath Pandurang Parab, Bombay 1890, Kâvyamâlâ : a Collection of Old and Rare Sanskrit Kâvyas ... [= 'collective' Kavyamala series], Part VI, pp. 66-118.



||prathamo vicāraḥ
praśāntāśeṣavighnāya darpasarpāpasarpaṇāt /
satyāmṛtanidhānāya svaprakāśavikāsine // 1.1 //
saṃsāravyatirekāya hṛtotsekāya cetasaḥ /
praśamāmṛtasekāya vivekāya namo namaḥ // 1.2 //
||yugmam
kṣemendraḥ suhṛdāṃ prītyā darpadoṣacikitsakaḥ /
svāsthyāya kurute yatnaṃ madhuraiḥ sūktibheṣajaiḥ // 1.3 //
kulaṃ vittaṃ śrutaṃ rūpaṃ śauryaṃ dānaṃ tapastathā /
prādhānyena manuṣyāṇāṃ saptaite madahetavaḥ // 1.4 //
ahaṃkārābhibhūtānāṃ bhūtānāmiva dehinām /
hitāya darpadalanaṃ kriyate mohaśāntaye // 1.5 //
kulaṃ kulaṃ kalayatāṃ mohānmithyābhimāninām /
lagnaḥ ko 'yaṃ na jānīmaḥ stabdhagrīvāgrahagrahaḥ // 1.6 //
kulasya kamalasyeva mūlamanviṣyate yadi /
doṣapaṅkaprastāntas tadāvaśyaṃ prakāśate // 1.7 //
yathā jātyaturaṃgasya na śakyajjātya(?)mucyate /
tathā guṇavataḥ sūnur nirguṇastatkulodbhavaḥ // 1.8 //
ekaścetpūrvapuruṣaḥ kule yajvā bahuśrutaḥ /
aparaḥ pāpakṛnmūrkhaḥ kulaṃ kasyānurvartatām // 1.9 //
loke kulaṃ kulaṃ tāvad yāvatpūrvasamanvayaḥ /
guṇaprabhāve vicchinne samāptaṃ sakalaṃ kulam // 1.10 //
kulābhimānaḥ kasteṣāṃ jaghanyasthānajanmanām /
kulakūlaṃkaṣā yeṣāṃ jananyo nimnagāḥ striyaḥ // 1.11 //
kulīnasya kulīnasya navadāridryalajjayā /
kiṃ kulenākulīnāgre yāñcādainyapralāpinaḥ // 1.12 //
guṇavatkulajāto 'pi nirguṇaḥ kena pūjyate /
dogdhrīkulodbhavā dhenur vandhyā kasyopayujyate // 1.13 //
svayaṃ kulakṛtastasmād vicārya tyajyatāṃ madaḥ /
guṇādhīnaṃ kulaṃ jñātvā guṇeṣvādhīyatāṃ matiḥ // 1.14 //
mūlānveṣaṇacintyamānamaniśaṃ nāstyeva puṃsāṃ kulaṃ strīṇāṃ yatra paramparaiva tanute saṃtānatantukramam /
etāsāṃ kṛtakaprapañcaracanālajjāvatīnāṃ puraḥ saṃsaktasmararūḍhagūḍhacaritaṃ tattvena jānāti kaḥ // 1.15 //
kulābhimānābharaṇasya mātā pitāmahī vā prapitāmahī vā /
yoṣitsvabhāvena yadi praduṣṭā tadeṣa doṣaḥ kulamūlaghātaḥ // 1.16 //
sūryavaṃśe triśaṅkuryaś caṇḍālo 'bhūnmahīpatiḥ /
dilīparaghurāmādyāḥ kṣitipāstatkulodbhavāḥ // 1.17 //
bhūbhujāṃ somavaṃśyānāṃ yaḥ pūrvapuruṣo budhaḥ /
gurutalpe sa candrasya jāto jagati viśrutaḥ // 1.18 //
kanyāyāstanayaḥ karṇaḥ kṣetrajāḥ pāṇḍunandanāḥ /
sāmānyakulacarcābhiḥ kimanyābhiḥ prayojanam // 1.19 //
mathurāyāmabhūtpūrvaṃ brāhmaṇaḥ śrīmatāṃ varaḥ /
yajvā śrutanidhirnāma śrutimānviśrutaśrutaḥ // 1.20 //
tasya muktālatā nāma prāṃśuvaṃśasamudbhavā /
babhūva vallabhā patnī lāvaṇyalalitākṛtiḥ // 1.21 //
tasyāṃ tasyābhavatkāntaḥ suvṛttaḥ saguṇāgraṇīḥ /
putrastejonidhirnāma vidyāvimaladarpaṇaḥ // 1.22 //
sa dhīmānvedavidvādī kaviḥ sarvakalālayaḥ /
sabhāsu viduṣāṃ cakre lajjayāvanataṃ śiraḥ // 1.23 //
taṃ darpadoṣajvaritaṃ grīvāstambhayutaṃ rahaḥ /
praśamāya pitā snehāt pathyaṃ vaktuṃ pracakrame // 1.24 //
putra mithyābhimānena kiṃ prayāto 'si mūḍhatām /
yanmadadviradārūḍhaḥ pūjyapūjāsu lajjase // 1.25 //
nāstyupāyaḥ sa saṃsāre darpaśvabhranipātinām /
mūḍhānāṃ kriyate yena kṣanaṃ hastāvalambanam // 1.26 //
kaṣṭaṃ kenopadiṣṭaste vinaṣṭavinayasmṛteḥ /
madaḥ sādhujanāviṣṭaḥ kulavidyādhanodbhavaḥ // 1.27 //
asthiraḥ kulasaṃbandhaḥ sadā vidyāvivādinī /
mado mohāya mithyaiva muhūrtanidhanaṃ dhanam // 1.28 //
etadeva kulīnatvam etadeva guṇārjanam /
yatsadaiva satāṃ satsu vinayāvanataṃ śiraḥ // 1.29 //
dayaiva viditā vidyā satyamevākṣayaṃ dhanam /
akalaṅkavivekānāṃ śīlamevāmalaṃ kulam // 1.30 //
abhogasubhagā bhūtir adainyadhavalaṃ kulam /
adarpaviśadā vidyā bhavatyunnatacetasām // 1.31 //
dveṣaḥ kasya na doṣāya prītiḥ kasya na bhūtaye /
darpaḥ kasya na pātāya nonnatyai kasya namratā // 1.32 //
tyāginā kiṃ daridreṇa kiṃ kulīnena pāpinā /
tuṣṭena kiṃ kadaryeṇa darpāndhena budhena kim // 1.33 //
vairāyate suhṛdbhāvaḥ pradānaṃ haraṇāyate /
darpabhūtābhibhūtasya vidyā maurkhyaśatāyate // 1.34 //
guṇināṃ matsaraḥ śatrur lubdhānāmatiyācakaḥ /
sarva eva sadarpāṇāṃ na kaścitpriyavādinām // 1.35 //
tasmātkāryastvayā putra nāhaṃkāraḥ kadācana /
darpodgrīvaḥ kilogreṇa mohagrāheṇa gṛhyate // 1.36 //
vaṃśenonnatiśālinā guṇagaṇenāntaścamatkāriṇā rūpeṇātimanohareṇa mahatā vittena vṛttena vā /
rohanmohamahātarurmadamayaḥ saṃjāyate yaḥ sadā tasyādau dṛḍharūḍhamūladalane kāryo 'bhiyogastvayā // 1.37 //
guṇeṣvanādaraṃ putra prāptaśrīrapi mā kṛthāḥ /
saṃpūrṇo 'pi ghaṭaḥ kūpād guṇacchinnaḥ patatyadhaḥ // 1.38 //
kulābhimānaṃ tyaja saṃvṛttāgraṃ dhanābhimānaṃ tyaja dṛṣṭanaṣṭam /
vidyābhimānaṃ tyaja paṇyarūpaṃ rūpābhimānaṃ tyaja kālalehyam // 1.39 //
putra prayatnena vibodhito 'si na muñcasi tvaṃ yadi darpamoham /
tadeṣa te yāsyati śalyabhāvaṃ tīvrābhitāpaprasavo 'bhimānaḥ // 1.40 //
vibhūtinalinīgajaḥ sujanamānabhaṅgaśaninipātapathadaiśikaḥ sukṛtacitradhūmodgamaḥ /
parāśayanavajvaraścaritacandrabimbāmbudaḥ sadā samadacetasāṃ guṇavināśaheturmadaḥ // 1.41 //
anityateyaṃ yadi nityatā syāt sarvaṃ na pāke virasaṃ yadi syāt /
kulārthavidyādikṛto 'bhimānas tadaiṣa te syānna viḍambanīyaḥ // 1.42 //
ahaṃ vādī vidyāparicayaguruḥ sarvaviduṣām ahaṃ mānī vāṇīprasaraparipākena sukaviḥ /
ahaṃ līlāhaṃsaḥ kuvalayadṛśāṃ mānasacaraḥ karotyantaḥ puṃsāmiti madapiśācaḥ paricayam // 1.43 //
lakṣmīḥ kṣaṇakṣayavatī parirakṣitāpi kāyo 'pyapāyanicayasya nikāya eva /
saṃbhogayogasukhasaṃgatirapyatathyā mithyābhimānakalanāghana eṣa śāpaḥ // 1.44 //
ityukto 'pyasakṛtpitrā līlāmīlitalocanaḥ /
sa yayau mattahastīva vegādagaṇitāṅkuśaḥ // 1.45 //
pādena kṣitimālikhanti samadāḥ kopoṣṇaniḥśvāsinas tiryagjihmanirīkṣaṇairvidadhati bhrūbhaṅgabhīmaṃ mukham /
sasvedāṅgulikandalīnikaṣaṇaistāmyallalāṭatvacaḥ kampante hitamantravādasamaye bhūtābhibhūtā iva // 1.46 //
sa kadācidvarāśveṣu sthiteṣu javakautukāt /
pratasthe kharamāruhya vayasyagṛhamutsave // 1.47 //
tena tīkṣṇapratodena codyamānaḥ punaḥ punaḥ /
kharastīvravyathārto 'bhūt prakṣaratkṣatajokṣitaḥ // 1.48 //
srutākṣaḥ kathitakleśaḥ svasvanocitasaṃjñayā /
so 'vadatsaṃmukhāyātāṃ gardabhīṃ jananīṃ nijām // 1.49 //
mātarbrāhmaṇaputro 'yaṃ paśya māmadhamāśayaḥ /
vidārayanpratodena vahantaṃ hantumudyataḥ // 1.50 //
kiṃ karomi yamenāhaṃ labdho 'nena durātmanā /
avaṭe pātayāmyenaṃ tanuṃ śvabhre kṣipāmi vā // 1.51 //
ityārtarāviṇaṃ putraṃ sāśrunetrātha gardabhī /
tamuvāca sasaṃtāpaṃ snehasaṃkrantatadyathā // 1.52 //
vahainaṃ durmadaṃ putra sahasva viṣamāṃ vyathām /
asya nāstyeva hṛdaye dāruṇe karuṇākaṇaḥ // 1.53 //
raudraḥ śūdreṇa jāto 'yaṃ brāhmaṇyāṃ brahmavarjitaḥ /
paraduḥkhaṃ na jānāti caṇḍaṃ caṇḍālaceṣṭitaḥ // 1.54 //
dayādaridraṃ hṛdayaṃ vacaḥ krakacakarkaśam /
yonisaṃkarajātānām etatpratyakṣalakṣaṇam // 1.55 //
navanītopamā vāṇī karuṇākomalaṃ manaḥ /
ekabījaprajātānāṃ bhavatyavanataṃ śiraḥ // 1.56 //
raṭati kaṭukāṭopaṃ kopādakāraṇavairavān spṛśati na dayāṃ dainyāpanne vijātitayā śaṭhaḥ /
kṣaṇarasikatālolaḥ sevāśritānavamanyate guṇiṣu kurute garvodgārānakharvagalaḥ khalaḥ // 1.57 //
iti duḥsahamākarṇya gardabhīvacanaṃ dvijaḥ /
sarvaprāṇisvanābhijñaḥ saṃmohābhihato 'patat // 1.58 //
sa labdhasaṃjñaḥ sucirān meruśṛṅgādiva cyutaḥ /
tatyāja sahasā darpaṃ naṣṭākhilakulonnatiḥ // 1.59 //
saṃmūrcchito viṣeṇeva sa gatvā māturantikam /
yathāśrutaṃ nivedyāsyai sarvaṃ papraccha tāṃ rahaḥ // 1.60 //
tanutyāgapravṛttena pṛṣṭā sā tena śāpitā /
adhomukhī tamavadad vailakṣyalulitākṣaraiḥ // 1.61 //
lajjākaramasatkarma kathaṃ tatkathayāmi te /
saṃsārādapi sāścaryaṃ gahanaṃ strīviceṣṭitam // 1.62 //
api kuñjarakarṇāgrād api pippalapallavāt /
api vidyudvilasitād vilolaṃ lalanāmanaḥ // 1.63 //
na bādhyante guṇaiḥ patyur na lakṣyante parīkṣakaiḥ /
na dhanena nivāryante śīlatyāgodyatāḥ striyaḥ // 1.64 //
dhanayauvanasaṃjātadarpakāluṣyaviplavāḥ /
kenonnataparibhraṣṭā vāryante nimnagāḥ striyaḥ // 1.65 //
dehapradāḥ prāṇaharā narāṇāṃ bhīrusvabhāvāḥ praviśanti vahnim /
krūrāḥ paraṃ pallavapeśalāṅgyo mugdhā vidagdhānapi vañcayanti // 1.66 //
ahaṃ pura rajaḥsnātā kāle kusumulāñchane /
ekākinī puṣpavane yauvanonmādinī sthitā // 1.67 //
vratadīkṣāpare patyau serṣyeva vinatānanā /
unnatastanavinyastahastā ciramacintayam // 1.68 //
etāḥ śvasanasotkampāḥ sajṛmbhāḥ ṣaṭpadasvanaiḥ /
sotkaṇṭhamiva gāyanti latāḥ puṣparajakhalāḥ // 1.69 //
udbhinnayauvanākrāntā priyabhogaviyoginī /
vrataśeṣajuṣaḥ patyur doṣeṇaivāsmi niṣphalā // 1.70 //
iti cintākṣaṇe tasmiṃl lagnābhimukhadarpaṇaḥ /
nāpitaḥ parihāsākhyaḥ śīlaśatrurivāyayau // 1.71 //
sa māmekākinīṃ dṛṣṭvā naṣṭasaṃvṛttikātarām /
pasparśotkampinīṃ pādanakhagrahaṇalīlayā // 1.72 //
tatrāhaṃ vṛttakartavyā nīcasaṃgamalajjayā /
adhomukhī cyutaṃ śīlaṃ vīkṣamāṇeva mūrcchitā // 1.73 //
avidūre carantī sā kharī sarvaṃ dadarśa tat /
gūḍhagarbhapradaṃ caitat karma me kulapātakam // 1.74 //
āstāṃ kimanayā putra guptavṛttāntacarcayā /
saṃvṛttānyeva śobhante śarīrāṇi kulāni ca // 1.75 //
iti māturvacaḥ śrutvā yātaḥ sa sahasāndhatām /
jātimānāvapatanān nirjīvita ivābhavat // 1.76 //
atha gatvā nirāhāraḥ sa kailāsāṭṭāhāsinīm /
āśāṃ brāhmaṇyabaddhāśaś cacāra suciraṃ tapaḥ // 1.77 //
tasyogratapasā tuṣṭaḥ svayameva śatakratuḥ /
brāhmaṇyaṃ yācamānasya na dadau durlabhaṃ bhuvi // 1.78 //
punaḥ punaḥ sa tapasā saṃtāpitajagattrayaḥ /
sahasrākṣavarātprāpa devatvaṃ na tu vipratām // 1.79 //
chandodevābhidhāno 'tha so 'bhavadbhuvi viśrutaḥ /
pratyabdamekadivase hy arcanīyo mṛgīdṛśām // 1.80 //
saṃmohapātālaviśālasarpas tasmānna kāryaḥ kulajātidarpaḥ /
śamakṣamādānadayāśrayāṇāṃ śīlaṃ viśālaṃ kulamāmananti // 1.81 //
mātā na yasyāstyavivekarāśiḥ punarbhavābdhirjanako na yasya /
yasya prasaktā dayitā na tṛṣṇā sa eva loke kuśalī kulīnaḥ // 1.82 //
||dvitīyo vicāraḥ
dhanena darpaḥ kimayaṃ narāṇāṃ lakṣmīkaṭākṣāñcalacañcalena /
yatkaṃdharābaddhamapi prayāti naikaṃ padaṃ kālagatasya paścāt // 2.1 //
surakṣitaṃ tiṣṭhati nirnimittam arakṣitaṃ tiṣṭhati daivayogāt /
sthitaṃ kadaryasya ca copayuktam unmattanṛttopamameva vittam // 2.2 //
karmoktinarmanirmāṇaiḥ prātaḥ prātaḥ pradhāvatām /
dhanaṃ dhanaṃ pralapatāṃ nidhanaṃ vismṛtaṃ nṛṇām // 2.3 //
vicchāyayornirvyayayoḥ kaṣṭakliṣṭakalatrayoḥ /
viśeṣaḥ kleśadoṣasya kaḥ kadaryadaridrayoḥ // 2.4 //
ye dhanādānasaṃnaddhā nekṣante nidhanāvadhim /
nindanto lubdhatāṃ teṣām ante 'nye bhuñjate dhanam // 2.5 //
uktaṃ parasyāmiṣatām anuktaṃ yātyadṛśyatām /
hṛdaye śalyatāṃ dhatte nidhane dhanināṃ dhanam // 2.6 //
dhanena jīviteneva kaṇṭhasthena nirīkṣate /
paryante 'pyaprakāśena bandhūnāṃ mukhamāturaḥ // 2.7 //
yadarjitaṃ parikleśair arjitaṃ yanna bhujyate /
vibhajyate yadante 'nyaḥ kasyacinmāstu taddhanam // 2.8 //
vidyā vivādāya dhanaṃ madāya prajñāprakarṣaḥ paravañcanāya /
atyunnatirlokaparābhavāya yeṣāṃ prakāśastimiraṃ hi teṣām // 2.9 //
aśāntāntastṛṣṇā dhanalavaṇavārivyatikarair gatacchāyaḥ kāyaściravirasarūkṣāśanatayā /
anidrā mando 'gnirnṛpasalilacaurānalabhayāt kadaryāṇāṃ kaṣṭaṃ sphuṭamadhanakaṣṭādapi param // 2.10 //
śrāvastyāṃ sārthavāho 'bhūd arthanātha ivāparaḥ /
nando nāma nirānandaḥ kīrtanenārthināmapi // 2.11 //
sa kadaryaḥ sadā sarvajanasyodvegaduḥsahaḥ /
mūrdhaśāyī nidhānānāṃ kālavyāla ivābhavat // 2.12 //
kṛtvā samastaṃ divasaṃ dhanānāṃ nidhānakumbhīgaṇanāvidhānam /
sa lājapeyāpalamānaśīlam aśnāti rātrāvudaraṃ saśūlam (?) // 2.13 //
nirvyañjanaṃ nirlavaṇaṃ vinaṣṭam amṛṣṭapākaṃ viniviṣṭakaṣṭam /
adṛṣṭahāsaṃ vyayasaṃnirodhāt tasyābhavadveśma saśokamūkam // 2.14 //
vicchāyaṃ niḥsukhānandaṃ nirdīpaṃ jalavarjitam /
tasya kaṣṭaṃ kadaryasya paralokamabhūdgṛham // 2.15 //
sa bhaktasaṃcaye nityam abhaktaḥ saṃtatāmayaiḥ /
suvarṇavānvivarṇo 'bhūt saṃpūrṇaścintayā kṛśaḥ // 2.16 //
puṇyaprāpyā matirnāma dhanarddhiriva rūpiṇī /
bhāryābhūttadayogyasya tasya daivaviparyayāt // 2.17 //
sadā pracchādya sā bhartuś cakārātithisatkriyām /
tena vyayavivādeṣu śoṣitā kalahāgninā // 2.18 //
tasyāṃ tasyābhavatsūnuḥ saguṇaścandanābhidhaḥ /
pitrā lobhāndhakāreṇa nītaḥ padma ivānyatām // 2.19 //
kadācitsvagṛhadvāri dṛṣṭvā labdhānnamarthinam /
cakāra kalahaṃ nandaḥ patnyā śoṇitapātanam // 2.20 //
so 'vadatkopadaṣṭauṣṭhaḥ śvasanbhāryāmadhomukhīm /
tatsparsapāpaṃ stanayoḥ kṣālayantīmivāśrubhiḥ // 2.21 //
mama dāsyati ko bhikṣāṃ tvatpāṇikṣīṇasaṃpadaḥ /
dāridryajananī yasya sthitā tvaṃ durbhagā gṛhe // 2.22 //
striyo yatra pragalbhante bharturācchādya kartṛtām /
gṛhaṃ bhavatyavaśyaṃ tadāspadaṃ paramāpadām // 2.23 //
gṛhamekaṃ gṛhasthasya gṛhāṇāṃ śatamarthinaḥ /
bhāryābharjitavittasya naṣṭā gṛhapatergatiḥ // 2.24 //
tṛptidaṃ darśanenāpi jantorjīvitajīvitam /
draviṇaṃ yena rakṣanti svakāyaṃ bhakṣayanti te // 2.25 //
jīvannapyakriyo niḥsvaḥ śavo 'pyarthena sakriyaḥ /
dāridryaṃ maraṇaṃ loke dhanamāyuḥ śarīriṇām // 2.26 //
etadevārthasāmarthyaṃ pratyakṣeṇopalakṣyate /
yatskandhabandhe jīvadbhiḥ śavaḥ śibikayohyate // 2.27 //
prayacchasi kimarthibhyas tvamannaṃ kleśasaṃcitam /
dīyate yatkila prāptyai tatprāptaṃ kiṃ na rakṣyate // 2.28 //
putradārādisaṃbandhaḥ puṃsāṃ dhananibandhanaḥ /
kṣīṇātputrāḥ palāyante dārā gacchanti cānyataḥ // 2.29 //
paṇḍitāḥ kavayaḥ śūrāḥ kalāvantastapasvinaḥ /
vaidyasyeva savittasya vīkṣante mukhamāturāḥ // 2.30 //
iti tasya vacaḥ śrutvā kṛpaṇasyārthaniṣkṛpam /
sā tamūce samucitaṃ sattvasyābhijanasya ca // 2.31 //
santaḥ kurvanti yatnena dharmasyārthe dhanārjanam /
dharmācāravinīnānāṃ draviṇaṃ malasaṃcayaḥ // 2.32 //
yatkarotyaruciṃ kleśaṃ tṛṣṇaṃ mohaṃ prajāgaram /
na taddhanaṃ kadaryāṇāṃ hṛdayavyādhireva tat // 2.33 //
vardhate yo dhanavyādhiḥ sukhabhogaviyogakṛt /
tasyāśu śamanaṃ pathaṃ rājavaidyacikitsayā // 2.34 //
lobhānnābhūdgṛhe yasya kadācitkaścidutsavaḥ /
nṛtyanti paṭahaistasya nidhane dhanabhāginaḥ // 2.35 //
kaṇācāmatuṣāṅgārān yatnena parirakṣasi /
mūṣakāpahṛtaṃ koṣe ratnarāśiṃ na paśyasi // 2.36 //

yatsadoṣamayogyaṃ vā kūṭadānena tena kim // 6.13 //
cirasevānurodhena lobhakṛcchrādanicchayā /
aprasādena yaddattaṃ balādānena tena kim // 6.14 //
yatpuṣpadhūpatilakapratipattipradarśitam /
dattamatyalpaniḥsāraṃ dambhadānena tena kim // 6.15 //
prabhūtabhārasaṃbhāraṃ rājacaurādiviplave /
dattvā yadduṣṭamudghuṣṭaṃ śalyadānena tena kim // 6.16 //
anāsvādyamavikreyam anādeyamanīpsitam /
dattaṃ nirupakāraṃ yad vandhyadānena tena kim // 6.17 //
ṛṇavaccirasaṃśodhyaṃ vacasā pratipāditam /
yannityayācanadveṣaṃ yācyadānena tena kim // 6.18 //
ekasmai pūrṇamanyasmai kṛśaṃ tulyaguṇodaye /
bhedādyadarpitaṃ rāgadveṣadānena tena kim // 6.19 //
ṛṇadaiḥ svajanaiḥ putrair labdhakṣāmapratigrahaḥ /
nityamāyāsyate yena kalidānena tena kim // 6.20 //
na parasyārtiśanamaṃ nātmanaḥ puṇyakāraṇam /
dattālpamūlyenāptaṃ yat svalpadānena tena kim // 6.21 //
durgraheṣu viruddheṣu daśāpāke 'tidāruṇe /
dīyate doṣaśāntyai yad bhayadānena tena kim // 6.22 //
mumūrṣastyaktasarvāśaḥ śayanastho dadāti yat /
mūrcchāsthānena manasā mohadānena tena kim // 6.23 //
dattaṃ priyaviyogograśokaśalyārtacetasā /
yatpaścāttāpajananaṃ bāṣpadānena tena kim // 6.24 //
purohitāya gurave śāntisvastividhāyine /
dīyate yatprasaṅgena bhṛtidānena tena kim // 6.25 //
yatsaṃtyaktaphalaspṛhaṃ yaducitaṃ sarvasvabhūtaṃ ca yan nānyāyena yadarjitaṃ paradhanasparśena śaptaṃ na yat /
dattvā duḥkhaśataṃ na yatsvavacasā paścānna yadgaṇyate taddānaṃ dhanabījavāpanipuṇaḥ śeṣaḥ prakāraḥ kṛṣeḥ // 6.26 //
prāptuṃ svargavarāṅganāstanataṭasparśātiriktaṃ sukhaṃ datto merurapi prayāti tṛṇatāmātmopakārecchayā /
āpannārtivilokane karuṇayā śraddhāsudhāpūritaṃ sattvotsāhasamanvitaṃ tṛṇamapi trailokyadānādhikam // 6.27 //
yudhiṣṭhirasya bhūbhartuḥ purā kanakavarṣiṇaḥ /
aśvamedhe vidhānena vartamāne mahākratau // 6.28 //
sajjāsu rājabhojyāsu vividhāsvannapāliṣu /
aniśaṃ ratnapātreṣu bhuñjāneṣu dvijanmasu // 6.29 //
vipreṣu pūryamāṇeṣu maṇikāñcanaśāsanaiḥ /
ucchiṣṭabhūmiṃ nakulaḥ svabilātsamupāyayau // 6.30 //
dīptakāñcanavarṇena pārśvenaikena śobhitaḥ /
apareṇāsuvarṇena vitīrṇajanakautukaḥ // 6.31 //
so 'bhyetya tūrṇamucchiṣṭa hemapātracyute 'mbhasi /
luloṭha śapharotphālaparivartavivartanaiḥ // 6.32 //
suvarṇapārśvaṃ nakulaṃ dṛṣṭvā sarve kutūhalāt /
majjantamucchiṣṭajale kṣitipāya nyavedayan // 6.33 //
prāptena bhūbhujā dṛṣṭvā dṛṣṭyā pṛṣṭa iveṣṭayā /
so 'vadadvismayabhuvā suspaṣṭākṣarayā girā // 6.34 //
rājannasyātidānasya na paśyāmyucitaṃ phalam /
asmātprabhūtasaṃbhārāt saktupātraṃ varaṃ param // 6.35 //
pravṛtte 'sminmahādāne mahataste mahīpate /
vitte vṛtte ca citte ca śuddhiṃ ko vetti tattvataḥ // 6.36 //
saṃbhāro 'yaṃ bhuvanabhavanavyāptiparyāptabhogaḥ sarvāśāsu pratatajanatāpūraṇe 'tyantatucchaḥ /
āpannārtipraśamanavidhau sattvaśuddhipradāne saṃnaddhānāmapi tṛṇakaṇaḥ kāñcanādritvameti // 6.37 //
śrūyatām yanmayā dṛṣṭaṃ bhūpate svayamadbhutam /
udetyudīrite yasmin kāye romāñcakañcukaḥ // 6.38 //
śiloñchavṛttinā pūrvaṃ vipreṇa kṣetracāriṇā /
upavāsakṛśenāptaṃ yavastokaṃ kalatriṇā // 6.39 //
saktupātre tataḥ siddhe kṛtadevapitṛkriyaḥ /
jāyāputravibhāgena svaṃ bhāgaṃ bhoktumudyayau // 6.40 //
sa prāṇāhutitoyārthī dadarśātithimāgatam /
kṣutkṣāmakukṣiṃ saṃkṣiptasarvāṅgaśithilākṛtim // 6.41 //
tasmai vihitasatkāraḥ saprasādena cetasā /
śraddhāsudhāvasiktaṃ tat sa dadau nijabhojanam // 6.42 //
nigīrṇo 'tithinā tasminn akṣīṇakṣudvikāriṇā /
tadbhāryāpyādaravatī tasmai svamaśanaṃ dadau // 6.43 //
tenāpyatṛptimālokya tatsūnuḥ śraddhayātithim /
svabhojanena vidadhe saṃpūrṇāśananirvṛttam // 6.44 //
gate bhuktvātithau tasminn upavāsakṛśo dvijaḥ /
sattvotsāhayutastasthau klānto 'pi niśi nirvyathaḥ // 6.45 //
athāhaṃ saktugandhena nirgataḥ kṣudhito bilāt /
prāptastatparṇakuṭikām utsṛṣṭocchiṣṭavartinīm // 6.46 //
tatrācamanatoyena spṛṣṭamātrasya me nṛpa /
pasya me dakṣiṇaṃ pārśvaṃ jataṃ hemamayacchavi // 6.47 //
tato 'haṃ vāmapārśvasya hemacchāyāptaye sadā /
nirnidraścintayā yātaḥ kṛśatāmeva kevalam // 6.48 //
yadyatprāpnoti puruṣaḥ karmayogātsamīhitam /
tattatsaṃpūraṇāyaiva yāti cintāvidheyatām // 6.49 //
adhunā vartamāne 'sminn aśvamedhe tava kratau /
hemapārśvāśayāyāto viprocchiṣṭāmahīmaham // 6.50 //
ratnakāñcanapātrāmbusiktasya luṭhataściram /
mama kāntilavo 'pyaṅge na kaścidiha dṛśyate // 6.51 //
sarvathā sattvaśuddhāya dānāyātilaghīyase /
namo mahāphalāyaiva na bhogāṅgaprasaṅgine // 6.52 //
ityuktvā nakule yāte tattatheti yudhiṣṭhiraḥ /
vicintya saṃtatocchvāsaḥ kṣaṇaṃ stimitatāṃ yayau // 6.53 //
tasmātsuvarṇāmbararatnabhūmidānairna darpaḥ puruṣeṇa kāryaḥ /
bhavatyudāraṃ karuṇārdrasattvaṃ dānaṃ sadā kasyacideva puṇyaiḥ // 6.54 //
||saptamo vicāraḥ
tapaḥ sadā rāgadhanābhimānamohaprahāṇāya satāmabhīṣṭam /
tenaiva darpo yadi kiṃ vṛthaiva tyakto nikāyaḥ kṣapitaśca kāyaḥ // 7.1 //
sarvātmanā śuddhadhiyā vidheyaḥ saṃsāradoṣapraśamāya yatnaḥ /
kopopataptaṃ dhanarāgadigdhaṃ karoti tīvraṃ na tapaḥ praśāntim // 7.2 //
cittaṃ viraktaṃ yadi kiṃ tapobhiś cittaṃ sarāgaṃ yadi kiṃ tapobhiḥ /
cittaṃ prasannaṃ yadi kiṃ tapobhiś cittaṃ sakopaṃ yadi kiṃ tapobhiḥ // 7.3 //
kopena śāpaśpuritādharāṇāṃ kāmena kampasphuritādharāṇām /
svedāmbhasā tulyasamudbhavena nistejasāṃ kiṃ tapasā munīnām // 7.4 //
bhāryāpyahalyā kila gautamasya kruddhasya śāpena śilā babhūva /
nīto vasiṣṭhena ruṣābhiśaptaś caṇḍālatāṃ bhūmipatistraśaṅkuḥ // 7.5 //
bhūmagnamūrtirvararatnalobhād vipāṭayantīṃ nayane sukanyām /
tatpāṇisaṃsparśasukhādareṇa sehe nikāraṃ cyavanaḥ sarāgaḥ // 7.6 //
pāṇḍuḥ priyākaṇṭhavilambibāhur yayau stananyastatanuryadastam /
dagdhaḥ parīkṣitphaṇiphūtkṛtairyat tapasvikopasya vijṛmbhitaṃ tat // 7.7 //
visārasaṃsāratarorna yena niḥśeṣamunmūlitameva mūlam /
śāpopatāpaprabhuṇā pareṣāṃ kiṃ tena mithyātapasā munīnām // 7.8 //
na rājasevārajasā viluptaṃ na bhūmividyādivivādataptam /
na dambhadīkṣākuhakākulaṃ yat kalyāṇamittraṃ vimalaṃ vrataṃ tat // 7.9 //
sasaṃcayaṃ guptakalatraputraṃ punargṛhītavyavāharabhāram /
dambhābhimānodbhavakaṣṭabhūtaṃ mithyāvrataṃ jīvitavṛttyupāyaḥ // 7.10 //
sarāgarogaṃ bahulapramohaṃ sarogasaṃbhāramakharvagarvam /
pradveṣadoṣoṣṇamameyamāyaṃ saṃsāracihnaṃ vratametadagryam // 7.11 //
jaṭākṣasūtrājinayogapaṭṭakanyādṛḍhagranthinipīḍyamānam /
vivekahīnaṃ virataprakāśaṃ vrataṃ bṛhadbandhanamāmananti // 7.12 //
sarāgakāṣāyakaṣāyacittaṃ śīlāṃśukatyāgadigambaraṃ vā /
laulyodbhavadbhasmabharaprahāsaṃ vrataṃ na veṣodbhaṭatulyavṛttam // 7.13 //
niḥsaṅgayogaṃ dhanabhogasaṅgaṃ vilambikaṅkālakapālamālam /
kopākulaṃ sparśavivarjanīyaṃ bhāravrataṃ tatkathayanti pāpam // 7.14 //
bālastapasvī kimato 'sti hāsyaṃ yuvā vanaiṣī kimato 'styayogyam /
vṛddha sarāgaḥ kimato 'sti nindyaṃ mūrkhaḥ pramātā kimato 'sti śocyam // 7.15 //
kṣamā śamaḥ śāsanamindriyāṇāṃ manaḥ prasiktaṃ karuṇāmṛtena /
tapo 'rhametatsajane vane vā kāyasya saṃśoṣaṇamanyadāhuḥ // 7.16 //
himācale śyāmaladevadāruvane purā nirjharacāruhāsye /
tapasyatāṃ śoṣajuṣāṃ munīnāṃ kālo yayau varṣasahasrasaṃkhyaḥ // 7.17 //
tataḥ kadācidbhagavānbhavārtihārī vihārāya nabhaḥpathena /
samaṃ bhavānyā vṛṣabhādhirūḍhaḥ samāyayau śītamayūkhamauliḥ // 7.18 //
tasyoditānāṃ vadanaprabhāṇāṃ dīrghīkṛtānekaśaśiprabhāṇām /
vilāsahāsyena nabho babhūva vibhaktisaṃsaktasitottarīyam // 7.19 //
devī vilokyātha tapaḥprayatnatīvraprayāsaprakaṭāsthiśeṣān /
munīnkṛpāveśaviṣaṇṇacittā śaśāṅkalekhābharaṇaṃ babhāṣe // 7.20 //
deva tvadārādhananiścalānāṃ saṃtyaktasarvāgrahanigrahāṇām /
tapa kriyāśoṣitavigrahāṇāṃ nādyāpi muktiḥ kimaho munīnām // 7.21 //

bhavatpadaṃ nityasukhāya naiva nirāmayaṃ tanmunayaḥ prayānti // 7.22 //

iti priyāyāḥ praṇayopapannam ākarṇya vākyaṃ giriśo 'bravīttām /

na nirvikāraṃ padamāpnuvanti kopena kāmena ca kṛṣyamāṇāḥ // 7.26 //
pratyakṣameṣāṃ manaso vikāraṃ saṃdarśayāmyeṣa niṣaktamantaḥ /
tīvravrataiḥ śuṣyati kāya eva na vāsanālīnaghanapramohaḥ // 7.27 //

bhajantyavikleśatapaḥprasaktā yuktāḥ prakāmaṃ padamavyayaṃ tat // 7.28 //
uktveti śaṃbhurvṛṣabhātsalīlaṃ girerivāgrādavatīrya bhūmim /
kṣaṇādabhūdadbhutarūparāśir nagnavrataḥ kāntisudhāvadātaḥ // 7.29 //
tasyāmarādhīśakirīṭaratnaśoṇaprabhārdrāviva pādapadmau /
pracakraturvidrumabālavallīnavaprarohādbhutagarvamurvyām // 7.30 //
suspaṣṭajānu pracitoruśobhi nābhihradāvartavibhaktamadhyam /
tattasya rūpaṃ pravilambibāhoḥ pīnāṃsamāsīnmukhapūrṇacandram // 7.31 //
ananyalāvaṇyasudhābdhimadhyasnātairivāṅgaiḥ sphaṭikāvadātaiḥ /
cakre daśāśāḥ sa pṛthuprakāśā digambaratvādiva jātahāsāḥ // 7.32 //
pāṇisthitaśyāmamayūrapicchacchāyācchaṭāvicchurito 'sya kaṇṭhaḥ /
rarāja līnāntarakālakūṭamiṣāgninevārpitadhūmalekhaḥ // 7.33* //

pūrvāpakārasmṛtijātalajjaṃ cakṣuḥ kṣaṇaṃ kvāpi yayau tṛtīyam // 7.35 //
babhau sa kāntaḥ kuṭilāsitena skandhaspṛśā kuntalasaṃcayena /
anveṣṭumiṣṭāṃ mukuṭendulekhāṃ niśāgaṇeneva samāgatena // 7.36 //
latāvadhūpallavapāṇimuktaiḥ smitāvadātairvibabhau ca puṣpaiḥ /
rūpāntare nihnutajahnukanyāphenāvaśeṣairiva kīrṇakeśaḥ // 7.37 //
tenānyarūpeṇa kṛtā navaiva kāntiścakāśe nijarūpaguptyai /

prāpuḥ prayātāḥ kṣitimakṣamālā bhrūbhaṅgatāṃ tasya tapovanasya // 7.58 //

āsītpramohapratimo 'ndhakāraḥ śāpākṣarāmairbhramarairbhramadbhiḥ // 7.59 //

kaṇṭhasthalālokanakālakūṭasaṃpūritākṣā iva mohamūrcchām // 7.60 //

patnīvikārogranikāramūcuḥ kaṇṭhāntaraśvāsavikīrṇavarṇāḥ // 7.61 //

anena saṃsūcayatā nigūḍharūpeṇa darpādakulīnabhāvam /
nītā pavitratvamiyaṃ munīnāṃ kāpālikeneva vanāntabhūmiḥ // 7.63 //
aho batāsya pratibhā prasahya satīsamāliṅganasaspṛhasya /
kenāpi kāmatkuhakakrameṇa kāntaṃ kṛtaṃ rūpamanena nūnam // 7.64 //
uktveti tasmai sasṛjuḥ sakopās te daṇḍapāṣāṇabṛsīśatāni /
dveṣāvṛtākṣṇāmavivekajanmā mohaḥ pramāde gurutāmupaiti // 7.65 //
dūre bhavatyatha śanaiḥ śiśirāṃśumaulau teṣaṃ prakopavipulānalatāpitānām /
taddarśanānusaraṇaprasṛtasya yatnaḥ patnījanasya sutarāṃ vinivartane 'bhūt // 7.66 //
atha sa bhagavānbhargaḥ svargāpagāpṛthunirjharaprasṛtahasitastasmādeśātkrameṇa tirohitaḥ /
praśamavimalaṃ vyoma vyāpya priyāmavadatsmayasmitasitamukhīṃ dṛṣṭaṃ devi tvayā municeṣṭitam // 7.67 //
bhasmasmeraśarīratā pṛthujaṭābandhaḥ śiromuṇḍanaṃ kuṇḍī daṇḍakamaṇḍalupraṇayitā carmākṣasūtragrahaḥ /
kāṣāyavyasanaṃ nirambararuciḥ kaṅkālamālādhṛtiḥ kāmakrodhavaśādviśeṣaruciraṃ sarvaṃ vṛthaiva vratam // 7.68 //
darpotkopātparaṇitajaṭāsūtrabandhacca mohād antaḥsīdatsarasaviṣayāsvādasaṃvādasaṅgāt /
āśāpāśavyasananicayādvāsanālīnadoṣān naiṣāṃ muktirbhavati tapasā kāyasaṃśoṣaṇena // 7.69 //
ityuktaṃ tripurāriṇā girisutā śrutvā yathārthaṃ vaco niścitya vratamapraśāntamanasāṃ mithyaiva kāyakṣayam /
saṃsāroparamāya moharajasaḥ śāntyai munīnāṃ paraṃ rāgadveṣavimuktaye ca dayayā cakre harasyārthanām // 7.70 //
devyārthito 'tha bhagavānkṛpayā smarāris teṣāmanugrahamayena vilokanena /
cakre smitasnapitadigvadano munīnāṃ līnasya moharajasaḥ sahasaiva śāntim // 7.71 //
tapoviśeṣairniśitaprayatnais tasmānna kāryaḥ pṛthumohadarpaḥ /
dveṣeṇa rāgeṇa mohadayena tapaḥ kṣayaṃ yāti saha smayena // 7.72 //
praśānto 'ntastṛṣṇāviṣamaparitāpaḥ śamajalair aśeṣaḥ saṃtoṣāmṛtavisarapānena vapuṣaḥ /
asaṅgaḥ saṃbhogaḥ kamaladalakīlālatulyā bhavāraṇye puṃsāṃ parahitamudāraṃ khalu tapaḥ // 7.73 //
