#Author:Samantabhadra
#Text:Āptamīmāṃsā
#Data entry:Diwakar Acharya
#Edition:Gajādhara Lal Jain. Benares : Jain Siddhant Prakashini Sanstha, 1914.



devāgamanabhoyānacāmarādivibhūtayaḥ /
māyāviṣvapi dṛśyante nātastvamasi no mahān // 1 //
adhyātmaṃ bahirapyeṣa vigrahādimahodayaḥ /
divyaḥ satyo divaukassvapyasti rāgādimatsu saḥ // 2 //
tīrthakṛtsamayānāṃ ca parasparavirodhataḥ /
sarveṣāmāptatā nāsti kaścideva bhavedguruḥ // 3 //
doṣāvaraṇayorhānirniḥśeṣāstyatiśāyanāt /
kvacidyathā svahetubhyo bahirantarmalakṣayaḥ // 4 //
sūkṣmāntaritadūrārthāḥ pratyakṣāḥ kasyacidyathā /
anumeyatvato 'gnyādiriti sarvajñasaṃsthitiḥ // 5 //
sa tvamevāsi nirdoṣo yuktiśāstrāvirodhavāk /
avirodho yadiṣṭaṃ te prasiddhena na bādhyate // 6 //
tvanmatāmṛtabāhyānāṃ sarvathaikāntavādinām /
āptābhimānadagdhānāṃ sveṣṭaṃ dṛṣṭena bādhyate // 7 //
kuśalākuśalaṃ karma paralokaśca na kvacit /
ekāntagraharakteṣu nātha svaparavairiṣu // 8 //
bhāvaikānte padārthānāmabhāvānāmapahnavāt /
sarvātmakamanādyantamasvarūpamatāvakam // 9 //
kāryadravyamanādi syātprāgabhāvasya nihnave /
pradhvaṃsasya ca dharmasya pracyave 'nantatāṃ vrajet // 10 //
sarvātmakaṃ tadekaṃ syādanyāpohavyatikrame /
anyatra samavāyena vyapadiśyeta sarvathā // 11 //
abhāvaikāntapakṣe 'pi bhāvāpahnavavādinām /
bodhavākyaṃ pramāṇaṃ na kena sādhanadūṣaṇam // 12 //
virodhānnobhayaikātmyaṃ syādvādanyāyavidviṣām /
avācyataikānte 'pyuktirnāvācyamiti yujyate // 13 //
kathaṃcitte sadeveṣṭaṃ kathañcidasadeva tat /
tathobhayamavācyaṃ ca nayayogānna sarvathā // 14 //
sadeva sarvaṃ ko necchetsvarūpādicatuṣṭayāt /
asadeva viparyāsānna cenna vyavatiṣṭhate // 15 //
kramārpitadvayād dvaitaṃ sahāvācyamaśaktitaḥ /
avaktavyottarāḥ śeṣāstrayo bhaṅgāḥ svahetutaḥ // 16 //
astitvaṃ pratiṣedhyenāvinābhāvyekadharmiṇi /
viśeṣaṇatvātsādharmyaṃ yathā bhedavivakṣayā // 17 //
nāstitvaṃ pratiṣedhyenāvinābhāvyekadharmiṇi /
viśeṣaṇatvādvaidharmyaṃ yathā 'bhedavivakṣayā // 18 //
vidheyapratiṣedhyātmā viśeṣyaḥ śabdagocaraḥ /
sādhyadharmo yathā heturahetuścāpyapekṣayā // 19 //
śeṣabhaṅgāśca netavyā yathoktanayayogataḥ /
na ca kaścidvirodho 'sti munīndra tava śāsane // 20 //
evaṃ vidhiniṣedhābhyāmanavasthitamarthakṛt /
neti cenna yathākāryaṃ bahirantarupādhibhiḥ // 21 //
dharme dharme 'nya evārtho dharmiṇo 'nantadharmaṇaḥ /
aṅgitve 'nyatamāntasya śeṣāntānāṃ tadaṅgatā // 22 //
ekānekavikalpādāvuttaratrāpi yojayet /
prakriyāṃ bhaṅginīmenāṃ nayairnayaviśāradaḥ // 23 //
advaitākāntapakṣe 'pi dṛṣṭo bhedo virudhyate /
kārakāṇāṃ kriyāyāśca naikaṃ svasmātprajāyate // 24 //
karmadvaitaṃ phaladvaitaṃ lokadvaitaṃ ca no bhavet /
vidyāvidyādvayaṃ na syādbandhamokṣadvayaṃ tathā // 25 //
hetoradvaitasiddhiśced dvaitaṃ syāddhetusādhyayoḥ /
hetunā cedvinā siddhirdvaitaṃ vāṅmātrato na kim // 26 //
advaitaṃ na vinā dvaitādaheturiva hetunā /
saṃjñinaḥ pratiṣedho na pratiṣedhyādṛte kvacit // 27 //
pṛthaktvaikāntapakṣe 'pi pṛthaktvādapṛthakkṛtau /
pṛthaktve na pṛthaktvaṃ syādanekastho hyasau guṇaḥ // 28 //
santānaḥ samudāyaśca sādharmyaṃ ca niraṅkuśaḥ /
pretyabhāvaśca tatsarvaṃ na syādekatvanihnave // 29 //
sadātmanā ca bhinnaṃ cejjñānaṃ jñeyād dvidhā 'pyasat /
jñānābhāve kathaṃ jñeyaṃ bahirantaśca te dviṣām // 30 //
sāmānyārthā giro 'nyeṣāṃ viśeṣo nābhilapyate /
sāmānyābhāvatasteṣāṃ mṛṣaiva sakalā giraḥ // 31 //
virodhānnobhayaikātmyaṃ syādvādanyāyavidviṣām /
avācyataikānte 'pyuktirnāvācyamiti yujyate // 32 //
anapekṣe pṛthaktvaikyaṃ hyavastu dvayahetutaḥ /
tadevaikyaṃ pṛthaktvaṃ ca svabhedaiḥ sādhanaṃ yathā // 33 //
satsāmānyāttu sarvaikyaṃ pṛthagdravyādibhedataḥ /
bhedābhedavivakṣāyāmasādhāraṇahetuvat // 34 //
vivakṣā cāvivakṣā ca viśeṣye 'nantadharmiṇi /
sato viśeṣaṇasyātra nāsatastaistadarthibhiḥ // 35 //
pramāṇagocarau santau bhedābhedau na saṃvṛtī /
tāvekatrāviruddhau te guṇamukhyavivakṣayā // 36 //
nityatvaikāntapakṣe 'pi vikriyā nopapadyate /
prāgeva kārakābhāvaḥ kva pramāṇaṃ kva tatphalam // 37 //
pramāṇakārakairvyaktaṃ vyaktaṃ cedindriyārthavat /
te ca nityaṃ vikāryaṃ kiṃ sādhoste śāsanādbahiḥ // 38 //
yadi satsarvathā kāryaṃ puṃvannotpattumarhati /
pariṇāmaprakḷptiśca nityatvaikāntabādhinī // 39 //
puṇyapāpakriyā na syātpretyabhāvaḥ phalaṃ kutaḥ /
bandhamokṣau ca teṣāṃ na yeṣāṃ tvaṃ nāsi nāyakaḥ // 40 //
kṣaṇikaikāntapakṣe 'pi pretyabhāvādyasambhavaḥ /
pratyabhijñādyabhāvānna kāryārambhaḥ kutaḥ phalam // 41 //
yadyasatsarvathā kāryaṃ tanmā jani khapuṣpavat /
mopādānaniyāmo bhūnmāśvāsaḥ kāryajanmani // 42 //
na hetuphalabhāvādiranyabhāvādananvayāt /
santānāntaravannaikaḥ santānastadvataḥ pṛthak // 43 //
anyeṣvananyaśabdo 'yaṃ saṃvṛtirna mṛṣā katham /
mukhyārthaḥ saṃvṛtirnāsti vinā mukhyānna saṃvṛtiḥ // 44 //
catuṣkoṭervikalpasya sarvānteṣūktyayogataḥ /
tattvānyatvamavācyaṃ ca tayoḥ santānatadvatoḥ // 45 //
avaktavyacatuṣkoṭivikalpo 'pi na kathyatām /
asarvāntamavastu syādaviśeṣyaviśeṣaṇam // 46 //
dravyādyantarabhāvena niṣedhaḥ saṃjñinaḥ sataḥ /
asadbhedo na bhāvastu sthānaṃ vidhiniṣedhayoḥ // 47 //
avastvanabhilāpyaṃ syātsarvāntaiḥ parivarjitam /
vastvevāvastutāṃ yāti prakriyāyā viparyayāt // 48 //
sarvāntāścedavaktavyāsteṣāṃ kiṃ vacanaṃ punaḥ /
saṃvṛttiścenmṛṣaivaiṣā paramārthaviparyayāt // 49 //
aśakyatvādavācyaṃ kimabhāvātkimabodhataḥ /
ādyatoktidvayaṃ na syātkiṃ vyājenocyatāṃ sphuṭam // 50 //
hinastyanabhisaṃdhātṛ na hinastyabhisandhimat /
badhyate taddvayāpetaṃ cittaṃ baddhaṃ na mucyate // 51 //
ahetukatvānāśasya hiṃsāheturna hiṃsakaḥ /
cittasantatināśaśca mokṣo nāṣṭāṅgahetukaḥ // 52 //
virūpakāryārambhāya yadi hetusamāgamaḥ /
āśrayibhyāmananyo 'sāvaviśeṣādayuktavat // 53 //
skandhāḥ santatayaścaiva saṃvṛtitvādasaṃskṛtāḥ /
sthityutpattivyayāsteṣāṃ na syuḥ kharaviṣāṇavat // 54 //
virodhānnobhayaikātmyaṃ syādvādanyāyavidviṣām /
avācyataikānte 'pyuktirnāvācyamiti yujyate // 55 //
nityaṃ tatpratyabhijñānānnākasmāttadavicchidā /
kṣaṇikaṃ kālabhedātte buddhyasañcaradoṣataḥ // 56 //
na sāmānyātmanodeti na vyeti vyaktamanvayāt /
vyetyudeti viśeṣātte sahaikatrodayādi sat // 57 //
kāryotpādaḥ kṣayo heturniyamāllakṣaṇātpṛthak /
na tau jātyādyavasthānādanapekṣāḥ khapuṣpavat // 58 //
ghaṭamaulisuvarṇārthī nāśotpādasthitiṣvayam /
śokapramohamādhyasthyaṃ jano yāti sahetukam // 59 //
payovrato na dadhyatti na payo 'tti dadhivrataḥ /
agorasavrato nobhe tasmāttattvaṃ trayātmakam // 60 //
kāryakāraṇanānātvaṃ guṇaguṇyanyatāpi ca /
sāmānyatadvadanyatvaṃ caikāntena yadīṣyate // 61 //
ekasyānekavṛttirna bhāgābhāvādbahūni vā /
bhāgitvādvāsya naikatvaṃ doṣo vṛtteranārhate // 62 //
deśakālaviśeṣe 'pi syādvṛttiryutasiddhivat /
samānadeśatā na syānmūrtakāraṇakāryayoḥ // 63 //
āśrayāśrayibhāvānna svātantryaṃ samavāyinām /
ityayuktaḥ sa sambandho na yuktaḥ samavāyibhiḥ // 64 //
sāmānyaṃ samavāyaścāpyekaikatra samāptitaḥ /
antareṇāśrayaṃ na syānnāśotpādiṣu ko vidhiḥ // 65 //
sarvathānabhisambandhaḥ sāmānyasamavāyayoḥ /
tābhyāmartho na sambaddhastāni trīṇi khapuṣpavat // 66 //
ananyataikānte 'ṇūnāṃ saṃghāte 'pi vibhāgavat /
asaṃhatatvaṃ syādbhūtacatuṣkaṃ bhrāntireva sā // 67 //
kāryabhrānteraṇubhrāntiḥ kāryaliṅgaṃ hi kāraṇam /
ubhayābhāvatastatsthaṃ guṇajātītaracca na // 68 //
ekatve 'nyatarābhāvaḥ śeṣābhāvo 'vinābhuvaḥ /
dvitvasaṅkhyāvirodhaśca saṃvṛtiścenmṛṣaiva sā // 69 //
virodhānnobhayaikātmyaṃ syādvādanyāyavidviṣām /
avācyataikānte 'pyuktirnāvācyamiti yujyate // 70 //
dravyaparyāyayoraikyaṃ tayoravyatirekataḥ /
pariṇāmaviśeṣācca śaktimacchaktibhāvataḥ // 71 //
saṃjñāsaṃkhyāviśeṣācca svalakṣaṇaviśeṣataḥ /
prayojanādibhedācca tannānātvaṃ na sarvathā // 72 //
yadyāpekṣikasiddhiḥ syānna dvayaṃ vyavatiṣṭhate /
anāpekṣikasiddhau ca na sāmānyaviśeṣatā // 73 //
virodhānnobhayaikātmyaṃ syādvādanyāyavidviṣām /
avācyataikānte 'pyuktirnāvācyamiti yujyate // 74 //
dharmadharmyavinābhāvaḥ sidhyatyanyonyavīkṣayā /
na svarūpaṃ svato hyetatkārakajñāpakāṅgavat // 75 //
siddhaṃ ceddhetutaḥ sarvaṃ na pratyakṣādito gatiḥ /
siddhaṃ cedāgamātsarvaṃ viruddhārthamatānyapi // 76 //
virodhānnobhayaikātmyaṃ syādvādanyāyavidviṣām /
avācyataikānte 'pyuktirnāvācyamiti yujyate // 77 //
vaktaryanāpte yaddhetoḥ sādhyaṃ taddhetusādhitam /
āpte vaktari tadvākyātsādhyamāgamasādhitam // 78 //
antaraṅgārthataikānte buddhivākyaṃ mṛṣākhilam /
pramāṇābhāsamevātastatpramāṇādṛte katham // 79 //
sādhyasādhanavijñapteryadi vijñaptimātratā /
na sādhyaṃ na ca hetuśca pratijñāhetudoṣataḥ // 80 //
bahiraṅgārthataikānte pramāṇābhāsanihnavāt /
sarveṣāṃ kāryasiddhiḥ syādviruddhārthābhidhāyinām // 81 //
virodhānnobhayaikātmyaṃ syādvādanyāyavidviṣām /
avācyataikānte 'pyuktirnāvācyamiti yujyate // 82 //
bhāvaprameyāpekṣāyāṃ pramāṇābhāsanihnavaḥ /
bahiḥprameyāpekṣāyāṃ pramāṇaṃ tannibhaṃ ca te // 83 //
jīvaśabdaḥ sabāhyārthaḥ saṃjñātvāddhetuśabdavat /
māyādibhrāntisaṃjñāśca māyādyaiḥ svaiḥ pramoktivat // 84 //
buddhiśabdārthasaṃjñāstāstisro buddhyādivācikāḥ /
tulyā buddhyādibodhāśca trayastatpratibimbikāḥ // 85 //
vaktṛśrotṛpramātṝṇāṃ vākyabodhapramāḥ pṛthak /
bhrāntāveva pramābhrāntau bāhyārthau tādṛśetarau // 86 //
buddhiśabdapramāṇatvaṃ bāhyārthe sati nāsati /
satyānṛtavyavasthaivaṃ yujyate 'rthāptyanāptiṣu // 87 //
daivādevārthasiddhiśceddaivaṃ pauruṣataḥ katham /
daivataścedanirmokṣaḥ pauruṣaṃ niṣphalaṃ bhavet // 88 //
pauruṣādeva siddhiścetpauruṣaṃ daivataḥ katham /
pauruṣāccedamoghaṃ syātsarvaprāṇiṣu pauruṣam // 89 //
virodhānnobhayaikātmyaṃ syādvādanyāyavidviṣām /
avācyataikānte 'pyuktirnāvācyamiti yujyate // 90 //
abuddhipūrvāpekṣāyāmiṣṭāniṣṭaṃ svadaivataḥ /
buddhipūrvavyapekṣāyāmiṣṭāniṣṭaṃ svapauruṣāt // 91 //
pāpaṃ dhruvaṃ pare duḥkhātpuṇyaṃ ca sukhato yadi /
acetanākaṣāyau ca badhyeyātāṃ nimittataḥ // 92 //
puṇyaṃ dhruvaṃ svato duḥkhātpāpaṃ ca sukhato yadi /
vītarāgo munirvidvāṃstābhyāṃ yuñjyānnimittataḥ // 93 //
virodhānnobhayaikātmyaṃ syādvādanyāyavidviṣām /
avācyataikānte 'pyuktirnāvācyamiti yujyate // 94 //
viśuddhisaṃkleśāṅgaṃ cetsvaparasthaṃ sukhāsukham /
puṇyapāpāsravau yuktau na cedvyarthastavārhataḥ // 95 //
ajñānācceddhruvo bandho jñeyānantyānna kevalī /
jñānastokādvimokṣaścedajñānād bahuto 'nyathā // 96 //
virodhānnobhayaikātmyaṃ syādvādanyāyavidviṣām /
avācyataikānte 'pyuktirnāvācyamiti yujyate // 97 //
ajñānānmohato bandho nājñānādvītamohataḥ /
jñānastokācca mokṣaḥ syādamohānmohito 'nyathā // 98 //
kāmādiprabhavaścitraḥ karmabandhānurūpataḥ /
tacca karma svahetubhyo jīvāste śuddhyaśuddhitaḥ // 99 //
śuddhyaśuddhī punaḥ śaktī te pākyāpākyaśaktivat /
sādyanādī tayorvyaktī svabhāvo 'tarkagocaraḥ // 100 //
tattvajñānaṃ pramāṇaṃ te yugapat sarvabhāsanam /
kramabhāvi ca yajjñānaṃ syādvādanayasaṃskṛtam // 101 //
upekṣā phalamādyasya śeṣasyādānahānadhīḥ /
pūrvaṃ vājñānanāśo vā sarvasyāsya svagocare // 102 //
vākyeṣvanekāntadyotī gamyaṃ prati viśeṣakaḥ /
syānnipāto 'rthayogitvāttava kevalināmapi // 103 //
syādvādaḥ sarvathaikāntayogātkiṃvṛttacidvidhiḥ /
saptabhaṅganayāpekṣo heyādeyaviśeṣakaḥ // 104 //
syādvādakevalajñāne sarvatattvaprakāśane /
bhedaḥ sākṣādasākṣācca hyavastvanyatamaṃ bhavet // 105 //
sadharmaṇaiva sādhyasya sādharmyādavirodhataḥ /
syādvādapravibhaktārthaviśeṣavyañjako nayaḥ // 106 //
nayopanayaikāntānāṃ trikālānāṃ samuccayaḥ /
avibhrāḍ bhāvasambandho dravyamekamanekadhā // 107 //
mithyāsamūho mithyā cenna mithyaikāntatāsti naḥ /
nirapekṣā nayā mithyā sāpekṣā vastu te 'rthakṛt // 108 //
niyamyate 'rtho vākyena vidhinā vāraṇena vā /
tathānyathā ca so 'vaśyamaviśeṣyatvamanyathā // 109 //
tadatadvastu vāgeṣā tadevetyanuśāsati /
na satyā syānmṛṣāvākyaiḥ kathaṃ tattvārthadeśanā // 110 //
vāksvabhāvo 'nyavāgarthapratiṣedhaniraṅkuśaḥ /
āha ca svārthasāmānyaṃ tādṛgvācyaṃ khapuṣpavat // 111 //
sāmānyavāgviśeṣe cenna śabdārthā mṛṣā hi sā /
abhipretaviśeṣāpteḥ syātkāraḥ satyalāñchanaḥ // 112 //
vidheyamīpsitārthāṅgaṃ pratiṣedhavirodhi yat /
tathaivādeyaheyatvamiti syādvādasaṃsthitiḥ // 113 //
itīyamāptamimāṃsā vihitā hitamicchatā /
samyaṅmithyopadeśārthaviśeṣapratipattaye // 114 //
jayati jagati kleśāveśaprapaṃcahimāṃśumān vihitaviṣamaikāntadhvāntapramāṇanayāṃśumān /
yatipatirajo yasyādhṛṣṭānmatāmbunidherlavān svamatamatayastīrthyā nānā pare samupāsate // 115 //
