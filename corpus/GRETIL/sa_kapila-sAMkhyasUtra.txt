atha trividhaduḥkhātyantanivṛttiḥ atyanta puruṣārthaḥ // 1.1 //
||paramapuruṣārthasvarūpaṃ
na dṛṣṭāt tatsiddhiḥ, nivṛtte 'pyanuvṛtti darśanāt // 1.2 //
||laukikopāyaiḥ duḥkhanivṛttyanupapattiḥ
prātyahika kṣutpratīkāravat tatpratīkāra ceṣṭanāt puruṣārthatvaṃ // 1.3 //
||laukikopāyaiḥ duḥkhanivṛttyanupapattiḥ
sarvāsambhavāt, sambhave 'pi sattāsambhavāt heyaḥ pramāṇakuśalaiḥ // 1.4 //
||laukikopāyaiḥ duḥkhanivṛttyanupapattiḥ
utkarṣādapi mokṣasya sarvotkarṣaśruteḥ // 1.5 //
||laukikopāyaiḥ duḥkhanivṛttyanupapattiḥ
aviśeṣaścobhayoḥ // 1.6 //
||vaidikopāyānāmapi duḥkhanivṛttyanupāyatā
na svabhāvato baddhasya mokṣasādhanopadeśavidhiḥ // 1.7 //
||bandhasya svābhāvikatvānupapattiḥ
svabhāvasyānapāyitvāt, ananuṣṭhānalakṣaṇamaprāmāṇyaṃ // 1.8 //
||bandhasya svābhāvikatvānupapattiḥ
nāśakyopadeśavidhiḥ, upadiṣṭe 'pyanupadeśaḥ // 1.9 //
||bandhasya svābhāvikatvānupapattiḥ
śuklapaṭavat bījavaccet // 1.10 //
||bandhasya svābhāvikatvānupapattiḥ
śaktyudbhavānudbhavābhyāṃ nāśakyopadeśaḥ // 1.11 //
||bandhasya svābhāvikatvānupapattiḥ
na kālayogato vyāpino nityasya sarvasambandhāt // 1.12 //
||bandhasya kāladeśanimittakatvānupapattiḥ

||bandhasya kāladeśanimittakatvānupapattiḥ

||bandhasya avasthāviśeṣanimittakatvānupapattiḥ

||bandhasya cittagātradharmatvānupapattiḥ

||bandhasya prakṛtinimittakatvānupapattiḥ
na nityaśuddha buddhamukta svabhāvasya tadyogastadyogāhate // 1.19 //
||bandhasya prakṛtinigittakatvānupapattiḥ
nāvidyāto ' pyavastunā bandhāyogāt // 1.20 //
||bandhasya avidyānimittakatvanirāsaḥ
vastutve sindhāntahāniḥ // 1.21 //
||bandhasya avidyānimittakatvanirāsaḥ
vijātīyadvaitāpattiśca // 1.22 //
||bandhasya avidyānimittakatvanirāsaḥ
viruddhobhayarūpā cet // 1.23 //
||bandhasya avidyānimittakatvanirāsaḥ pū
na tādṛkpadārthāpratīteḥ // 1.24 //
||bandhasya avidyānimittakatvanirāsaḥ si
na vayaṃ ṣaṭpadārthavādinaḥ vaiśeṣikādivat // 1.25 //
||bandhasya avidyānimittakatvanirāsaḥ pū
aniyatatve 'pi nāyauktikasya saṃgrahaḥ, anyathā bālonmattādisamatvaṃ // 1.26 //
||bandhasya avidyānimittakatvanirāsaḥ si
nānādiviṣayoparāga nimittako 'pyasya // 1.27 //
||bandhasya viṣayavāsanānimittakatvanirāsaḥ

||bandhasya viṣayavāsanānimittakatvanirāsaḥ

||bandhasya viṣayavāsanānimittakatvanirāsaḥ
adṛṣṭavaśāccet // 1.30 //

||bandhādīnāṃ sarveṣāmapi kṣaṇikatva vādanirāsaḥ

||bandhādīnāṃ sarveṣāmapi kṣaṇikatva vādanirāsaḥ

||bandhādīnāṃ sarveṣāmapi kṣaṇikatva vādanirāsaḥ

||ātmano gativiśeṣasya bandhanimittatvanirāsaḥ
