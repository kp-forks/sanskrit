#Author:Kṣemendra
#Text:Sudhanakinnaryavadāna
#Data entry:Klaus Wille
#Edition:Martin Straube: Prinz Sudhana und die Kinnarī - Eine buddhistische Liebesgeschichte von Kṣemendra. Texte, Übersetzung, Studie. Marburg 2006 (Indica et Tibetica, 46).



abhinavakisalayakomalamanasām api kuliśakaṭhinadhairyāṇām /
mahatāṃ maṇivimalānāṃ api bhavati na rāgasaṃkrāntiḥ // 64.1 //
śākyālaye darśanam eva pitrā sarvānukampī bhṛśam arthyamānaḥ yadā yadānandasudhārasendur viveśa śāstā nijarājadhānīṃ // 64.2 //
tadā tadā harmyagatā mṛgākṣī karṇāvataṃsīkṛtanetrakāntiḥ /
kāntyā diśāṃ vismayam ullikhantaṃ yaśodharā taṃ dayitaṃ vilokya // 64.3 //
tatsaṃgamāliṅganayor nirāśā bhrāntākhilāśā viṣamūrcchiteva /
dhṛtiṃ vayasyām iva vārayantīṃ nirasya saudhāt tanum utsasarja // 64.4 //
yadā yadā pallavapeśalāṅgī dehaṃ samutsṛṣṭavatī satī sā /
tadā tadā manmathamohitāṃ tāṃ dayārdracakṣur bhagavān rarakṣa // 64.5 //
tataḥ kadā cid bhagavān vanānte tatkautukād bhikṣugaṇena pṛṣṭaḥ /
jagāda dantadyuticandrikābhir nivārayan rāgam ivādharasya // 64.6 //
yaśodharā madviraheṇa nityaṃ karoti yat sāhasam etad ārtā /
haranti dhairyaṃ vitaranti mohaṃ eṣa svabhāvaḥ smaravibhramāṇāṃ // 64.7 //
mayāpi tasyā viraheṇa pūrvaṃ janmāntare māravimohitena /
saṃsaktasaṃtāpanimittabhūtaḥ khedaḥ prabhūtavyaśano 'nubhūtaḥ // 64.8 //
purā jitāmarapure satpure hastināpure /
rājā dhanābhidhāno 'bhūn nidhānaṃ guṇasaṃpadāṃ // 64.9 //
bhujenāliṅgitā bhūmiḥ kṛtā kaṇṭhe sarasvatī /
lakṣmīḥ prasādhitā yena kīrtir eva pravāsitā // 64.10 //
rāmāyāṃ tasya jāyāyāṃ tanayaḥ sudhanābhidhaḥ /
abhavat sahasaṃjātanidhānaśataviśrutaḥ // 64.11 //
sarvavidyākumudinīvikāsarasikaḥ sadā /
kalāvān nirmalarucir yaḥ pūrṇendur ivābabhau // 64.12 //
babhūva tasya bhūbhartur bhūpatir bhūmyanantaraḥ /
mānī mahendrasenākhyaḥ prakhyātapṛthuvikramaḥ // 64.13 //
tena duḥsahadaṇḍena sarvasarvasvahāriṇā /
akālakālakalpena pīḍitāḥ śuśucuḥ prajāḥ // 64.14 //
tasyādharmapravṛttasya nivṛttasukṛtotsave /
tīvropatāpasaṃtapte pure na vavṛṣur ghanāḥ // 64.15 //
pratikūle mahīpāle tatra durbhikṣaviplavaḥ /
babhūva vyasaneṣv eva bhavanti vyasanodayāḥ // 64.16 //
tataḥ kleśaviśeṣārtāḥ sarve puranivāsinaḥ /
kupativyasanodvegād ekībhūtā vyacintayan // 64.17 //
rājñā doṣākareṇeyaṃ jaḍena janatā param /
pratyagrakarapātena nalinīva nimīlitā // 64.18 //
eṣa durvyasanī nityam asatyāmātyasaṃmataḥ /
asmān nipīḍya puṣṇāti viṭaceṭakagāyanān // 64.19 //
kaṣṭā tatrāpy anāvṛṣṭiḥ pāpaśāpena bhūpateḥ /
durbhikṣajananī jātā janasaṃkṣayasākṣiṇī // 64.20 //
sutīkṣṇaḥ kṣmāpālaḥ kharamukharamūrkhaḥ parijanaḥ kadaryāś cāmātyāḥ kaṭukapaṭakauṭilyapaṭavaḥ /
padasthāḥ kāyasthā viṣamamukharogāḥ prakupitāḥ kathaṃ sahyaḥ so 'yaṃ kṛpaṇadalano dāruṇagaṇaḥ // 64.21 //
dhanaḥ śrimān kṣitipatiḥ śrūyate rāṣṭravardhanaḥ /
gacchāmas tatpuraṃ so 'smān vatsalaḥ pālayiṣyati // 64.22 //
apatyam iva rāṣṭraṃ yaḥ sadā paśyati bhūpatiḥ /
viṣaye jīvyate tasya janakasyeva veśmani // 64.23 //
iti niścayam ādhāya yayus te hastināpuraṃ /
tyājyaḥ kāyo 'pi sāpāyaḥ kiṃ punar deśasaṃśrayaḥ // 64.24 //
jñātvā mahendraseno 'pi svapuraṃ nirjanaṃ nṛpaḥ /
tīvrānutāpaḥ sāmarṣaṃ mahāmātyān abhāṣata // 64.25 //
dhanasya dhanino rājñaḥ puraṃ matpuravāsinaḥ /
prayātā iti naś cāraiḥ kathitaṃ gūḍhacāribhiḥ // 64.26 //
durbhikṣakhinnā yadi te yātā mama ripoḥ puram /
tat sarvatra bhavanty eva paryāyair devaviplavāḥ // 64.27 //
atha vā rājadoṣena gatās te sukhavāñchayā /
viṣṭidaṇḍakaronmuktaḥ kasya rājñaḥ pure janaḥ // 64.28 //
prāyaḥ paricitadveṣāt sadā navanavaiṣiṇaḥ /
dūrastha eva sarvasya sarvo bhavati vallabhaḥ // 64.29 //
asmadabhyadhikaḥ ko 'sau tasya bhūmipater guṇaḥ /
pareṣāṃ harate yena mugdhā jāyā iva prajāḥ // 64.30 //
upāyaś cintyatāṃ tāvat tasya darpapraśāntaye /
samṛddhikāraṇaṃ yac ca tadvighāto vidhīyatāṃ // 64.31 //
iti rājavacaḥ śrutvā mahāmātyā babhāṣire /
śrūyatāṃ deva yenāsau rājā dhanajanorjitaḥ // 64.32 //
viṣaye nāgarājo 'sti citro nāma bahūdakaḥ /
tasya kṣitipater mūrtaḥ puṇyabaddha ivodayaḥ // 64.33 //
akāle sasyaniṣpattis tatprabhāveṇa jāyate /
kṛṣisaṃpattimūlāś ca bhūbhujāṃ sarvasaṃpadaḥ // 64.34 //
tasmād vidyābalān nāgaḥ sa hartuṃ yadi śakyate /
tadā yānti svayaṃ tasya prajās tvām eva saṃśrayam // 64.35 //
dīptamantrabalaṃ kaṃ cit tasmād anviṣya sādhakam /
nāgarājasya haraṇe kriyatāṃ tūrṇam udyamaḥ // 64.36 //
ity amātyavacaḥ śrutvā tathety āha mahīpatiḥ /
svaguṇādhānavikalāḥ paradoṣodyatāḥ khalāḥ // 64.37 //
tataḥ prabhūtakanakapradānapaṇaghoṣaṇām /
kṛtvā te mantriṇaḥ prāpur nāgabandhanamantriṇam // 64.38 //
sa tair vidyādharo nāma suvarṇārpaṇasaṃvidā /
prārthitaś citram ānetuṃ prayayau hastināpuram // 64.39 //
tatra kānanaparyante snigdhaśyāmalapādape /
bhavanaṃ bhogināṃ bhartuḥ sa dadarśa nabhaḥprabham // 64.40 //
valadbakulamālāyā vilasattilakaśriyaḥ /
vanalakṣmyā ivāsannamaṇḍanaṃ maṇidarpaṇam // 64.41 //
tad vilokyāmalajalaṃ spṛhāmalinamānasaḥ /
mantrānudhyānasaṃnaddhaḥ siddhyai digbandham ādadhe // 64.42 //
digbandhe vihite tena vratenātyugratejasā /
śirovyathā nāgapater abhūt taptaphaṇāmaṇeḥ // 64.43 //
athādṛśyaḥ samutthāya taṃ dṛṣṭvā mantrasādhakam /
kampamānaḥ phaṇipatir bandhatrāsād acintayat // 64.44 //
babhrubhrūśmaśruśabalas taḍitkapilalocanaḥ /
akālakālasaṃkāśaḥ kulaṃ hantuṃ mamāgataḥ // 64.45 //
vane saṃprati digbandhaḥ kṛto 'nena durātmanā /
badhnāty eṣa na yāvan māṃ tāvad yuktikramaḥ kṣamaḥ // 64.46 //
tīropānte vasaty asmin maharṣir valkalāyanaḥ /
jāne so 'pi na me sādhu paritrāṇe pragalbhate // 64.47 //
tasyāśramapade yo 'sau paricaryāparaḥ sadā /
lubdhakaḥ padmako nāma sa me saṃrakṣaṇakṣamaḥ // 64.48 //
iti niścitya manasā sa gatvā lubdhakāntikam /
svavṛttāntaṃ nivedyāsmai bandharakṣām ayācata // 64.49 //
sādhakasya vadhāyaiva nāgarājena so 'rthitaḥ /
dhanvī taṃ deśam abhyetya mantradhīraṃ vyalokayat // 64.50 //
asminn avasare mantrasādhako 'pi hutānalaḥ /
ākṛṣṭiṃ nāgarājasya vidadhe bandhanotsukaḥ // 64.51 //
mantrākṛṣṭe phaṇipatau kṣaṇaṃ tadbhavanodakam /
saviṣādaṃ rurodeva prodyadbudbudagadgadam // 64.52 // saṃtrāsavihvalabhujaṅgavadhūviṣādaniśvāsavegavihitojjvalaphenamālam /
rakṣārthanām iva cakāra taraṅgahastasaktāñjali prabalakampavilolam ambhaḥ // 64.53 //
ātte vidyābalāt tena nāge garudatejasā /
saṃkocakuṭilābhoge kṣipte ca jalabhājane // 64.54 //
taṃ hemalubdhaṃ bāṇena trāsasaṃdigdhalocanam /
vivyādha dhanur ākṛṣya viṣadigdhena lubdhakaḥ // 64.55 //
magnabāṇaṃ tam abhyetya bhūyastyaktabhujaṅgamaṃ /
karālakaravālena viśiraskaṃ cakāra saḥ // 64.56 //
sā vidyābandhakī tasya lobhād anyopayoginī /
abhūt siddhāpi mugdhasya svavināśāya kevalam // 64.57 //
yuktāḥ paropatāpāya vidyāvibhavaśaktayaḥ /
sahasaiva vimūḍhānāṃ vinaśyanti sahāsubhiḥ // 64.58 //
tataḥ praharṣasaṃpūrṇaḥ kṛtajñaḥ phaṇināyakaḥ /
taṃ nināya svabhavanaṃ lubdhakaṃ snehalubdhadhīḥ // 64.59 //
tatra ratnalatodyānavimānamaṇiveśmasu /
abhyarcyamānaś citreṇa kaṃ cit kālam uvāsa saḥ // 64.60 //
kadā cid atha nāgena pūjyamānaṃ savismayaḥ /
vidyuddāmopamaṃ pāśam amoghākhyaṃ dadarśa saḥ // 64.61 //
taṃ dṛṣṭvā tatprabhāvaṃ ca śrutvā nāganiveditam /
yatnāt tadarthanāṃ cakre lubdhako lubdhamānasaḥ // 64.62 //
ajayyaṃ samarodyoge surāṇām api bandhanaṃ /
taṃ jīvitādhikaṃ tasmai dadau prītyā phaṇīśvaraḥ // 64.63 //
tatas taṃ pāśam ādāya citram āmantrya lubdhakaḥ /
uttīrya nāgabhavanāl labdhārthaḥ svapadaṃ yayau // 64.64 //
nāgaprabhāvasaṃprāptāṃ ciraṃ bhuktvā sa saṃpadam /
putrāyotpalakākhyāya pāśaṃ dattvā vyapadyata // 64.65 //
pālayan pitur ācāraṃ so 'pi lubdhakumārakaḥ /
munes tasya kulasthityā paricaryāparo 'bhavat // 64.66 //
tataḥ kadā cit sa muner viśrāntasya sthitaḥ puraḥ /
śuśrāva kākalīgītaṃ kīrṇaṃ karṇarasāyanam // 64.67 //
gītaśravaṇaniṣpandanilīnahariṇaṃ vanam /
citranyastam ivālokya so 'pṛcchad vismito munim // 64.68 //
kamalābandhasaṃruddhamadhupadhvanisodaraḥ /
kuto 'yaṃ kokilālāpalalitaḥ śrūyate svanaḥ // 64.69 //
iti lubdhakaputreṇa pṛṣṭas taṃ munir abravīt /
gāyanti madhurālāpam etaṃ kinnarakanyakāḥ // 64.70 //
drumasya kinnarapateḥ kanyā kanyāśatair vṛtā /
pañcabhir nāgabhavane krīḍaty eṣā manoharā // 64.71 //
śrutvaitat kautukākrāntaḥ sa punar munim abhyadhāt /
api śaknoti puruṣaḥ prāptuṃ kinnarakanyakām // 64.72 //
tam uvāca munir yasya bhavet pāśaḥ karāgragaḥ /
amoghākhyaḥ sa śaknoti hartuṃ kinnarakāminīm // 64.73 //
etad ākarṇya sotsāhas taṃ praṇamyātha lubdhakaḥ /
prayayau pāśam ādāya bhogīndrabhavanāntikam // 64.74 //
tatra kelivilāsāṅkaṃ so 'paśyat kinnarīgaṇam /
bibhrāṇam anilālolahemavallīvanaśriyam // 64.75 //
madbye snānotthitāṃ tāsāṃ sa dadarśa manoharām /
smarasyeva trinetrāgninirvāṇajaladevatām // 64.76 //
anaṅgavibhramārambhanirbharormiṇi yauvane /
majjataḥ śaiśavasyeva kurvāṇām avalambanam // 64.77 //
mekhalābaddhasaṃnaddhadivyāmbaramanoramām /
jalakelikalāsaktasphāraphenāvalīm iva // 64.78 //
lāvaṇyaprasṛteneva pravāhena mahīyasā /
harantīṃ tārahāreṇa śaśiśubhraniśāśriyam // 64.79 //
kalayantīṃ jalollīḍhāṃ līlāpattralatāṃ punaḥ /
karṇābharaṇaratnāṃśukalāpena kapolayoḥ // 64.80 //
kastūrīlekhayā sakhyā likhyamānalalāṭikām /
kalaṅkakalanāklaibyaṃ śamayantīṃ himatviṣaḥ // 64.81 //
tāṃ dṛṣṭvā vismayāveśapāśenākṛṣṭamānasaḥ /
cakre pāśam amoghākhyaṃ sajjaṃ sapadi lubdhakaḥ // 64.82 //
tataḥ saṃtrāsataralāḥ sahasā hariṇekṣaṇāḥ /
kinnaryo divam utpetuḥ pāśahastaṃ vilokya taṃ // 64.83 //
laghuhastatayā kṣiptvā pāśaṃ cakitalocanām /
manoharāṃ sa jagrāha hariṇīm iva lubdhakaḥ // 64.84 //
sā pāśavivaśā tena kṛṣṭā kaṣṭadaśāṃ śritā /
kim etad iti nājñāsīt kṣaṇamūrcchānimīlitā // 64.85 //
yūthabhraṣṭeva hariṇī svajanālokanāśayā /
nirīkṣamāṇā kakubhaḥ sā saṃbhrāntā tam abhyadhāt // 64.86 //
muñca muñca dṛḍhākṣiptāṃ mā sprākṣīḥ parirakṣa mām /
krūrā api bhavanty eva śokārtiṣu kṛpālavaḥ // 64.87 //
lobhād anuttamapadaprayuktā divyakanyakā /
sadyaḥ pradīptā vidyeva nirdahaty eva sādhakam // 64.88 //
māṃ vicārya dhiyā dhīmann ucitāya prayacchataḥ /
bhaviṣyati tavāvaśyam mahān dharmadhanāgamaḥ // 64.89 //
pīḍāṃ pāśakṛtām etāṃ na sahe muñca bandhanam /
svayaṃ vrajāmy ahaṃ tatra yatra te 'bhimatā gatiḥ // 64.90 //
divam utplutya gacchāmi na tv ahaṃ muktabandhanā /
yadvaśān me gatir vyomni cūḍāratnaṃ gṛhāṇa tat // 64.91 //
ity uktaḥ sa tayā sāsraṃ prayātaḥ karuṇārdratām /
cūḍāmaṇiṃ samādāya pāśaṃ muktvā jagāda tām // 64.92 //
samāśvasihi kalyāṇi na śokaṃ kartum arhasi /
na nāmānucitāya tvāṃ prayacchāmi nijecchayā // 64.93 //
asti rājasutaḥ śrīmān guṇaratnamahodadhiḥ /
pūritāḥ kakubhaḥ sarvā yena kīrtyamṛtormibhiḥ // 64.94 //
sa vidyāvadanādarśaḥ kalākelivibhūṣaṇaḥ /
suvṛttaḥ sudhano nāma nijavaṃśaviśeṣakaḥ // 64.95 //
sa te samucitaḥ subhru vibhramābharaṇaṃ bhuvaḥ /
tyāgopabhogasubhagaḥ sukhotsava iva śriyaḥ // 64.96 //
surakinnaragandharvavidyādharavilāsinām /
yaḥ kharvīkurute garvam urvarāśarvaripatiḥ // 64.97 //
iti tena kṛtāśvāsā bandhuvargaviyoginī /
kurarīvātikaruṇaṃ vilalāpa manoharā // 64.98 //
atrāntare vindhyataṭīṃ mṛgayākelikautukī /
prasthitaḥ sudhano dhanvī śanais taṃ deśam āyayau // 64.99 //
rathanemisvanais tasya pranṛttaśikhimaṇḍalī /
vanalakṣmyāḥ kṣaṇaṃ lebhe lolanīladukūlatām // 64.100 //
sa kapolapraṇayibhir babhau svedodabindubhiḥ /
saṃkrāntaiḥ kuṇḍalaprānte kāntamuktāphalair iva // 64.101 //
syandanāśvakhuroddhūtaṃ dantadīdhitibhiḥ puraḥ /
harann iva rajaḥpuñjaṃ vyājahāra sa sārathim // 64.102 //
aho manoratheneva rathenānilaraṃhasā /
dūrojjhitā svasainyena kiyatī laṅghitā kṣitiḥ // 64.103 //
etā bālānilollāsalolapippalapallavāḥ /
haranti haritacchāyāṃ hariṇābharaṇā bhuvaḥ // 64.104 //
etā bālapravāloṣṭhyaḥ stabakastanabandhurāḥ /
sotkaṇṭhā iva jṛmbhante mañjaryaḥ śvasanākulāḥ // 64.105 //
imā marakataśyāmaśaṣpasaṃcayakañcukāḥ /
rājante kausumarajorañjitā vanarājayaḥ // 64.106 //
etā vivalitagrīvā hariṇyas trāsavidrutāḥ /
nīlotpalavanānīva sṛjanti taralekṣitaiḥ // 64.107 //
ete niśākarakarāṅkurakāntadantāḥ pallīpatipraṇayiṇīstanatulyakumbhāḥ /
paśyanti dantiśiśavaḥ parito rathaṃ me nemisvanāpaḥrtacāpalalīnakarṇāḥ // 64.108 //
ete nirmalanarmadāparisaravyājṛmbhivallīvalatpuṣpoddāmamadhūtsavapraṇayinaḥ kṣībā ivāghūrṇitāḥ /
saṃnaddhāḥ śabarīnitambavilasanmāyūrapattrāvalīlīlāndolananarmavibhramakalābandheṣu vindhyānilāḥ // 64.109 //
iti bruvāṇaḥ kalayan vanalakṣmīṃ nṛpātmajaḥ /
śuśrāva nirjane kīrṇaṃ kinnaryāḥ karuṇaṃ svanam // 64.110 //
śrutvaiva kautukākṛṣṭas tatsamīpaṃ kṛpānidhiḥ /
so 'bhyetya sadguṇādarśas tāṃ dadarśa mṛgīdṛśam // 64.111 //
yācamānām paritrāṇaṃ lubdhakaṃ sāśrulocanām /
vanecarabhayodvignām iva kānanadevatāṃ // 64.112 //
anveṣṭuṃ lubdhakākṛṣṭaṃ svam aṅkaṃrgam āgatām /
khinnām iva vanabhrāntyā lakṣmīṃ hariṇalakṣmaṇaḥ // 64.113 //
tāṃ vilokya sa sāścaryarūpātiśayavismitaḥ /
abhilāṣapaṭe kṣipraṃ citranyasta ivābhavat // 64.114 //
so 'cintayad aho ramyanirmāṇābhyāsakāriṇaḥ /
asmin mukhasamullekhe rekhāpariṇatir vidheḥ // 64.115 //
durlabhā bhogibhavane martyaloke kathaiva kā /
manye lāvaṇyamudreyaṃ svarge 'py abhinavoditā // 64.116 //
tāruṇyena nipītaśaiśavatayā sānaṅgaśṛṅgāriṇī tanvaṅgyā sakalāṅgasaṃgamasakhī bhaṅgir navāṅgīkṛtā /
niḥsaṃrambhaparākramaḥ pṛthutarārambhābhiyogaṃ vinā sāṃrājye jagatāṃ yayā vijayate devo vilāsāyudhaḥ // 64.117 //
iti vismayagarbheṇa sābhilāṣeṇa cakṣuṣa /
tāṃ pibantam ivābhyetya praṇamya prāha lubdhakaḥ // 64.118 //
kulakalpadrumasyeyaṃ drumasya dayitā sutā /
deva kinnararājasya pāśenāpaḥrtā mayā // 64.119 //
ānītā tvatkṛte divyakanyaiṣā pratigṛhyatām /
tvam evāsyā guṇodāra bhartā bhuva ivocitaḥ // 64.120 //
asyāś cūḍāmaṇir ayaṃ mayā svecchāgatipradaḥ /
gṛhītas tadvirahitā neyaṃ yāti vihāyasā // 64.121 //
rakṣyo maṇir ayaṃ nāsti datte 'smin saṃgamo 'nayā /
ity uktvāsmai dadau kanyāratnaṃ ratnaṃ ca lubdhakaḥ // 64.122 //
gṛhītā rājaputreṇa sā mahīṃrgalakṣmaṇā /
sudhāsikteva tatyāja svadeśavirahānalam // 64.123 //
utkaṇṭhālokanālolāṃ tāṃ bālahariṇīm iva /
babandha lubdhakatyaktāṃ rāgavāgurayā smaraḥ // 64.124 //
kinnarīṃ ratham āropya ratnair āpūrya lubdhakam /
harṣapūrṇaḥ svanagaraṃ pratasthe pārthivātmajaḥ // 64.125 //
sa hastināpuraṃ prāpya nivedya svakathāṃ pituḥ /

līlāvilolā lalanā vanānte lateva dṛṣṭā stabakāvanaṃrā // 64.247 //
anena nūnaṃ vanakuñjareṇa sā rājarambhā parirambhalobhāt /
ākṛṣya nītā ghanasaṃnibhena saṃcchāditā vā śaśinaḥ kaleva // 64.248 //
iti kānanakuñjeṣu tasyonmādapralāpinaḥ /
śokād iva vivarṇenduvadanā rajanī yayau // 64.249 //
sa nāgabhavanāsannatīropāntatapovanam /
śanaiḥ praviśya papraccha maharṣiṃ valkalāyanaṃ // 64.250 //
ihavirahavicintāśokaniśvāsamūrcchanmadanadahanadhūmaśyāmasaktaikaveṇī /
api śamitaśaśāṅkoddāmasaundaryadarpā munivara hariṇākṣī kinnarī kāpi dṛṣṭā // 64.251 //
iti kāntāviyuktasya prāptasyonmādinīṃ daśām /
muniḥ śrutvā vacas tasya parijñāya tam abravīt // 64.252 //
samāśvasihi viśramya saṃtāpas tyajyatām ayam /
dṛṣṭā sā tava kalyāṇī mayā mānasacandrikā // 64.253 //
yūthabhraṣṭeva hariṇī nirapekṣāpi jīvite /
pāśākṛṣṭeva kariṇī dhāryate sā tvadāśayā // 64.254 //
pāṇau śete vadanakamalaṃ prastare pallavānāṃ tāpaklāntā taralavalanasraṃsinī gātralekhā /
āśābandhe dhṛtir iva mates tvadviyogākulāyās tasyā naiva kva cid api manaḥ kiṃ tu viśrāntim eti // 64.255 //
drumasya kinnarapateḥ sthitāhaṃ bhavane pituḥ /
āgantavyaṃ tvayā tūrṇam iti tvāṃ saṃdideśa sā // 64.256 //
vīryasattvabalopāyadhairyotsāhavatām api /
agamye kinnarapure kramād vartma śaśaṃsa ca // 64.257 //
idaṃ ca tvatkṛte dattaṃ tayā ratnāṅgulīyakam /
yasya snigdhaprabhābhyaṅgair diśo yānti piśaṅgatām // 64.258 //
ity ānandasudhāsiktam uktvā dhairyāvalambanam /
aṅgulīyaṃ dadau tasmai mārgaṃ cākathayan muniḥ // 64.259 //
pathā tadupadiṣṭena sopāyena nṛpātmajaḥ /
dhīraḥ pracakrame gantuṃ dhanadādhyuṣitāṃ diśam // 64.260 //
sa siddhāṃ ghṛtapākena sudhāṃ pītvā mahauṣadhim /
labdharddhibalamāhātmyaḥ sāyudhaḥ prayayau śanaiḥ // 64.26 //
ṛddhyā saṃnihitaṃ tasya sarvopakaraṇaṃ pathi /
abhūt sattvasahāyānāṃ svādhīnāḥ sarvasaṃpadaḥ // 64.262 //
atha vidyādharavadhūvilāsahasitadyutim /
himavantam atikramya kukūlādrim avāpa saḥ // 64.263 //
phalopahāraiḥ svīkṛtya tatra vānarayūthapam /
vāyuvegākhyam āruhya sa taṃ śailam alaṅghayat // 64.264 //
athājapathanāmānam aticakrāma bhūdharam /
nihatyājagaraṃ ghoraṃ vighnasaṃgham iveṣunā // 64.265 //
vīṇāsvanair vaśīkṛtya rākṣasīṃ kāmarūpiṇīm /
kāmarūpādrim ullaṅghya prayayau kinnarīpriyaḥ // 64.266 //
balavān mudgarāghātanikhātaiḥ śastraśaṅkubhiḥ /
ekadhāraṃ tataḥ śailam ārurohātisāhasaḥ // 64.267 //
athogrataram āruhya vajrakākhyaṃ sa parvatam /
gṛdhrarūpāṃ samālokya rākṣasīṃ piśitaiṣiṇīm // 64.268 //
parivṛttena saṃcchannaḥ samāṃsamṛgacarmaṇā /
sa pādamūle tasyādres tasthau niścalavigrahaḥ // 64.269 //
māṃsalubdhā tam utkṣipya gṛdhrarūpā niśācarī /
nidadhe śikharasyāgre bhoktuṃ bhīṣaṇavigrahā // 64.270 //
mṛgacarma samutsṛjya tāṃ nihatya sa vīryavān /
nīrandhrakhadirākīrṇam prāpa khādirabhūdharam // 64.271 //
praviśya tadguhāṃ lebhe śilāṃ vyasya mahauṣadhim /
śītātapatamaḥsarparākṣasādibhayāpahām // 64.272 //
sa yantraparvatau prāpya saṃghaṭtaiḥ prāṇahāriṇau /
yantrakīlaṃ śarāgreṇa chittvā cakre viniścalau // 64.273 //
yantrakīlasamucchedair yantradvāraṃ vidārya saḥ /
chedanam yantracakraṃ ca yantrayuktau tathāyasau // 64.274 //
tīvraprahārau puruṣau yantrameṣau ca duḥsahau /
yantrogradantaniṣpeṣau tathā makararākṣasau // 64.275 //
ghorāndhakāragambhīraṃ guhākūpaṃ vilaṅghya ca /
tuṅgāṃ saritam uttīrya hatvā tatkūlarākṣasān // 64.276 //
āśīviṣāvṛtajalāṃ pataṃgākhyāṃ ca nimnagām /
rodinīṃ ca nadīṃ tīre yasyāḥ kinnaraceṭikāḥ // 64.277 //
kurvanti rodanaravair vighnaṃ tadgatacetasām /
tadvidhāṃ hāsinīṃ nāma hāsāpaḥrtacetasām // 64.278 //
diśanti puline yasyā vyasanaṃ kinnarāṅganāḥ /
laṅghayitvāpagāś cānyā vetrāṃ prāpya nadīṃ tataḥ // 64.279 //
kūlavetralatālambī tasyām atha titīrṣayā /
pavanapreritāṃ pāravetravallīm avāpa saḥ // 64.280 //
tayā prāpya paraṃ pāram āśayevātidīrghayā /
dadarśa kinnarapuraṃ sphārasphaṭikamandiram // 64.281 //
praviśya sa śanaiḥ prapya kāntāṃ kanakapadminīm /
tattīratarum āruhya tasthau ratnalatāvṛtaḥ // 64.282 //
sa dadarśāmbujarajaḥpuñjaiḥ surabhipiñjaram /
hemakumbhair jalaṃ tatra nayantīḥ kinnarāṅganāḥ // 64.283 //
kumbhotkṣepe śramārtāyās tatraikasyāḥ sametya saḥ /
hastālambena sāhāyyaṃ kṛtvā papraccha tārn śanaiḥ // 64.284 //
mātaḥ kasya kṛte toyaṃ idaṃ yatnena nīyate /
yadbhaktyā gaṇyate nāyaṃ bhavatībhiḥ pariśramaḥ // 64.285 //
iti tena priyagirā sā pṛṣṭā tam abhāṣata /
mādhuryadhuryasaundarye pakṣapātavatī kṣaṇāt // 64.286 //
martyāmodāpanodāya sadā surabhivāribhiḥ /
pituḥ kinnararājasya girā snāti manoharā // 64.287 //
tayeti kathitaṃ śrutvā sudhāsikta iva kṣaṇāt /
sa hemakumbhe cikṣepa tadabhijñāṅgulīyakam // 64.288 //
snāntyās tatas tad abhihāri manoharāyāḥ kumbhāt papāta kucakumbhayuge 'ṅgulīyam /
yasyāṃśumatpratimaratnamayūkhalekhāḥ kṣipraṃ nakhakṣatavilāsatulām avāpuḥ // 64.289 //
mūrtaṃ tataḥ svaṃ anurāgam ivākalayya ratnāṅgulīyakam anaṅgakathāntaraṅgam /
sā kāntam āgatam avetya kutas tvayedaṃ saṃprāptam ity avadad ucchaliteva dāsīm // 64.290 //
tatas tām avadad dāsī devi puṣkariṇītaṭe /
sthitaḥ ko 'pi yuvā kāntaḥ pratyakṣa iva manmathaḥ // 64.291 //
nikṣiptaṃ hemakumbhe 'sminn idaṃ tenāṅgulīyakam /
bhajate yatprabhāgarbham payaḥ kuṅkumakāntatām // 64.292 //
iti tadvacanaṃ tanvī priyaṃ śrutvā manoharā /
niścitya dayitaṃ prāptam ānināya tayaiva tam // 64.293 //
tayā guptatare nyastaṃ kāntam udyānamandire /
kumudvatīva śaśinaṃ gatvāpaśyan manoharā // 64.294 //
parasparālokanavibhrameṇa viyogasaṃtāpanivedanena /
tayoḥ praharṣānubhavena lebhe śobhām aśeṣāṅgavatīm anaṅgaḥ // 64.295 //
yad yat kiṃ cid virahasamaye cintayābhyastam antar yad yat prauḍhapramadasuhṛdā manmathenopadiṣṭam /
yad yat premṇaḥ sadṛśam ucitaṃ yad yad autsukyarāśes tat tat sarvaṃ praṇayasubhagaṃ dampatī cakratus tau // 64.296 //
tataḥ pracchannavṛttāntaṃ lajjamānā manoharā /
nivedyādarśayat pitroḥ patiṃ bhūmimanobhavam // 64.297 //
tataḥ kinnararājas tāṃ kopaprasphuritādharaḥ /
uvāca darśanapathaṃ parihṛtyātmajāpateḥ // 64.298 //
aho pramādapatitā daivād anucite jane /
na vimuñcasi durvṛtte kṣālyamānāpi raktatām // 64.299 //
yauvanāvanilāvaṇyaṃ spṛhaṇīyaṃ divaukasām /
bata prayātam etat te martyasnehena śocyatāṃ // 64.300 //
udagragotraprabhavā ghanayauvananirbharā /
nimnage kṣobhavibhraṣṭā yātāsi sutarām adhaḥ // 64.301 //
vidvadudvegajananī kulavailakṣyakāriṇī /
malinā khalavidyeva saṃmatāsi na kasya cit // 64.302 //
rūpamātreṇa tuṣṭā tvaṃ yadi martyavaśaṃ gatā /
tat kiṃ na ramase hemanirmāṇapuruṣatviṣā // 64.303 //
prabhāvaguṇahīnasya puṃsaś cārutarākṛteḥ /
ālekhyapuruṣasyeva saundaryaṃ bhittirañjanam // 64.304 //
vadhyo me tvatpatiḥ pāpe hīnasaṃbandhalajjayā /
mukhaṃ draṣṭuṃ na śaknomi tvadyācñāptadivaukasām // 64.305 //
satyaṃ utsāhayuktasya kulasyonnatiśālinaḥ /
jareva kila kāyasya kanyā saṃkocakāriṇī // 64.306 //
iti sā bhartsitā pitrā tam uvāca natānanā /
asūtrahāraṃ kurvāṇā kucayor bāṣpabindhubhiḥ // 64.307 //
kopād anucitaṃ tāta naitan māṃ vaktum arhasi /
na śrūyante prabhāveṇa kiṃ narāḥ kinnarādhikāḥ // 64.308 //
garuḍasyāpi durlaṅghyām imām ullaṅghya yaḥ kṣitim /
prabhāvabhūmir āyāti manuṣyaḥ sa kathaṃ bhavet // 64.309 //
guṇasaṃvādinī mūrtir bhavaty eva śaririṇām /
karoty ānanda saṃvādaṃ dyutir eva himatviṣaḥ // 64.310 //
jātyā kiṃ kriyate tāta svabhāvānuguṇā guṇāḥ /
pūrṇendur amṛtasyandī kālakūṭasya sodaraḥ // 64.311 //
bhavanty antarguṇāḥ ke cic channadoṣās tathāpare /
aparīkṣya na kartavyā maṇīnāṃ mūlyalaṅghanā // 64.312 //
śrutvaitat kinnarapatis tat tatheti vicintya ca /
jāmātaraṃ samāhūya guṇajijñāsayābravīt // 64.313 //
tvayā kāntyā jitās tāvad ete kinnaradārakāḥ /
saṃdarśitaprabhāvas tu divyasaṃbandham arhasi // 64.314 //
atyāyataṃ śaravanaṃ kṛtvoddhṛtaśaraṃ kṣaṇāt /
vyuptam anyūnam uccitya punar dehi tilādhakam // 64.315 //
saṃdarśaya dhanurvede dṛḍhalakṣyādikauśalam /
tataḥ kīrtipatākeyaṃ tavāyattā manoharā // 64.316 //
ity aśakye 'pi kauṭilyāt preritas tena karmaṇi /
sarvaṃ kāntānurāgeṇa kumāraḥ kartum udyayau // 64.317 //
mithyāśramakleśaphale pravṛttaṃ śarapāṭane /
taṃ vijñāya sahasrākṣaḥ pakṣapātād acintayat // 64.318 //
kiṃ bhādrakalpiko bodhisattvo 'yaṃ pārthivātmajaḥ /
niyuktaḥ kinnarendreṇa niṣphale kleśakarmaṇi // 64.319 //
asyāsmin samayāyāse kāryaṃ sāhāyyakaṃ mayā /
iti saṃcintya śakro 'sya karmaniṣpattim ādadhe // 64.320 //
śatakratusamādiṣṭair yakṣaiḥ sūkararūpibhiḥ /
utpāṭite śaravane same vyuptaṃ tilādhakam // 64.321 //
ekīkṛtaṃ samuccitya śakrasṛṣṭaiḥ pipīlakaiḥ /
kumāraḥ kinnarendrāya vismitāya nyavedayat // 64.322 //
sa saptakanakastambhān saptatālān balorjitaḥ /
sūkarīcakrasaṃyuktān viddhvā niśitapattriṇā // 64.323 //
śastrāstravikramakalāśilpaśaktim adarśayat /
yayāsya petur mukuṭe divyāḥ kusumavṛṣṭayaḥ // 64.324 //
vismito 'pi prabhāveṇa tasya kinnarabhūpatiḥ /
punas tadvañcanāyaiva tāṃ tāṃ yuktim acintayat // 64.325 //
dadhati vipulāścarye maunaṃ hasanti satāṃ stutau malinavadanāḥ kīrtyutkarṣe vrajanti vivāditām /
api guṇaśatair nārādhyante viruddhadhiyaḥ paraṃ paraparibhavakṣobhārambhasthirābhiniveśinaḥ // 64.326 //
sa tam ūce prabhāvo 'yaṃ divyaḥ prakaṭitas tvayā /
prajñāprakarṣam adhunā saṃdarśayitum arhasi // 64.327 //
abhinnavarṇarūpāṇāṃ tulyābharaṇavāsasām /
kinnarīṇāṃ sthitāṃ madhye gṛhāṇa nijavallabhām // 64.328 //
ity uktas tena sa punaḥ kinnarīśatapañcakam /
tulyavarṇavayoveśaṃ dadarśāvyagram agrataḥ // 64.329 //
tāsāṃ madhye parijñāya sa jagrāha manoharām /
vallarīvanasaṃcchannāṃ bhrṅgaś cūtalatām iva // 64.330 //
devo 'yam iti niścitya tatas taṃ kinnareśvaraḥ /
tuṣṭas tasmai dadau divyaratnaiḥ saha manoharām // 64.331 //
ślāghyopabhogavibhavaiḥ pūjitas tena sādaram /
jāyāsakhas tam āmantrya kumāraḥ svapuraṃ yayau // 64.332 //
manoharāgragaṃ putraṃ prāptam ālokya bhūpatiḥ /
rākendudarśanodbhūtaḥ sudhāmbudhir ivābabhau // 64.333 //
putraṃ tataś caritacandrasitātapatre rājye 'bhiṣicya paritāpaharaṃ prajānāṃ /
saṃtoṣaśītalavivekasukhābhirāmāṃ chāyām asevata nṛpaḥ praśamadrumasya // 64.334 //
prāptābhiṣekaṃ sudhanam prabhāte taṃ saptaratnāni pṛthupratāpam /
navaprabhāvaprabhubhāvabhītyā sevānivāsārtham ivopajagmuḥ // 64.335 //
sa bodhisattvaḥ sudhano 'ham eva yaśodharā sāpi manoharābhūt tadviprayogavyasanānalārtiḥ kāmānubandhād iyatī mayāptā // 64.336 //
tasmāt kāmaḥ kamalavadanānetraparyantavāsī varjyaḥ sadbhiḥ śamaṃrgavadhūbandhakeliḥ kirātaḥ / sphūrjatpuṣpaprasarajarajaḥpuñjahālāhalāgrair lokaḥ śokavyasanaviśikhair mohanair yena viddhaḥ // 64.337 //
iti bhikṣugaṇaḥ svayaṃ jinena svakathāṃ tāṃ viniveditāṃ niśamya /
sarasaṃ pṛthumūlam eva mene śataśākhasya manobhavam bhavasya // 64.338 //
