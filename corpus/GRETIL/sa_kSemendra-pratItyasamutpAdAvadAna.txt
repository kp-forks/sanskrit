#Author:Kṣemendra
#Text:Pratītyasamutpādāvadāna
#Data entry:Klaus Wille
#Edition:M. Mejor. Kṣemendra's Bodhisattvāvadānakalpalatā. Studies and Materials. Tokyo 1992 (Studia Philologica Buddhica, Monograph Series, 8), pp. 16-22.



sarvam avidyāmūlaṃ saṃsārataruprakāravaicitryam /
jñātum vaktuṃ hantuṃ kaḥ śakto 'nyatra sarvajñāt // 75.1 //
śrāvastyāṃ svastimān pūrvaṃ jino jetavane sthitaḥ /
aśeṣadarśī bhagavān bhikṣusaṃgham abhāṣata // 75.2 //
śṛṇuta śreyase jñānālokanirmalamānasāḥ /
pratītyasamutpādaṃ vaḥ kathayāmi yathākramam // 75.3 //
avidyāvāsanaiveyaṃ duḥkhaskandhasya bhūyasaḥ /
saṃsāraviṣavṛkṣasya mūlabandhavidhāyinī // 75.4 //
tatpratyayas tu saṃskārāḥ kāyavāṅmānasātmakāḥ /
saṃskārotthaṃ ca vijñānaṃ manaḥṣaṣṭhendriyātmakamn // 75.5 //
tatpratyayaṃ nāmarūpaṃ saṃjñāsaṃdarśanābhidham /
manaḥṣaṣṭhendriyasthānaṃ ṣaḍāyatanam apy ataḥ // 75.6 //
ṣaḍāyatanasaṃśleṣaḥ sparśa ity abhidhīyate /
ṣaṭsparśānubhavo yaś ca vedanā sā prakīrtitā // 75.7 //
tayā viṣayasaṃkleśarāgāt tṛṣṇā prajāyate /
kāmādiṣu tadudbhūtam upādānaṃ pravartate // 75.8 //
upādānodbhavaḥ kāmarūpārūpyamayo bhavaḥ /
nānāyoniparāvṛttyā jātir bhavasamudbhavā // 75.9 //
jarāmaraṇaśokādisaṃtatir jātisaṃśrayā /
avidyādinirodhena teṣāṃ vyuparamakramaḥ // 75.10 //
pratītyotpādo 'yaṃ bahugatir avidyākṛtapadaḥ sa cintyo yuṣmābhir vijanavanaviśrāmaśamibhiḥ /
parijñātaḥ samyag vrajati kila kālena tanutāṃ tanutvaṃ saṃprāptaḥ sukhataranivāryaś ca bhavati // 75.11 //
||iti kṣemendraviracitāyāṃ bodhisattvāvadānakalpalatāyāṃ pratītyasamutpādāvadānaṃ pañcasaptatitamaḥ pallavaḥ
