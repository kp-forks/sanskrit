#Text:Candrāloka
#Data entry:Réginald Beyaert

||pīyuṣavarṣaśrījayadevakavikṛtaḥ candrālokaḥ
uccairasyati mandatāmarasatāṃ jāgratkalaṅkairavadhvaṃsaṃ hastayate ca yā sumanasām ullāsinī mānase /
dhṛṣṭodyanmadanāśanārcir amalā lokatrayīdarśikā sā netratritayīva khaṇḍaparaśor vāgdevatā dīvyatu // 1.1 //
haṃ ho cinmayacittacandramaṇayaḥ saṃvardhayadhvaṃ rasān re re svairiṇi nirvicārakavite māsmat prakāśībhava /
ullāsāya vicāravīcinicayālaṅkāravārām nidheś candrālokamayaṃ svayaṃ vitanute pīyūṣavarṣaḥ kṛtī // 1.2 //
yuktyāsvādyalasadrasaikavasatiḥ sāhityasārasvatakṣīrāmbhodhir agādhatām upadadhat sevyaḥ samāśrīyatām /
śrīr asmād upadeśakauśalamayaṃ pīyūṣam asmājjagaj jāgradbhāsurapadmakeśarayaśaḥśītāṃśur asmād budhāḥ // 1.3 //
taṃ pūrvācāryasūryoktijyotiḥ stomodgamaṃ stumaḥ /
yaṃ prastūya prakāśante madguṇās trasareṇavaḥ // 1.4 //
nāśaṅkanīyam eteṣāṃ matam etena dūṣyate /
kiṃ tu cakṣur mṛgākṣīṇāṃ kajjaleneva bhūṣyate // 1.5 //
pratibhaiva śrutābhyāsasahitā kavitāṃ prati /
hetur mṛdambusambaddhā bījamālā latāmiva // 1.6 //
nirdoṣā lakṣaṇavatī sarītir guṇabhūṣaṇā /
sālaṃkārarasānekavṛttir vāk kāvyanāmabhāk // 1.7 //
aṅgīkaroti yaḥ kāvyaṃ śabdārthāv analaṃkṛtī /
asau na manyate kasmād anuṣṇam analaṃ kṛtī // 1.8 //
vibhaktyutpattaye yogyaḥ śāstrīyaḥ śabda iṣyate /
rūḍhayaugikatanmiśraiḥ prabhedaiḥ sa punas tridhā // 1.9 //
avyaktayoganiryogayogābhāsais tridhādimaḥ /
te ca vṛkṣādibhūvādimaṇḍapādyā yathākramam // 1.10 //
śuddhatanmūlasaṃbhinnaprabhedair yaugikas tridhā /
te ca bhrāntisphuratkāntikaunteyādisvarūpiṇaḥ // 1.11 //
tanmiśro 'nyonyasāmānyaviśeṣaparivartanāt /
nīradhiḥ paṅkajaṃ saudhaḥ sāgaro bhūruhaḥ śaśī // 1.12 //
kṣīranīradhir ākāśapaṅkajaṃ tena siddhayati /
vibhaktyantaṃ padaṃ vākyaṃ tadvyūho 'rthasamāptitaḥ // 1.13 //
yuktārthatāṃ tāṃ ca vinā khaṇḍavākyaṃ sa iṣyate /
vākyaṃ ca khaṇḍavākyaṃ ca padam ekam api kvacit // 1.14 //
dhūmavattvād iti yathā devety āmantraṇaṃ yathā /
vākyāny ekārthaviśrāntāny āhur vākyakadambakam // 1.15 //
mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau /
anenāsāv ādyaḥ sukavijayadevena racite ciraṃ candrāloke sukhayatu mayūkhaḥ sumanasaḥ // 1.16 //
syāc ceto viśatā yena sakṣatā ramaṇīyatā /
śabde 'rthe ca kṛtonmeṣaṃ doṣam udghoṣayanti tam // 2.1 //
bhavec chrutikaṭur varṇaḥ śravaṇodvejane paṭuḥ /
saṃvidrate vyākaraṇaviruddhaṃ cyutasaṃskṛti // 2.2 //
aprayuktaṃ daivatādiśabde puṃlliṅgatādikam /
asamarthaṃ tu hantyādeḥ prayogo gamanādiṣu // 2.3 //
sa hanti hanta kāntāre kāntaḥ kuṭilakuntalaḥ /
nihatārthaṃ lohitādau śoṇitādiprayogataḥ // 2.4 //
vyanakty anucitārthaṃ yat padam āhus tadeva tat /
iyam adbhutaśākhyagrakelikautukavānarī // 2.5 //
nirarthakaṃ tuhītyādi pūraṇaikaprayojanam /
arthe vidadhadityādau dadhadādyam avācakam // 2.6 //
dhatte nabhastalaṃ bhāsvān aruṇaṃ taruṇaiḥ karaiḥ /
ekākṣaraṃ vinā bhūbhrūkṣmādikaṃ khatalādivat // 2.7 //
aślīlaṃ trividhaṃ vrīḍājugupsāmaṅgalātmanā /
āhlādasādhanaṃ vāyuḥ kāntānāśe bhavet katham // 2.8 //
syād dvyartham iha sandigdhaṃ nadyāṃ yānti patatriṇaḥ /
syād apratītaṃ śāstraikagamyaṃ vītānumādivat // 2.9 //
śithilaṃ śayane lilye maccitaṃ te śaśiśriyi /
mastapiṣṭakaṭīloṣṭagallādi grāmyam ucyate // 2.10 //
neyārthaṃ lakṣaṇātyantaprasarād amanoharam /
himāṃśor hāradhikkārajāgare yāmikāḥ karāḥ // 2.11 //
kliṣṭam artho yadīyo 'rthaśreṇiniḥśreṇim ṛcchati /
haripriyāpitṛvadhūpravāhapratimaṃ vacaḥ // 2.12 //
avimṛṣṭavidheyāṃśaḥ samāsapihite vidhau /
viśanti viśikhaprāyāḥ kaṭākṣāḥ kāmināṃ hṛdi // 2.13 //
aparādhīna ityādi viruddhamatikṛn matam /
anyasaṅgatam uttuṅgahāraśobhipayodharau // 2.14 //
rasādyanucite varṇe pratikūlākṣaraṃ viduḥ /
na mām aṅgada jānāsi rāvaṇaṃ raṇadāruṇam // 2.15 //
yasminn upahato lupto visarga iha tat tathā /
kusandhiḥ paṭav āgaccha visandhir nṛpatī imau // 2.16 //
hatavṛttam anukto 'pi chandodoṣaś cakāsti cet /
viśālalocane! paśyā-mbaraṃ tārātaraṅgitam // 2.17 //
nyūnaṃ tvatkhaḍgasaṃbhūtayaśaḥpuṣpaṃ nabhastaṭam /
adhikaṃ bhavataḥ śatrūn daśaty asilatāphaṇī // 2.18 //
kathitaṃ punaruktā vāk śyāmābjaśyāmalocanā /
vikṛtaṃ dūravivṛtair aiyaruḥ kuñjarāḥ puram // 2.19 //
patatprakarṣaṃ hīnānuprāsāditve yathottaram /
gambhīrārambhadambholipāṇir eṣaḥ samāgataḥ // 2.20 //
samāptapunarāttaṃ syād eṣa pīyūṣabhājanam /
netrānandī tuṣārāṃśur udety ambudhibāndhavaḥ // 2.21 //
ardhāntarapadāpekṣi krīḍānṛtyeṣu sasmitam /
moghārambhaṃ stumaḥ śambhum ardharambhoruvigraham // 2.22 //
abhavanmatayogaḥ syān na ced abhimato 'nvayaḥ /
yena baddho 'mbudhir yasya rāmasyānucarā vayam // 2.23 //
dviṣāṃ sampadam ācchidya yaḥ śatrūn samapūrayat /
asthānasthasamāsaṃ na vidvajjanamanoramam // 2.24 //
mithaḥ pṛthagvākyapadaiḥ saṃkīrṇaṃ yat tad eva tat /
vaktreṇa bhrājate rātriḥ kāntā candreṇa rājate // 2.25 //
brahmāṇḍaṃ tvadyaśaḥpūragarbhitaṃ bhūmibhūṣaṇa /
ākarṇaya payaḥpūrṇasuvarṇakalaśāyate // 2.26 //
bhagnaprakramam ārabdhaśabdanirvāhahīnatā /
akramaḥ kṛṣṇa pūjyante tvām anabhyarcya devatāḥ // 2.27 //
amatārthāntaraṃ mukhye 'mukhyenārthe virodhakṛt /
tyaktahāram uraḥkṛtvā śokenāliṅgitāṅganā // 2.28 //
apuṣṭārtho viśeṣye cen na viśeṣo viśeṣaṇāt /
viśanti hṛdayaṃ kāntākaṭākṣāḥ khañjanatviṣaḥ // 2.29 //
kaṣṭaḥ spaṣṭāvabodhārtham akṣamo vācyasannibhaḥ /
vyāhataś cedvirodhaḥ syān mithaḥ pūrvāparārthayoḥ // 2.30 //
sahasrapatramitraṃ te vaktraṃ kenopamīyate /
kutastatropamā yatra punaruktaḥ sudhākaraḥ // 2.31 //
duṣkramagrāmyasandigdhās trayo doṣāḥ kramād amī /
tvadbhaktaḥ kṛṣṇa! gaccheyaṃ narakaṃ svargam eva vā // 2.32 //
ekaṃ me cumbanaṃ dehi tava dāsyāmi kañcukam /
brūta kiṃ sevyatāṃ candramukhīcandrakirīṭayoḥ // 2.33 //
anaucityaṃ kīrtilatāṃ taraṅgayati yaḥ sadā /
prasiddhyā vidyayā vāpi viruddhaṃ dvividhaṃ matam // 2.34 //
nyasteyaṃ paśya kandarpapratāpadhavaladyutiḥ /
ketakī śekhare śambhor dhatte candrakalātulām // 2.35 //
sāmānyaparivṛttiḥ syāt kuṇḍalacchavivigrahā /
viśeṣaparivṛttiḥ syād vanitā mama cetasi // 2.36 //
tathā sahacarācāruviruddhānyonyasaṃgatī /
dhvāṅkṣāḥ santaś ca tanayaṃ svaṃ paraṃ ca na jānate // 2.37 //
sarojanetra putrasya mukhendum avalokaya /
pālayiṣyati te gotram asau narapurandaraḥ // 2.38 //
pade tadaṃśe vākyāṃśe vākye vākyakadambake /
yathānusāram abhyūhed doṣān śabdārthasaṃbhavān // 2.39 //
doṣam āpatitaṃ svānte prasarantaṃ viśṛṅkhalam /
nivārayati yas tredhā doṣāṅkuśam uśanti tam // 2.40 //
doṣe guṇatvaṃ tanute doṣatvaṃ vā nirasyati /
bhavantam athavā doṣaṃ nayaty atyājyatāmasau // 2.41 //
mukhaṃ candraśriyaṃ dhatte śvetaśmaśrukarāṅkuraiḥ /
atra hāsyarasoddeśe grāmyatvaṃ guṇatāṃ gatam // 2.42 //
tava dugdhābdhisaṃbhūteḥ kathaṃ jātā kalāṅkitā /
kavīnāṃ samayād vidyāviruddho 'doṣatāṃ gataḥ // 2.43 //
dadhāra gaurī hṛdaye devaṃ himakarāṅkitam /
atra śleṣodayān naiva tyājyaṃ hīti nirarthakam // 2.44 //
mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau /
dvitīyas tenāsau sukavijayadevena racite ciraṃ candrāloke sukhayatu mayūkhaḥ sumanasaḥ // 2.45 //
alpākṣarā vicitrārthakhyātir akṣarasaṃhatiḥ /
uṣākāntenānugataḥ śūraḥ śaurir ayaṃ punaḥ // 3.1 //
śobhā khyātāpi yaddoṣo guṇakīrtyā niṣidhyate /
mudhā nindanti saṃsāraṃ kaṃsārir yatra pūjyate // 3.2 //
abhimāno vicāraś ced ūhitārthaniṣedhakṛt /
indur yadi kathaṃ tīvraḥ sūryo yadi kathaṃ niśi // 3.3 //
hetus tyaktvā bahūn pakṣān yuktyaikasyāvadhāraṇam /
nendur nārko 'yam aurvāgniḥ sāgarād utthito dahan // 3.4 //
pratiṣedhaḥ prasiddhānāṃ kāraṇānām anādaraḥ /
na yuddhena bhruvoḥ spande-naiva vīrā nipātitāḥ // 3.5 //
niruktaṃ syān nirvacanaṃ nāmnaḥ satyaṃ tathānṛtam /
īdṛśaiś caritai rājan satyaṃ doṣākaro bhavān // 3.6 //
syān mithyādhyavasāyaś ced asatī sādhyasādhane /
candrāṃśusūtragrathitāṃ nabhaḥpuṣpasrajaṃ vaha // 3.7 //
siddhiḥ khyāteṣu cen nāma kīrtyate tulyatoktaye /
yuvām eveha vikhyātau tvaṃ balair jaladhir jalaiḥ // 3.8 //
yuktir viśeṣasiddhiś ced vicitrārthāntarānvayāt /
navas tvaṃ nīradaḥ ko 'pi svarṇair varṣasi yan muhuḥ // 3.9 //
kāryaṃ phalopalambhaś ced vyāpārād vastuto 'tha vā /
asāv udeti śītāṃśur mānacchedāya subhruvām // 3.10 //
ityādi lakṣaṇaṃ bhūri kāvyasyāhur maharṣayaḥ /
svarṇabhrājiṣṇubhālatvaprabhṛtīva mahībhujaḥ // 3.11 //
mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau /
tṛtīyas tenāsau sukavijayadevena racite ciraṃ candrāloke sukhayatu mayūkhaḥ sumanasaḥ // 3.12 //
śleṣo vighaṭamānārthaghaṭamānatvavarṇanam /
sa tu śābdaḥ sajātīyaiḥ śabdair bandhaḥ sukhāvahaḥ // 4.1 //
ullasattanutāṃ nīte 'nante pulakakaṇṭakaiḥ /
bhītayā mānavatyaiva śriyāśliṣṭaṃ hariṃ stumaḥ // 4.2 //
yasmād antaḥsthitaḥ sarvaḥ svayam artho 'vabhāsate /
salilasyeva sūktasya sa prasāda iti smṛtaḥ // 4.3 //
samatālpasamāsatvaṃ varṇādyais tulyatātha vā /
śyāmalā komalā bālā ramaṇaṃ śaraṇaṃ gatā // 4.4 //
samādhir arthamahimā lasadghanarasātmanā /
syād antarviśatā yena gātramaṅkuritaṃ satām // 4.5 //
mādhuryaṃ punaruktasya vaicitryaṃ cārutāvaham /
vayasya paśya paśyāsyāś cañcalaṃ locanāñcalam // 4.6 //
ojaḥ syāt prauḍhir arthasya saṃkṣepo vātibhūyasaḥ /
ripuṃ hatvā yaśaḥ kṛtvā tvadasiḥ kośam āviśat // 4.7 //
saukumāryam apāruṣyaṃ paryāyaparivartanāt /
sa kathāśeṣatāṃ yātaḥ samāliṅgya marutsakham // 4.8 //
udāratā tu vaidagdhyam agrāmyatvāt pṛthaṅmatā /
mānaṃ muñca priye kiṃcil locanāntam udañcaya // 4.9 //
śṛṅgāre ca prasāde ca kāntyarthavyaktisaṃgrahaḥ /
amī daśa guṇāḥ kāvye puṃsi śauryādayo yathā // 4.10 //
tilakādyam iva strīṇāṃ vidagdhahṛdayaṅgamam /
vyatiriktam alaṅkāraṃ prakṛter bhūṣaṇaṃ girām // 4.11 //
vicitralakṣaṇo nyāso nirvāhaḥ prauḍhir aucitī /
śāstrāntararahasyoktiḥ saṃgraho dik pradarśitā // 4.12 //
mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau /
caturthas tenāsau sukavijayadevena racite ciraṃ candrāloke sukhayatu mayūkhaḥ sumanasaḥ // 4.13 //
śabdārthayoḥ prasiddhyā vā kaveḥ prauḍhivaśena vā /
hārādivad alaṅkāraḥ sanniveśo manoharaḥ // 5.1 //
svaravyañjanasaṃdohavyūhāmandohadohadā /
gaur jagajjāgradutsekā chekānuprāsabhāsurā // 5.2 //
āvṛttavarṇasampūrṇaṃ vṛttyanuprāsavad vacaḥ /
amandānandasandohasvacchandāspadamandiram // 5.3 //
lāṭānuprāsabhūr bhinnā-bhiprāyā punaruktatā /
yatra syān na punaḥ śatror garjitaṃ taj jitaṃ jitam // 5.4 //
ślokasyārdhe tadardhe vā varṇāvṛttir yadi dhruvā /
tadā matā matimatāṃ sphuṭānuprāsatā satām // 5.5 //
upameyopamānādāv arthānuprāsa iṣyate /
candanaṃ khalu govindacaraṇadvandvavandanam // 5.6 //
punaruktapratīkāśaṃ punaruktārthasannibham /
aṃśukāntaṃ śaśī kurvann ambarāntam upaity asau // 5.7 //
āvṛttavarṇastabakaṃ stavakandāṅkuraṃ kaveḥ /
yamakaṃ prathamā dhuryamādhuryavacaso viduḥ // 5.8 //
kāvyavitpravaraiścitraṃ khaḍgabandhādi lakṣyate /
teṣv ādyam ucyate ślokadvayīsajjanarañjikā // 5.9 //
kāminīva bhavet khaḍgalekhā cārukarālikā /
kāśmīrasekā raktāṅgī śatrukaṇṭhāntikāśritā // 5.10 //
upamā yatra sādṛśyalakṣmīr ullasati dvayoḥ /
hṛdaye khelator uccais tanvaṅgīstanayoriva // 5.11 //
upamānopameyatve yatraikasyaiva jāgṛtaḥ /
indur indurivetyādau bhaved evam ananvayaḥ // 5.12 //
paryāyeṇa dvayos tac ced upameyopamā matā /
dharmo 'rtha iva pūrṇaśrīr artho dharma iva tvayi // 5.13 //
vikhyātasyopamānasya yatra syād upameyatā /
indur mukham ivetyādau syāt pratīpopamā tadā // 5.14 //
upamāne tu līlādipadāḍhye lalitopamā /
tvannetrayugalaṃ dhatte līlāṃ nīlāmbujanmanoḥ // 5.15 //
anekasyārthayugmasya sādṛśyaṃ stabakopamā /
śrito 'smi caraṇau viṣṇor bhṛṅgas tāmarasaṃ yathā // 5.16 //
syāt saṃpūrṇopamā yatra dvayor api vidheyatā /
padmānīva vinidrāṇi netrāṇy āsann aharmukhe // 5.17 //
yatropamānacitreṇa sarvathāpy uparajyate /
upameyamayī bhittis tatra rūpakam iṣyate // 5.18 //
samānadharmayuksādhyāropāt sopādhirūpakam /
utsiktakṣitibhṛllakṣyapakṣacchedapurandaraḥ // 5.19 //
pṛthakkathitasādṛśyaṃ dṛśyaṃ sādṛśyarūpakam /
ullasatpañcaśākhas te rājate bhujabhūruhaḥ // 5.20 //
syād aṅgayaṣṭirityevaṃvidham ābhāsarūpakam /
aṅgayaṣṭidhanurvallītyādi rūpitarūpakam // 5.21 //
pariṇāmo 'nayor yasminn abhedaḥ paryavasyati /
kāntena pṛṣṭā rahasi maunam evottaraṃ dadau // 5.22 //
bahubhirbahudhollekhād ekasyollekhitā matā /
strībhiḥ kāmaḥ priyaiś candraḥ kālaḥ śatrubhir aikṣi saḥ // 5.23 //
atathyam āropayituṃ tathyāpāstir apahnutiḥ /
nāyaṃ sudhāṃśuḥ kiṃ tarhi vyomagaṅgāsaroruham // 5.24 //
paryastāpahnutir yatra dharmamātraṃ niṣidhyate /
nāyaṃ sudhāṃśuḥ kiṃ tarhi sudhāṃśuḥ preyasīmukham // 5.25 //
bhrāntāpahnutir anyasya śaṅkayā tathyanirṇaye /
tāpaṃ tanoti sotkampaṃ jvaraḥ kiṃ na sakhi smaraḥ // 5.26 //
chekāpahnutir anyasya śaṅkayā tathyanihnave /
prajalpan matpade lagnaḥ kāntaḥ kiṃ na hi nūpuraḥ // 5.27 //
kaitavāpahnutir vyakte vyājādyair nihnave padaiḥ /
niryānti smaranārācāḥ kāntādṛkpātakaitavāt // 5.28 //
utprekṣonnīyate yatra hetvādir nihnutiṃ vinā /
tvanmukhaśrīkṛte nūnaṃ padmair vairāyate śaśī // 5.29 //
ivādikapadābhāve gūḍhotprekṣāṃ pracakṣate /
yatkīrtir vibhramaśrāntā viveśa svarganimnagām // 5.30 //
syāt smṛtibhrāntisaṃdehais tad evālaṃkṛtitrayam /
paṅkajaṃ paśyatas tasyā mukhaṃ me gāhate manaḥ // 5.31 //
ayaṃ pramattamadhupas tvanmukhaṃ veda paṅkajam /
paṅkajaṃ vā sudhāṃśur vety asmākaṃ tu na nirṇayaḥ // 5.32 //
mīlitaṃ bahusādṛśyād bhedavac cen na lakṣyate /
raso nālakṣi lākṣāyāś caraṇe sahajāruṇe // 5.33 //
sāmānyaṃ yadi sādṛśyād bheda eva na lakṣyate /
padmākarapraviṣṭānāṃ mukhaṃ nālakṣi subhruvām // 5.34 //
hetoḥ kuto 'pi vaiśiṣṭyāt sphūrtir unmīlitaṃ matam /
lakṣitāny udite candre padmāni ca mukhāni ca // 5.35 //
anumānaṃ ca kāryādeḥ kāraṇādyavadhāraṇam /
asti kiñcid yadanayā māṃ vilokya smitaṃ manāk // 5.36 //
arthāpattiḥ svayaṃ sidhyet padārthāntaravarṇanam /
sa jitas tvanmukhenenduḥ kā vārtā sarasīruhām // 5.37 //
syāt kāvyaliṅgaṃ vāgartho nūtanārthasamarthakaḥ /
jito 'si manda kandarpa maccitte 'sti trilocanaḥ // 5.38 //

eṣu saptadaśasv ekaṃ parityajyāsphuṭakramam // 7.10 //
ye ṣoḍaśādyā dviguṇās te syur vākyapadāśrayāt /
prabandhe 'pi dvādaśa syur arthaśaktibhuvo bhidaḥ // 7.11 //
dvātriṃśad dvādaśaikaḥ ṣaṭ sarvasaṃkalitadhvaneḥ /
bhedāḥ syur ekapañcāśat saṃbhinnās tu sahasraśaḥ // 7.12 //
vaktṛsyūtaṃ bodhayituṃ vyaṅgyaṃ vaktur abhīpsitam /
svāṅkuritam atadrūpaṃ svayam ullasitaṃ giraḥ // 7.13 //
kaścit sādhāraṇaḥ kaścid āmantrya pratibodhitaḥ /
kaścit taṭasthaḥ kaścic ca bodhitapratibodhitaḥ // 7.14 //
ityevaṃ boddhṛvaicitryād vaktṛsyūtaṃ caturvidham /
upekṣānihnavābhyāṃ ca dvidhā svāṅkuritaṃ matam // 7.15 //
bhūtādikālabhedena nihnavaḥ syād anekadhā /
anenāpi prabhedena vyaktivallī vijṛmbhate // 7.16 //
nānāprabhedā niyatā kvacit prakaraṇādinā /
arthe 'rtham anyaṃ yaṃ vakti tad vācyavyaṅgyam iṣyate // 7.17 //
mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau /
mayūkhastenāsau sukavijayadevena racite ciraṃ candrāloke sukhayatu mahati munisaṃkhyaḥ // 7.18 //
yad vyajyamānaṃ manasaḥ staimityāya sa no dhvaniḥ /
anyathā tu guṇībhūtavyaṅgyam āpatitaṃ tridhā // 8.1 //
vyaktir eva kvacid vyaṅgyaḥ kvacid arthasvabhāvataḥ /
kvacic cārutarasyāgre sa vimuñcati cārutām // 8.2 //
agūḍhaṃ kalayed arthā-ntarasaṃkramitādikam /
vismṛtaḥ kim apāṃnātha sa tvayā kumbhasaṃbhavaḥ // 8.3 //
aparasya rasādeś ced aṅgamanyad rasādikam /
hā hā! matkucakāśmīraliptaṃ bhinnam uraḥ śaraiḥ // 8.4 //
tathā vācyasya siddhyaṅgaṃ naur artho vāridher yathā /
saṃśritya taraṇiṃ dhīrās taranti vyādhivāridhīn // 8.5 //
asphuṭaṃ stanayor atra kokasādṛśyavan matam /
kuṅkumāktaṃ stanadvandvaṃ mānasaṃ mama gāhate // 8.6 //
saṃdigdhaṃ yadi saṃdeho dairghyādyutpalayoriva /
saṃprāpte nayane tasyāḥ śravaṇottaṃsabhūmikām // 8.7 //
tulyaprādhānyam indutvam iva vācyena sāmyabhṛt /
kānte tvadānanarucā glānim eti saroruham // 8.8 //
asundaraṃ yadi vyaṅgyaṃ syād vācyād amanoharam /
sarasyāmīladambhoje cakraḥ kāntāṃ vilokate // 8.9 //
kākusthaṃ praṇato 'mbhodhir adya mādyatu rāvaṇaḥ /
ity aṣṭadhā guṇībhūtavyaṅgyam aṅgīkṛtaṃ budhaiḥ // 8.10 //
mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau /
mayūkhastenāsau sukavijayadevena racite ciraṃ candrāloke mahati vasusaṃkhyaḥ sukhayatu // 8.11 //
mukhyārthasyāvivakṣāyāṃ pūrvārvācī ca rūḍhitaḥ /
prayojanāc ca saṃbaddhaṃ vadantī lakṣaṇā matā // 9.1 //
lakṣaṇīyasvaśabdasya mīlanāmīlanād dvidhā /
lakṣaṇā sā tridhā siddhasādhyasādhyāṅgabhedataḥ // 9.2 //
sphuṭāsphuṭaprabhedena prayojanam api dvidhā /
viduḥ sphuṭaṃ taṭasthatvād arthagatvād dvidhā budhāḥ // 9.3 //
asphuṭaṃ cārthaniṣṭhatvāt taṭasthatvād api dvidhā /
lakṣyalakṣakaniṣṭhatvād arthasaṃstham api dvidhā // 9.4 //
lakṣakasthaṃ sphuṭaṃ yatra sā vicakṣaṇalakṣaṇā /
asphuṭatvaṃ taṭasthatvaṃ lakṣyasthatvam amuṣya ca // 9.5 //
anyās tisra iti vyaktā śaktitaḥ sā caturvidhā /
indur evaiṣa tadvaktram utkarṣo lakṣyate mukhe // 9.6 //
pradīpaṃ vardhayetyatra taṭasthaṃ maṅgalodayaḥ /
paṭo 'yaṃ dagdha ityādau sphuṭaṃ nāsti prayojanam // 9.7 //
amṛtaṃ sūktam ityādau lakṣyastham atihṛdyatā /
ābhimukhyāt saṃnidhānāt tathākārapratītitaḥ // 9.8 //
kāryakāraṇabhāvāt sā vācyavācakabhāvataḥ /
ityevamādeḥ saṃbandhāt kiṃcānyasmāc catuṣṭayāt // 9.9 //
sādṛśyāt samavāyāt sā vaiparītyāt kriyānvayāt /
sāropādhyavasānākhye gauṇaśuddhe pṛthak pṛthak // 9.10 //
gauṇaṃ sāropam uddiṣṭam indur mukham itīdṛśam /
gauṇaṃ sādhyavasānaṃ syād indur evedam īdṛśam // 9.11 //
śuddhaṃ sāropam uddiṣṭam āyur ghṛtam itīdṛśam /
śuddhaṃ sādhyavasānaṃ syād āyur evedam īdṛśam // 9.12 //
upādānārpaṇadvāre dve cānye iti ṣaḍvidhā /
kuntā viśanti gaṅgāyāṃ ghoṣo nivasatīti ca // 9.13 //
lakṣyalakṣakavaiśiṣṭyavaiśiṣṭyād dvividhā punaḥ /
sarasaṃ kāvyam amṛtaṃ vidyā sthirataraṃ dhanam // 9.14 //
tathā sahetur atathā-bhedabhinnā ca kutracit /
saundaryeṇaiṣa kandarpaḥ sā ca mūrtimatī ratiḥ // 9.15 //
śabde padārthe vākyārthe saṃkhyāyāṃ kārake tathā /
liṅge ceyam alaṅkārā-ṅkurabījatayā sthitā // 9.16 //
mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau /
mayūkhastenāsau sukavijayadevena racite ciraṃ candrāloke mahati navasaṃkhyaḥ sukhayatu // 9.17 //
dharmaṃ kaṃcit puraskṛtya prāyaḥ śabdaḥ pravartate /
yayārthaṃ spaṣṭam ācaṣṭe śabdas tām abhidhāṃ viduḥ // 10.1 //
jātyā guṇena kriyayā vastuyogena saṃjñayā /
nirdeśena tathā prāhuḥ ṣaḍvidhām abhidhāṃ budhāḥ // 10.2 //
gaur nīlaḥ pācako daṇḍī ḍitthaḥ kaṃsa iti kramāt /
kaṃ saṃhinasti kaṃsārir naraṃ ca kaṃ samāśritam // 10.3 //
na yogāder āyatanaṃ na saṅketaniketanam /
vṛttyā nirdeśaśabdo 'yaṃ mukhyayā svābhidheyayā // 10.4 //
pīyūṣavarṣaprabhavaṃ candrālokamanoharam /
sudhānidhānam āsādya śrayadhvaṃ vibudhā mudam // 10.5 //
jayanti yājñikaśrīmanmahādevāṅgajanmanaḥ /
sūktipīyūṣavarṣasya jayadevakaver giraḥ // 10.6 //
mahādevaḥ satrapramukhamakhavidyaikacaturaḥ sumitrā tadbhaktipraṇihitamatir yasya pitarau /
mayūkhas tenāsau sukavijayadevena racite ciraṃ candrāloke mahati daśasaṃkhyaḥ sukhayatu // 10.7 //
