||1. dānapāramitāsamāsaḥ

namo buddhāya //
tathāgatānāṃ padam ārurukṣur āśritya ratnatrayam ādareṇa /
bodhau nidhāyāvicalaṃ manaśca kuryāt parātmavyatihāram ādau // 1.1 //
tataḥ paraṃ dānavidhau prayogaḥ kāryastathā lokahitonmukhena /
yathā svagātrāṇyapi yācitasya na yogasaṃkocavirūpatā syāt // 1.2 //
mātsaryadoṣopacayāya yat syān na tyāgacittaṃ paribṛṃhayed vā /
tattyaktumevārhati bodhisattvaḥ parigrahacchadmamayaṃ vighātam // 1.3 //
tad bodhisattvaḥ katham ādadīta ratnaṃ dhanaṃ vā divi vāpi rājyam /
yat tyāgacittapratipakṣadakṣaṃ saṃbodhimārgāvaraṇaṃ karoti // 1.4 //
saṃsmṛtya caryātiśayaṃ munīnāṃ tadunmukhīṃ svāmapi ca pratijñām /
parigrahasnehavinigrahārthaṃ kuryād imāṃścetasi sadvitarkān // 1.5 //
yadā nisṛṣṭo jagate mayāyaṃ kāyo 'pī tattyāgakṛto 'pi dharmaḥ /
bāhye tadā vastuni saṅgacittaṃ na me gajasnānam ivānurūpam // 1.6 //
māṃsārthino māṃsamidaṃ harantu majjānamapyuddharaṇāt tadarthī /
ahaṃ hi lokārthamidaṃ bibharmi śarīrakaṃ kiṃ bata vastu bāhyam // 1.7 //
yathaiva bhaiṣajyamahīruhasya tvakpattrapuṣpādi janā haranti /
madīyamete 'paharanti ceti naivaṃ vikalpāḥ samudācaranti // 1.8 //
tathaiva lokārthasamudyatena svalpo 'pi kāryo na mayā vikalpaḥ /
duḥkhe kṛtaghne satatāśucau ca dehe parasmāyupayujyamāne // 1.9 //
ādhyātmike caiva mahījalādye bāhye mahābhūtagaṇe ca tulye /
idaṃ mamedaṃ na mameti ko 'yam ajñānapaṅkāṅkavidhirmayāpi // 1.10 //
gṛhṇīta gātrāṇyapi me yatheṣṭaṃ mā kārṣurasmin parakīyabuddhim /
yuṣmākameva svamidaṃ kimarthaṃ nātmābhimāno mama kaścidatra // 1.11 //
ityadbhutā yasya bhavantyabhīkṣṇaṃ saṃbuddhabhāvānuguṇā vitarkāḥ /
taṃ bodhisattvātiśayaṃ vadanti buddhā mahāsattvamacintyasattvāḥ // 1.12 //
evaṃ sa dānapratipattiśūraḥ karoti kāye 'pi na jātvapekṣām /
tasyāprayatnādupayānti śuddhiṃ karmāṇi vākkāyamanomayāni // 1.13 //
viśuddhakarmā ca hitaṃ pareṣām āyāsaduḥkhena vinā karoti /
itthaṃ sa sattvārthamabhiprayatno nayānaye kauśalamabhyupaiti // 1.14 //
bhūyastaraṃ prāpya balaṃ sa dānāt saddharmadānena tataḥ karoti /
bhavāndhakāre bhramatāṃ janānāṃ sūryodayāt spaṣṭataraṃ prakāśam // 1.15 //
sādhāraṇī lokahitārthasiddhiḥ sarvajñabhāvābhyudayapratiṣṭhā /
ato 'sya puṇyākṣayatābhyudeti prabheva bhānorudayasthitasya // 1.16 //
ityadbhutā dānamayā guṇaughā ye bodhisattvābharaṇībhavanti /
tasmāt tadīyaṃ parikarma cittaṃ dānasya kāruṇyapuraḥsarasya // 1.17 //
āyuḥpratībhānabalādi bauddhaṃ niṣpādayeyaṃ jagatāmanena /
sattvā mayā cāmiṣasaṃgṛhītāḥ saddharmapātrāṇyapi me bhaveyuḥ // 1.18 //
ityannadānaṃ pradadāti vidvān na svargasaṃpattiparigrahāya /
pānānyapi kleśatṛṣaḥ śamāya lokasya lokārthacaro dadāti // 1.19 //
bauddhasya caivarddhiviceṣṭitasya nirvāṇasaukhyasya ca sarvalokaḥ /
lābhī kathaṃ syāditi lokanātho yānaṃ mahāyānaratirdadāti // 1.20 //
saṃbuddhavarṇasya ca hemabhāso lajjāmayasyaiva ca bhūṣaṇasya /
niṣpattaye vastravidhīnudārān satkṛtya kālānuguṇaṃ dadāti // 1.21 //
saṃbodhimaṇḍāsanam āsanāni śayyāśca śayyātrayam īkṣamāṇaḥ /
sarvajñacakṣuḥpratilabdhaye ca caityeṣu rathyāsu ca dīpamālām // 1.22 //
vādyāni divyaśrutisaṃgrahārthaṃ saṃbuddhaśīlāya ca gandhadānam /
sabhāprapārāmavihāragehāñ śaraṇyabhāvābhimukho dadāti // 1.23 //
dānaṃ rasānāṃ tu susaṃskṛtāṇāṃ rasārasāgratvaparigrahāya /
bhaiṣajyadānānyajarāmaratvaṃ lokān imān prāpayituṃ dadāti // 1.24 //
bhujiṣyatāmātmasamaṃ ninīṣurdāśīkṛtān kleśagaṇena lokān /
sa dāsadāsyādi sadā dadāti dāsānudāsānaparākariṣyan // 1.25 //
dadāti putrān duhitṛḥ priyāśca bodhipriyatvādanavadyadānam /
ekāntasaddharmaratipriyaśca krīḍāviśeṣān ratihetubhūtān // 1.26 //
suvarṇamuktāmaṇividrumādīn dadāti sallakṣaṇasaṃpadartham /
ratnapradīptāni ca bhūṣaṇāni citrāṇyanuvyañjanasauṣṭhavāya // 1.27 //
dhyānārthamudyānatapovanāni saddharmakoṣāya ca vittakoṣam /
munīndrarājyāya dadātyakhinno rājyāni cājñāpanamaṇḍitāni // 1.28 //
cakrāṅkitābhyāṃ caraṇottamābhyām saṃbodhimaṇḍākramaṇotsukatvāt /
sa nirvikāraścaraṇapradānaṃ lokārthaniṣpattikaro dadāti // 1.29 //
duḥkhāpagāyāmatiśīghragāyāṃ magnasya lokasya kathaṃ na dadyām /
saddharmahastāniti saṃpradatte hastānvikoṣāmburuhaprakāśān // 1.30 //
śraddhendriyādipratipūraṇārthaṃ sa karṇanāsādi dadātyakhinnaḥ /
cakṣuśca cakṣurvimalīkariṣyaṃllokasya sarvāvaraṇaprahāṇāt // 1.31 //
utkṛtya māṃsāni saśoṇitāni dadāti kāruṇyavaśena nāthaḥ /
bhūmyagnivāyvambuvadeva me syāllokopajīvyaḥ katham eṣa kāyaḥ // 1.32 //
lokottamajñānasamāpanārthaṃ sa uttamāṅgairapi satkaroti /
abhyāgatasyārthijanasya yācñāṃ prāgeva dehāvayavaistadanyaiḥ // 1.33 //
majjānamapyadbhutavīraceṣṭo dadāti lokasya kathaṃ na kuryām /
tathāgataṃ vigrahamapradhṛṣyaṃ vṛṣṭyāpi vajrojjvalayā patantyā // 1.34 //
ityevamādyaṃ satatānavadyaṃ tadbodhisattvāmbudharapramuktam /
prahlādya dānāmbu jagatsamagraṃ sarvajñatāsāgaramabhyupaiti // 1.35 //
anviṣya bhogānviṣameṇa nāsau dadāti notpīḍanayā parasya /
na trāsalajjāpratikārahetorna dakṣinīyān parimārgamāṇaḥ // 1.36 //
na ca praṇīte sati rūkṣadānam adakṣiṇīyā iti vāvamanya /
vipākakāṅkṣākṛpaṇīkṛtaṃ vā satkārahīnaṃ vijugupsitaṃ vā // 1.37 //
naivonnatiṃ śīlavate prayacchan viparyayaṃ gacchati netarasmai /
nātmānamutkarṣati naiva nindāṃ karoti so 'nyasya samaprayogaḥ // 1.38 //
na cāsya mithyāśayadānamasti naivāstyanadhyāśayadānamasya /
na krodhadoṣopahataṃ dadāti naivānutāpaṃ kurute sa dattvā // 1.39 //
na ślāghyamāno vipulaṃ dadāti nāślāghyamāno 'nyataraṃ dadāti /
na yācakānāmupaghātadānaṃ yad vā bhaved vipratipattihetuḥ // 1.40 //
nākāladānaṃ sa dadāti kiṃcid dadāti kāle viṣame 'pi naiva /
na devabhāvāya na rājyahetorna hīnayānaspṛhayālubhāvāt // 1.41 //
nāsau mukhollokanayā dadāti na kīrtiśabdāya na hāsyahetoḥ /
paryāptametacca mameti naivaṃ yadvā vihiṃsāhasitaṃ pareṣām // 1.42 //
sarvajñabhāvāpariṇāmitaṃ vā sagarhitaṃ vā sa dadāti naiva /
tato 'sya tat pāramitābhidhānaṃ parāṃ viśuddhiṃ samupaiti dānam // 1.43 //
dānodbhavaṃ tasya ca puṇyarāśiṃ lokāt samagrādapi piṇḍitāni /
puṇyāni naivābhibhavanti yasmāllokottamattvaṃ sa tato 'bhyupaiti // 1.44 //
pañcasvabhijñāsu viniścitātmā lokāya yadvarṣati dānavarṣam /
samantatastasya kutaḥ pramāṇaṃ parikṣayo vā satatapravṛtteḥ // 1.45 //
yadakṣayāṇāṃ jagatāṃ hitāya jñānasya hetuśca yadakṣayasya /
traidhātukena kṣayiṇā na tacca saṃlipyate vyomavadambudena // 1.46 //
tacchūnyatākārasamāhitaṃ ca nimittadoṣaiḥ parivarjitaṃ ca /
akiṃcanakleśaviyogasiddhestenākṣayaṃ tatkathitaṃ munīndraiḥ // 1.47 //
asmin punaḥ satpuruṣāvadāne dāne nidāne sukhavistārāṇām /
cikīrṣatā yogamanityasaṃjñā bhogeṣu kāryā karuṇā ca loke // 1.48 //
bhogānanityānabhivīkṣamānaḥ sātmyaṃ gatāyāṃ ca tataḥ kṛpāyām /
sa niścayaṃ gacchati dīyate yad etān madīyāṃ na tu yadgṛhe me // 1.49 //
yaddattam asmān na bhayaṃ kadā cid gehe yadasmādbhayamabhyupaiti /
sādhāraṇaṃ rakṣyamatarpakaṃ ca datte tu naite prabhavantyanarthāḥ // 1.50 //
sukhaṃ paratrāpi karoti dattam ihaiva duḥkhaṃ prakarotyadattam /
ulkāsvabhāvaṃ hi dhanaṃ narāṇām atyajyamānaṃ vyasanaṃ dadāti // 1.51 //
adīyamānaṃ nidhanaṃ prayāti nidhānatāṃ yāti hi dīyamānam /
dhanasya niḥsāralaghoḥ sa sāro yaddīyate lokahitonmukhena // 1.52 //
yaddattametadviduṣāṃ praśasyaṃ bālo janastannicayapraśaṃsī /
prāyo viyogo hi parigrahebhyo dānādbhavatyabhyudayo yaśaśca // 1.53 //
dattaṃ na tatkleśaparigrahāya kleśāya mātsaryamanāryadharmaḥ /
yaddīyate satpatha eṣa tasmād ato 'nyathā kāpathamāhurāryāḥ // 1.54 //
abhyāgate yācanake ca tena saṃbodhisaṃbhāravivṛddhihetau /
tatpreṣyasaṃjñātmani saṃniveśyā kalyāṇamitrapriyatā ca tasmin // 1.55 //
mahātmanāṃ yatpratanūbhavanti rāgādayo yācanakānniśamya /
tenotsavābhyāgamamapyatītya teṣāṃ priyaṃ yācanakopayānam // 1.56 //
sa cetpunaryācanake 'pi labdhe dātuṃ na śaknotyatidurbalatvāt /
tenānuneyo madhureṇa sāmnā sa yācakaḥ syānna yathā samanyuḥ // 1.57 //
kāryaśca mātsaryavinigrahāya mohaprahāṇāya ca tena yatnaḥ /
tathā yathā yācanakaḥ kadā cid vaimukhyadīno na tato vyāpaiti // 1.58 //
saṃbodhicittaṃ kuta eva tasya dravye 'pi yo matsaramabhyupaiti /
vāsaḥ sucittasya hi nāsti doṣairambhonidhānasya śavairyathaiva // 1.59 //
tasmāt tyaktvā sarvataḥ sarvadoṣān bodhiprārthī sarvadā sarvadaḥ syāt /
trātuṃ lokānekavīraḥ kva cittaṃ ceṣṭā dainyānūrjiteyaṃ kva caiva // 1.60 //
mūlaṃ dānasyāsya saṃbodhicittaṃ tanna tyājyaṃ ditsatā dānamīdṛk /
taṃ saṃbuddhāstyāgināmagramāhuryo lokeṣu tyāgamādhitsuragram // 1.61 //
// dānapāramitāsamāsaḥ //
||2. śīlapāramitāsamāsaḥ
saṃbuddhaśīlābharaṇābhirāmān kartuṃ janānutpatitādareṇa /
svameva śīlaṃ pariśodhyamādau śīlaṃ hi śakterbalamādadhāti // 2.1 //
loke tathā prema niveśayeta svapne 'pi na droharuciryathā syāt /
paropakāraikarasaḥ pareṣāṃ bhogānahīnāmiva na spṛśecca // 2.2 //
dvandvapravṛttervinivṛttabuddhiḥ prāgeva dārapraṇayāt parasya /
kurvīta lokasya hitārthakartā kāyena ceṣṭāḥ sujanasya ceṣṭāḥ // 2.3 //
mādhuryaramyāmapi kālayuktāṃ satyānukūlāmavibhedinīṃ ca /
saddharmatattvādhigamāya vāṇīṃ brūyādvipakṣāduparamya tasyāḥ // 2.4 //
kāryaṃ prayatnena mayā yadasmai tatsādhanena svayameva labdham /
parasya saukhyeṣviti tuṣṭacittaḥ kuryān manonirviṣayāmabhidhyām // 2.5 //
mamaiva daurabalyamidaṃ yadeṣa kleśāsvatantraḥ svahitaṃ na vetti /
parāparādheṣvapi kārya evaṃ vyāpādavahnipraśamāya yatnaḥ // 2.6 //
kudṛṣṭisaṃjñaṃ ca tamaḥpratānaṃ jñānaprakāśairmanaso nirasya /
kuryādahāryāṃ naradevavarye bhaktiṃ guṇābhyāsavirūḍhamūlām // 2.7 //
svargasya mokṣasya ca satpathebhyo naivoccalet karmapathebhya ebhyaḥ /
atra sthitānāṃ hi jagaddhitārthāścintāviśeṣāḥ saphalībhavanti // 2.8 //
samāsataḥ śīlamidaṃ vadanti yaḥ saṃvaraḥ kāyavacomanastaḥ /
kārtsnyena cātraiva yataḥ sa tasmād etānyayatnena viśodhayecca // 2.9 //
hiṃsānivṛttapraṇayo dadāti saumyasvabhāvādabhyaṃ janānām /
yā vāsanā doṣakṛtāsya citte tāṃ cāprayatnena samucchinatti // 2.10 //
maitrīviśeṣānugate ca citte vairānubandheṣu śamam gateṣu /
sukhaprabodhaḥ sukhameva śete kṣīṇāśubhasvapnavikāradoṣaḥ // 2.11 //
kurvanti rakṣāṃsyapi cāsya rakṣāṃ na durgatibhyo bhayamabhyupaiti /
prāpnoti cārogyaguṇābhirāmamāyuḥ prakṛṣṭaṃ sugatipratiṣṭham // 2.12 //
ataśca saṃbodhimupāgatānāṃ tathāgatānāmamitaprayāmam /
nirvartate cittavaśānuvarti lokasya saukhyopacayāya vāyuḥ // 2.13 //
anādadānastu parasya bhogān āpnoti bhogānmahataḥ paratra /
narendradāyādagaṇairahāryān girīniva śvāsanavairahāryān // 2.14 //
ācāraśuddhyānugatapriyatvaṃ viśvāsapātratvamihaiva yāti /
ataḥ paropakramanirviśaṅko gatipratīghātamupaiti naiva // 2.15 //
asārabuddhirdhanavistareṣu bhavatyayatnena viśuddhaśīlaḥ /
tasmādupakleśaviśuddhabuddhiranuttarāṃ ca svayameti bodhim // 2.16 //
kāmeṣu mithyācaraṇānnivṛtto jitendriyatvāt praśamābhirāmaḥ /
prāpnoti lokastutibhiḥ samantāt kīrtiṃ diganteṣu vikīryamāṇam // 2.17 //
na cāpi kaṃ citpramadāsu rāgaṃ karoti mātṛriva vīkṣamāṇaḥ /
asmācca puṇyopacayān munīndraḥ saṃjāyate vāraṇavastikoṣaḥ // 2.18 //
vāco 'nṛtāyāstu nivartamānaḥ prāmodyavāñchāṭhyavimuktacittaḥ /
ādeyasiddhyā vacanasya sattvān karoti dharmābhimukhān ayatnāt // 2.19 //
divaukasāṃ ca priyatāṃ yadeti satyapriyaścitramidaṃ na tādṛk /
devasvabhāvo guṇapakṣapātī pratyakṣiṇastaccariteṣu te ca // 2.20 //
pramāṇabhūto bhavati priyaśca yallaukikānāmidamatra citram /
prāyeṇa loko hi guṇairdaridraḥ svenānumānena parānminoti // 2.21 //
tyaktveva nīlotpalinīvanāni viśeṣadarśī kamalāyamāne /
tasyānane saṃśrayamabhyupaiti prahlādano gandhavidhirmanojñaḥ // 2.22 //
bhrājiṣṇunā durgatitārakeṇa jñānena paśyaṃśca samāsamāni /
sa ātmasākṣī samupaiti lajjāṃ yādṛcchikairapyaśubhairvitarkaiḥ // 2.23 //
evaṃ sa śuddhaprakṛtiḥ krameṇa na śaṅkyate 'nyairna ca śaṅkate 'nyān /
tato 'sya satyābhyanuvartanī vāgarakṣatāṃ yāti tathāgatatve // 2.24 //
kāyaḥ paropakramaṇairabhedyāḥ parairahāryā parivārasaṃpat /
paiśūnyamuktasya bhavatyabhedyā śraddhā ca dharme pratipattisārā // 2.25 //
maitrīmabhedyāmavisaṃvadantīṃ kṛpāṃ ca lokārthamasaṃtyajantīm /
prāpnoti cābhedyatamān munitve janmāntarasthānapi śiṣyasaṃghān // 2.26 //
krodhasya sainyāgrarajaḥpratānaṃ saṃkalpacaṇḍānilaviprakīrṇam /
yaśovapurdhvaṃsanamityapāsyaṃ maitryambuvāhaiḥ paruṣābhidhānam // 2.27 //
asmānnivṛtto madhurairvacobhirlokasya cetāṃsi vaśīkaroti /
lokasya ca premṇi virūḍhamūle saivāsya vāggrāhyataratvameti // 2.28 //
ataśca lokāñchataśo vinīya teṣāṃ samāvṛtya ca duḥkhamārgam /
na durgatiṃ gacchati puṇyakarmā dharmo hi rakṣeha paratra caiva // 2.29 //
dūrādapi vyaktapadānunādaḥ śrīmān adūre 'pi sukhasvabhāvaḥ /
meghasvanodagratarastato 'sya brahmasvaro vaktram alaṃkaroti // 2.30 //
abaddhavākyādvirataḥ priyatvam ekāntato yāti vicakṣaṇānām /
satyābhidhāne kramate sabuddhiḥ prāpnoti māhātmyam akṛtrimaṃ ca // 2.31 //
asmācca puṇyān munirājabhāve gāmbhīryagūḍhān paripṛcchamānaḥ /
praśnānanekānapi caikakāle niḥsaṃśayaṃ vyākurute sa vācā // 2.32 //
pretyeha cānarthaphalairavandhyāṃ vandhyāmabhidhyāṃ samapāsya buddhyā /
anīrṣyabhāvādatikāṅkṣitāṃ sa prāpnoti vistīrṇatarāṃ samṛddhim // 2.33 //
citte viśuddhe ca tadāśrayāṇi vākkāyakarmāṇi śucībhavanti /
nabhastale kālaguṇābhirāme tārāgaṇānāmiva maṇḍalāni // 2.34 //
puṇyādhipatyātkramate ca buddhistasyopabhogeṣu sadottameṣu /
prayāti rājñāmapi saṃmatatvam adhṛṣyatāṃ ca pratigarvitānām // 2.35 //
vaikalyamāyānti na cendriyāṇi satkarmanirvṛttabalāni tasya /
ataśca lokatrayapūjya ekaḥ śāstā bhavatyaprativartyacakraḥ // 2.36 //
vyāpādadāhajvaravipramuktaḥ sādhusvabhāvābhinayo nayena /
vyaktīkarotīva manaḥprasādaṃ svasthapraśāntena viceṣṭitena // 2.37 //
hiṃsātmake vigrahasaṃhite vā karmaṇyanāryācarite śaṭhe vā /
na cāsya buddhiḥ kramate kadācin maitrīsukhāsvādaviśeṣalābhāt // 2.38 //
loke vrajatyāryajanena sāmyaṃ saṃmānyate daivatavajjanena /
na brahmaloko 'pi ca durlabho 'sya prasnigdhakarmaṇyamanaḥpathasya // 2.39 //
hitābhinandī jagatāmayatnāt prasādayatyeva ca mānasāni /
ramyaḥ śaratkāla ivāpagānāṃ toyāni meghāgamadūṣitāni // 2.40 //
rūpeṇa sarvapriyadarśanena jñānāspadenādbhutaceṣṭitena /
ekīkarotīva tato munitve lokasya vijñānapṛthaktvasiddhim // 2.41 //
kudṛṣṭipaṅkakramaṇaṃ lasaṃstu prāpnoti kalyāṇahṛdaḥ sahāyān /
karmasvako 'stīti ca karma pāpaṃ viśasyamāno 'pi karoti naiva // 2.42 //
bhavatyakampyā ca jiṇe 'sya bhaktirnāyasyate kautukamaṅgalaiśca /
ārye ca mārge labhate pratiṣṭhāṃ viśeṣagāmitvamato 'bhyupaiti // 2.43 //
satkāyadṛṣṭyuccalitaḥ sa yāti na durgatiṃ hetuparikṣayeṇa /
jñānena cānāvaraṇena yukto divaḥpṛthivyorvicaratyasaṅgaḥ // 2.44 //
pratyekabuddhairapi cānavāptāḥ sarve tato 'syābhimukhībhavanti /
jagaddhitārtheṣu vijṛmbhamāṇāḥ sarvajñabhāvāya munīndradharmāḥ // 2.45 //
imāṃ vibhūtiṃ guṇaratnacitrāṃ ślāghyāṃ svayaṃgrāhaguṇābhirāmām /
ko nāma vidvān na samādadīta viśeṣataḥ sattvahitābhilāṣī // 2.46 //
divyābhirāmā manujeṣu saṃpat prakṛṣṭasaukhyaikarasā ca divyā /
śīlādyadi syāt kimivātra citraṃ yasmāt prarohantyapi buddhadharmāḥ // 2.47 //
śīlacyutastvātmahite 'pyaśaktaḥ kasmin parasyārthavidhau samarthaḥ /
tasmād viśeṣeṇa parārthasādhorna nyāyyamasmiñchīthilādaratvam // 2.48 //
vivarjayedaṇvapi varjanīyaṃ tasmādbhayaṃ tīvramavekṣamāṇaḥ /
na bodhisattvābhyucitaṃ ca śīlaṃ vikhaṇḍayedātmasukhodayena // 2.49 //
na cchidradoṣaiḥ parijarjaraṃ vā strīkelisaṃvāhanavīkṣaṇādyaiḥ /
na durjanakleśaparigrahād vā kurvīta śīlaṃ śabalaprakāram // 2.50 //
kalmāṣadoṣāpagataṃ niṣevyam ekāntaśuklopacayena śīlam /
svecchāgatitvācca bhujiṣyavṛttaṃ vidvatpraśaṃsābharaṇānavadyam // 2.51 //
samagraśikṣāpadapūraṇācca saṃpūrṇamāmarṣavivarjitaṃ ca /
cetoviśuddhipratibimbabhūtaistīvraiḥ parārthaikarasaiḥ prayogaiḥ // 2.52 //
smṛtyāśrayāccendriyasaṃvareṇa śīlasya saṃrakṣaṇatatparaḥ syāt /
lokasya dauḥśīlyamabhipravṛddhaṃ tamaḥ sahasrāṃśurivāpaneṣyan // 2.53 //
duḥkhapratīkāranimittasevyaiḥ kāyavraṇālepanaveṣṭanādyaiḥ /
nyāyopalabdhaiḥ parituṣṭacitto 'parānanollokanakātaraḥ syāt // 2.54 //
ślāghyeṣu sarveṣvapi vartamānaḥ śīlānukūleṣu guṇodayeṣu /
avismitatvādaparādhamānī kīrterbibhīyācca tadudbhavāyāḥ // 2.55 //
lābhaprakāro hi guṇaprakāśācchatrutvamabhyeti suhṛnmukhena /
saroruhāṇāmiva śītaraśmiḥ śreyaḥ pramāthī śithilavratānām // 2.56 //
śīlaṃ guṇābhyāsavidhiṃ vadanti saṃbodhicitte ca guṇāḥ samagrāḥ /
abhyasyate tacca kṛpāguṇena kāruṇyaśīlaḥ satataṃ tataḥ syāt // 2.57 //
yanniśritaṃ kāmabhave 'pi naiva saṃtiṣṭhate naiva ca rūpadhātau /
ārūpyadhātau yadasaṃsthitaṃ ca tattattvataḥ śīlamudāharanti // 2.58 //
yo lokadhātuṣvamiteṣu sattvāñchīle pratiṣṭhāpayiṣuḥ samagrān /
niṣevate lokahitāya śīlaṃ taducyate pāramiteti tajjñaiḥ // 2.59 //
śīlaṃ viśeṣādhigamasya mārgo dāyādyabhūtaṃ karuṇātmakānām /
jñānaprakarṣasya śucisvabhāvo naṣṭoddhavā maṇḍanajātiragrā // 2.60 //
lokatrayavyāpi manojñāgandhaṃ vilepanaṃ pravrajitāvirodhi /
tulyākṛtibhyo 'pi pṛthagjanebhyaḥ śīlaṃ viśeṣaṃ kurute narāṇām // 2.61 //
akatthanānāmapi dhīrabhāvād vināpi vāgbhedapariśrameṇa /
atrāsanābhyānatasarvalokaṃ tyaktāvalepoddhavamīśvaratvam // 2.62 //
apyaprakāśānvayasaṃstavānām akurvatāmapyupakārasāram /
niṣkevale śīlavidhau sthitānām asaṃstutānāmapi yannarāṇām // 2.63 //
rajāṃsi pādāśrayapāvitāni praṇāmalabdhāni samudvahanti /
cūḍāgralagnāni manuṣyadevāḥ śrīmattaraṃ śīlamataḥ kulebhyaḥ // 2.64 //
tasmān na durgatibhayena na rājyahetor na svargasaṃpadabhilāṣasamudbhavena /
seveta śīlamamalaṃ na hi tattathā syāllokārthasiddhiparamastu bhajeta śīlam // 2.65 //
// śīlapāramitāsamāsaḥ //
||3. kṣāntipāramitāsamāsaḥ
saṃmohanīṃ manmathapakṣamāyāṃ prāhuḥ sukhāṃ caiva vimokṣamāyām /
tasyāṃ na kuryāt kaiva kṣamāyāṃ prayatnamekāntahitakṣamāyām // 3.1 //
parāparādheṣu sadānabhijñā vyavasthitiḥ sattvavatāṃ manojñā /
guṇābhinirvartitacārusaṃjñā kṣameti lokārthacarī kṛpājñā // 3.2 //
parārthamabhyudyatamānasānāṃ dīkṣāṃ titikṣāṃ prathamāṃ vadanti /
seturjalānīva hi roṣadoṣaḥ śreyāṃsi lokasya samāvṛṇoti // 3.3 //
alaṃkriyā śaktisamanvitānāṃ tapodhanānāṃ balasaṃpadagrā /
vyāpādadāvānalavāridhārā pretyeha ca kṣāntiranarthaśāntiḥ // 3.4 //
kṣamāmaye varmaṇi sajjanānāṃ vikuṇṭhitā durjanavākyabāṇāḥ /
prāyaḥ praśaṃsākusumatvametya tatkīrtimālāvayavā bhavanti // 3.5 //
pratikriyā durjanavāgviṣāṇāṃ prahlādanī jñānaniśākarābhā /
dhīraprakārā prakṛtiryatīnāṃ kṣāntirguṇānām adhivāsabhūmiḥ // 3.6 //
sattvasya gāmbhīryamayasya sāro ghanāgamaḥ krodhanidāghaśāntyai /
vyatītavelasya guṇārṇavasya vyāpī svanaḥ kṣāntimayo 'bhyudeti // 3.7 //
ā brahmalokādadhirohāṇārthā sopānapaṅktirgatakhedadoṣā /
karmāntaśālā guṇaśībharasya rūpasya sallakṣaṇabhūṣaṇasya // 3.8 //
unmūlanī vairaphalācitānāṃ kṣamāsariddoṣamahādrumāṇām /
saṃbodhicittasya vivardhitasya guṇāmburaśeḥ satatānukūlā // 3.9 //
śubhā paratrāpi hite samṛddhirjagaddhitārthasya parā vivṛddhiḥ /
śubhasvabhāvātiśayaprasiddhiḥ kṣāntirmanaḥkāyavacoviśuddhiḥ // 3.10 //
saṃsāradoṣairna ca cchedameti sattvān kṛpāsnigdhamavekṣamāṇaḥ /
satkarmabhirlokahitaiḥ samantād yaśomayatvaṃ vrajatīva loke // 3.11 //
na spṛśyate vismayavācyadoṣairjñānāvadānena titikṣureva /
anityatākṣāntibalodayācca praharṣamāyāti sukhe 'pi naiva // 3.12 //
saṃkocamāyāti na cāyaśobhirvisāriṇā kṣāntibalaśrayeṇa /
ataśca śeṣairapi lokadharmairaniśritatvānna sa cāpalīti // 3.13 //
tīvraprakārairapi viprakārairna vikriyāṃ yānti satāṃ manāṃsi /
dṛḍhābhilāṣāṇi munīndrabhāve kṣāntyā balādhānasusaṃskṛtāni // 3.14 //
sa kṣāntidhīreṇa ca mānasena kaṣṭāni saṃdarśayate tapāṃsi /
darponnatiṃ tīrthakṛtāṃ manaḥsu nīcaiḥ kariṣyan hitakāmyayaiva // 3.15 //
loko 'yamātmābhiniveśasamūḍhaḥ śeṣān parānityabhimanyamānaḥ /
tadviprakārairabhibhūtacetā<;ḥ>; kṣamāviyogāt parikhedameti // 3.16 //
kṛpāsanāthāni satāṃ manāṃsi kṣāntyā kṛtasvastyayanakriyāṇi /
naṣṭātmadṛṣṭiṇi parāpakārān na vikriyāṃ yānti guṇānurāgāt // 3.17 //
mithyāvikalpo hṛdayajvarasya krodhasya heturdhṛtidurbalānām /
samyagvikalpastu samādadhāti kṣāntiprakārāṃ manasaḥ praśāntim // 3.18 //
vikalpasanniśrayasaṃśritāyāṃ kṣāntyāṃ na tu syāccalitāvakāśaḥ /
pratyūṣavātasphurite 'mbhasīva saṃpūrṇacandrapratibimbalakṣayāḥ // 3.19 //
vikalpaśāntiṃ paramārthatastu kṣāntiṃ kṣamātattvavido vadanti /
tasmādvikalpopaśame yateta svapnopamaṃ lokamavekṣamāṇaḥ // 3.20 //
cakṣuḥ kim ākrośati cakṣuretacchrotrādi vākrośati kiṃ tadādi /
yaivaṃ kṣamā sāyatanānvavekṣā na kṣāntireṣā paramārthatastu // 3.21 //
vaktā vacaścaitadanityameva śrutirvikalpo 'pi ca yo mamāyam /
anityabhāvapravikalpanaiṣā na kṣāntimetām paramāṃ vadanti // 3.22 //
kartāpakārasya na kaścidasti naivāsti kaścitkriyāte ca yasya /
nairātmasaṃdarśanasiddhireṣā na kṣāntireṣāpi gataprakarṣā // 3.23 //
tattatpratītya prabhavanti bhāvā nindāpraśaṃsāsukhaduḥkhasaṃjñāḥ /
pratītyasiddheravatārabhūmirna kṣāntiratyantasamāhitaiṣā // 3.24 //
yadyesā saṃmohamahāgraheṇa paryastacetā nanu nāhamevam /
ityunnate cāvanate ca citte kṣāntiprakarṣasya kuto 'vakāśaḥ // 3.25 //
pradhvaṃsinī varṇalavapratiśrudyantrādivaikaikaśa uccarantī /
kuryāṃ kathaṃ kasya ca kāṃ ca pīḍām eṣāpi na kṣāntiratiprakṛṣṭā // 3.26 //
yadyesā matpāpaparikṣayārthaṃ na vīkṣate svāmapi dharmapīḍām /
asmān na kalyāṇataraṃ hi mitram asāvapi kṣāntyupacārā eva // 3.27 //
karmasvatāṃ eva hi vīkṣamāṇastitikṣate tadguṇadarśanācca /
naivaṃprakārāpi hi naiṣṭhikatvaṃ kṣāntirvikalpopahatā prayāti // 3.28 //
anityaduḥkhāśuciniḥsvabhāvatā mama kṣamante na tu tadviparyayāḥ /
iyaṃ vipakṣapraśamakṣamā kṣamā dvayapravṛtterna tu pāramārthikī // 3.29 //
ayatnatattvārthavicakṣaṇo janaḥ paropakāreṣu yataḥ pravartate /
kṣamā na caivaṃ samatāṃ sameti yā yataḥ kṣamaivaṃ na vikalpanakṣayā // 3.30 //
nirodham āyānti yadā tvaśeṣatāḥ samādhikanyūnavikalpanakramāḥ /
anuttarāṃ kṣāntimamānagocarāṃ vadanti tāmadvāyamārgacāriṇāḥ // 3.31 //
svataḥ parasmādubhayādahetuto yathā na bhāvāḥ prabhavanti ke cana /
svataḥ parasmād ubhayād ahetutastathā na bhāvā vibhavanti ke cana // 3.32 //
naṣṭād anaṣṭād ubhayāc na nobhayān na jātu kāryaṃ khalu vidyate kva cit /
tathāpi kāryaṃ samudeti vastuno yetthaṃ kṣamā sā dvayavarjitā kṣamā // 3.33 //
sato 'sato vāsti na janma janmanā vinā nirodho 'pi na kasya cit kva cit /
svabhāvaśūnyāmiti bhāvakalpanāṃ vipaśyataḥ kṣāntirudeti naiṣṭhikī // 3.34 //
avāpya yāṃ vyākriyate sahasraśo jinairasau nāma jino bhaviṣyati /
pravartate lokahitakriyāvidhiḥ samāhitasyaiva ca tasya sarvadā // 3.35 //
yāvacca bhāvābhiniviṣṭabuddhiratra dvayaṃ tāvadupaiti mohāt /
tathānimittaṃ ca vimokṣaheturdure bhavatyasya yathā kṣiteḥ kham // 3.36 //
upaiti dharmapraṇidhānakarmasu prabhutvamṛddhāvadhimuktijanmasu /
tathā pariṣkāravidhau svacetasi prakarṣiṇi jñānabale tathāyuṣi // 3.37 //
avāpya caitadvaśitāmayaṃ dhanaṃ prakṛṣṭaṃ akṣiṣṇu parārthasādhanam /
janasya kṛcchreṣu patiṣyataḥ sataḥ sa jāyate dhāraṇakāraṇaṃ vibhuḥ // 3.38 //
tasmāt parārthamahatīṃ dhuramudvahadbhiḥ kṣānterupāyavidhireṣa sadānugamyaḥ /
atra sthitasya hi bhavanti parārthacittāḥ sarvā<;ḥ>; kriyā guṇaphalābharaṇābhirāmāḥ // 3.39 //
asyāṃ hi bhaktirapi yā pravirūḍhamūlā tāmabhyasanti munayo munirājabhāve /
śraddhānuviddhamanasāṃ na hi dharmamārge dṛṣṭo manoratharathasya yato 'kṣabhaṅgaḥ // 3.40 //
// kṣāntipāramitāsamāsaḥ //
||4. vīryapāramitāsamāsaḥ
sarvaṃsahe kṣāntibale ca rūḍhe sarvādbhutānyārabhate sa śauryāt /
vīryeṇa kāryāntamahābalena yasmāt sa devānapi yātyatītya // 4.1 //
sudṛśyapārāṇyapi laukikāni kāryāṇi nirvīryaduruttarāṇi /
aprāpyarūpaṃ tu na kiṃ cid asti khedānabhijñena parākrameṇa // 4.2 //
ārabdham evotsahate na hīna ārabhya madhyastu viṣādameti /
parārtham aśrāntaparākramāste nirvāṇamutsṛjya samārabhante // 4.3 //
prāyeṇa dainyopahato jano 'yaṃ svādhīnavīryo 'pi gurusvakāryaḥ /
ahīnavīryasya tu merusāro 'pyakhedasādhyaḥ parakāryabhāraḥ // 4.4 //
saṃsārakoṭyorubhayoḥ samānaiḥ prayāmasārairdivasairyadi syuḥ /
saṃvatsarāstatpracayātidīrghaiḥ kalpaiḥ samudrodakabindutulyaiḥ // 4.5 //
utpādayeyaṃ yadi bodhicittam ekaikametena parākrameṇa /
saṃbhāraśeṣaṃ cinuyāṃ tathāpi bhūyaḥsamutsāritakhedadainyaḥ // 4.6 //
ekaikamevaṃ yadi bodhicittaṃ prāpyeta saṃbhāravidhiśca śeṣaḥ /
tathāpi bodhiṃ samudānayeyaṃ kṛpāsamutsāhitadhairyasāraḥ // 4.7 //
saṃsāraduḥkhaṃ svamacintayitvā saṃnāhadārḍhyaṃ yadacintyamevam /
ādyaṃ samādānamidaṃ vadanti vīravratānāṃ karuṇātmakānām // 4.8 //
padbhyāṃ atikramya kukūlakalpāṃ kṛtsnāṃ mahīmāyudhasaṃvṛtāṃ vā /
yaddraṣṭum apyutsahate munīndrān pātuṃ śivaṃ dharmarasāyanaṃ vā // 4.9 //
saṃsārapaṅkājjanatā mayeyam uddhṛtya nirvāṇasukhe niveṣyā /
utkṣepanikṣepavidhau padānāṃ yaccittamevaṃ ca samādadāti // 4.10 //
yad vā hitārthaṃ kramate parasya puṇyāni vā lokahitāya cittam /
parākramaḥ so 'kṣayavikramāṇāṃ śrīmatsamādānavidhau dvitīyaḥ // 4.11 //
puṇyasya cotpādasamānakālaṃ saṃbuddhabhāve pariṇāmanaṃ yat /
tadakṣayatvaṃ samudāgamāya śubhaṃ samādānam udāharanti // 4.12 //
mahatsu vāmbhaḥsu yathā niṣikto naivodabinduḥ kṣayamabhyupaiti /
saṃbuddhabhāve pariṇāmitasya tathaiva puṇyasya na saṃkṣayo 'sti // 4.13 //
tathā hi kāruṇyaviśuddhabuddhiḥ sarvajñabhāvāya phalantyamūni /
puṇyāni lokasya carācarasyetyevaṃ sa tānyārabhate susattvaḥ // 4.14 //
mahātrisāhasragataṃ janaughaṃ nirvāpayedekadine na kaścit /
kalpaṃ tathā naiva ca sattvadhātostenāpi kiṃ cit paripācitaṃ syāt // 4.15 //
śrutvāpi sattvākṣayatāṃ imāṃ yaḥ sattvānaśeṣān vininīṣureva /
viṣādadoṣānavalīḍhavīryaḥ kastasya dūrastha ihārthasāraḥ // 4.16 //
yaḥ puṇyarāśirjagatāṃ samagrastāvatpramāṇairdaśabhirjinasya /
nivṛttimāgacchati romakūpa ekaika ekaikasujātaromā // 4.17 //
śatena bhūyo guṇitena tena puṇyena romāspadasaṃśritena /
bhavatyanuvyañjanamekameva śeṣāṇi tasya prabhavanti kāye // 4.18 //
tāvadguṇādeva ca puṇyarāśestasmād anuvyañjanasaṃpraviṣṭāt /
pratyekaśastasya jinatvaśaṃsi nirvartate lakṣaṇacitrakarma // 4.19 //
sallakṣaṇotpattinimittabhūtāt sahasrasaṃkhyāguṇitācca puṇyāt /
nirvartate tasya manojñavarṇā saṃpūrṇacandrasphuṭakāntirūrṇā // 4.20 //
ūrṇābhinirvṛttikarmaṃ ca puṇyaṃ śatapramāṇairguṇitaṃ sahasraiḥ /
karoti tasyānavalokanīyaṃ chattarābhamuṣṇīṣalalāmaśīrṣam // 4.21 //
ayaṃ mayā puṇyanidhiḥ parārthaṃ saṃceya ityuttamabodhicitte /
vīryonmukhe kena mukhena tasmiṃllayapravṛttirlabhatāṃ praveśam // 4.22 //
sarve 'pi sattvā yadi lokadhātau pratyekabuddhaiḥ sadṛśā bhaveyuḥ /
jñānena tebhyo 'bhyadhikaprabhāvaḥ kṣāntistha eko 'pi hi bodhisattvaḥ // 4.23 //
tathaiva ca kṣāntibalasthitebhyo viśeṣaṃ āyātyavivartanīyaḥ /
aśrāntavīryaḥ kuśalaprayoge yallaukike caiva taduttare ca // 4.24 //
tebhyaḥ puṇaścādhika eva dūraṃ ya ekajātipratibaddhabodhiḥ /
ka eva vādo dṛḍhavīryavatsu ye bodhimūle prathamaṃ niṣaṇṇāḥ // 4.25 //
tādṛgvidhajñānaviśuddhipūrṇaḥ syād yadyaśeṣena ca lokadhātuḥ /
yāyāt kalāṃ so 'pi na bodhimūle sthitasya mārātikṛtāntyajāteḥ // 4.26 //
tādṛgvidhajñānaviśuddhacittāḥ syuryadyaśeṣena ca sarvalokāḥ /
balapradeśasya muneratulyāḥ kalāpradeśairapi te samagrāḥ // 4.27 //
ityadbhutajñānasamudramekaḥ kṛpātmako nistarituṃ prayāti /
avyāhatājñaḥ paracittacāre prajñāvabhāsaṃ ca nabho viśālam // 4.28 //
sarveṣu sattveṣu ca tasya mātrā samānahārdā karuṇābhyudeti /
saṃbuddhadharmāśca tato 'vaśeṣāstasyādbhutāḥ saṃprabhavantyaśeṣāḥ // 4.29 //
ebhiḥ samādānaguṇairupetaḥ śuddhaśravaiḥ pelavasattvasattvaiḥ /
aṣṭābhiraṅgairiva tattvamārgo vīryaprakarṣādadhikaṃ vibhāti // 4.30 //
vīryaṃ tridhā yaḥ kuśalaprayogastasmācca vākkāyamanoviśeṣāḥ /
prasthānaviṣṭhānasamāhitasya vīryaprakarṣasya manomayasya // 4.31 //
yo bodhicittapraṇayaḥ samaśca kṛpā ca nairātmagatau kṣamā ca /
caturvikalpo janasaṃgrahaśca sarveṣu dharmeṣvanavagrahaśca // 4.32 //
saṃsārapaṅke yadakhinnatā ca traidhātukasyaiva ca nopalabdhiḥ /
sarvasvadānaṃ na ca tena mānaḥ samagraśikṣasya na śikṣayā ca // 4.33 //
parāpakārairavikāri dhairyaṃ cittasya cātyantamavikṣatiryā /
ārambhadārḍhayaṃ kuśalakriyāsu prītirvivekaikarasā ca citte // 4.34 //
caturvidhadhyānasamāpanaṃ ca cittasya nidhyaptiranātmataśca /
atṛptatā ca śrutavistareṇa nyāyapraveśastadavekṣaṇācca // 4.35 //
yā deśanā caiva yathāśrutānāṃ jñānaṃ ca dharmānabhilāpyatāyām /
pañcasvabhijñāsu ca yatprabhutvam abhyāsamātrā ca taduttarāyām // 4.36 //
yadṛddhipādeṣvabhinirhṛtatvaṃ paṭvī na cāyāsamayī kriyā ca /
samyakprahāṇeṣu ca yaḥ prayogaḥ śubhāśubhādeva ca yā vimuktiḥ // 4.37 //
yatkauśalaṃ cendriyanirṇayeṣu nirindriyān paśyati yacca dharmān /
mārgasya saṃbhāravimārgaṇaṃ ca na cāsya kiṃ cid gamanaṃ kutaścit // 4.38 //
ityevamādyaṃ pṛthucitravīryaṃ prasthānaviṣṭhānaviśeṣacitram /
asyākṣayatvapratipūraṇārthaṃ prasthānakarmaiva viśeṣahetuḥ // 4.39 //
nimittakarmasvapi na pravartate vitiṣṭhate jñānamaye ca karmaṇi /
kṛpāguṇādyan na jahāti saṃskṛtaṃ na coruvīryo 'pi patatyasaṃskṛte // 4.40 //
apūrvadharmaśrutiralparogatā durāsadetvaṃ śrutadharmadhāraṇam /
amānuṣebhyo 'pi parigrahodayaḥ samādhigotrapratilambha eva ca // 4.41 //
vrajantyavandhyā yadaharniśaṃ kriyā guṇairna hāniṃ yadutpaiti mauśalīm /
vivṛddha evotpalavacca yadguṇairmanuṣyadharmādadhikaprayojanaiḥ // 4.42 //
yaśo viśālam ca sukhaṃ sukhodayaṃ vinītakārpaṇyamanastvamuttamam /
guṇāśca teṣāmiha dṛṣṭadhārmikā bhavanti vīryāditi ko 'tra vismayaḥ // 4.43 //
trailokyapūjyamamitoruguṇaṃ saṃbuddhabhāvamapi yānti yadā /
vīryavyāpāśrayadṛḍhāḥ puruṣā na syād ataḥ ka iva vīryaparaḥ // 4.44 //
// vīryapāramitāsamāsaḥ //
||5. dhyānapāramitāsamāsaḥ
atha dhyānavidhau yogaṃ kuryāj jñānavivṛddhaye /
sukhaṃ hi kartuṃ lokānāṃ jñānālokādanugraham // 5.1 //
prasīdatyadhikaṃ jñānaṃ dhyānānmanasi nirmale /
śaradutsāritaghane nabhasīvendumaṇḍalam // 5.2 //
viśuddhaśīlaḥ kalyāṇaiḥ sahāyaiḥ sahitairhitaiḥ /
alpakṛtyaḥ praśāntātma smṛtyadhiṣṭhānaveṣṭitaḥ // 5.3 //
nivasan vṛkṣamūleṣu śādvalāstīrṇabhūmiṣu /
anupaskṛtaramyeṣu vanapuṣpasugandhiṣu // 5.4 //
dhyānasācivyadhīreṣu saṃtuṣṭajanaveśmasu /
jananirghoṣamūkatvād gambhīrāvasthiteṣviva // 5.5 //
pratyaraṇyaniviṣṭeṣu śūnyeṣvāyataneṣu vā /
kuñjeṣu ca mahīdhrāṇāṃ siṃhanādānunādiṣu // 5.6 //
yatra kva cana vā deśe saṃsargakleśavarjite /
paryaṅkena sukhāsīnaḥ śarīram ṛju dhārayan // 5.7 //
upasthāpya smṛtimayīṃ rakṣāṃ abhimukhīṃ hṛdi /
kṛpayā kuvitarkāṇāṃ kṛtvevākṣaṇaghoṣaṇām // 5.8 //
prajñāparicayasyāyāṃ kālo me na tu nirvṛteḥ /
na hi sattvān anirvāpya svayaṃ nirvātum utsahe // 5.9 //
iti lokahitāvekṣī buddhabhāvagataspṛhaḥ /
kuryāt sātatyayogena dhyānārambhasamudyamam // 5.10 //
na hi viśramya viśramya mathnannagnimavāpnuyāt /
sa eva yogo yoge 'pi viśeṣādhigamādṛte // 5.11 //
ekatraiva ca badhnīyād dṛḍham ālambane manaḥ /
anyānyālambanagrāhaḥ kliśnātyevākulaṃ manaḥ // 5.12 //
vidarśanād vīryabalāllīyamānaṃ samuddharet /
uddhanyamānaṃ ca manaḥ praśamena nivārayet // 5.13 //
samyaggatamupekṣitaṃ samādhibalaniścalam /
tatrāpi vā tanmayaḥ syāt sugatajñānalabdhaye // 5.14 //
na ca dhyānasukhāsvādaḥ pāratantryamanukramet /
na hi svasukhamātrārthamayam ārambhavistaraḥ // 5.15 //
śarīrajīvitāpekṣī dainyopahatamānasaḥ /
na kuryād vīryaśaithilyamapyādīpte svamūrdhani // 5.16 //
lakṣayitvā nimittāni manastasya samādhaye /
bhraśyamānaṃ prayuñjīta smṛtyāvahitayā punaḥ // 5.17 //
mano nivaraṇebhyaśca vipakṣairvinivartayet /
svecchāprayātaṃ dviradam aṅkuśākarṣaṇairiva // 5.18 //
atha nīvaraṇavyādhināśaprasvasthamānasaḥ /
dāridryādiva nirmukto mahato vyasanādiva // 5.19 //
prītiyuktena manasā kāmadoṣān vicārayet /
tadviyogopalabhyāṃ ca parāṃ sukhaparamparām // 5.20 //
vidyududdyotacapalāḥ phenāṃśukanibhātmakāḥ /
svapnavat pelavāsvādā vañcanārthamivoditāḥ // 5.21 //
pitṛṇāmapi putreṣu putrāṇāṃ ca pitṛṣvapi /
prītisarvasvabhūteṣu suhṛtsu suhṛdāmapi // 5.22 //
guṇapracayabaddhasya vyūḍhesu samareṣvapi /
darśitasthairyasārasya snehasetorvidāriṇaḥ // 5.23 //
iha paryeṣṭiduḥkhasya paratra narakasya ca /
hetubhūtā yataḥ kāmāḥ kāmayeta na tān ataḥ // 5.24 //
yadāśrayo vitarko 'pi prajñācakṣurnimīlanaḥ /
ātmano 'pi parasyāpi vighātāya pravartate // 5.25 //
ātmakāmairapi ca ye sarvathāpi vivarjitāḥ /
parārthakāmastān kāmāṃstyaktvā kathamanusmaret // 5.26 //
tṛptireṣāṃ na saṃprāptyā nāhanyahani sevayā /
naiva saṃnicayenāpi ko 'nyo vyādhirataḥ paraḥ // 5.27 //
yadāsvādahato naiva svārthamapyavabudhyate /
unmattapānapratimān kastān sahṛdayaḥ smaret // 5.28 //
ityevaṃ sarvato duṣṭān kāmāṃstasyānupaśyataḥ /
tataḥ saṃkucitaṃ cittaṃ naiṣkramye 'bhiprasīdati // 5.29 //
vivekajaṃ prītisukhaṃ tataḥ prasrabdhilabdhijam /
prāpnoti cittasyaikāgryaṃ prathamadhyānasaṃjñitam // 5.30 //
sa vitarkavicārāṇāṃ kāmānāmiva duṣṭatām /
puṣyāṃstatpraśamānveṣī samādhiprītijaṃ sukham // 5.31 //
adhyātmasaṃprasādācca cittaikāgratayā ca tat /
dvitīyaṃ dhyānamityāhuradvitīyā maharṣayaḥ // 5.32 //
utplavaṃ manaso dṛṣṭvā prīteratha virajya saḥ /
tṛtīyaṃ dhyānamāpnoti smṛtyupekṣāsamanvitam // 5.33 //
sukhabhogamapi tyaktvā sukhaduḥkhanirākṛtam /
viśuddhaṃ smṛtyupekṣābhyāṃ caturthaṃ dhyānamaśnute // 5.34 //
abhijñā labhate pañca sa ca tatrānugāminīḥ /
rājyastha iva dharmātmā hrīkīrtiśrīmatidyutīḥ // 5.35 //
pratyekajinalabdhāśca śrāvakīyā vyatītya ca /
tā bhavantyadhikā dūraṃ parārthasamudāgamāt // 5.36 //
sa hi matsariṇastyāge śīle tadvikalānapi /
kopanān kṣāntisauratye kusīdān vīryasaṃpadi // 5.37 //
vikṣiptacetaso dhyāne prajñāyāṃ tannirākṛtān /
niyojayati kāruṇyādaśrāntācāravikramaḥ // 5.38 //
ato 'cyutābhirdīptābhirbhābhirlokāvabhāsanam /
marīcibhirivādityāḥ kurute 'nantagocaram // 5.39 //
atha pāpakṛtaḥ sattvān patato narakādiṣu /
kṣīṇapuṇyāyuṣaścaiva devāñchāśvatamāninaḥ // 5.40 //
taistairduḥkhaviśeṣaiśca lokaṃ kāraṇayāhatam /
tatra divyaprabhāveṇa cakṣuṣā sā vilokayan // 5.41 //
tīvramāyāti kāruṇyaṃ kāruṇyān na pramādyati /
parārtheṣvapramattaśca yātyacintyaprabhāvatām // 5.42 //
athānyalokadhātusthān sampaśyati tathāgatān /
buddhakṣetraguṇavyūhān saṃghasyaiva ca saṃpadaḥ // 5.43 //
bodhisattvarṣabhāṇāṃ ca viśuddhācāragocaram /
sarvalokahitodarkaṃ śrīmaccaritamīkṣate // 5.44 //
tatra ca praṇidhistasya sukhenaiva samṛdhyati /
parārthapariṇāmācca śīlasyaiva ca saṃpadaḥ // 5.45 //
atimānuṣayā śrutyā divyayārthaviśuddhayā /
śṛṇvannuccāvacā vāco vidūre 'pyavidūravat // 5.46 //
kṛpādūracarairuktāḥ pāruṣyavirasākṣarāḥ /
antardīptasya kopāgnerniścarantīrivārciṣaḥ // 5.47 //
apsarogītasacivān bhūṣaṇasvanaśībharān /
divyatūryaninādāṃśca vināśaikarasānapi // 5.48 //
niṣevyamāṇān rāgāndhairamitrānmitrārūpiṇaḥ /
vīkṣya vrajati kāruṇyaṃ teṣāṃ vāñcanayā tayā // 5.49 //
bhayādduḥkhaviśeṣācca so 'vispaṣṭapadākṣaraiḥ /
nārakairārtarasitairhṛdīvābhihatastataḥ // 5.50 //
paramālambate vīryaṃ majjāgatamahākṛpaḥ /
tīkṣṇāgreṇa pratodena sadaśva iva coditaḥ // 5.51 //
nānālokasthitebhyo 'tha jinebhyo dharmadeśanāḥ /
śṛṇoti sarvasattvānāṃ nirvāṇakāṅkṣayākṣayāḥ // 5.52 //
tataḥ sa paracitteṣu vijñāyānuśayāśayān /
puṇyāṅkurān ropayati jñānasādhanavānnavān // 5.53 //
smṛtvā pūrvanivāsaṃ ca kalpakoṭisahasraśaḥ /
paśyan puṇyāni lokānāṃ tathendriyabalābalam // 5.54 //
tadāśrayavaśādṛddhyā so 'nekīkṛtavigrahaḥ /
avandhyakathanaṃ yāti yathābhājanadeśanāt // 5.55 //
kva cid arkasahasradīptināpyavisaṃvāditakāntisaṃpadā /
vapuṣā munirājalakṣaṇaḥ sphuṭacitreṇa samantalakṣmaṇā // 5.56 //
janayannayanotsavaṃ nṛṇāṃ vacasā hlādaviśeṣamācaran /
sa karotyamṛtaprakāśanaṃ jinabhāvāya jinādhimuktiṣu // 5.57 //
praśamottarayā muniśriyā kva cid atyārthaviśiṣṭaceṣṭayā /
kurute muniśiṣyarūpabhṛdvinayaṃ tadvinayārhacetasām // 5.58 //
abhisāritapādapaṅkajaḥ suracūḍāmaṇibhirmahendravat /
dhanado dhanado yathā kva cit kuha cidbrahmavadadbhutadyutiḥ // 5.59 //
kva cid unmiṣitatrilocanaḥ śaśīlekhāmalamaulibhūṣaṇaḥ /
amarādhipabhāsuradyutirbujagendraśriyamudvahan kva cit // 5.60 //
kuliśānalapiṅgalāṅguliḥ kuha cid guhyakarājarājavat /
amitājinalakṣmavān kva cid guṇaraśmirmunicandramā iva // 5.61 //
sphuṭakaustubharatnaraśmibhirvipuloraḥsthalabhāsuradyutiḥ /
garuḍadhvajarājadṛk kva cit kuha ciccaiva halāyudhadyutiḥ // 5.62 //
sitaśaktiracintyaśaktimān kuha ciccāruśikhaṇḍivāhanaḥ /
udayāstanagendrabhūṣaṇaḥ śaśisūryāmalarūpavān kva cit // 5.63 //
kuha cid dhutabhuṅmarutvatāṃ vapuṣānyatra narāśrayāśinām /
vāruṇadyutim udvahan kva cit kuha cinmanmathacāruvigrahaḥ // 5.64 //
lalitāṃ pramadānarākṛtiṃ naranārīratisaṃgalālasaḥ /
kva cid eva tapodhanaśriyaṃ vidadhat kāmaviraktamānasaḥ // 5.65 //
hṛdayāni harannṛṇāṃ kva cid guruśiṣyakṣitipālavṛttibhiḥ /
narakeṣvapi ca parabhāvato vidadhadduḥkhavimokṣaṇakṣaṇam // 5.66 //
jagatām adhimuktivistarairatha so 'nekavidhairviceṣṭitaiḥ /
karuṇāguṇasaṃtatastataḥ kurute lokahitaṃ tatastataḥ // 5.67 //
samavāpya viśeṣasaṃpadaṃ vipulāṃ dhyānaguṇaśrayādimām /
prayateta viśeṣavattaraṃ nidhicihneṣvavisaṃvadatsviva // 5.68 //
kuśale sthitirapyanūrjitā kim ahāniḥ śithilavratocitā /
prayateta vivṛddhaye tataḥ parihāṇistu viparyayādataḥ // 5.69 //
sulabhaśca samādhirudyamād anurakṣā punarasya duṣkarā /
sahasā vijigīṣuṇā yathā vijitasya praśamapratikriyā // 5.70 //
manasaḥ parivṛttilāghavaṃ paramaṃ tatra na viśvasedataḥ /
anavāpya mahīmivācalām acalāṃ bhūmimabhīradurgamām // 5.71 //
abhisaṃskṛta[mārgacāriṇaḥ] patanāntā hi samādhivistarāḥ /
ata uttamamārgabhāvanām avalambeta vikalpavarjanāt // 5.72 //
sucinityasukhātmakalpanaṃ kapaṭam saṃskṛtadambhasaṃbhavām /
samavekṣya na bhāvakalpanāpraṇayavyāpṛtamānaso bhavet // 5.73 //
viśade 'pyupalambhasaṃbhave vrajati kleśaśaravyatām ataḥ /
vyatiyāti tu māragocaraṃ tamanarthaṃ praśamayya sarvathā // 5.74 //
na hi niśrayadośaduṣito bhavati dhyānavidhirviśuddhaye /
calatānugato hi niśrayaḥ sakhaṭuṅkastata eva kathyate // 5.75 //
vyavahāravidhiprasiddhaye pratipattadbhavatīti kathyate /
na hi kiṃ cid udeti kutra cit sadasatsaṃbhavayuktyasaṃbhavāt // 5.76 //
gaganena samānamānasastribhavādapyu atha vītaniśrayaḥ /
avikalpitadhīraceṣṭito vacanenāpratiyatnaśobhinā // 5.77 //
kurute sa ca laukikīṃ kriyāṃ jagadekāntahitānuvartinīm /
na samādhibalācca hīyate vaśavartitvamavāpya cetasaḥ // 5.78 //
tataḥ paraṃ parahitatatparodyataiḥ samādhibhirvidhivihitaprayojanaiḥ /
vivardhate ghanasamaye yathodadhiḥ saridvadhūsamupahṛtairnavāmbubhiḥ // 5.79 //
// dhyānapāramitāsamāsaḥ //
||6. prajñāpāramitāsamāsaḥ
puṇyāni dānaprabhṛtīnyamūni prajñāsanāthānyadhikaṃ vibhānti /
hiraṇmayānīva vibhūṣaṇāni pratyuptaratnadyutibhāsvarāṇi // 6.1 //
kriyāsu sāmarthyaguṇaṃ hi teṣāṃ prajñaiva vistāriṇamādadhāti /
svārthapravṛttau viśadakramāṇāṃ yathā manaḥsaṃtatirindriyāṇāṃ // 6.2 //
kriyāsvayogyāni śarīrayantrāṇyāyurviyuktāni yathā na bhānti /
tathaiva kāryāṇi na bhānti loke prajñāviyogena jadīkṛtāni // 6.3 //
śraddhādikānāmapi cendriyāṇāṃ prajñāgraṇī buddhirivendriyāṇām /
guṇāguṇān vetti hi tatsanāthaḥ kleśakṣaye naipuṇametyataśca // 6.4 //
prajñāviyogāt phalalālasānāṃ naiva svatodānaviśuddhirasti /
tyāgaṃ parārthaṃ hi vadanti dānaṃ śeṣastu vṛddhyārthamiva prayogaḥ // 6.5 //
prajñāsamunmīlitacakṣuṣastu dattvā svamāṃsānyapi bodhisattvāḥ /
naivonnatiṃ nāvanatiṃ prayānti bhaiṣajyavṛkṣā iva nirviklapāḥ // 6.6 //
evaṃ sa bhūmiṃ prathamāmupaiti lokottarasyārthavidhipratiṣṭhām /
akrodhanaḥ prītisamṛddhacetā dānairmahadbhirjagadarthacetāḥ // 6.7 //
prāyeṇa yasyāṃ balacakravartī bhavatyasaṃhāryamatiśca bodheḥ /
prajñāguṇādeśitasatpatho 'tha karmeṇa bhūmiṃ vimalāmupaiti // 6.8 //
yasyāṃ prakṛtyaiva viśuddhaśīlaścaturmahādvīpapatiḥ sa bhūtvā /
narendracūḍāmaṇisatkṛtājñaḥ sūryārhatāmeti yathā munīndraḥ // 6.9 //
tataḥ paraṃ kāmiṣu daivateṣu loke 'pi ca dvitrisahasrasaṃkhye /
aiśvaryamāpnoti tataḥ paraṃ ca bhūmiṃ viśodhya prabhavāṃ prabhāyāḥ // 6.10 //
śīlasya śuddhiḥ kuta eva tasya yaḥ prajñayā nāpahṛtāndhakāraḥ /
prāyeṇa śīlāni hi tadviyogād āmarṣadoṣaiḥ kaluṣīkriyante // 6.11 //
nātmārthamapyasti tu yasya śīlaṃ prājñasya tasyāsti kathaṃ parārtham /
yo dṛṣṭadoṣo bhavabandhanānāṃ lokān samastāṃstata ujjihīrṣuḥ // 6.12 //
prajñāvipakṣairhṛdi soparāge kṣamāguṇaḥ kena dhṛtiṃ labheta /
guṇāguṇāavekṣaṇakātarākṣe khyāto guṇairvīrā iva kṣitīśe // 6.13 //
prajñānvitānāṃ tu parāpakārāḥ kṣamāguṇāḥ sthairyakarā bhavanti /
bhadrātmakānāmiva vāraṇānāṃ karamāśrayā naikavidhā viṣeṣāḥ // 6.14 //
niṣkevalaṃ vīryamapi śramāya prajñāsanāthasya tu tasya kārye /
anuttaraḥ siddhiguṇo 'bhyudeti hartā tadutthasya pariśramasya // 6.15 //
yasmāt paraṃ sūkṣamataraṃ na kiṃ cid yannaipuṇānāṃ paramaḥ prakarṣaḥ /
yatkāmadoṣādibhirāvṛtānāṃ manaḥpathaṃ naiva kadā cideti // 6.16 //
tad dhyānamekāntasukhābhirāmaṃ kathaṃ pravekṣyantyastāṃ manāṃsi /
sthūlāni doṣopacayairmahadbhiḥ prajñotpathaṃ nyāyamivāśritānī // 6.17 //
prajñānirudyogamaterhi dṛṣṭirnāyāti śuddhiṃ tadṛte na śīlam /
samyakasmādhis tadṛte na labhyo duḥkhakṣayastadvirahāttathaiva // 6.18 //
prājñastu doṣādbhayamīkṣamāṇaḥ sukhānubaddhaṃ ca sukhaṃ guṇebhyaḥ /
vihāya doṣāñjagadarthakāmo guṇābhirāmeṇa pathā prayāti // 6.19 //
samudyatastena samādhimetya prāpnoti vākkāyamanoviśuddhīḥ /
ato 'navadyena balena yuktaḥ pravartate lokahitodayeṣu // 6.20 //
dānena cābhīpsitabhūyasaiva priyairadīnairvacanāmṛtaiśca /
naiṣkāraṇorjasvalayā ca vṛttyā parārthacaryāsu samaṃ samantāt // 6.21 //
sāmānyamartheṣu ca darśayitvā premṇā vaśīkṛtya manāṃsi teṣām /
karoti nirvāṇasukhe pratiṣṭhāṃ prajñāguṇāvyāhatadharmacakraḥ // 6.22 //
prajñādyarogaiśca balairamībhiradhyāsitaṃ nābhyupayātumīsā /
ajīvikādurgatimṛtyunindāśāradyadoṣāśrayaṇī bhayārtiḥ // 6.23 //
bhayāni sarvāṇi hi doṣajāni prajñā na doṣaiḥ sahavāsameti /
śaradvyapoḍhābhragavākṣapakṣā bhā bhāskarasyeva tamaḥpratānaiḥ // 6.24 //
sahasraraśmerudaye 'pi yāni tamāṃsi rundhanti jagadgatāni /
nāmaikaśeṣāṇi karoti tāni prajñāprabhāyāḥ prasaraprabhāvaḥ // 6.25 //
na tatra bhūyaḥ karaṇīyam asti yatra prabhā sā balatāmupaiti /
yugāntakālānalasaṃhṛte hi loke na dagdhavyakathāḥ prathante // 6.26 //
jyotīṃṣi sarvāṇyapi saṃhitāni prajñāprabhāṃ nālamathopayātum /
atastayā nāsti parātivṛddhirgarīyasī vāparihāṇijātiḥ // 6.27 //
saṃpūrṇatāṃ yāti sukhena śikṣā śīlāya cittapraśamāya caiva /
prajñābhiyuktasya yatastato 'syāṃ sarvābhisāreṇa parākrameta // 6.28 //
yā skandhadhātvāyatanapravṛttau satyāśrayā pratyayitā parīkṣā /
kālatraye 'pyeṣa samāsayuktyā prajñāvadātairviṣayapraveśaḥ // 6.29 //
kīrtiṃ vitanvanti jinātmajānāṃ prajñāvadātāścaritapradeśāḥ /
guṇadvīṣāmapyatiduṣkuhāṇāṃ romāñcitā vismayapāratantryāt // 6.30 //
prajñābalaṃ dīptataraprabhāvaṃ nālaṃ prasoḍhuṃ sabalo 'pi māraḥ /
prajñāṃśavo vibhramayānti cakṣurna draṣṭum īśo hi yataḥ sa eva // 6.31 //
kandarpanārācanipātasāhī prajñāmayaṃ varma vitatya citte /
vyūḍhāni rūpaprabhṛtīnyanekānyeko 'pi nirbhīrabhibhūya yāti // 6.32 //
adhīrasātmyaṃ bhayaviklavaṃ vā mūḍhocitaṃ śokaparigrahaṃ vā /
svalpātmacitteṣvavagāḍhamūlaṃ roṣoparāgaṃ parijihmitaṃ vā // 6.33 //
dīneṣu kārpaṇyamalīmasatvaṃ kṛtāspadaṃ rāgiṣu cāpalaṃ vā /
tejovihīneṣvalasatvasattvaṃ samuddhateṣvapraśamātmakatvam // 6.34 //
tāṃstāṃśca līnānapi doṣaleśān pṛthagvidhiṣvāśrayagahvareṣu /
samudbhavantyeva parākaroti prajñā pratijñeva jagaddhitārthā // 6.35 //
niveśya doṣakṣayadhīrasaumyāṃ bhavasya tasyopari dṛṣṭilakṣmīm /
svayaṃ munīndrairabhiṣicyate yat prahlādinā vyākaraṇāmṛtena // 6.36 //
ūrṇāprabhābhiśca mahāmunīnāṃ niśīthacandradyutihāsinībhiḥ /
yadājyadhārābhirivādhvaragnirvibhāti mūrdhanyabhiṣicyamānaḥ // 6.37 //
avāpya yasmān muniyauvarājyaṃ samaṃ samantād visṛtātmabhāvaḥ /
lokasya duḥkhaṃ praśamatyayatnād rajo mahāmegha iva pravṛṣṭaḥ // 6.38 //
prajñāprabhāvopanataḥ sa sarvaḥ prabhāvisāraḥ sugatātmajānām /
ko vismayo vātra sutapriyāyā mātuḥ samīyādyadiyaṃ vibhūtiḥ // 6.39 //
daśaprakāro 'pi yadā munīnāṃ tadāśrayādeva balaprakarṣaḥ /
udetyasādhāraṇasundaraśca śeṣo 'pyasaṃkhyo guṇaratnarāśiḥ // 6.40 //
śāstrāṇi cakṣuḥpratimāni loke nidhānabhūtāṃśca kalāviśeṣān /
mantrān paritrāṇakṛto vicitrān dharmavyavasthāśca pṛthagviśeṣāḥ // 6.41 //
paryāyacitraṃ ca vimokṣamārgaṃ tattacca lokasya hitopapādi /
yadbodhisattvāḥ pravidarśayanti prajñāprabhāvābhyudayaḥ sa sarvaḥ // 6.42 //
divyapratispardhibhirindriyārthairnarendrabhāve 'pi hi bodhisattvāḥ /
na yadvirūpāṃ prakṛtiṃ vrajanti prajñā guṇāmātyasanāthatā sā // 6.43 //
paropakāraikarasā ca maitrī rāgoparāgaprativarjitā ca /
parasya duḥkheṣu parā dayā ca na śokabhārālasatāṃ gatā ca // 6.44 //
anuddhātatvaṃ mudite 'pi citte tamonirārambhamupekṣitaṃ ca /
te te guṇā<;ś>;cābhyadhikaṃ vibhānti prajñāniruddhapratipakṣamārgāḥ // 6.45 //
ko nāma lokasya parārthasādhurduḥkhaikahetūni tamāṃsi hanyāt /
avyāhatā jñānaśayāśayeṣu prajñā na cet syādatisūryādīptiḥ // 6.46 //
tatprāptaye śrutam aśītivikalpacitraṃ saṃceyam āśrayasahaṃ gurumabhyupetya /
dvātriṃśatā tadadhigamya vivardhayeta samyaṅmanaḥ samavadhānakṛtairviśeṣaiḥ // 6.47 //
alpaśruto 'ndha iva vetti na bhāvanāyā mārgaṃ vicintayati kāni ca tadvihīnaḥ /
tasmācchrutaṃ prati yateta tadāśrayā hi prajñā samudbhavati cintanabhāvanābhyām // 6.48 //
praśnairavigrahamukhaiśca kathāviśeṣairmīmāṃsayārthagativīkṣaṇayā svayaṃ ca /
prajñāvivṛddhimabhitaḥ prayateta nityaṃ dhyānena tadguṇavivṛddhikareṇa caiva // 6.49 //
prajñābhyupāyavidhireṣa samāsatastu dhyānaṃ tadarthaniyataḥ śrutivistaraśca /
tābhyāṃ samudbhavati hi prabhavo guṇānāṃ prajñāprabhāsamudayo 'gnirivāraṇībhyāṃ // 6.50 //
vidvajjanācaritamārgasamāśrayācca saṃmohahetugahanāni vivarjayeta /
tairāvṛto na hi vibhātyudayasthito 'pi toyāvalambijaladāntaritaḥ śaśīva // 6.51 //
ālasyajṛmbhitamatitvam asatsahāyā nidrānivṛttiraviniścayaśīlatā ca /
jñāne muneriva kutūhalitānivṛttirmithyābhimānaparisaṃkucitāśca pṛcchāḥ // 6.52 //
dainyena cātmaparitāpasamudbhavena vidvajjanābhigamanādarakātaratvam /
mithyāvikalpapaṭutā vitathā ca dṛṣṭirmohāya tatpraśamanāya tu tadvipakṣāḥ // 6.53 //
skandheṣu sāyatanadhātuṣu satyayuktyo<;r>; hetudbhaveṣu śucayānavinirṇaye ca /
dharmeṣu kauśalamaśeṣata eva yacca prajñāprayogaviṣayo 'ṣṭavikalpa eṣaḥ // 6.54 //
niḥsāraphenanicayairaviśeṣi rūpaṃ tisro 'pi budbudalavā iva vedanāśca /
saṃjñāpi kāmaguṇaviprasṛtān satṛṣṇān bālān mṛgāniva vilobhayate marīciḥ // 6.55 //
saṃskārajātirapi tulyaguṇā kadalyā vijñānato 'pi na ca yuktatarāsti māyā /
yanniśrayādbhramati naikavikalpaceṣṭaṃ bhūtābhibhūtakuṇapapratimaṃ śarīram // 6.56 //
nātmā tadīyamapi cātra na kiṃ cid asti saṃghāta eṣa vividhāśūcisaṃnidhānaḥ /
bālān pralambhayata eva ca sattvasaṃjñā svacchandaceṣṭa iva yantravidhau suyukte // 6.57 //
ātmā na cakṣurapi ca kṣaṇabhaṅguratvāt tadvan na cakṣuṣi na cātra yathaiva cakṣuḥ /
ādhyātmikāyatanaśeṣam aśeṣam evam ātmīyavastuviṣayo 'pi ca tadvivekī // 6.58 //
bāhyeṣu dhātuṣu śarīrasamāśritānāṃ nālpo 'pi lakṣaṇavirodhakṛto 'sti bhedaḥ /
vijñānadhāturapi ca kṣaṇikaḥ sa nātmā tasmātparo 'pi ca nabhaḥkusumaiḥ samānaḥ // 6.59 //
ityetadudbhavati kevalameva duḥkhaṃ tṛṣṇāvimūḍhamanaso vigmāttu tasyāḥ /
śāntiḥ parā bhavati tarṣaharastu mārgaḥ śīlaṃ samādhipariśuddhatayā ca dṛṣṭiḥ // 6.60 //
tattatpratītya bhavatīti viśuddhadṛṣṭirnāstyasti veti samupaiti sa naiva kiṃ cit /
māyāmayaṃ jagadidaṃ pratibhāti tasya tasmāt sukhādiṣu bhavatyavikāradhīraḥ // 6.61 //
āsīdbhaviṣyati ca yat tad apīdṛgeva kaḥ saṃbhavo yadasukhaṃ na bhaved bhavebhyaḥ /
evaṃ vyatītaviṣayeṣvapi vītarāgo naivābhinandati bhavāṃśca bhaviṣyato 'pi // 6.62 //
ākārabhedaparuṣe puruṣo 'parādhī ko nāma gūḍhanakharasphuṭadṛṣṭicihne /
tatpraiṣyavṛttikapaṭānyanucintya rajyed viśvāsameva ca yathocitamatra yāyāt // 6.63 //
evaṃ vimuktamatirapyanukampakastu kleśāntaraṃ jagadanāthamavekṣamāṇaḥ /
hīneṣu niṣpraṇayabuddhirudārabhāvān nirvātumicchati na buddhaguṇānalabdhvā // 6.64 //
lokārthasādhanavidhāvasamartharūpaṃ yānadvayaṃ samavādhūya sa pūrvameva /
kāruṇyadeśitapatho munirājayānam ātasthivān parahitaikarasasvabhāvam // 6.65 //
hīnociteṣu na matirnamati praṇītā saṃtiṣṭhate mahati nāmahatī kadā cit /
saṃsyandate śucibhireva śucisvabhāvaṃ tulyaistathānyadapi śāsvata eṣa yogaḥ // 6.66 //
svapnopamāni vigaṇayya sukhāsukhāni saṃmohadoṣakṛpaṇāṃ janatāṃ ca teṣu /
ātmārtha eva gurutāṃ katham asya yāyād vyāpārabhāramavadhūya parārtharamyam // 6.67 //
yaḥ sarvalokahitakāraṇasarvaceṣṭastyaktvātmadṛṣṭiviṣayaṃ vitathābhimānam /
sarvatra śāntamatiradvayamārgacārī so 'tyadbhutaścaritanirvṛta eva loke // 6.68 //
prajñāviśuddhikaramuttamayānametat sarvajñatā tadudayā hi mahāmunīnām /
lokasya yā nayanatāmiva saṃprayāti dīptāṃśumaṇḍalatalotpatitā prabheva // 6.69 //
saṃsāradoṣabharanirmathito 'pi naiva prajñāvivecanatayā parikhīdyate yaḥ /
nātmābhikhedapariviklavatāṃ sa yāti yānasya buddhaguṇasaṃjananasya loke // 6.70 //
paśyanti cābhutamayaṃ sugataprabhāvaṃ romāñcakañcukitasarvaśarīradeśāḥ /
tadgāminaṃ pariharanti ca yānamārgaṃ kiṃ nāma kāraṇamṛte śaṭhaceṣṭitebhyaḥ // 6.71 //
ko nāma mārakalinānabhibhūtacetāḥ saṃbuddhadharmaguṇaratnanidhānabhūtam /
sarvajñayānamapayānam anarthapaṅkād ākroṣṭumarhati na cejjagato 'sya vairam // 6.72 //
lokārthasādhanapare jinarājavaṃśe prajñānimīlitanayeṣu pariskhalatsu /
cittaṃ narasya karuṇāmṛdu kasya na syāt tanmohadoṣaśamanāya dṛḍhaṃ ca vīryam // 6.73 //
prajñāyā janayati yaḥ parāṃ viśuddhiṃ nirmokṣaḥ kathamiva tasya dūrataḥ syāt /
naivāsmātparataramasti śīlam anyat tattasmād bhajata vimokṣakāṅkṣiṇo hi // 6.74 //
// prajñāpāramitāsa[māsa]ścāyaṃ pāramitāsamāsaḥ //
viśuddhamaunīndramanastaḍāgaprasūtasūtrāntasaroruhebhyaḥ /
ādāya śurabhramareṇa samyag madhūrjitaṃ pāramitāsamāse //
