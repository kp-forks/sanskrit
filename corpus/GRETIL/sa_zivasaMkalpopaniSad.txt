#Text:Śivasaṃkalpopaniṣad
#Data entry: Maximilian Mehner
#Edition:Scheftelowitz, Isidor: »Śivasaṅkalpopaniṣad.« In: Zeitschrift der Deutschen Morgenländischen Gesellschaft 75 (1921), pp. 201-212.


yenedaṃ bhūtaṃ bhuvanaṃ bhaviṣyat parigṛhītam amṛtena sarvam /
yena yajñas tāyate saptahotā tan me manaś śivasaṃkalpam astu // 1 //

yena karmāṇy apaso manīṣiṇo yajñe kṛṇvanti vidatheṣu dhīrāḥ /
yad apūrvaṃ yakṣam antaḥ prajānāṃ tan me manaś śivasaṃkalpam astu // 2 //

yat prajñānam uta ceto dhṛtiś ca yaj jyotir antar amṛtaṃ prajāsu /
yasmān na ṛte kiṃ cana karma kriyate tan me manaś śivasaṃkalpam astu // 3 //

yaj jāgrato dūram udaiti daivaṃ tad u suptasya tathaivaiti /
dūraṃgamaṃ jyotiṣāṃ jyotir ekaṃ tan me manaś śivasaṃkalpam astu // 4 //

yasmin ṛcas sāma yajūṃṣi yasmin pratiṣṭhitā rathanābhāv ivārāḥ /
yasmiṃś cittaṃ sarvam otam prajānāṃ tan me manaś śivasaṃkalpam astu // 5 //

suṣārathir aśvān iva yan manuṣyān nenīyatebhīśubhir vājina iva /
hṛtpratiṣṭhaṃ yad ajiraṃ javiṣṭhaṃ tan me manaś śivasaṃkalpam astu // 6 //

yad atra ṣaṣṭhaṃ triśataṃ śarīraṃ yajñasya guhyaṃ navanābham ādyam /
daśa pañcatriṃśataṃ yat paraṃ ca tan me manaś śivasaṃkalpam astu // 7 //

ye pañca pañcā daśataṃ śataṃ ca sahasraṃ ca niyutaṃ nyarbudaṃ ca /
te yajñacitteṣṭakāttaṃ śarīraṃ tan me manaś śivasaṃkalpam astu // 8 //

vedāham etaṃ puruṣaṃ mahāntam ādityavarṇaṃ tamasaḥ parastāt /
tasya yoniṃ paripaśyanti dhīrās tan me manaś śivasaṃkalpam astu // 9 //

yena karmāṇi pracaranti dhīrā viprā vācā manasā karmaṇā ca /
saṃvidam anu saṃyanti prāṇinas tan me manaś śivasaṃkalpam astu // 10 //

ye mano hṛdayaṃ ye ca devā ye antarikṣe bahudhā caranti /
ye śrotraṃ cakṣuṣī saṃcaranti tan me manaś śivasaṃkalpam astu // 11 //

yena dyaur ugrā pṛthivī cāntarikṣaṃ ye parvatāḥ pradiśo diśaś ca /
yenedaṃ jagad vyāptaṃ prajānāṃ tan me manaś śivasaṃkalpam astu // 12 //

yenedaṃ sarvaṃ jagato babhūvur ye devā api mahato jātavedāḥ /
tad evāgnitapaso jyotir ekaṃ tan me manaś śivasaṃkalpam astu // 13 //

acintyaṃ cāprameyaṃ ca vyaktāvyaktaparaṃ ca yat /
sūkṣmāt sūkṣmataraṃ jñānaṃ tan me manaḥ śivasaṃkalpam astu // 14 //

asti vināśayitvā sarvam idaṃ nāsti punas tathaivā dṛṣṭaṃ dhruvaṃ /
asti nāsti hitaṃ madhyamaṃ padaṃ tan me manaḥ śivasaṃkalpam astu // 15 //

asti nāsti viparīto pravāho 'sti nāsti sarvaṃ vā idaṃ guhyam /
asti nāsti parāt paro yat paraṃ tan me manaḥ śivasaṃkalpam astu // 16 //

parāt parataraṃ yasya tatparāc caiva tat paraṃ /
tatparāt paratojñeyaṃ tan me manaḥ śivasaṃkalpam astu // 17 //

parāt parataro brahma tatparāt parato hariḥ /
tatparāt parato hy eṣa tan me manaḥ śivasaṃkalpam astu // 18 //

gobhir juṣṭo dhanena hy āyuṣā ca balena ca /
prajayā paśubhiḥ puṣkalādyaṃ tan me manaḥ śivasaṃkalpam astu // 19 //

prayataḥ praṇavo nityaṃ paramaṃ puruṣottamaṃ /
oṃkāraṃ param ātmānaṃ tan me manaḥ śivasaṃkalpam astu // 20 //

yo vai vedādiṣu gāyatrī sarvavyāpī maheśvaraḥ /
yad dviruktaṃ tathādvyaiśyaṃ tan me manaḥ śivasaṃkalpam astu // 21 //

yo vai veda mahādevam paramaṃ puruṣottamaṃ /
yaḥ sarvaṃ yasya cit sarvaṃ tan me manaḥ śivasaṃkalpam astu // 22 //

yo 'sau sarveṣu vedeṣu pathate hy aja īśvaraḥ /
akāyo nirguṇo dhyātvā tan me manaḥ śivasaṃkalpam astu // 23 //

kailāśaśikhare ramye śaṃkarasya śubhe gṛhe /
devatās tatra modyanti tan me manaḥ śivasaṃkalpam astu // 24 //

kailāśaśikharābhāsā hi bhavad girisaṃsthitāḥ /
nīlakaṇṭhaṃ trinetraṃ ca tan me manaḥ śivasaṃkalpam astu // 25 //

ābrahmastambaparyantaṃ trailokyaṃ sacarācaram /
utpātitaṃ jagad vyāptaṃ tan me manaḥ śivasaṃkalpam astu // 26 //

ya imaṃ śivasaṃkalpam sadā dhyāyanti brāhmaṇāḥ /
te paramokṣaṃ gamiṣyanti tan me manaḥ śivasaṃkalpam astu // 27 //

tryambakaṃ yajāmahe sugandhiṃ puṣṭivardhanaṃ /
urvārukam iva bandhanān mṛtyor mukṣīya māmṛtāt tan me manaḥ śivasaṃkalpam astu // 28 //
