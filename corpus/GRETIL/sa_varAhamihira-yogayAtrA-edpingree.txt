||chap.1 daivapuruṣakāra
yasminn udayati vikasati kamalam ivāstaṃ prayāti saṃkucati /
sacarācaraṃ tribhuvanaṃ sa jayati kiraṇālayaḥ savitā // 1.1 //
yakṣeśamedhena vijitya dhātrīm ity evam abhyudyamino nṛpasya /
vinighnato vighnakarān na pāpaṃ kriyāt krameṇaiva paśūn makheṣu // 1.2 //
utsāhamantraprabhuśaktiyukto dhīmān vinītendriyabhṛtyavargaḥ /
prajānurakto dhṛtimān sahiṣṇur vṛddhopasevī vijigīṣur iṣṭaḥ // 1.3 //
sāmvatsaras tasya vinītaveṣo dhīmān svatantrāṅgapaṭuḥ kulīnaḥ /
dakṣaḥ pragalbho 'vikalo vinītas tādṛg vidhastasya purohito 'pi // 1.4 //
tasya daivanarakārasametāṃ varṇayanti kavayaḥ phalasiddhim /
tatra kecid avadhūya nṛkāraṃ daivam eva phaladāyakam āhuḥ // 1.5 //
labdhavyāny eva labhyate gantavyāny eva gacchati /
prāptavyāny eva prāproti duḥkhāni ca sukhāni ca // 1.6 //
utthānāc ced bhavet siddhir na kaścit prāpnuyāt phalam /
ahorātraṃ viceṣṭanto dṛśyante vṛttikarśitāḥ // 1.7 //
vihaṅga iva vadho hi(syāt) nipatyāryamanīśvaraḥ /
vidhānavihito 'deśyo nānyeṣāṃ nātmanaḥ prabhuḥ // 1.8 //
tasmād daivaṃ pradhānaṃ hi na kuryāt karma mānuṣam /
niśceṣṭam api loke 'smin bhajate bhuktir uttamā // 1.9 //
āghātadurjayanayaṃ paribhūya daivam ātmābhimānacapalāḥ pracaranti ye 'rīn /
teṣāṃ cirād api kṛtāntamahāpaśūnāṃ hastagrahaṃ na samupaiti jitāpi lakṣmī // 1.10 //
tathā pare nāpeksyaiva daivaṃ dairyāvalambinaḥ /
pratyekṣinaḥ kriyāsiddhau kevalaṃ jagur udyamam // 1.11 //
utthānahīno rājā hi buddhimān api sarvadā /
pradharṣanīyaḥ śatrūṇāṃ bhujaṅga iva nirviṣaḥ // 1.12 //
utthānadhīraḥ puruṣo vāgdhīrān adhitiṣṭhati /
utthānadhīraṃ vāgdhīrā ramayanta upāsyante // 1.13 //
vidhānagaṇanājaḍaḥ puruṣakārasuptādaro manorathapariśramair na paricumbati śrīmukham /
parākramaviniścitaikasunayo hi sadyaḥ śrutaṃ harir madasuvāsitaṃ pibati kuṃjarāśruṃ madhu // 1.14 //
kṛṣivṛṣṭisamānugā dṛśyante phalasiddhayaḥ /
asminn arthe śṛṇu ślokān dvipāyanamukhodgatān // 1.15 //
na vinā mānuṣaṃ daivaṃ daivaṃ vā mānuṣaṃ vinā /
naikaṃ nirvartayaty artham ekāraṇirivānalam // 1.16 //
siddhyante sarva ārambhāḥ saṃyogāt karmaṇo 'rddhyoḥ /
daivāt purusakārāc ca na tv ekasmāt kathājcana // 1.17 //
anuśāsti naraṃ daivam ihāsyety anuśāsakaḥ /
iha nāsya bhaviṣyāmi kutaḥ phalamanīhitam // 1.18 //
yatnena sampadya manuṣyakāraṃ yatnāvakāśe puruṣo niruddhaḥ /
pratīkṣate daivamataṃ dvitīyas tam āpado nātmakṛtāḥ spṛśanti // 1.19 //
sattvārjitaṃ karmaphalaṃ sukhena vipakṣyate rājasamudyame[na] /
kṛcchreṇa śeṣaṃ mahatā yato 'to bhājyāni mṛgyāṇi nṛbhir nṛkāraiḥ // 1.20 //
||chap.2 tithiguṇa
yātrākālam ataḥ paraṃ munimatāny ālokya saṃkhyāpayābhyuddhuṣya pṛthūktam akṣaragurau sampannnam īśaṃ bhuvaḥ /
jñāte janmani tasya yānasamayaṃ vāṃchanti horāvidas tv ajñāte 'py aphaleṣu nākṣaram iva prāhātra ratnāvalim // 2.1 //
vidite horārāśau sthānabalaparigrahe grahāṇāṃ ca /
āyuṣi ca parijñāte śubham aśubhaṃ vā phalaṃ vācyam // 2.2 //
anye vadanty avidite (tva)pi janmakāle yogyaḥ prayāṇasamayo manujeśvarāṇām /
praśnodayodbhavaphal(ānvita)siddhibhāgyas tattvārthavinniga(da)ti sma vaco vaśiṣṭhaḥ // 2.3 //
apṛcchataḥ pṛcchato vā yiyāsor yasya kasya cit /
horākendratrikoṇebhyas tasya vindyāc chubhāśubham // 2.4 //
daivajñasya hi daivena sadasatphalavāṃchataḥ /
avaśo gocaraṃ martyaḥ sarvaḥ samupanīyate // 2.5 //
aśrauṣīc ca purā viṣṇor jñānārthaṃ samupasthitaḥ /
vacanaṃ lokanāthasya niḥsṛtaṃ mukhapaṅkajāt // 2.6 //
yat sāraṃ pṛcchataḥ puṃso graharā(śyā)śritaṃ phalam /
tatsāraṃ tasya janma syād ya(dya)py anyagṛhe bhavet // 2.7 //
budhvā śāstraṃ yathānyāyaṃ balābalavidhānataḥ /
yathoktaṃ jātake sarvaṃ tathaivātrāpi cintayet // 2.8 //
tasmān nṛpaḥ kusumaratnaphalāgrahastaḥ prātaḥ praṇamya ravaye haridiṅmukhasthaḥ /
horāṅgaśāstrakuśalān hitakāriṇaś ca saṃhṛtya daivagaṇakān sakṛd eva pṛccheta // 2.9 //
atha nṛpatisamīpe daivavit pṛṣṭamātrah phalam udyanimittais tarkayec chāstradṛṣṭyā /
sad iti sadasi vācyaṃ yady api syād asat tat sphuṭam pai kathanīyaṃ bhūbhuje mantriṇe vā // 2.10 //
sumadhuraphalapuṣpakṣīravṛkṣāvṛtāyāṃ caraṇagatisukhāyāṃ gośakṛtpeyavatyām /
salilakusumavatyāṃ snānino bhadramūrdhvaṃ grahabhagaṇagatijñaḥ prādiśet pṛcchakasya // 2.11 //
stanacaraṇatalauṣṭhāṅguṣṭhahastoparāṅgaṃ śravaṇavadananāsāguhyarandhrāṇi bhūpaḥ /
spṛśati yadi karāgrair gaṇḍakaṭyaṃsakaṃ vā pravadati śubhaśabdaṃ vāhinī śāsti śatrūn // 2.12 //
udayam udayapaṃ vā janmabhaṃ janmapaṃ vā tad upacayagṛhaṃ vā vīkṣya lagne yiyāsoḥ /
vinihatam aripakṣaṃ viddhi śatror idaṃ vā yadi hibukasametaṃ pṛcchato 'stasthitaṃ vā // 2.13 //
pārṣṇigrāhaḥ pṛṣṭhato bhāskarasya prāg yātavyastigmaraśmeś ca yānaḥ /
ākrandārkāt saptame yaḥ sthitaś ca tattulyāste śaktitaś cintayīyāḥ // 2.14 //
jīvajñārkasitāparaiḥ sutasuhṛdduścitkalagnāyagaiḥ siddhārtho 'rigaṇān vijitya na cirāt praṣṭā sametyālayam /
lagne vā kujamandayoḥ sutagate jīve ravau karmage lābhe karmaṇi vā sitendusutayoḥ praṣṭur jayo niścitaḥ // 2.15 //
tasminn eva bhaved yoge yadi śukro 'ṣṭam āśritaḥ /
praṣṭāram udyamād eva hataśaktir viśasty ariḥ // 2.16 //
gurvarkaśaśibhiḥ siddhir lagnāridaśamasthitaiḥ /
tadval lagnārirandhrasthairjīvaśukradivākaraiḥ // 2.17 //
sacivāptir gurāvāye karmaṇy asitavakrayoḥ /
caturthe jñe 'ṣṭame candre jāyāsthe ca site jayaḥ // 2.18 //
karmaṇy āre ravāvāye tṛtīyasthe 'rkanandane /
candre ṣaṣṭhe vilagnasthaiḥ śeṣaiḥ praṣṭurdhruvo jayaḥ // 2.19 //
sahaje 'rkārkibhūputraiḥ saumyair lagnagatair jayaḥ /
gurau vā lagnage pāpair lābhe karmaṇi vā sthitaiḥ // 2.20 //
māheyama(nda)yoḥ ṣaṣṭhe lagne guruśaśāṅkayoḥ /
sitacandrajayor ante jayaḥ karmaṇi tīkṣṇagau // 2.21 //
ity yogāḥ śubhāḥ proktāḥ praśnakāle jayāśritāḥ /
aśubhās tu bhavanty atra tān pravakṣyāmy ataḥ param // 2.22 //
yuddhe bhaṅgo yamendvārair navamātmajalagnagaiḥ /
śaśāṅkayamayor lagne mṛtyur bhūsutadṛṣṭayoḥ // 2.23 //
savakre nidhane mande mṛtyur lagne divākare /
candre 'smin tryāyamṛtyusthe sasūrye vā vaded vadham // 2.24 //
vakrajñaśaśibhir dyūne praṣṭur nāśo 'pi gacchataḥ /
kṣunmārāśatruvṛddhiś ca lagne māheyaśukrayoḥ // 2.25 //
candrāvanījayor mūrtau ṣaṣṭhe śaśijaśukrayoḥ /
nidhanasthe sahasrāṃśau vijñeyo mantriṇo vadhaḥ // 2.26 //
tanayasya vadhaḥ praṣṭuḥ pāpair udayaputragaiḥ /
sacandre rudhire lagne bhaṅgaḥ sūryātmajekṣite // 2.27 //
dyūnanaidhanage candre lagnaṃ yāte divākare /
viparyaye vā yātasya trāsabhaṅgavadhāgamaḥ // 2.28 //
dvitrikendrasthitaiḥ pāpaiḥ saumyaiś ca balavarjitaiḥ /
aṣṭamasthe niśānāthe praṣṭur bandhavadhātyayāḥ // 2.29 //
kujacandramasor dyūne svabhedo 'rkabudhodaye /
tadvan mandārayor yuddhe bhaṅgaḥ saumyārkayos tathā // 2.30 //
sarvais tu navame rājā hanti mantripurohitān /
yoge 'sminn udaye candre sutasthapatideśikān // 2.31 //
arkārkasutayor lagne dṛṣṭayoḥ kṣitisūnunā /
candre 'ste vibalaiḥ saumyaiḥ praṣṭuḥ senāpater vadhaḥ // 2.32 //
vidhane vakrayamayoścandre 'ste lagnage ravau /
jñe tṛtīye ca vikramya samantrī hanyate nṛpaḥ // 2.33 //
mūrghodayaṃ śubhasuhṛdyutavīkṣitaṃ vā lagnaṃ śubhāś ca balinaḥ śubhavargalagne /
siddhipradaṃ bhavati neṣṭam ato 'nyathā taj janmaprayāṇaphalayuktibhir anyad udyam // 2.34 //
nidhanahibukahorāsaptamārtheṣu pāpā na śubhaphalakarāḥ syuḥ pṛcchatāṃ mānavānām /
daśamabhavanayukteṣv eṣu saumyāḥ praśastāḥ sadasad idam aśeṣaṃ yānakāle 'pi cintyam // 2.35 //
||chap.3 titiguṇa
nandā bhadrā vijayāś cātha riktāḥ pūrṇāś caitāḥ phalam evaṃ vidadhyuḥ /
neṣṭā madhyā pravarāś ceti śukle kṛṇṇe cintyās tithayas tāḥ pratīpāḥ // 3.1 //
sarvāḥ śastās tithayaḥ śuklapakṣe hitvā tryṃśaṃ bahule 'ntyaṃ ca kaiścit /
ṣaṣṭhyaṣṭamyor vasulabdhāpi yātā dvādaśyāṃ vā nāśam āśu prayāti // 3.2 //
pratipadi phalam eke pañcadaśyāṃ ca neṣṭaṃ jagur idam iti tajjñair yāpyam evāvadhāryam /
kim api kim api nūnaṃ taiḥ samīkṣyoktam etat samavacanavighāte yuktatā kena cintyā // 3.3 //
||chap.4 nakṣatrabala
diśi diśi bahulādyāḥ saptakāḥ prākpravṛttāḥ pavanadahanadiksthas tiryag atyugradaṇḍaḥ /
surapatir api kṛcchraṃ yāti taṃ laṃghayitvā nanu bhavati virodho dikṣu daṇḍaikagāsu // 4.1 //
prāgdvārikair analadiṅ na virodham eti śeṣāḥ pradakṣiṇagatāḥ vidiśaḥ prakalpyāḥ /
ullaṃghya daṇḍam api kāmam iyān narendraḥ śūlaṃ vihāya yadi diṅmukhalagnaśuddhiḥ // 4.2 //
jyeṣṭhāyāṃ puruhūtadiṅmukhagataḥ prāpto balir bandhanaṃ yāmyām ājapade muraś calitavān yāto murārer vaśam /
rohiṇyāṃ namuciḥ pratīcy abhigataḥ pūrtiṃ gato vajriṇā saumyām aryamadaivate ca gatavān mṛtyor vaśaṃ śambaraḥ // 4.3 //
puṣyo 'tha hastaḥ śravaṇaḥ śraviṣṭhā prācyādimukhyāny udag aśviyuk ca /
naiśākaraṃ tvāṣṭram athānurādhā pauṣṇaṃ ca madhyāni tathāhur eke // 4.4 //
sarvadvārikasaṃjñitāni gurubhaṃ hastāśvimaitrāṇi ca śreṣṭhāny aindavapauṣṇaviṣṇuvasubhāny ādyaiḥ sahāṣṭau sadā /
raudrājāhiyamānilānalamaghāśāktāgnibhir ninditaṃ yānaṃ tvaṣṭranilāntare 'gniyamayoḥ pitryoraje cāntare // 4.5 //
sve sve karmaṇi pūjitāni munibhiḥ śuddhāni sarvāṇy api tyaktvārkodayam ānale 'riviṣayaṃ yātrā dighakṣoḥ śubhā /
rohiṇyāṃ triṣu cottareṣu vijayo yātur viśākhāsu ca tyaktvā vāsarapūrvabhāgam avadad gargo 'dhirājyārthinaḥ // 4.6 //
mūlendūragaśaṅkareṣv arivadhe yāyād dinārdhaṃ vinā bhṛtyarthaṃ pavanāśvisūryagurubheṣv ahno 'parārdhaṃ vinā /
rātryādau tu na pauṣṇamaitraśaśibhatvāṣṭreṣu vāṃchan dyutiṃ rātrer madhyam apāsya pūrvabharaṇīpitryeṣu hantuṃ parāḥ // 4.7 //
niśāntabhāge triṣu vaiṣṇavādyeṣv iyād dhanārthī na punarvasau ca /
niṣedhayanty ambupasaṃjñakasya madhyāhnam eke saniśāntabhāgam // 4.8 //
yatheṣṭavelāgamanaṃ praśastaṃ hastaindavopendrasurejyabheṣu /
kṛtvā prayāṇaṃ śravaṇe śriyo 'rthī viśen na jātuḥ kṣitipaḥ svasīmni // 4.9 //
saumye gatvādhyuṣya raudre 'ditīśe saṃprasthātā bādhate śatrusaṃghān /
maitre gatvā pauruhūte samuṣya? mūle yāyāc chatrunāśāya bhūpaḥ // 4.10 //
haste gatvā svāticitre samuṣya śakrāgnyṛkṣe prasthito bādhate 'rīn /
tiṣye pauṣṇe vāsave caikarātraṃ sīmni sthitvā bhūtim āpnoti yātā // 4.11 //
ṛkṣe pāpaiḥ saṃyute 'ntaḥ prakopo lakṣmībhraṃśo rāhuṇārkeṇa rogaḥ /
ketūlkābhyāṃ pīḍite deśanāśaḥ kleśaḥ saure soṣṇaraśmyātmaje ca // 4.12 //
diṅm(ārga)dhūpapariveśavikampapāṃsuyukteṣv athānayagadaśramarogapīḍāḥ /
bhinne graheṇa vaśatām upayāti śatroḥ sandhyārātre 'śanihate 'javighāta uktaḥ // 4.13 //
janmādi karma tato dṛśan na(va)maṃ saṃghātikaṃ coḍubhāgam /
samudayam (ā)dyād daśa(maṃ) bhramavināśasaṃjñaṃ ca trayorviṃśam // 4.14 //
ādyāt tu paṃcaviṃśaṃ mānasametaṃ naraḥ ṣaḍṛkṣaḥ syāt /
mānavanakṣatro nṛpatir ādyāt tad evābhiṣekarkṣam // 4.15 //
nāmānurūpam eṣāṃ sadasatphalam iṣṭapāpaguṇadoṣaiḥ /
janmarkṣādiviśuddhau yātā śatrūn jayaty acirāt // 4.16 //
janmabhakarmādhānoditās tu sampadvipatkarāḥ kṣemāḥ /
pratyarisādhakavaināśikāḥ syur mitrātimitre ca // 4.17 //
mitrātimitrasādhakasampadaḥ kṣemeṣu karmabhe yathā /
prāpnoti nāvaśeṣeṣv abhiṣekapurastād yātas tu // 4.18 //
sitasiddhārthakaśālipriyaṅgumadayantikāmbubhiḥ puṣye /
snātvā yāyāt prācīṃ saumye sahasramaṇitoyaiḥ // 4.19 //
gośṛṅgāpyanadīgirivalmīkataḍāgamṛtsamāyuktaiḥ /
toyaiḥ snātvā nṛpatir yāyād yāmyāṃ diśaṃ haste // 4.20 //
saridabhyākulamṛḍbhiḥ snātvā (ca)caitre prāgdiśaṃ yāyāt /
śravaṇe tu nadīsaṅgamamṛtsalilaiḥ paccimām eva // 4.21 //
madhubhadradārusahitaiḥ snātvā toyair udak śraviṣṭhāsu /
gogajaviṣāṇakośair madhughṛtapūrṇair udak pauṣṇe // 4.22 //
udag vai vaiśvadeve snātvā madhupūthikāpūrtasalilaiḥ /
naksatradevamantraiś cāgniṃ hutvā jayaṃ labhate // 4.23 //
akṣatamāṣāḥ khinnās tilasahitās taṇḍulā dadhi ca gavyam /
vṛṣabhapiśitaṃ mṛgasya ca pañcānām āśvinādīnām // 4.24 //
rudhiravilāyanapāyasabhujaṅgamāṃsāni śāṅkarādīnām /
pitrye tilaudanaṃ ṣaṣṭikānnam ṛkṣadvaye parataḥ // 4.25 //
prāśyāḥ priyaṅgucitrāṇḍajāḥ phalaṃ yāvakaṃ kulatthāś ca /
madhusarpiṣī ca hastān mūlāny ambūni saktavo mūlāt // 4.26 //
śravaṇādīnām adyāc chāliṃ śākaṃ biḍālamāṃsaṃ ca /
ājyaṃ yatheṣṭamāṃsaṃ ca saktavo māṣasaṃyuktāḥ // 4.27 //
prāśyādi ghṛtaṃ tilaudanaṃ matsyāḥ kṣīram iti pradakṣiṇam /
adyān nṛpatir yathādiśaṃ nakṣatrābhihitaṃ ca siddhaye // 4.28 //
asvādu cyutakacamakṣikānuviddhaṃ durgandhi kṣayakṛd abhūri yac ca dagdham /
susvinnaṃ mudu ruciraṃ manonukūlaṃ svādvannaṃ bahu vijayāya yānakāle // 4.29 //
guṇavati tithāv ṛkṣe 'niṣṭe divā gamanaṃ hitaṃ niśi ca bhagane śastaṃ yānaṃ tithau guṇavarjite /
bhatithigaditān doṣān prāpnoty ataḥ pratilomago guṇam api tayoḥ samyag yātur jagāda bhṛgur muniḥ // 4.30 //
||chap.5 vāraphala
udaranayanarogaśvāpadāraṇyavādhāḥ savitṛdivasayātāpyaśnute 'rthakṣayaṃ ca /
anilakaphajarogān śaktimānānnahāniṃ salilajanitapīḍāṃ cāhni yātā himāṃśoḥ // 5.1 //
jvalanavadhaviṣāsṛkpittaruk śatrupīḍām avanijadinayātā vādhyate śatrusaṃghaiḥ /
ahani savitṛsūnor dainyam āpnoti gacchan svajanadhanaviyogaṃ mṛtyubandhāmayāṃś ca // 5.2 //
budhadivasagato 'rīn bādhate mantraśaktyā sravaṇasukhakathāptiṃ śilpimitrāgamāṃś ca /
kṣitidhanajayaratnastrīpratāpapramodān atibalam acireṇa svīkaroty ahni sūreḥ // 5.3 //
pravarayuvatiśayyāvastragandhānnapānasmarasukhadhanaratrāny ahni bhuṃkte sitasya /
anupacayagato 'haḥ śastam apy apraśastaṃ śubham upacayasaṃsthaḥ pāpasaṃjño vidhatte // 5.4 //
||chap.6 muhūrta
garavaṇijaviṣṭiparivarjitāni karaṇāni yātur iṣṭāni /
garam api kaiś cic chastaṃ vaṇijaṃ ca vaṇikkriyāsv eva // 6.1 //
śivabhujagamitrapitṛvasujalaviśvaviriñcipaṅkajaprabhavāḥ /
indrāgnīndraniśācaravaruṇāryamayonayaś cāhni // 6.2 //
rudrājāhirbudhnyāḥ pūṣā dasrāntakāgnidhātāraḥ /
indvaditiguruhariravitvaṣṭranilākhyāḥ kṣaṇā rātrau // 6.3 //
ahnaḥ pañcadaśāṃśo rātreś caivaṃ muhūrta iti saṃjñā//
sa ca vijñeyas tajjñaiś chāyāyantrāmbubhir yuktyā // 6.4 //
nakṣatravat kṣaṇānaṃ parighādi tadīśvaraiḥ samaṃ cintyaṃ /
phalam api tad eva dṛṣṭaṃ gargādyais tatra ca ślokāḥ // 6.5 //
ahorātraṃ ca sampūrṇaṃ candranakṣatrayojitam /
tannakṣatramuhūrtāś ca samakarmaguṇāḥ smṛtāḥ // 6.6 //
aṣṭame 'rdhadivase same śubho yo viriñcavibhusaṃjñitaḥ kṣaṇaḥ /
tena yānam apahāya dakṣiṇāṃ sarvadikṣv abhijitā praśasyate // 6.7 //
||chap.7 candrabala
satya āha gatikarmakalpahā janmagas tu himaguryiyāsataḥ /
tatra dehanavate 'vajāyate tena karmasu na sādhakaḥ smṛtaḥ // 7.1 //
tac ca tasya bahubhir viśodhitaṃ lagnam eva sutarāṃ yatas tanuḥ /
tulyadehanavatāṃ prapadya kiṃ sodayodayam uvāca śobhanam // 7.2 //
aṣṭavargapariśodhitaḥ śaśī śreṣṭhatāṃ samanuvartate yadā /
janmago 'pi hi tadā praśasyate yo 'ṣṭavargaśubhadaḥ sa śobhanaḥ // 7.3 //
saptamāyadaśaṣaṭtrijanmago neṣṭado dvinidhanopagair grahaiḥ /
bandhuriḥphanavapañcamasthitaiś ceṣṭado yadi vilomaveśmagaḥ // 7.4 //
yasya gocaraphalapramāṇatā tasya vedhaphalam iṣyate na vā /
prāyaśo na bahusaṃmataṃ tv idaṃ sthūlamārgaphalado hi gocaraḥ // 7.5 //
yasyotsṛjaty uḍupatiḥ puruṣo 'pasavyaṃ janmarkṣam āpadam upaiti sa bhūmipālaḥ /

pakṣasy ādāv apacayagṛhe hānir āpūryamāṇe vṛddhis tūktā hrasati himagau kṛṣṇapakṣādisaṃsthe // 7.7 //
upacayagṛhayuktaḥ savyagaḥ śuklapakṣe śubham abhilaṣamāṇaḥ saumyamadhyasthito vā /
sakhivaśigṛhayuktaḥ kārakarkṣe 'pi cendur jayasukhaphaladātā tatprahartāny athātaḥ // 7.8 //
prakṣīṇe 'pya anupacayasthite 'pe yāyād viśrabdhaṃ yadi sakhivaśyakārakarkṣe /
naivaṃ ced udayagṛhāt svajanmabhād vā sampūrṇe 'py upacayage 'pi na prayāyāt // 7.9 //
bhṛgusutabudhabhinne ruk sureḍyena mṛtyur bhayam asitakujābhyāṃ ketunā strīpraṇāśaḥ /
apaṭukiraṇakāntau gacchataḥ śatruvṛddhiḥ savikṛtapariveṣe netranāśaḥ śaśāṅke // 7.10 //
||chap.8 lagnabala
dvipadavaśagāḥ sarve siṃhaṃ vihāya catuṣpadāḥ salilanilayābharkṣāvaśyāḥ sarīsṛpajātayaḥ /
mṛgapativaśe tiṣṭhanty ete vihāya sarīsṛpān na kathitagṛheṣūhyaṃ vaśyaṃ janavyavahārataḥ // 8.1 //
sthalajalajasarīsṛpāḥ svakānāṃ balam abhivīkṣya vidheyatāṃ bhajante /
viṣamagṛhavaśe samā dyusaṃsthā niśi viṣamā vaśavartinaḥ samānām // 8.2 //
śīrṣodaye samabhivāṃchitakāryasiddhiḥ pṛṣṭhodaye viphalatāṃ balavidravaś ca /
yātavyadiṅmukhagatasya sukhena siddhir vyarthaśramo bhavati dikpratilomalagen // 8.3 //
śastaṃ divā dinabale niśi naktavīrye rāśau viparyayabale gamanaṃ na śastam /
indvarkalagnasahiteṣu gamaścarādyeṣv anyasvakobhayamahīprativṛddhidātā // 8.4 //
mīnodaye pravasataḥ kuṭilaḥ sadoṣo mārge bhavaty akṛtakāryanivartanam ca /
anyāṃśakeṣv api jhaṣe phalam etad eva mīnāṃśakeṣu ca pararkṣasamāśriteṣu // 8.5 //
ripunidhane ripunidhanaṃ ripuṣaṣṭhe lagnage vadho yātuḥ /
satyānuśāsanam idaṃ vāsiṣṭhe nāyam ekāntaḥ // 8.6 //
śatroḥ prasūtisamaye yadi nidhanaṃ saṃyutaṃ grahaiḥ pāpaiḥ /
saumyayutaṃ vā ṣaṣṭhaṃ sadasatphalatā tatas tābhyām // 8.7 //
kleśād vinā phalam arikṣayam arthasiddhiṃ prāpnoti lagnasahite pravasan svalagne /
arthakṣayaṃ śramamanarthamato dvitīye kalyāṇasaukhyavibhavāgamam āhur eke // 8.8 //
bhṛtyārthavāhanasahāyajayās tṛtīye bandhārthanāśabhayasainyavadhāś caturthe /
mantropajāpavipadātmajabhe 'rivṛddhiḥ ṣaṣṭhe 'rivittabaladīptijayāgamāś ca // 8.9 //
dyūne 'dhvavāhanavipatkṣudatiśramārtir bandho vadhaḥ paribhavo nidhane rujaś ca /
dharme 'rthanāśagadakāryavipadbhayāni kāryaṃ vinā balabhayaṃ daśame kṣayaś ca // 8.10 //
kecid vadanty upacayopagṛhītam etat tasmāc chubhaṃ daśamabhe gamanaṃ vilagen /
arthāptidīptiśubhasiddhijayāś ca lābhe ripphe chalaṃ vyaaybhaye vijite 'pi bhedaḥ // 8.11 //
rogāj janmagṛhodaye sutavadho 'siddhiś ca tatpañcame dyūne kleśam avāpya janmabhavanāt prāpnoti paścāt sukham /
mārgād eva nivartanaṃ navamabhe meṣūraṇe 'rthāgamaḥ śeṣarkṣeṣu yathaiva lagnabhavanāt tadvat phalaṃ janmataḥ // 8.12 //
kṣuttṛṣṇārtimārganāśo 'kṣirogaḥ kleśāvāptis tigmagoḥ prāgvilagne /
śastā cāndre devatārthaṃ svadeśe kauje pittavyālaśastrāgnipīḍā // 8.13 //
baudhe tuṣṭir vāṃchitāptir yaśaś ca jaive 'thāptiḥ sthānamānārināśāḥ /
strīratnāptiḥ kāryasiddhiś ca śaukre mānde bandhavyādhinīcāvamānāḥ // 8.14 //
saumyo 'pi janmani na yaḥ śubhapuṣṭidātā sthānaṃ na tasya śubhadaṃ vrajato vilagen /
pāpo 'pi yaḥ śubhaphalaṃ prakaroti puṃsāṃ sthānaṃ vilagnagatam iṣṭam uśanti tasya // 8.15 //
avidheyaṃ bhavanaṃ yat svajanmalagnarkṣayoḥ prayātṛṇām /
apy anukūlaṃ lagnaṃ dhanakṣayāyāsadaṃ bhavati // 8.16 //
sakhivaśyatām upete svajanmalagnarkṣayor vilagnarkṣe /
apacayakare 'pi yātur jayadhanamānāgamāḥ kṣipram // 8.17 //
kecid vadanti yasmin graho niraṃśaṃ karoti tallagnam /
bhavanaṃ yātur aniṣṭaṃ bhayaśokodvegadaṃ yasmād // 8.18 //
vṛṣavṛścikakarkaṭair nṛṇām anukūlair api lagnam āśritaiḥ /
gamanaṃ pravadnay aśobhanaṃ munayo 'nyarkṣasamāśritair api // 8.19 //
veśirvilagnopagato yiyāsor vināpi yatnāt kurute phalāptim /
gauyānam iṣṭaṃ jalarāśilagne tad aṃśake cānyagṛhodaye 'pi // 8.20 //
||chap.9 lagnabheda
ūrdhvā tiryag vānatā syāc ca horā muktā krāntā nāgatārkā krameṇa /
ity etāḥ syur jīvaśarmopadeśe jñeyo rāśir brahmaṣaṇḍasya pakṣe // 9.1 //
ūrdhvā syād yā ṣaṭcarādyeṣu horā bhūyo bhūyo maṇḍale kaiścid uktā /
rāśau rāśau prāgvilagnopayātaiḥ drekkāṇais tu prāha horāṃ vasiṣṭhaḥ // 9.2 //
sarvaṃ pramāṇaṃ municoditatvāt kiṃ tv atra yojyaṃ dalam eva horā /
yasmin sahasrāṃśuravasthito 'rdhe tiryaṅmukhī sā gaṇaye tato 'nyāḥ // 9.3 //
dhatte vāṃchitakāryam ūrdhvavadanā kleśād vinā lagnagā kleśāyāsaparikṣayāṃś ca kurute tiryaṅmukhī gacchataḥ /
sainyabhraṃśam adhomukhī prakurute kṛcchrād gṛhe cāgamaṃ srvāḥ puṣṭaphalapradāḥ svapatinā dṛṣṭā na pāpagrahaiḥ // 9.4 //
grahadivasaphalaṃ yad eva yātur tad akhilam eva karoti tasya horā /
himasalilasṛjo vināha kaścid yuvatisamāśrayaratnadeti cāndrī // 9.5 //
drekkāṇākāraceṣṭāguṇasadṛśaphalaṃ yojayed vṛddhihetor drekkāṇe saumyarūpe kusumaphalayute ratnabhāṇḍānvite ca /
saumyair dṛṣṭe jayaḥ syāt praharaṇasahite pāpadṛṣṭe ca bhaṅgaḥ sāgnau dāho 'tha bandhaḥ sabhujaganigale pāśayukte ca yātuḥ // 9.6 //
nṛpabhūrtir lagnagate drekkāṇe mantriṇo dvitīye ca /
vaidyapurohitasāmbatsarās tṛtīye pare bhṛtyāḥ // 9.7 //
sthānaparākramacintā vyāpāraparākramau camūpabalam /
abhyavahāryaṃ śayanāsane ca tatpaṃcasu kramaśaḥ // 9.8 //
yānāsanaśayyāvāhanāni daśame 'pare 'nnapānāni /
vāhanayodhāḥ paratas trayodaśe bhavati yuvarājā // 9.9 //
jñeyaṃ balaṃ prayātur mantrasya viniścayaś ca tatparatah /
tarataḥ pañcadaśādye drekkāṇacatuṣṭaye ripavaḥ // 9.10 //
ekonaviṃśake sainikāmbuśayane dhanāni parato 'rthāḥ /
parato 'sya daṇḍanetā saimyasyopadravaḥ prarataḥ // 9.11 //
senācchidraṃ tasmāt senānetā bhavec caturviṃśe /
sainyārogyaṃ sainyaṃ catuṣpadaṃ ca kramāt tritaye // 9.12 //
kāryaṃ kośaḥ phalasiddhayas traye bhūmipās traye parataḥ /
dharmakriyātha yodhārcanaṃ ca yātrāsamāptiś ca // 9.13 //
ity udayādyā bhāvā drekkāṇair ye mayā samuddiṣṭāḥ /
sadasatphalam ādeśyaṃ sadasadyutavīkṣaṇāt teṣām // 9.14 //
navabhāge tigmāṃśorvāhananāśo vilagnasaṃprāpte /
kṛcchrāt svagṛhāgamanaṃ pratāpamṛdutā ca candrāṃśe // 9.15 //
kauje 'gnibhayaṃ baudhe mitraprāptir dhanāgamo jaive /
bhogavivṛddhiḥ śaukre bhṛtyavināśo ravisutāṃśe // 9.16 //
yad udayati phalaṃ grahe pradiṣṭaṃ janayati tasya navāṃśako vlagne /
subhabhavananavāṃśe sahāyo ripubalabhāgam upaiti yātur atra // 9.17 //
yat proktaṃ rāśy udaye dvādaśabhāge 'pi tat phalaṃ vācyam /
yac ca navāṃśakavihitaṃ triṃśāṃśasyodaye tat syāt // 9.18 //
||chap.10 grahasthānabala
santāpaśokagadavighnakṛd udgame 'rkaḥ kalyāṇamānabalahārdiharo dvitīye /

putrāpadaṃ sutagṛhe 'dhvani cārthasiddhiṃ ṣaṣṭhe 'bhivāṃchitaphalāptim arikṣayaṃ ca /
dyūne kalatrakalahaṃ dhanasaṃkṣayaṃ ca mṛtyuṃ karoti nidhane savitā rujaṃ ca // 10.2 //
dharmaṃ hinasti navame savitārthadaś ca hatvā viyatyaviditaṃ śramakarmadātā /
ratnāgamaṃ subahu lābhagataḥ karoti kṛtvā vyayaṃ vyayagataḥ kurute 'rtham alpam // 10.3 //

dharmaṃ na sādhayati dharmagato mahījaḥ śasto 'mbare na śubhadaḥ kathito 'pariś ca /

kīrtir lagne 'rthāthasiddhir dvitīye duścikyasthe kṣucchramārtiḥ surejye /
pātālasthe dharmatatvābhimānā kāryaṃ sidhyaty ātmajasthe 'py asādhyam // 10.12 //
ṣaṣṭhe jīve śatrurāyāti vaśyaṃ kecit prāhur vaśyatāṃ yāti śatroḥ /
vindanty aste 'riṣṭayoṣāyaśāṃsi mṛtyau prāṇān hanty athānye jagur na // 10.13 //
putrotpattir dharmasiddhiś ca dharme jīve karmaṇy arthasiddhir yaśaś ca /
lābhe kāryaṃ vāñchitaṃ yāti siddhiḥ riṣphe prāpte kliśyate 'nekaduḥkhaiḥ // 10.14 //
veśyārthāmbaramālyabhojanasukhaprāptir vilagne bhṛgau lābho 'rthe sahaje na sīdati gataḥ prāpnoti śreṣṭhāṃ śrutim /

dattvā strīdhanam astagaḥ svaviṣayavyucchittido bhārgavaḥ kāryaṃ sādhayate 'ṣṭame 'tha navame kṣipraṃ karotīpsitam /
yātuḥ karmagataḥ prabhūtadhanado lābhe jayārthaprado vyarthaṃ dvādaśago vyayaṃ prakurute śasto 'parair dvādaśe // 10.16 //

||chap.11 utsargāpavāda
saumyāsaumyeṣv etal lagnādstheṣu yat phalaṃ proktam /

janmasamaye śaśāṅkodayopacayasaṃsthitā grahāḥ kecit /

avaśeṣasthānagato vadhabandhodvegadaḥ kṣipram // 11.20 //
uḍupatir udayaṃ prāptaḥ sarvasthānopagaḥ prayātṛṇām /

saptāhāntarbalabhayakarī vṛṣṭir anyartujātā kecid yātrām asitajaladaiḥ protthitāṃ pūjayanti /
citrābhrāyāṃ balapativadho ruk ca pītāmbudāyāṃ kṣc coṣṇāyāṃ bhavati na cirād ambudāyāṃ vināśaḥ // 31.7 //
vāhanāni savyago hanti yoṣito 'nyathā /
paṃjarākṛtisthitaḥ sarvato baleśvaram // 31.8 //
cāpam aindram anulomam akhaṇḍaṃ projjvalaṃ bahulam āyatam iṣṭam /
sphūrjanaṃ salilakumbhaniṣekaṃ kṣobhitārṇavasamaṃ vijayāya // 31.9 //
tanvī pralambā vijayāya dīrghā taḍid ghanasphūrjathuvarjitā ca /
śastāśaniś cāpy asamīpajātaḥ pradakṣiṇenāpsyati pātaśabdaḥ // 31.10 //
lagne 'rkendū nighnatī śasti paurān ulkā hanyād yātupaurān grahāś ca /
yātuḥ śastā sambhṛtā vā dhruvarkṣe neṣṭāto 'nyā dhūminī śyāvaraktā // 31.11 //
hanyān nṛpapaurarāṣṭrikān bhṛtyāmbhoruhataskarān dvijāṃś ca /
velā kramaśo dināditaḥ kāṣṭhāyāṃ ca sa yatra bhūsvanaś ca // 31.12 //
śastā śāntadvijamṛgaghuṣṭā sandhyā snigdhā mṛduṣavanā ca /
pāṃśudhvastā janapadanāśaṃ dhatte rūkṣā rudhiranibhā vā // 31.13 //
||chap.32 utpāta
pariveṣo 'rkaśaśinoḥ śatrupakṣagrahaiḥ saha /
snigdho 'khaṇḍaś ca jayadaḥ pāpado bahumaṇḍalaḥ // 32.1 //
pratilomo 'tibahulo neṣṭa iṣṭaḥ pradakṣiṇaḥ /
nīhāraḥ pāṃśupātaś ca hitaḥ snigdhaḥ pradakṣiṇaḥ // 32.2 //
gandharvanagaraṃ hanti prākprabhṛty avanīśvaram /
baleśaṃ yuvarājaṃ ca purohitam iti kramāt // 32.3 //
sitādyaṃ brāhmaṇādīṃś ca vidikṣu varṇasaṅkarān /
pradakṣiṇaṃ tu yātrāyāṃ jayadaṃ neṣṭam anyathā // 32.4 //
diśāṃ dāho 'tha jayadaḥ pradakṣiṇa upāgataḥ /
śvabhronnate vadho rājño jayāya vyatyayena ca // 32.5 //
ghṛtāmbupayasāṃ srāve cyute vṛkṣāc ca tatkṣayaḥ /
taile ca nāśo mukhyānāṃ rakte śastrakṛtaṃ bhayam // 32.6 //
madyena tu mitho bhedo rugbhayaṃ dadhinisrave /
kusumāt kusumotpattau paracakrāgamaṃ vadet // 32.7 //
bahutoye 'tisaṃśoṣaḥ kūpe 'sṛkpūrtivisravaḥ /
rājño 'bdārdhād vadhaṃ kuryuḥ sroto 'lpatve parāgamaḥ // 32.8 //
vṛṣṭividyutsvanair vyabhre carasthiraviparyaye /
dinolkāyāṃ ca pīḍyante janā bhūpatibhiḥ saha // 32.9 //
saptāhaṃ santatāny ato vṛṣṭir hanyān narādhipam /
anagnijvalane puṃsāṃ vyādhir āśv eva jāyate // 32.10 //
pīḍā rāṣṭrasya ṣaṇmāsād uṣṇaśītaviparyaye /
māse 'ṣṭame nṛpasyānto nināde paśupakṣiṇām // 32.11 //
nagarāraṇyasattvānām araṇyapurasevane /
rāṣṭrapīḍādhike varṣe mukhyalokasya ca kṣayaḥ // 32.12 //
stambhopalakusūlārcāpīṭhaśayyāpasarpaṇe /
gadane cāpi deśasya nāśaṃ māsatrayād vadet // 32.13 //
gonāgāśvavadho 'bdārdhe vyatyayo vā nṛpāntakṛt /
deśanāśas tathā strīṇāṃ khagāhipaśusambhave // 32.14 //
dhānyarasamadyabhojanaphalapuṣpasnehamāṃsavastrāṇām /
sikatārajorasātmakasaśarkarāṅgāravarṣeṣu // 32.15 //
sutapatnī kṣudrājñā nṛpadeśacamūpamaraṇasaṃkleśāḥ /
bhedo 'pado 'tha mantrivyādhivināśabhūpakrodhāḥ // 32.16 //
niśi suracāpaṃ sitavarṇādyaṃ janayati pīḍāṃ dvijapūrvāṇām /
bhavati ca yasyāṃ diśi taddiśyaṃ narapatimukhyaṃ na cirād dhanyāt // 32.17 //
mṛgapaśurāsabhavājigajānāṃ viyataravo na cirād bhayadātā /
vibudhapatidhvajatoraṇapātaḥ phalam idam eva karoti nṛpāṇām // 32.18 //
||chap.33 puradurgālabdho
śuddhair dvādaśakendranaidhanagataiḥ pāpais triṣaṣṭhāyagair lagne kendragate 'thavā suragurau daiteyapūjye 'thavā /
sarvārambhaphalaprasiddhir udaye rāśau ca kartuḥ śubhe svagrāmyasthirabhodaye subhavanaṃ kāryaṃ praveśo 'pi vā // 33.1 //
krūrodaye tadahani vyatipātayoge rikte tithāv ṛtusamāptiṣu vaidhṛte vā /
tīkṣṇograbheṣu ca hutāśaviṣābhighātaraudrāṇi siddhim upayāny asite ca pakṣe // 33.2 //
ketūlkārkajarāhukīlakakujā bimbapraviṣṭā yataḥ sūryendvoḥ pariveṣakhaṇḍam athavā dṛśyeta yasyāṃ diśi // 33.3 //
kroṣṭuśvāhipipīlikāśaśamṛgādhvāṃkṣādayo vā pure sainye vāpi yato viśanti hi tataḥ śastroḥ puraṃ ghātayet // 33.4 //
pātālarkṣe rāhuketvoḥ pure 'res toyocchittiḥ sālapātaś ca kāryaḥ /
jāmitrasthe bhūmije 'syāṃśake vā putreṇendor vīkṣite 'gniḥ pradeyaḥ // 33.5 //
||chap.34 jayottara
digdāhakṣatajarajośmavṛṣṭipātair nirghātakṣiticalanād ivaikṛtaiś ca /
yuddhānte mṛgaśakunaiś ca dīptanādair no bhadraṃ bhavati jaye 'pi pārthivasya // 34.1 //
śubhā mṛgapatatriṇo mṛdusamīraṇo hlādakṛt grahāḥ sphuṭamarīcayo vigatareṇudiṅmaṇḍalam // 34.2 //
yadānyad api vaikṛtaṃ na vijayāvasāne bhavet tadā sukham akaṇṭakaṃ nṛpatir atti deśaṃ ripoḥ /
udvāham akālotsavam abhiṣekaṃ cātmajasya yaḥ kṛtvā /
prasthānāvihataḥ so 'bhyeti tataś cotsavadineṣu // 34.3 //
paraviṣayapurāptau sādhudevadvijasvaṃ kulajanavanitāś ca kṣmādhipo noparundhyāt /
vigajaturagaśastrānnārtibhītāṃś ca hanyāc chubhatithidivasarkṣe hṛṣṭasainyo vaśec ca // 34.4 //
svaviṣayam upagamya mānavendro 'valim upayācitakāni cādhikāni /
nigaditavidhinaiva saṃpradadyāt prathamagaṇāsurabhūtadaivatebhyaḥ // 34.5 //
iti manujapatir yathopadeśaṃ bhagaṇavidāṃ prakaroti yo vacāṃsi /
sa sakalanṛpamaṇḍalādhipatyaṃ vrajati diviva purandaro 'cireṇa // 34.6 //
