#Text:Mālinīvijayottaratantra
#Data entry:Somadeva Vasudeva
#Edition:Adhikaras 1-4, 7, 11-17: critical edition by Somadeva Vasudeva, Wolfson College, Oxford, July 2000 (critical apparatus not included) (also publ. Pondichery c2004, Collection indologie, 97); Adhikaras 5, 6, 8-10, 18-23: ed. by Madhusudan Kaul Shastri, Bombay 1922 (Kashmir series of texts and studies ; 37).


||sṛṣṭyadhikāraḥ prathamaḥ
jayanti jagadānandavipakṣakṣapaṇakṣamāḥ // 1.1ab //
parameśamukhodbhūtajñānacandramarīcayaḥ // 1.1cd //
jagadarṇavamagnānāṃ tārakaṃ tārakāntakam // 1.2ab //
sanatkumārasanakasanātanasanandanāḥ // 1.2cd //
nāradāgastyasaṃvartavasiṣṭhādyā maharṣayaḥ // 1.3ab //
jijñāsavaḥ paraṃ tattvaṃ śivaśaktyunmukhīkṛtāḥ // 1.3cd //
samabhyarcya vidhānena te tam ūcuḥ praharṣitāḥ // 1.4ab //
bhagavadyogasaṃsiddhikāṅkṣiṇo vayam āgatāḥ // 1.4cd //
sā ca yogaṃ vinā yasmān na bhavet tam ato vada // 1.5ab //
ṛṣibhir yogam icchadbhiḥ sa tair evam udāhṛtaḥ // 1.5cd //
pratyuvāca prahṛṣṭātmā namaskṛtya maheśvaram // 1.6ab //
śṛṇudhvaṃ saṃpravakṣyāmi sarvasiddhiphalapradam // 1.6cd //
mālinīvijayaṃ tantraṃ parameśamukhodgatam // 1.7ab //
bhuktimuktipradātāram umeśam amarārcitam // 1.7cd //
svasthānastham umā devī praṇipatyedam abravīt // 1.8ab //
siddhayogeśvarītantraṃ navakoṭipravistaram // 1.8cd //
yat tvayā kathitaṃ pūrvaṃ bhedatrayavisarpitam // 1.9ab //
mālinīvijaye tantre koṭitritayalakṣite // 1.9cd //
yogamārgas tvayā proktaḥ suvistīrṇo maheśvara // 1.10ab //
bhūyas tasyopasaṃhāraḥ prokto dvādaśabhis tathā // 1.10cd //
sahasraiḥ so 'pi vistīrṇo gṛhyate nālpabuddhibhiḥ // 1.11ab //
atas tam upasaṃhṛtya samāsād alpadhīhitam // 1.11cd //
sarvasiddhikaraṃ brūhi prasādāt parameśvara // 1.12ab //
evam uktas tadā devyā prahasyovāca viśvarāṭ // 1.12cd //
śṛṇu devi pravakṣyāmi siddhayogeśvarīmatam // 1.13ab //
yan na kasya cid ākhyātaṃ mālinīvijayottaram // 1.13cd //
mayāpy etat purā prāptam aghorāt paramātmanaḥ // 1.14ab //
upādeyaṃ ca heyaṃ ca vijñeyaṃ paramārthataḥ // 1.14cd //
śivaḥ śaktiḥ savidyeśā mantrā mantreśvarāṇavaḥ // 1.15ab //
upādeyam iti proktam etat ṣaṭkaṃ phalārthinām // 1.15cd //
malaḥ karma ca māyā ca māyīyam akhilaṃ jagat // 1.16ab //
sarvaṃ heyam iti proktaṃ vijñeyaṃ vastu niścitam // 1.16cd //
etaj jñātvā parityajya sarvasiddhiphalaṃ labhet // 1.17ab //
tatreśaḥ sarvakṛc chāntaḥ sarvajñaḥ sarvakṛt prabhuḥ // 1.17cd //
sakalo niṣkalo 'nantaḥ śaktir apy asya tadvidhā // 1.18ab //
sa sisṛkṣur jagat sṛṣṭer ādāv eva nijecchayā // 1.18cd //
vijñānakevalān aṣṭau bodhayām āsa pudgalān // 1.19ab //
aghoraḥ paramo ghoro ghorarūpas tadānanaḥ // 1.19cd //
bhīmaś ca bhīṣaṇaś caiva vamanaḥ pivanas tathā // 1.20ab //
etān aṣṭau sthitidhvaṃsarakṣānugrahakāriṇaḥ // 1.20cd //
mantramantreśvare śuddhe saṃniyojya tataḥ punaḥ // 1.21ab //
mantrāṇām asṛjat tadvat saptakoṭīḥ samaṇḍalāḥ // 1.21cd //
sarve 'py ete mahātmāno mantrāḥ sarvaphalapradāḥ // 1.22ab //
ātmā caturvidho jñeyas tatra vijñānakevalaḥ // 1.22cd //
malaikayuktas tatkarmayuktaḥ pralayakevalaḥ // 1.23ab //
malam ajñānam icchanti saṃsārāṅkurakāraṇam // 1.23cd //
dharmādharmātmakaṃ karma sukhaduḥkhādilakṣaṇam // 1.24ab //
īśvarecchāvaśād asya bhogecchā saṃprajāyate // 1.24cd //
bhogasādhanasaṃsiddhyai bhogecchor asya mantrarāṭ // 1.25ab //
jagad utpādayām āsa māyām āviśya śaktibhiḥ // 1.25cd //
sā caikā vyāpinī sūkṣmā niṣkalā jagato nidhiḥ // 1.26ab //
anādyantāśiveśānī vyayahīnā ca kathyate // 1.26cd //
asūta sā kalātattvaṃ yad yogād abhavat pumān // 1.27ab //
jātakartṛtvasāmarthyo vidyārāgau tato 'sṛjat // 1.27cd //
vidyā vivecayaty asya karma tatkāryakāraṇe // 1.28ab //
rāgo 'nurañjayaty enaṃ svabhogeṣv aśuciṣv api // 1.28cd //
niyatir yojayaty enaṃ svake karmaṇi pudgalam // 1.29ab //
kālo 'pi kalayaty enaṃ tuṭyādibhir avasthitaḥ // 1.29cd //
tata eva kalātattvād avyaktam asṛjat tataḥ // 1.30ab //
guṇān aṣṭaguṇāṃ tebhyo dhiyaṃ dhīto 'py ahaṅkṛtim // 1.30cd //
tat tridhā taijasāt tasmān mano 'kṣeśam ajāyata // 1.31ab //
vaikārikāt tato 'kṣāṇi tanmātrāṇi tṛtīyakāt // 1.31cd //
śrotraṃ tvakcakṣuṣī jihvā ghrāṇaṃ buddhīndriyāṇi tu // 1.32ab //
karmendriyāṇi vākpāṇipāyūpasthāṅghrayaḥ kramāt // 1.32cd //
kalādikṣitiparyantam etat saṃsāramaṇḍalam // 1.33ab //
samudrādi jagat kṛtsnaṃ parivartayatīcchayā // 1.33cd //
bhedaḥ paraḥ kalādīnāṃ bhuvanatvena yaḥ sthitaḥ // 1.34ab //
asṛjat tam asāv eva bhogināṃ bhogasiddhaye // 1.34cd //
ity anena kalādyena dharāntena samanvitāḥ // 1.35ab //
pumāṃsaḥ sakalā jñeyās tadavasthājighāṃsubhiḥ // 1.35cd //
avasthātritaye 'py asmiṃs tirobhāvanaśīlayā // 1.36ab //
śivaśaktyā samākrāntāḥ prakurvanti viceṣṭitam // 1.36cd //
evaṃ jagati sarvatra rudrāṇāṃ yogyatāvaśāt // 1.37ab //
aṅguṣṭhamātrapūrvāṇāṃ śatam aṣṭādaśottaram // 1.37cd //
anugṛhya śivaḥ sākṣān mantreśatve niyuktavān // 1.38ab //
te svagocaram āsādya bhuktimuktiphalārthinām // 1.38cd //
brahmādīnāṃ prayacchanti svabalena samaṃ phalam // 1.39ab //
ṛṣibhyas te 'pi te cānu manvantebhyo mahādhipāḥ // 1.39cd //
heyopādeyavijñānaṃ kathayanti śivoditam // 1.40ab //
brahmādistambaparyante jātamātre jagaty alam // 1.40cd //
mantrāṇāṃ koṭayas tisraḥ sārdhāḥ śivaniyojitāḥ // 1.41ab //
anugṛhyāṇusaṃghātaṃ yātāḥ padam anāmayam // 1.41cd //
evam asyātmanaḥ kāle kasmiṃś cid yogyatāvaśāt // 1.42ab //
śaivī saṃbadhyate śaktiḥ śāntā muktiphalapradā // 1.42cd //
tatsaṃbandhāt tataḥ kaś cit tatkṣaṇād apavṛjyate // 1.43ab //
ajñānena sahaikatvaṃ kasya cid vinivartate // 1.43cd //
rudraśaktisamāviṣṭaḥ sa yiyāsuḥ śivecchayā // 1.44ab //
bhuktimuktiprasiddhyarthaṃ nīyate sadguruṃ prati // 1.44cd //
tam ārādhya tatas tuṣṭād dīkṣām āsādya śāṅkarīm // 1.45ab //
tatkṣaṇād vopabhogād vā dehapāte śivaṃ vrajet // 1.45cd //
yogadīkṣāṃ samāsādya jñātvā yogaṃ samabhyaset // 1.46ab //
yogasiddhim avāpnoti tadante śāśvataṃ padam // 1.46cd //
anena kramayogena saṃprāptaḥ paramaṃ padam // 1.47ab //
na bhūyaḥ paśutām eti śuddhe svātmani tiṣṭhati // 1.47cd //
ātmā caturvidho hy eṣa punar eṣa caturvidhaḥ // 1.48ab //
ācāryatvādibhedena śuddhātmā paripaṭhyate // 1.48cd //
nityāditritayaṃ kuryād guruḥ sādhaka eva ca // 1.49ab //
nityam eva dvayaṃ cānyo yāvajjīvaṃ śivājñayā // 1.49cd //
upādeyaṃ ca heyaṃ ca tad etat parikīrtitam // 1.50ab //
jñātvaitaj jñeyasarvasvaṃ sarvasiddhyaraho bhavet // 1.50cd //
||iti śrīmālinīvijayottare tantre prathamo 'dhikāraḥ samāptaḥ
||vyāptyadhikāro dvitīyaḥ
athaiṣām eva tattvānāṃ dharādīnām anukramāt // 2.1ab //
prapañcaḥ kathyate leśād yogināṃ yogasiddhaye // 2.1cd //
śaktimacchaktibhedena dharātattvaṃ vibhidyate // 2.2ab //
svarūpasahitaṃ tac ca vijñeyaṃ daśapañcadhā // 2.2cd //
śivādisakalātmāntāḥ śaktimantaḥ prakīrtitāḥ // 2.3ab //
tacchaktayaś ca vijñeyās tadvad eva vicakṣaṇaiḥ // 2.3cd //
evaṃ jalādimūlāntaṃ tattvavrātam idaṃ mahat // 2.4ab //
pṛthag bhedair imair bhinnaṃ vijñeyaṃ tatphalepsubhiḥ // 2.4cd //
anenaiva vidhānena puṃstattvāt tu kalāntikam // 2.5ab //
trayodaśavidhaṃ jñeyaṃ rudravat pralayākalaḥ // 2.5cd //
tadvan māyāpi vijñeyā navadhā jñānakevalaḥ // 2.6ab //
mantrāḥ saptavidhās tadvat pañcadhā mantranāyakāḥ // 2.6cd //
tridhā mantreśvareśānāḥ śivaḥ sākṣān na bhidyate // 2.7ab //
bhedaḥ prakathito leśād ananto vistarād ayam // 2.7cd //
evaṃ bhuvanamālāpi bhinnā bhedair imaiḥ sphuṭam // 2.8ab //
vijñeyā yogasiddhyarthaṃ yogibhir yogapūjitā // 2.8cd //
eteṣām eva tattvānāṃ bhuvanānāṃ ca śāṅkari // 2.9ab //
ya ekam api jānāti so 'pi yogaphalaṃ labhet // 2.9cd //
yaḥ punaḥ sarvatattvāni vetty etāni yathārthataḥ // 2.10ab //
sa gurur matsamaḥ prokto mantravīryaprakāśakaḥ // 2.10cd //
dṛṣṭāḥ saṃbhāṣitās tena spṛṣṭāś ca prītacetasā // 2.11ab //
narāḥ pāpaiḥ pramucyante saptajanmakṛtair api // 2.11cd //
ye punar dīkṣitās tena prāṇinaḥ śivacoditāḥ // 2.12ab //
te yatheṣṭaṃ phalaṃ prāpya gacchanti paramaṃ padam // 2.12cd //
rudraśaktisamāveśas tatra nityaṃ pratiṣṭhitaḥ // 2.13ab //
sati tasmiṃś ca cihnāni tasyaitāni vilakṣayet // 2.13cd //
tatraitat prathamaṃ cihnaṃ rudre bhaktiḥ suniścalā // 2.14ab //
dvitīyaṃ mantrasiddhiḥ syāt sadyaḥpratyayakārikā // 2.14cd //
sarvasattvavaśitvaṃ ca tṛtīyaṃ lakṣaṇaṃ smṛtam // 2.15ab //
prārabdhakāryaniṣpattiś cihnam āhuś caturthakam // 2.15cd //
kavitvaṃ pañcamaṃ proktaṃ sālaṃkāraṃ manoharam // 2.16ab //
sarvaśāstrārthavettṛtvam akasmāc cāsya jāyate // 2.16cd //
rudraśaktisamāveśaḥ pañcadhā paripaṭhyate // 2.17ab //
bhūtatattvātmamantreśaśaktibhedād varānane // 2.17cd //
pañcadhā bhūtasaṃjñas tu tathā triṃśatidhā paraḥ // 2.18ab //
ātmākhyas trividhaḥ prokto daśadhā mantrasaṃjñakaḥ // 2.18cd //
dvividhaḥ śaktisaṃjño 'pi jñātavyaḥ paramārthataḥ // 2.19ab //
pañcāśadbhedabhinno 'yaṃ samāveśaḥ prakīrtitaḥ // 2.19cd //
āṇavo 'yaṃ samākhyātaḥ śākto 'py evaṃvidhaḥ smṛtaḥ // 2.20ab //
evaṃ śāmbhavam apy ebhir bhedair bhinnaṃ vilakṣayet // 2.20cd //
uccārakaraṇadhyānavarṇasthānaprakalpanaiḥ // 2.21ab //
yo bhavet sa samāveśaḥ samyag āṇava ucyate // 2.21cd //
uccārarahitaṃ vastu cetasaiva vicintayan // 2.22ab //
yam āveśam avāpnoti śāktaḥ so 'trābhidhīyate // 2.22cd //
akiṃciccintakasyaiva guruṇā pratibodhataḥ // 2.23ab //
jāyate yaḥ samāveśaḥ śāmbhavo 'sāv udīritaḥ // 2.23cd //
sārdham etac chataṃ proktaṃ bhedānām anupūrvaśaḥ // 2.24ab //
saṃkṣepād vistarād asya parisaṃkhyā na vidyate // 2.24cd //
saṃvittiphalabhedo 'tra na prakalpyo manīṣibhiḥ // 2.25ab //
bhedo 'paro 'pi saṃkṣepāt kathyamāno 'vadhāryatām // 2.25cd //
jāgratsvapnādibhedena sarvāveśakramo budhaiḥ // 2.26ab //
pañcabhinnaḥ parijñeyaḥ svavyāpārāt pṛthak pṛthak // 2.26cd //
tatra svarūpaṃ śaktiś ca sakalaś ceti tattrayam // 2.27ab //
iti jāgradavastheyaṃ bhede pañcadaśātmake // 2.27cd //
akalau dvau parijñeyau samyak svapnasuṣuptayoḥ // 2.28ab //
mantrāditatpatīśānavargas turya iti smṛtaḥ // 2.28cd //
śaktiśaṃbhū parijñeyau turyātīte varānane // 2.29ab //
trayodaśātmake bhede svarūpam akalāv ubhau // 2.29cd //
mantramantreśvareśānāḥ śaktiśaṃbhū ca kīrtitau // 2.30ab //
pralayākalabhede 'pi svaṃ vijñānakalāv ubhau // 2.30cd //
mantramantreśvareśānāḥ śaktīśāv api pūrvavat // 2.31ab //
navadhā kīrtite bhede svaṃ mantrā mantranāyakāḥ // 2.31cd //
tadīśāḥ śaktiśaṃbhū ca pañcāvasthāḥ prakīrtitāḥ // 2.32ab //
pūrvavat saptabhede 'pi svaṃ mantreśeśaśaktayaḥ // 2.32cd //
śivaś ceti parijñeyāḥ pañcaiva varavarṇini // 2.33ab //
svaṃ śaktiḥ sanijeśānā śaktiśaṃbhū ca pañcake // 2.33cd //
trike svaśaktiśaktīcchāśivapadaṃ vilakṣayet // 2.34ab //
svavyāpārādhipatvena taddhīnaprerakatvataḥ // 2.34cd //
icchānivṛtteḥ svasthatvād abhinnam api pañcadhā // 2.35ab //
iti pañcātmake bhede vijñeyaṃ vastu kīrtitam // 2.35cd //
bhūyo 'py āsām avasthānāṃ saṃjñābhedaḥ prakāśyate // 2.36ab //
piṇḍasthaḥ sarvatobhadro jāgrannāmadvayaṃ matam // 2.36cd //
dvisaṃjñaṃ svapnam icchanti padasthaṃ vyāptir ity api // 2.37ab //
rūpasthaṃ tu mahāvyāptiḥ suṣuptasyāpi taddvayam // 2.37cd //
pracayaṃ rūpātītaṃ ca samyak turyam udāhṛtam // 2.38ab //
mahāpracayam icchanti turyātītaṃ vicakṣaṇāḥ // 2.38cd //
pṛthak tattvaprabhedena bhedo 'yaṃ samudāhṛtaḥ // 2.39ab //
sarvāṇi eva tattvāni pañcaitāni yathā śṛṇu // 2.39cd //
bhūtatattvābhidhānānāṃ yo 'ṃśo 'dhiṣṭheya iṣyate // 2.40ab //
piṇḍastham iti taṃ prāhuḥ padastham aparaṃ viduḥ // 2.40cd //
mantrās tatpatayaḥ seśā rūpastham iti kīrtyate // 2.41ab //
rūpātītaṃ parā śaktiḥ savyāpārāpy anāmayā // 2.41cd //
niṣprapañco nirābhāsaḥ śuddhaḥ svātmany avasthitaḥ // 2.42ab //
sarvātītaḥ śivo jñeyo yaṃ viditvā vimucyate // 2.42cd //
caturvidhaṃ tu piṇḍastham abuddhaṃ buddham eva ca // 2.43ab //
prabuddhaṃ suprabuddhaṃ ca padasthaṃ ca caturvidham // 2.43cd //
gatāgataṃ suvikṣiptaṃ saṅgataṃ susamāhitam // 2.44ab //
caturdhā rūpasaṃsthaṃ tu jñātavyaṃ yogacintakaiḥ // 2.44cd //
uditaṃ vipulaṃ śāntaṃ suprasannam athāparam // 2.45ab //
manonmanam anantaṃ ca sarvārthaṃ satatoditam // 2.45cd //
pracaye tatra saṃjñeyam ekaṃ tanmahati sthitam // 2.46ab //
ity evaṃ pañcadhādhvānaṃ tridhedānīṃ nigadyate // 2.46cd //
vijñānākalaparyantam ātmatattvam udāhṛtam // 2.47ab //
īśvarāntaṃ ca vidyāhvaṃ śeṣaṃ śivapadaṃ viduḥ // 2.47cd //
evaṃ bhedair imair bhinnas tatrādhvā parikīrtitaḥ // 2.48ab //
yugapat sarvamārgāṇāṃ prabhedaḥ procyate 'dhunā // 2.48cd //
pārthivaṃ prākṛtaṃ caiva māyīyaṃ śāktam eva ca // 2.49ab //
iti saṅkṣepataḥ proktam etad aṇḍacatuṣṭayam // 2.49cd //
pṛthag dvayam asaṅkhyātam ekam ekaṃ pṛthak pṛthak // 2.50ab //
ādyaṃ dhārikayā vyāptaṃ tatraikaṃ tattvam iṣyate // 2.50cd //
ekam ekaṃ pṛthak kṣārṇaṃ padārṇamanuṣu smaret // 2.51ab //
kālāgnibhuvanād yāvad vīrabhadrapurottamam // 2.51cd //
puraṣoḍaśakaṃ jñeyaṃ ṣaḍvidho 'dhvā prakīrtitaḥ // 2.52ab //
āpyāyinyā dvitīyaṃ ca tatra tattvāni lakṣayet // 2.52cd //
trayoviṃśatyabādīni tadvad dhādyakṣarāṇi ca // 2.53ab //
padāni pañca mantrāś ca ṣaṭpañcāśatpurāṇi ca // 2.53cd //
tattvāni sapta bodhinyā tac caturdhā purāṇi ca // 2.54ab //
tṛtīye sapta varṇāḥ syuḥ padamantradvayaṃ dvayam // 2.54cd //
utpūyinyā caturthaṃ tu tatra tattvatrayaṃ viduḥ // 2.55ab //
varṇatrayaṃ mantram ekaṃ padam ekaṃ ca lakṣayet // 2.55cd //
aṣṭādaśa vijānīyād bhuvanāni samāsataḥ // 2.56ab //
śivatattvaṃ paraṃ śāntaṃ kalā tatrāvakāśadā // 2.56cd //
svaraṣoḍaśakaṃ mantraṃ padaṃ caikaṃ vilakṣayet // 2.57ab //
ity evaṃ ṣaḍvidho 'py adhvā samāsāt parikīrtitaḥ // 2.57cd //
śuddhāśuddhaṃ jagatsarvaṃ brahmāṇḍaprabhavaṃ yataḥ // 2.58ab //
tasmāc chuddham imaiḥ śuddhair brahmāṇḍaiḥ sarvam iṣyate // 2.58cd //
brahmā viṣṇuś ca rudraś ca īśvaraś ceti suvrate // 2.59ab //
pṛthag eteṣu boddhavyaṃ śāntaṃ paticatuṣṭayam // 2.59cd //
yo hi yasmād guṇotkṛṣṭaḥ sa tasmād ūrdhva ucyate // 2.60ab //
etat te kathitaṃ sarvaṃ kim anyat paripṛcchasi // 2.60cd //
||iti śrīmālinīvijayottare tantre vyāptyadhikāro dvitīyaḥ samāptaḥ
||mantroddhārādhikāras tṛtīyaḥ
evam uktā mahādevī jagadānandakāriṇā // 3.1ab //
praṇipatya punar vākyam idam āha jagatpatim // 3.1cd //
evam etan mahādeva nānyathā samudāhṛtam // 3.2ab //
yathākhyātaṃ tathā jñātam āditaḥ samanukramāt // 3.2cd //
śivādivasturūpāṇāṃ vācakān parameśvara // 3.3ab //
sāṃprataṃ śrotum icchāmi prasādād vaktum arhasi // 3.3cd //
ity uktaḥ sa maheśānyā jagadārtiharo haraḥ // 3.4ab //
vācakān avadan mantrān pāramparyakramāgatān // 3.4cd //
yā sā śaktir jagaddhātuḥ kathitā samavāyinī // 3.5ab //
icchātvaṃ tasya sā devi sisṛkṣoḥ pratipadyate // 3.5cd //
saikāpi saty anekatvaṃ yathā gacchati tac chṛṇu // 3.6ab //
